# 작곡가 대결 (Composer Battle)

> 주어진 재료로 짧은 곡을 만드세요.
> 가장 아름답거나
> 가장 감동적이거나
> 가장 재미있는 곡이 이깁니다.
> 누구나 작곡가가 될 수 있습니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 창의적 표현, 음악적 구성, 감성 표현 |

## 게임 구성

```
음표 카드: 도·레·미·파·솔·라·시 각 8장
리듬 카드: 온음표·2분음표·4분음표·8분음표 각 10장
분위기 카드: 20장 (슬픔·기쁨·긴장·평화 등)
박자 카드: 2/4·3/4·4/4박자
작곡 용지: 4마디 악보 용지
```

## 기본 규칙

```
재료 배분:
  음표 카드 6장 + 리듬 카드 4장
  분위기 카드 1장 공개 (모두 같은 분위기)

작곡 (5분):
  받은 카드로 4마디 멜로디 구성
  작곡 용지에 기보

연주:
  목소리로 "라라라" 또는 악기로 연주

평가:
  분위기 카드와 얼마나 맞는지
  창의성·완성도 종합 평가

득점:
  분위기 일치: +2점
  독창성: +1-3점 (투표)
```

## 작곡 가이드

```
4마디 구조:
  1마디: 주제 제시
  2마디: 주제 반복·변형
  3마디: 대비·전개
  4마디: 마무리·귀결

감정 표현 기법:
  슬픔: 단음계·느린 템포·낮은 음역
  기쁨: 장음계·빠른 템포·도약
  긴장: 반음계·불규칙 리듬
  평화: 순차 진행·중간 템포
```

## 심화 모드

```
협동 작곡:
  2명이 각 2마디씩 담당
  자연스럽게 연결되어야 함

변주 게임:
  한 사람의 멜로디를
  다른 분위기로 변주
```

## 교육적 효과

```
음악 창작:
  이론이 아닌 실습으로 작곡
  → 음악적 자신감

감성 표현:
  감정을 소리로 변환
  → 예술적 표현력

구조 이해:
  음악의 형식과 구조
  → 음악 감상력 향상
```
