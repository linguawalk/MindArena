# 음계 탐정 (Scale Detective)

> 들리는 음이 어떤 음계인지 맞혀보세요.
> 음과 음 사이의 관계를 이해하면
> 음악이 달리 들립니다.
> 귀를 훈련하는 게임입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 음감, 음계 이해, 청각 분석 |

## 게임 구성

```
음계 카드: 장조·단조·펜타토닉·블루스 각 10장
멜로디 카드: 30장 (간단한 멜로디 악보)
음정 카드: 2도·3도·5도·옥타브
악기 선택 카드: 피아노·기타·목소리
```

## 기본 규칙

```
멜로디 연주:
  진행자가 멜로디 카드의 선율 부르거나
  악기로 연주

추측:
  어떤 음계인지 카드 선택
  장조·단조·펜타토닉·블루스

이유 설명:
  왜 그 음계라고 생각하는지
  (밝은 느낌·어두운 느낌·독특한 느낌)

채점:
  정답: +2점
  설명이 맞으면: +1점 (답 틀려도)
```

## 음계별 특징

```
장조 (Major):
  밝고 명랑한 느낌
  도-레-미-파-솔-라-시-도
  예) 학교종·반짝반짝

단조 (Minor):
  어둡고 슬픈 느낌
  도-레-미♭-파-솔-라♭-시♭-도
  예) 장례 행진곡·비창 소나타

펜타토닉 (Pentatonic):
  5음 음계·동양적 느낌
  도-레-미-솔-라
  예) 아리랑·동요 많음

블루스 (Blues):
  구슬프고 감성적
  특수 음정 포함
  예) 재즈·R&B
```

## 상대 음감 훈련

```
음정 맞추기:
  두 음의 간격 추측
  "이 두 음의 거리는?"

화음 감별:
  "이것은 장화음인가 단화음인가?"

조옮김 인식:
  "같은 멜로디를 다른 음높이로
   연주했을 때 같은 곡인가?"
```

## 교육적 효과

```
음감 발달:
  음과 음의 관계 인식
  → 음악적 귀 훈련

감성 어휘:
  음악의 느낌을 언어로 표현
  → 감성 지능

문화 이해:
  음계마다 다른 문화적 배경
  → 음악의 다양성
```
