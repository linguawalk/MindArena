# 음악 기억 (Music Memory)

> 멜로디를 기억하는 것은
> 언어를 기억하는 것과 같습니다.
> 음악적 기억력을 훈련하고
> 음악과 감정의 연결을 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 음악적 기억, 감정-음악 연결, 음악 지식 |

## 게임 구성

```
멜로디 카드: 50장 (유명 곡의 첫 소절)
감정 카드: 기쁨·슬픔·흥분·평화·긴장 각 10장
작곡가 카드: 20장
연대기 카드: 시대별 음악 분류
```

## 기본 규칙

```
모드 1 - 멜로디 맞추기:
  진행자가 멜로디 허밍
  제목·작곡가 맞히기

모드 2 - 감정 연결:
  멜로디 들려주기
  어떤 감정인지 카드 선택
  창작자 의도와 비교

모드 3 - 시대 배치:
  멜로디를 시대순으로 배열
  바로크→고전→낭만→현대

득점:
  각 모드별 정답 +2점
  이유 설명 +1점
```

## 멜로디 카드 예시

```
한국 음악:
  아리랑·애국가·동요 등

서양 클래식:
  베토벤 5번 교향곡 첫 소절
  모차르트 터키 행진곡
  쇼팽 야상곡

대중음악:
  시대별 히트곡
  영화 OST
```

## 음악과 감정 연구

```
과학적 발견:
  장조 → 대체로 긍정적 감정
  단조 → 대체로 부정적 감정
  빠른 템포 → 흥분·기쁨
  느린 템포 → 슬픔·평온

문화 차이:
  서양 단조 = 슬픔
  일부 문화 단조 = 신성·의례
  → 감정-음악 연결은 문화적

개인 차이:
  같은 음악, 다른 감정
  개인 경험과 연결
```

## 교육적 효과

```
음악 지식:
  다양한 장르·시대 음악 접촉
  → 음악 교양

감정 인식:
  음악으로 감정 표현·인식
  → 감성 지능

문화 이해:
  음악을 통한 시대·문화 이해
  → 인문학적 감수성
```
