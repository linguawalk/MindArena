# 보드게임 - 음악·리듬 (Music & Rhythm)

> 음악은 수학이자 언어이자 감정입니다.
> 박자를 느끼고 음계를 이해하며
> 소리로 소통하는 능력을 훈련합니다.
> 귀와 뇌를 동시에 깨우는 게임들입니다.
