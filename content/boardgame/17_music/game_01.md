# 박자 배틀 (Rhythm Battle)

> 박자는 음악의 심장입니다.
> 손뼉·발구르기·목소리로
> 복잡한 리듬을 만들고 따라하세요.
> 리듬 감각이 뛰어난 자가 이깁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 리듬 감각, 청각 기억, 신체 협응 |

## 게임 구성

```
리듬 카드: 40장 (악보 표기)
박자 주사위: 4분음표·8분음표·16분음표
템포 카드: 느림·보통·빠름
점수판
```

## 기본 규칙

```
리듬 카드 공개:
  카드에 적힌 리듬 패턴 확인
  예) 손-손-발-손-쉬기

시연:
  현재 플레이어가 리듬 1회 시연

따라하기:
  다른 플레이어 즉시 따라하기
  정확도 평가 (자기 평가·상호 평가)

난이도 상승:
  성공할수록 리듬 길이 증가
  템포 카드로 속도 조절

득점:
  정확한 재현: +2점
  창작 리듬 제시: +1점 (맞추면 추가 +1)
```

## 리듬 표기법

```
기호:
  ■ = 손뼉 (강박)
  □ = 손뼉 (약박)
  ● = 발 구르기
  ○ = 무릎 치기
  _ = 쉬기

예시 패턴:
  Level 1: ■-■-■-■ (4분음표 4개)
  Level 2: ■-□-□-■-□-□ (3박자 리듬)
  Level 3: ■-●-□-■-_-□-●-■
  Level 4: ■-□-□-□-●-■-□-_-□-●
```

## 신체 리듬 확장

```
오스티나토 (반복 리듬):
  한 명이 반복 리듬 유지
  나머지가 다른 리듬 추가
  → 앙상블 체험

캐논 리듬:
  2박 뒤에 같은 리듬 시작
  → 돌림노래 원리

폴리리듬:
  2박자 vs 3박자 동시
  → 아프리카 음악 원리
```

## 교육적 효과

```
리듬 감각:
  박자 분할·조합 체험
  → 음악의 기초

신체 협응:
  손·발·입의 동시 제어
  → 운동 신경 발달

청각 기억:
  들은 리듬 즉시 재현
  → 청각 처리 능력
```
