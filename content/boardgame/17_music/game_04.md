# 리듬 전달 (Rhythm Pass)

> 귓속말처럼 리듬을 전달합니다.
> 처음과 끝이 같을 수 있을까요?
> 리듬이 왜곡되는 과정을 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 청각 기억, 정확한 전달, 소통 왜곡 이해 |

## 기본 규칙

```
준비:
  일렬로 앉기
  첫 번째 사람이 리듬 카드 확인
  (다른 사람은 보지 못함)

전달:
  첫 번째가 두 번째 어깨 두드려
  리듬 전달 (소리 없이 신체로만)
  두 번째→세 번째→ ... 마지막까지

비교:
  마지막 사람이 리듬 크게 재현
  원본 카드 공개 후 비교

득점:
  완벽 전달: 팀 전체 +5점
  1-2개 오류: +3점
  3개 이상 오류: +1점
```

## 리듬 전달 방법

```
신체 신호만 사용:
  어깨 두드리기: 강박
  등 두드리기: 약박
  꼬집기: 쉬기
  두 번 빠르게: 8분음표

소리 추가 모드:
  "딱"·"툭"·"쉿" 등
  단, 말은 금지
```

## 왜곡 분석

```
어디서 왜곡됐는가?
  각 전달 단계 기록
  오류 발생 지점 찾기

왜 왜곡됐는가?
  청각 기억의 한계
  신체 신호의 모호성
  집중력 저하

개선 방법:
  더 명확한 신호 약속
  반복 확인 허용
  집중 환경 조성
```

## 교육적 효과

```
소통 왜곡:
  정보가 전달되며 변하는 원리
  → 루머·오해 발생 원리 이해

청각 정확성:
  정확하게 듣고 재현
  → 청각 집중력

팀 소통:
  명확한 신호의 중요성
  → 소통 능력 향상
```

## 게임 후 토론

```
"리듬이 전달되면서 어떻게 변했나요?
 왜 그렇게 됐을까요?"

"이것이 실제 소문이나 뉴스가
 퍼지는 방식과 비슷한가요?"

"정확한 소통을 위해
 무엇이 필요한가요?"
```
