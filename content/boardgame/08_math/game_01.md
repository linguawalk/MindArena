# 확률 도박사 (Probability Gambler)

> 주머니 속 공의 색을 맞혀야 합니다.
> 정보가 쌓일수록 확률이 바뀝니다.
> 언제 배팅하고, 언제 기다릴 것인가?
> 수학적 직관이 승패를 가릅니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 확률 직관, 베이즈 추론, 의사결정 |

## 게임 구성

```
불투명 주머니: 5개
컬러 공: 빨강 15개 / 파랑 15개 / 노랑 10개
배팅 칩: 각 플레이어 20개
정보 카드: 30장
```

## 기본 규칙

```
준비:
  주머니 5개에 공을 섞어 넣기
  (구성은 진행자만 알고 있음)
  각 플레이어 칩 20개 지급

라운드 진행:
  ①공개 단계:
    주머니에서 공 1개 꺼내 색 공개
    다시 넣기 (복원 추출)

  ②정보 축적:
    3번 공개 후 배팅 시작 가능

  ③배팅:
    다음 공의 색 예측 + 칩 배팅
    (최소 1개, 최대 보유 칩의 절반)

  ④공개:
    실제 공 색 확인

  ⑤정산:
    맞히면 배팅 칩의 2배 획득
    틀리면 배팅 칩 잃음
```

## 전략 포인트

```
베이즈 업데이트:
  공개될수록 확률 갱신
  예) 빨강 3번 연속 → 빨강 비율 높음 추정

켈리 기준:
  확률 우위에 비례하여 배팅
  전 재산 배팅은 위험

정보 가치:
  몇 번 공개 후 배팅이 최적인가?
```

## 교육적 효과

```
확률 직관:
  이론 확률 → 실제 느낌으로 체득
  → 통계적 사고력 기초

베이즈 추론:
  새 정보로 판단 업데이트
  → 과학적 사고 방식

리스크 관리:
  불확실 상황의 자원 배분
  → 실생활 의사결정 능력
```

## 변형 모드

```
정보 비대칭:
  플레이어마다 다른 공 정보 제공
  → 정보 거래 게임으로 확장

팀 모드:
  팀별 공동 배팅
  → 집단 의사결정 훈련
```
