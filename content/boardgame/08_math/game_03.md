# 균형 저울 (Balance Scale)

> 저울이 기울면 집니다.
> 숫자를 배치해 균형을 맞추거나
> 상대의 균형을 무너뜨리세요.
> 등식의 아름다움을 게임으로.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 등식 이해, 수의 균형, 연산 능력 |

## 게임 구성

```
저울 보드: 양팔 저울 그림 (개인용)
숫자 타일: 1-20 각 3세트
연산 카드: +, -, ×, ÷ 각 10장
목표 카드: 50장 (목표 합계)
```

## 기본 규칙

```
목표:
  목표 카드의 숫자를 만족하는
  등식 완성하기

진행:
  ①목표 카드 공개 (예: "양쪽 합 = 20")
  ②손패 숫자 타일 5장
  ③연산 카드 2장

  ④타일 + 연산으로 등식 구성
    왼쪽 = 오른쪽

  ⑤완성 시: "균형!" 선언
  ⑥검증 후 맞으면 득점

득점:
  기본: +2점
  보너스: 적은 타일 사용 시 추가 점수
  오류 시: -1점
```

## 예시

```
목표: 양쪽 합 = 24

가능한 등식:
  왼쪽: 3 × 8 = 24
  오른쪽: 12 + 12 = 24

또는:
  왼쪽: 6 × 4 = 24
  오른쪽: 20 + 4 = 24
```

## 난이도 조절

```
초급: 덧셈·뺄셈만
중급: 사칙연산
고급: 거듭제곱 포함
심화: 분수·소수 포함
```

## 교육적 효과

```
등식 개념:
  양변이 같아야 한다는 원리
  → 대수학 기초

다양한 풀이:
  같은 답에 다른 경로
  → 수학적 유연성

연산 속도:
  실전으로 빨라지는 계산
  → 수리 능력 향상
```
