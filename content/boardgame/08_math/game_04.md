# 기댓값 경매 (Expected Value Auction)

> 모든 것에는 기댓값이 있습니다.
> 가치를 정확히 계산한 사람이
> 경매에서 이깁니다.
> 과대 지불도, 기회 포기도 패배입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 기댓값 계산, 경매 전략, 의사결정 |

## 게임 구성

```
경매 카드: 60장 (확률 복권형 상품)
자금 칩: 각 플레이어 100점
주사위: 6면체 2개
결과 표: 기댓값 계산용
```

## 기본 규칙

```
경매 카드 구조 예시:
  "주사위 합이 7 이상이면 50점,
   6 이하이면 10점"
  → 기댓값: 50 × (21/36) + 10 × (15/36) ≈ 33.6점

진행:
  ①카드 공개
  ②각자 기댓값 계산 (1분)
  ③비밀 입찰 (칩 배팅)
  ④최고 입찰자 낙찰
  ⑤주사위로 실제 결과 결정
  ⑥낙찰가 차감, 결과값 획득

승리:
  10라운드 후 최다 점수 보유자
```

## 카드 예시

```
카드 A:
  "주사위 1개: 짝수 → 60점 / 홀수 → 0점"
  기댓값: 30점

카드 B:
  "주사위 2개 합: 12 → 200점 / 나머지 → 5점"
  기댓값: 200×(1/36) + 5×(35/36) ≈ 10.4점

카드 C:
  "동전 3번: 모두 앞면 → 80점 / 아니면 20점"
  기댓값: 80×(1/8) + 20×(7/8) = 27.5점
```

## 전략 포인트

```
정확한 계산:
  기댓값보다 싸게 사면 장기적 이득
  비싸게 사면 장기적 손해

경쟁 심리:
  상대가 과대 지불하도록 유도
  내가 원하는 카드는 조용히

포트폴리오:
  고위험 고수익 vs 안정적 수익
  균형 있는 카드 확보
```

## 교육적 효과

```
기댓값 체득:
  수식이 아닌 게임으로 이해
  → 통계적 사고의 핵심

경제적 사고:
  가치 vs 가격 판단
  → 실생활 소비 판단력

감정 통제:
  손실 후 냉정한 재계산
  → 손실 회피 편향 극복
```
