# 진화 게임 (Evolution Game)

> 환경이 변하면 생존자가 바뀝니다.
> 가장 강한 자가 아닌
> 가장 잘 적응한 자가 살아남습니다.
> 다윈의 진화론을 직접 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 적응·진화 사고, 시스템 변화, 자연 선택 |

## 게임 구성

```
생물 보드: 플레이어당 1개 (종 관리)
특성 카드: 60장 (생존에 유리한 특성)
환경 카드: 20장 (환경 변화)
먹이 토큰: 50개
포식자 말: 5개
```

## 기본 규칙

```
종 설계:
  매 라운드 특성 카드 추가 가능
  (크기·속도·위장·독·협력 등)

환경 변화:
  환경 카드 공개
  예) "빙하기": 두꺼운 털 필요
      "가뭄": 물 저장 능력 필요
      "포식자 급증": 방어 능력 중요

생존 판정:
  현재 환경에서 특성 점수 계산
  점수 낮은 종은 개체 수 감소

번성:
  높은 점수 → 먹이 많이 획득
  → 개체 수 증가

승리:
  10라운드 후 최다 개체 수
```

## 특성 카드 예시

```
공격형:
  날카로운 발톱 (+2 사냥)
  독 분비 (+3 방어, +1 사냥)
  큰 몸집 (+2 방어, -1 속도)

방어형:
  위장색 (+3 생존, -1 사냥)
  단단한 껍질 (+2 방어)
  집단 생활 (+1 방어, +1 번식)

적응형:
  온도 조절 (+2 기후 변화 대응)
  잡식성 (+1 모든 환경)
  수면 조절 (+2 자원 부족 환경)
```

## 환경 카드 예시

```
빙하기: 온도 조절·털 특성 +3
화산 폭발: 모든 종 개체 수 -2, 적응 재시작
풍요의 시대: 먹이 2배, 모든 종 번성
새로운 포식자: 방어 특성 없으면 -2/턴
섬 고립: 특화 종 유리, 일반 종 불리
```

## 교육적 효과

```
자연 선택:
  환경에 따른 생존 차이 체험
  → 진화의 핵심 원리

적응 사고:
  변화에 맞는 전략 수정
  → 유연한 사고력

시스템 사고:
  생태계 상호작용 이해
  → 복잡계 사고
```
