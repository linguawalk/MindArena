# 가설 배틀 (Hypothesis Battle)

> 과학은 가설을 세우고
> 실험으로 검증합니다.
> 좋은 가설을 세우는 능력이
> 과학적 사고의 핵심입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 가설 설정, 변수 통제, 과학적 추론 |

## 게임 구성

```
현상 카드: 50장 (설명할 자연 현상)
변수 카드: 40장 (원인이 될 수 있는 요소)
실험 카드: 30장 (검증 방법)
반증 카드: 20장 (가설을 깨는 증거)
```

## 기본 규칙

```
현상 공개:
  현상 카드 1장 공개
  예) "키가 큰 나무는 낙뢰를 더 자주 맞는다"

가설 제시:
  변수 카드를 활용해 가설 구성
  "왜냐하면 ___이기 때문이다"

가설 형식:
  "___하면 ___할 것이다
   왜냐하면 ___이기 때문이다"

검증 방법:
  실험 카드로 검증 방법 제시

반증 시도:
  다른 플레이어의 반증 카드 사용

평가:
  가장 과학적인 가설 투표 선정
```

## 현상 카드 예시

```
자연:
  "북극곰은 털이 희다"
  "철새는 남쪽으로 이동한다"
  "화산 근처에 식물이 잘 자란다"

인체:
  "운동 후 기분이 좋아진다"
  "잠을 못 자면 판단력이 떨어진다"

사회:
  "경제가 성장하면 범죄가 줄어든다"
  "SNS 사용 시간과 우울증의 관계"
```

## 좋은 가설의 조건

```
검증 가능성:
  실험으로 확인할 수 있어야 함

구체성:
  측정 가능한 변수 포함

반증 가능성:
  틀렸음을 증명할 수 있어야 함
  (포퍼의 과학철학)

단순성:
  오컴의 면도날
  → 불필요한 가정 배제
```

## 교육적 효과

```
과학적 방법:
  관찰→가설→실험→결론
  → 과학의 본질 이해

비판적 사고:
  가설의 약점 찾기
  → 논리적 사고력

지식의 한계:
  모든 가설은 잠정적
  → 과학적 겸손함
```

## 게임 후 토론

```
"과학적 사실과 의견의 차이는?"

 과학적 가설인가요?
 언제 반증될 수 있나요?"

"가짜 뉴스를 과학적으로
 판별하는 방법은?"
```
