# 과학자 탐정 (Scientist Detective)

> 단서를 모아 현상을 설명하세요.
> 탐정처럼 증거를 수집하고
> 과학자처럼 결론을 내립니다.
> 귀납 추론의 즐거움.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 귀납 추론, 증거 기반 사고, 과학적 설명 |

## 게임 구성

```
미스터리 현상 카드: 30장
증거 카드: 각 현상당 6장
  (관련 3장 + 무관련 3장)
설명 보드: 개인용
과학 원리 카드: 40장 (설명에 사용)
```

## 기본 규칙

```
현상 공개:
  미스터리 현상 1가지 제시
  예) "왜 하늘은 파란색인가?"

증거 수집:
  증거 카드 3장 공개
  관련 증거인지 판단 후 선택

설명 구성:
  과학 원리 카드를 활용해
  현상 설명 작성 (3분)

발표 및 평가:
  설명 발표
  증거와 원리의 연결이
  얼마나 논리적인지 평가

득점:
  정확한 증거 선택: +1점/개
  논리적 설명: +1~3점
  실제 과학적 설명과 일치: +2점
```

## 미스터리 현상 예시

```
물리:
  "왜 무거운 것과 가벼운 것이
   같은 속도로 떨어지는가?"
  "왜 무지개는 항상 반원형인가?"

화학:
  "왜 철은 녹이 슬지만 금은 슬지 않는가?"
  "왜 요리할 때 고기 색이 변하는가?"

생물:
  "왜 인간은 다른 동물보다 오래 사는가?"
  "왜 쌍둥이도 성격이 다를 수 있는가?"

지구:
  "왜 지진은 특정 지역에 집중되는가?"
  "왜 달은 항상 같은 면만 보이는가?"
```

## 과학 원리 카드 예시

```
물리: 빛의 산란 / 중력 / 관성 / 파동
화학: 산화 반응 / 단백질 변성 / 촉매
생물: 자연 선택 / 유전자 발현 / 면역
지구: 판구조론 / 조석력 / 대기 순환
```

## 교육적 효과

```
귀납 추론:
  개별 증거 → 일반 원리
  → 과학적 사고의 핵심

증거 평가:
  관련 증거와 무관 증거 구별
  → 비판적 정보 처리

과학적 설명:
  현상을 원리로 설명
  → 과학 이해력 심화
```

## 게임 후 토론

```
"과학적 설명과 미신의 차이는
 무엇인가요?"

"증거가 없으면 믿으면 안 되나요?
 증거가 있으면 반드시 믿어야 하나요?"

"당신 주변에서 과학적으로
 설명하고 싶은 현상이 있나요?"
```
