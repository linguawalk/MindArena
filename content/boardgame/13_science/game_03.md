# 원소 전쟁 (Element War)

> 주기율표는 자연의 설계도입니다.
> 원소를 조합해 화합물을 만들고
> 상대의 반응을 예측하세요.
> 화학이 전략이 됩니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 원소 지식, 화학 결합, 주기율표 이해 |

## 게임 구성

```
원소 카드: 30종 (자주 쓰이는 원소)
  수소(H) 산소(O) 탄소(C) 질소(N)
  나트륨(Na) 염소(Cl) 철(Fe) 등
화합물 목표 카드: 40장
반응 카드: 20장 (특수 화학 반응)
주기율표 참고 카드: 플레이어당 1장
```

## 기본 규칙

```
목표:
  화합물 목표 카드의 분자식 완성
  예) H₂O, NaCl, CO₂

손패:
  원소 카드 5장

턴 진행:
  ①원소 카드 1장 뽑기
  ②화합물 완성 시도
    (목표 카드의 원소 조합)
  ③완성 시 점수 획득

특수 반응:
  반응 카드로 상대 화합물 분해
  또는 촉매 반응으로 2배 완성
```

## 화합물 목표 카드 예시

```
쉬운 화합물 (1점):
  H₂O (물): 수소 2 + 산소 1
  NaCl (소금): 나트륨 1 + 염소 1
  CO₂ (이산화탄소): 탄소 1 + 산소 2

중간 화합물 (2점):
  H₂SO₄ (황산): 수소·황·산소
  NH₃ (암모니아): 질소·수소
  C₆H₁₂O₆ (포도당): 탄소·수소·산소

어려운 화합물 (3점):
  DNA 구성 요소
  철강 합금
  의약품 분자
```

## 주기율표 학습 연계

```
원소 카드에 표시:
  원자번호 / 원소 기호 / 원자량
  족·주기 정보
  주요 특성 (금속/비금속/반도체)

패턴 발견:
  같은 족 원소는 성질이 비슷
  → 주기율표의 규칙성 체험
```

## 교육적 효과

```
원소 암기:
  게임으로 자연스럽게 원소 기호 학습
  → 화학 기초 지식

화학 결합:
  원소 조합의 규칙 이해
  → 분자 구조 사고

과학적 규칙:
  자연에 숨겨진 패턴 발견
  → 과학적 호기심 자극
```
