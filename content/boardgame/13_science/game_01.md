# 발명 공방 (Invention Workshop)

> 문제가 있는 곳에 발명이 있습니다.
> 주어진 재료로 세상의 문제를 해결하세요.
> 가장 창의적이고 실용적인
> 발명품이 이깁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 창의적 문제 해결, 발명 사고, 실용적 설계 |

## 게임 구성

```
문제 카드: 40장 (해결할 문제)
재료 카드: 60장 (사용 가능한 재료)
평가 기준 카드: 15장
발명 설계 용지: 인원수 × 10장
```

## 기본 규칙

```
문제 공개:
  문제 카드 1장 공개
  예) "노인이 혼자 약을 챙겨 먹기 어렵다"

재료 지급:
  각자 재료 카드 5장 받기
  예) 센서·알람·앱·물리적 용기·AI

설계 (5분):
  재료 카드를 조합해 발명품 설계
  설계 용지에 그림+설명

발표 (각 2분):
  발명품 소개
  "이 발명품은 ___입니다.
   작동 원리는 ___이고
   이 문제를 ___방식으로 해결합니다."

평가:
  평가 기준 카드 기준으로 투표
  자신 제외
```

## 문제 카드 예시

```
사회 문제:
  "독거노인의 고독사를 예방하려면?"
  "음식물 쓰레기를 줄이려면?"
  "교통 약자의 이동을 돕는 방법은?"

환경 문제:
  "플라스틱 없이 음식을 포장하려면?"
  "도시의 열섬 현상을 줄이려면?"

미래 문제:
  "화성에서 식물을 키우려면?"
  "AI가 일자리를 빼앗을 때 대비하려면?"
```

## 평가 기준 카드

```
실용성: 실제로 작동할 수 있는가?
창의성: 아무도 생각 못한 아이디어인가?
경제성: 저렴하게 만들 수 있는가?
지속성: 오래 사용할 수 있는가?
접근성: 누구나 쉽게 사용할 수 있는가?
```

## 교육적 효과

```
디자인 씽킹:
  공감 → 문제 정의 → 아이디어 → 프로토타입
  → 혁신 방법론 체험

제약 속 창의성:
  제한된 재료로 최선의 해결책
  → 창의성의 본질

실용적 사고:
  아름다운 아이디어 vs 작동하는 아이디어
  → 공학적 사고
```

## 실제 발명 연결

```
각 라운드 후 실제 유사 발명 소개:
  "비슷한 문제를 실제로 이렇게 해결했습니다"
  → 학습 동기 강화
  → 발명의 현실성 확인
```
