# 영역 정복 (Territory)

> 보드 위에 돌을 놓아 영역을 만들고
> 가장 넓은 영역을 차지한 플레이어가 승리하는
> 순수 전략 추상 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 20-30분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 공간 분할, 영역 계산, 전략적 사고 |

---

## 게임 목표

게임이 끝났을 때 자신의 돌로 둘러싸인
**영역(빈 셀의 수)이 가장 넓은 플레이어**가 승리합니다.
돌 자체도 1점으로 계산합니다.

---

- 12×12 격자 보드

## 게임 규칙

### 기본 규칙

**1. 선공 결정**
연령이 가장 적은 플레이어가 선공입니다.
이후 시계 방향으로 진행합니다.

**2. 턴 진행**
자신의 턴에 다음 중 하나를 선택합니다.

①**돌 놓기**: 빈 셀에 자신의 돌을 1개 놓습니다.
②**패스**: 돌을 놓지 않고 턴을 넘깁니다.
  (단, 연속 패스는 게임 종료 조건이 됩니다)

**3. 영역 규칙**
빈 셀이 상하좌우로 인접한 모든 돌이
같은 색일 때 그 빈 셀은 해당 플레이어의 영역입니다.

```
영역 예시 (흑의 영역):
  ●  ●  ●
  ●  .  ●   ← 가운데 빈 셀 = 흑의 영역 (1점)
  ●  ●  ●
```

**4. 포위 및 제거**
상대방의 돌 그룹이 완전히 포위되면
(상하좌우 모든 인접 셀이 자신의 돌 또는 보드 경계)
그 돌들을 제거하고 포로로 가져갑니다.
포로 1개당 나중에 1점 감점입니다.

```
포위 예시:
  ●  ●  ●
  ●  ○  ●   ← 흑이 백 1개를 완전 포위
  ●  ●  ●   → 백 돌 제거, 흑의 포로 1개
```

**5. 게임 종료**
다음 조건 중 하나가 발생하면 게임이 끝납니다.
- 모든 플레이어가 연속으로 패스
- 모든 플레이어의 돌이 소진
- 합의로 종료

### 점수 계산

```
최종 점수 = 자신의 영역 셀 수
           + 보드 위 자신의 돌 수
           - 상대방에게 잡힌 포로 수
```

## 전략 가이드

### 초보자 전략

**1. 코너와 변 활용**
보드의 코너와 변은 적은 돌로 큰 영역을 만들 수 있습니다.

```
코너 영역 예시:
  ●  ●  .
  ●  .  .   ← 적은 돌로 넓은 영역
  .  .  .
  (보드 경계가 벽 역할)
```

**2. 효율적 영역 확보**
돌 1개로 영역을 최대화하세요.
정사각형 형태가 가장 효율적입니다.

**3. 상대 영역 방해**
상대방이 큰 영역을 만들려 할 때
그 안에 돌을 하나 놓아 영역을 분리하세요.

### 중급 전략

**1. 이중 위협**
동시에 두 곳의 영역을 위협하세요.
상대방은 하나만 막을 수 있습니다.

**2. 효율적 포위**
적은 돌로 많은 상대 돌을 포위하는 구조를 찾으세요.

```
효율적 포위:
  ●  ●  ●  ●
  ●  ○  ○  ●   ← 흑 8개로 백 2개 포위
  ●  ●  ●  ●
```

**3. 사활(死活) 판단**
자신의 돌 그룹이 살 수 있는지(두 눈 확보)
항상 확인하세요.

```
두 눈 구조 (살아있는 그룹):
  ●  ●  ●  ●  ●
  ●  .  ●  .  ●   ← 빈 셀 2개 = 두 눈 = 살아있음
  ●  ●  ●  ●  ●
```

**4. 연결 유지**
자신의 돌 그룹이 연결되어 있어야
포위 위험이 줄어듭니다.

### 고급 전략

**1. 세력 싸움**
상대방보다 더 넓은 세력권을 확보하세요.
실제로 돌을 놓지 않아도
세력만으로 상대방의 진입을 막을 수 있습니다.

**2. 끝내기 계산**
게임 후반 남은 빈 셀을 정확히 계산하고
최적의 순서로 영역을 확보하세요.

**3. 3인·4인 외교 전술**
다인 게임에서는 선두를 공동으로 견제하는
외교적 판단이 중요합니다.
단, 어떤 약속도 규칙상 강제되지 않습니다.

## 점수 계산 예시

```
게임 종료 후:

흑의 영역 셀: 18개
흑의 보드 위 돌: 22개
흑의 포로 (잡힌 돌): 3개
흑의 최종 점수: 18 + 22 - 3 = 37점

백의 영역 셀: 20개
백의 보드 위 돌: 20개
백의 포로 (잡힌 돌): 1개
백의 최종 점수: 20 + 20 - 1 = 39점

→ 백 승리!
```

## 변형 규칙

### 빠른 영역 정복
- 보드 크기: 8×8
- 돌 개수: 20개씩
- 포위·제거 규칙 없음
- 순수 영역 확보만으로 승부
- 소요 시간: 10-15분

### 협력 영역 정복
- 2 vs 2 팀전
- 같은 팀의 돌은 서로 연결된 것으로 봄
- 팀 영역의 합산 점수로 승부
- 팀 전략 소통은 자유롭게 허용

### 제한 영역 정복
- 각 플레이어는 턴당 돌 2개 놓기 가능
- 게임이 더 빠르게 진행
- 더 넓은 전략적 시야 필요

## 교육적 활용

### 수학 연계
- **면적 계산**: 영역의 셀 수 계산
- **집합론**: 영역의 포함·배제 관계
- **최적화**: 최소 돌로 최대 영역 확보

### 사고력 연계
- **공간 분할**: 2D 공간의 구조적 이해
- **자원 배분**: 제한된 돌의 효율적 사용
- **다자 관계**: 3-4인 게임에서 세력 균형

### 수업 활용 방법
1. 8×8 빠른 버전으로 규칙 익히기 (10분)
2. 2인 정식 게임 1판 (25분)
3. 점수 계산 연습 (5분)
4. "더 좋은 수가 있었나?" 복기 (10분)

## 자주 묻는 질문

**Q: 영역은 어떻게 확정하나요?**
A: 빈 셀의 상하좌우 인접 셀이
모두 같은 색의 돌이거나 보드 경계일 때 확정됩니다.
대각선은 영역 계산에 포함되지 않습니다.

**Q: 포위된 돌을 즉시 제거해야 하나요?**
A: 네. 포위가 완성되는 순간 즉시 제거합니다.

**Q: 3인 게임에서 두 플레이어가 협력해도 되나요?**
A: 암묵적 협력은 가능하지만 어떤 약속도 강제되지 않습니다.
언제든지 전략을 바꿀 수 있습니다.

**Q: 자신의 돌을 스스로 포위하면 어떻게 되나요?**
A: 자신의 돌이 완전히 포위되어도
두 눈(빈 셀 2개 이상)이 있으면 살아있습니다.
두 눈이 없는 상태에서 자신이 포위를 완성하면
그 돌들은 제거됩니다.
