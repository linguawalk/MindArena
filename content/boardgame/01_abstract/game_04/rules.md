# 균형의 탑 (Balance Tower)

> 보드 위에 숫자 타일을 배치하여
> 행과 열의 합을 균형있게 맞추는
> 수학적 추상 전략 게임입니다.
> 상대방보다 더 높은 균형 점수를 얻으세요.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 20-25분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 수 감각, 덧셈·뺄셈, 전략적 배치, 균형 개념 |

---

## 게임 목표

게임이 끝났을 때 자신이 배치한 타일로 만들어진
**행과 열의 합이 목표값에 가장 가까운 플레이어**가
높은 점수를 얻습니다.
**최종 점수가 가장 높은 플레이어**가 승리합니다.

---

## 준비물

- 6×6 게임 보드
- 숫자 타일 (1-9) 각 4개씩 = 총 36개
- 목표값 카드 12장 (각 행·열의 목표 합)
- 플레이어 마커 (색상별)
- 점수 기록판

### 보드 구조

```
        열1  열2  열3  열4  열5  열6   목표합
행1  [  ]  [  ]  [  ]  [  ]  [  ]  [  ]  = T1
행2  [  ]  [  ]  [  ]  [  ]  [  ]  [  ]  = T2
행3  [  ]  [  ]  [  ]  [  ]  [  ]  [  ]  = T3
행4  [  ]  [  ]  [  ]  [  ]  [  ]  [  ]  = T4
행5  [  ]  [  ]  [  ]  [  ]  [  ]  [  ]  = T5
행6  [  ]  [  ]  [  ]  [  ]  [  ]  [  ]  = T6
     ↓    ↓    ↓    ↓    ↓    ↓
     T7   T8   T9  T10  T11  T12
```

T1~T12: 각 행·열의 목표 합 (게임 시작 전 카드로 결정)

---

## 게임 준비

### 1. 목표값 설정

목표값 카드를 무작위로 뽑아
각 행과 열의 목표 합을 설정합니다.

**목표값 카드 예시:**
```
행 목표값 카드:
  T1=18  T2=22  T3=15  T4=27  T5=20  T6=18

열 목표값 카드:
  T7=20  T8=17  T9=19  T10=25  T11=16  T12=23
```

**균형 확인:**
행 목표값의 합 = 열 목표값의 합이어야 합니다.
(같은 타일들이 배치되므로 수학적으로 같아야 함)
```
18+22+15+27+20+18 = 120
20+17+19+25+16+23 = 120  ✓
```

### 2. 타일 배분

숫자 타일을 잘 섞어서 각 플레이어에게 6개씩 나눠줍니다.
나머지 타일은 뒤집어서 더미로 쌓습니다.

### 3. 선공 결정

숫자가 가장 높은 타일을 가진 플레이어가 선공입니다.

---

## 게임 규칙

### 턴 진행

자신의 턴에 다음 중 하나를 선택합니다.

**①타일 배치:**
손에 있는 타일 1개를 보드의 빈 셀에 놓습니다.
배치한 타일은 자신의 마커로 표시합니다.

**②타일 교환:**
손에 있는 타일 1개를 더미 맨 위 타일과 교환합니다.
교환한 타일은 더미 맨 아래로 들어갑니다.

**③타일 드로우:**
더미에서 타일 1개를 가져와 손패에 추가합니다.
(손패 최대 8개)

### 배치 제한

- 각 셀에는 타일 1개만 배치 가능
- 한 번 배치된 타일은 이동 불가
- 자신의 마커가 있는 타일만 이동 가능
  (단, 이동은 같은 행 또는 같은 열 안에서만)

### 방해 행동

상대방의 타일을 직접 제거할 수 없지만
다음 방법으로 방해할 수 있습니다.

**셀 봉쇄:**
상대방이 배치하려는 셀 옆에 먼저 타일을 놓아
목표값 달성을 어렵게 만들 수 있습니다.

**목표값 왜곡:**
특정 행·열에 큰 숫자를 먼저 배치해서
상대방이 목표값에 맞추기 어렵게 만들 수 있습니다.

---

## 점수 계산

### 게임 종료 조건

다음 중 하나가 발생하면 게임이 끝납니다.
- 보드가 완전히 채워짐
- 모든 플레이어가 연속으로 패스
- 타일 더미 소진 후 모든 플레이어 패스

### 점수 계산 방법

**행 점수:**
각 행에서 타일 합산값이 목표값에 가까울수록 높은 점수.

```
행 점수표:
  목표값과 차이 0: 10점
  목표값과 차이 1: 8점
  목표값과 차이 2: 6점
  목표값과 차이 3: 4점
  목표값과 차이 4: 2점
  목표값과 차이 5 이상: 0점
```

**열 점수:**
행 점수와 동일한 방식으로 계산.

**기여 점수:**
각 행·열에서 가장 많은 타일을 배치한 플레이어가
해당 행·열 점수의 50%를 추가로 받습니다.

**보너스 점수:**
- 완벽한 행(차이 0): +5점 추가
- 완벽한 열(차이 0): +5점 추가
- 완벽한 행+열 동시: +15점 추가

---

## 점수 계산 예시

```
게임 종료 후 보드 상태:

        열1  열2  열3  열4  열5  열6   실제합  목표합  차이
행1  [ 3] [ 5] [ 2] [ 4] [ 1] [ 3]  =  18    18     0  → 10점
행2  [ 7] [ 2] [ 4] [ 5] [ 3] [ 1]  =  22    22     0  → 10점
행3  [ 1] [ 3] [ 5] [ 2] [ 3] [ 2]  =  16    15     1  → 8점
행4  [ 9] [ 4] [ 3] [ 6] [ 2] [ 4]  =  28    27     1  → 8점
행5  [ 2] [ 6] [ 4] [ 3] [ 2] [ 3]  =  20    20     0  → 10점
행6  [ 5] [ 1] [ 2] [ 4] [ 3] [ 3]  =  18    18     0  → 10점
실제합  27   21   20   24   14   16
목표합  20   17   19   25   16   23
차이    7    4    1    1    2    7
점수    0    2    8    8    6    0

행 점수 합계: 10+10+8+8+10+10 = 56점
열 점수 합계: 0+2+8+8+6+0 = 24점
총 기본 점수: 80점 (전체)

플레이어별 기여 점수 계산:
(각 행·열의 다수 타일 보유자가 해당 점수의 50% 추가)
```

## 전략 가이드

### 초보자 전략

**1. 목표값 분석**
게임 시작 전 행·열 목표값의 패턴을 파악하세요.
- 목표값이 높은 행: 큰 숫자 타일 배치
- 목표값이 낮은 행: 작은 숫자 타일 배치

**2. 손패 관리**
항상 다양한 숫자의 타일을 손에 유지하세요.
큰 숫자만 있거나 작은 숫자만 있으면 배치가 어렵습니다.

**3. 빈 셀 파악**
자신이 배치해야 할 셀과 상대방의 타일을
항상 파악하고 있어야 합니다.

### 중급 전략

**1. 행과 열 동시 고려**
타일 하나가 행과 열 모두에 영향을 줍니다.
배치할 때 행과 열 양쪽의 목표값을 동시에 고려하세요.

```
예시: 행 목표값 = 20, 열 목표값 = 15
현재 행 합계 = 12 (8 부족)
현재 열 합계 = 11 (4 부족)
→ 어떤 숫자를 놓아야 두 목표에 가장 가까울까?
```

**2. 기여 점수 전략**
한 행·열에 집중적으로 타일을 배치해서
기여 점수를 독점하는 전략도 유효합니다.

**3. 상대방 방해**
상대방이 완벽한 행·열을 만들려는 것을 파악하고
핵심 셀에 먼저 타일을 배치하세요.

### 고급 전략

**1. 보너스 점수 狙이**
완벽한 행과 열이 교차하는 셀을 찾아
보너스 점수를 극대화하세요.

**2. 타일 교환 타이밍**
필요한 숫자가 손에 없을 때
교환 타이밍을 잘 잡아야 합니다.
게임 후반에 교환하면 배치 기회를 잃을 수 있습니다.

**3. 확률 계산**
더미에 남은 타일의 숫자 분포를 추적하면
필요한 타일이 나올 확률을 계산할 수 있습니다.

## 변형 규칙

### 마이너스 균형의 탑
- 타일에 음수 포함: -1, -2, -3 타일 추가
- 목표값에 음수 포함
- 더 복잡한 수 계산 필요
- 권장 연령: 12세 이상

### 곱셈 균형의 탑
- 합산 대신 곱셈으로 목표값 달성
- 목표값: 각 행의 타일 곱
- 수의 곱셈 감각 훈련
- 권장 연령: 11세 이상

### 협력 균형의 탑
- 모든 플레이어가 협력해서 전체 보드 균형 달성
- 경쟁 없이 공동 목표 추구
- 소통과 협력 강조

### 빠른 균형의 탑
- 4×4 보드 사용
- 손패 4개
- 소요 시간: 10-15분
- 초보자 입문용

---

## 교육적 활용

### 수학 연계
- **덧셈·뺄셈**: 행·열 합산 계산
- **수 감각**: 목표값에 맞는 수 조합
- **최적화**: 최적의 타일 배치 전략
- **확률**: 남은 타일 분포 추적

### 사고력 연계
- **전략적 사고**: 장기 계획과 즉각 대응
- **다차원 사고**: 행과 열 동시 고려
- **자원 관리**: 손패 타일 효율적 사용

### 수업 활용 방법
1. 4×4 빠른 버전으로 규칙 익히기 (10분)
2. 2인 정식 게임 1판 (25분)
3. "최적의 배치는?" 토론 (10분)
4. 곱셈 변형 버전 도전 (15분)

## 자주 묻는 질문

**Q: 행과 열 목표값이 맞지 않으면 어떻게 하나요?**
A: 목표값 카드를 다시 뽑아 새로 설정합니다.
행 목표값 합 = 열 목표값 합이 되어야 합니다.

**Q: 손패가 8개를 초과하면?**
A: 드로우 액션을 선택할 수 없습니다.
배치 또는 교환만 가능합니다.

**Q: 더미가 소진되면?**
A: 더미 없이 배치와 패스만 가능합니다.
모든 플레이어가 연속 패스하면 게임 종료.

**Q: 기여 점수에서 동수가 나오면?**
A: 해당 행·열에 타일을 먼저 배치한 플레이어가
기여 점수를 가져갑니다.
