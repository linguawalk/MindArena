# 추상 게임 (Abstract Games)

> 운이 아닌 순수한 사고력으로 승부하는 게임입니다.
> 정보가 완전히 공개되고, 주사위나 카드 없이
> 오직 논리와 전략으로 플레이합니다.

---

## 카테고리 특징

- **완전 정보**: 모든 플레이어가 같은 정보를 가짐
- **운 요소 없음**: 주사위, 카드, 무작위 요소 없음
- **순수 사고력**: 논리, 공간 인식, 패턴 인식이 핵심
- **교육적 가치**: 수학적 사고, 집중력, 계획 능력 향상

## 교육 목표
