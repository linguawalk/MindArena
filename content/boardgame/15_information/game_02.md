# 신호와 소음 (Signal and Noise)

> 진짜 정보와 가짜 정보가 섞여 있습니다.
> 신호를 찾고 소음을 걸러내야 합니다.
> 정보 과잉 시대의 필수 능력을 훈련합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 정보 평가, 출처 판별, 비판적 사고 |

## 게임 구성

```
정보 카드: 80장
  신호 (사실): 40장
  소음 (거짓·오해): 40장
출처 카드: 각 정보당 1장
  (신뢰도 높음/중간/낮음)
판별 보드: 신호/소음 구분대
점수판
```

## 기본 규칙

```
정보 공개:
  정보 카드 5장 공개
  (신호·소음 섞여 있음)

출처 공개 (선택):
  출처 카드 구매 가능
  구매할수록 점수 차감

판별:
  각 정보 신호/소음 분류
  이유 1문장 필수

채점:
  정확한 판별: +2점
  구매한 출처 수 차감
  오판: -1점
```

## 정보 카드 예시

```
신호 (사실):
  "백신 접종은 집단 면역 형성에 기여한다"
  "지구 온도는 산업화 이후 상승했다"
  "수면 부족은 인지 능력을 저하시킨다"

소음 (거짓·오해):
  "백신이 자폐증을 유발한다" (반증된 연구)
  "인간은 뇌의 10%만 사용한다" (신화)
  "설탕이 아이들을 과잉행동하게 만든다" (미확인)
```

## 출처 신뢰도

```
신뢰도 높음:
  피어리뷰 과학 논문
  공신력 있는 기관 발표
  → 정보 정확도 높음

신뢰도 중간:
  일반 뉴스 매체
  전문가 개인 의견
  → 확인 필요

신뢰도 낮음:
  소셜미디어 게시물
  익명 블로그
  → 의심 필요
```

## 교육적 효과

```
미디어 리터러시:
  정보 출처 평가 능력
  → 가짜 뉴스 대응

비판적 수용:
  모든 정보를 의심하되
  모두 거부하지도 않기
  → 균형 잡힌 사고

확증 편향 극복:
  믿고 싶은 것만 믿는 경향
  → 인지 편향 인식
```
