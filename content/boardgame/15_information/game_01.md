# 내부자 게임 (Insider Game)

> 그룹 안에 내부자가 있습니다.
> 내부자는 답을 알고 있지만
> 들키지 않아야 합니다.
> 일반인은 내부자를 찾아야 합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 사회적 추론, 신호 읽기, 속임수 탐지 |

## 게임 구성

```
역할 카드: 마스터 1장·내부자 1장·일반인 나머지
정답 카드: 50장 (단어·개념)
타이머: 5분
투표 용지
```

## 기본 규칙

```
역할 배정 (비밀):
  마스터: 정답 알고 있음, 게임 진행
  내부자: 정답 알고 있음, 숨겨야 함
  일반인: 정답 모름, 질문으로 추측

게임 진행:
  일반인이 예/아니오 질문
  마스터가 대답
  내부자도 함께 질문 (들키지 않게)

정답 맞추기:
  5분 내 정답 맞추면 일반인 성공

내부자 찾기:
  정답 맞춘 후 내부자 투표
  내부자 지목 성공: 마스터+일반인 승리
  실패: 내부자 승리
```

## 내부자 전략

```
너무 좋은 질문은 금물:
  직접적인 힌트 → 의심받음

너무 나쁜 질문도 금물:
  엉뚱한 질문 → 역시 의심받음

자연스럽게:
  일반인처럼 헤매면서
  슬쩍 방향 잡아주기
```

## 일반인 전략

```
질문 패턴 관찰:
  누가 유독 도움이 되는 질문을?
  누가 게임 흐름을 아는 것 같은?

반응 관찰:
  정답 근처에서 미세한 반응
  눈빛·표정·자세 변화

메타 관찰:
  게임 진행 방향 유도하는 사람?
```

## 교육적 효과

```
사회적 추론:
  행동에서 지식 상태 추론
  → 마음 이론(Theory of Mind)

속임수 탐지:
  언어·비언어 신호 읽기
  → 사회적 지능

전략적 소통:
  정보 공개 수준 조절
  → 협상·외교 능력 기초
```

## 정답 카드 주제별

```
쉬운 단어: 바나나·자동차·학교
중간 단어: 민주주의·중력·공감
어려운 단어: 아이러니·패러독스·엔트로피
```
