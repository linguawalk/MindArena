# 이중 첩자 (Double Agent)

> 당신은 두 편을 동시에 섬기는 첩자입니다.
> 양쪽 모두를 만족시키면서
> 정보를 흘리고 임무를 완수해야 합니다.
> 이중 게임의 심리전.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐⭐ |
| 목표 역량 | 전략적 소통, 다중 목표 관리, 신뢰 구축 |

## 게임 구성

```
팀 카드: A팀·B팀 각 1장
첩자 카드: 1-2장 (이중 첩자)
임무 카드: 각 팀당 5장
정보 토큰: 20개
신뢰 점수판
```

## 기본 규칙

```
역할 배정 (비밀):
  A팀: 절반의 플레이어
  B팀: 나머지 플레이어
  이중 첩자: 1-2명 (양팀 카드 보유)

임무 수행:
  각 팀 임무 카드 달성 시도
  정보 공유·교환으로 임무 완수

이중 첩자 목표:
  양팀 임무를 모두 일부 달성
  들키지 않으면서

정체 폭로:
  의심되는 플레이어 지목 가능
  정체 확인 시 이중 첩자 탈락

승리:
  이중 첩자: 양팀 임무 50% 이상 달성
  각 팀: 임무 완수 + 이중 첩자 발견
```

## 임무 카드 예시

```
A팀 임무:
  "B팀의 핵심 정보 토큰 3개 확보"
  "B팀 내 이중 첩자 1명 발견"

B팀 임무:
  "A팀의 계획 카드 2장 파악"
  "A팀 내 이중 첩자 발견"

이중 첩자 임무:
  "양팀 임무 각 1개씩 달성"
  "끝까지 정체 숨기기"
```

## 심리전 전략

```
이중 첩자:
  한 팀에 너무 충성하지 않기
  의심받으면 과잉 충성으로 반전
  정보는 조금씩 흘리기

일반 팀원:
  정보 공유 패턴 관찰
  누가 양쪽에 이익이 되는 행동을?
  신뢰와 의심의 균형
```

## 교육적 효과

```
다중 목표:
  상충되는 목표 동시 관리
  → 복잡한 의사결정

신뢰 경제:
  신뢰의 가치와 비용
  → 사회적 자본 이해

전략적 소통:
  말하는 것과 숨기는 것
  → 협상 능력
```
