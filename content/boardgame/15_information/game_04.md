# 정보 시장 (Information Market)

> 정보는 곧 돈입니다.
> 누가 무엇을 알고 있는지가
> 시장에서 우위를 결정합니다.
> 정보를 사고팔고 활용하는 게임.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 정보 가치 평가, 전략적 거래, 정보 경제학 |

## 게임 구성

```
정보 카드: 60장 (다양한 가치의 정보)
시장 보드: 거래 진행용
자본 칩: 플레이어당 50점
정보 봉투: 각 플레이어 비밀 정보 보유
결과 카드: 10장 (최종 가치 결정)
```

## 기본 규칙

```
초기 설정:
  각 플레이어 비밀 정보 3개 보유
  (다른 플레이어는 내용 모름)

정보 거래:
  정보를 팔거나 교환 가능
  가격은 협상으로 결정
  단, 거짓 정보 판매도 허용!

결과 공개:
  5라운드 후 결과 카드 공개
  특정 정보 보유자에게 보너스

최종 자산:
  자본 + 정보 가치 합산
```

## 정보 카드 예시

```
고가치 정보:
  "다음 라운드 결과 카드 미리 봄"
  "특정 플레이어의 비밀 정보 파악"

중간 정보:
  "시장 가격 변동 예측"
  "다른 플레이어 자본 규모"

저가치 정보:
  "공개된 사실의 확인"
  "이미 알려진 정보 재확인"
```

## 정보 거래 전략

```
정보 판매:
  진짜 정보: 신뢰 쌓기
  가짜 정보: 단기 수익, 장기 신뢰 손상

정보 구매:
  가격이 가치보다 낮을 때만
  출처 신뢰도 확인

정보 독점:
  핵심 정보 혼자 보유
  적절한 시점에 활용
```

## 교육적 효과

```
정보 경제학:
  정보의 가격과 가치
  → 현대 경제의 핵심 이해

비대칭 정보:
  레몬 시장·역선택 개념
  → 아커로프 노벨경제학 이해

신뢰 자본:
  거짓말의 장단기 비용
  → 평판 경제학
```
