# 진실 게임 (Truth Game)

> 세 가지 진술 중 하나는 거짓입니다.
> 어느 것이 거짓인지 찾아내세요.
> 하지만 거짓말쟁이도 진실처럼 말합니다.
> 인간의 진실 탐지 능력은 얼마나 정확한가?

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 거짓 탐지, 자기 표현, 사회적 관찰력 |

## 기본 규칙

```
진술:
  각 플레이어 차례로
  자신에 관한 진술 3개 발표
  그 중 2개는 진실, 1개는 거짓

추측:
  다른 플레이어들이
  어느 것이 거짓인지 추측

공개:
  정답 공개

득점:
  거짓 맞힘: +2점
  모두 틀리게 한 발표자: +3점
  1명만 맞힘: 맞힌 사람 +2, 발표자 +1
```

## 좋은 거짓말 만들기

```
원칙:
  진실처럼 들리는 거짓
  거짓처럼 들리는 진실

기법:
  세부 사항 추가 (거짓에도)
  감정 실어서 말하기
  진실 2개와 비슷한 스타일 유지

함정:
  너무 평범한 거짓 → 쉽게 들킴
  너무 황당한 진실 → 거짓으로 오해
```

## 거짓 탐지 관찰 포인트

```
언어적 신호:
  지나친 세부 사항
  방어적 표현
  주제 회피

비언어적 신호:
  눈 맞춤 변화
  말하는 속도
  미세 표정

맥락적 단서:
  이전에 알려진 정보와 일치?
  너무 완벽한 이야기?
```

## 심화 모드

```
주제 제한:
  "여행에 관한 진술만"
  "두려움에 관한 진술만"
  → 자기 개방 촉진

감정 버전:
  "감정·경험에 관한 진술"
  → 깊은 자기 이해

예측 버전:
  "미래에 대한 진술"
  (실현 여부 나중에 확인)
```

## 교육적 효과

```
사회적 인식:
  타인의 행동에서 정보 읽기
  → 사회적 지능

자기 표현:
  자신에 대해 흥미롭게 말하기
  → 자기 개방 능력

인지 편향:
  거짓 탐지 능력의 한계 인식
  (인간 정확도: 평균 54%)
  → 과신 편향 극복
```

## 게임 후 토론

```
"거짓말을 잘 탐지하는 사람은
 어떤 특징이 있었나요?"

"인간은 거짓말을 잘 탐지할 수 있을까요?
 연구 결과는 어떨까요?"

"거짓말이 사회에서 어떤 역할을 하나요?
 모든 거짓말은 나쁜가요?"
```
