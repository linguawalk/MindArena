# 외교관 게임 (Diplomat)

> 각 플레이어가 서로 다른 국가를 대표해서
> 협상, 동맹, 거래를 통해
> 자국의 이익을 극대화하는
> 외교 시뮬레이션 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-6명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 외교, 국제 관계, 협상, 논증, 전략적 소통 |

---

## 게임 목표

8라운드 후 **국가 점수(경제력 + 안보력 + 국제 신뢰도)**가
가장 높은 플레이어가 승리합니다.

---

## 준비물

- 세계 지도 보드
- 국가 카드 6장
- 국가 이익 카드 (비밀 목표)
- 외교 자원 토큰 (경제력·군사력·문화력 각 20개)
- 협정 카드 30장
- 분쟁 카드 20장
- 동맹 토큰 15개
- 신뢰도 트랙 (국가별)
- 협상 기록지
- 발언 타이머 (2분)

### 국가 카드

```
강대국 A (Great Power A):
  초기 경제력: 8
  초기 군사력: 7
  초기 문화력: 5
  특성: 거부권 1회/게임
  약점: 모든 협상에서 더 많은 양보 요구받음

강대국 B (Great Power B):
  초기 경제력: 7
  초기 군사력: 8
  초기 문화력: 4
  특성: 군사 위협 1회/게임
  약점: 문화 협상에서 신뢰도 -1

중견국 A (Middle Power A):
  초기 경제력: 5
  초기 군사력: 4
  초기 문화력: 7
  특성: 중재자 역할 가능 (+보너스)
  약점: 군사 분쟁에서 불리

중견국 B (Middle Power B):
  초기 경제력: 6
  초기 군사력: 5
  초기 문화력: 6
  특성: 다자 협상에서 +2 보너스
  약점: 양자 협상에서 불리

소국 A (Small State A):
  초기 경제력: 3
  초기 군사력: 2
  초기 문화력: 8
  특성: 연대 형성 시 효과 2배
  약점: 강대국 압박에 취약

소국 B (Small State B):
  초기 경제력: 4
  초기 군사력: 3
  초기 문화력: 7
  특성: 정보 수집 능력 우수
  약점: 단독 행동 시 효과 절반
```

## 비밀 국가 이익 카드

각 플레이어는 비밀 목표를 받습니다.

```
비밀 목표 예시:

영토 확장:
  목표: 영토 분쟁 2건 승리
  보너스: +15점

경제 패권:
  목표: 경제 협정 3건 이상 체결
  보너스: +20점

평화 유지:
  목표: 모든 분쟁을 협상으로 해결
  보너스: +25점

동맹 구축:
  목표: 동맹국 3개 이상 유지
  보너스: +15점

문화 외교:
  목표: 문화 협정 2건 + 신뢰도 최상위
  보너스: +20점

고립 정책:
  목표: 어떤 동맹도 맺지 않고 생존
  보너스: +30점 (달성 어려움)
```

## 게임 준비

### 1. 국가 배분

플레이어 수에 맞게 국가 카드를 배분합니다.
남는 국가는 "중립국"으로 보드에 배치합니다.

### 2. 비밀 목표 배분

각 플레이어에게 비밀 국가 이익 카드 1장을 줍니다.
게임 종료까지 공개하지 않습니다.

### 3. 초기 분쟁 설정

분쟁 카드 3장을 공개해서
초기 국제 갈등 상황을 설정합니다.

---

## 게임 규칙

### 라운드 구조

```
1단계: 국제 이벤트 공개
2단계: 외교 회의 (자유 협상)
3단계: 공식 제안 및 투표
4단계: 분쟁 해결
5단계: 국가 발전
```

### 1단계: 국제 이벤트

이벤트 카드 1장을 공개합니다.

```
이벤트 카드 예시:

경제 위기:
  효과: 모든 국가 경제력 -2
        경제 협력 협정 체결 시 +5 보너스

기후 재난:
  효과: 모든 국가 식량 자원 -1
        환경 협정 체결 촉구

테러 위협:
  효과: 군사력 낮은 국가 안보 위협
        군사 동맹 체결 인센티브 +3

문화 충돌:
  효과: 이념 갈등 분쟁 카드 1장 추가
        문화 외교 가치 +2

기술 혁신:
  효과: 최초 기술 협정 체결 국가 +5
        기술 격차 분쟁 가능성

자원 발견:
  효과: 특정 지역 자원 추가
        영토 분쟁 가능성 증가
```

### 2단계: 외교 회의 (10분)

모든 플레이어가 자유롭게 협상합니다.

```
협상 유형:

양자 협상:
  두 국가 간 1:1 협상
  장소: 테이블 한쪽
  내용: 비밀 유지 가능

다자 협상:
  3개국 이상 공개 협상
  장소: 테이블 중앙
  내용: 모든 플레이어 청취 가능

비밀 외교:
  귓속말 또는 메모로 비밀 협상
  약속은 게임상 강제되지 않음
```

**협상 원칙:**
```
①모든 약속은 자발적
②거짓말 허용 (외교의 현실)
③강압적 발언 금지 ("안 하면 공격한다" 형태 외)
④발언 시간: 2분 제한 (타이머 사용)
```

### 3단계: 공식 제안 및 투표

한 플레이어가 공식 제안을 합니다.

```
공식 제안 유형:

①양자 협정:
  두 국가 간 합의
  내용: 자원 교환, 군사 지원, 문화 교류 등
  투표: 두 당사국만

②다자 협정:
  3개국 이상 합의
  내용: 국제 조약, 공동 선언 등
  투표: 모든 플레이어

③결의안:
  특정 국가에 대한 집단 행동
  내용: 제재, 지원, 비난 등
  투표: 과반수 찬성

④거부권 행사:
  강대국만 가능
  특정 결의안 무효화
  게임당 1회
```

**투표 절차:**
```
①제안자: 2분 내 제안 설명
②반대자: 1분 내 반론
③비밀 투표: 동시에 찬반 카드 공개
④결과: 과반수 찬성 시 채택
```

### 4단계: 분쟁 해결

현재 진행 중인 분쟁을 처리합니다.

```
분쟁 해결 방법:

①외교적 해결:
  협상으로 합의
  양측 타협점 찾기
  결과: 양측 일부 이익 + 신뢰도 +1

②중재:
  제3국이 중재
  중재국: 중재 성공 시 +3점
  결과: 중재국 제안 수용 시 해결

③군사적 해결:
  군사력 비교 + 주사위
  결과: 승자 이익, 패자 손실
        신뢰도 -2 (양측)

④국제 재판:
  다자 투표로 해결
  결과: 투표 결과에 따름
```

### 5단계: 국가 발전

체결된 협정 효과를 적용합니다.

```
협정 효과 적용:
  경제 협정: 경제력 +2 (양측)
  군사 협정: 군사력 +1 (양측)
  문화 협정: 문화력 +2 (양측)
             신뢰도 +1 (전체)

고립 패널티:
  협정 없는 라운드: 경제력 -1
```

## 협정 카드 목록

```
자유 무역 협정 (FTA):
  효과: 양측 경제력 +3
        무역 분쟁 발생 시 자동 해결
  기간: 영구

군사 동맹:
  효과: 군사력 공유 (평균값 사용)
        공격받으면 자동 지원 의무
  기간: 3라운드

문화 교류:
  효과: 양측 문화력 +2, 신뢰도 +1
        문화 충돌 분쟁 면역
  기간: 영구

기술 협력:
  효과: 양측 경제력 +2, 문화력 +1
        기술 분쟁 면역
  기간: 2라운드

환경 협약:
  효과: 참여 국가 모두 신뢰도 +2
        기후 이벤트 피해 면역
  기간: 영구

불가침 조약:
  효과: 군사 분쟁 금지 (기간 내)
        양측 안보 비용 절감
  기간: 2라운드

전략적 동반자:
  효과: 모든 협상에서 +1 보너스
        정보 공유 의무
  기간: 3라운드
```

## 분쟁 카드 목록

```
영토 분쟁:
  당사국: 2개국
  해결 방법: 협상 또는 군사력 비교
  미해결 시: 양측 신뢰도 -1/라운드

무역 전쟁:
  당사국: 2개국
  해결 방법: 경제 협정 또는 WTO 중재
  미해결 시: 양측 경제력 -2/라운드

이념 갈등:
  당사국: 이념 다른 2개국
  해결 방법: 문화 외교 또는 상호 존중 선언
  미해결 시: 다자 갈등으로 확대 가능

자원 경쟁:
  당사국: 자원 접근 원하는 국가들
  해결 방법: 자원 분배 협정
  미해결 시: 군사 충돌 위험

난민 위기:
  당사국: 관련 모든 국가
  해결 방법: 다자 지원 협약
  미해결 시: 인도적 위기 악화
             모든 참여국 신뢰도 -1
```

## 신뢰도 시스템

```
신뢰도 트랙: 0-10 (국가별)
시작: 5

신뢰도 증가:
  협정 준수: +1
  중재 성공: +2
  환경 협약 참여: +1
  약속 이행: +1

신뢰도 감소:
  약속 파기: -3
  군사 공격: -2
  거짓 정보 발각: -2
  협정 파기: -2

신뢰도 효과:
  신뢰도 8+: 협상 시 +2 보너스
  신뢰도 5-7: 기본
  신뢰도 3-4: 협상 시 -1 페널티
  신뢰도 1-2: 대부분의 국가가 협상 거부
  신뢰도 0: 국제 제재 자동 발동
```

## 점수 계산

### 게임 종료 (8라운드 후)

```
기본 점수:
  경제력 × 2점
  군사력 × 1점
  문화력 × 1.5점
  신뢰도 × 3점

협정 보너스:
  체결 협정 수 × 2점

비밀 목표 달성:
  해당 보너스 점수

외교 성과:
  해결한 분쟁 수 × 3점
  중재 성공 × 5점
```

## 전략 가이드

### 초보자 전략

**1. 신뢰도 유지**
약속을 지키는 것이 가장 중요합니다.
신뢰도가 낮으면 아무도 협상하려 하지 않습니다.

**2. 초반 동맹**
초반에 1-2개의 안정적인 동맹을 구축하세요.
혼자서는 강대국에 맞서기 어렵습니다.

**3. 분쟁 회피**
초반에는 분쟁보다 협정에 집중하세요.
분쟁은 양측 모두 손해입니다.

### 중급 전략

**1. 중재 역할 활용**
분쟁 중인 두 국가 사이에서 중재하면
큰 점수와 신뢰도를 얻을 수 있습니다.

**2. 비밀 목표 숨기기**
자신의 비밀 목표를 들키면
다른 플레이어들이 방해합니다.

**3. 다자 협정 활용**
소국은 다자 협정으로 강대국과 균형을 맞출 수 있습니다.

### 고급 전략

**1. 거부권 타이밍**
강대국의 거부권은 자신에게 가장 불리한 결의안에 사용하세요.

**2. 정보 교환**
다른 플레이어의 비밀 목표를 파악하고
그것을 방해하거나 활용하세요.

**3. 동맹 배신**
필요하다면 동맹을 배신할 수 있습니다.
단, 신뢰도 손실을 감수해야 합니다.

## 변형 규칙

### 역사 외교 시뮬레이션
- 실제 역사적 외교 상황 재현
- 1차 세계대전 전야, 냉전 시대 등
- 역사 교육 연계

### 기후 외교
- 모든 협상이 기후 변화 해결에 초점
- 탄소 감축 목표 협상
- 환경 교육 연계

### 단순화 외교
- 분쟁 카드 없음
- 순수 협정 체결 경쟁
- 초보자용 입문 버전

---

## 교육적 활용

### 사회·역사 교육 연계
- **국제 관계**: 외교의 기본 원리
- **협상 이론**: 윈-윈과 제로섬 협상
- **세계 문제**: 실제 국제 이슈 시뮬레이션

### 소통 교육 연계
- **설득 논증**: 자국 이익 설명
- **경청**: 상대방 입장 이해
- **갈등 해결**: 분쟁의 평화적 해결

### 수업 활용 방법
1. 국제 관계 기본 개념 설명 (10분)
2. 역할 배분 + 비밀 목표 확인 (5분)
3. 본게임 진행 (35분)
4. 비밀 목표 공개 + 전략 분석 토론 (10분)
