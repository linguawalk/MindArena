# 세계 협약 (World Pact)

> 인류가 직면한 거대한 문제를 해결하기 위해
> 서로 다른 이해관계를 가진 국가들이
> 협상 테이블에 앉습니다.
> 모두의 이익을 반영한 세계 협약을
> 만들어낼 수 있을까요?

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 4-6명 |
| 소요 시간 | 40-50분 |
| 권장 연령 | 13세 이상 |
| 난이도 | ⭐⭐⭐⭐ (상) |
| 교육 분야 | 국제 협상, 다자 외교, 합의 도출, 글로벌 이슈 |

---

## 게임 목표

**팀 목표**: 세계 협약을 체결해서
글로벌 위기 지수를 50 이하로 낮추면 모든 플레이어가 승리합니다.

**개인 목표**: 협약 체결과 동시에
자국 이익 점수가 가장 높은 플레이어가
최우수 외교관으로 선정됩니다.

---

## 준비물

- 세계 지도 보드
- 국가 카드 6장
- 글로벌 위기 트랙 (0-100)
- 자국 이익 트랙 (국가별)
- 협약 초안 보드
- 협약 조항 카드 (50장)
- 위기 이벤트 카드 (30장)
- 협상 자원 토큰
- 거부권 카드 (강대국 전용)
- 합의 마커
- 발언 타이머

### 국가 카드

```
선진국 A (Developed Nation A):
  경제력: 9
  군사력: 8
  기술력: 9
  외교력: 6
  특성: 거부권 보유
        기술 지원 협약 주도 가능
  약점: 역사적 책임 요구받음
        개발도상국의 반발

선진국 B (Developed Nation B):
  경제력: 8
  군사력: 7
  기술력: 8
  외교력: 7
  특성: 거부권 보유
        금융 지원 협약 주도 가능
  약점: 자국 이익 우선 의심받음

신흥국 A (Emerging Nation A):
  경제력: 6
  군사력: 5
  기술력: 6
  외교력: 7
  특성: 개발도상국 연대 주도 가능
        빠른 성장으로 협상력 증가
  약점: 환경 감축 의무 논란

신흥국 B (Emerging Nation B):
  경제력: 5
  군사력: 6
  기술력: 5
  외교력: 8
  특성: 중재자 역할 우수
        다자 연합 구성 능력
  약점: 경제 지원 의존도 높음

개발도상국 A (Developing Nation A):
  경제력: 3
  군사력: 3
  기술력: 3
  외교력: 6
  특성: 피해국 공감 호소 강력
        국제 여론 동원 가능
  약점: 협약 이행 능력 제한

개발도상국 B (Developing Nation B):
  경제력: 2
  군사력: 2
  기술력: 2
  외교력: 7
  특성: 연대 구성 전문
        도덕적 권위 활용
  약점: 재정 지원 없이 협약 이행 불가
```

## 글로벌 위기 시스템

### 위기 항목

```
글로벌 위기 트랙 (총점 0-100):

기후 변화:      [0-20] 시작: 15
생물다양성 손실: [0-20] 시작: 12
핵 위협:        [0-20] 시작: 10
경제 불평등:    [0-20] 시작: 14
식량 위기:      [0-20] 시작: 9

총 글로벌 위기: 60 (게임 시작)
목표: 모든 항목 합산 50 이하로 낮추기
```

### 위기 악화

매 라운드 위기 이벤트가 위기를 악화시킵니다.

```
위기 이벤트 예시:

극단적 기상이변:
  기후 변화 +3
  식량 위기 +2
  영향국: 개발도상국 A·B 경제력 -1

생물종 대량 멸종:
  생물다양성 손실 +4
  식량 위기 +1
  영향: 농업 생산성 하락

핵 긴장 고조:
  핵 위협 +3
  경제 불평등 +1
  영향: 군비 증가로 복지 예산 감소

글로벌 경기 침체:
  경제 불평등 +4
  식량 위기 +2
  영향: 모든 국가 경제력 -1

팬데믹 발생:
  모든 위기 +1
  식량 위기 +2
  영향: 국제 협력 필요성 증가
```

## 협약 조항 카드

### 기후 협약 조항

```
탄소 중립 선언:
  효과: 기후 변화 -4
  비용: 선진국 GDP 2% 기여
  찬성 조건: 선진국 동의 필요
  반대 이유: 경제적 부담

기후 기술 이전:
  효과: 기후 변화 -3, 기술 격차 감소
  비용: 선진국 기술 무상 제공
  찬성 조건: 개발도상국 열렬 지지
  반대 이유: 선진국 지적재산권 손실

탄소세 도입:
  효과: 기후 변화 -3
  비용: 모든 국가 산업 비용 증가
  찬성 조건: 과반수 동의
  반대 이유: 경쟁력 저하

개발도상국 적응 지원:
  효과: 기후 변화 -2, 경제 불평등 -2
  비용: 선진국 자금 지원
  찬성 조건: 쉽게 합의 가능
  반대 이유: 거의 없음 (낮은 비용)
```

### 핵 비확산 조항

```
핵 감축 조약:
  효과: 핵 위협 -5
  비용: 군사력 감소 (강대국)
  찬성 조건: 강대국 양측 동의 필요
  반대 이유: 안보 우려

핵 사찰 제도:
  효과: 핵 위협 -3
  비용: 주권 일부 양도
  찬성 조건: 과반수 + 강대국 동의
  반대 이유: 주권 침해

핵 없는 세계 선언:
  효과: 핵 위협 -2 (상징적)
  비용: 없음
  찬성 조건: 쉬움
  반대 이유: 실효성 의문
```

### 경제 평등 조항

```
글로벌 최저 법인세:
  효과: 경제 불평등 -4
  비용: 조세 경쟁 포기 (일부 국가)
  찬성 조건: 과반수 + 주요국 동의
  반대 이유: 세수 감소 우려

개발 지원 증가:
  효과: 경제 불평등 -3
  비용: 선진국 ODA 증가
  찬성 조건: 선진국 자발적 동의
  반대 이유: 재정 부담

공정 무역 체제:
  효과: 경제 불평등 -3
  비용: 기존 무역 관행 변화
  찬성 조건: 협상 필요
  반대 이유: 기득권 손실
```

## 게임 규칙

### 라운드 구조

```
1단계: 위기 이벤트 공개 (악화)
2단계: 자유 협상 (5분)
3단계: 협약 조항 제안
4단계: 협상 및 수정
5단계: 최종 투표
6단계: 효과 적용
```

### 1단계: 위기 이벤트

위기 이벤트 카드 1장을 공개하고
위기 트랙을 업데이트합니다.

### 2단계: 자유 협상 (5분)

플레이어들이 자유롭게 협상합니다.

```
협상 내용:
  ①어떤 조항을 제안할 것인가?
  ②비용 분담 방식은?
  ③패키지 딜 (여러 조항 묶음) 구성
  ④거부권 사용 협박 또는 협상
  ⑤연합 형성
```

### 3단계: 협약 조항 제안

한 플레이어가 협약 조항 카드 1장을 제안합니다.

```
제안 형식:
  "저는 [조항 이름]을 제안합니다.
   이 조항은 [위기 항목] -[수치]를 달성하며
   비용은 [분담 방식]으로 합니다."
```

### 4단계: 협상 및 수정

다른 플레이어들이 조항 수정을 요구할 수 있습니다.

```
수정 요구 유형:

비용 재분배:
  "선진국이 비용의 70%를 부담해야 합니다."

조건부 동의:
  "우리가 찬성하는 대신 X 조항도 함께 채택해야 합니다."

패키지 딜:
  "A 조항 + B 조항을 묶어서 한 번에 채택합시다."

보상 요구:
  "이 조항에 동의하는 대신 경제 지원을 요청합니다."

시한 연장:
  "의무 이행 기간을 10년에서 20년으로 늘려주세요."
```

### 5단계: 최종 투표

수정된 조항에 대해 투표합니다.

```
투표 방식:
  찬성·반대·기권 카드 동시 공개

통과 조건:
  일반 조항: 과반수 찬성
  중요 조항: 2/3 이상 찬성
  핵심 조항: 전원 찬성 (거부권 없이)

거부권:
  선진국만 사용 가능
  게임당 2회
  사용 시 자국 이익 -3 (국제 비난)
  거부된 조항 해당 라운드 폐기
```

### 6단계: 효과 적용

채택된 조항의 효과를 즉시 적용합니다.

```
효과 적용:
  위기 트랙 감소
  자국 이익 변화 (비용 부담에 따라)
  신뢰도 변화
```

## 자국 이익 시스템

### 이익 점수 계산

```
자국 이익 = 경제 지수 + 안보 지수 + 외교 지수

경제 지수 변화:
  경제 협약 비용 부담: -비용만큼
  경제 협약 수혜: +수혜만큼
  경제 위기 이벤트: -이벤트 효과

안보 지수 변화:
  핵 감축 참여: -군사력 손실
  안보 협력: +안보 강화
  핵 위협 감소: +전반적 안정

외교 지수 변화:
  협약 주도 성공: +5
  거부권 행사: -3
  중재 성공: +4
  약속 불이행: -5
```

### 자국 이익 vs 세계 이익

```
핵심 딜레마:
  자국 이익 최대화 추구 → 세계 협약 어려움
  세계 협약 우선 → 자국 이익 일부 포기

최우수 외교관:
  세계 협약 체결 + 자국 이익 최대화
  = 진정한 외교 기술
```

## 특수 메커니즘

### 패키지 딜

여러 조항을 묶어서 한 번에 협상합니다.

```
패키지 딜 장점:
  서로 다른 이해관계를 조율 가능
  "A 조항으로 네가 이득, B 조항으로 내가 이득"
  모두가 일부 얻고 일부 양보

패키지 딜 예시:
  [탄소 중립 선언 + 기술 이전 + 경제 지원]
  = 선진국: 탄소 감축 성과
  = 개발도상국: 기술 + 경제 지원 획득
```

### 사이드 딜

공식 협약 외에 국가 간 비밀 협정을 체결합니다.

```
사이드 딜 규칙:
  두 국가 간 비공개 약속
  게임 종료 시 공개
  이행 여부로 신뢰도 변화

사이드 딜 예시:
  "나는 탄소 조항에 찬성할게요.
   대신 당신은 다음 라운드 경제 조항에 찬성해줘요."
```

### 국제 여론

개발도상국은 국제 여론을 활용할 수 있습니다.

```
국제 여론 카드:
  도덕적 호소:
    효과: 선진국의 특정 조항 반대 시
          자국 이익 -2 (국제 비난)
  
  피해국 연대:
    효과: 개발도상국 연합 투표 가중치 +1
  
  역사적 책임:
    효과: 선진국의 비용 분담 거부 시
          외교 지수 -3
```

## 협약 성공 시나리오

### 시나리오 #01: 기후 협약 우선

```
목표: 기후 변화 위기 해결에 집중
필요 조항: 탄소 중립 + 기술 이전 + 적응 지원
예상 결과: 기후 위기 -10, 선진국 비용 높음
협상 핵심: 선진국 설득 + 개발도상국 지지 확보
```

### 시나리오 #02: 경제 평등 우선

```
목표: 남북 격차 해소에 집중
필요 조항: 공정 무역 + 개발 지원 + 최저 법인세
예상 결과: 경제 불평등 -10, 기후 위기 미해결
협상 핵심: 신흥국·개발도상국 연합 형성
```

### 시나리오 #03: 균형 협약

```
목표: 모든 위기 균등하게 해결
필요 조항: 각 위기별 1-2개 조항
예상 결과: 모든 위기 -3~4씩 감소
협상 핵심: 패키지 딜 + 이해관계 조율
```

## 협약 체결 평가

### 게임 종료 조건

```
성공 조건:
  글로벌 위기 합산 ≤ 50
  + 체결 조항 최소 6개 이상

실패 조건:
  8라운드 내 협약 미체결
  글로벌 위기 100 도달
  강대국 거부권 남용으로 협약 불가
```

### 최우수 외교관 선정

```
협약 체결 성공 시:
  자국 이익 점수 최고 플레이어
  = 최우수 외교관 (개인 승리)

추가 평가:
  가장 많은 조항 주도: 창의적 외교관
  가장 많은 중재 성공: 평화 외교관
  비용 최소 부담: 실리 외교관
  가장 높은 신뢰도: 신뢰 외교관
```

## 전략 가이드

### 선진국 전략

**1. 거부권 아끼기**
거부권은 최후의 수단으로 사용하세요.
자국 이익 -3 패널티가 크게 누적됩니다.

**2. 패키지 딜 주도**
여러 조항을 묶어서 제안하면
개발도상국의 지지를 얻으면서
자국 비용을 분산할 수 있습니다.

**3. 기술 지원 활용**
기술 이전은 비용이 있지만
개발도상국의 무조건적 지지를 얻는 열쇠입니다.

### 신흥국 전략

**1. 캐스팅 보트 활용**
선진국과 개발도상국 사이에서
결정적 한 표를 가진 중재자 역할이 유리합니다.

**2. 보상 협상**
찬성 투표의 대가로 경제 지원이나
다른 조항에서의 양보를 요구하세요.

**3. 연합 전략**
개발도상국과 연합을 형성해서
협상력을 높이세요.

### 개발도상국 전략

**1. 국제 여론 활용**
도덕적 호소와 피해국 연대가 가장 강력한 도구입니다.
선진국의 역사적 책임을 상기시키세요.

**2. 연대 유지**
개발도상국들이 단결하면 협약 통과를 막을 수 있습니다.
이 거부 능력을 협상 카드로 사용하세요.

**3. 조건부 지지**
선진국의 기술·경제 지원을 조건으로
협약 찬성을 약속하세요.

### 역사 협약 시뮬레이션

  (교토의정서, 파리협약, 핵비확산조약)
- 역사 결과와 비교
- 역사·사회 교육 연계

### 단일 위기 협약
- 기후 변화만 집중
- 더 깊은 협상 가능
- 소요 시간: 30분

### 완전 협력 모드
- 자국 이익 없음
- 순수 글로벌 위기 해결
- 협력의 가치 체험

## 교육적 활용

### 사회·역사 교육 연계
- **국제 관계**: 강대국·약소국 역학
- **글로벌 이슈**: 기후·핵·불평등 이해
- **협약 과정**: 실제 국제 협약 이해

### 협상 교육 연계
- **다자 협상**: 여러 이해관계 조율
- **패키지 딜**: 복합 협상 전략
- **합의 도출**: 모두가 수용하는 합의

### 수업 활용 방법
1. 글로벌 이슈 현황 설명 (10분)
2. 역할 배분 + 국가 특성 파악 (5분)
3. 본게임 진행 (40분)
4. 실제 국제 협약과 비교 (10분)
