# 배심원 토론 (Jury Debate)

> 찬반 입장으로 나뉘어 논거를 제시하고
> 배심원을 설득하는 토론 게임입니다.
> 논리적 사고와 설득력 있는 표현이
> 승부를 결정합니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 4-8명 |
| 소요 시간 | 35-45분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐⭐ (상) |
| 교육 분야 | 논증, 설득, 비판적 사고, 공감, 토론 능력 |

---

## 게임 목표

토론자: 배심원 과반수를 자신의 입장으로 설득하면 승리.
배심원: 가장 설득력 있는 논거를 제시한 팀에게 투표.

---

## 준비물

- 토론 주제 카드 40장
- 논거 카드 (찬성·반대 각 30장)
- 배심원 점수판
- 발언 타이머 (1분·2분·3분)
- 찬반 마커
- 논거 강도 트랙
- 반박 카드 20장
- 감정 호소 카드 10장
- 진행자 카드

### 역할 구성

```
4인 게임:
  토론자 2명 (찬성 1 + 반대 1)
  배심원 2명

5인 게임:
  토론자 2명 (찬성 1 + 반대 1)
  배심원 3명

6인 게임:
  토론자 4명 (찬성 2 + 반대 2)
  배심원 2명

7-8인 게임:
  토론자 4명 (찬성 2 + 반대 2)
  배심원 3-4명
```

## 토론 주제 카드

### 사회 이슈

```
주제 #01:
  "인공지능이 인간의 일자리를 빼앗는다."
  찬성: AI가 반복 업무 대체
  반대: AI가 새 일자리 창출

주제 #02:
  "소셜 미디어는 민주주의를 위협한다."
  찬성: 가짜 뉴스·극단화 문제
  반대: 정보 공유·참여 확대

주제 #03:
  "사형제도는 폐지되어야 한다."
  찬성: 인권·오판 가능성
  반대: 범죄 억제·피해자 정의

주제 #04:
  "동물 실험은 금지되어야 한다."
  찬성: 동물 권리·대안 기술
  반대: 의학 발전 필요성

주제 #05:
  "기본소득제를 도입해야 한다."
  찬성: 빈곤 해결·자동화 대응
  반대: 재정 부담·근로 의욕 감소
```

### 과학·기술 이슈

```
주제 #06:
  "유전자 편집 아기는 허용되어야 한다."
  찬성: 유전 질환 예방
  반대: 생명 윤리·불평등 심화

주제 #07:
  "핵에너지는 친환경 에너지다."
  찬성: 탄소 배출 없음·안정적 공급
  반대: 방사성 폐기물·사고 위험

주제 #08:
  "우주 개발에 대규모 투자가 필요하다."
  찬성: 기술 발전·인류 생존
  반대: 지구 문제 우선

주제 #09:
  "자율주행차는 인간 운전자보다 안전하다."
  찬성: 사고 감소·데이터 기반
  반대: 해킹·법적 책임 문제
```

### 교육·문화 이슈

```
주제 #10:
  "학교 시험은 폐지되어야 한다."
  찬성: 창의성 억압·스트레스
  반대: 평가·동기 부여 필요

주제 #11:
  "영어는 세계 공용어가 되어야 한다."
  찬성: 소통 효율성
  반대: 문화 다양성 훼손

주제 #12:
  "예술 지원은 국가 의무다."
  찬성: 문화 가치·국민 삶의 질
  반대: 시장에 맡겨야

주제 #13:
  "온라인 교육이 오프라인을 대체할 수 있다."
  찬성: 접근성·비용 효율
  반대: 사회화·실습 한계
```

## 논거 카드 시스템

### 논거 유형

```
사실 논거 (Factual Argument):
  강도: ⭐⭐⭐
  형태: 통계·연구·사례
  예: "연구에 따르면 자동화로 X% 일자리 감소"
  효과: 배심원 설득력 +3

논리 논거 (Logical Argument):
  강도: ⭐⭐⭐⭐
  형태: 인과관계·연역·귀납
  예: "A이면 B, B이면 C, 따라서 A이면 C"
  효과: 배심원 설득력 +4

윤리 논거 (Ethical Argument):
  강도: ⭐⭐⭐
  형태: 가치·권리·의무
  예: "인간의 존엄성을 위해..."
  효과: 배심원 설득력 +3

감정 호소 (Emotional Appeal):
  강도: ⭐⭐
  형태: 공감·사례·이야기
  예: "한 아이가 이 정책으로..."
  효과: 배심원 설득력 +2
       (단, 과다 사용 시 -1 페널티)

전문가 인용 (Expert Citation):
  강도: ⭐⭐⭐
  형태: 권위 있는 출처 인용
  예: "노벨상 수상자 XXX에 따르면..."
  효과: 배심원 설득력 +3

반례 제시 (Counter-example):
  강도: ⭐⭐⭐
  형태: 상대 논거의 예외 제시
  예: "하지만 Y의 경우 그렇지 않았습니다"
  효과: 상대 설득력 -2
```

### 반박 카드

```
논리적 오류 지적:
  효과: 상대방의 논거 무효화
  조건: 실제 논리적 오류 지적해야 함
  종류: 허수아비 논증, 미끄러운 경사,
        인신공격, 순환논증

추가 증거 요구:
  효과: 상대방이 다음 턴에 증거 제시 의무
        제시 못하면 해당 논거 -2

맥락 오류 지적:
  효과: 상대방의 통계·사례 재해석
        설득력 -1

전제 공격:
  효과: 논거의 기반 가정 도전
        성공 시 해당 논거 무효화
```

## 게임 규칙

### 토론 구조

```
1단계: 주제 선정 및 입장 배정
2단계: 준비 시간 (5분)
3단계: 입론 (찬성 → 반대)
4단계: 반론 (반대 → 찬성)
5단계: 재반론 (찬성 → 반대)
6단계: 최종 변론 (양측)
7단계: 배심원 투표
```

### 1단계: 주제 및 입장 배정

주제 카드를 무작위로 뽑습니다.
주사위로 찬반 입장을 배정합니다.

```
중요:
  자신의 실제 의견과 반대 입장을 맡을 수도 있음
  → 다양한 관점 이해 훈련
```

### 2단계: 준비 시간 (5분)

각 팀이 비밀리에 논거를 준비합니다.

```
준비 내용:
  ①핵심 논거 3-4개 선택
  ②논거 카드 배분
  ③반박 예상 및 대응
  ④발언 순서 결정
  ⑤감정 호소 전략
```

### 3단계: 입론 (각 팀 3분)

각 팀이 자신의 주요 논거를 제시합니다.

```
입론 형식:
  "저희 팀은 [입장]을 지지합니다.
   그 이유는 세 가지입니다.
   첫째, [논거 1] + 근거
   둘째, [논거 2] + 근거
   셋째, [논거 3] + 근거
   따라서 [입장]이 타당합니다."

입론 후:
  논거 카드 공개 → 강도 계산
  배심원: 메모 가능
```

### 4단계: 반론 (각 팀 2분)

상대방의 논거에 반박합니다.

```
반론 형식:
  "상대측의 [논거]에 대해 반박합니다.
   [반박 내용 + 반박 카드 사용]
   또한 [추가 논거]를 제시합니다."

반론 규칙:
  반박 카드 최대 2장 사용
  새 논거도 추가 가능 (최대 2개)
```

### 5단계: 재반론 (각 팀 1분)

반론에 대한 재반론을 합니다.

```
재반론 형식:
  "상대측의 반론은 [이유]로 타당하지 않습니다.
   저희의 원래 논거가 여전히 유효한 이유는..."
```

### 6단계: 최종 변론 (각 팀 1분)

핵심 논거를 정리하고 배심원에게 호소합니다.

```
최종 변론 형식:
  "배심원 여러분,
   이번 토론에서 우리는 [핵심 내용]을 보여드렸습니다.
   [감정 호소 or 강력한 마무리 논거]
   따라서 [입장]을 지지해 주시길 부탁드립니다."
```

### 7단계: 배심원 투표

배심원들이 평가하고 투표합니다.

```
배심원 평가 기준:
  ①논거의 논리성 (1-5점)
  ②증거와 사례의 적절성 (1-5점)
  ③반박의 효과성 (1-5점)
  ④전달력과 명확성 (1-5점)

투표:
  비밀 투표로 더 설득력 있는 팀 선택
  과반수 지지 팀이 승리
```

## 논거 강도 트랙

```
논거 강도 트랙:
  찬성팀 [0─────────10]
  반대팀 [0─────────10]

강도 변화:
  새 논거 제시: +논거 강도
  반박 성공: -상대 논거 강도
  논리적 오류 지적 성공: 해당 논거 무효

최종 강도가 높은 팀이 유리하지만
배심원 투표가 최종 결정
```

## 논리적 오류 카드

```
허수아비 논증 (Straw Man):
  정의: 상대 주장을 왜곡해서 반박
  예: "AI를 도입하자는 것은 인간을 무시하는 것"
  (원래 주장: "효율성을 위해 AI 도입")

미끄러운 경사 (Slippery Slope):
  정의: 한 단계에서 극단적 결과 도출
  예: "게임을 허용하면 결국 모두 중독자"

인신공격 (Ad Hominem):
  정의: 논거가 아닌 사람을 공격
  예: "그 회사는 나쁜 회사니까 연구 결과 틀림"

순환 논증 (Circular Reasoning):
  정의: 결론을 전제로 사용
  예: "이것이 사실인 이유는 사실이기 때문"

거짓 이분법 (False Dichotomy):
  정의: 두 가지 선택만 있는 것처럼 제시
  예: "AI를 지지하지 않으면 기술 발전 반대"

성급한 일반화 (Hasty Generalization):
  정의: 소수 사례로 전체 결론
  예: "내 친구가 실패했으니 이 정책은 실패"
```

## 변형 규칙

### 역할 전환 토론
- 입론 후 찬반 입장 교환
- 같은 주제를 반대 입장에서 토론
- 다각적 사고 훈련

### 소크라테스 토론
- 질문과 답변 형식
- 한 팀이 질문, 다른 팀이 답변
- 논리적 일관성 검증

### 즉흥 토론
- 준비 시간 없음
- 주제 발표 즉시 토론 시작
- 즉각적 사고와 표현 훈련

### 다자 토론
- 3가지 이상 입장
- 각자 다른 관점에서 논거
- 복잡한 사회 문제 탐구

## 교육적 활용

### 논리·언어 교육 연계
- **논증**: 전제-추론-결론 구조
- **수사학**: 설득의 기술
- **비판적 사고**: 논거 평가

### 사회 교육 연계
- **다양한 관점**: 사회 문제의 복잡성
- **민주주의**: 토론으로 결정하는 과정
- **시민 교육**: 공론장 참여 능력

### 수업 활용 방법
1. 논증 구조 설명 (10분)
2. 교사 시범 토론 (10분)
3. 학생 토론 (30분)
4. 논리적 오류 찾기 복기 (10분)
