# 자원 협정 (Resource Treaty)

> 서로 다른 자원을 보유한 플레이어들이
> 협상과 거래를 통해
> 자신에게 필요한 자원을 확보하고
> 최적의 조합을 만드는 협상 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-5명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 협상, 자원 분배, 경제적 사고, 상호 이익 |

---

## 게임 목표

8라운드 후 **자원 조합 점수**가
가장 높은 플레이어가 승리합니다.
자원 조합이 완성될수록 더 높은 점수를 얻습니다.

---

## 준비물

- 자원 카드 (6종 × 15장 = 90장)
- 목표 조합 카드 (플레이어별 비밀)
- 협정 기록 보드
- 자원 토큰 (6종 × 20개)
- 협상 타이머 (3분)
- 거래 기록지
- 점수 계산표

### 자원 종류

```
물 (Water):          파랑 카드
식량 (Food):         초록 카드
에너지 (Energy):     노랑 카드
광물 (Mineral):      회색 카드
기술 (Technology):   보라 카드
인력 (Labor):        주황 카드
```

---

## 목표 조합 카드 (비밀)

각 플레이어는 비밀 목표 조합을 받습니다.
이 조합을 완성할수록 높은 점수를 얻습니다.

```
목표 조합 예시:

도시 건설:
  필요: 물×3 + 식량×2 + 광물×4 + 인력×3
  기본 점수: 25점
  완성 보너스: +10점

에너지 독립:
  필요: 에너지×5 + 기술×3 + 광물×2
  기본 점수: 20점
  완성 보너스: +15점

농업 혁명:
  필요: 물×4 + 식량×4 + 인력×4 + 기술×1
  기본 점수: 22점
  완성 보너스: +12점

기술 혁신:
  필요: 기술×5 + 에너지×3 + 인력×2
  기본 점수: 18점
  완성 보너스: +20점

무역 제국:
  필요: 모든 자원 각 2개 이상
  기본 점수: 30점
  완성 보너스: +5점

생태 사회:
  필요: 물×3 + 식량×3 + 에너지×2 (재생)
  기본 점수: 20점
  완성 보너스: +15점 + 환경 보너스
```

## 게임 준비

### 1. 자원 배분

자원 카드를 섞어서 각 플레이어에게 동등하게 나눠줍니다.

```
3인 게임: 각 8장
4인 게임: 각 6장
5인 게임: 각 5장
나머지: 공개 시장에 배치
```

### 2. 목표 조합 카드 배분

각 플레이어에게 목표 조합 카드를 비밀로 1장 줍니다.
게임 종료 전까지 공개하지 않습니다.

### 3. 공개 시장 설정

나머지 자원 카드를 시장 보드에 공개합니다.
시장에서 자원을 구매할 수 있습니다.

```
시장 가격 (기본):
  물: 2 코인
  식량: 2 코인
  에너지: 3 코인
  광물: 3 코인
  기술: 4 코인
  인력: 2 코인

시작 자금: 20 코인 (플레이어당)
```

---

## 게임 규칙

### 라운드 구조

```
1단계: 시장 가격 변동
2단계: 자유 협상 (3분)
3단계: 공식 거래 선언
4단계: 시장 거래
5단계: 조합 점수 체크
```

### 1단계: 시장 가격 변동

주사위를 굴려서 시장 가격을 변동시킵니다.

```
가격 변동표:
  1: 물 +1 코인 (가뭄)
  2: 식량 -1 코인 (풍작)
  3: 에너지 +2 코인 (에너지 위기)
  4: 기술 -1 코인 (기술 발전)
  5: 광물 +1 코인 (광물 부족)
  6: 모든 자원 +1 코인 (인플레이션)
```

### 2단계: 자유 협상 (3분)

플레이어들이 자유롭게 거래를 협상합니다.

```
협상 유형:

단순 교환:
  "내 에너지 2개와 당신의 물 3개를 교환해요."

조건부 거래:
  "이번에 물 2개를 주면, 다음 라운드에
   기술 1개를 드릴게요."

다자 거래:
  "A에게 에너지, B에게 식량, C에게 광물을
   각각 주고 모두에게서 인력을 받아요."

정보 거래:
  "제 목표를 일부 알려드릴게요.
   대신 기술 카드 1개만 주세요."

거짓 협상:
  허위 정보 제공 가능
  단, 발각 시 신뢰도 하락
```

### 3단계: 공식 거래 선언

협상에서 합의한 거래를 공식적으로 선언합니다.

```
공식 거래 절차:
  ①거래 내용 공개 선언
  ②양측이 동시에 카드 교환
  ③거래 기록지에 기록
  ④분쟁 시 기록지가 근거
```

**거래 취소:**
협상 중 합의한 거래도
공식 선언 전에는 취소 가능합니다.

### 4단계: 시장 거래

공식 거래 후 시장에서 추가 거래를 합니다.

```
시장 거래 규칙:
  현재 시장 가격으로 구매·판매
  한 번에 최대 3카드
  보유 코인 범위 내에서만 구매
```

### 5단계: 조합 점수 체크

현재 보유한 자원으로 목표 조합을 얼마나
달성했는지 확인합니다.

```
중간 점수 (라운드당):
  목표 자원 50% 달성: +2점
  목표 자원 75% 달성: +5점
  목표 자원 100% 달성: 즉시 완성 선언 가능

완성 선언:
  목표 조합 완전 달성 시 선언
  기본 점수 + 완성 보너스 획득
  단, 다른 플레이어도 계속 게임
  (더 늦게 완성한 플레이어와 최종 비교)
```

## 협상 전략 카드

게임 시작 시 각 플레이어에게 전략 카드 2장을 줍니다.
게임 중 1회 사용 가능합니다.

```
정보 탐색 (Intel Gathering):
  효과: 한 플레이어의 보유 자원 전체 확인
  사용 시기: 협상 단계
  비용: 없음

시장 조작 (Market Manipulation):
  효과: 원하는 자원의 시장 가격 ±2 조정
  사용 시기: 가격 변동 단계
  비용: 5 코인

긴급 수입 (Emergency Import):
  효과: 시장에서 자원 5개를 기본 가격으로 구매
  사용 시기: 시장 거래 단계
  비용: 없음 (가격만 지불)

거래 거부 (Trade Refusal):
  효과: 이번 라운드 다른 플레이어의
        거래 요청 1건 강제 거부
  사용 시기: 협상 단계
  비용: 없음

연대 협정 (Solidarity Pact):
  효과: 특정 플레이어와 이번 라운드
        모든 거래 수수료 없음
  사용 시기: 협상 단계
  비용: 없음

자원 덤핑 (Resource Dumping):
  효과: 원하는 자원을 시장에 대량 방출
        해당 자원 가격 -3 (1라운드)
  사용 시기: 가격 변동 단계
  비용: 자원 3개 시장 반납
```

## 특수 거래 규칙

### 선물 거래 (Futures)

미래의 자원을 미리 거래합니다.

```
선물 거래 규칙:
  현재 자원 없이 미래 자원 약속
  약속 기간: 최대 2라운드
  이행 불이행 시: 신뢰도 -3, 코인 -5

선물 거래 카드:
  "2라운드 후 에너지 3개를 드리겠습니다.
   지금 광물 2개를 주세요."
```

### 연합 거래 (Coalition Trade)

3명 이상이 함께 하는 복합 거래입니다.

```
연합 거래 효과:
  3인 연합: 각자 거래 시보다 10% 유리한 조건
  4인 연합: 각자 거래 시보다 20% 유리한 조건
  5인 연합: 시장 가격 영향 가능

연합 조건:
  모든 참여자 동의 필요
  한 명이라도 거부 시 무효
```

### 독점 거래 (Monopoly Trade)

특정 자원을 독점적으로 보유할 때
유리한 거래 조건을 얻을 수 있습니다.

```
독점 조건:
  특정 자원 전체의 60% 이상 보유

독점 효과:
  해당 자원 판매 가격 +50%
  다른 플레이어 필요 시 독점 가격 수용
  단, 다른 플레이어 연합으로 독점 해제 가능
```

## 분쟁 해결

### 거래 분쟁

```
분쟁 유형:
  ①약속 불이행: 합의된 거래를 이행하지 않음
  ②허위 정보: 거짓 정보로 거래 유도
  ③강압 거래: 불리한 조건 강요

분쟁 해결:
  다른 플레이어들의 다수결로 판정
  
결과:
  약속 불이행 확인: -5 코인, 신뢰도 -3
  허위 정보 확인: -3 코인, 신뢰도 -2
  강압 거래 확인: 거래 무효, 경고
```

## 점수 계산

### 게임 종료 (8라운드 후)

```
기본 점수:
  목표 자원 달성률 × 기본 점수

완성 보너스:
  조합 완성 시 보너스 점수

조기 완성 보너스:
  5라운드 이내 완성: +10점
  6라운드 이내 완성: +7점
  7라운드 이내 완성: +5점

자원 가치 점수:
  미사용 자원 × 현재 시장 가격 × 0.5

거래 효율 보너스:
  총 거래 수익 (판매가 - 구매가) > 20: +5점

최종 점수 = 목표 달성 점수 + 완성 보너스
           + 조기 완성 + 자원 가치 + 거래 효율
```

## 전략 가이드

### 초보자 전략

**1. 목표 조합 집중**
모든 자원을 조금씩 모으는 것보다
목표 조합에 필요한 자원에 집중하세요.

**2. 정보 관리**
자신의 목표 조합을 너무 일찍 공개하면
다른 플레이어들이 필요한 자원을 올리거나
제공을 거부할 수 있습니다.

**3. 초기 거래**
초반에 빠르게 핵심 자원을 확보하세요.
나중에는 경쟁이 심해집니다.

### 중급 전략

**1. 상대 목표 파악**
다른 플레이어들의 거래 패턴을 보면
그들의 목표 조합을 추측할 수 있습니다.

**2. 희소 자원 선점**
기술 자원은 가장 비싸고 희귀합니다.
초반에 확보해두면 유리한 조건으로 거래 가능합니다.

**3. 시장 가격 활용**
에너지 가격이 높을 때 에너지를 팔고
낮을 때 사는 시장 타이밍 전략이 효과적입니다.

### 고급 전략

**1. 독점 전략**
특정 자원을 독점하면 강력한 협상력을 얻습니다.
단, 다른 플레이어들의 연합 반격을 조심하세요.

**2. 다자 연합 활용**
3인 이상의 연합 거래는 개별 거래보다 유리합니다.
연합을 주도하는 사람이 가장 이익을 얻습니다.

**3. 정보 거래**
자신의 목표 일부를 공개해서
상대방의 목표 정보를 얻는 교환도 유효합니다.

## 변형 규칙

### 협력 자원 협정
- 모든 플레이어가 공동 목표 달성
- 외부 "자원 고갈" 카드가 방해
- 협력으로 공동 목표 완성

### 경쟁 경매 자원 협정
- 자유 거래 없음
- 모든 자원을 경매로만 거래
- 경매 전략 집중

### 정보 공개 자원 협정
- 모든 목표 조합 공개
- 순수 협상 전략만 경쟁
- 투명한 거래 환경

---

## 교육적 활용

### 경제 교육 연계
- **수요·공급**: 자원 희소성과 가격
- **무역 이익**: 비교우위와 교역
- **시장 구조**: 경쟁·독점·협력

### 소통 교육 연계
- **협상**: 상호 이익 찾기
- **설득**: 거래 조건 제시
- **정보 관리**: 전략적 정보 공개

### 수업 활용 방법
1. 자원 분배와 무역 이익 개념 (10분)
2. 3인 게임 3라운드 체험 (15분)
3. 본게임 진행 (30분)
4. "최선의 거래 전략은?" 토론 (10분)

| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 협상, 자원 분배, 경제적 사고, 상호 이익 |

## 게임 목표

8라운드 후 **자원 조합 점수**가
가장 높은 플레이어가 승리합니다.
자원 조합이 완성될수록 더 높은 점수를 얻습니다.

---

## 준비물

- 자원 카드 (6종 × 15장 = 90장)
- 목표 조합 카드 (플레이어별 비밀)
- 협정 기록 보드
- 자원 토큰 (6종 × 20개)
- 협상 타이머 (3분)
- 거래 기록지
- 점수 계산표

### 자원 종류

```
물 (Water):          파랑 카드
식량 (Food):         초록 카드
에너지 (Energy):     노랑 카드
광물 (Mineral):      회색 카드
기술 (Technology):   보라 카드
인력 (Labor):        주황 카드
```

## 목표 조합 카드 (비밀)

각 플레이어는 비밀 목표 조합을 받습니다.
이 조합을 완성할수록 높은 점수를 얻습니다.

```
목표 조합 예시:

도시 건설:
  필요: 물×3 + 식량×2 + 광물×4 + 인력×3
  기본 점수: 25점
  완성 보너스: +10점

에너지 독립:
  필요: 에너지×5 + 기술×3 + 광물×2
  기본 점수: 20점
  완성 보너스: +15점

농업 혁명:
  필요: 물×4 + 식량×4 + 인력×4 + 기술×1
  기본 점수: 22점
  완성 보너스: +12점

기술 혁신:
  필요: 기술×5 + 에너지×3 + 인력×2
  기본 점수: 18점
  완성 보너스: +20점

무역 제국:
  필요: 모든 자원 각 2개 이상
  기본 점수: 30점
  완성 보너스: +5점

생태 사회:
  필요: 물×3 + 식량×3 + 에너지×2 (재생)
  기본 점수: 20점
  완성 보너스: +15점 + 환경 보너스
```

## 게임 준비

### 1. 자원 배분

자원 카드를 섞어서 각 플레이어에게 동등하게 나눠줍니다.

```
3인 게임: 각 8장
4인 게임: 각 6장
5인 게임: 각 5장
나머지: 공개 시장에 배치
```

### 2. 목표 조합 카드 배분

각 플레이어에게 목표 조합 카드를 비밀로 1장 줍니다.
게임 종료 전까지 공개하지 않습니다.

### 3. 공개 시장 설정

나머지 자원 카드를 시장 보드에 공개합니다.
시장에서 자원을 구매할 수 있습니다.

```
시장 가격 (기본):
  물: 2 코인
  식량: 2 코인
  에너지: 3 코인
  광물: 3 코인
  기술: 4 코인
  인력: 2 코인

시작 자금: 20 코인 (플레이어당)
```

## 게임 규칙

### 라운드 구조

```
1단계: 시장 가격 변동
2단계: 자유 협상 (3분)
3단계: 공식 거래 선언
4단계: 시장 거래
5단계: 조합 점수 체크
```

### 1단계: 시장 가격 변동

주사위를 굴려서 시장 가격을 변동시킵니다.

```
가격 변동표:
  1: 물 +1 코인 (가뭄)
  2: 식량 -1 코인 (풍작)
  3: 에너지 +2 코인 (에너지 위기)
  4: 기술 -1 코인 (기술 발전)
  5: 광물 +1 코인 (광물 부족)
  6: 모든 자원 +1 코인 (인플레이션)
```

### 2단계: 자유 협상 (3분)

플레이어들이 자유롭게 거래를 협상합니다.

```
협상 유형:

단순 교환:
  "내 에너지 2개와 당신의 물 3개를 교환해요."

조건부 거래:
  "이번에 물 2개를 주면, 다음 라운드에
   기술 1개를 드릴게요."

다자 거래:
  "A에게 에너지, B에게 식량, C에게 광물을
   각각 주고 모두에게서 인력을 받아요."

정보 거래:
  "제 목표를 일부 알려드릴게요.
   대신 기술 카드 1개만 주세요."

거짓 협상:
  허위 정보 제공 가능
  단, 발각 시 신뢰도 하락
```

### 3단계: 공식 거래 선언

협상에서 합의한 거래를 공식적으로 선언합니다.

```
공식 거래 절차:
  ①거래 내용 공개 선언
  ②양측이 동시에 카드 교환
  ③거래 기록지에 기록
  ④분쟁 시 기록지가 근거
```

**거래 취소:**
협상 중 합의한 거래도
공식 선언 전에는 취소 가능합니다.

### 4단계: 시장 거래

공식 거래 후 시장에서 추가 거래를 합니다.

```
시장 거래 규칙:
  현재 시장 가격으로 구매·판매
  한 번에 최대 3카드
  보유 코인 범위 내에서만 구매
```

### 5단계: 조합 점수 체크

현재 보유한 자원으로 목표 조합을 얼마나
달성했는지 확인합니다.

```
중간 점수 (라운드당):
  목표 자원 50% 달성: +2점
  목표 자원 75% 달성: +5점
  목표 자원 100% 달성: 즉시 완성 선언 가능

완성 선언:
  목표 조합 완전 달성 시 선언
  기본 점수 + 완성 보너스 획득
  단, 다른 플레이어도 계속 게임
  (더 늦게 완성한 플레이어와 최종 비교)
```

## 협상 전략 카드

게임 시작 시 각 플레이어에게 전략 카드 2장을 줍니다.
게임 중 1회 사용 가능합니다.

```
정보 탐색 (Intel Gathering):
  효과: 한 플레이어의 보유 자원 전체 확인
  사용 시기: 협상 단계
  비용: 없음

시장 조작 (Market Manipulation):
  효과: 원하는 자원의 시장 가격 ±2 조정
  사용 시기: 가격 변동 단계
  비용: 5 코인

긴급 수입 (Emergency Import):
  효과: 시장에서 자원 5개를 기본 가격으로 구매
  사용 시기: 시장 거래 단계
  비용: 없음 (가격만 지불)

거래 거부 (Trade Refusal):
  효과: 이번 라운드 다른 플레이어의
        거래 요청 1건 강제 거부
  사용 시기: 협상 단계
  비용: 없음

연대 협정 (Solidarity Pact):
  효과: 특정 플레이어와 이번 라운드
        모든 거래 수수료 없음
  사용 시기: 협상 단계
  비용: 없음

자원 덤핑 (Resource Dumping):
  효과: 원하는 자원을 시장에 대량 방출
        해당 자원 가격 -3 (1라운드)
  사용 시기: 가격 변동 단계
  비용: 자원 3개 시장 반납
```

## 특수 거래 규칙

### 선물 거래 (Futures)

미래의 자원을 미리 거래합니다.

```
선물 거래 규칙:
  현재 자원 없이 미래 자원 약속
  약속 기간: 최대 2라운드
  이행 불이행 시: 신뢰도 -3, 코인 -5

선물 거래 카드:
  "2라운드 후 에너지 3개를 드리겠습니다.
   지금 광물 2개를 주세요."
```

### 연합 거래 (Coalition Trade)

3명 이상이 함께 하는 복합 거래입니다.

```
연합 거래 효과:
  3인 연합: 각자 거래 시보다 10% 유리한 조건
  4인 연합: 각자 거래 시보다 20% 유리한 조건
  5인 연합: 시장 가격 영향 가능

연합 조건:
  모든 참여자 동의 필요
  한 명이라도 거부 시 무효
```

### 독점 거래 (Monopoly Trade)

특정 자원을 독점적으로 보유할 때
유리한 거래 조건을 얻을 수 있습니다.

```
독점 조건:
  특정 자원 전체의 60% 이상 보유

독점 효과:
  해당 자원 판매 가격 +50%
  다른 플레이어 필요 시 독점 가격 수용
  단, 다른 플레이어 연합으로 독점 해제 가능
```

## 분쟁 해결

### 거래 분쟁

```
분쟁 유형:
  ①약속 불이행: 합의된 거래를 이행하지 않음
  ②허위 정보: 거짓 정보로 거래 유도
  ③강압 거래: 불리한 조건 강요

분쟁 해결:
  다른 플레이어들의 다수결로 판정
  
결과:
  약속 불이행 확인: -5 코인, 신뢰도 -3
  허위 정보 확인: -3 코인, 신뢰도 -2
  강압 거래 확인: 거래 무효, 경고
```

## 점수 계산

### 게임 종료 (8라운드 후)

```
기본 점수:
  목표 자원 달성률 × 기본 점수

완성 보너스:
  조합 완성 시 보너스 점수

조기 완성 보너스:
  5라운드 이내 완성: +10점
  6라운드 이내 완성: +7점
  7라운드 이내 완성: +5점

자원 가치 점수:
  미사용 자원 × 현재 시장 가격 × 0.5

거래 효율 보너스:
  총 거래 수익 (판매가 - 구매가) > 20: +5점

최종 점수 = 목표 달성 점수 + 완성 보너스
           + 조기 완성 + 자원 가치 + 거래 효율
```

## 전략 가이드

### 초보자 전략

**1. 목표 조합 집중**
모든 자원을 조금씩 모으는 것보다
목표 조합에 필요한 자원에 집중하세요.

**2. 정보 관리**
자신의 목표 조합을 너무 일찍 공개하면
다른 플레이어들이 필요한 자원을 올리거나
제공을 거부할 수 있습니다.

**3. 초기 거래**
초반에 빠르게 핵심 자원을 확보하세요.
나중에는 경쟁이 심해집니다.

### 중급 전략

**1. 상대 목표 파악**
다른 플레이어들의 거래 패턴을 보면
그들의 목표 조합을 추측할 수 있습니다.

**2. 희소 자원 선점**
기술 자원은 가장 비싸고 희귀합니다.
초반에 확보해두면 유리한 조건으로 거래 가능합니다.

**3. 시장 가격 활용**
에너지 가격이 높을 때 에너지를 팔고
낮을 때 사는 시장 타이밍 전략이 효과적입니다.

### 고급 전략

**1. 독점 전략**
특정 자원을 독점하면 강력한 협상력을 얻습니다.
단, 다른 플레이어들의 연합 반격을 조심하세요.

**2. 다자 연합 활용**
3인 이상의 연합 거래는 개별 거래보다 유리합니다.
연합을 주도하는 사람이 가장 이익을 얻습니다.

**3. 정보 거래**
자신의 목표 일부를 공개해서
상대방의 목표 정보를 얻는 교환도 유효합니다.

- 순수 협상 전략만 경쟁
- 투명한 거래 환경
