# 딜레마 협상 (Dilemma Deal)

> 게임 이론의 고전적 딜레마 상황을
> 다양한 형태로 경험하는 게임입니다.
> 협력과 배신 사이에서 최선의 선택을 찾고
> 상대방의 전략을 예측하세요.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 25-35분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 게임 이론, 협력과 경쟁, 신뢰, 전략적 사고 |

---

## 게임 목표

10라운드 후 **누적 점수가 가장 높은**
플레이어가 승리합니다.

---

## 준비물

- 딜레마 보드 (결과 매트릭스 표시)
- 선택 카드 (협력·배신 각 플레이어당 1장씩)
- 딜레마 시나리오 카드 30장
- 점수 토큰
- 신뢰 트랙 (플레이어 간)
- 협상 타이머 (2분)
- 비밀 메모지
- 이력 기록판

### 기본 딜레마 매트릭스

```
죄수의 딜레마 기본 보수표:

            상대: 협력    상대: 배신
나: 협력    (3, 3)       (0, 5)
나: 배신    (5, 0)       (1, 1)

숫자 = 각 플레이어의 점수
(나의점수, 상대점수)

해석:
  둘 다 협력: 각 3점 (최선의 공동 이익)
  내가 배신, 상대 협력: 나 5점, 상대 0점
  내가 협력, 상대 배신: 나 0점, 상대 5점
  둘 다 배신: 각 1점 (최악의 공동 결과)
```

## 딜레마 시나리오 카드

### 기본 시나리오

```
시나리오 #01: 고전 죄수의 딜레마
  상황: 두 용의자가 각자 심문을 받고 있습니다.
  협력 = 묵비권 행사
  배신 = 자백

  보수표:
    둘 다 협력: 각 3년 감형 (3점)
    나 배신·상대 협력: 나 석방(5점), 상대 10년(0점)
    나 협력·상대 배신: 나 10년(0점), 상대 석방(5점)
    둘 다 배신: 각 5년 복역(1점)

시나리오 #02: 공유지의 비극
  상황: 공유 목초지에서 가축을 키웁니다.
  협력 = 적정 수준 방목
  배신 = 과잉 방목

  보수표:
    둘 다 협력: 각 4점 (지속 가능)
    나 배신·상대 협력: 나 6점, 상대 2점
    나 협력·상대 배신: 나 2점, 상대 6점
    둘 다 배신: 각 0점 (목초지 황폐화)

시나리오 #03: 군비 경쟁
  상황: 두 국가가 군비를 결정합니다.
  협력 = 군비 축소
  배신 = 군비 확장

  보수표:
    둘 다 협력: 각 4점 (평화 배당)
    나 배신·상대 협력: 나 7점, 상대 0점
    나 협력·상대 배신: 나 0점, 상대 7점
    둘 다 배신: 각 1점 (군비 부담)
```

### 고급 시나리오

```
시나리오 #04: 치킨 게임
  상황: 두 차가 마주보고 달려옵니다.
  협력 = 먼저 핸들 꺾기
  배신 = 직진

  보수표:
    둘 다 협력: 각 2점 (무승부)
    나 배신·상대 협력: 나 5점, 상대 1점
    나 협력·상대 배신: 나 1점, 상대 5점
    둘 다 배신: 각 -5점 (충돌!)

시나리오 #05: 사냥꾼의 딜레마 (사슴 사냥)
  상황: 사슴을 함께 사냥하거나 토끼를 혼자 잡습니다.
  협력 = 사슴 사냥 (팀 필요)
  배신 = 토끼 사냥 (혼자)

  보수표:
    둘 다 협력: 각 5점 (사슴 획득)
    나 배신·상대 협력: 나 3점, 상대 0점
    나 협력·상대 배신: 나 0점, 상대 3점
    둘 다 배신: 각 3점 (토끼 획득)

시나리오 #06: 환경 협약
  상황: 탄소 감축 비용을 누가 부담할지.
  협력 = 비용 부담
  배신 = 무임승차

  보수표:
    둘 다 협력: 각 4점 (환경 개선 공유)
    나 배신·상대 협력: 나 7점, 상대 1점
    나 협력·상대 배신: 나 1점, 상대 7점
    둘 다 배신: 각 0점 (환경 악화)
```

## 게임 규칙

### 라운드 구조

```
1단계: 시나리오 공개
2단계: 협상 단계 (2분)
3단계: 동시 선택
4단계: 결과 공개 및 점수 부여
5단계: 전략 분석
```

### 1단계: 시나리오 공개

시나리오 카드 1장을 공개합니다.
모든 플레이어가 상황과 보수표를 확인합니다.

### 2단계: 협상 단계 (2분)

플레이어들이 자유롭게 협상합니다.

```
협상 규칙:
  ①말로 어떤 약속도 가능
  ②그러나 약속은 강제되지 않음
  ③거짓말 허용 (단, 발각 시 신뢰 하락)
  ④침묵도 전략 (협상 거부 가능)
```

**협상 전략 유형:**
```
설득형:
  "우리 둘 다 협력하면 각 3점이에요.
   배신하면 결국 둘 다 1점밖에 못 받아요."

위협형:
  "만약 배신하면 저도 다음 라운드에
   무조건 배신할 거예요."

동맹형:
  "저는 항상 협력할게요.
   신뢰 관계를 쌓아요."

허풍형:
  (협력하겠다고 말하고 배신)
```

### 3단계: 동시 선택

모든 플레이어가 협력 또는 배신 카드를
동시에 뒤집어서 공개합니다.

```
선택 카드:
  협력 카드 (파랑): [협력]
  배신 카드 (빨강): [배신]
```

### 4단계: 결과 및 점수

보수표에 따라 점수를 부여합니다.

```
2인 게임:
  기본 보수표 직접 적용

3-4인 게임:
  각 플레이어 쌍별로 계산 후 합산

예시 (3인):
  A 협력, B 배신, C 협력
  A vs B: A=0점, B=5점
  A vs C: A=3점, C=3점
  B vs C: B=5점, C=0점
  합계: A=3, B=10, C=3
```

### 5단계: 전략 분석

라운드 후 짧은 토론을 진행합니다.

```
토론 질문:
  "왜 이 선택을 했나요?"
  "상대방의 선택을 예상했나요?"
  "다음 라운드 전략을 바꿀 건가요?"
```

## 신뢰 트랙 시스템

### 신뢰도 변화

```
협력 → 협력: 신뢰 +1
협력 → 배신: 신뢰 -3 (배신당한 쪽)
배신 → 협력: 신뢰 +0 (신뢰 회복 없음)
배신 → 배신: 신뢰 -1

신뢰 효과:
  신뢰 8+: 협상 시 상대방 발언 신뢰도 높음
           다음 협력 시 +1 보너스
  신뢰 5-7: 기본
  신뢰 3-4: 상대방이 협상에 소극적
  신뢰 0-2: 상대방이 협상 거부
            자동 배신 선언 가능
```

## 전략 이론 소개

### 반복 게임 전략

```
팃-포-탯 (Tit-for-Tat):
  1라운드: 협력
  이후: 상대방의 이전 선택을 따라함
  효과: 장기적으로 가장 높은 평균 점수
  원칙: 착하고(협력 시작), 단호하고(배신엔 배신),
        관대하고(한번 용서), 명확함

모두 협력 (Always Cooperate):
  항상 협력
  결과: 착취당하면 낮은 점수

모두 배신 (Always Defect):
  항상 배신
  결과: 단기 이득, 장기 낮은 점수

무작위 (Random):
  무작위 선택
  결과: 예측 불가, 중간 점수

응보 전략 (Grim Trigger):
  협력 시작
  한 번 배신당하면 영원히 배신
  효과: 강력한 억제력, 유연성 없음
```

### 내쉬 균형 분석

```
죄수의 딜레마:
  내쉬 균형: (배신, 배신)
  이유: 상대가 뭘 해도 배신이 개인 최선
  파레토 최적: (협력, 협력)
  딜레마: 개인 합리성 → 집단 비합리성

사슴 사냥:
  내쉬 균형: (협력, 협력) 또는 (배신, 배신)
  이유: 조정 문제 (어느 균형이 더 나은가?)
  해결: 소통으로 균형 선택 가능

치킨 게임:
  내쉬 균형: (협력, 배신) 또는 (배신, 협력)
  이유: 둘 다 같은 선택 시 최악
  문제: 누가 먼저 물러설 것인가?
```

## 특수 라운드 규칙

### 공개 라운드
- 협상 없이 바로 선택
- 상대방의 진짜 전략 파악

### 소통 금지 라운드
- 협상 단계 없음
- 비언어적 신호만 가능

### 다자 딜레마 라운드
- 3-4인이 동시에 딜레마 상황
- 연합 형성 가능 (일부만 협력 선택)

### 반복 딜레마 라운드
- 같은 시나리오를 3번 연속
- 전략 진화 관찰

---

## 변형 규칙

### 진화 딜레마
- 점수가 낮은 플레이어는 전략을 바꿔야 함
- 최고 점수 전략이 살아남는 진화 관찰

### 협력 딜레마
- 모든 플레이어가 외부 위협에 맞서 협력
- 내부 배신자 찾아내기

### 토너먼트 딜레마
- 다양한 전략끼리 토너먼트
- 최고 전략 결정

---

## 교육적 활용

### 수학·경제 연계
- **게임 이론**: 보수 행렬과 내쉬 균형
- **의사결정**: 불확실한 상황의 최적 선택
- **확률**: 상대방 전략의 확률적 예측

### 사회·윤리 연계
- **협력의 가치**: 공동 이익의 중요성
- **신뢰**: 신뢰가 만드는 사회적 가치
- **딜레마**: 개인과 집단 이익의 충돌

### 수업 활용 방법
1. 게임 이론 기초 설명 (10분)
2. 시나리오 #01 체험 (10분)
3. 다양한 시나리오 플레이 (20분)
4. 실생활 딜레마 토론 (10분)
