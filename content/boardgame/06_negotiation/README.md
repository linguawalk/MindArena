# 협상 게임 (Negotiation Games)

> 언어와 논리, 심리와 전략으로
> 상대방을 설득하고 최선의 합의를 이끌어내는
> 게임입니다.
> 소통 능력과 전략적 사고가 핵심입니다.

---

## 카테고리 특징

- **정보 비대칭**: 각자 다른 정보와 목표를 가짐
- **설득과 논증**: 논리적 근거로 상대방 설득
- **상호 이익**: 모두에게 유리한 합의 도출
- **교육적 가치**: 소통, 논증, 갈등 해결, 공감 능력

---

---

## 교육 목표

- 논리적 설득과 논증 능력
- 적극적 경청과 공감 능력
- 갈등 해결과 합의 도출
- 다양한 관점 이해와 통합
