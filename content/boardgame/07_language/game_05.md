# 은유 배틀 (Metaphor Battle)

> "인생은 ___이다."
> 빈칸을 채우는 것이 아닙니다.
> 왜 그것인지 설득하는 것입니다.
> 가장 탁월한 비유가 이깁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 40-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 추상적 사고, 논증 능력, 창의적 연결 |

## 게임 구성

```
주제 카드: 50장 (추상 개념들)
비유어 카드: 100장 (구체적 사물·현상)
논증 점수판
```

## 기본 규칙

```
준비:
  주제 카드 1장 공개 (예: "우정")
  모든 플레이어 비유어 카드 3장 받기

선택 (1분):
  손에 든 카드 중 1장 선택
  "우정은 [선택한 카드]이다" 구성

발표 및 논증 (각 2분):
  왜 그 비유가 적절한지 설명
  유사점 최소 3가지 제시

평가:
  다른 플레이어 투표
  (자신 제외)

득점:
  1위: +3점 / 2위: +1점
  가장 독창적 비유: 보너스 +2점
```

## 주제 카드 예시

```
우정 / 시간 / 자유 / 성공 / 두려움
지식 / 사랑 / 변화 / 희망 / 실패
민주주의 / 창의성 / 책임 / 고독 / 성장
```

## 비유어 카드 예시

```
뿌리 / 거울 / 씨앗 / 파도 / 나침반
지도 / 불꽃 / 다리 / 그림자 / 열쇠
퍼즐 / 길 / 계절 / 빛 / 무게
```

## 논증 예시

```
주제: "두려움"
비유어: "그림자"

논증:
  ①항상 따라다닌다
  ②빛이 강할수록 커진다
  ③실체가 없지만 영향을 미친다
  ④완전히 없앨 수 없다
  ⑤때로는 방향을 알려주기도 한다
```

## 교육적 효과

```
추상적 사고:
  개념의 본질 파악
  → 철학적 사고력 기초

유추 능력:
  서로 다른 영역 연결
  → 창의성의 핵심 메커니즘

논증 능력:
  주장 + 근거 구조 훈련
  → 논리적 설득력 향상
```

## 심화 토론

```
게임 후 토론 질문:

"가장 설득력 있었던 비유는 무엇이었나요?
 왜 그것이 탁월했나요?"

"같은 개념을 완전히 다르게 표현한
 두 비유가 있었다면,
 둘 다 맞을 수 있나요?"

"당신이 가장 두렵게 생각하는 것을
 무엇에 비유하겠습니까?"
```
