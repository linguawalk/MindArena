# 이야기 짓기 (Story Builder)

> 카드 3장으로 이야기를 만드세요.
> 황당할수록, 논리적일수록,
> 감동적일수록 점수를 얻습니다.
> 최고의 이야기꾼이 되어보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 창의적 사고, 서사 구성, 표현력 |

## 게임 구성

```
인물 카드: 40장 (직업·성격·특징)
장소 카드: 40장 (배경·환경·시대)
사건 카드: 40장 (갈등·반전·결말 요소)
평가 토큰: 각 플레이어 5개
```

## 기본 규칙

```
준비:
  각 덱을 섞어 뒤집어 놓기
  각 플레이어 인물·장소·사건 카드 1장씩 뽑기

이야기 만들기 (3분):
  뽑은 3장 카드를 모두 포함한
  1-2분 분량의 이야기 구성

발표:
  차례대로 이야기 발표

평가:
  발표자 제외 나머지 플레이어
  토큰 0-2개로 평가
  (0: 아쉬움 / 1: 좋음 / 2: 훌륭함)

득점:
  받은 토큰 합산
  보너스: 카드 3장 모두 자연스럽게 연결 시 +2
```

## 카드 예시

```
인물 카드:
  "퇴직한 우주비행사"
  "말을 못하는 마법사"
  "시간을 파는 상인"

장소 카드:
  "빛이 없는 도서관"
  "물속의 사막"
  "어제의 서울"

사건 카드:
  "중요한 것을 잃어버렸다"
  "두 가지 선택 앞에 서다"
  "예상치 못한 만남"
```

## 이야기 구성 가이드

```
좋은 이야기의 3요소:
  ①시작: 인물과 상황 소개
  ②전개: 사건과 갈등
  ③결말: 해결 또는 여운

평가 기준:
  창의성: 카드를 얼마나 독창적으로 활용했는가
  논리성: 이야기 흐름이 자연스러운가
  감동/재미: 듣는 사람이 즐거웠는가
```

## 심화 모드

```
장르 제한:
  추가 장르 카드 뽑기
  (공포 / 코미디 / 로맨스 / SF)
  해당 장르로만 이야기 구성

협력 모드:
  한 문장씩 돌아가며 이야기 이어가기
  마지막 플레이어가 결말 담당

즉흥 모드:
  준비 시간 없이 즉석 발표
```

## 교육적 효과

```
서사 사고:
  시작·전개·결말 구조 이해
  인과관계 사고력 향상

창의적 연결:
  무관해 보이는 요소 연결
  → 창의성 핵심 능력

공감 능력:
  다른 사람의 이야기 경청·평가
  → 소통 능력 향상
```
