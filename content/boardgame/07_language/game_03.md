# 금지어 탈출 (Taboo Escape)

> 이 단어를 설명하되
> 금지어는 절대 말하면 안 됩니다.
> 가장 핵심적인 단어들이 막힐 때
> 진짜 언어 능력이 드러납니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 어휘력, 설명 능력, 우회적 사고 |

## 게임 구성

```
설명 카드: 100장
  각 카드: 목표 단어 1개 + 금지어 5개
모래시계: 60초
버저: 팀별 1개
점수 칩: 각 색상 20개
```

## 기본 규칙

```
팀 구성:
  2팀으로 나누기 (짝수 인원)
  홀수 시: 심판 1명

설명자 역할:
  카드를 보고 목표 단어 설명
  금지어 5개 사용 불가
  몸짓·그림 불가

팀원 역할:
  설명 듣고 목표 단어 맞추기
  60초 내 최대한 많이

상대팀 역할:
  금지어 사용 감시
  금지어 사용 시 버저!

득점:
  맞힌 단어: +1점
  금지어 사용: -1점
  패스: 0점 (무제한)
```

## 카드 예시

```
카드 1:
  목표: 사랑
  금지어: 마음 / 좋아 / 감정 / 연인 / 행복

카드 2:
  목표: 도서관
  금지어: 책 / 공부 / 조용 / 빌리다 / 열람

카드 3:
  목표: 민주주의
  금지어: 투표 / 자유 / 선거 / 국민 / 권리
```

## 난이도 조절

```
쉬움: 금지어 3개
보통: 금지어 5개
어려움: 금지어 7개
최고난도: 금지어 + 관련 품사 전체 금지
```

## 교육적 효과

```
우회적 표현:
  직접 표현 불가 시 다른 방법 탐색
  → 언어 유연성 향상

개념 이해:
  단어의 본질을 다른 말로 설명
  → 깊은 이해력 확인

협력 소통:
  팀원과 빠른 언어 신호 교환
  → 소통 효율성 훈련
```

## 자체 제작 카드 가이드

```
좋은 금지어 선택 기준:
  ①가장 먼저 떠오르는 단어
  ②목표 단어의 상위 개념
  ③목표 단어의 핵심 속성
  ④연관 행동·동사
  ⑤자주 함께 쓰이는 단어
```
