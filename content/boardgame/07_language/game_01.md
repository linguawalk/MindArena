# 단어 사슬 (Word Chain Battle)

> 끊어지면 지는 게임.
> 마지막 글자로 새 단어를 만들되
> 상대보다 더 오래, 더 넓게 연결해야 합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 어휘력, 순발력, 연상 사고 |

## 게임 구성

```
카드 덱: 주제 카드 60장
          (자연 / 도시 / 음식 / 직업 / 감정)
모래시계: 30초
점수판: 6인용
```

## 기본 규칙

```
1. 주제 카드 1장 공개
   → 해당 주제 관련 단어만 사용

2. 시작 플레이어가 단어 1개 말하기

3. 다음 플레이어:
   → 앞 단어의 마지막 글자로 시작하는
   → 같은 주제 단어 말하기
   → 30초 내에

4. 실패 조건:
   → 시간 초과
   → 이미 사용된 단어
   → 주제 벗어난 단어
   → 끝말 불일치

5. 실패 시: 1점 감점
   라운드 종료: 5번 실패 시
```

## 고급 규칙 (심화)

```
더블 체인:
  두 글자 이상 겹치는 단어 연결 시 +2점

주제 전환 카드:
  패스 대신 주제를 바꿀 수 있음 (1회)

블로킹:
  의도적으로 이어가기 어려운 단어 선택
  → 전략적 사고 유도
```

## 교육적 효과

```
어휘 확장:
  다양한 주제의 단어를 빠르게 탐색
  → 어휘 범주화 능력 향상

순발력:
  제한 시간 내 언어 처리
  → 유창성 훈련

연상 사고:
  주제 내 단어 네트워크 형성
  → 개념 연결 능력
```

## 변형 규칙

```
영어 모드:
  한영 교차 사용 허용
  (언어 학습 효과)

감정 사슬:
  감정 관련 단어만 사용
  (정서 어휘 확장)

학습 모드:
  틀린 단어도 설명하면 점수 유지
  (오류를 학습 기회로)
```

## 진행 예시

```
주제: 자연

플레이어 A: "바람"
플레이어 B: "람사르" (X - 고유명사)
           "람쥐" (X - 주제 벗어남)
           "람" (X - 단어 아님)
           → 실패

플레이어 A: "바람"
플레이어 B: "람"은 안 되니... "바다" (X - 첫글자 불일치)
플레이어 B: "람풍" → 없는 단어
→ 30초 경과 → 실패

올바른 예:
A: "구름" → B: "름" → "름"으로 시작하는 단어 없음
A: "나무" → B: "무지개" → C: "개울" → A: "울창한숲"
```
