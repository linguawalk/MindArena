# 한 문장 게임 (One Sentence)

> 같은 상황을 한 문장으로 표현합니다.
> 가장 정확한 문장,
> 가장 아름다운 문장,
> 가장 재미있는 문장.
> 언어는 관점입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 문장 구성력, 관점 사고, 간결한 표현 |

## 게임 구성

```
상황 카드: 60장 (그림 또는 짧은 묘사)
평가 기준 카드: 15장
메모지: 인원수 × 20장
펜: 인원수
```

## 기본 규칙

```
준비:
  상황 카드 1장 공개
  평가 기준 카드 1장 공개
    (정확성 / 감동 / 유머 / 간결함 / 독창성)

작성 (2분):
  각자 해당 상황을 한 문장으로 표현
  평가 기준에 맞게

발표 및 평가:
  익명으로 섞어서 읽기
  투표로 최고의 문장 선정

득점:
  최고 문장 선정: +3점
  2위: +1점
```

## 상황 카드 예시

```
"비가 오는 창가에 홀로 앉은 노인"
"처음으로 바다를 본 아이"
"오래된 친구와의 재회"
"실패한 다음 날 아침"
"로봇이 눈물을 흘리는 장면"
```

## 평가 기준 카드 예시

```
정확성: 상황을 가장 정확하게 담은 문장
감동: 가장 마음을 움직이는 문장
유머: 가장 웃긴 문장
간결함: 가장 짧고 강렬한 문장
독창성: 아무도 생각 못한 표현
```

## 교육적 효과

```
문장 구성:
  주어-서술어-목적어 구조 이해
  → 글쓰기 기초 능력

관점 다양성:
  같은 상황, 다른 표현
  → 다양한 시각 체험

함축적 표현:
  한 문장에 모든 것을 담는 훈련
  → 핵심 파악 능력
```

## 변형 모드

```
시 모드:
  운율이 있는 한 문장

뉴스 모드:
  기자처럼 객관적 한 문장

철학 모드:
  깊은 의미가 담긴 한 문장

어린이 모드:
  어린이도 이해할 수 있는 한 문장
```
