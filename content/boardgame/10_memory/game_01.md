# 순서 기억 (Sequence Memory)

> 순서를 기억하는 것은
> 단순해 보이지만
> 길어질수록 누구나 무너집니다.
> 마지막까지 버티는 자가 승자입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 작업 기억, 순서 기억, 집중력 |

## 게임 구성

```
컬러 타일: 6색 × 5개 (총 30개)
순서 보드: 타일 놓는 줄
기억 카드: 방해 요소 카드 20장
점수판
```

## 기본 규칙

```
진행자 역할 (돌아가며):
  타일을 하나씩 놓으며
  색과 위치 보여주기
  다 보여준 후 뒤집기

기억자 역할:
  본 순서대로 타일 재현
  틀린 위치부터 탈락

라운드:
  3개부터 시작
  성공 시 1개씩 추가
  실패 시 그 수 유지 (2회 실패 시 -1)

승리:
  10개 순서 기억 성공
  또는 최장 순서 기억자
```

## 방해 요소 카드

```
색깔 말하기:
  타일을 보여줄 때
  다른 색 이름 말하기
  → 스트룹 효과 유발

숫자 세기:
  기억하는 동안
  7부터 거꾸로 세기

질문 받기:
  기억 중 간단한 질문에 답하기
```

## 난이도 조절

```
초급: 3색만 사용
중급: 6색 사용
고급: 방해 카드 포함
심화: 색+모양 동시 기억
```

## 교육적 효과

```
작업 기억:
  정보를 잠시 보유하며 처리
  → 학습의 핵심 인지 능력

주의 집중:
  방해 요소에도 집중 유지
  → 선택적 주의력

간섭 극복:
  스트룹 효과 훈련
  → 인지 유연성
```
