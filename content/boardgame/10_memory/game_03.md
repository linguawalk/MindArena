# 이중 과제 (Dual Task)

> 두 가지를 동시에 해야 합니다.
> 하나에 집중하면 다른 하나가 무너집니다.
> 주의 자원을 나누는 훈련.
> 멀티태스킹의 한계를 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 분할 주의, 작업 기억, 인지 유연성 |

## 게임 구성

```
주 과제 카드: 40장 (기억·판단 과제)
부 과제 카드: 40장 (간단한 동작·말하기)
수행 평가표
모래시계: 45초
```

## 기본 규칙

```
주 과제:
  카드 5장 순서 기억
  또는 패턴 판단

부 과제 (동시 수행):
  숫자 거꾸로 세기
  또는 특정 단어 반복
  또는 손가락 두드리기 패턴

수행:
  45초 동안 두 과제 동시 수행
  끝나면 주 과제 결과 답하기

평가:
  주 과제 정확도 + 부 과제 완성도
  두 가지 모두 평가
```

## 과제 조합 예시

```
조합 1 (쉬움):
  주: 색깔 카드 4장 순서 기억
  부: "하나 둘 셋" 반복

조합 2 (중간):
  주: 숫자 카드 5장 합산
  부: 7부터 거꾸로 세기

조합 3 (어려움):
  주: 단어 카드 6장 범주 분류
  부: 왼손 검지로 박자 두드리기
```

## 교육적 효과

```
분할 주의:
  두 채널 동시 처리
  → 실제 생활의 멀티태스킹

한계 인식:
  "완벽한 멀티태스킹은 없다"
  → 집중력의 소중함 체험

인지 유연성:
  과제 전환 능력
  → 실행 기능 훈련
```
