# 집중력 배틀 (Focus Battle)

> 방해를 무시하고
> 목표에만 집중할 수 있는가?
> 소음 속에서도 흔들리지 않는
> 집중력을 훈련합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 선택적 주의, 방해 억제, 집중 지속 |

## 게임 구성

```
목표 카드: 개인당 1장 (찾을 패턴)
탐색 보드: 10×10 그리드 (패턴 가득)
방해 카드: 30장
타이머: 60초
```

## 기본 규칙

```
준비:
  각자 목표 카드 1장 받기
  (찾아야 할 특정 패턴)
  탐색 보드 공개

탐색 (60초):
  보드에서 목표 패턴 찾기
  찾으면 위치 메모

방해:
  다른 플레이어들이
  방해 카드 사용 가능
  (말걸기·소리내기·틀린 정보)

채점:
  찾은 패턴 수 × 정확도
```

## 목표 패턴 예시

```
숫자 패턴: 보드에서 "3-1-4" 연속 찾기
색깔 패턴: 빨-파-노 대각선 찾기
모양 패턴: ○-□-△ 순서 찾기
문자 패턴: 특정 단어 숨어있는 것 찾기
```

## 방해 카드 예시

```
"거기 있다!" (틀린 위치 지적)
"시간 10초 남았어!" (거짓 경고)
"내 패턴 찾았어!" (집중 흔들기)
"그거 아니야" (확인된 답 부정)
손뼉 치기 (청각 방해)
```

## 집중력 레벨

```
Level 1: 방해 없음
Level 2: 청각 방해만
Level 3: 언어 방해 포함
Level 4: 모든 방해 허용
Level 5: 진행자도 방해 참여
```

## 교육적 효과

```
선택적 주의:
  목표 자극에만 집중
  방해 자극 무시
  → 칵테일 파티 효과 훈련

집중 지속:
  방해에도 흔들리지 않기
  → 실제 학습 환경 시뮬레이션

억제 능력:
  충동·방해 억제
  → 실행 기능의 핵심
```

## 게임 후 토론

```
"어떤 방해가 가장 효과적이었나요?
 왜 그것이 집중을 방해했을까요?"

"집중력을 높이기 위해
 일상에서 무엇을 바꿀 수 있을까요?"

"완전한 집중이 가능한 환경은
 어떤 환경인가요?"
```
