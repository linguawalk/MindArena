# 변화를 찾아라 (Spot the Change)

> 1분 전과 지금이 다릅니다.
> 무엇이 바뀌었나요?
> 관찰력과 기억력이
> 동시에 필요합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-8명 |
| 시간 | 15-25분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 관찰력, 시각 기억, 변화 감지 |

## 게임 구성

```
장면 보드: 5×5 그리드에 물건 배치
물건 타일: 25종 × 2세트
변화 카드: 추가·제거·이동·교체 30장
모래시계: 30초 (관찰용)
```

## 기본 규칙

```
관찰 단계 (30초):
  장면 보드 전체 관찰
  메모 불가

가리기:
  스크린으로 보드 가리기
  변화 카드 1-3장 적용

발견 단계:
  스크린 제거
  변화된 것 찾아 말하기

득점:
  변화 1개 발견: +1점
  모든 변화 발견: 보너스 +2점
  오답 지적: -1점
```

## 변화 유형

```
추가: 없던 물건 등장
제거: 있던 물건 사라짐
이동: 위치 변경
교체: 비슷한 물건으로 교체
회전: 방향 변경
색상: 색깔 변경
```

## 난이도 조절

```
초급:
  변화 1개 / 관찰 60초
  큰 변화만

중급:
  변화 2-3개 / 관찰 30초
  작은 변화 포함

고급:
  변화 4-5개 / 관찰 15초
  매우 미세한 변화
```

## 테마 변형

```
일상 장면:
  부엌·교실·공원 테마

범죄 현장:
  탐정 역할로 증거 변화 찾기

역사 장면:
  역사적 배경 그림 활용
  → 역사 학습 연계
```

## 교육적 효과

```
관찰력:
  세밀한 차이 포착
  → 과학적 관찰 능력

시각 기억:
  장면 전체를 기억에 저장
  → 사진 기억력 훈련

변화 감지:
  이전과 현재 비교
  → 비판적 사고 기초
```
