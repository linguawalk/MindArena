# 기억 궁전 (Memory Palace)

> 고대 그리스인들이 사용한
> 최강의 기억술.
> 장소에 기억을 걸어두는 방법으로
> 놀라운 양의 정보를 기억합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 기억술, 연상 기억, 공간 활용 |

## 기억 궁전이란?

```
방법론:
  친숙한 장소(집·학교·길)를
  머릿속에 떠올리고
  기억할 내용을 각 장소에
  생생한 이미지로 배치

예시:
  기억할 것: 사과·자동차·음악·책·바다

  현관: 거대한 사과가 문을 막고 있다
  거실: 소파 위에 자동차가 올라가 있다
  주방: 냄비에서 음악이 흘러나온다
  방: 책이 침대를 덮고 있다
  화장실: 변기에서 바다가 넘친다
```

## 게임 구성

```
장소 보드: 방 5개짜리 집 그림 (개인용)
기억 카드: 단어·숫자·그림 100장
이미지 토큰: 기억 이미지 표시용
채점 시트
```

## 기본 규칙

```
준비 (5분):
  기억 카드 10장 공개
  각자 기억 궁전에 배치
  (어느 방에, 어떤 이미지로)

기억 (3분):
  조용히 기억 궁전 순회
  각 장소의 이미지 강화

회상:
  카드 모두 뒤집기
  기억 궁전 순회하며
  순서대로 회상

득점:
  순서 정확 기억: 2점/개
  순서 틀려도 기억: 1점/개
```

## 이미지 강화 팁

```
생생하게:
  흑백보다 컬러
  정적보다 동적
  보통보다 과장

감각 활용:
  시각뿐 아니라
  소리·냄새·촉감 상상

감정 연결:
  웃긴 / 무서운 / 신기한
  → 감정이 강할수록 기억 강함
```

## 심화 모드

```
긴 목록:
  20개 / 30개 / 50개 기억

숫자 기억:
  원주율 소수점 이하 기억
  (세계 기록: 7만 자리 이상!)

역순 회상:
  거꾸로 순서로 회상
```

## 교육적 효과

```
기억술 습득:
  실생활에서 바로 적용 가능
  → 시험·발표 준비에 활용

공간 기억:
  인간의 가장 강한 기억 시스템 활용
  → 해마 활성화

창의적 연상:
  단어→이미지 변환
  → 창의적 사고력
```
