# 문화 충돌 (Culture Clash)

> 같은 행동이 문화마다 다른 의미를 갖습니다.
> 선의로 한 행동이 오해를 만들기도 합니다.
> 문화 충돌의 순간을 체험하고
> 이해와 존중의 방법을 찾습니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 문화 감수성, 관점 전환, 오해 해결 |

## 게임 구성

```
문화 카드: 20장 (다양한 나라·문화권)
상황 카드: 40장 (문화 충돌 상황)
반응 카드: 각 문화당 3장 (문화별 해석)
해결 카드: 20장 (갈등 해결 방법)
```

## 기본 규칙

```
역할 배정:
  각자 문화 카드 1장 (특정 문화권 사람)
  문화 특성 읽기

상황 공개:
  문화 충돌 상황 제시

반응:
  자신의 문화 관점으로 해석
  "내 문화에서 이것은 ___를 의미합니다"

충돌 분석:
  왜 오해가 생겼는가?
  어느 쪽이 잘못인가?

해결:
  해결 카드 활용
  어떻게 화해할 수 있는가?
```

## 문화 카드 예시

```
한국 문화:
  어른께 두 손으로 드리기
  침묵은 불편함의 표시
  밥 먹었냐는 인사

일본 문화:
  명함은 두 손으로 정중히
  직접 거절보다 에둘러 표현
  집에서 신발 벗기

미국 문화:
  눈 맞춤 = 자신감·신뢰
  이름으로 부르기 선호
  개인 공간 중요

중동 문화:
  왼손 사용 금기
  남녀 악수 제한
  시간 개념이 다름
```

## 충돌 상황 카드 예시

```
상황 1:
  "한국인 직원이 일본인 상사에게
   선물을 한 손으로 건넸다"
  → 한국: 자연스러움 / 일본: 무례함

상황 2:
  "미국인이 한국 어른에게
   이름으로 불렀다"
  → 미국: 친근함 / 한국: 무례함

상황 3:
  "핀란드인과 한국인이 회의 중
   한국인이 계속 말했다"
  → 한국: 적극적 참여 / 핀란드: 상대 무시
```

## 문화 차원 이론

```
홉스테드의 문화 차원:
  권력 거리: 위계 수용도
  개인주의: 개인 vs 집단
  불확실성 회피: 변화 두려움
  장기 지향: 미래 vs 현재

예시:
  한국: 높은 권력 거리, 집단주의
  미국: 낮은 권력 거리, 개인주의
  일본: 높은 불확실성 회피
```

## 교육적 효과

```
문화 감수성:
  다름을 틀림으로 보지 않기
  → 글로벌 시민 의식

관점 전환:
  내 문화가 기준이 아님
  → 탈중심화 사고

갈등 해결:
  문화 차이로 인한 오해 해소
  → 다문화 소통 능력
```

## 게임 후 토론

```
"오늘 가장 놀라웠던 문화 차이는?
 왜 그런 차이가 생겼을까요?"

"문화 차이가 있을 때
 누가 맞추어야 할까요?"

"한국 문화 중 외국인이
 오해하기 쉬운 것은 무엇인가요?"
```
