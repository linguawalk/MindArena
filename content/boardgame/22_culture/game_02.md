# 고정관념 탐정 (Stereotype Detective)

> "아프리카는 모두 가난하다"
> "아시아인은 수학을 잘한다"
> 고정관념은 어디서 오는가?
> 어떻게 극복할 수 있는가?

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 고정관념 인식, 비판적 사고, 편견 극복 |

## 게임 구성

```
고정관념 카드: 40장 (다양한 집단에 관한)
반증 카드: 각 고정관념당 3장
출처 카드: 20장 (고정관념의 기원)
극복 전략 카드: 15장
```

## 기본 규칙

```
고정관념 카드 공개:
  특정 집단에 관한 고정관념 제시
  예) "노인은 기술을 잘 못 다룬다"

분석 단계:
  ①이 고정관념이 얼마나 퍼져있는가?
  ②어디서 왔는가? (출처 카드)
  ③얼마나 사실인가?
  ④어떤 해를 끼치는가?

반증:
  반증 카드로 예외 사례 제시

극복 전략:
  이 고정관념을 어떻게 극복할 수 있는가?
```

## 고정관념 카드 예시

```
성별:
  "여자는 수학을 못한다"
  "남자는 감정을 표현 안 한다"

나이:
  "젊은 사람은 책임감이 없다"
  "노인은 변화를 싫어한다"

국적:
  "한국 사람은 열심히 일한다"
  "프랑스 사람은 낭만적이다"

직업:
  "예술가는 가난하다"
  "의사는 부자다"
```

## 고정관념의 기원 분석

```
미디어:
  영화·드라마·광고의 반복 이미지

역사적 경험:
  과거 사건의 일반화

인지 효율:
  빠른 판단을 위한 단순화

내집단 편향:
  우리 집단 vs 저들 집단
```

## 고정관념 vs 문화적 경향

```
고정관념: 모든 개인에게 적용
문화적 경향: 집단 평균의 차이

예시:
  "한국인은 모두 집단주의다" (고정관념)
  "한국 문화는 개인보다 집단 가치를
   상대적으로 더 강조하는 경향이 있다"
  (문화적 경향 - 개인차 존재)
```

## 교육적 효과

```
고정관념 인식:
  내 안의 편견 발견
  → 자기 인식

비판적 수용:
  일반화의 위험성 이해
  → 미디어 리터러시

다양성 존중:
  개인을 집단으로 환원하지 않기
  → 인권 감수성
```

## 게임 후 토론

```
"자신이 속한 집단에 대한
 고정관념 중 가장 불편한 것은?"

"고정관념이 완전히 없는 세상은
 가능할까요?"

"고정관념과 통계적 사실은
 어떻게 다른가요?"
```
