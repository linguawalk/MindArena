# 세계 관습 퀴즈 (World Customs Quiz)

> 세계는 넓고 관습은 다양합니다.
> 어떤 나라에서는 당연한 것이
> 다른 나라에서는 금기입니다.
> 세계 문화 상식을 넓힙니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-8명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 세계 문화 지식, 문화 다양성 인식 |

## 게임 구성

```
관습 카드: 80장 (세계 각국 관습)
  앞면: 관습 설명
  뒷면: 나라 + 배경 설명
국가 카드: 30장
맞추기 보드
점수판
```

## 기본 규칙

```
모드 1 - 나라 맞추기:
  관습 설명 읽기
  어느 나라 관습인지 맞히기

모드 2 - 의미 맞추기:
  특정 관습 + 나라 제시
  그 관습의 의미·유래 설명

모드 3 - OX 퀴즈:
  "이 나라에서 이것은 예의 바른 행동이다"
  맞으면 O, 틀리면 X
```

## 관습 카드 예시

```
식사:
  "식사 중 트림하는 것이 맛있다는 표현"
  → 중국·중동 일부

  "식사 후 접시를 깨끗이 비우면 실례"
  → 중국 (더 먹고 싶다는 의미가 없어짐)

  "밥그릇을 들고 먹는다"
  → 일본 (한국은 반대)

인사:
  "코를 맞대고 인사"
  → 마오리족 (뉴질랜드), 이누이트족

  "볼에 키스로 인사"
  → 프랑스·스페인·이탈리아

  "악수할 때 왼손을 오른쪽 팔꿈치에"
  → 아프리카 일부 지역

색깔:
  "결혼식에 흰 드레스"
  → 서양 (동양은 전통적으로 붉은색)

  "장례식에 흰 옷"
  → 한국·중국 전통

숫자:
  "4층이 없는 건물"
  → 한국·중국·일본 (4 = 死)

  "13층이 없는 건물"
  → 미국·유럽 (13 = 불길)
```

## 놀라운 관습 TOP 10

```
핀란드: 사우나에서 비즈니스 회의
일본: 면을 소리내어 먹기 (칭찬)
인도: 식사 후 손 핥기 (예의)
불가리아: 고개 끄덕임 = 아니요
태국: 머리는 신성해서 만지면 안 됨
그리스: OK 사인 = 모욕
브라질: 엄지척 = 매우 부정적
이란: 엄지척 = 심한 욕설
이누이트: 코 문지르기 = 인사
사우디: 커피잔 흔들기 = 그만 주세요
```

## 교육적 효과

```
세계 지식:
  각국 문화·관습 습득
  → 글로벌 교양

문화 상대주의:
  옳고 그름이 아닌 다름
  → 가치 판단 유보

여행 준비:
  실제 여행 시 문화 실수 예방
  → 실용적 국제 감각
```
