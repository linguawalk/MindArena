# 이민자의 하루 (Immigrant's Day)

> 새로운 나라에서 하루를 보냅니다.
> 언어도 모르고 문화도 낯섭니다.
> 이민자·난민·외국인 노동자의
> 경험을 직접 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 공감 능력, 사회적 포용, 이민자 이해 |

## 게임 구성

```
역할 카드: 이민자 4-6종 (배경 스토리)
과제 카드: 40장 (일상 과제)
언어 장벽 카드: 20장 (소통 어려움)
도움 카드: 10장 (지역 주민 도움)
장벽 토큰: 30개
```

## 역할 카드 예시

```
카드 1: 베트남 요리사 (35세)
  한국어: 기초 수준
  한국 거주: 6개월
  목표: 식당 취업
  어려움: 서류 작업, 한국 관습

카드 2: 시리아 난민 (28세)
  한국어: 없음
  한국 거주: 1개월
  목표: 자녀 학교 등록
  어려움: 언어, 차별, 행정

카드 3: 한국계 중국인 (45세)
  한국어: 유창하지만 사투리 억양
  한국 거주: 2년
  목표: 운전면허 취득
  어려움: '완전한 한국인' 아님의 인식
```

## 기본 규칙

```
역할 배정:
  각자 이민자 역할 카드 받기
  배경 스토리 읽기

과제 수행:
  일상 과제 카드 뽑기
  예) "은행 계좌 개설"
      "자녀 학교 상담"
      "병원 진료 예약"

장벽 처리:
  언어 장벽 카드 발생
  어떻게 해결할 것인가?

도움 결정:
  도움 카드 활용 여부
  도움 받는 것이 쉬운가?

성찰:
  과제 완수 후 감정 공유
```

## 과제 카드 예시

```
쉬운 과제:
  "편의점에서 물건 구매"
  "버스 타기"

중간 과제:
  "휴대폰 개통"
  "집 계약"

어려운 과제:
  "경찰서에서 도움 요청"
  "직장 면접"
  "차별 경험 후 신고"
```

## 언어 장벽 카드

```
"직원이 외국인이라는 것을 알고
 말을 짧고 크게 한다"

"서류가 한국어로만 되어 있다"

"다른 고객들이 쳐다본다"

"직원이 다른 한국인 직원을 부른다"
```

## 교육적 효과

```
공감 체험:
  이민자 어려움 직접 경험
  → 사회적 공감

제도 이해:
  이민자를 위한 사회 지원 필요성
  → 사회 정책 관심

포용성:
  나는 이민자에게 어떻게 대했나?
  → 시민 의식
```

## 게임 후 토론

```
"가장 힘들었던 과제는?
 실제 이민자는 얼마나 더 힘들까요?"

"한국 사회는 이민자에게
 얼마나 친화적인가요?"

"이민자를 도울 수 있는
 가장 쉬운 방법은?"
```
