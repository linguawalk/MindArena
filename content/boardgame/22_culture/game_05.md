# 문화 번역가 (Cultural Translator)

> 번역은 단어가 아닌 문화를 옮기는 것입니다.
> 직역하면 틀리는 표현들.
> 문화를 이해해야 진정한 소통이 가능합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 문화 간 소통, 언어와 문화의 관계, 번역 사고 |

## 게임 구성

```
표현 카드: 60장 (번역하기 어려운 표현)
문화 배경 카드: 각 표현당 1장
번역 시도 용지: 개인용
평가 기준: 정확성·자연스러움·문화 이해
```

## 번역 불가능한 표현들

```
한국어:
  눈치: 상황 파악 능력
  정 (情): 오랜 애정·유대감
  한 (恨): 억울함·슬픔·체념의 복합 감정
  빨리빨리: 빠른 속도의 문화적 압박

일본어:
  木漏れ日 (Komorebi): 나뭇잎 사이로 비치는 햇살
  侘寂 (Wabi-sabi): 불완전함의 아름다움
  木枯らし (Kogarashi): 초겨울 찬바람

포르투갈어:
  Saudade: 그리움·향수·있지 않는 것에 대한 갈망

독일어:
  Schadenfreude: 남의 불행에서 느끼는 즐거움
  Waldeinsamkeit: 숲속 고독의 평화로움

덴마크어:
  Hygge: 아늑함·편안함·행복한 순간
```

## 기본 규칙

```
표현 카드 공개:
  번역하기 어려운 표현 제시

번역 시도 (3분):
  해당 개념을 다른 언어로 표현
  단어·문장·비유 모두 가능

발표:
  각자 번역 발표

평가:
  얼마나 정확히 전달했는가?
  문화적 맥락을 담았는가?

문화 배경 카드:
  실제 배경 공개 후 비교
```

## 번역 과제 예시

```
과제 1:
  "눈치"를 영어로 설명하세요
  → "Reading the room" (부분적)
  → 완전한 번역은 불가능

과제 2:
  "Hygge"를 한국어로 설명하세요
  → 편안함? 아늑함? 행복?
  → 덴마크 문화 없이 완전히 이해 불가

과제 3:
  "恨(한)"을 일본어로 설명하세요
  → 역사·문화적 맥락 설명 필요
```

## 언어와 사고의 관계

```
사피어-워프 가설:
  "언어가 사고를 형성한다"

예시:
  눈 관련 단어:
    영어: snow 1개
    이누이트어: 50개 이상
  → 더 세밀하게 인식

색깔 인식:
  러시아어: 밝은 파랑·어두운 파랑 별개 단어
  → 색 구별 능력 더 발달
```

## 교육적 효과

```
언어 감수성:
  언어 속에 담긴 문화
  → 언어 학습의 깊이

문화 번역:
  직역 아닌 의역의 중요성
  → 국제 소통 능력

창의적 표현:
  없는 개념 새로 표현하기
  → 언어 창조 능력
```

## 게임 후 토론

```
"번역할 수 없는 한국어 표현이
 더 있다면 무엇인가요?"

"언어가 없으면 개념도 없는 것인가요?
 아니면 개념이 먼저이고 언어가 따르는가요?"

"완벽한 번역은 가능할까요?"
```
