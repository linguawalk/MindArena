# 위기 탈출 (Crisis Escape)

> 팀 전원이 협력해서
> 연쇄적으로 발생하는 위기를 해결하고
> 제한된 시간 안에 탈출하는
> 순수 협력 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-5명 |
| 소요 시간 | 25-35분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 협력, 역할 분담, 우선순위 결정, 위기 관리 |

---

## 게임 목표

팀 전원이 협력해서 **10라운드 이내에**
모든 위기를 해결하고 탈출 포인트에 도달하면
팀 전체가 승리합니다.

한 명이라도 탈출에 실패하거나
해결되지 않은 위기가 임계치를 초과하면
팀 전체가 패배합니다.

---

## 준비물

- 탈출 맵 보드 (5×5 격자)
- 위기 카드 덱 (60장)
  - 긴급 위기 (빨강): 20장
  - 일반 위기 (노랑): 25장
  - 예비 위기 (초록): 15장
- 역할 카드 (8가지)
- 자원 토큰 (체력·도구·정보 각 20개)
- 위기 마커
- 탈출 타이머 트랙 (10칸)
- 행동 토큰 (플레이어당 3개)
- 팀 사기 트랙

### 탈출 맵 보드

```
   A    B    C    D    E
1 [시작][  ] [  ] [  ] [  ]
2 [  ] [  ] [위기][  ] [  ]
3 [  ] [위기][  ] [위기][  ]
4 [  ] [  ] [자원][  ] [  ]
5 [  ] [  ] [  ] [  ][탈출]

이동: 상하좌우 1칸씩
탈출 포인트: E5
```

---

## 역할 카드

```
지휘관 (Commander):
  특수 능력: 매 라운드 팀원 1명에게
             행동 토큰 1개 추가 부여
  약점: 자신의 행동 토큰 2개 (다른 역할보다 1개 적음)
  역할: 전략적 방향 결정, 자원 배분 조율

엔지니어 (Engineer):
  특수 능력: 도구 자원 사용 시 효과 2배
  약점: 정보 자원 효과 절반
  역할: 기계·구조적 위기 해결 전문

의료관 (Medic):
  특수 능력: 팀원 체력 회복 (매 턴 1명 +2 체력)
  약점: 전투·물리적 위기 해결 능력 없음
  역할: 팀원 생존 지원

정보관 (Intelligence):
  특수 능력: 위기 카드를 미리 1장 확인 가능
  약점: 체력 자원 효과 절반
  역할: 적 정보 수집, 위기 예측

탐험가 (Explorer):
  특수 능력: 이동 시 2칸 이동 가능
  약점: 위기 해결 시 자원 1개 추가 소모
  역할: 빠른 이동, 자원 수집

과학자 (Scientist):
  특수 능력: 정보 자원으로 위기 난이도 1단계 낮춤
  약점: 이동 속도 절반 (2라운드에 1칸)
  역할: 복잡한 위기 분석·해결

외교관 (Diplomat):
  특수 능력: 팀원 간 자원 이전 비용 없음
  약점: 긴급 위기 해결 불가
  역할: 팀 자원 배분, 협상

전략가 (Strategist):
  특수 능력: 위기 카드 발생 전 1회 취소 가능
              (게임당 2번)
  약점: 자원 보유량 최대 3개
  역할: 장기 계획 수립
```

## 게임 준비

### 1. 역할 배분

플레이어 수에 따라 역할 카드를 배분합니다.

```
2인: 지휘관 + 엔지니어 (또는 자유 선택)
3인: 지휘관 + 엔지니어 + 의료관
4인: 위 3개 + 정보관
5인: 위 4개 + 탐험가
```

### 2. 시작 자원 배분

```
체력 자원: 플레이어당 3개
도구 자원: 팀 공유 풀에 5개
정보 자원: 팀 공유 풀에 3개
```

### 3. 초기 위기 배치

맵 위에 위기 마커를 3개 배치합니다.
```
초기 위기 위치:
  B2: 일반 위기
  C3: 긴급 위기
  D3: 일반 위기
```

### 4. 팀 사기 트랙

팀 사기 트랙을 최대값(10)에 설정합니다.
사기가 0이 되면 팀 즉시 패배합니다.

---

## 게임 규칙

### 라운드 구조

각 라운드는 3단계로 진행됩니다.

```
1단계: 위기 확산 (위기 카드 1장 공개)
2단계: 팀 토론 (2분)
3단계: 행동 실행 (플레이어 순서대로)
```

### 1단계: 위기 확산

위기 카드 덱에서 1장을 공개합니다.

```
위기 카드 예시:

긴급 위기 (빨강):
  "화재 발생!"
  위치: C3
  해결 조건: 도구 자원 3개 사용
  미해결 시: 인접 칸으로 확산
             팀 사기 -2

일반 위기 (노랑):
  "통신 두절"
  위치: B4
  해결 조건: 정보 자원 2개 사용
  미해결 시: 팀 사기 -1
             다음 라운드 행동 토큰 1개 감소

예비 위기 (초록):
  "부상 발생"
  위치: 특정 플레이어
  해결 조건: 체력 자원 2개 사용
  미해결 시: 해당 플레이어 행동 토큰 1개 감소
```

### 2단계: 팀 토론

2분 동안 팀원들이 자유롭게 토론합니다.

```
토론 내용:
  - 어떤 위기를 먼저 해결할 것인가?
  - 누가 어디로 이동할 것인가?
  - 자원을 어떻게 배분할 것인가?
  - 탈출 경로는?
```

### 3단계: 행동 실행

각 플레이어가 행동 토큰을 사용해서 행동합니다.

**가능한 행동:**

```
①이동 (토큰 1개):
  인접 칸으로 1칸 이동
  (탐험가는 2칸)

②위기 해결 시도 (토큰 1개):
  현재 위치의 위기에 자원 투입
  조건 충족 시 위기 해결

③자원 수집 (토큰 1개):
  자원 위치에서 자원 토큰 2개 획득

④자원 전달 (토큰 1개):
  같은 칸의 팀원에게 자원 1-3개 전달
  (외교관은 토큰 소모 없음)

⑤팀원 지원 (토큰 1개):
  같은 칸의 팀원에게 행동 토큰 1개 전달

⑥탈출 준비 (토큰 2개):
  탈출 포인트(E5)에서 사용
  탈출 카운터 +1
  (모든 플레이어의 탈출 카운터가 1이 되면 탈출 성공)
```

## 위기 해결 시스템

### 위기 해결 절차

**①위기 위치 이동:**
해결하려는 위기가 있는 칸으로 이동합니다.

**②자원 투입:**
필요한 자원을 투입합니다.
한 번의 행동으로 부분 투입도 가능합니다.

**③해결 판정:**
필요 자원이 모두 투입되면 위기가 해결됩니다.
위기 마커를 제거합니다.

```
부분 해결 예시:
  화재 위기: 도구 자원 3개 필요
  이번 턴: 도구 자원 2개 투입
  → 위기 마커에 부분 투입 표시
  → 다음 턴: 도구 자원 1개 더 투입 → 해결
```

### 위기 확산

미해결 위기는 매 라운드 확산됩니다.

```
긴급 위기 확산:
  미해결 1라운드: 사기 -2, 인접 위기 추가
  미해결 2라운드: 사기 -3, 해당 구역 봉쇄
  미해결 3라운드: 팀 즉시 패배

일반 위기 확산:
  미해결 2라운드: 사기 -1
  미해결 4라운드: 긴급 위기로 전환

예비 위기 확산:
  미해결 3라운드: 일반 위기로 전환
```

## 탈출 조건

**탈출 포인트(E5)에 도달 후:**
모든 위기가 해결된 상태에서
탈출 준비 행동을 완료해야 합니다.

```
탈출 진행:
  라운드 8: 탈출 준비 시작 가능
  라운드 9: 모든 플레이어 탈출 포인트 도달 필요
  라운드 10: 최종 탈출 (마지막 기회)
```

---

## 패배 조건

다음 중 하나가 발생하면 즉시 팀 패배입니다.

```
①팀 사기가 0이 됨
②한 명의 플레이어가 라운드 11에 탈출 못함
③긴급 위기가 3라운드 이상 미해결
④위기 마커가 맵에 7개 이상
```

## 난이도 조정

```
쉬움:
  시작 자원 +2개씩
  위기 확산 속도 절반
  라운드 12까지 연장

보통:
  기본 규칙 그대로

어려움:
  시작 자원 -1개씩
  매 라운드 위기 카드 2장
  라운드 8까지 단축

전문가:
  어려움 + 통신 금지
  (토론 단계 없음, 손짓으로만 소통)
```

## 시나리오

### 시나리오 #01: 실험실 탈출
```
배경: 화학 실험실에서 위험 물질이 유출됐습니다.
특수 규칙:
  - 모든 이동에 방호복 자원 1개 필요
  - 과학자 역할이 없으면 위기 해결 난이도 +1
  - 탈출 포인트에 비상구 코드 입력 필요 (정보 자원 5개)
```

### 시나리오 #02: 우주 정거장
```
배경: 우주 정거장에서 산소 공급 시스템이 고장났습니다.
특수 규칙:
  - 매 라운드 팀 전체 체력 -1 (산소 부족)
  - 산소 자원을 수집해서 체력 보충 가능
  - 무중력 구역: 이동 비용 2배
```

### 시나리오 #03: 도시 재난
```
배경: 대규모 자연재해가 도시를 강타했습니다.
특수 규칙:
  - 맵 크기 7×7로 확장
  - 구조 대상 NPC 5명 추가 (구조하면 사기 +1)
  - 여진 이벤트: 매 3라운드 맵 일부 봉쇄
```

## 전략 가이드

### 소통 원칙

**1. 정보 공유 우선**
자신의 자원 상태, 위치, 계획을 항상 공유하세요.
숨기는 정보가 없어야 팀이 최적 결정을 합니다.

**2. 역할 활용**
각 역할의 특수 능력을 최대한 활용하세요.
엔지니어가 있으면 도구 위기를 우선 배정합니다.

**3. 우선순위 결정**
긴급 위기 > 일반 위기 > 예비 위기 순으로 처리하세요.
단, 탈출 경로의 위기를 먼저 처리하는 것도 중요합니다.

### 자원 관리

**1. 자원 분산 금지**
한 위기에 집중적으로 자원을 투입하세요.
여러 위기에 조금씩 투입하면 모두 해결 못할 수 있습니다.

**2. 비상 자원 확보**
항상 팀 공유 풀에 자원 2-3개를 비축하세요.
갑작스러운 긴급 위기에 대비합니다.

**3. 외교관 활용**
외교관이 있으면 자원 이전 비용 없이
필요한 플레이어에게 자원을 집중시킬 수 있습니다.

## 교육적 활용

### 협력 교육 연계
- **역할 분담**: 각자의 강점 활용
- **의사소통**: 제한된 시간 내 효율적 소통
- **우선순위**: 긴급성과 중요성 판단

### 사고력 연계
- **위기 관리**: 복수 위기 동시 대응
- **자원 최적화**: 제한된 자원의 효율적 사용
- **계획 수립**: 단기·장기 목표 균형

### 수업 활용 방법
1. 역할 설명 + 규칙 숙지 (10분)
2. 쉬운 난이도 1판 (25분)
3. "더 잘할 수 있었던 순간" 복기 (10분)
4. 어려운 난이도 도전 (30분)
