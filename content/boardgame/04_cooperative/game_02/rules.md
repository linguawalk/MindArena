# 지식 원정대 (Knowledge Expedition)

> 팀원들이 각자의 전문 지식을 활용해서
> 미지의 세계를 탐험하고
> 잃어버린 지식의 조각을 모아
> 위대한 발견을 완성하는 협력 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-6명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 지식 협력, 역할 전문화, 집단 지성, 탐구 능력 |

---

## 게임 목표

팀원들이 협력해서 **지식 조각 12개**를 모두 수집하고
**발견 보드**를 완성하면 팀 전체가 승리합니다.

제한 라운드(8라운드) 내에 완성하지 못하거나
팀원이 탐험 중 실종되면 팀 전체가 패배합니다.

---

## 준비물

- 탐험 지도 보드 (6×6 격자, 타일로 구성)
- 지도 타일 36장 (탐험하며 공개)
- 지식 조각 카드 12장 (목표)
- 전문가 역할 카드 6장
- 장애물 카드 20장
- 발견 보드 1개
- 탐험 마커 (플레이어별)
- 지식 토큰 (30개)
- 위험 주사위
- 팀 결속력 트랙

### 탐험 지도 타일 종류

```
지식 유적지 (6장):
  지식 조각이 숨겨진 장소
  도착 시 지식 문제 도전 가능

자원 지대 (8장):
  지식 토큰 2-4개 획득

위험 지대 (10장):
  위험 주사위 굴려서 장애물 등장

안전 지대 (8장):
  특별한 이벤트 없음
  휴식 가능

비밀 통로 (4장):
  지도 내 다른 비밀 통로로 순간이동
```

## 전문가 역할 카드

```
고고학자 (Archaeologist):
  특수 능력: 지식 유적지 도착 시
             문제 난이도 1단계 감소
  전문 분야: 역사·문화
  행동력: 3

언어학자 (Linguist):
  특수 능력: 암호·언어 장애물 자동 해제
  전문 분야: 언어·문학
  행동력: 3

수학자 (Mathematician):
  특수 능력: 수학·논리 문제 30초 추가 시간
  전문 분야: 수학·논리
  행동력: 2

생물학자 (Biologist):
  특수 능력: 자연 지대에서 지식 토큰 2배 획득
  전문 분야: 자연·생물
  행동력: 3

물리학자 (Physicist):
  특수 능력: 위험 지대 피해 50% 감소
  전문 분야: 과학·기술
  행동력: 3

역사학자 (Historian):
  특수 능력: 지도 타일을 뒤집기 전 미리 확인 가능
  전문 분야: 역사·사회
  행동력: 4
```

## 게임 준비

### 1. 탐험 지도 설정

지도 타일을 섞어서 6×6 격자에 뒤집어 놓습니다.
단, 시작 타일(A1)은 안전 지대로 고정합니다.

```
초기 지도 (? = 미탐험):
   A    B    C    D    E    F
1 [시작][?]  [?]  [?]  [?]  [?]
2 [?]  [?]  [?]  [?]  [?]  [?]
3 [?]  [?]  [?]  [?]  [?]  [?]
4 [?]  [?]  [?]  [?]  [?]  [?]
5 [?]  [?]  [?]  [?]  [?]  [?]
6 [?]  [?]  [?]  [?]  [?]  [?]
```

### 2. 지식 조각 배치

지식 조각 카드 12장을 무작위로 6개의
지식 유적지 타일 아래에 2장씩 숨겨놓습니다.

### 3. 역할 배분

플레이어 수에 맞게 역할 카드를 배분합니다.

```
3인: 고고학자 + 수학자 + 생물학자
4인: 위 3개 + 언어학자
5인: 위 4개 + 물리학자
6인: 모든 역할
```

### 4. 시작 자원

팀 공유 지식 토큰: 10개
각 플레이어 개인 토큰: 없음

## 게임 규칙

### 라운드 구조

```
1단계: 이벤트 카드 공개
2단계: 팀 토론 (2분)
3단계: 탐험 행동 (플레이어 순서대로)
4단계: 라운드 종료 처리
```

### 1단계: 이벤트 카드

매 라운드 시작 시 이벤트 카드 1장을 공개합니다.

```
이벤트 카드 예시:

폭풍 경보:
  이번 라운드 모든 이동 행동력 +1 소모

지식의 빛:
  다음 지식 문제 정답 시 지식 토큰 +3

신비로운 안내자:
  팀원 1명 즉시 원하는 위치로 이동 가능

시간의 압박:
  이번 라운드 문제 시간 절반

발견의 기쁨:
  지식 조각 획득 시 팀 결속력 +2
```

### 2단계: 팀 토론

팀원들이 이번 라운드의 전략을 토론합니다.

```
토론 내용:
  - 누가 어느 유적지를 탐험할 것인가?
  - 위험 지대를 어떻게 처리할 것인가?
  - 자원이 부족한 팀원을 어떻게 지원할 것인가?
  - 지식 조각 수집 우선순위는?
```

### 3단계: 탐험 행동

각 플레이어가 행동력을 사용해서 행동합니다.

**①이동 (행동력 1):**
인접한 칸으로 이동합니다.
미탐험 칸으로 이동 시 타일을 공개합니다.

**②지식 문제 도전 (행동력 1):**
지식 유적지에서 지식 조각을 획득하기 위해
지식 문제에 도전합니다.

```
지식 문제 도전 절차:
  ①문제 카드 공개
  ②도전 팀원 결정 (여러 명 가능)
  ③45초 내 답변
  ④정답 시: 지식 조각 획득
   오답 시: 지식 토큰 2개 소모 후 재도전 가능
             (재도전 1회만 허용)
```

**③자원 수집 (행동력 1):**
자원 지대에서 지식 토큰을 획득합니다.

**④팀원 지원 (행동력 1):**
같은 칸의 팀원에게 지식 토큰 1-3개 전달합니다.

**⑤장애물 해제 (행동력 1):**
위험 지대의 장애물을 해제합니다.
지식 토큰 소모 또는 전문 능력 활용.

**⑥발견 보드 기록 (행동력 2):**
수집한 지식 조각을 발견 보드에 배치합니다.
시작 칸(A1)에서만 사용 가능.

## 지식 조각과 발견 보드

### 지식 조각 카드 종류

```
과학 조각 (2장):
  조각 A: "물질의 기본 단위 = 원자"
  조각 B: "에너지 보존 법칙"

역사 조각 (2장):
  조각 C: "문명의 발생 조건"
  조각 D: "혁명의 공통 원인"

수학 조각 (2장):
  조각 E: "무한의 개념"
  조각 F: "확률의 원리"

언어 조각 (2장):
  조각 G: "언어의 기원"
  조각 H: "의미와 기호의 관계"

자연 조각 (2장):
  조각 I: "생태계 균형의 원리"
  조각 J: "진화의 메커니즘"

예술 조각 (2장):
  조각 K: "아름다움의 보편성"
  조각 L: "창의성의 본질"
```

### 발견 보드

```
┌─────────────────────────────┐
│         위대한 발견          │
│    "지식의 통합 이론"        │
├──────┬──────┬──────┬────────┤
│ 과학  │ 역사  │ 수학  │ 언어  │
│[A][B]│[C][D]│[E][F]│[G][H] │
├──────┴──────┼──────┴────────┤
│    자연      │     예술      │
│  [I]  [J]  │  [K]   [L]   │
└─────────────┴───────────────┘

모든 칸 채우기 = 팀 승리!
```

## 위험 지대 처리

위험 지대 타일에 도착하면
위험 주사위를 굴려서 장애물을 결정합니다.

```
위험 주사위 결과:

1: 지식 수수께끼
   특정 분야 문제 즉시 도전
   실패 시: 해당 칸 통과 불가 (다음 턴 재시도)

2: 함정
   지식 토큰 3개 소모 (없으면 팀 결속력 -1)

3: 미로
   다음 이동 시 추가 행동력 2 필요

4: 경쟁자 등장
   다음 라운드 지식 조각 획득 불가
   (한 라운드 동안)

5: 자연재해
   팀 전체 이번 라운드 행동력 -1

6: 행운!
   해당 칸 위험 무효화
   지식 토큰 2개 획득
```

### 장애물 해제 방법

```
지식 수수께끼: 해당 분야 전문가가 해제 가능
함정: 지식 토큰 3개 소모
미로: 수학자가 자동 해제 / 다른 역할은 토큰 2개
경쟁자: 외교적 해결 (팀 토론 후 결속력 1 사용)
자연재해: 생물학자 자동 해제 / 다른 역할은 토큰 2개
```

## 팀 결속력 트랙

```
팀 결속력 트랙 (0-10):
  시작: 5
  최대: 10
  최소: 0 → 팀 즉시 패배

결속력 증가:
  지식 조각 획득: +1
  발견 이벤트 카드: +2
  장애물 협력 해제: +1

결속력 감소:
  팀원 실종: -3
  긴급 실패: -1
  함정 자원 부족: -1

결속력 효과:
  결속력 8+: 모든 문제 시간 +15초
  결속력 6-7: 기본
  결속력 4-5: 자원 수집 -1
  결속력 1-3: 행동력 -1
```

## 지식 문제 예시

### 고고학자 전문 문제 (역사·문화)

```
유적지 #01 — "문명의 요람":
  문제: 세계 4대 문명 발생지를 모두 말하세요.
  정답: 메소포타미아·이집트·인더스·황하 문명
  난이도: ⭐⭐

유적지 #02 — "제국의 흔적":
  문제: 로마 제국이 서유럽 문화에 미친
        영구적 영향 3가지는?
  정답: 법률 체계 / 라틴어 계열 언어 /
        기독교 문화 (외 다수)
  난이도: ⭐⭐⭐
```

### 수학자 전문 문제 (수학·논리)

```
유적지 #03 — "수의 신비":
  문제: 소수의 정의와 소수를 찾는
        에라토스테네스의 체를 설명하세요.
  정답: 1과 자신만으로 나누어지는 수 /
        배수를 순차적으로 제거하는 방법
  난이도: ⭐⭐⭐

유적지 #04 — "무한의 탐구":
  문제: 힐베르트 호텔 역설을 설명하고
        무한의 특성을 논하세요.
  정답: 무한히 많은 방이 있어도 새 손님 수용 가능
        = 무한 + 1 = 무한
  난이도: ⭐⭐⭐⭐
```

### 생물학자 전문 문제 (자연·생물)

```
유적지 #05 — "생명의 기원":
  문제: 지구 최초 생명체의 탄생 조건과
        원시 바다의 역할을 설명하세요.
  정답: 원시 대기 + 에너지(번개·자외선) →
        유기물 합성 → 원시 바다에서 세포 형성
  난이도: ⭐⭐⭐

유적지 #06 — "진화의 증거":
  문제: 자연선택설의 4가지 핵심 원리를 설명하세요.
  정답: 변이 / 유전 / 과잉 생산 /
        환경 적합성에 따른 선택
  난이도: ⭐⭐⭐⭐
```

## 변형 규칙

### 조용한 원정대
- 토론 단계 없음
- 제스처와 카드로만 소통
- 비언어 소통 능력 훈련

### 전문가 원정대
- 각 유적지가 특정 전문가만 입장 가능
- 팀 구성 전략이 더 중요
- 역할 배분의 중요성 강조

### 경쟁 원정대
- 팀 vs 팀 방식
- 먼저 발견 보드를 완성한 팀이 승리
- 협력과 경쟁의 균형

### 초급 원정대
- 지식 조각 8개로 축소
- 위험 지대 없음
- 소요 시간: 20분

## 교육적 활용

### 지식 교육 연계
- **다분야 통합**: 여러 분야 지식 동시 활용
- **전문화**: 각 분야의 깊이 있는 탐구
- **발견 학습**: 지식 조각을 모아 큰 그림 완성

### 협력 교육 연계
- **집단 지성**: 팀원의 지식 결합
- **역할 인식**: 자신과 타인의 강점 파악
- **협력적 문제 해결**: 복합 과제 공동 해결

### 수업 활용 방법
1. 학기 초반 친목 도모 게임으로 활용 (35분)
2. 특정 교과 지식 복습 (문제 카드 교체) (30분)
3. 팀 빌딩 활동 (40분)
4. 학생이 유적지 문제 직접 출제 (25분)
