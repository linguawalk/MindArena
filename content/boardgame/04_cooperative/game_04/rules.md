# 타임머신 수리단 (Time Machine Repair)

> 고장난 타임머신을 수리하기 위해
> 팀원들이 각자 다른 시대로 파견되어
> 필요한 부품과 지식을 수집해서
> 돌아오는 협력 게임입니다.
> 역사 지식과 과학 원리가 핵심입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-5명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐⭐ (상) |
| 교육 분야 | 역사 지식, 과학 원리, 협력, 시간 관리 |

---

## 게임 목표

팀원들이 각자 다른 시대로 파견되어
**수리 부품 8개**와 **시대 지식 토큰 12개**를 수집하고
**기준점(현재)으로 모두 귀환**하면 팀이 승리합니다.

제한 라운드(10라운드) 내에 귀환하지 못하거나
타임머신 고장 지수가 최대치를 초과하면 팀 패배입니다.

---

## 준비물

- 타임라인 보드 (6개 시대 구역)
- 시대 카드 (각 시대별 사건·인물·발명 카드)
- 수리 부품 카드 8장
- 시대 지식 토큰 (30개)
- 역할 카드 5장
- 시간 이동 비용 카드
- 타임머신 고장 트랙
- 역사 지식 문제 카드 60장
- 과학 원리 문제 카드 30장
- 행동 토큰 (플레이어당 3개)
- 귀환 카운터

### 타임라인 보드 구조

```
┌──────────┬──────────┬──────────┐
│  고대     │  중세     │  근세     │
│ BC~AD500 │500~1500  │1500~1800 │
│  지수:[8] │  지수:[6] │  지수:[5] │
├──────────┼──────────┼──────────┤
│  근대     │  현대     │  미래     │
│1800~1900 │1900~2000 │2000~2100 │
│  지수:[4] │  지수:[7] │  지수:[3] │
└──────────┴──────────┴──────────┘

현재(기준점): 현대 구역 중앙
각 시대 지수: 해당 시대의 위험도 (높을수록 위험)
```

## 역할 카드

```
타임머신 엔지니어 (Time Engineer):
  특수 능력: 수리 부품 획득 비용 절반
             타임머신 고장 트랙 -1 (매 라운드)
  전문 시대: 근대 (산업혁명 시대)
  행동력: 3
  약점: 역사 지식 문제 시간 -15초

역사 전문가 (History Expert):
  특수 능력: 모든 역사 지식 문제 시간 +20초
             시대 지식 토큰 획득 2배
  전문 시대: 고대·중세
  행동력: 3
  약점: 과학 원리 문제 난이도 +1

과학 탐구자 (Science Explorer):
  특수 능력: 과학 원리 문제 자동 정답 (1회/라운드)
             미래 시대 위험도 무시
  전문 시대: 현대·미래
  행동력: 3
  약점: 시간 이동 비용 +1

시간 항법사 (Time Navigator):
  특수 능력: 시간 이동 비용 절반
             2개 시대 동시 방문 가능
  전문 시대: 모든 시대
  행동력: 4
  약점: 수리 부품 획득 불가 (다른 역할에 전달만)

문화 해석가 (Culture Interpreter):
  특수 능력: 모든 시대 문화 적응 자동
             팀원에게 힌트 제공 무제한
  전문 시대: 근세·근대
  행동력: 3
  약점: 단독 행동 불가 (항상 팀원과 함께)
```

## 게임 준비

### 1. 시대 카드 배치

각 시대 구역에 해당 시대의 사건·인물·발명 카드를
섞어서 뒤집어 놓습니다.

```
고대 시대 카드 예시:
  이집트 피라미드 건설 / 알렉산더 대왕 /
  아르키메데스 / 실크로드 개설 / ...

중세 시대 카드 예시:
  흑사병 / 십자군 원정 / 마그나 카르타 /
  화약 발명 / 나침반 전래 / ...

근세 시대 카드 예시:
  코페르니쿠스 지동설 / 구텐베르크 인쇄술 /
  콜럼버스 항해 / 갈릴레이 / 뉴턴 / ...

근대 시대 카드 예시:
  산업혁명 / 증기기관 / 전화기 발명 /
  다윈 진화론 / 아인슈타인 / ...

현대 시대 카드 예시:
  인터넷 / DNA 발견 / 우주 탐험 /
  컴퓨터 혁명 / 기후 변화 / ...

미래 시대 카드 예시:
  AI 혁명 / 화성 이주 / 나노 기술 /
  양자 컴퓨터 / 탄소 중립 / ...
```

### 2. 수리 부품 배치

수리 부품 카드 8장을 무작위로
각 시대 구역에 1-2장씩 숨겨놓습니다.

```
부품 배치 예시:
  고대: 시간 결정체 (1개)
  중세: 역사 나침반 (1개)
  근세: 중력 렌즈 (1개)
  근대: 증기 코어 (2개)
  현대: 디지털 메모리 (2개)
  미래: 양자 칩 (1개)
```

### 3. 역할 배분 및 시작 위치

```
3인: 엔지니어 + 역사 전문가 + 항법사
4인: 위 3개 + 과학 탐구자
5인: 모든 역할

시작 위치: 모두 현대(기준점)에서 출발
```

### 4. 타임머신 고장 트랙

```
타임머신 고장 트랙: 0-10
시작: 3
최대치(10): 팀 즉시 패배
매 라운드: 고장 지수 +1 (자동 악화)
```

## 게임 규칙

### 라운드 구조

```
1단계: 타임머신 악화 (고장 지수 +1)
2단계: 시대 이벤트 공개
3단계: 팀 토론 (2분)
4단계: 팀원 행동
5단계: 귀환 체크
```

### 1단계: 타임머신 악화

매 라운드 시작 시 고장 트랙 +1.
엔지니어가 있으면 +1 후 -1 = 현상 유지.

### 2단계: 시대 이벤트

각 시대에서 이벤트 카드 1장씩 공개합니다.

```
시대 이벤트 예시:

고대 이벤트 — "피라미드 건설 현장":
  방문자: 노동력 제공 → 시대 지식 +2
  해결 조건: 역사 문제 정답

중세 이벤트 — "흑사병 창궐":
  방문자: 체력 -1 (위험)
  해결 조건: 과학 원리 문제 정답 (의학 지식)
  회피: 다른 시대로 즉시 이동

근세 이벤트 — "종교 재판":
  방문자: 행동력 -1
  해결 조건: 역사·문화 문제 정답
  문화 해석가: 자동 회피

근대 이벤트 — "산업혁명 공장":
  방문자: 수리 부품 획득 비용 -1
  해결 조건: 과학 원리 문제 정답

현대 이벤트 — "디지털 혁명":
  방문자: 시대 지식 토큰 +3
  해결 조건: 현대 기술 문제 정답

미래 이벤트 — "AI 반란":
  방문자: 즉시 귀환 강제
  해결 조건: 미래 과학 문제 + 철학 문제 정답
  과학 탐구자: 피해 무시
```

### 3단계: 팀 토론

```
토론 핵심:
  - 누가 어느 시대로 이동할 것인가?
  - 수리 부품 수집 우선순위는?
  - 위험한 이벤트를 어떻게 처리할 것인가?
  - 귀환 타이밍은?
```

### 4단계: 팀원 행동

각 플레이어가 행동력을 사용해서 행동합니다.

**①시간 이동 (행동력 1-3):**
다른 시대로 이동합니다.

```
시간 이동 비용:
  인접 시대: 행동력 1
  2단계 건너뜀: 행동력 2
  3단계 이상: 행동력 3
  
예시: 고대 → 근대 = 3단계 = 행동력 3
      현대 → 미래 = 1단계 = 행동력 1
```

**②시대 탐험 (행동력 1):**
현재 시대의 카드 1장을 공개합니다.
수리 부품이 있으면 발견합니다.

**③지식 습득 (행동력 1):**
시대 지식 문제에 도전합니다.
정답 시 시대 지식 토큰 1-3개 획득.

**④수리 부품 획득 (행동력 2):**
발견한 수리 부품을 획득합니다.
지식 문제 정답 필요.

**⑤타임머신 수리 (행동력 2):**
현재(기준점)에서만 사용 가능.
수리 부품 1개 사용 → 고장 트랙 -2.

**⑥귀환 (행동력 1):**
현재(기준점)로 즉시 귀환합니다.
단, 다음 라운드에 다시 시간 이동 가능.

### 5단계: 귀환 체크

모든 플레이어가 현재(기준점)에 있고
수리 부품 8개 + 시대 지식 토큰 12개가 모이면
즉시 팀 승리.

## 지식 문제 시스템

### 시대별 지식 문제

**고대 시대 문제:**

```
문제 1 (⭐):
  Q: 피타고라스가 발견한 수학 정리는?
  A: a²+b²=c² (직각삼각형 세 변의 관계)
  토큰: 1개

문제 2 (⭐⭐):
  Q: 아르키메데스의 부력 원리를 설명하고
     일상 속 응용 예시를 드세요.
  A: 물체가 밀어낸 유체의 무게 = 부력.
     응용: 배, 잠수함, 열기구
  토큰: 2개

문제 3 (⭐⭐⭐):
  Q: 알렉산더 대왕의 동방 원정이
     동서 문화 교류에 미친 영향을 분석하세요.
  A: 헬레니즘 문화 형성 / 그리스 문화 동방 전파 /
     실크로드 개척 기반
  토큰: 3개
```

**근세 시대 문제:**

```
문제 1 (⭐):
  Q: 코페르니쿠스의 지동설 핵심 내용은?
  A: 지구가 태양 주위를 돈다.
  토큰: 1개

문제 2 (⭐⭐):
  Q: 구텐베르크 인쇄술이 유럽 사회에 미친
     혁명적 변화 3가지는?
  A: 성경 보급 확대 / 종교 개혁 촉진 /
     지식의 민주화
  토큰: 2개

문제 3 (⭐⭐⭐):
  Q: 갈릴레이가 아리스토텔레스의 물리학을
     어떻게 반증했는지 설명하세요.
  A: 낙하 실험으로 무거운 물체가 먼저 떨어지지 않음 증명.
     관측으로 지동설 지지.
  토큰: 3개
```

**과학 원리 문제:**

```
문제 1 (⭐⭐):
  Q: 뉴턴의 운동 법칙 3가지를 간단히 설명하세요.
  A: 1법칙: 관성의 법칙
     2법칙: F=ma
     3법칙: 작용·반작용
  토큰: 2개

문제 2 (⭐⭐⭐):
  Q: 열역학 제1·2법칙을 설명하고
     타임머신에 이 법칙들이 어떤 장벽이 되는지?
  A: 1법칙: 에너지 보존 (에너지 무한 공급 불가)
     2법칙: 엔트로피 증가 (시간 역행 불가)
  토큰: 3개
```

```
시간 결정체 (Time Crystal):

  획득 조건: 고대 역사 문제 ⭐⭐ 정답
  수리 효과: 고장 트랙 -2

역사 나침반 (History Compass):
  설명: 과거 시대를 정확히 탐색하는 장치
  획득 조건: 중세 지식 문제 ⭐⭐ 정답
  수리 효과: 시간 이동 비용 -1 (영구)

중력 렌즈 (Gravity Lens):
  설명: 시공간을 구부리는 렌즈
  획득 조건: 근세 과학 문제 ⭐⭐⭐ 정답
  수리 효과: 고장 트랙 -3

증기 코어 (Steam Core) ×2:
  설명: 타임머신의 에너지 공급원
  획득 조건: 근대 과학 문제 ⭐⭐ 정답
  수리 효과: 행동력 +1 (2라운드)

디지털 메모리 (Digital Memory) ×2:
  설명: 시간 좌표를 저장하는 메모리
  획득 조건: 현대 기술 문제 ⭐⭐ 정답
  수리 효과: 지식 토큰 보너스 +2

양자 칩 (Quantum Chip):
  설명: 양자 컴퓨팅 기반 항법 시스템
  획득 조건: 미래 과학 문제 ⭐⭐⭐ 정답
  수리 효과: 타임머신 완전 수리 (고장 트랙 0)
```

## 시간 역설 이벤트

게임 중 특수한 상황이 발생할 수 있습니다.

```
나비 효과 경고:
  발생 조건: 같은 시대에 3명 이상 방문
  효과: 해당 시대 이벤트 효과 2배
  해결: 분산 이동 필요

시간 폭풍:
  발생 조건: 타임머신 고장 트랙 7 이상
  효과: 모든 시간 이동 비용 +1
  해결: 수리 부품 즉시 사용

역사 왜곡:
  발생 조건: 오답 3회 연속
  효과: 수집한 시대 지식 토큰 3개 반납
  해결: 팀원의 힌트 활용

시간 가속:
  발생 조건: 라운드 8 이후 자동 발생
  효과: 매 라운드 고장 트랙 +2
  해결: 빠른 귀환만이 답
```

## 변형 규칙

### 역사 여행 모드
- 과학 원리 문제 없음
- 순수 역사 지식만
- 역사 수업 복습에 최적

### 과학 탐구 모드
- 역사 문제 최소화
- 과학 원리 집중
- 과학 수업 연계

### 미래 예측 모드
- 미래 시대에만 집중
- 현재 과학 기술로 미래 예측
- 창의적 사고 훈련

### 초급 수리단
- 수리 부품 5개
- 시대 지식 8개
- 라운드 12까지 연장

## 교육적 활용

### 역사 교육 연계
- **시대 이해**: 각 시대의 특징과 사건
- **인과 관계**: 역사적 사건의 연쇄 영향
- **시대적 맥락**: 발명과 사회 변화의 연결

### 과학 교육 연계
- **과학사**: 과학 발견의 역사적 맥락
- **원리 이해**: 물리·화학·생물 핵심 원리
- **응용**: 과학 원리의 현실 적용

### 수업 활용 방법
1. 역할 설명 + 시대 카드 소개 (10분)
2. 3인 축약 게임 (25분)
3. "어느 시대가 가장 어려웠나?" 토론 (10분)
4. 수리 부품을 주제로 과학사 학습 (20분)
