# 생태계 수호자 (Ecosystem Guardian)

> 팀원들이 협력해서 무너져가는 생태계를
> 복원하고 균형을 되찾는 협력 게임입니다.
> 생태학적 지식과 협력적 사고가
> 지구를 구합니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-5명 |
| 소요 시간 | 25-35분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 생태학, 환경 과학, 시스템 사고, 협력 |

---

## 게임 목표

팀원들이 협력해서 **6개 생태 구역의
생태 지수를 모두 5 이상**으로 회복하면
팀 전체가 승리합니다.

어느 한 구역의 생태 지수가 0이 되면
해당 구역이 멸종 위기에 처하고
3개 구역이 동시에 0이 되면 팀 전체가 패배합니다.

---

## 준비물

- 생태계 보드 (6개 구역)
- 생태 지수 트랙 (구역별 0-10)
- 종 카드 (동물·식물·미생물 각 20장)
- 위협 카드 덱 (60장)
- 복원 카드 덱 (40장)
- 수호자 역할 카드 5장
- 연결 토큰 (먹이사슬 연결용) 30개
- 오염 마커 20개
- 행동 토큰 (플레이어당 3개)
- 생태 지식 문제 카드 50장

### 생태계 보드 구조

```
┌──────────┬──────────┬──────────┐
│  열대우림  │   초원    │  사막     │
│ 지수: [8] │ 지수: [6] │ 지수: [4] │
├──────────┼──────────┼──────────┤
│  해양     │  습지    │  극지방   │
│ 지수: [7] │ 지수: [5] │ 지수: [3] │
└──────────┴──────────┴──────────┘

각 구역의 생태 지수는 0-10 사이
목표: 모든 구역 5 이상으로 회복
```

## 수호자 역할 카드

```
생물학자 수호자 (Biologist Guardian):
  특수 능력: 멸종 위기 종 복원 비용 절반
  전문 구역: 열대우림·습지
  행동력: 3
  지식 보너스: 생물 관련 문제 정답 시 +2 복원력

기후 과학자 (Climate Scientist):
  특수 능력: 기후 위협 카드 효과 50% 감소
  전문 구역: 극지방·해양
  행동력: 3
  지식 보너스: 기후 문제 정답 시 구역 지수 +2

환경 공학자 (Environmental Engineer):
  특수 능력: 오염 마커 제거 비용 절반
  전문 구역: 사막·초원
  행동력: 3
  지식 보너스: 오염 문제 정답 시 주변 구역 정화

생태 외교관 (Eco Diplomat):
  특수 능력: 구역 간 생태 지수 이전 가능
  전문 구역: 모든 구역
  행동력: 4
  지식 보너스: 국제 환경 문제 정답 시 팀 행동력 +2

자연 탐험가 (Nature Explorer):
  특수 능력: 모든 구역 이동 비용 없음
  전문 구역: 없음 (모든 구역 동등)
  행동력: 4
  지식 보너스: 발견 문제 정답 시 새 복원 카드 획득
```

## 게임 준비

### 1. 초기 생태 지수 설정

```
열대우림: 8  초원: 6  사막: 4
해양: 7  습지: 5  극지방: 3
```

목표는 낮은 구역(사막·극지방·습지)을 먼저 회복하는 것입니다.

### 2. 역할 배분

```
2인: 생물학자 수호자 + 기후 과학자
3인: 위 2개 + 환경 공학자
4인: 위 3개 + 생태 외교관
5인: 모든 역할
```

### 3. 초기 위협 배치

위협 카드 덱에서 3장을 공개해서
각 생태 구역에 즉각적인 위협을 설정합니다.

### 4. 복원 카드 배분

각 플레이어에게 복원 카드 3장을 나눠줍니다.

## 생태 종 카드

### 동물 종 카드 예시

```
흰긴수염고래:
  구역: 해양
  생태 지수 기여: +2
  먹이사슬: 크릴새우 → 흰긴수염고래
  위협: 포경·해양 오염
  복원 조건: 해양 지수 4 이상

시베리아호랑이:
  구역: 극지방 인접 초원
  생태 지수 기여: +3
  먹이사슬: 사슴 → 시베리아호랑이
  위협: 밀렵·서식지 파괴
  복원 조건: 초원 지수 5 이상

열대우림개구리:
  구역: 열대우림
  생태 지수 기여: +1
  먹이사슬: 곤충 → 개구리 → 뱀
  위협: 서식지 파괴·기후 변화
  복원 조건: 습도 지수 조건 충족
```

### 식물 종 카드 예시

```
맹그로브:
  구역: 습지·해안
  생태 지수 기여: +2
  역할: 해안 보호·탄소 흡수·서식지 제공
  복원 조건: 습지 지수 3 이상

사바나 아카시아:
  구역: 초원
  생태 지수 기여: +2
  역할: 그늘 제공·질소 고정·코끼리 먹이
  복원 조건: 초원 지수 2 이상

툰드라 이끼:
  구역: 극지방
  생태 지수 기여: +1
  역할: 토양 형성·순록 먹이·탄소 저장
  복원 조건: 극지방 지수 1 이상
```

## 게임 규칙

### 라운드 구조

```
1단계: 위협 카드 공개 (환경 악화)
2단계: 팀 토론 (2분)
3단계: 수호자 행동 (플레이어 순서대로)
4단계: 먹이사슬 연결 보너스 확인
```

### 1단계: 위협 카드 공개

위협 카드 덱에서 2장을 공개합니다.

```
위협 카드 예시:

산업 오염:
  대상: 해양·습지
  효과: 각 구역 생태 지수 -2
        오염 마커 2개 배치
  해결: 환경 공학자의 정화 행동

기후 변화 심화:
  대상: 극지방
  효과: 생태 지수 -3
        빙하 감소 마커 추가
  해결: 기후 과학자의 복원 행동

불법 벌목:
  대상: 열대우림
  효과: 생태 지수 -2
        종 1개 멸종 위기
  해결: 생물학자 수호자의 보호 행동

사막화 확산:
  대상: 초원·사막 인접 구역
  효과: 초원 지수 -1, 사막 지수 -1
  해결: 식물 복원 카드 사용

해양 온난화:
  대상: 해양·극지방
  효과: 각 지수 -1
        산호초 위기 마커 추가
  해결: 온도 복원 카드 사용
```

### 2단계: 팀 토론

```
토론 핵심 질문:
  - 가장 위급한 구역은?
  - 각자의 역할로 무엇을 할 수 있는가?
  - 어떤 복원 카드를 사용할 것인가?
  - 먹이사슬 연결을 어떻게 구성할 것인가?
```

### 3단계: 수호자 행동

각 플레이어가 행동력을 사용해서 행동합니다.

**①구역 이동 (행동력 1):**
인접한 생태 구역으로 이동합니다.
자연 탐험가는 비용 없이 이동.

**②생태 복원 (행동력 1):**
현재 구역의 생태 지수를 +1 회복합니다.
복원 카드 사용 시 효과 증폭 가능.

**③종 복원 (행동력 2):**
종 카드를 현재 구역에 배치합니다.
해당 종의 생태 지수 기여값만큼 지수 증가.

**④오염 제거 (행동력 1):**
오염 마커 1개를 제거합니다.
지식 문제 정답 시 2개 제거.

**⑤지식 도전 (행동력 1):**
생태 지식 문제에 도전합니다.
정답 시 역할별 보너스 획득.

**⑥먹이사슬 연결 (행동력 1):**
연결 토큰으로 종 카드를 연결합니다.
완전한 먹이사슬 완성 시 큰 보너스.

### 4단계: 먹이사슬 연결 보너스

완전한 먹이사슬이 구성되면 보너스를 받습니다.

```
먹이사슬 예시:
  [식물성 플랑크톤] → [크릴새우] → [고래]
  완성 시: 해양 지수 +3 추가 보너스

  [풀] → [초식동물] → [육식동물] → [분해자]
  완성 시: 초원 지수 +4 추가 보너스

  먹이사슬 길이 보너스:
    2단계: +1
    3단계: +2
    4단계: +4
    5단계 이상: +7
```

## 생태 지식 문제

### 열대우림 문제

```
문제 1:
  Q: 열대우림이 "지구의 허파"라고 불리는 이유는?
  A: 막대한 양의 CO₂를 흡수하고 O₂를 생산.
     지구 산소의 20% 공급.

문제 2:
  Q: 열대우림의 층위 구조 4가지를 설명하세요.
  A: 최상층(캐노피) → 하층 수관 → 관목층 → 지면층

문제 3:
  Q: 키스톤 종(keystone species)의 정의와
     예시를 설명하세요.
  A: 생태계 균형 유지에 핵심적인 역할을 하는 종.
     예: 늑대(옐로스톤), 해달(켈프 숲), 코끼리(사바나)
```

### 해양 문제

```
문제 1:
  Q: 산호초가 해양 생태계에서 차지하는
     중요성을 3가지 설명하세요.
  A: 해양 생물의 25% 서식지 /
     해안 보호 / 관광·수산업 경제 가치

문제 2:
  Q: 해양 산성화의 원인과 산호초에 미치는 영향은?
  A: CO₂ 증가 → 탄산 형성 → pH 감소.
     산호 골격 형성 방해 → 백화 현상

문제 3:
  Q: 플라스틱 오염이 해양 먹이사슬에 미치는
     연쇄 영향을 설명하세요.
  A: 미세 플라스틱 → 플랑크톤 섭취 →
     어류 → 인간 (생물 농축)
```

### 극지방 문제

```
문제 1:
  Q: 북극과 남극의 생태학적 차이를 설명하세요.
  A: 북극: 빙하 위 생태계, 북극곰·바다코끼리.
     남극: 대륙 위 생태계, 펭귄·크릴새우.

문제 2:
  Q: 영구동토층(permafrost)의 융해가
     기후 변화를 가속화하는 메커니즘은?
  A: 동토층 → 탄소 저장.
     융해 시 CO₂·메탄 방출 → 온실 효과 강화
     → 추가 융해 (양성 피드백)

문제 3:
  Q: 해빙(sea ice) 감소가 알베도에 미치는
     영향을 설명하세요.
  A: 흰 얼음(반사율 80%) → 어두운 바다(반사율 6%)
     → 열 흡수 증가 → 추가 온난화
```

## 복원 카드 목록

```
자연 재조림:
  효과: 선택한 구역 생태 지수 +3
  조건: 해당 구역에 식물 종 없음
  비용: 행동력 2

해양 보호구역 설정:
  효과: 해양 지수 +2
        해양 오염 마커 1개 제거
  조건: 없음
  비용: 행동력 1

멸종 위기 종 보호 프로그램:
  효과: 멸종 위기 종 1개 즉시 복원
        해당 구역 지수 +2
  조건: 생물학자 수호자 필요
  비용: 행동력 2

탄소 포집 기술:
  효과: 기후 위협 카드 효과 완전 무효화
  조건: 기후 과학자 또는 환경 공학자
  비용: 행동력 1

국제 환경 협약:
  효과: 다음 3라운드 위협 카드 효과 50% 감소
  조건: 생태 외교관 + 지식 문제 정답
  비용: 행동력 2

생태 회랑 조성:
  효과: 두 구역을 연결, 지수 평균화
  조건: 인접 구역
  비용: 행동력 1
```

## 변형 규칙

### 기후 위기 모드
- 매 라운드 기후 위협 자동 강화
- 위협 카드 3장 공개
- 최고 난이도 생태 게임

### 교육 모드
- 위협 카드 없음
- 순수 생태계 복원에 집중
- 지식 문제만으로 진행

### 지역 생태계 모드
- 특정 지역 생태계만 집중
- 예: 한국 생태계 (DMZ·갯벌·백두대간)
- 지역 생태 교육에 활용

---

## 교육적 활용

### 과학 교육 연계
- **생태학**: 먹이사슬·생태 지수·생물 다양성
- **환경 과학**: 오염·기후 변화·복원 방법
- **시스템 사고**: 생태계 상호 연결성

### 협력 교육 연계
- **공동 목표**: 생태계 회복이라는 명확한 목표
- **역할 전문화**: 각 수호자의 고유 능력 활용
- **집단 의사결정**: 복원 우선순위 협의

### 수업 활용 방법
1. 생태계 기본 개념 설명 (10분)
2. 2인 기초 게임 (20분)
3. 먹이사슬 연결 토론 (10분)
4. 실제 환경 문제 연결 토론 (15분)
