# 사회 계약 (Social Contract)

> 우리는 왜 규칙을 따르는가?
> 사회가 없다면 어떻게 살 것인가?
> 새로운 사회의 규칙을
> 직접 만들어 보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 사회 계약론, 공정성, 협력적 규칙 설계 |

## 배경 설정

```
시나리오:
  당신들은 무인도에 표류했습니다.
  구조는 6개월 후입니다.
  함께 살아가기 위한 규칙을
  처음부터 만들어야 합니다.

자원:
  음식: 하루치 × 30일
  물: 충분
  도구: 제한적
  지식: 각자 다른 기술 보유
```

## 게임 구성

```
자원 카드: 식량·도구·기술 각 20장
규칙 제안 용지: 개인당 5장
투표 토큰: 각 플레이어 10개
갈등 카드: 20장 (사건·위기)
```

## 기본 규칙

```
1단계: 자원 분배
  자원 카드 배분 (불평등하게)
  → 현실의 불평등 반영

2단계: 규칙 제안
  각자 규칙 3개 제안
  예) "노동은 능력에 따라"
      "분배는 필요에 따라"
      "의사결정은 다수결로"

3단계: 합의
  규칙 투표·협상
  과반 찬성 시 채택

4단계: 갈등 해결
  갈등 카드 발생
  → 만든 규칙으로 해결
  → 규칙 수정 필요 시 재협상
```

## 갈등 카드 예시

```
"음식이 떨어져 갑니다.
 누가 더 받아야 하는가?
 - 더 많이 일한 사람?
 - 더 배고픈 사람?
 - 아픈 사람?"

"한 명이 규칙을 어겼습니다.
 어떤 벌칙을 줄 것인가?
 벌칙이 없다면 규칙의 의미는?"

"리더가 필요합니다.
 어떻게 뽑을 것인가?
 뽑지 않을 것인가?"
```

## 롤스의 무지의 베일

```
심화 모드:
  자원 배분 전
  자신이 어떤 처지가 될지 모르는 상태에서
  규칙을 먼저 만들기

→ "무지의 베일" 실험
→ 모르면 공정하게 만든다는 롤스의 주장 체험
```

## 교육적 효과

```
사회 계약론:
  홉스·로크·루소·롤스 사상 체험
  → 정치철학 기초

공정성 감각:
  나에게 불리해도 공정한 규칙
  → 도덕 발달

민주주의 경험:
  규칙 만들기·합의·수정
  → 민주적 과정 체험
```

## 게임 후 토론

```
"만든 규칙 중 가장 잘 작동한 것은?
 왜 잘 작동했나요?"

"실제 사회의 법과
 오늘 만든 규칙의 차이는?"

"공정한 사회는 어떤 사회인가요?"
```
