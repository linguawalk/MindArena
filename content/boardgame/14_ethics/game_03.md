# 철학자 배틀 (Philosopher Battle)

> 소크라테스라면 어떻게 말했을까?
> 니체라면? 공자라면?
> 역사 속 철학자의 관점으로
> 현대 문제를 논합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 철학적 사고, 관점 채택, 논증 능력 |

## 게임 구성

```
철학자 카드: 20장 (동서양 철학자)
현대 문제 카드: 30장
철학자 사상 요약 카드: 20장 (참고용)
논증 점수판
```

## 철학자 카드 목록

```
서양:
  소크라테스 (대화·무지의 지)
  플라톤 (이데아·철인정치)
  아리스토텔레스 (중용·행복)
  칸트 (의무·정언명령)
  밀 (공리주의·자유)
  니체 (권력의지·초인)
  롤스 (정의·무지의 베일)
  마르크스 (계급·역사유물론)

동양:
  공자 (인·예·충서)
  노자 (무위자연·도)
  맹자 (성선설·민본)
  순자 (성악설·예)
  석가모니 (연기·중도)
  원효 (화쟁·일심)
```

## 기본 규칙

```
역할 배정:
  철학자 카드 1장씩 받기
  사상 요약 카드로 철학자 이해

현대 문제 공개:
  예) "SNS는 민주주의에 도움이 되는가?"

토론 (각 2분):
  자신의 철학자 관점으로 주장
  "___는 이렇게 말할 것입니다.
   왜냐하면 ___이 때문입니다."

배심원 평가:
  가장 설득력 있는 철학적 논증
  (내용 + 철학자 관점 일치도)
```

## 현대 문제 카드 예시

```
기술:
  "AI에게 권리를 부여해야 하는가?"
  "소셜미디어를 규제해야 하는가?"

사회:
  "기본소득은 정의로운가?"
  "사형제는 폐지해야 하는가?"

환경:
  "현재 세대가 미래 세대에게
   빚을 지고 있는가?"

개인:
  "거짓말이 정당화될 수 있는가?"
  "자유는 무제한일 수 있는가?"
```

## 철학자 관점 예시

```
문제: "거짓말이 정당화될 수 있는가?"

칸트:
  "거짓말은 절대 안 된다.
   모든 사람이 거짓말하는 세상을
   원하지 않는다면 거짓말하면 안 된다"

공리주의(밀):
  "선의의 거짓말은 정당하다.
   더 많은 사람에게 이익이 된다면
   거짓말도 허용될 수 있다"

소크라테스:
  "우리는 진실이 무엇인지도 모른다.
   그러니 거짓말도 모르는 것이다"
```

## 교육적 효과

```
철학 지식:
  주요 철학자 사상 습득
  → 인문학적 기초

관점 채택:
  다른 사람 눈으로 보기
  → 탈중심화 사고

논증 구조:
  주장 + 원리 + 적용
  → 논리적 사고력
```
