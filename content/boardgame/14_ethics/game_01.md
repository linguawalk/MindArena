# 트롤리 문제 변형 (Trolley Problem Variants)

> 달리는 트롤리가 5명을 향해 갑니다.
> 레버를 당기면 1명이 죽습니다.
> 당기겠습니까?
> 이 단순한 질문이 왜 이렇게 어려운가?

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 윤리적 추론, 결과론 vs 의무론, 가치 충돌 |

## 게임 구성

```
딜레마 카드: 40장 (트롤리 문제 변형)
입장 카드: 찬성·반대·기권
논거 카드: 50장 (윤리 이론 근거)
설득 점수판
```

## 기본 규칙

```
딜레마 공개:
  카드 1장 공개 후 낭독

비밀 투표:
  각자 찬성·반대·기권 카드 선택
  동시 공개

토론 (각 2분):
  자신의 선택 논거 발표

재투표:
  토론 후 마음이 바뀌었는지 재투표

득점:
  자신의 논거에 동의받은 수
  마음을 바꾸게 한 수 (설득 성공)
```

## 딜레마 카드 예시

```
고전:
  "레버를 당겨 5명 대신 1명을 희생시킬 것인가?"

변형 1 - 육교:
  "육교에서 뚱뚱한 사람을 밀어
   트롤리를 멈출 것인가?"
  (같은 결과, 다른 행동 방식)

변형 2 - 의사:
  "건강한 환자 1명의 장기를 적출해
   5명의 환자를 살릴 것인가?"

변형 3 - 아는 사람:
  "희생될 1명이 당신의 가족이라면?"

변형 4 - AI:
  "자율주행 차가 사고 시
   탑승자와 보행자 중 누구를 보호해야 하는가?"

변형 5 - 미래:
  "현재 1명을 희생해
   미래 100명을 살릴 수 있다면?"
```

## 윤리 이론 논거 카드

```
결과론 (공리주의):
  "최대 다수의 최대 행복"
  수를 세어 더 많은 이익 선택

의무론 (칸트):
  "사람을 수단으로 쓰면 안 된다"
  결과와 무관하게 옳은 행동

덕 윤리:
  "좋은 사람이라면 어떻게 할 것인가?"

계약론:
  "모든 사람이 동의할 수 있는 규칙은?"
```

## 교육적 효과

```
윤리적 추론:
  직관과 이론의 충돌 체험
  → 도덕 철학의 핵심 문제

관점 이해:
  같은 상황, 다른 판단
  → 다양한 가치관 존중

논증 능력:
  윤리적 주장의 근거 제시
  → 도덕적 추론 능력
```

"당신은 결과론자인가요,
