# 도덕적 딜레마 법정 (Moral Dilemma Court)

> 법적으로는 유죄지만 도덕적으로 옳을 수 있습니다.
> 도덕적으로는 나쁘지만 법적으로 무죄일 수 있습니다.
> 법과 도덕의 경계에서 판결을 내려보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐⭐ |
| 목표 역량 | 법·도덕 구분, 상황 윤리, 판단력 |

## 게임 구성

```
사건 카드: 30장 (도덕·법 딜레마 사건)
역할 카드: 판사·검사·변호사·배심원·증인
증거 카드: 각 사건당 5장
판결 용지
```

## 사건 카드 예시

```
사건 1: 장 발장
  "배고픈 아이를 위해 빵을 훔쳤다.
   법적: 유죄 / 도덕적: ?"

사건 2: 내부 고발
  "회사의 불법을 폭로하기 위해
   기밀 문서를 유출했다"

사건 3: 안락사
  "극심한 고통의 환자가 요청해
   의사가 죽음을 도왔다"

사건 4: 거짓말
  "나치에게 숨겨준 유대인을
   보호하기 위해 거짓말했다"

사건 5: AI 결정
  "AI 의사가 1명을 희생해
   5명을 살리는 결정을 내렸다"

사건 6: 시민 불복종
  "불의한 법에 저항하기 위해
   법을 어기고 시위했다"
```

## 재판 구조

```
①사건 낭독 (2분)

②역할별 준비 (3분):
  검사: 법적 유죄 논거
  변호사: 도덕적 정당성 논거
  증인: 상황 설명

③검사 측 논거 (2분)

④변호사 측 논거 (2분)

⑤배심원 질의 (2분)

⑥최후 진술 (각 1분)

⑦배심원 평결:
  법적 판결: 유죄/무죄
  도덕적 판결: 옳음/그름
  (두 판결이 다를 수 있음!)
```

## 판결 유형

```
법적 유죄 + 도덕적 옳음:
  → 악법 문제 논의

법적 무죄 + 도덕적 그름:
  → 법의 허점 논의

법적 유죄 + 도덕적 그름:
  → 일반적 범죄

법적 무죄 + 도덕적 옳음:
  → 모범적 행동
```

## 교육적 효과

```
법·도덕 구분:
  법과 도덕이 항상 일치하지 않음
  → 시민 의식 함양

상황 윤리:
  맥락에 따라 달라지는 판단
  → 절대주의 vs 상대주의

시민 불복종:
  마틴 루터 킹·간디의 논리 이해
  → 민주주의 깊은 이해
```
