# 가치관 경매 (Values Auction)

> 당신에게 가장 중요한 것은 무엇인가?
> 자유, 평등, 안전, 행복, 진실.
> 돈이 있다면 무엇을 먼저 사겠습니까?
> 선택이 곧 당신의 가치관입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 가치 명료화, 우선순위 결정, 자기 이해 |

## 게임 구성

```
가치 카드: 30장
경매 칩: 플레이어당 100점
가치 이해 카드: 각 가치당 1장 (정의 설명)
기록 시트
```

## 가치 목록

```
개인적 가치:
  자유 / 행복 / 건강 / 성취 / 사랑
  지식 / 아름다움 / 모험 / 평화 / 안전

사회적 가치:
  평등 / 정의 / 공동체 / 민주주의 / 법질서
  환경 / 다양성 / 연대 / 투명성 / 책임

초월적 가치:
  진실 / 선함 / 의미 / 영성 / 유산
  창의성 / 지혜 / 용기 / 겸손 / 감사
```

## 기본 규칙

```
경매 진행:
  가치 카드 1장씩 경매
  비밀 입찰 (동시 공개)
  최고 입찰자 낙찰

예산 배분:
  100점으로 30가지 가치 중 선택
  → 무엇에 얼마를 쓸 것인가?

결과 공유:
  각자 구입한 가치 공개
  "왜 이것을 선택했나?" 설명

토론:
  같은 가치를 다른 이유로 선택
  다른 가치를 같은 이유로 선택
```

## 심화 변형

```
딜레마 라운드:
  두 가치 중 하나만 선택
  "자유 vs 안전"
  "평등 vs 효율"
  "진실 vs 친절"

사회 설계:
  팀으로 사회 헌법 만들기
  상위 5개 가치 합의
```

## 교육적 효과

```
가치 명료화:
  막연한 가치관을 구체화
  → 자기 이해 심화

우선순위:
  모두 중요해 보이는 것 중 선택
  → 실제 의사결정 훈련

다양성 이해:
  다른 사람의 가치관 이해
  → 공감 능력 향상
```

## 게임 후 토론

```
"가장 비싸게 산 가치는 무엇인가요?
 왜 그것이 가장 중요한가요?"

"포기한 가치 중 아쉬운 것은?
 왜 포기했나요?"

"10년 후 같은 경매를 한다면
 선택이 달라질까요?"
```
