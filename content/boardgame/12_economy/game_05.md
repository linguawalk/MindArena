# 가격 탐정 (Price Detective)

> 이 물건의 적정 가격은 얼마인가?
> 너무 비싸면 안 팔리고
> 너무 싸면 손해입니다.
> 가치와 가격의 차이를 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 가치 평가, 가격 결정, 소비자 심리 |

## 게임 구성

```
상품 카드: 60장 (다양한 물건·서비스)
정보 카드: 각 상품당 4장
  (원가·수요·경쟁·브랜드)
가격 추측 보드: 1-1000 범위
정답 봉투: 실제 가격 담긴 봉투
```

## 기본 규칙

```
상품 공개:
  상품 카드 1장 공개
  (예: "스타벅스 아메리카노 한 잔")

정보 공개 (선택):
  정보 카드 구매 가능
  구매할수록 점수 깎임

가격 예측:
  각자 비밀리에 가격 적기

공개 및 채점:
  정답 봉투 개봉
  실제 가격과 가장 가까운 사람 승리

득점:
  오차 10% 이내: +3점
  오차 25% 이내: +1점
  오차 50% 초과: 0점
  정확: +5점 보너스
```

## 상품 카드 예시

```
일상:
  편의점 삼각김밥 / 택시 기본요금
  영화관 팝콘 / 대학교 1년 등록금

특별:
  에르메스 버킨백 / 람보르기니
  국제우주정거장 1일 체류비
  월드컵 결승 티켓

서비스:
  변호사 시간당 상담료
  성형외과 쌍꺼풀 수술
  헬리콥터 1시간 임대
```

## 가격 결정 요인 토론

```
정보 카드가 알려주는 것:
  원가: 만드는 데 얼마?
  수요: 사려는 사람이 얼마나?
  경쟁: 대체품이 있나?
  브랜드: 이미지가 얼마나 중요?

가격 ≠ 가치:
  "물은 생존에 필수지만 싸고
   다이아몬드는 없어도 되지만 비싸다"
  → 희소성의 역설
```

## 교육적 효과

```
가격 원리:
  원가·수요·경쟁·브랜드의 역할
  → 미시경제학 기초

소비자 심리:
  왜 비싼 것을 더 원하는가?
  → 베블런 효과·사치재

가치 판단:
  무엇이 진짜 가치 있는가?
  → 비판적 소비 의식
```

## 게임 후 토론

```
"가장 놀라웠던 가격은 무엇이었나요?
 왜 그렇게 비싸거나 쌌을까요?"

"가격이 공정하다는 것은
 어떤 의미인가요?"

"당신이 가장 가치 있다고 생각하는 것의
 가격은 적절한가요?"
```
