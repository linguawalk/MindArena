# 시장을 움직여라 (Move the Market)

> 수요와 공급이 가격을 결정합니다.
> 하지만 모든 플레이어가
> 동시에 시장에 개입한다면?
> 보이지 않는 손을 직접 느껴보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 수요·공급 원리, 시장 균형, 전략적 행동 |

## 게임 구성

```
상품 보드: 5가지 상품 (쌀·철·직물·향신료·금)
가격 트랙: 각 상품 1-20
수요·공급 카드: 각 50장
자본 칩: 플레이어당 100점
거래 기록판
```

## 기본 규칙

```
목표:
  10라운드 후 최대 자산 보유

매 턴:
  ①비밀 행동 선택:
    매입 (수요 증가 → 가격 상승)
    매도 (공급 증가 → 가격 하락)
    보유 (관망)

  ②동시 공개

  ③가격 변동:
    순매입 > 순매도: 가격 +1
    순매도 > 순매입: 가격 -1
    균형: 가격 유지

  ④정산:
    보유 상품 × 현재 가격

특수 카드:
  풍작: 공급 급증
  전쟁: 특정 상품 수요 급증
  독점: 한 플레이어 가격 통제
```

## 전략 포인트

```
정보 비대칭:
  다른 플레이어 의도 파악
  → 역발상 투자

타이밍:
  저점 매입 / 고점 매도
  → 시장 사이클 이해

집단 행동:
  모두 같은 방향이면 수익 없음
  → 게임 이론적 사고
```

## 교육적 효과

```
수요·공급:
  가격 결정 원리 체험
  → 경제학 기초 개념

시장 심리:
  탐욕과 공포의 순환
  → 행동 경제학 이해

전략적 상호작용:
  내 행동이 시장에 미치는 영향
  → 게임 이론 기초
```

```
"모든 사람이 동시에 팔려고 하면
 어떤 일이 벌어지나요?"

"실제 주식시장도 이렇게 작동하나요?"
```
