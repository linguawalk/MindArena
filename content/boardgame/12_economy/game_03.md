# 버블과 붕괴 (Bubble and Crash)

> 가격이 계속 오릅니다.
> 언제 팔아야 할까요?
> 버블인 줄 알면서도 올라타는 이유.
> 금융 위기의 심리를 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 버블 인식, 탐욕·공포 심리, 군중 심리 |

## 게임 구성

```
자산 보드: 가격 트랙 1-100
투자 칩: 플레이어당 50개
버블 주사위: 특수 주사위
붕괴 카드: 10장 (덱에 숨겨짐)
공황 토큰: 5개
```

## 기본 규칙

```
가격 상승 단계:
  매 턴 버블 주사위 굴리기
  결과에 따라 가격 +1~+5

투자 결정:
  매입: 현재 가격에 칩 투자
  보유: 현재 포지션 유지
  매도: 현재 가격에 칩 회수

붕괴 트리거:
  붕괴 카드가 나오면
  가격 즉시 1/3로 폭락
  (덱에 언제 있는지 모름)

매도 타이밍:
  붕괴 전 매도 = 수익
  붕괴 후 매도 = 손실
```

## 버블 심리 메커니즘

```
탐욕 단계:
  가격 계속 상승 → 더 오를 것 같음
  → 매도 못함

군중 심리:
  다른 플레이어도 안 팜 → 나도 보유
  → 집단적 비합리성

공포 단계:
  붕괴 후 → 더 떨어질 것 같음
  → 바닥에 매도
```

## 실제 버블 사례 카드

```
튤립 버블 (1637):
  튤립 구근 1개 = 집 한 채 가격
  → 한순간에 폭락

닷컴 버블 (2000):
  인터넷 기업 가격 폭등
  → 3년간 -78%

부동산 버블 (2008):
  주택 가격 영원히 오를 것 같았음
  → 글로벌 금융위기
```

## 교육적 효과

```
버블 인식:
  비합리적 가격 상승 패턴
  → 금융 위기 이해

심리적 편향:
  손실 회피·군중 심리 체험
  → 행동 경제학 핵심

출구 전략:
  언제 빠져나올지 결정
  → 실전 투자 판단력
```
