# 협력 경제 (Cooperative Economy)

> 혼자는 할 수 없습니다.
> 분업과 교역이 모두를 풍요롭게 합니다.
> 비교 우위를 이해하고
> 협력의 이익을 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-5명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 비교 우위, 분업, 교역의 이익 |

## 게임 구성

```
지역 보드: 플레이어당 1개
  (각기 다른 자원 생산 효율)
자원 타일: 식량·목재·철·직물·기술 각 20개
교역 카드: 40장
목표 카드: 공동 목표 10장
```

## 각 지역 특성

```
농업 지역: 식량 2배 생산 / 철 절반
산림 지역: 목재 2배 / 식량 절반
광산 지역: 철 2배 / 목재 절반
직물 지역: 직물 2배 / 기술 절반
기술 지역: 기술 2배 / 직물 절반
```

## 기본 규칙

```
개별 생산:
  자기 지역 자원 생산
  효율에 따라 수량 결정

교역:
  다른 플레이어와 자원 교환
  가격은 협상으로 결정

공동 목표:
  5가지 자원 모두 일정량 필요한
  공동 프로젝트 완성

승리:
  공동 프로젝트 완성 = 전원 승리
  실패 = 전원 패배

단, 개인 기여도로 순위 결정
```

## 비교 우위 실험

```
혼자 모든 자원 생산 시:
  턴당 자원: 5종류 × 1개 = 5개

분업 후 교역 시:
  특화 생산: 1종류 × 4개
  교역 후: 4종류 × 1개 이상

→ 분업이 항상 효율적임을 체험
```

## 교육적 효과

```
비교 우위:
  절대 우위보다 비교 우위가 중요
  → 국제 무역의 핵심 원리

분업의 이익:
  특화가 전체 파이를 키움
  → 애덤 스미스 핀 공장 원리

협상:
  교역 조건 결정
  → 실생활 협상 능력
```

"자유 무역의 장점과 단점은
 무엇인가요?"
