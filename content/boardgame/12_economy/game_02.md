# 포트폴리오 대전 (Portfolio Battle)

> 계란을 한 바구니에 담지 마세요.
> 하지만 너무 분산하면 수익도 없습니다.
> 위험과 수익의 균형을 찾는
> 투자의 기술을 배웁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 분산 투자, 위험 관리, 기댓값 사고 |

## 게임 구성

```
자산 카드: 6종류 × 10장
  주식 (고위험·고수익)
  채권 (저위험·저수익)
  부동산 (중위험·중수익)
  금 (안전 자산)
  스타트업 (초고위험·초고수익)
  현금 (무위험·무수익)

시장 사건 카드: 30장
초기 자본: 플레이어당 200점
```

## 기본 규칙

```
투자 단계:
  자본을 6가지 자산에 배분
  각 자산 최소 0 / 최대 자본의 60%

시장 사건:
  매 라운드 사건 카드 1장
  자산별 수익률 결정

예시 사건 카드:
  "경기 호황": 주식 +20%, 채권 +5%
  "금리 인상": 채권 -10%, 현금 +3%
  "전쟁 발발": 금 +30%, 주식 -25%
  "기술 혁신": 스타트업 +50%, 부동산 -5%

수익 계산:
  투자금 × 수익률 = 이익/손실

5라운드 후:
  최대 자산 보유자 승리
```

## 전략 유형

```
공격형:
  스타트업·주식 집중
  고위험 고수익 추구

방어형:
  채권·금·현금 집중
  손실 최소화

균형형:
  6가지 골고루
  안정적 성장
```

## 교육적 효과

```
분산 투자:
  "모든 달걀을 한 바구니에"
  → 실제 투자 원칙 체험

위험 인식:
  수익과 위험은 비례
  → 금융 리터러시 기초

장기 사고:
  단기 손실 견디기
  → 투자 심리 훈련
```
