# 색깔 감정 (Color Emotion)

> 빨강은 정말 열정인가요?
> 파랑은 항상 슬픔인가요?
> 색과 감정의 관계를 탐구하고
> 색채 언어를 배웁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 색채 감각, 감정 표현, 문화적 색 이해 |

## 게임 구성

```
색상 카드: 50장 (다양한 색과 명도·채도)
감정 카드: 30장 (다양한 감정·상황)
문화 카드: 10장 (나라별 색 의미)
연결 점수판
```

## 기본 규칙

```
모드 1 - 색→감정:
  색상 카드 공개
  각자 어울리는 감정 카드 선택
  일치할수록 점수

모드 2 - 감정→색:
  감정 카드 공개
  각자 어울리는 색 선택 (그리기·말하기)
  가장 독창적 or 공감 많은 선택 점수

모드 3 - 문화 차이:
  같은 색이 나라마다 다른 의미
  어느 나라에서 어떤 의미인지 맞히기
```

## 색채 심리학 데이터

```
빨강:
  서양: 열정·위험·사랑
  중국: 행운·축제
  인도: 순수·결혼

흰색:
  서양: 순수·결혼
  동아시아: 죽음·애도
  중동: 순수·평화

노랑:
  서양: 행복·주의
  프랑스: 질투·배신
  이집트: 행복·번영

검정:
  서양: 죽음·권위·세련
  일본: 신비·우아
  아프리카 일부: 성숙·남성성
```

## 색채 구성 게임

```
테마 팔레트:
  주제를 3가지 색으로 표현
  예) "봄": 연두·분홍·하늘
  다른 사람이 주제 맞히기

감정 그라데이션:
  기쁨→슬픔 사이를
  색 변화로 표현
  → 색의 연속성 이해
```

## 교육적 효과

```
색채 감각:
  색의 심리적 효과 이해
  → 디자인·예술 기초

문화 감수성:
  같은 색의 다른 의미
  → 문화적 다양성 이해

감정 표현:
  비언어적 감정 표현
  → 예술적 소통 능력
```

## 게임 후 토론

```
"가장 의견이 갈린 색은 무엇이었나요?
 왜 그랬을까요?"

"당신이 가장 좋아하는 색은?
 그 색이 주는 느낌은?"

"광고나 상품 디자인에서
 색이 어떻게 활용되는지 예를 들어보세요."
```
