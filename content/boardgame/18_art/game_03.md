# 디자인 심판 (Design Judge)

> 좋은 디자인이란 무엇인가?
> 아름다운 것과 기능적인 것
> 그리고 의미 있는 것.
> 디자인을 보는 눈을 기릅니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 디자인 사고, 심미안, 비판적 시각 |

## 게임 구성

```
디자인 카드: 60장 (실제 디자인 사례)
  로고·포스터·제품·건축·UI
평가 기준 카드: 20장
디자인 원칙 카드: 10장
투표 토큰: 플레이어당 5개
```

## 기본 규칙

```
디자인 공개:
  디자인 카드 2장 나란히 공개
  같은 목적의 두 디자인 비교

평가 기준 공개:
  기능성·심미성·독창성·전달력 중 1개

토론 (3분):
  각자 더 나은 디자인 주장
  평가 기준에 따른 논거 제시

투표:
  더 나은 디자인 선택
  이유 1문장 필수

득점:
  다수 선택: +2점
  설득력 있는 논거: +1점
```

## 디자인 원칙 카드

```
대비 (Contrast):
  서로 다른 요소로 강조

반복 (Repetition):
  통일성을 위한 반복

정렬 (Alignment):
  보이지 않는 선으로 질서

근접 (Proximity):
  관련 요소끼리 가까이

여백 (White Space):
  비움으로 강조하기

계층 (Hierarchy):
  중요한 것을 크게·먼저
```

## 디자인 카드 주제

```
로고 디자인:
  유명 브랜드 로고 변천사
  성공한 리브랜딩 vs 실패한 리브랜딩

포스터:
  사회 캠페인 포스터
  영화 포스터

제품:
  인체공학적 디자인 vs 심미적 디자인
  지속 가능한 디자인

UI/UX:
  앱 인터페이스 비교
  사용하기 쉬운 vs 어려운 디자인
```

## 교육적 효과

```
디자인 사고:
  목적·사용자·맥락 고려
  → 문제 해결 사고

심미 교육:
  좋은 디자인 보는 눈
  → 시각 문화 이해

논증 능력:
  주관적 감각을 논리로 표현
  → 비판적 사고
```

## 게임 후 토론

```
"좋은 디자인의 조건은
 무엇인가요?"

"아름답지만 불편한 디자인과
 못생겼지만 편리한 디자인.
 어느 것이 더 좋은 디자인인가요?"

"당신 주변에서 가장 잘 디자인된
 것은 무엇인가요?"
```
