# 명화 탐정 (Masterpiece Detective)

> 위대한 그림에는 이야기가 있습니다.
> 그림을 보고 숨겨진 의미를 찾아보세요.
> 미술사 지식과 관찰력을 동시에 훈련합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 미술 감상, 상징 해석, 역사적 맥락 이해 |

## 게임 구성

```
명화 카드: 40장 (동서양 유명 작품)
질문 카드: 각 작품당 3-5장
상징 사전 카드: 30장 (미술 상징 모음)
시대 배경 카드: 20장
```

## 기본 규칙

```
명화 공개:
  명화 카드 1장 공개
  1분간 조용히 관찰

질문 단계:
  질문 카드 1장 뽑기
  예) "이 그림에서 가장 먼저 눈에 띄는 것은?"
      "화가는 어떤 감정을 표현했는가?"
      "이 그림의 배경은 어떤 시대인가?"

답변:
  각자 답변 준비 (1분)
  순서대로 발표

채점:
  정답이 있는 질문: 정확도
  해석 질문: 근거의 풍부함
```

## 명화 카드 예시

```
서양:
  모나리자 (다빈치): 미소의 비밀
  별이 빛나는 밤 (반 고흐): 소용돌이의 의미
  게르니카 (피카소): 전쟁의 참상
  진주 귀걸이 소녀 (베르메르): 빛의 표현

한국:
  씨름 (김홍도): 조선 민중의 삶
  월하정인 (신윤복): 남녀 관계의 묘사
  청자 상감운학문 매병: 고려의 미의식
```

## 미술 감상 5단계

```
1단계 - 기술 (Description):
  보이는 것을 객관적으로 묘사
  "무엇이 보이는가?"

2단계 - 분석 (Analysis):
  구성 요소의 관계 파악
  "어떻게 배치되어 있는가?"

3단계 - 해석 (Interpretation):
  의미와 감정 탐구
  "무슨 뜻인가? 어떤 감정인가?"

4단계 - 판단 (Judgment):
  가치와 중요성 평가
  "왜 이 그림이 위대한가?"

5단계 - 맥락 (Context):
  역사적·문화적 배경
  "이 시대에 왜 이렇게 그렸는가?"
```

## 교육적 효과

```
미술 교양:
  동서양 명화 지식
  → 문화적 소양

상징 해석:
  그림 속 숨겨진 의미 찾기
  → 상징적 사고

역사 연결:
  그림을 통한 역사 이해
  → 통합적 사고
```
