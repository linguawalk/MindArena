# 관찰 드로잉 배틀 (Observation Drawing Battle)

> 정확하게 보는 것이 먼저입니다.
> 그림 실력보다 관찰 능력이 중요합니다.
> 눈앞의 대상을 얼마나 정확히
> 보고 표현할 수 있는가?

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 관찰력, 시각적 표현, 세부 인식 |

## 게임 구성

```
관찰 대상 카드: 40장 (사물·동물·건물 사진)
드로잉 용지: 플레이어당 10장
채점 루브릭: 관찰 정확도 기준
타이머: 2분
```

## 기본 규칙

```
대상 공개:
  관찰 대상 카드 30초 동안 공개
  이후 카드 뒤집기

드로잉 (2분):
  기억과 관찰로 그리기
  최대한 정확하게

대상 재공개:
  원본과 비교

평가:
  비율 정확도: 전체 형태
  세부 요소: 특징적 부분
  위치 관계: 각 요소의 배치

득점:
  세부 요소 1개당 +1점
  전체 형태 유사: +2점
  독특한 특징 포착: +1점
```

## 관찰 훈련법

```
전체→부분:
  먼저 전체 윤곽 파악
  그 다음 세부 요소

비율 관찰:
  "머리가 몸의 1/4 크기"
  "창문이 문 높이의 절반"

음영 관찰:
  빛이 어디서 오는가
  그림자는 어디에 생기는가

특징 포착:
  이 대상만의 독특한 점
  가장 먼저 눈에 띄는 것
```

## 변형 모드

```
시간 연장:
  30초→10초로 줄이기
  → 더 빠른 관찰 훈련

맹목 드로잉:
  종이 보지 않고 그리기
  → 손-눈 협응 훈련

반쪽 완성:
  그림 절반만 주고
  나머지 완성하기
  → 대칭·비율 이해
```

## 교육적 효과

```
관찰력:
  세밀한 시각적 주의
  → 과학적 관찰 능력과 연결

시각 기억:
  본 것을 기억에 저장
  → 시각적 사고력

표현력:
  본 것을 표현하는 능력
  → 시각 언어 습득
```
