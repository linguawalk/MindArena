# 추상 표현 (Abstract Expression)

> 형태 없이 감정을 표현할 수 있는가?
> 선 하나, 점 하나에도 의미가 있습니다.
> 추상 미술의 언어를 배우고
> 순수한 표현의 자유를 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 추상적 표현, 감정 시각화, 예술적 해석 |

## 게임 구성

```
감정·개념 카드: 40장
추상 표현 도구:
  선 종류: 직선·곡선·점선·지그재그
  형태: 원·삼각·사각·자유형
  색상: 기본색 + 혼합
드로잉 용지: 플레이어당 10장
```

## 기본 규칙

```
카드 공개:
  감정·개념 카드 1장
  예) "외로움" "희망" "혼돈" "평화"

표현 (3분):
  선·형태·색만 사용
  구체적 형상 금지
  (사람·집·꽃 등 불가)

발표:
  그림 공개
  어떤 감정·개념인지 다른 플레이어가 맞히기

득점:
  맞힌 사람: +2점
  맞힌 사람 수 × 1점 (표현자에게)
```

## 추상 표현 가이드

```
선의 감정:
  수직선: 강함·안정·권위
  수평선: 평화·안정·죽음
  대각선: 움직임·긴장·역동
  곡선: 부드러움·유연·생명
  지그재그: 긴장·불안·에너지

형태의 감정:
  원: 완전·조화·따뜻함
  삼각형: 긴장·방향·공격성
  사각형: 안정·질서·신뢰
  불규칙: 혼란·자유·창의

색의 감정:
  (색깔 감정 게임 참고)
```

## 칸딘스키 원칙

```
바실리 칸딘스키 (추상 미술의 선구자):
  "예술은 내적 필연성을 표현한다"
  색과 형태가 직접 감정에 작용한다고 주장

점:
  정적·집중·시작

선:
  점의 움직임
  내면의 소리

면:
  선의 움직임
  2차원적 생명
```

## 심화 모드

```
공동 추상:
  모든 플레이어가 같은 종이에
  순서대로 선·형태 추가
  → 집단 창작 체험

음악 추상:
  음악 들으며 즉흥 표현
  → 공감각적 표현

반응 추상:
  상대방 그림에 반응하는 그림
  → 시각적 대화
```

## 교육적 효과

```
추상적 사고:
  구체 없이 본질 표현
  → 추상화 능력

감정 시각화:
  내면을 형태로 표현
  → 정서 표현 능력

예술 이해:
  추상 미술이 왜 예술인지 체험
  → 미술 감상 폭 확장
```

## 게임 후 토론

```
"왜 추상 미술이 나타났을까요?
 사진기 발명과 어떤 관계가 있을까요?"

"추상 미술은 예술인가요?
 아무나 할 수 있는 것 아닌가요?"
```
