# 숨겨진 역할 (Hidden Role)

> 플레이어 중 일부는 비밀 역할을 가지고 있습니다.
> 누가 아군이고 누가 적군인지 추론하면서
> 자신의 팀 목표를 달성하는
> 사회적 추론 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 4-8명 |
| 소요 시간 | 25-35분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 사회적 추론, 심리 분석, 논증, 협력과 경쟁 |

---

## 게임 목표

**시민 팀**: 숨어있는 방해꾼을 모두 찾아내면 승리.
**방해꾼 팀**: 시민들의 중요 임무를 3번 방해하면 승리.

---

## 준비물

- 역할 카드 (시민·방해꾼·특수 역할)
- 임무 카드 30장
- 투표 토큰 (찬성·반대 각 플레이어당 1개)
- 증거 토큰 20개
- 방해 토큰 5개
- 게임 진행 보드
- 비밀 메모지

### 역할 카드 구성

```
플레이어 수별 역할 배분:

4명: 시민 3명 + 방해꾼 1명
5명: 시민 3명 + 방해꾼 1명 + 탐정 1명
6명: 시민 4명 + 방해꾼 2명
7명: 시민 4명 + 방해꾼 2명 + 탐정 1명
8명: 시민 5명 + 방해꾼 2명 + 탐정 1명
```

### 역할 설명

```
시민 (Citizen):
  팀: 시민 팀
  능력: 임무 수행, 방해꾼 의심 및 투표
  목표: 방해꾼을 모두 추방

방해꾼 (Saboteur):
  팀: 방해꾼 팀
  능력: 임무 방해, 시민으로 위장
  목표: 임무 3번 방해

탐정 (Detective):
  팀: 시민 팀
  능력: 매 라운드 1명의 역할 확인 가능
  목표: 방해꾼 추방 지원
  제약: 탐정 정체를 밝히면 능력 상실

내부고발자 (Whistleblower):
  팀: 시민 팀 (특수)
  능력: 방해꾼 1명의 정체를 알고 시작
  목표: 그 방해꾼을 추방
  제약: 직접 지목 불가, 간접적 암시만 가능
```

---

## 게임 준비

### 1. 역할 배분

역할 카드를 섞어서 각 플레이어에게 1장씩 나눠줍니다.
자신의 역할 카드를 비밀로 확인합니다.

### 2. 방해꾼 확인 (야간 단계)

모든 플레이어가 눈을 감습니다.

**방해꾼 확인:**
진행자: "방해꾼은 눈을 뜨세요."
방해꾼들이 눈을 뜨고 서로를 확인합니다.
진행자: "방해꾼은 눈을 감으세요."

**탐정 확인:** (탐정 역할 포함 시)
진행자: "탐정은 눈을 뜨세요."
탐정이 눈을 뜨고 한 명을 가리킵니다.
진행자가 방해꾼 여부를 엄지 위/아래로 표시합니다.
진행자: "탐정은 눈을 감으세요."

모든 플레이어가 눈을 뜹니다.

### 3. 게임 보드 설정

```
게임 진행 보드:

임무 결과: [  ] [  ] [  ] [  ] [  ]
           1라  2라  3라  4라  5라

방해꾼 방해 횟수: [  ] [  ] [  ]
시민 성공 횟수:   [  ] [  ] [  ]

추방된 플레이어:
  ___ / ___ / ___ / ___
```

## 게임 규칙

### 게임 진행 순서

각 라운드는 3단계로 진행됩니다.

```
1단계: 토론 (3분)
2단계: 임무팀 선발 투표
3단계: 임무 수행
```

### 1단계: 토론

모든 플레이어가 자유롭게 토론합니다.

**토론 내용:**
- 누가 방해꾼인지 의심
- 자신의 결백 주장
- 지난 라운드 임무 결과 분석
- 임무팀 구성 제안

**토론 규칙:**
- 자신의 역할 카드를 보여줄 수 없음
- 거짓말 허용 (방해꾼의 기본 전략)
- 다른 플레이어의 말을 끊을 수 없음
- 진행자는 토론에 참여하지 않음

### 2단계: 임무팀 선발

**팀장 선정:**
첫 라운드는 무작위로 팀장을 정합니다.
이후 시계 방향으로 팀장이 넘어갑니다.

**팀원 지명:**
팀장이 임무팀 인원을 지명합니다.

```
라운드별 임무팀 인원:

4-5명 게임: 2명 / 3명 / 2명 / 3명 / 3명
6-7명 게임: 2명 / 3명 / 4명 / 3명 / 4명
8명 게임:   3명 / 4명 / 4명 / 5명 / 5명
```

**승인 투표:**
모든 플레이어가 동시에 찬성/반대 토큰을 냅니다.
과반수 찬성이면 임무팀 확정.
과반수 반대이면 팀장이 다음 플레이어로 넘어가고
새 팀장이 다시 팀원을 지명합니다.

```
투표 거부 3회 연속:
→ 방해꾼 팀 1점 획득 (임무 방해 성공으로 처리)
```

### 3단계: 임무 수행

임무팀으로 선발된 플레이어들만 참여합니다.

**임무 카드 제출:**
임무팀 플레이어 각자가 비밀리에
성공 카드 또는 방해 카드 1장을 냅니다.

```
시민: 반드시 성공 카드를 내야 함
방해꾼: 성공 또는 방해 카드 선택 가능
```

**결과 공개:**
카드를 섞어서 한 장씩 공개합니다.

```
방해 카드 0장: 임무 성공 → 시민 팀 1점
방해 카드 1장 이상: 임무 실패 → 방해꾼 팀 1점
```

**예외 규칙 (4라운드):**
4라운드 임무는 방해 카드가 2장 이상이어야 실패.
(방해꾼 1명이 방해해도 성공)

### 추방 투표

임무가 2번 연속 실패하거나
플레이어 과반수가 요청하면 추방 투표를 진행합니다.

**추방 후보 지명:**
각 플레이어가 의심 가는 1명을 지목합니다.
가장 많이 지목받은 플레이어가 후보가 됩니다.

**추방 투표:**
후보를 추방할지 투표합니다.
과반수 찬성이면 추방.

**추방 결과:**
```
방해꾼 추방: 시민 팀 유리
시민 추방: 방해꾼 팀 유리 + 방해꾼 팀 1점
```

추방된 플레이어는 게임에서 제외되지 않고
발언만 금지됩니다. (관찰자로 남음)

---

## 승패 조건

### 시민 팀 승리

다음 중 하나 달성:
①임무 3번 성공.
②방해꾼 전원 추방.

### 방해꾼 팀 승리

다음 중 하나 달성:
①임무 3번 방해.
②시민이 너무 많이 추방되어
  방해꾼 수 ≥ 남은 시민 수.

## 전략 가이드

### 시민 전략

**1. 행동 패턴 관찰**
방해꾼은 자연스럽지 않은 행동을 합니다.
- 과도하게 결백을 주장
- 특정 플레이어를 집중 공격
- 임무 실패 후 침묵

**2. 임무 결과 분석**
임무 실패 시 팀원들의 반응을 관찰합니다.
방해꾼은 의외로 침착할 수 있습니다.

**3. 소수 의견 경청**
방해꾼은 다수의 의견을 따르는 경향이 있습니다.
소수 의견을 주장하는 사람이 오히려 시민일 수 있습니다.

**4. 증거 기반 주장**
"느낌"이 아닌 구체적 근거로 주장하세요.
```
나쁜 주장: "지호가 왠지 수상해요."
좋은 주장: "지호는 2라운드에서 임무 실패 후
          즉시 다른 사람을 의심하기 시작했어요."
```

### 방해꾼 전략

**1. 자연스러운 위장**
처음 라운드는 성공 카드를 내서 신뢰를 쌓으세요.

**2. 희생양 만들기**
의심을 받기 전에 먼저 다른 시민을 의심하세요.

**3. 타이밍 선택**
중요한 라운드(4·5라운드)에 방해하는 것이 효과적입니다.

**4. 공동 방해 조율**
방해꾼 2명일 때 한 명은 방해, 한 명은 성공 카드를 내서
의심을 분산하는 전략도 유효합니다.

## 특수 시나리오

### 시나리오 #01: 연구소 침투
```
배경: 비밀 연구소에서 중요한 데이터가 유출되고 있습니다.
      시민들은 보안 시스템을 복구해야 합니다.
      방해꾼들은 데이터 유출을 계속 진행합니다.

특수 규칙:
  3라운드부터 탐정의 능력이 2회로 증가.
  방해 성공 시 다음 임무팀에 방해꾼 1명 강제 포함.
```

### 시나리오 #02: 도시 위기
```
배경: 도시에 비상사태가 발생했습니다.
      시민들은 위기를 해결해야 하지만
      방해꾼들이 복구를 방해합니다.

특수 규칙:
  임무 성공 시 증거 토큰 1개 획득.
  증거 토큰 3개로 방해꾼 1명 강제 추방 가능.
```

### 시나리오 #03: 우주 기지
```
배경: 우주 기지에서 산소 공급 시스템이 고장났습니다.
      팀이 협력해서 수리해야 하지만
      일부 대원이 시스템 수리를 방해하고 있습니다.

특수 규칙:
  5라운드 이내에 임무 5번 완료해야 시민 팀 승리.
  시간 초과 시 방해꾼 팀 자동 승리.
```

## 변형 규칙

### 단순 숨겨진 역할
- 특수 역할 없음 (시민·방해꾼만)
- 토론 시간 2분으로 단축
- 초보자 입문용

### 이중 방해꾼
- 방해꾼 중 1명이 '수장' 역할 추가
- 수장이 추방되면 방해꾼 팀 즉시 패배
- 시민 팀은 수장을 찾는 추가 목표

### 협력 숨겨진 역할
- 방해꾼 없음
- 모든 플레이어가 시민
- 외부 '위협 카드'가 임무를 방해
- 순수 협력 버전

---

## 교육적 활용

### 논리 연계
- **귀납적 추론**: 행동 패턴에서 역할 추론
- **논증**: 근거 기반 주장과 반박
- **소거법**: 불가능한 역할 제거

### 사회 연계
- **심리 분석**: 타인의 행동 동기 파악
- **소통**: 설득력 있는 주장
- **협력**: 팀 내 정보 공유와 신뢰

### 수업 활용 방법
1. 규칙 설명 + 시범 게임 (10분)
2. 5인 기본 게임 1판 (30분)
3. "방해꾼을 어떻게 찾았나?" 복기 (10분)
4. 특수 시나리오 도전 (30분)
