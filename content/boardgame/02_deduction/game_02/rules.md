# 증인 심문 (Witness)

> 각 플레이어가 서로 다른 정보를 가지고 있습니다.
> 제한된 질문과 답변으로 정보를 공유하고
> 함께 사건의 진실을 밝혀내는
> 협력형 추론 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-6명 |
| 소요 시간 | 20-30분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 정보 공유, 논리적 추론, 소통 능력, 협력 |

---

## 게임 목표

각 플레이어는 사건의 일부 정보만 알고 있습니다.
제한된 질문을 통해 정보를 공유하고
**사건 카드의 4가지 핵심 요소**
(용의자·장소·시간·도구)를 모두 알아내면
팀 전체가 승리합니다.

---

## 준비물

- 사건 카드 30장
- 증인 카드 (플레이어별 비밀 정보)
- 용의자 카드 12장
- 장소 카드 12장
- 시간 카드 8장
- 도구 카드 10장
- 질문 토큰 (플레이어당 3개)
- 추론 기록지
- 모래시계 (30초)

### 카드 예시

```
용의자 카드:
  김민준 / 이서연 / 박지호 / 최유나
  정승현 / 한소희 / 오태양 / 윤지수
  임채원 / 강다은 / 신준서 / 백아린

장소 카드:
  도서관 / 실험실 / 옥상 / 지하실
  카페 / 체육관 / 미술실 / 음악실
  교장실 / 창고 / 보건실 / 컴퓨터실

시간 카드:
  오전 8시 / 오전 10시 / 정오 / 오후 2시
  오후 4시 / 오후 6시 / 오후 8시 / 자정

도구 카드:
  열쇠 / 노트 / 카메라 / 망치
  독약 / 밧줄 / 촛대 / 칼
  독침 / USB
```

---

## 게임 준비

### 1. 사건 카드 선택

사건 카드 1장을 무작위로 뽑아
뒤집어서 게임 보드 중앙에 놓습니다.
(아직 아무도 보지 않음)

```
사건 카드 예시 (뒷면):
┌─────────────────────┐
│     ??? 사건        │
│  용의자: _______    │
│  장소:   _______    │
│  시간:   _______    │
│  도구:   _______    │
└─────────────────────┘
```

### 2. 증인 카드 배분

각 플레이어에게 증인 카드를 나눠줍니다.
증인 카드에는 사건의 일부 정보가 적혀 있습니다.

```
증인 카드 예시 (3인 게임):

플레이어 1의 카드:
  ┌─────────────────┐
  │ 나는 보았다...  │
  │ 용의자: 김민준  │
  │ 장소:   도서관  │
  │ 시간:   ???     │
  │ 도구:   ???     │
  └─────────────────┘

플레이어 2의 카드:
  ┌─────────────────┐
  │ 나는 보았다...  │
  │ 용의자: ???     │
  │ 장소:   ???     │
  │ 시간:   오후 4시│
  │ 도구:   열쇠    │
  └─────────────────┘

플레이어 3의 카드:
  ┌─────────────────┐
  │ 나는 보았다...  │
  │ 용의자: 김민준  │
  │ 장소:   ???     │
  │ 시간:   오후 4시│
  │ 도구:   ???     │
  └─────────────────┘
```

**배분 원칙:**
- 각 정보는 최소 1명, 최대 플레이어 수-1명이 알고 있음
- 어떤 정보는 여러 명이 공유
- 어떤 정보는 1명만 알고 있음
- 모든 정보를 아는 사람은 없음

### 3. 질문 토큰 배분

각 플레이어에게 질문 토큰 3개를 나눠줍니다.

---

## 게임 규칙

### 라운드 진행

게임은 **최대 5라운드**로 진행됩니다.
각 라운드에서 플레이어들은 번갈아 질문합니다.

### 질문 단계

**①질문자 선택:**
자신의 턴에 다른 플레이어 1명에게 질문할 수 있습니다.
질문 토큰 1개를 사용합니다.

**②질문 유형:**
다음 중 하나의 질문만 허용됩니다.

```
허용 질문:
  "당신의 카드에 [용의자/장소/시간/도구] 정보가 있나요?"
  → 예/아니오로만 답변

  "당신의 카드에서 알고 있는 것을 하나 알려주세요."
  → 카드에서 알고 있는 정보 1가지 공개

금지 질문:
  "당신의 카드를 보여주세요." (X)
  "모든 정보를 말해주세요." (X)
  "누가 범인인 것 같나요?" (X)
```

**③답변:**
질문받은 플레이어는 30초 이내에 정직하게 답변합니다.
모래시계로 시간을 측정합니다.

**④기록:**
모든 플레이어가 공개된 정보를 추론 기록지에 기록합니다.

### 추론 단계

질문 라운드가 끝나면 팀이 함께 추론합니다.
- 지금까지 공개된 정보 종합
- 논리적 추론으로 미공개 정보 추정
- 최종 답안 합의

### 최종 답안 제출

팀이 합의한 4가지 요소를 기록지에 작성합니다.

```
최종 답안:
  용의자: ___________
  장소:   ___________
  시간:   ___________
  도구:   ___________
```

사건 카드를 뒤집어 정답을 확인합니다.

### 승패 판정

```
4가지 모두 정확: 완벽한 해결! (+10점)
3가지 정확:     훌륭한 추론! (+6점)
2가지 정확:     보통 추론  (+3점)
1가지 이하:     미해결     (+0점)

사용한 라운드가 적을수록 보너스:
  1라운드: +5점
  2라운드: +4점
  3라운드: +3점
  4라운드: +2점
  5라운드: +1점
```

## 전략 가이드

### 정보 공유 전략

**1. 중복 확인**
여러 명이 같은 정보를 알고 있을 수 있습니다.
"당신도 용의자를 아나요?"라고 물어
중복 여부를 파악하세요.

**2. 빈 정보 우선**
아직 아무도 공개하지 않은 정보부터
물어보는 것이 효율적입니다.

**3. 교차 검증**
두 플레이어가 같은 정보를 알고 있다면
신뢰도가 높아집니다.

### 질문 효율화

**나쁜 질문 패턴:**
```
질문: "당신의 카드에서 하나 알려주세요."
답변: "저는 장소를 알고 있어요. 도서관입니다."
→ 이미 알고 있던 정보일 수 있음 (토큰 낭비)
```

**좋은 질문 패턴:**
```
질문: "당신의 카드에 시간 정보가 있나요?"
답변: "네, 있어요."
질문: "그 시간을 알려주세요."
답변: "오후 4시입니다."
→ 확인 후 구체적 정보 획득 (효율적)
```

### 논리적 추론

**소거법 활용:**
```
알고 있는 정보:
  플레이어 1: 용의자=김민준, 장소=도서관
  플레이어 2: 시간=오후4시, 도구=열쇠
  플레이어 3: 용의자=김민준, 시간=오후4시

추론:
  용의자: 3명 중 2명이 김민준 → 높은 신뢰도
  장소: 1명만 알고 있음 → 도서관 가능성
  시간: 2명이 오후4시 → 높은 신뢰도
  도구: 1명만 알고 있음 → 열쇠 가능성
```

## 사건 카드 예시

### 사건 #01
```
제목: 도서관의 비밀
용의자: 이서연
장소:   도서관
시간:   오후 4시
도구:   열쇠

배경 이야기:
방과 후 도서관이 잠겨 있어야 했는데
누군가 열쇠를 가지고 몰래 들어갔습니다.
```

### 사건 #02
```
제목: 실험실 사고
용의자: 박지호
장소:   실험실
시간:   오전 10시
도구:   독약

배경 이야기:
실험실에서 원인 불명의 사고가 발생했습니다.
누군가 실험 도구를 무단으로 사용한 것 같습니다.
```

### 사건 #03
```
제목: 옥상의 목격자
용의자: 최유나
장소:   옥상
시간:   자정
도구:   카메라

배경 이야기:
자정에 학교 옥상에서 누군가가
카메라로 무언가를 촬영하는 것이 목격됐습니다.
```

## 변형 규칙

### 경쟁 증인 심문
- 협력 대신 경쟁
- 각 플레이어가 혼자 답안 제출
- 가장 많이 맞힌 플레이어 승리
- 정보 공유 전략이 더 복잡해짐

### 이중 첩자 증인 심문
- 1명의 플레이어가 잘못된 정보 카드 보유
- 다른 플레이어들은 누가 첩자인지 모름
- 잘못된 정보를 걸러내는 추론 능력 필요

### 고급 증인 심문
- 질문 토큰 2개로 감소
- 사건 요소 5가지로 증가 (범행 동기 추가)
- 더 정밀한 정보 공유 전략 필요

### 침묵의 증인
- 예/아니오 질문만 허용
- 구체적 정보 직접 공개 금지
- 더 많은 논리적 추론 필요

---

## 교육적 활용

### 논리 연계
- **연역적 추론**: 주어진 정보에서 결론 도출
- **귀납적 추론**: 여러 단서에서 패턴 발견
- **소거법**: 불가능한 경우 제거

### 소통 연계
- **효율적 소통**: 제한된 질문으로 최대 정보
- **경청**: 다른 플레이어의 답변 주의 깊게 듣기
- **협력**: 팀 전체의 정보 종합

### 수업 활용 방법
1. 교사가 증인 역할, 학생들이 질문 (10분)
2. 3인 게임 1판 (25분)
3. "더 적은 질문으로 해결할 수 있었나?" 토론 (10분)
4. 이중 첩자 변형 도전 (15분)
