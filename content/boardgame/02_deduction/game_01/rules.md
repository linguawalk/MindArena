# 암호 해독가 (Code Breaker)

> 상대방이 만든 비밀 암호를
> 논리적 추론과 소거법으로 해독하는 게임입니다.
> 매 시도마다 얻는 단서를 분석해서
> 가장 적은 시도로 암호를 맞히세요.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2명 |
| 소요 시간 | 15-20분 |
| 권장 연령 | 9세 이상 |
| 난이도 | ⭐⭐ (보통) |
| 교육 분야 | 논리적 추론, 소거법, 정보 분석, 가설 검증 |

---

## 게임 목표

상대방이 설정한 **4자리 비밀 암호**를
**10번 이내의 시도**로 먼저 해독한 플레이어가 승리합니다.

---

## 준비물

- 암호 보드 2개 (플레이어별 1개)
- 색깔 코드 핀 (6가지 색: 빨강·주황·노랑·초록·파랑·보라) 각 8개
- 단서 핀 (검정·흰색) 각 20개
- 차단막 1개 (상대방 암호 가리기)
- 추론 기록지

### 암호 보드 구조

```
[차단막]
[ 비밀 암호 ]  ← 상대방이 보지 못하게 설정
━━━━━━━━━━━━━━━━━━━━━━━━━━━
시도 10: [○][○][○][○]  단서: [●][●][ ][ ]
시도  9: [○][○][○][○]  단서: [●][ ][ ][ ]
시도  8: [○][○][○][○]  단서: [●][●][●][ ]
시도  7: [○][○][○][○]  단서: [●][●][ ][ ]
시도  6: [○][○][○][○]  단서: [●][●][●][ ]
시도  5: [○][○][○][○]  단서: [●][●][●][●] ← 해독!
시도  4: [○][○][○][○]  단서: [●][●][ ][ ]
시도  3: [○][○][○][○]  단서: [●][ ][ ][ ]
시도  2: [○][○][○][○]  단서: [●][○][ ][ ]
시도  1: [○][○][○][○]  단서: [ ][ ][ ][ ]
```

---

## 게임 준비

### 1. 비밀 암호 설정

각 플레이어는 차단막 뒤에서 몰래
**4자리 비밀 암호**를 설정합니다.

**설정 규칙:**
- 6가지 색 중 4가지를 선택
- 같은 색을 중복 사용 가능
- 순서가 중요 (빨파초노 ≠ 파빨초노)

```
비밀 암호 예시:
  [빨강][빨강][파랑][초록]
  [노랑][보라][노랑][빨강]
  [초록][초록][초록][초록]  ← 중복 허용
```

### 2. 선공 결정

가위바위보로 선공을 정합니다.
선공 플레이어가 먼저 추측을 시작합니다.

---

## 게임 규칙

### 턴 진행

두 플레이어가 번갈아 가며 진행합니다.

**①추측 선언:**
4가지 색 핀을 순서대로 보드에 놓습니다.

```
추측 예시:
  [빨강][파랑][노랑][초록]
```

**②단서 제공:**
상대방은 추측 결과에 따라 단서 핀을 제공합니다.

```
단서 핀 규칙:
  검정 핀 (●): 색과 위치가 모두 정확
  흰색 핀 (○): 색은 맞지만 위치가 틀림
  빈칸 ( ): 해당 색은 암호에 없음
```

**단서 핀 제공 예시:**
```
비밀 암호: [빨강][파랑][초록][노랑]
추측:      [빨강][초록][파랑][보라]

결과:
  빨강 → 위치 1 정확 → 검정 핀 ●
  초록 → 위치 3에 있어야 하는데 위치 2 → 흰색 핀 ○
  파랑 → 위치 2에 있어야 하는데 위치 3 → 흰색 핀 ○
  보라 → 암호에 없음 → 빈칸

단서: [●][○][○][ ]
(단서 핀의 순서는 추측 순서와 무관하게 검정 먼저)
```

**③기록:**
추측과 단서를 보드에 기록합니다.

**④해독 성공:**
단서가 모두 검정 핀(●●●●)이면 해독 성공!

### 게임 종료

**승리 조건:**
10번 이내에 상대방의 암호를 해독하면 승리.

**패배 조건:**
10번 시도 후에도 해독하지 못하면 패배.

**동시 해독:**
같은 턴에 두 플레이어 모두 해독하면
더 적은 시도로 해독한 플레이어가 승리.
시도 횟수도 같으면 무승부.

---

## 논리적 추론 가이드

### 소거법 기본

단서를 활용해 가능한 경우의 수를 줄여나갑니다.

**예시 추론 과정:**

```
시도 1: [빨강][빨강][빨강][빨강]
단서:   [ ][ ][ ][ ]
→ 빨강은 암호에 없음. 완전 소거.

시도 2: [파랑][파랑][파랑][파랑]
단서:   [●][ ][ ][ ]
→ 파랑이 정확히 1개 있음.
  어느 위치인지는 아직 모름.

시도 3: [파랑][노랑][노랑][노랑]
단서:   [●][●][ ][ ]
→ 파랑 또는 노랑이 각각 1개씩 맞음.
  시도 2와 비교: 노랑이 정확히 1개 있음.
  파랑의 위치 1이 맞을 가능성 높음.
```

### 체계적 추론 전략

**1단계: 색 확인**
처음 몇 번의 시도는 어떤 색이 암호에 있는지 파악합니다.

```
추천 초기 시도:
  시도 1: [빨강][파랑][초록][노랑]
  시도 2: [보라][주황][빨강][파랑]
→ 두 시도로 6가지 색의 존재 여부 파악 가능
```

**2단계: 위치 확인**
어떤 색이 있는지 파악한 후 정확한 위치를 찾습니다.

```
색 A가 있다고 확인됐을 때:
  시도 N:   [A][?][?][?]  → 위치 1 테스트
  시도 N+1: [?][A][?][?]  → 위치 2 테스트
  ...
```

**3단계: 조합 확정**
위치가 확인된 색을 고정하고 나머지를 추론합니다.

### 고급 추론 기법

**정보 최대화 추측:**
매 시도가 최대한 많은 정보를 제공하도록 구성합니다.

```
나쁜 추측 (이미 아는 정보 반복):
  시도 5: [빨강][빨강][파랑][초록]
  (빨강이 없다는 것을 알고 있는데 사용)

좋은 추측 (새로운 정보 획득):
  시도 5: [노랑][보라][파랑][초록]
  (아직 확인 안 된 노랑·보라 포함)
```

**교차 검증:**
두 시도의 결과를 비교해서 특정 색의 위치를 확정합니다.

```
시도 3: [A][B][C][D] → 단서 ●●○○
시도 4: [A][C][B][D] → 단서 ●○●○

비교:
  위치 1(A): 둘 다 ● → A의 위치 1 확정!
  위치 4(D): 둘 다 ○ → D는 있지만 위치 4 아님 확정
```

## 난이도 변형

### 초급 암호 해독가
- 암호 길이: 3자리
- 색 종류: 4가지
- 중복 없음
- 시도 횟수: 8번
- 권장 연령: 7세 이상

### 표준 암호 해독가
- 암호 길이: 4자리
- 색 종류: 6가지
- 중복 허용
- 시도 횟수: 10번

### 고급 암호 해독가
- 암호 길이: 5자리
- 색 종류: 8가지
- 중복 허용
- 시도 횟수: 12번
- 권장 연령: 12세 이상

### 수학 암호 해독가
- 색 대신 숫자(1-6) 사용
- 암호: 4자리 숫자
- 수학적 단서 추가:
  "합이 짝수", "첫 두 자리의 합 = 7" 등
- 수 감각 + 논리 결합

---

## 최적 해독 전략 (수학적 분석)

### 가능한 경우의 수

```
표준 버전 (6색, 4자리, 중복 허용):
  6^4 = 1,296가지

최적 첫 추측:
  [빨강][빨강][파랑][파랑]
  (두 가지 색을 각각 두 번 사용)
  → 평균 4.340번 만에 해독 가능

Donald Knuth 알고리즘:
  항상 5번 이내에 해독 가능한 전략 존재
  (1977년 수학적으로 증명)
```

### 정보 이론 관점

```
각 시도로 얻을 수 있는 최대 정보:
  가능한 단서 결과: 14가지
  (검정/흰색 핀의 조합)
  이상적 추측 = 각 결과가 균등하게 나오는 추측
```

---

## 교육적 활용

### 수학 연계
- **경우의 수**: 가능한 암호 조합 계산
- **확률**: 각 추측의 정보량 분석
- **집합론**: 가능한 암호 집합의 축소

### 사고력 연계
- **연역적 추론**: 단서에서 결론 도출
- **소거법**: 불가능한 경우 제거
- **가설 검증**: 추측과 단서의 일치 확인

### 수업 활용 방법
1. 교사가 암호 설정, 학생들이 함께 해독 (10분)
2. 2인 대전 2판 (20분)
3. "몇 번이면 반드시 해독할 수 있을까?" 토론 (10분)
4. 수학 버전 도전 (10분)
