# 논리 격자 (Logic Grid)

> 주어진 단서들을 논리적으로 분석해서
> 격자 표에 정확한 답을 채워 넣는
> 순수 논리 추론 게임입니다.
> 연역적 사고의 정수를 경험하세요.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 1-4명 |
| 소요 시간 | 15-25분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 연역적 추론, 소거법, 논리 분석, 집중력 |

---

## 게임 목표

주어진 단서 카드들을 분석해서
**논리 격자 표를 완전히 채우면** 승리합니다.

1인 플레이: 가장 짧은 시간 안에 완성.
다인 플레이: 가장 먼저 완성한 플레이어 승리.

---

## 준비물

- 논리 격자 보드 (4×4 기본)
- 단서 카드 덱 (난이도별)
- 마킹 토큰 (O·X 각 32개)
- 정답 카드 (봉투에 밀봉)
- 타이머
- 추론 메모지


## 게임 준비

### 1. 난이도 선택

```
초급 (⭐):
  카테고리: 2가지
  항목: 3가지씩
  단서: 4-5개
  예: 사람 3명 × 직업 3가지

중급 (⭐⭐):
  카테고리: 3가지
  항목: 4가지씩
  단서: 6-8개
  예: 사람 4명 × 직업 4가지 × 취미 4가지

고급 (⭐⭐⭐):
  카테고리: 4가지
  항목: 4가지씩
  단서: 8-10개
  예: 사람 4명 × 직업 × 취미 × 색깔

최고급 (⭐⭐⭐⭐):
  카테고리: 5가지
  항목: 5가지씩
  단서: 12-15개
```

### 2. 단서 카드 배분

선택한 난이도의 단서 카드를 준비합니다.

**1인 플레이:** 모든 단서 카드를 펼쳐놓고 시작.
**다인 플레이:** 단서 카드를 같은 수로 나눠 가짐.

### 3. 정답 카드 보관

정답 카드를 봉투에 넣어 보이지 않게 보관합니다.

---

## 단서 유형

### 직접 단서
```
유형 A — 직접 연결:
  "민준은 의사입니다."
  → 민준-의사 칸에 O 표시
  → 민준-(다른 직업) 칸에 X 표시
  → (다른 사람)-의사 칸에 X 표시

유형 B — 직접 부정:
  "서연은 요리사가 아닙니다."
  → 서연-요리사 칸에 X 표시
```

### 관계 단서
```
유형 C — 같음:
  "지호와 같은 직업을 가진 사람은 파랑을 좋아합니다."
  → 지호의 직업 = 파랑을 좋아하는 사람의 직업

유형 D — 다름:
  "유나는 수영을 좋아하는 사람과 직업이 다릅니다."
  → 유나의 직업 ≠ 수영을 좋아하는 사람의 직업

유형 E — 이웃:
  "민준과 서연의 취미는 알파벳 순서로 인접합니다."
  → (등산-독서) 또는 (수영-음악) 등
```

### 조건 단서
```
유형 F — 조건:
  "의사라면 반드시 독서를 좋아합니다."
  → 의사인 사람은 독서 취미

유형 G — 비교:
  "화가는 선생님보다 나이가 많습니다."
  → 나이 순서에 활용

유형 H — 개수:
  "빨강을 좋아하는 사람은 3명입니다."
  → 특정 카테고리의 개수 정보
```

## 게임 규칙

### 1인 플레이

**시작:**
타이머를 시작하고 단서 카드를 분석합니다.

**진행:**
①단서 카드를 읽고 논리 격자에 O/X 표시.
②새로운 O가 생기면 연쇄 추론 적용.
③모든 칸이 채워지면 완성 선언.

**검증:**
정답 카드와 비교해서 정확성 확인.

**점수:**
```
완벽 정답 + 소요 시간으로 점수 계산:
  5분 이내: 100점
  10분 이내: 80점
  15분 이내: 60점
  20분 이내: 40점
  오류 1개당: -10점
```

### 다인 플레이

**정보 공개 단계:**
각 플레이어는 자신의 단서 카드를 1장씩 공개합니다.
공개된 단서는 모든 플레이어가 활용합니다.

**추론 단계:**
각 플레이어가 개별적으로 격자를 채웁니다.

**완성 선언:**
격자를 먼저 완성한 플레이어가 "완성!"을 외칩니다.

**검증:**
정답 카드와 비교합니다.
오류가 있으면 해당 플레이어는 탈락.
게임을 계속 진행합니다.

**승리:**
정확하게 완성한 첫 번째 플레이어가 승리합니다.

## 추론 방법 가이드

### 기본 추론 절차

**Step 1: 직접 단서 적용**
```
단서: "민준은 의사입니다."

적용:
  민준-의사: O ✓
  민준-선생: X
  민준-요리사: X
  민준-화가: X
  서연-의사: X
  지호-의사: X
  유나-의사: X
```

**Step 2: 연쇄 추론**
```
민준-의사: O (확정)
단서: "의사는 독서를 좋아합니다."

연쇄 추론:
  → 민준은 독서를 좋아함
  민준-독서: O ✓
  민준-등산: X
  민준-수영: X
  민준-음악: X
  서연-독서: X
  지호-독서: X
  유나-독서: X
```

**Step 3: 소거 추론**
```
현재 상태:
  서연-선생: ?
  서연-요리사: ?
  서연-화가: ?
  지호-선생: X (다른 단서로 확정)
  지호-요리사: ?
  지호-화가: ?
  유나-선생: ?
  유나-요리사: X (다른 단서로 확정)
  유나-화가: ?

선생님 열에서:
  민준-선생: X
  지호-선생: X
  유나-선생: ?
  서연-선생: ?

아직 2명 남음 → 추가 단서 필요
```

**Step 4: 완전 소거**
```
요리사 열에서:
  민준-요리사: X
  서연-요리사: ?
  지호-요리사: ?
  유나-요리사: X

단서: "지호는 요리사가 아닙니다."
  → 지호-요리사: X
  → 서연-요리사: O (나머지 1명!)
  → 서연-선생: X
  → 서연-화가: X

연쇄:
  → 유나-선생: ? 또는 화가?
  → 지호-화가: ?
```

## 예시 퍼즐 (초급)

### 퍼즐 #01 — "세 친구의 직업"

```
등장인물:
  사람: 민준, 서연, 지호
  직업: 의사, 선생님, 화가

단서:
  1. 민준은 의사가 아닙니다.
  2. 서연은 화가입니다.
  3. 지호는 선생님이 아닙니다.

풀이:
  단서 2: 서연-화가: O
         → 민준-화가: X, 지호-화가: X

  단서 1: 민준-의사: X
         → 민준은 선생님만 가능
         민준-선생님: O

  단서 3: 지호-선생님: X
         → 지호는 의사만 가능
         지호-의사: O

정답:
  민준 = 선생님
  서연 = 화가
  지호 = 의사
```

## 예시 퍼즐 (중급)

### 퍼즐 #02 — "네 친구의 직업과 취미"

```
등장인물:
  사람: 민준, 서연, 지호, 유나
  직업: 의사, 선생님, 요리사, 화가
  취미: 독서, 등산, 수영, 음악

단서:
  1. 민준은 의사입니다.
  2. 서연은 수영을 좋아합니다.
  3. 화가는 음악을 좋아합니다.
  4. 요리사는 독서를 좋아하지 않습니다.
  5. 유나는 요리사입니다.
  6. 지호는 등산을 좋아하지 않습니다.

풀이 과정:
  단서 1: 민준-의사: O
  단서 5: 유나-요리사: O
  → 서연, 지호는 선생님 또는 화가

  단서 2: 서연-수영: O
  단서 3: 화가-음악: O
  → 서연은 수영이므로 화가가 아님 (화가는 음악)
  → 서연-선생님: O
  → 지호-화가: O

  단서 3 연쇄: 지호-음악: O
  단서 6: 지호-등산: X
  → 지호-음악: O (이미 확정)

  나머지 취미 배분:
  민준: 독서 또는 등산
  유나: 독서 또는 등산
  단서 4: 요리사(유나)-독서: X
  → 유나-등산: O
  → 민준-독서: O

정답:
  민준 = 의사, 독서
  서연 = 선생님, 수영
  지호 = 화가, 음악
  유나 = 요리사, 등산
```

## 나만의 퍼즐 만들기

직접 논리 격자 퍼즐을 만들 수 있습니다.

**제작 순서:**
①카테고리와 항목 결정.
②정답 표 먼저 작성.
③정답에서 역방향으로 단서 추출.
④단서만으로 풀 수 있는지 검증.
⑤불필요한 단서 제거.

**좋은 퍼즐의 조건:**
- 단서가 최소화되어 있음 (군더더기 없음)
- 단일 정답만 존재
- 여러 단계의 추론이 필요
- 직접 단서와 간접 단서가 균형 있게 포함

## 변형 규칙

### 협력 논리 격자
- 단서 카드를 플레이어가 나눠 가짐
- 자신의 단서 카드를 보여줄 수 없음
- 구두로 설명해서 팀이 함께 격자 완성
- 소통 + 논리 결합

### 시간 제한 논리 격자
- 각 단서 적용에 30초 제한
- 시간 초과 시 다음 단서로 넘어감
- 빠른 추론 능력 훈련

### 오류 포함 논리 격자
- 단서 중 1-2개가 거짓
- 어떤 단서가 거짓인지 찾아야 함
- 고급 논리 훈련

## 교육적 활용

### 수학 연계
- **논리학**: 연역적 추론의 기본 구조
- **집합론**: 포함·배제 관계
- **경우의 수**: 가능한 조합 분석

### 사고력 연계
- **분석적 사고**: 단서를 체계적으로 분석
- **체계적 사고**: 격자를 순서대로 채우기
- **집중력**: 복잡한 정보 동시 처리

### 수업 활용 방법
1. 초급 퍼즐로 규칙 익히기 (10분)
2. 중급 퍼즐 개인 도전 (15분)
3. 고급 퍼즐 팀 협력 (15분)
4. 직접 퍼즐 만들기 (20분)
