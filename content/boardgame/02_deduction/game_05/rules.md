# 탐정 노트 (Detective Notes)

> 복잡한 사건 현장을 조사하고
> 여러 단서를 종합해서
> 범인·범행 장소·범행 도구를 밝혀내는
> 고급 추론 보드게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-5명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐⭐ (상) |
| 교육 분야 | 복합 추론, 증거 분석, 가설 수립, 비판적 사고 |

---

## 게임 목표

사건 카드에 숨겨진 **범인·장소·도구** 3가지를
다른 플레이어보다 먼저 정확하게 추리해서
**최종 고발**에 성공한 플레이어가 승리합니다.

---

## 준비물

- 사건 보드 1개
- 용의자 카드 6장
- 장소 카드 6장
- 도구 카드 6장
- 단서 카드 덱 (60장)
- 사건 봉투 1개 (정답 보관)
- 탐정 노트 (플레이어별 1개)
- 행동 토큰 (플레이어당 4개)
- 이동 마커 (플레이어별 1개)
- 증거 토큰 30개

### 카드 구성

```
용의자 카드 (6장):
  김민준 (30세, 회사원)
  이서연 (25세, 학생)
  박지호 (45세, 의사)
  최유나 (38세, 교사)
  정승현 (52세, 사업가)
  한소희 (29세, 예술가)

장소 카드 (6장):
  서재      (1층 북쪽)
  주방      (1층 남쪽)
  정원      (외부 동쪽)
  지하실    (지하)
  침실      (2층 북쪽)
  테라스    (2층 남쪽)

도구 카드 (6장):
  독약      (액체)
  밧줄      (섬유)
  칼        (금속)
  촛대      (금속)
  열쇠      (금속)
  카메라    (전자)
```

### 사건 보드

```
┌─────────────────────────────────────┐
│           저택 평면도               │
│  ┌──────┐    ┌──────┐              │
│  │ 침실  │    │테라스│  [2층]       │
│  └──────┘    └──────┘              │
│  ┌──────┐    ┌──────┐              │
│  │ 서재  │    │ 주방 │  [1층]       │
│  └──────┘    └──────┘              │
│  ┌──────┐    ┌──────┐              │
│  │지하실│    │ 정원 │  [외부]       │
│  └──────┘    └──────┘              │
└─────────────────────────────────────┘

이동 경로:
  인접한 방끼리만 이동 가능
  서재 ↔ 주방 ↔ 정원
  서재 ↔ 침실 ↔ 테라스
  주방 ↔ 테라스
  서재 ↔ 지하실
```

## 게임 준비

### 1. 사건 봉투 준비

용의자·장소·도구 카드를 각각 섞어서
각 더미에서 1장씩 뽑아 사건 봉투에 넣습니다.
**아무도 보지 않습니다.**

### 2. 나머지 카드 배분

봉투에 넣지 않은 카드들(용의자 5장·장소 5장·도구 5장)을
모두 섞어서 플레이어들에게 균등하게 나눠줍니다.
남는 카드는 공개해서 보드 옆에 놓습니다.

```
5인 게임 예시:
  15장 카드 ÷ 5명 = 3장씩
  (나머지 0장)

3인 게임 예시:
  15장 카드 ÷ 3명 = 5장씩
  (나머지 0장)

2인 게임 예시:
  15장 카드 ÷ 2명 = 7장씩
  나머지 1장 공개
```

### 3. 탐정 노트 배분

각 플레이어에게 탐정 노트 1개를 줍니다.
자신이 가진 카드에 체크 표시를 합니다.
(자신이 가진 카드 = 봉투 안에 없는 것이 확실)

### 4. 시작 위치

모든 플레이어는 서재에서 시작합니다.

## 게임 규칙

### 턴 진행

플레이어는 번갈아 가며 자신의 턴을 진행합니다.
각 턴에 행동 토큰 2개를 사용합니다.

### 가능한 행동

**①이동 (토큰 1개)**
현재 위치에서 인접한 방으로 이동합니다.

```
이동 경로:
  서재 → 주방 / 침실 / 지하실
  주방 → 서재 / 정원 / 테라스
  침실 → 서재 / 테라스
  테라스 → 침실 / 주방
  정원 → 주방
  지하실 → 서재
```

**②심문 (토큰 1개)**
같은 방에 있는 다른 플레이어 1명에게
용의자·장소·도구 각 1가지씩 제시하며 심문합니다.

```
심문 예시:
  "김민준이 서재에서 독약을 사용했나요?"

심문받은 플레이어 응답:
  자신의 카드 중 제시된 3장 중 1장 이상 있으면
  그 중 1장을 심문자에게만 비밀리에 보여줍니다.
  아무것도 없으면 "없습니다"라고 말합니다.
```

**③단서 조사 (토큰 1개)**
현재 위치의 방에서 단서 카드 1장을 뽑습니다.

```
단서 카드 예시:
  "이 방에서 금속 도구의 흔적이 발견됐습니다."
  → 도구: 칼, 촛대, 열쇠 중 하나 가능성

  "목격자: 용의자는 2층에서 목격됐습니다."
  → 장소: 침실 또는 테라스

  "용의자는 단독 행동이 가능한 직업입니다."
  → 용의자: 특정 직업군으로 범위 축소
```

**④증거 수집 (토큰 1개)**
현재 위치의 방에서 증거 토큰 1개를 가져옵니다.
증거 토큰은 최종 고발 시 정확도 보너스로 사용합니다.

**⑤최종 고발 (토큰 2개)**
범인·장소·도구를 모두 확신할 때 고발합니다.
반드시 서재에 있어야 고발 가능합니다.

---

## 최종 고발

### 고발 선언

"최종 고발!"을 외치고
범인·장소·도구를 기록지에 적습니다.

```
고발 기록:
  범인: ___________
  장소: ___________
  도구: ___________
```

### 고발 검증

사건 봉투를 열어 정답을 확인합니다.

**성공:** 3가지 모두 정확 → 즉시 승리
**부분 성공:** 2가지 정확 → 게임 계속, 한 번 더 기회
**실패:** 1가지 이하 → 해당 플레이어 탈락

### 점수 계산

```
기본 점수:
  3가지 정확: 30점
  2가지 정확: 10점
  1가지 정확: 0점

보너스 점수:
  증거 토큰 1개당: +2점
  단서 카드 사용 없이 해결: +10점
  첫 번째 고발 성공: +5점

페널티:
  잘못된 고발 시도: -5점
  탈락: 남은 점수의 50% 감점
```

## 전략 가이드

### 초보자 전략

**1. 자신의 카드 먼저 체크**
게임 시작 즉시 탐정 노트에
자신이 가진 카드를 모두 체크합니다.
이 카드들은 봉투에 없습니다.

**2. 공개 카드 체크**
나눠주고 남은 공개 카드도 체크합니다.

**3. 이동 효율화**
심문하려는 플레이어가 있는 방으로 이동하세요.
불필요한 이동은 행동 토큰 낭비입니다.

### 중급 전략

**1. 심문 최적화**
심문 시 3가지를 모두 모르는 항목으로 구성하세요.
이미 아는 항목을 넣으면 정보 획득 기회를 낭비합니다.

```
나쁜 심문:
  내가 가진 카드: 서재(장소)
  심문: "김민준이 서재에서 독약을?"
  → 서재가 정답 아님을 이미 알면서 낭비

좋은 심문:
  내가 모르는 것만: "이서연이 지하실에서 칼을?"
```

**2. 타인의 심문 관찰**
다른 플레이어들의 심문 내용과 응답을 기록하세요.
"없습니다" 응답은 중요한 소거 정보입니다.

**3. 단서 카드 우선순위**
여러 카테고리에 걸친 단서가 가장 가치 있습니다.

### 고급 전략

**1. 역추론**
다른 플레이어가 무엇을 아는지 파악하면
자신이 모르는 것을 간접적으로 알 수 있습니다.

**2. 고발 타이밍**
확신이 90% 이상일 때 고발하세요.
너무 빠른 고발은 실패 위험, 너무 늦으면 다른 플레이어에게 선점당합니다.

**3. 증거 토큰 관리**
확신이 있을 때 바로 고발하는 것이
증거 토큰을 모으는 것보다 중요할 수 있습니다.

## 사건 시나리오

### 시나리오 #01: 저택의 밤
```
배경:
오래된 저택에서 밤사이 사건이 발생했습니다.
저택에는 6명의 손님이 있었고
그 중 한 명이 범인입니다.

특수 규칙:
  야간 턴: 이동 비용 2배
  단서 카드: 시간대 단서 추가
  ("밤에만 접근 가능한 장소" 등)
```

### 시나리오 #02: 박물관 도난
```
배경:
박물관에서 귀중한 유물이 사라졌습니다.
범인은 박물관 관계자 중 한 명입니다.

특수 규칙:
  도구 대신 "훔친 방법" (열쇠 복제 / 경보 해제 / 창문 파손)
  장소 대신 "전시실" (고대관 / 근대관 / 특별전시관)
```

### 시나리오 #03: 학교 미스터리
```
배경:
학교에서 중요한 시험 문제가 유출됐습니다.
범인은 학교 구성원 중 한 명입니다.

특수 규칙:
  용의자: 교장 / 교감 / 담임교사 / 학생 / 학부모 / 경비원
  장소: 교장실 / 교무실 / 인쇄실 / 도서관 / 컴퓨터실 / 창고
  도구: USB / 복사기 / 카메라 / 이메일 / 메모리카드 / 사진
```

## 변형 규칙

### 협력 탐정 노트
- 경쟁 없이 팀이 함께 사건 해결
- 행동 토큰을 팀 공유 자원으로 사용
- 더 복잡한 사건 시나리오 가능

### 이중 사건 탐정 노트
- 사건 봉투 2개 (두 개의 사건 동시 진행)
- 어느 사건을 먼저 해결할지 선택
- 멀티태스킹 추론 능력 필요

### 제한 심문 탐정 노트
- 심문 시 예/아니오만 허용
- 더 많은 심문 횟수 필요
- 논리적 질문 설계 능력 훈련

## 교육적 활용

### 논리 연계
- **복합 추론**: 여러 단서의 통합 분석
- **가설 수립**: 증거 기반 가설 설정
- **비판적 사고**: 단서의 신뢰성 평가

### 소통 연계
- **정보 관리**: 아는 것과 모르는 것의 구분
- **전략적 질문**: 효율적 심문 설계
- **기록**: 체계적인 노트 작성

### 수업 활용 방법
1. 간단한 3인·3항목 버전으로 시작 (15분)
2. 정식 5인 게임 (35분)
3. "어떤 단서가 결정적이었나?" 복기 (10분)
4. 직접 시나리오 만들기 도전 (20분)
