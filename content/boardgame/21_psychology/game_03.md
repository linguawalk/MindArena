# 감정 날씨 (Emotion Weather)

> 감정은 날씨 같습니다.
> 맑음·흐림·비·폭풍.
> 감정을 인식하고 표현하고
> 다루는 능력을 기릅니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 감정 인식, 감정 표현, 정서 조절 |

## 게임 구성

```
날씨 카드: 맑음·흐림·비·폭풍·무지개 각 10장
상황 카드: 40장 (다양한 일상 상황)
감정 색상판: 감정 바퀴
대응 카드: 30장 (감정 대처 방법)
```

## 감정 날씨 체계

```
맑음 (☀️): 기쁨·만족·감사·평화
흐림 (☁️): 불안·걱정·우울·지루함
비 (🌧️): 슬픔·실망·그리움·외로움
폭풍 (⛈️): 분노·공포·좌절·혐오
무지개 (🌈): 복합 감정·성장·희망
```

## 기본 규칙

```
상황 카드 공개:
  일상적 상황 제시

날씨 선택:
  그 상황에서 느낄 감정 날씨 선택
  이유 설명

공유:
  각자 선택한 날씨 공개
  같은 상황, 다른 감정 비교

대응 카드:
  해당 날씨에 맞는 대처법 논의

득점:
  감정 표현의 풍부함
  대응 방법의 창의성
```

## 상황 카드 예시

```
"시험에서 예상보다 낮은 점수를 받았다"
"오랫동안 연락 없던 친구에게 메시지가 왔다"
"열심히 준비한 발표가 잘 됐다"
"아무 이유 없이 기분이 좋지 않다"
"실수로 중요한 약속을 잊었다"
"처음 가보는 새로운 환경"
```

## 대응 카드 예시

```
맑음 날씨 대응:
  "감사 일기 쓰기"
  "기쁨 나누기"

흐림 날씨 대응:
  "한 발짝 산책"
  "좋아하는 음악 듣기"

비 날씨 대응:
  "울어도 괜찮다"
  "신뢰하는 사람에게 이야기하기"

폭풍 날씨 대응:
  "10초 멈추기"
  "깊은 호흡 3번"
  "자리 잠깐 피하기"
```

## 심화 활동

```
날씨 일기:
  오늘 나의 감정 날씨는?
  무엇이 날씨를 바꿨는가?

날씨 예보:
  내일 어떤 상황이 예상되는가?
  어떤 날씨가 올 것 같은가?
  어떻게 준비할 것인가?

날씨 변화:
  같은 상황에서 날씨가 바뀐 경험
  무엇이 날씨를 바꿨는가?
```

## 교육적 효과

```
감정 어휘:
  다양한 감정 이름 알기
  → 정서 지능 기초

감정 수용:
  좋은 감정·나쁜 감정 없음
  모두 자연스럽고 소중한 신호

대처 능력:
  감정에 따른 건강한 대응
  → 정서 조절 능력
```

## 게임 후 토론

```
"같은 상황에서 다른 날씨를 선택한
 사람이 있었나요? 왜 달랐을까요?"

"어떤 날씨가 가장 다루기 어렵나요?
 어떻게 하면 도움이 될까요?"

"감정을 숨기는 것과 표현하는 것
 어느 것이 더 건강할까요?"
```
