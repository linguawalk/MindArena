# 편향 사냥꾼 (Bias Hunter)

> 우리는 모두 편향되어 있습니다.
> 하지만 편향을 알면
> 더 나은 판단을 할 수 있습니다.
> 일상 속 인지 편향을 찾아보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 비판적 사고, 인지 편향 인식, 객관적 판단 |

## 게임 구성

```
상황 카드: 50장 (일상·뉴스·의사결정 상황)
편향 카드: 20장 (주요 인지 편향)
판단 용지: 개인용
편향 점수판
```

## 주요 편향 카드

```
확증 편향:
  믿고 싶은 정보만 받아들이기
  "내가 옳다는 증거만 찾는다"

가용성 편향:
  쉽게 떠오르는 것을 더 중요하게 여기기
  "뉴스에 나온 것이 더 위험해 보인다"

닻 내리기 효과:
  처음 본 숫자에 판단이 끌리기
  "처음 가격이 기준이 된다"

후광 효과:
  한 가지 좋은 점이 전체를 좋게 보이게
  "잘생긴 사람이 더 능력있어 보인다"

집단 사고:
  집단 의견에 반대하기 어려움
  "다들 좋다고 하니까..."

손실 회피:
  잃는 것이 얻는 것보다 크게 느껴짐
  "1만원 잃는 것이 1만원 얻는 것보다 아프다"

자기 위주 편향:
  성공은 내 덕, 실패는 남 탓
  "잘 되면 내 실력, 안 되면 운이 나빠서"
```

## 기본 규칙

```
상황 카드 공개:
  일상적 판단 상황 제시
  예) "뉴스에서 비행기 사고를 봤다.
       다음 여행에 비행기를 안 타기로 했다"

개인 판단:
  어떤 편향이 작동했는지 카드 선택
  이유 작성

발표 및 토론:
  각자 선택한 편향 발표
  다른 의견 토론

득점:
  다수 일치: +2점
  독창적 분석: +1점 (소수의견)
```

## 상황 카드 예시

```
"유명인이 추천한 제품을 샀다"
→ 후광 효과

"로또 1등이 연속 같은 번호 안 나왔으니
 이번엔 나올 것 같다"
→ 도박사의 오류

"회의에서 팀장 의견에 아무도 반대 안 했다"
→ 집단 사고

"내가 좋아하는 후보의 나쁜 뉴스는
 믿기지 않는다"
→ 확증 편향

"50% 할인 후 20만원이 싸 보인다"
→ 닻 내리기 효과
```

## 탈편향 전략

```
확증 편향 극복:
  반대 증거 의도적으로 찾기
  "내 생각이 틀렸다면 왜일까?"

가용성 편향 극복:
  실제 통계 확인하기
  "실제 확률은 얼마인가?"

닻 내리기 극복:
  첫 숫자 무시하고 독립적 판단
  "처음 제안이 없었다면 얼마라고 했을까?"
```

## 교육적 효과

```
메타인지:
  자신의 사고 과정 관찰
  → 비판적 사고의 핵심

판단력:
  편향 없는 더 나은 결정
  → 의사결정 능력

겸손:
  "나도 편향될 수 있다"
  → 지적 겸손
```

## 게임 후 토론

```
"오늘 발견한 편향 중
 자신에게 가장 해당하는 것은?
 어떤 상황에서 그 편향이 나타났나요?"

"편향을 완전히 없앨 수 있을까요?"

"어떤 편향은 도움이 되기도 할까요?"
```
