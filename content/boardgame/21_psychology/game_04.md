# 방어기제 게임 (Defense Mechanism Game)

> 우리는 마음의 고통으로부터
> 자신을 보호하려 합니다.
> 그 방법이 방어기제입니다.
> 건강한 방어와 건강하지 않은 방어를 구분합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 심리 기제 이해, 자기 인식, 건강한 대처 |

## 주요 방어기제 카드

```
억압 (Repression):
  불쾌한 기억·감정을 무의식으로 밀어내기
  "그런 일 없었어" (기억하지 못함)

부정 (Denial):
  현실을 인정하지 않기
  "나는 괜찮아" (분명히 괜찮지 않은데)

투사 (Projection):
  내 감정을 남에게 돌리기
  "저 사람이 나를 싫어해" (내가 싫어하는데)

합리화 (Rationalization):
  그럴듯한 이유 만들기
  "어차피 저 포도는 셨을 거야"

반동형성 (Reaction Formation):
  반대 감정으로 표현하기
  "좋아하는 사람을 괴롭힌다"

승화 (Sublimation):
  충동을 사회적으로 인정받는 방식으로 표현
  "분노를 운동으로 풀기" (건강한 방어)

유머 (Humor):
  어려운 상황을 웃음으로 대처
  (비교적 건강한 방어)
```

## 기본 규칙

```
상황 공개:
  심리적 어려움 상황 제시
  예) "시험에 떨어진 후"

반응 선택:
  등장인물의 반응 제시
  어떤 방어기제인지 카드 선택

분석:
  건강한 방어인지 평가
  더 건강한 대안은?

토론:
  같은 방어기제도 상황에 따라 다름
  언제 건강하고 언제 문제인가?
```

## 상황 카드 예시

```
상황: "직장에서 해고됐다"
반응 A: "사실 그 회사 별로였어" (합리화)
반응 B: "전혀 신경 안 써" (억압/부정)
반응 C: "글을 써서 감정 정리" (승화)
반응 D: "그 회사 사람들이 날 싫어했어" (투사)

→ 어떤 반응이 가장 건강한가?
```

## 방어기제 건강도

```
미성숙한 방어 (주로 문제):
  부정·투사·행동화

신경증적 방어 (중간):
  억압·합리화·반동형성

성숙한 방어 (대체로 건강):
  승화·유머·이타주의·예상
```

## 교육적 효과

```
자기 인식:
  자신의 방어 패턴 발견
  → 심리적 자기 이해

타인 이해:
  남의 이상한 행동 이해
  → 판단 줄이기

정신 건강:
  건강한 대처 방법 학습
  → 심리적 유연성
```

## 게임 후 토론

```
"자신이 자주 사용하는
 방어기제는 무엇인 것 같나요?"

"방어기제가 완전히 없다면
 어떤 문제가 생길까요?"

"가장 건강한 방어기제는
 어떻게 키울 수 있을까요?"
```
