# 동기 탐정 (Motivation Detective)

> 사람들은 왜 그런 행동을 할까요?
> 겉으로 보이는 이유와
> 진짜 이유는 다를 수 있습니다.
> 인간 행동의 동기를 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 동기 이해, 공감, 행동 분석 |

## 게임 구성

```
행동 카드: 40장 (사람들의 다양한 행동)
동기 카드: 30장 (내적·외적 동기)
욕구 피라미드 보드: 매슬로우 욕구 5단계
분석 용지
```

## 매슬로우 욕구 단계

```
5단계 - 자아실현:
  잠재력 실현·성장·창의성

4단계 - 존중:
  인정·성취·자존감·지위

3단계 - 사회적:
  소속감·사랑·우정·관계

2단계 - 안전:
  신체·직업·재정적 안전

1단계 - 생리적:
  음식·물·수면·온도
```

## 기본 규칙

```
행동 카드 공개:
  사람의 특정 행동 제시
  예) "SNS에 하루 20번 게시물 올린다"

동기 추측:
  어떤 욕구를 충족하려는 행동인지
  매슬로우 단계 + 동기 카드 선택

발표:
  각자 분석 발표
  다른 사람 반응 관찰

토론:
  겉으로 보이는 동기 vs 숨겨진 동기
  여러 동기가 동시에 작용?

득점:
  가장 통찰력 있는 분석 투표
```

## 행동 카드 예시

```
"비싼 명품 가방을 산다"
→ 3단계(인정)? 4단계(지위)? 자기만족?

"밤새 공부한다"
→ 4단계(성취)? 2단계(안전/취업)?

"SNS 팔로워를 늘리려 한다"
→ 3단계(사회적)? 4단계(존중)?

"자원봉사를 한다"
→ 5단계(자아실현)? 3단계(소속)?

"매일 운동한다"
→ 1단계(건강)? 4단계(자존감)?
```

## 동기 카드 예시

```
외적 동기:
  보상·처벌·인정·경쟁

내적 동기:
  호기심·즐거움·성장·의미

복합 동기:
  두려움 극복·상처 치유
  증명하고 싶음·보상받고 싶음
```

## 교육적 효과

```
행동 이해:
  표면 행동 뒤의 동기 파악
  → 인간 이해 심화

자기 이해:
  내 행동의 진짜 이유
  → 자기 인식

공감:
  다른 사람 행동 이해
  → 판단 줄이기
```

## 게임 후 토론

```
"오늘 내 행동 중
 진짜 동기가 무엇인지 모르는 것이 있나요?"

"외적 동기와 내적 동기 중
 어느 것이 더 오래 지속될까요?"

"동기를 안다고 해서
 행동을 바꿀 수 있을까요?"
```
