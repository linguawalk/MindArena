# 심리 유형 배틀 (Personality Type Battle)

> 사람마다 다르게 생각하고 느끼고 행동합니다.
> 유형이 다른 사람들이 만날 때
> 무슨 일이 일어나는가?
> 다양성을 이해하고 협력하는 법을 배웁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 성격 유형 이해, 다양성 존중, 협력 전략 |

## 유형 체계

```
4가지 차원 (MBTI 간소화):

에너지 방향:
  E (외향): 사람에게서 에너지 얻음
  I (내향): 혼자 있을 때 에너지 충전

정보 수집:
  S (감각): 구체적 사실 중심
  N (직관): 패턴·가능성 중심

판단 방식:
  T (사고): 논리·분석 중심
  F (감정): 가치·관계 중심

생활 방식:
  J (판단): 계획·구조 선호
  P (인식): 유연성·즉흥 선호
```

## 게임 구성

```
유형 카드: E/I/S/N/T/F/J/P 각 4장
상황 카드: 40장 (팀 과제·갈등 상황)
강점 카드: 각 유형별 강점 목록
협력 카드: 유형별 최적 역할
```

## 기본 규칙

```
유형 배정:
  각자 유형 카드 조합으로 유형 받기
  예) E+N+T+J = 전략가형

상황 카드:
  팀 과제 상황 공개

유형별 반응:
  자신의 유형 특성으로 반응
  "이 유형이라면 어떻게 할까?"

갈등·협력:
  다른 유형과의 충돌 지점 찾기
  어떻게 협력할 수 있는가?

득점:
  팀 과제 완수도 + 협력 점수
```

## 유형별 특성 카드

```
전략가 (E+N+T+J):
  강점: 큰 그림·체계적 계획
  약점: 감정 고려 부족

중재자 (I+N+F+P):
  강점: 공감·창의적 해결
  약점: 결단력 부족

탐험가 (E+S+T+P):
  강점: 즉각 대응·실용적
  약점: 장기 계획 어려움

보호자 (I+S+F+J):
  강점: 세심한 배려·안정성
  약점: 변화 적응 어려움
```

## 상황 카드 예시

```
"기한이 3일 남은 팀 프로젝트"
→ J형: 계획표 만들기 vs P형: 일단 시작

"팀원이 실수했을 때"
→ T형: 원인 분석 vs F형: 감정 먼저 챙기기

"새로운 방향 제안"
→ N형: 가능성 탐색 vs S형: 현실 검토

"팀 내 갈등 발생"
→ E형: 즉시 대화 vs I형: 생각 정리 먼저
```

## 중요한 경고

```
유형론의 한계:
  사람은 유형보다 훨씬 복잡함
  상황에 따라 달라짐
  유형으로 사람 판단하면 안 됨

올바른 사용:
  자기 이해의 출발점으로만
  다름을 이해하는 도구로
  우열 없음 - 모든 유형 장단점 있음
```

## 교육적 효과

```
다양성 이해:
  다른 방식도 옳을 수 있음
  → 포용적 사고

팀 구성:
  다양한 유형이 강한 팀
  → 다양성의 가치

자기 이해:
  나의 자연스러운 경향 파악
  → 강점 개발
```

## 게임 후 토론

```
"자신의 유형이 정확하게 맞는다고
 느꼈나요? 어떤 부분이 달랐나요?"

"가장 협력하기 어려운 유형은?
 어떻게 하면 더 잘 협력할 수 있을까요?"

"완벽한 팀을 구성한다면
 어떤 유형들을 모을 것인가요?"
```
