# AI 판사 (AI Judge)

> AI가 판결을 내립니다.
> 공정한가? 편향되지 않았는가?
> AI의 결정을 인간이 어떻게
> 평가하고 통제해야 하는가?

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | AI 윤리, 알고리즘 편향, 기술 거버넌스 |

## 게임 구성

```
사건 카드: 30장 (AI 판단 사례)
AI 판결 카드: 각 사건당 2-3장
편향 카드: 15장 (학습 데이터 편향)
인간 판결 용지: 개인용
이의 제기 카드: 10장
```

## AI 판결 사례

```
사례 1 - 대출 심사:
  AI가 특정 우편번호 거주자의
  대출을 자동 거부
  → 역사적 차별 지역과 일치
  → 인종 차별인가?

사례 2 - 채용 AI:
  AI가 여성 지원자를 낮게 평가
  → 과거 남성 위주 채용 데이터 학습
  → 편향 재생산

사례 3 - 재범 예측:
  AI가 특정 집단 재범률 높게 예측
  → 역사적 불평등 반영
  → 미래 차별 강화?

사례 4 - 의료 진단:
  AI가 특정 피부색 환자 진단 오류
  → 학습 데이터 다양성 부족
  → 의료 불평등

사례 5 - 자율주행 판단:
  사고 불가피 시 누구를 보호?
  → 탑승자? 보행자?
  → 프로그래밍된 윤리
```

## 기본 규칙

```
사건 공개:
  AI 판결 사례 제시
  AI의 결정 공개

분석:
  ①AI 판결이 공정한가?
  ②어떤 편향이 작동했는가?
  ③인간이라면 어떻게 결정했는가?

역할별 입장:
  AI 개발자: 기술적 한계 설명
  피해자: 불공정 경험 주장
  판사: 법적 판단
  시민: 사회적 영향 평가

결론:
  AI 판결 수용·수정·폐기
  재발 방지 방법 제안
```

## 편향 카드 종류

```
역사적 편향:
  과거 불평등한 데이터 반영

대표성 편향:
  특정 집단 데이터 부족

측정 편향:
  측정 방법 자체의 불공정

피드백 루프:
  편향된 결정 → 편향된 데이터 → 더 편향
```

## AI 거버넌스 원칙

```
투명성:
  AI 결정 과정 설명 가능해야

책임성:
  잘못된 결정의 책임 주체 명확

공정성:
  다양한 집단에 동등한 결과

인간 통제:
  중요 결정은 인간이 최종 확인
```

## 교육적 효과

```
AI 리터러시:
  AI의 작동 원리와 한계
  → 기술 이해력

비판적 사고:
  기술을 무조건 신뢰하지 않기
  → 디지털 시민 의식

윤리적 설계:
  공정한 AI를 만들려면?
  → 기술 윤리학
```

## 게임 후 토론

```
"AI가 인간보다 더 공정할 수 있을까요?
 어떤 조건에서 그럴 수 있나요?"

"AI의 실수와 인간의 실수.
 어느 것이 더 용납될 수 있나요?"

"AI 결정에 이의를 제기할 권리가
 있어야 하는가요?"
```
