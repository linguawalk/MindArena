# 특이점 이후 (Post-Singularity)

> 기술적 특이점이 왔습니다.
> AI가 인간을 능가했습니다.
> 인간은 무엇을 하며 살 것인가?
> 가장 근본적인 질문을 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐⭐ |
| 목표 역량 | 인간 정체성, 기술 철학, 미래 가치관 |

## 세계관 설정

```
2045년, 기술적 특이점 도래:
  AI가 모든 지적 작업 수행
  로봇이 모든 육체 노동 수행
  인간의 노동 필요성 소멸
  기본소득으로 생계 해결

새로운 질문들:
  인간은 왜 사는가?
  일이 없는 삶의 의미는?
  AI와 인간의 관계는?
  인간만이 할 수 있는 것은?
```

## 게임 구성

```
삶의 방식 카드: 30장 (특이점 이후 삶의 선택)
가치 카드: 인간 고유 가치 20장
AI 능력 카드: AI가 할 수 있는 것 25장
철학 딜레마 카드: 15장
토론 점수판
```

## 삶의 방식 카드

```
창조:
  AI가 더 잘 만들어도 직접 만드는 이유?
  예술·음악·글쓰기

관계:
  AI 친구·연인도 가능한 시대
  인간 관계의 의미

탐험:
  AI가 더 빨리 발견해도
  직접 경험하는 이유

수행:
  완벽한 AI 운동 코치가 있어도
  직접 땀 흘리는 이유

명상·영성:
  AI가 답을 알아도
  스스로 깨달아야 하는 것
```

## 핵심 딜레마 카드

```
딜레마 1 - 창의성:
  "AI가 베토벤보다 아름다운 음악을 만든다.
   인간이 음악을 만드는 이유가 있는가?"

딜레마 2 - 관계:
  "AI 배우자가 인간 배우자보다
   더 이해해주고 사랑해준다면?"

딜레마 3 - 성장:
  "AI가 모든 지식을 즉시 전달할 수 있다면
   교육의 의미는?"

딜레마 4 - 목적:
  "일이 없어도 행복할 수 있는가?
   인간은 목적 없이 살 수 있는가?"

딜레마 5 - 정체성:
  "뇌에 AI 칩을 이식해 지능을 높인다면
   그것도 인간인가?"
```

## 기본 규칙

```
세계관 공유:
  특이점 이후 세계 이해

삶의 방식 선택:
  각자 어떤 삶을 선택할 것인지
  카드 조합으로 표현

딜레마 토론:
  철학 딜레마 카드 공개
  각자 입장 발표·토론

인간 고유 가치:
  AI가 절대 대체할 수 없는 것은?
  팀 합의로 5가지 선정

득점:
  논거의 깊이와 설득력
```

## 인간 고유 가치 탐구

```
철학자들의 답:

실존주의 (사르트르):
  "인간은 자신을 만들어가는 존재"
  → 선택과 책임이 본질

불교:
  "깨달음은 경험으로만 가능"
  → 과정이 목적

인본주의:
  "인간의 잠재력 실현"
  → 자아실현

공동체주의:
  "관계 속에서만 인간"
  → 연결이 본질
```

## 교육적 효과

```
인간 정체성:
  인간이란 무엇인가?
  → 철학적 자기 이해

기술 철학:
  기술과 인간의 관계
  → 비판적 기술 수용

미래 가치관:
  변하는 세상의 불변 가치
  → 가치관 정립
```

```
 우리가 준비해야 할 것은?"
```
