# 미래 도시 설계 (Future City Design)

> 2050년의 도시를 설계합니다.
> 기후 변화·인구 변화·기술 발전.
> 모든 사람이 행복하게 살 수 있는
> 도시는 어떻게 만들어지는가?

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-5명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 도시 계획, 지속가능성, 통합적 사고 |

## 게임 구성

```
도시 보드: 10×10 그리드
구역 타일:
  주거·상업·공업·녹지·교통·에너지 각 20개
시민 카드: 40장 (다양한 시민 유형)
문제 카드: 25장 (도시 문제)
지속가능성 트랙: 환경·경제·사회 3개
```

## 도시 설계 원칙

```
15분 도시 (파리 모델):
  걸어서 15분 내 모든 필수 시설
  → 자동차 의존도 감소

스마트 도시:
  IoT·AI로 도시 최적화
  → 에너지·교통·안전

자연 친화 도시:
  녹지·수공간 통합
  → 열섬 현상 감소

포용적 도시:
  모든 계층이 살 수 있는
  → 젠트리피케이션 방지
```

## 기본 규칙

```
도시 건설:
  구역 타일 배치
  제약 조건: 녹지 20% 이상
             대중교통 접근성 확보

시민 요구:
  시민 카드 뽑기
  각 시민 유형의 요구 충족 확인

문제 발생:
  문제 카드 공개
  예) "폭염 발생" "홍수" "인구 급증"
  도시 설계로 대응 가능한가?

평가:
  지속가능성 3개 트랙 점수
  환경·경제·사회 균형
```

## 시민 카드 예시

```
청년 직장인:
  대중교통 + 직주 근접
  임대료 부담 가능한 주거

노인:
  의료 시설 접근성
  보행자 친화 환경
  커뮤니티 공간

어린이 가족:
  학교·공원 근접
  안전한 놀이 환경

장애인:
  무장애 환경 전반
  이동 편의 시설

이민자:
  문화 공간
  언어 지원 서비스
```

## 문제 카드 예시

```
기후:
  "폭염 40°C: 냉방 시설 없는 취약계층"
  "100년 만의 홍수: 저지대 주거지 침수"

사회:
  "인구 30% 급증: 주거 부족"
  "젠트리피케이션: 원주민 이탈"

기술:
  "자율주행 전환: 택시 기사 실업"
  "해킹으로 스마트 시티 마비"
```

## 교육적 효과

```
도시 이해:
  도시 문제의 복잡성
  → 사회 구조 이해

통합적 사고:
  환경·경제·사회 동시 고려
  → 지속가능발전 사고

민주적 설계:
  모든 시민을 위한 도시
  → 포용성 가치
```
