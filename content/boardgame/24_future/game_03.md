# 디지털 시민 (Digital Citizen)

> 온라인에서도 시민입니다.
> 디지털 공간의 권리와 책임.
> 가짜 뉴스·개인정보·사이버 폭력.
> 디지털 시대의 시민 의식을 훈련합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 디지털 리터러시, 온라인 윤리, 개인정보 이해 |

## 게임 구성

```
상황 카드: 50장 (디지털 공간 상황)
선택 카드: 각 상황당 3-4개 선택지
결과 카드: 각 선택의 결과
디지털 시민 점수판: 개인 평판 트랙
```

## 상황 카드 유형

```
가짜 뉴스:
  "친구가 충격적인 뉴스 공유 요청"
  "내가 믿는 내용이지만 출처 불명"

개인정보:
  "무료 앱이 연락처 접근 요청"
  "편리한 서비스를 위해 위치 공유"

사이버 폭력:
  "친구가 특정인 욕하는 단톡방 초대"
  "익명으로 누군가 비난하는 댓글"

저작권:
  "유튜브 영상 무단 다운로드"
  "과제에 이미지 무단 사용"

디지털 발자국:
  "취업 전 SNS 과거 게시물 발견"
  "잊고 싶은 사진이 온라인에 존재"
```

## 기본 규칙

```
상황 카드 공개:
  디지털 상황 제시

선택:
  각자 비밀리에 선택지 고르기
  동시 공개

결과 확인:
  선택에 따른 결과 카드 공개
  단기·장기 결과 모두 포함

토론:
  왜 그 선택을 했는가?
  더 나은 선택이 있었는가?

평판 트랙:
  선택에 따라 디지털 시민 점수 변화
```

## 선택지 예시

```
상황: "친구가 확인되지 않은 뉴스 공유 요청"

선택 A: 바로 공유 (+좋아요 / -신뢰도)
선택 B: 무시 (안전하지만 소극적)
선택 C: 출처 확인 후 결정 (올바른 선택)
선택 D: 친구에게 가짜 뉴스임을 알림 (적극적)

결과:
  A 선택 시: 가짜 뉴스가 퍼짐, 내 신뢰도 하락
  C 선택 시: 사실 확인 능력 인정
  D 선택 시: 친구 관계 불편하지만 옳은 일
```

## 디지털 시민 10원칙

```
1. 디지털 접근: 모든 사람의 참여 권리
2. 디지털 상거래: 안전한 거래
3. 디지털 소통: 책임있는 소통
4. 디지털 리터러시: 비판적 사용 능력
5. 디지털 예절: 온라인 매너
6. 디지털 법률: 법 준수
7. 디지털 권리·책임: 권리만큼 책임
8. 디지털 건강: 균형있는 사용
9. 디지털 보안: 자신·타인 보호
10. 디지털 환경: 지속가능한 사용
```

## 교육적 효과

```
디지털 리터러시:
  비판적 정보 소비
  → 가짜 뉴스 대응

온라인 윤리:
  온라인도 현실의 연장
  → 사이버 예절

개인정보 인식:
  디지털 발자국의 영구성
  → 개인정보 보호
```
