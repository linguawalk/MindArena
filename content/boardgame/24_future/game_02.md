# 기술 예언자 (Technology Prophet)

> 10년 후 세상은 어떻게 변할까요?
> 기술 발전의 패턴을 이해하고
> 미래를 예측해 보세요.
> 예측의 정확성보다 논거가 중요합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 미래 예측, 기술 트렌드 이해, 논리적 추론 |

## 게임 구성

```
기술 카드: 40장 (현재 개발 중인 기술)
예측 용지: 개인용
시간 카드: 5년·10년·20년·50년
평가 기준 카드: 논거 평가 기준
트렌드 카드: 20장 (기술 발전 패턴)
```

## 기술 카드 예시

```
AI·컴퓨팅:
  범용 AI (AGI)
  양자 컴퓨터
  뇌-컴퓨터 인터페이스

바이오:
  노화 역전 기술
  장기 프린팅
  유전자 편집 (CRISPR)

에너지:
  핵융합 발전
  고체 배터리
  우주 태양광

물리:
  개인 비행체
  초고속 교통 (하이퍼루프)
  우주 관광 대중화
```

## 기본 규칙

```
기술 카드 공개:
  현재 기술 1개 제시

예측 작성 (3분):
  ①언제 상용화될 것인가? (시간 카드)
  ②어떻게 세상이 바뀔 것인가?
  ③누가 혜택을 받고 누가 피해를 받는가?
  ④어떤 새로운 문제가 생길 것인가?

발표 및 토론:
  각자 예측 발표
  논거 평가

득점:
  가장 설득력 있는 예측 투표
```

## 기술 발전 패턴

```
가트너 하이프 사이클:
  기대 과잉 → 환멸 → 안정화

무어의 법칙:
  2년마다 컴퓨팅 성능 2배
  (현재 둔화 중)

기술 수용 곡선:
  혁신가 → 초기 수용자 → 조기 다수 → 후기 다수 → 지체자

기술 융합:
  단독 기술보다 융합에서 혁신
  AI + 바이오 = ?
  AI + 에너지 = ?
```

## 과거 예측 vs 현실

```
맞은 예측:
  1984년: "개인 컴퓨터 시대 올 것"
  1990년: "인터넷이 세상 바꿀 것"

틀린 예측:
  2000년: "날아다니는 자동차 2020년까지"
  1977년: "전 세계 컴퓨터 5대면 충분"

교훈:
  기술 방향은 맞지만 시기는 틀리는 경우 많음
  사회적 수용이 기술만큼 중요
```

## 교육적 효과

```
미래 사고:
  현재에서 미래로 연결
  → 전략적 사고

기술 이해:
  현재 기술의 발전 방향
  → 디지털 리터러시

논리적 추론:
  근거 있는 예측
  → 비판적 사고력
```

"당신이 살고 싶은 2050년의 세상은
