# 미로 건축가 (Maze Architect)

> 한 명은 미로를 설계하고
> 다른 한 명은 탈출해야 합니다.
> 건축가는 막고, 탐험가는 뚫는다.
> 공간 전략의 대결.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 35-45분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 공간 전략, 경로 탐색, 상대 모델링 |

## 게임 구성

```
미로 보드: 8×8 그리드
벽 타일: 30개
출발·도착 말: 각 1개
이동 카드: 방향 카드 40장
시야 차단 판: 2개 (정보 숨김용)
```

## 기본 규칙

```
역할:
  건축가: 미로 설계 + 방해
  탐험가: 출발→도착 최단 경로

건축가 턴:
  벽 타일 1개 추가 또는 이동
  탐험가 경로 차단 시도

탐험가 턴:
  이동 카드 1장 사용
  상하좌우 1-2칸 이동

승리 조건:
  탐험가: 20턴 내 도착
  건축가: 탐험가 20턴 내 막기
```

## 전략 포인트

```
건축가 전략:
  막다른 골목 만들기
  출구 근처 집중 방어
  탐험가 경로 예측

탐험가 전략:
  여러 경로 동시 탐색
  건축가 패턴 읽기
  최단 경로 계속 재계산
```

## 협력 모드

```
2인 협력:
  함께 최단 경로 찾기
  건축가 없이 시간 내 탈출

팀 대결 (4인):
  건축가 팀 vs 탐험가 팀
  역할 교대로 2라운드
```

## 교육적 효과

```
경로 탐색:
  BFS·DFS 알고리즘 직관적 이해
  → 컴퓨터 과학 기초

공간 전략:
  2D 공간에서 자원 배치
  → 전략적 공간 사고

상대 모델링:
  상대 다음 수 예측
  → 마음 이론(Theory of Mind)
```
