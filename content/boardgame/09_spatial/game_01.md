# 도형 퍼즐 레이스 (Shape Puzzle Race)

> 조각을 빠르게 맞추는 자가 이깁니다.
> 하지만 속도보다 정확성이 먼저입니다.
> 공간을 읽는 눈을 훈련합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 공간 지각, 도형 인식, 순발력 |

## 게임 구성

```
퍼즐 보드: 5×5 그리드 (개인용 4개)
도형 타일: 테트로미노 7종 × 4세트
목표 카드: 40장 (완성 형태 그림)
모래시계: 60초
```

## 기본 규칙

```
준비:
  목표 카드 1장 공개
  모든 플레이어 동일한 타일 세트 보유

경쟁:
  60초 내에 목표 형태 완성
  먼저 완성하고 "완성!" 외치기

검증:
  목표 카드와 비교
  정확하면 +3점
  오류 시 -1점 + 계속 진행

라운드:
  10라운드 후 최다 점수 승리
```

```
      □

     □_
     □□
J형: _□
      □
     □□
```

## 난이도 조절

```
초급: 목표 형태 단순 (3-4조각)
중급: 목표 형태 복잡 (5-6조각)
고급: 회전·반전 포함
심화: 3D 형태 목표
```

## 교육적 효과

```
공간 지각:
  2D 도형의 조합 예측
  → 공간 추론 능력

도형 회전:
  머릿속 회전 시뮬레이션
  → 정신적 회전 능력

속도 + 정확성:
  빠르면서도 정확한 판단
  → 인지 효율성
```
