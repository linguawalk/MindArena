# 시점 탐정 (Viewpoint Detective)

> 같은 물체를 다른 방향에서 보면
> 전혀 다른 모습입니다.
> 제시된 시점에서 어떻게 보이는지
> 가장 먼저 맞히는 자가 이깁니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 시점 전환, 공감 사고, 공간 추론 |

## 게임 구성

```
3D 구조물 카드: 40장 (블록 구조)
시점 카드: 앞·뒤·왼·오른·위·아래 6방향
시점 그림 카드: 각 구조물당 6장
정답 토큰
```

## 기본 규칙

```
문제 제시:
  3D 구조물 카드 공개
  시점 카드 1장 공개
  (예: "오른쪽에서 본 모습")

추론:
  해당 시점에서 보이는 모습
  그림 카드 4장 중 선택

동시 공개:
  모든 플레이어 동시에 선택 카드 공개

득점:
  정답: +2점
  가장 빠른 정답자: 추가 +1점
```

## 문제 예시

```
구조물:
  1층: 2×2 블록
  2층: 1×1 블록 (왼쪽 앞)
  3층: 없음

위에서 본 모습:
  □□
  □_

앞에서 본 모습:
  _□
  □□

오른쪽에서 본 모습:
  □_
  □□
```

## 심화 모드

```
제작자 모드:
  직접 구조물 만들고
  6방향 그림 모두 그리기

탐정 모드:
  3장의 시점 그림으로
  원래 구조물 역추론

건축가 모드:
  시점 그림대로 블록 조립
  실물 조작 포함
```

## 교육적 효과

```
시점 전환:
  다른 위치에서 보는 연습
  → 탈중심화 사고

공간 추론:
  3D↔2D 변환 자유자재
  → 도면·설계 이해력

공감 능력:
  "저 사람 눈에는 어떻게 보일까"
  → 타인 관점 이해의 기초
```

## 게임 후 토론

```
"같은 상황을 다른 사람은
 어떻게 볼까요?
 시점이 다르면 무엇이 달라지나요?"

"일상에서 시점 차이로
 오해가 생긴 경험이 있나요?"
```
