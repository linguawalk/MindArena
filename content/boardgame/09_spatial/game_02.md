# 그림자 맞추기 (Shadow Match)

> 3D 물체의 그림자를 보고
> 원래 형태를 추론합니다.
> 보이지 않는 것을 상상하는 능력.
> 공간 지능의 핵심입니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 3D 공간 추론, 시각적 추론, 논리 사고 |

## 게임 구성

```
그림자 카드: 60장 (정면·측면·위 3방향 그림자)
물체 카드: 60장 (정답 3D 형태)
오답 카드: 각 문제당 3장
점수 토큰
```

## 기본 규칙

```
문제 제시:
  그림자 카드 1-3장 공개
  (정면만 / 정면+측면 / 3방향)

추론:
  어떤 3D 물체인지 판단
  물체 카드 4장 중 1개 선택

득점:
  1장 그림자로 맞추면: +5점
  2장 그림자로 맞추면: +3점
  3장 그림자로 맞추면: +1점
  오답: 0점
```

## 문제 예시

```
그림자 세트 1:
  정면: 원
  측면: 직사각형
  위: 원
  → 정답: 원기둥

그림자 세트 2:
  정면: 삼각형
  측면: 삼각형
  위: 정사각형
  → 정답: 사각뿔

그림자 세트 3:
  정면: 십자형
  측면: 직사각형
  위: 직사각형
  → 정답: 십자형 기둥
```

## 심화 모드

```
역방향 모드:
  3D 물체를 보고
  그림자 3장 직접 그리기

건축 모드:
  블록을 쌓아 그림자 재현
  실물 조작 + 공간 추론
```

## 교육적 효과

```
3D→2D 변환:
  입체를 평면으로 이해
  → 도면 읽기 능력

역추론:
  결과에서 원인 유추
  → 역방향 사고

시각 지능:
  비언어적 추론 능력
  → IQ 핵심 요소 훈련
```
