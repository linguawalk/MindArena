# 회전 배틀 (Rotation Battle)

> 같은 도형도 회전하면 달라 보입니다.
> 상대보다 빠르게 일치 여부를 판단하세요.
> 머릿속 회전이 빠를수록 유리합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 정신적 회전, 순발력, 공간 인식 |

## 게임 구성

```
기준 카드: 40장 (기준 도형)
비교 카드: 각 기준당 4장
  (동일 회전 2장 + 거울 반전 2장)
버저: 플레이어당 1개
점수판
```

## 기본 규칙

```
문제 제시:
  기준 카드 1장 + 비교 카드 1장 공개

판단:
  비교 카드가 기준 카드를
  회전만으로 일치 가능한가?
  (거울 반전은 불가)

버저:
  판단 완료 시 버저 누르기
  "같음" 또는 "다름" 선언

득점:
  정답 + 가장 빠름: +2점
  정답 + 느림: +1점
  오답: -1점
```

## 판단 기준

```
같음 (회전으로 일치):
  90° / 180° / 270° 회전 후 동일

다름 (거울 반전):
  어떻게 회전해도 일치 불가
  좌우 또는 상하 반전된 형태
```

## 훈련 팁

```
초보자:
  실제로 카드를 돌려보며 확인
  패턴 익히기

중급자:
  머릿속으로만 회전
  특징점 찾기 (돌출부, 오목부)

고급자:
  0.5초 내 판단
  직관적 인식 훈련
```

## 교육적 효과

```
정신적 회전:
  IQ 검사 핵심 항목
  → 공간 지능 직접 훈련

시각 처리 속도:
  빠른 형태 인식
  → 인지 처리 속도 향상

집중력:
  순간 집중 + 빠른 판단
  → 주의력 훈련
```
