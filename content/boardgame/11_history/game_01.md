# 문명의 선택 (Civilization Choice)

> 당신이 고대 문명의 지도자입니다.
> 농업, 군사, 문화, 교역 중
> 무엇에 투자할 것인가?
> 모든 선택에는 대가가 따릅니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 60-90분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 자원 배분, 장기 전략, 역사적 인과 사고 |

## 시대 설정

```
고대 문명 4개 (선택):
  메소포타미아 (농업·법률 강점)
  이집트 (건축·종교 강점)
  그리스 (철학·민주주의 강점)
  중국 (기술·관료제 강점)
```

## 게임 구성

```
문명 보드: 플레이어당 1개
자원 큐브: 식량·금·지식·병력 각 20개
발전 카드: 40장 (기술·제도·문화)
사건 카드: 20장 (역사적 위기)
시대 트랙: 고대→중세→근대
```

## 기본 규칙

```
매 턴 구조:
  ①자원 생산 (보드 상태에 따라)
  ②발전 카드 구매 (자원 소비)
  ③사건 카드 대응 (위기 관리)
  ④이웃 문명과 교역·전쟁

발전 트리:
  농업 → 도시화 → 제국
  군사 → 정복 → 패권
  문화 → 철학 → 보편 종교
  교역 → 화폐 → 금융

승리 조건:
  5시대 후 가장 높은 문명 점수
  (군사·문화·경제·기술 합산)
```

## 역사적 사건 카드

```
기근: 식량 자원 -3
      농업 발전국은 -1로 감소

침략: 인접 문명이 군사 행동
      방어력 부족 시 발전 카드 잃음

전염병: 전 문명 인구 -2
        의학 발전국은 면역

황금기: 이번 턴 자원 생산 2배
        문화 발전국은 3배
```

## 역사 연결

```
각 발전 카드에 실제 역사 사례 표기:
  "관개 농업" → 메소포타미아 티그리스강
  "민주주의" → 아테네 클레이스테네스
  "만리장성" → 진시황의 방어 전략

게임 후 토론:
  "왜 로마는 멸망했는가?"
  "문명의 흥망에 공통 패턴이 있는가?"
```

## 교육적 효과

```
역사적 인과:
  원인→결과 사고
  → 역사 이해의 핵심

자원 배분:
  제한된 자원의 전략적 투자
  → 경제적 사고

장기 계획:
  단기 이익 vs 장기 발전
  → 전략적 사고력
```
