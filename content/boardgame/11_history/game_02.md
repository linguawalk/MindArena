# 역사 재판 (History Trial)

> 역사적 인물이 법정에 섰습니다.
> 한쪽은 변호, 한쪽은 검사.
> 역사의 판단을 다시 내려보세요.
> 정답은 없습니다. 논거만 있습니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 역사 이해, 논증 능력, 다면적 사고 |

## 재판 대상 인물 예시

```
나폴레옹: 영웅인가, 침략자인가?
콜럼버스: 발견자인가, 파괴자인가?
진시황: 통일자인가, 폭군인가?
마키아벨리: 현실주의자인가, 악인인가?
히로시마 결정자: 전쟁 종결자인가, 학살자인가?
```

## 게임 구성

```
인물 카드: 20장
증거 카드: 각 인물당 10장
  (찬성 5장 + 반대 5장)
역할 카드: 판사·검사·변호사·배심원
논증 점수판
```

## 기본 규칙

```
역할 배정:
  판사: 1명 (진행)
  검사: 1-2명 (유죄 주장)
  변호사: 1-2명 (무죄 주장)
  배심원: 나머지 (평결)

재판 구조:
  ①모두 발언 (각 2분)
  ②증거 제시 (증거 카드 활용)
  ③반박 (각 1분)
  ④최후 진술 (각 1분)
  ⑤배심원 평결 + 이유 발표

득점:
  설득한 배심원 수
  논증의 질 (판사 평가)
```

## 증거 카드 예시 (나폴레옹)

```
검사 측 증거:
  "유럽 전쟁으로 사망자 300만 명"
  "이집트 원정 실패, 무의미한 희생"
  "황제 즉위로 공화정 이념 배신"

변호사 측 증거:
  "나폴레옹 법전으로 근대 법률 기초"
  "봉건제 폐지로 유럽 자유화"
  "능력주의 사회 실현"
```

## 논증 평가 기준

```
근거의 질:
  역사적 사실에 기반한가?

논리 구조:
  전제→근거→결론이 명확한가?

반박 능력:
  상대 논증의 약점 지적

설득력:
  배심원의 마음을 움직였는가?
```

## 교육적 효과

```
역사적 맥락:
  단순 사실 암기가 아닌
  배경·원인·결과 이해

다면적 사고:
  한 사건의 여러 측면 동시 고려
  → 비판적 사고력

논증 능력:
  주장 + 근거 + 반박
  → 토론·글쓰기 기초
```

## 게임 후 토론

```
"역사를 현재의 기준으로
 판단하는 것이 공정한가요?"

"역사적 맥락을 고려하면
 나쁜 행동도 정당화될 수 있나요?"

"역사에서 배울 수 있는
 가장 중요한 교훈은 무엇인가요?"
```
