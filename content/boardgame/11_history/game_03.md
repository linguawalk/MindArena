# 타임라인 배틀 (Timeline Battle)

> 언제 일어난 일인지 아는 것이
> 역사를 이해하는 첫걸음입니다.
> 사건을 시간 순서에 맞게 배치하며
> 역사의 흐름을 몸으로 익힙니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-6명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 역사 연대기, 인과 관계, 역사 지식 |

## 게임 구성

```
역사 카드: 120장
  앞면: 사건 이름·그림
  뒷면: 연도·설명
타임라인 트랙: 테이블 중앙에 배치
```

## 기본 규칙

```
준비:
  카드 섞어 각자 4장 받기
  (뒷면 보지 않고)
  첫 카드 1장 공개해 트랙 시작

자기 턴:
  손패 카드 1장 선택
  타임라인 트랙에 위치 예측·배치

확인:
  카드 뒤집어 연도 확인
  올바른 위치: 카드 유지
  틀린 위치: 카드 버리고 1장 추가

승리:
  손패 먼저 소진한 플레이어
```

## 카드 예시

```
한국사:
  고조선 건국 (기원전 2333년)
  한글 창제 (1443년)
  임진왜란 (1592년)
  3·1 운동 (1919년)
  한국전쟁 (1950년)

세계사:
  로마 건국 (기원전 753년)
  마그나카르타 (1215년)
  구텐베르크 인쇄술 (1450년)
  프랑스 혁명 (1789년)
  달 착륙 (1969년)
```

## 테마 덱

```
한국사 덱: 한국 역사만
세계사 덱: 주요 세계사
과학사 덱: 발명·발견 역사
문화사 덱: 예술·문학 역사
전쟁사 덱: 주요 전쟁
혼합 덱: 모든 테마 혼합
```

## 교육적 효과

```
연대기 이해:
  역사의 순서와 흐름
  → 역사 시험 준비에 직접 활용

인과 관계:
  어떤 사건이 다른 사건의 원인?
  → 역사적 사고력

배경 지식:
  자연스럽게 역사 지식 습득
  → 즐거운 역사 학습
```
