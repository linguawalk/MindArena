# 역사적 순간 (Historic Moment)

> 역사의 분기점에 서 있습니다.
> 당신이 그 결정을 내린다면?
> 다른 선택을 했다면 역사가 어떻게 달라졌을까?
> 반사실적 사고로 역사를 깊이 이해합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 반사실적 사고, 역사 이해, 의사결정 |

## 게임 구성

```
결정 카드: 50장 (역사적 갈림길)
결과 카드: 각 결정당 2-3장 (실제 + 가상)
논증 점수판
배심원 토큰: 플레이어당 3개
```

## 기본 규칙

```
결정 카드 공개:
  역사적 인물의 결정 순간 제시

역할 배정:
  실제 결정 옹호자: 1명
  대안 결정 제안자: 1-2명
  배심원: 나머지

논증 (각 2분):
  실제 결정이 왜 최선이었나
  또는 다른 결정을 했다면 어땠을까

배심원 평결:
  더 설득력 있는 주장에 토큰
  이유 1문장 필수 발표

득점:
  받은 토큰 수
```

## 결정 카드 예시

```
결정 1:
  "1592년 조선 선조는
   임진왜란 때 한양을 버리고 피난했습니다.
   만약 버티며 항전했다면?"

결정 2:
  "1945년 미국은 일본에 원자폭탄을 투하했습니다.
   협상으로 종전을 시도했다면?"

결정 3:
  "1939년 영국·프랑스는 뮌헨 협정으로
   히틀러에게 양보했습니다.
   즉각 전쟁을 선택했다면?"

결정 4:
  "세종대왕은 한글을 창제했습니다.
   한자를 계속 사용했다면?"
```

## 반사실적 사고 가이드

```
좋은 반사실적 논증:
  ①당시 상황·제약 조건 고려
  ②단기·장기 결과 모두 분석
  ③역사적 증거에 기반한 추론
  ④과장하지 않는 합리적 상상

피해야 할 논증:
  현재 지식으로 과거 판단
  단순한 "더 좋았을 것" 주장
  역사적 맥락 무시
```

## 교육적 효과

```
반사실적 사고:
  "만약 ~이었다면" 사고
  → 창의적·비판적 사고의 핵심

역사 이해 심화:
  왜 그 결정을 했는지 이해
  → 표면적 암기 탈피

의사결정 훈련:
  불완전한 정보로 선택
  → 실생활 의사결정 능력
```

## 게임 후 토론

```
"역사는 영웅 개인이 만드는가,
 구조적 흐름이 만드는가?"

"역사에서 '운'의 역할은 얼마나 클까요?"

"지금 우리가 내리는 결정 중
 미래 역사책에 실릴 수 있는 것은?"
```
