# 제국의 흥망 (Rise and Fall)

> 모든 제국은 흥하고 망합니다.
> 당신의 제국은 얼마나 오래갈 수 있는가?
> 내부의 부패와 외부의 위협.
> 균형을 잃으면 역사 속으로 사라집니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 50-70분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 시스템 사고, 균형 관리, 역사적 패턴 인식 |

## 게임 구성

```
제국 보드: 플레이어당 1개
  (영토·군사·경제·문화·부패 트랙)
사건 덱: 60장 (위기·기회)
동맹 카드: 20장
역사 카드: 실제 제국 붕괴 사례 10장
```

## 핵심 메커니즘

```
5가지 균형 유지:
  영토: 너무 크면 관리 불가
  군사: 너무 약하면 침략 당함
  경제: 고갈되면 모든 것 중단
  문화: 낮으면 내부 분열
  부패: 높아지면 효율 감소

상호작용:
  군사 강화 → 경제 소비
  영토 확장 → 관리 비용 증가
  문화 투자 → 단기 손해, 장기 이득
  부패 방치 → 서서히 붕괴
```

## 실제 제국 패턴 카드

```
로마 제국:
  "너무 넓어진 제국은 수비가 불가능하다"
  → 영토 트랙 최대치 페널티 강화

몽골 제국:
  "정복은 쉬우나 통치는 어렵다"
  → 영토 확장 후 문화 트랙 급락

대영 제국:
  "무역이 곧 힘이다"
  → 경제 트랙 보너스, 군사 의존도 낮춤
```

## 승리 조건

```
기본: 10라운드 후 생존한 제국 중 최고 점수

추가 승리:
  "역사에 남은 제국":
    문화 점수 최고 시 보너스
  "불패의 군대":
    군사 점수 최고 유지 시 보너스
  "번영의 시대":
    경제 트랙 최고 유지 시 보너스
```

## 교육적 효과

```
시스템 사고:
  여러 요소의 상호작용 이해
  → 복잡계 사고력

역사적 패턴:
  흥망의 공통 원인 발견
  → 역사 법칙 귀납

균형 감각:
  단기·장기 이익 조율
  → 의사결정 능력
```
