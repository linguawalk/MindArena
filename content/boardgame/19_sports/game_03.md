# 올림픽 기획자 (Olympic Planner)

> 올림픽은 단순한 스포츠 대회가 아닙니다.
> 외교·경제·문화·도시가 얽힌
> 세계 최대의 이벤트.
> 성공적인 올림픽을 기획해 보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 프로젝트 기획, 이해관계자 관리, 글로벌 사고 |

## 게임 구성

```
도시 보드: 개인용 (올림픽 준비 현황)
자원 카드: 예산·인력·시간·외교력 각 20개
이슈 카드: 30장 (돌발 상황)
평가 지표: 성공·경제·환경·이미지 트랙
```

## 기본 규칙

```
도시 선택:
  각자 다른 가상 도시 선택
  초기 자원 배분 (불평등)

준비 단계 (5라운드):
  매 라운드 자원 투자 결정
  시설·교통·보안·문화 프로그램

이슈 카드:
  매 라운드 돌발 상황 발생
  대응 방식 선택

개회식:
  5라운드 후 성과 평가
  4가지 지표 합산
```

## 투자 분야

```
시설 (예산 많이):
  경기장·선수촌·미디어센터
  투자 많을수록 성공 지표 상승
  하지만 환경 지표 하락

교통 (예산·시간):
  도로·지하철·공항 확충
  관광객 만족도 상승

보안 (예산·인력):
  테러 예방·선수 보호
  부족하면 이슈 카드 피해 증가

문화 (외교력):
  성화 봉송·문화 행사
  국가 이미지 상승
```

## 이슈 카드 예시

```
긍정 이슈:
  "스타 선수 참가 확정"
  → 관심도 +10

중립 이슈:
  "예상치 못한 더위"
  → 냉방 시설 추가 비용

부정 이슈:
  "건설 노동자 파업"
  → 시설 완공 지연

외교 이슈:
  "특정 국가 보이콧 선언"
  → 외교력으로 해결 가능
```

## 올림픽 레거시 토론

```
올림픽 개최의 빛:
  국제적 위상 상승
  인프라 개선
  관광 수입

올림픽 개최의 그림자:
  막대한 비용
  환경 파괴
  시설 사후 관리 문제

실제 사례:
  2004 아테네: 재정 위기 촉발
  2008 베이징: 국가 위상 강화
  2018 평창: 사후 활용 고민
```

## 교육적 효과

```
프로젝트 관리:
  자원·시간·우선순위 관리
  → 실무 역량 기초

이해관계자:
  다양한 요구 조율
  → 사회적 사고

글로벌 사고:
  국제 이벤트의 복잡성
  → 세계 시민 의식
```
