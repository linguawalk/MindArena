# 스포츠 과학 (Sports Science)

> 왜 어떤 선수는 더 빠른가?
> 어떻게 훈련해야 더 강해지는가?
> 스포츠와 과학의 만남.
> 인체의 가능성을 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 스포츠 과학 지식, 데이터 분석, 신체 이해 |

## 게임 구성

```
선수 카드: 40장 (다양한 신체 능력치)
훈련 카드: 30장 (훈련 방법)
과학 원리 카드: 20장 (생리학·역학)
대회 카드: 15장 (경기 조건)
```

## 기본 규칙

```
선수 육성:
  선수 카드 1장 받기
  (능력치: 속도·지구력·근력·유연성·정신력)

훈련 선택:
  3라운드 동안 훈련 카드 선택
  능력치 향상

대회 출전:
  대회 카드 조건에 맞는 능력치 비교
  과학 원리 카드로 보너스·페널티

승리:
  3번의 대회 합산 점수
```

## 훈련 카드 예시

```
유산소 훈련:
  지구력 +3 / 근력 +1
  회복 시간: 1라운드

무산소 훈련:
  근력 +3 / 속도 +2
  부상 위험: 10%

유연성 훈련:
  유연성 +4 / 부상 방지
  다른 능력치 소폭 향상

멘탈 훈련:
  정신력 +4
  압박 상황 보너스
```

## 과학 원리 카드

```
지레 원리:
  긴 팔다리 = 속도 유리
  → 수영·달리기 적용

에너지 시스템:
  ATP-PC계: 순간 폭발력
  젖산계: 1-2분 지속
  유산소계: 장시간 지속

근섬유 유형:
  속근 (백근): 폭발력·피로 빠름
  지근 (적근): 지구력·피로 느림

심박수:
  최대 심박수 = 220 - 나이
  운동 강도에 따른 심박수 목표
```

## 스포츠 과학 퀴즈 모드

```
질문 예시:
  "마라톤 선수와 단거리 선수의
   근육 구성이 다른 이유는?"

  "고지대 훈련이 효과적인 이유는?"

  "스트레칭은 운동 전에 해야 하나,
   운동 후에 해야 하나?"

  "도핑은 왜 금지되는가?
   어떤 종류가 있는가?"
```

## 교육적 효과

```
인체 이해:
  근육·심폐·신경 시스템
  → 생물학 기초 연결

데이터 사고:
  능력치 비교·분석
  → 수치 기반 의사결정

건강 교육:
  올바른 운동 방법
  → 평생 건강 습관
```
