# 스포츠 룰 메이커 (Sports Rule Maker)

> 규칙이 없으면 스포츠도 없습니다.
> 공정하고 재미있는 규칙은
> 어떻게 만들어지는가?
> 새로운 스포츠를 직접 설계합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 규칙 설계, 공정성 사고, 창의적 문제 해결 |

## 게임 구성

```
기본 요소 카드: 도구·장소·인원·목표 각 20장
규칙 제약 카드: 15장 (설계 조건)
평가 기준 카드: 안전·공정·재미·실현가능성
프로토타입 설계지
```

## 기본 규칙

```
재료 뽑기:
  각 카테고리에서 카드 1장씩
  예) 도구: 풍선 / 장소: 운동장
      인원: 6명 / 목표: 점수 누적

규칙 제약 카드:
  추가 조건 1장
  예) "신체 접촉 금지"
      "모든 팀원 참여 필수"

스포츠 설계 (10분):
  4가지 재료 + 제약 조건으로
  새로운 스포츠 규칙 설계
  이름·규칙·승리 조건 작성

발표 및 평가:
  각자 설계한 스포츠 발표
  평가 기준으로 투표
```

## 재료 카드 예시

```
도구: 풍선·훌라후프·줄넘기·공·신문지·의자

장소: 운동장·체육관·교실·복도·수영장

인원: 2명·4명·6명·팀전·모두

목표: 점수 누적·시간 내 완수·상대 점수 차단
      마지막까지 생존·먼저 도착
```

## 규칙 설계 가이드

```
좋은 스포츠 규칙의 조건:

공정성:
  모든 참여자에게 같은 기회
  특정 신체 조건 유리하지 않음

안전성:
  부상 위험 최소화
  명확한 금지 행동

재미:
  적절한 긴장감
  역전 가능성 존재

명확성:
  판단 기준이 명확
  심판 없어도 진행 가능
```

## 실제 스포츠 탄생 사례

```
농구 (1891):
  제임스 네이스미스가
  겨울 실내 운동으로 설계
  처음 규칙: 13가지뿐

배구 (1895):
  농구보다 덜 격렬한 스포츠로
  사무직 직원을 위해 설계

피클볼 (1965):
  아이들을 위해
  탁구·배드민턴·테니스 결합

교훈:
  모든 스포츠는 누군가가 설계했다
```

## 교육적 효과

```
규칙 사고:
  규칙의 목적과 기능
  → 법·제도 이해의 기초

공정성:
  모두에게 공평한 조건 설계
  → 사회 정의 사고

창의 설계:
  제약 속 새로운 것 만들기
  → 디자인 씽킹
```
