# 페어플레이 딜레마 (Fair Play Dilemma)

> 이기기 위해 반칙을 해도 되는가?
> 규정을 어기지 않았지만
> 정신을 어긴 플레이는 정당한가?
> 스포츠맨십의 본질을 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 스포츠맨십, 윤리적 판단, 규칙의 정신 이해 |

## 게임 구성

```
딜레마 카드: 35장 (스포츠 윤리 상황)
입장 카드: 허용·불허·조건부 허용
논거 카드: 스포츠맨십 원칙 20장
심판 역할 카드
```

## 기본 규칙

```
상황 공개:
  딜레마 카드 1장 낭독

비밀 입장:
  허용·불허·조건부 중 선택
  동시 공개

토론 (각 90초):
  자신의 입장 논거 발표

재투표:
  마음이 바뀌었으면 변경 가능

득점:
  다수 입장: +1점
  가장 설득력 있는 논거: +2점
```

## 딜레마 카드 예시

```
딜레마 1 - 시간 끌기:
  "1-0으로 이기는 팀이
   후반 추가시간에 의도적으로
   공을 코너에 몰아 시간을 끈다"
  허용? 불허? 왜?

딜레마 2 - 부상 연기:
  "상대 선수가 부상당했지만
   의도적으로 경기를 멈추지 않고
   득점했다"

딜레마 3 - 도핑 경계:
  "금지 약물은 아니지만
   효과가 비슷한 보충제를 복용했다"

딜레마 4 - 심판 속이기:
  "파울을 당하지 않았지만
   파울 당한 척 쓰러졌다"

딜레마 5 - 상대 정보:
  "상대 팀의 전술 회의를
   우연히 엿들었다"

딜레마 6 - 규정 허점:
  "규정 허점을 이용해
   이득을 봤다. 규정 위반은 아님"
```

## 스포츠맨십 원칙 카드

```
존중:
  상대·심판·규칙에 대한 존중

정직:
  결과를 조작하지 않는 태도

공정:
  모두에게 동등한 기회

탁월성:
  최선을 다하는 태도

즐거움:
  게임 자체를 즐기는 태도
```

## 교육적 효과

```
규칙의 정신:
  문자적 규칙 vs 규칙의 목적
  → 법 정신 이해

스포츠맨십:
  승리보다 중요한 가치
  → 인성 교육

윤리적 판단:
  상황에 따른 도덕적 선택
  → 실천 윤리학
```

```
  "문화 표현인가, 스포츠맨십 위반인가?"
```
