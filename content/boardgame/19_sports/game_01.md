# 전술 감독 (Tactical Coach)

> 감독은 11명을 어떻게 배치할 것인가?
> 공격과 수비의 균형.
> 상대 전술을 읽고 대응하는
> 스포츠 전략의 세계.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 전술적 사고, 자원 배분, 상대 모델링 |

## 게임 구성

```
전술 보드: 축구장 그리드
선수 타일: 공격·미드필더·수비·골키퍼
전술 카드: 40장 (포메이션·전략)
상황 카드: 20장 (게임 중 상황)
점수판
```

## 기본 규칙

```
포메이션 선택:
  선수 타일로 포메이션 구성
  4-3-3 / 4-4-2 / 3-5-2 등
  상대 포메이션 모름

동시 공개:
  양측 포메이션 동시 공개

상황 카드:
  게임 상황 카드 3장 뽑기
  각 상황에서 포메이션 유불리 계산

득점:
  유리한 상황 수로 점수
  3전 2선승제
```

## 포메이션 특성

```
4-3-3 (공격형):
  공격: 강함 (+3)
  중원: 보통 (+2)
  수비: 약함 (+1)
  카운터 공격에 취약

4-4-2 (균형형):
  공격: 보통 (+2)
  중원: 강함 (+3)
  수비: 보통 (+2)
  모든 상황 무난

5-3-2 (수비형):
  공격: 약함 (+1)
  중원: 보통 (+2)
  수비: 강함 (+3)
  세트피스에 강함
```

## 상황 카드 예시

```
"후반 추가시간, 1골 차 리드"
→ 수비형 포메이션 유리

"상대 에이스 부재"
→ 공격형 포메이션 유리

"비와 강풍"
→ 롱볼 전술 유리

"전반전 퇴장으로 10명"
→ 수비 집중 포메이션
```

## 전술 카드

```
프레싱: 높은 라인 유지
          상대 빌드업 차단
          체력 소모 큼

스위퍼: 최후방 수비수 추가
         골키퍼 앞 보호막

윙 어택: 측면 공격 강화
          크로스 활용

티키타카: 짧은 패스 연결
           볼 점유율 극대화
```

## 교육적 효과

```
전술 사고:
  자원(선수) 최적 배치
  → 경영·군사 전략과 연결

상대 모델링:
  상대 전술 예측·대응
  → 게임 이론적 사고

상황 판단:
  변수에 따른 전략 수정
  → 유연한 의사결정
```

## 다른 스포츠 확장

```
농구 전술:
  맨투맨 vs 지역 방어
  픽앤롤·아이솔레이션

야구 전술:
  수비 시프트
  번트·도루 타이밍

배구 전술:
  서브 리시브 대형
  공격 패턴
```
