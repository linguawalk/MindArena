# 장면 전환 (Scene Switch)

> 이야기는 계속 흘러야 합니다.
> "전환!" 한 마디에
> 완전히 다른 장면이 펼쳐집니다.
> 빠른 전환 속에서 이야기를 이어갑니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 적응력, 창의적 연결, 빠른 사고 |

## 기본 규칙

```
기본 장면 시작:
  2명이 즉흥 연기 시작
  관객이 지켜보기

전환 신호:
  진행자 또는 관객이 "전환!" 외치기
  연기자 즉시 멈추기

새 장면:
  다른 2명이 교체 또는
  같은 연기자가 완전히 다른 장면 시작

연결:
  새 장면은 이전 장면과
  어떤 식으로든 연결되어야 함
  (단어·이미지·감정·주제)
```

## 전환 유형

```
인물 전환:
  같은 장소, 다른 인물
  이전 인물 대사가 힌트

시간 전환:
  같은 인물, 다른 시간
  10년 전·후

공간 전환:
  같은 시간, 다른 장소
  동시에 일어나는 일

감정 전환:
  반대 감정으로 전환
  슬픔 → 기쁨
```

## 진행자 지시 카드

```
"느린 동작으로"
"모두 침묵으로"
"노래로만"
"동물처럼"
"로봇처럼"
"역방향으로 (뒤로 걷기)"
```

## 교육적 효과

```
적응력:
  갑작스러운 변화에 즉각 반응
  → 변화 대응 능력

창의적 연결:
  이질적인 것들 연결
  → 연상 사고력

팀 호흡:
  서로의 리듬 맞추기
  → 협력 능력
```

## 고급 장면 전환

```
타임루프:
  같은 장면이 반복되지만
  매번 조금씩 달라짐

평행 우주:
  같은 순간, 다른 선택
  두 장면 동시 진행

이야기 속 이야기:
  연기 중 다른 이야기 시작
  → 메타 픽션 구조
```
