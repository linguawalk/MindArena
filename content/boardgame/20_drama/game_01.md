# 즉흥극 배틀 (Improv Battle)

> 대본 없이 무대에 서세요.
> 상황이 주어지면 즉시 연기합니다.
> 실수가 허용되는 안전한 공간에서
> 즉흥성과 창의성을 훈련합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 즉흥성, 경청, 협력적 창작, 표현력 |

## 게임 구성

```
상황 카드: 50장 (장소·사건·관계)
캐릭터 카드: 30장 (직업·성격·특징)
제약 카드: 20장 (연기 조건)
평가 토큰: 플레이어당 3개
```

## 기본 규칙

```
팀 구성:
  2-3명씩 팀 나누기

상황 뽑기:
  상황 카드 1장 + 캐릭터 카드 각자 1장
  30초 준비

즉흥 연기 (3분):
  주어진 상황에서 즉흥 연기
  캐릭터 유지하며 이야기 전개

평가:
  관객(다른 팀)이 토큰으로 평가
  자연스러움·창의성·협력도

득점:
  받은 토큰 합산
```

## 즉흥극 황금 규칙

```
"Yes, And..." 원칙:
  상대방의 설정을 절대 거부하지 않기
  받아들이고 발전시키기
  예) "저는 외계인이에요" →
      "네, 그래서 지구 음식 처음 드셔보셨겠네요"

경청:
  상대가 만드는 세계에 집중
  내 차례만 기다리지 않기

실수 환영:
  "실수는 선물"
  예상치 못한 전개가 더 재미있음

캐릭터 유지:
  어떤 상황에도 캐릭터 일관성
```

## 상황 카드 예시

```
"로봇이 처음으로 감정을 느낀 순간"
"실수로 시간여행을 한 역사학자"
"서로 언어가 다른 두 사람의 만남"
"마지막 피자 한 조각을 두고"
"무인도에서 핸드폰이 터지는 순간"
```

## 캐릭터 카드 예시

```
직업: 탐정·요리사·우주비행사·철학자
성격: 지나치게 낙관적·모든 것을 의심·말이 없는
특징: 노래로만 말하는·왼손잡이 강조·시간 강박
```

## 제약 카드 예시

```
"대사 중 숫자 사용 금지"
"3초마다 위치 바꾸기"
"한 문장씩 번갈아 말하기"
"감정을 과장해서 표현"
"느린 동작으로만"
```

## 교육적 효과

```
즉흥성:
  계획 없이 반응하는 능력
  → 실생활 대화 능력

협력 창작:
  함께 이야기 만들기
  → 팀워크·경청

자기표현:
  몸·목소리·표정 활용
  → 비언어 소통 능력
```

## 게임 후 토론

```
"즉흥극에서 가장 어려운 순간은?
 상대가 예상 밖의 설정을 했을 때
 어떻게 반응했나요?"

"'Yes, And' 원칙이 일상 대화에서도
 적용될 수 있을까요?"
```
