# 감정 연기 (Emotion Acting)

> 같은 대사도 감정에 따라 달라집니다.
> "안녕하세요"가 분노로,
> 슬픔으로, 기쁨으로 표현될 때
> 얼마나 다른지 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 25-35분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 감정 표현, 감정 인식, 공감 능력 |

## 게임 구성

```
대사 카드: 30장 (일상적인 짧은 대사)
감정 카드: 20장 (다양한 감정)
강도 카드: 약함·보통·강함
평가 토큰
```

## 기본 규칙

```
대사 + 감정 뽑기:
  대사 카드 1장 + 감정 카드 1장
  강도 카드 1장

연기 준비 (30초):
  해당 감정으로 대사 표현 방법 생각

연기:
  대사를 감정·강도에 맞게 표현

맞히기:
  관객이 어떤 감정인지 맞히기

득점:
  맞힌 관객: +2점
  표현자: 맞힌 수 × 1점
```

## 대사 카드 예시

```
"오늘 날씨 참 좋네요"
"밥 먹었어요?"
"그거 어디서 샀어요?"
"잠깐만요"
"알겠습니다"
"고마워요"
"괜찮아요"
```

## 감정 카드 예시

```
기본 감정:
  기쁨·슬픔·분노·공포·혐오·놀라움

복합 감정:
  질투·그리움·당황·자랑스러움
  죄책감·안도·지루함·설렘

미묘한 감정:
  억울함·서운함·뿌듯함·허탈함
```

## 비언어 표현 요소

```
목소리:
  높낮이·빠르기·크기·떨림·쉼

표정:
  눈썹·눈·입·근육 긴장도

몸짓:
  자세·손 동작·움직임·거리

호흡:
  빠른·느린·얕은·깊은
```

## 심화 모드

```
감정 변화:
  한 대사 안에서 감정이 바뀌기
  예) 기쁨으로 시작 → 슬픔으로 끝

감정 숨기기:
  슬프지만 기쁜 척하기
  → 위장된 감정 표현

문화 비교:
  같은 감정, 다른 문화권 표현 방식
```

## 교육적 효과

```
감정 어휘:
  다양한 감정 이름 알기
  → 정서 지능 기초

감정 인식:
  타인의 감정 읽기
  → 공감 능력

자기 표현:
  내면 감정을 밖으로 표현
  → 정서 표현 능력
```

## 게임 후 토론

```
"어떤 감정이 가장 표현하기 어려웠나요?
 왜 그랬을까요?"

"같은 감정인데 사람마다
 표현 방식이 달랐나요?"

"감정을 잘 읽는 능력이
 실생활에서 어떻게 도움이 될까요?"
```
