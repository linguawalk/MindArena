# 한 줄 연극 (One Line Theater)

> 한 줄의 대사로 모든 것을 표현합니다.
> 배우는 한 줄만 말할 수 있습니다.
> 어떤 한 줄을 선택하느냐가
> 이야기를 결정합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-8명 |
| 시간 | 20-30분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 간결한 표현, 핵심 전달, 창의적 해석 |

## 기본 규칙

```
장면 설정:
  진행자가 상황 설명
  예) "재회한 오랜 친구들"

한 줄 규칙:
  각 플레이어 딱 한 문장만 말하기
  순서대로 돌아가며

이야기 전개:
  한 줄씩 모여 이야기 완성
  모순 없이 자연스럽게

마무리:
  마지막 플레이어가 결말 짓기
```

## 한 줄의 힘

```
좋은 한 줄의 조건:
  구체적: "좋았어" 보다 "그날 비가 왔지"
  전진: 이야기를 앞으로 나아가게
  여운: 다음 사람이 이어갈 여지
  캐릭터: 그 사람만의 말투·관점

피해야 할 한 줄:
  이야기 막기: "아무것도 없었다"
  너무 결론적: 혼자 끝내려는 시도
  무관한 내용: 흐름 끊기
```

## 장면 카드 예시

```
인간 관계:
  "20년 만에 만난 첫사랑"
  "오늘 처음 만난 룸메이트"
  "마지막 수업의 선생님과 학생"

특수 상황:
  "우주선 마지막 산소 30분"
  "타임머신에서 내린 사람들"
  "모두가 같은 꿈을 꾼 아침"

일상:
  "오래된 카페의 단골 손님들"
  "엘리베이터에 갇힌 낯선 사람들"
```

## 변형 모드

```
장르 한 줄:
  공포·로맨스·코미디 장르로 한 줄
  같은 상황 다른 장르

침묵 한 줄:
  대사 없이 몸짓만으로 한 줄
  비언어 표현 훈련

반전 한 줄:
  이전 내용을 뒤집는 한 줄
  → 반전 구성 훈련
```

## 교육적 효과

```
간결함:
  핵심만 남기는 능력
  → 글쓰기·발표 능력

협력 서사:
  함께 만드는 이야기
  → 공동 창작 능력

경청:
  앞 사람 말을 듣고 이어가기
  → 대화 능력
```
