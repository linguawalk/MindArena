# 캐릭터 빌더 (Character Builder)

> 인물은 어떻게 만들어지는가?
> 직업·욕망·두려움·과거.
> 입체적인 캐릭터를 설계하고
> 그 인물로 살아보세요.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 캐릭터 이해, 공감 능력, 창의적 설계 |

## 게임 구성

```
캐릭터 요소 카드:
  직업 카드: 30장
  욕망 카드: 20장 (원하는 것)
  두려움 카드: 20장 (피하고 싶은 것)
  과거 카드: 20장 (배경 사건)
  버릇 카드: 15장 (특이한 습관)

상황 카드: 25장
캐릭터 시트: 개인용
```

## 기본 규칙

```
캐릭터 생성:
  각 카테고리에서 카드 1장씩 뽑기
  조합으로 캐릭터 완성

캐릭터 시트 작성:
  이름·나이 자유 설정
  5가지 요소 기록
  한 줄 자기소개 작성

상황 카드:
  상황 카드 공개
  각자 자신의 캐릭터로 반응

평가:
  캐릭터 일관성 + 창의성 평가
```

## 캐릭터 카드 예시

```
직업: 은퇴한 마술사 / 심해 탐험가
     AI 윤리학자 / 향수 조향사

욕망: 잊혀지고 싶다 / 모든 것을 알고 싶다
     누군가의 영웅이 되고 싶다

두려움: 평범해지는 것 / 혼자 남겨지는 것
       진실을 알게 되는 것

과거: 어린 시절 기억을 잃었다
     한 번의 실수로 모든 것을 잃었다
     비밀을 하나 품고 살아간다

버릇: 항상 모자를 쓴다
     대화 중 눈을 마주치지 않는다
     모든 것을 수첩에 기록한다
```

## 상황 카드 예시

```
"갑자기 정전이 됐다"
"옛 친구를 예상치 못한 곳에서 만났다"
"중요한 물건을 잃어버렸다"
"누군가 도움을 요청해 왔다"
"비밀이 드러날 위기에 처했다"
```

## 입체적 캐릭터의 조건

```
평면 캐릭터:
  착한 사람 / 나쁜 사람
  한 가지 특성만

입체 캐릭터:
  선하지만 두려움 있음
  욕망과 두려움이 충돌
  과거가 현재에 영향

좋은 캐릭터 공식:
  원하는 것 (욕망)
  + 가로막는 것 (두려움·장애)
  + 이유 (과거)
  = 입체적 인물
```

## 교육적 효과

```
공감 능력:
  다른 사람의 관점으로 보기
  → 사회적 상상력

서사 이해:
  인물이 이야기를 만든다
  → 문학 이해력

자기 이해:
  캐릭터 설계를 통한 자기 탐구
  → 심리적 자기 인식
```

## 게임 후 토론

```
"만든 캐릭터 중 자신과
 가장 비슷한 면이 있는 캐릭터는?
 어떤 점이 비슷했나요?"

"욕망과 두려움이 충돌할 때
 인물은 어떻게 행동할까요?"

"좋아하는 소설·영화 속 인물을
 이 방법으로 분석해 보면?"
```

