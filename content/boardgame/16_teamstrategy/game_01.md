# 침묵의 팀 (Silent Team)

> 말 없이 협력할 수 있는가?
> 언어 없이 팀원과 소통하며
> 공동 목표를 달성하는 게임.
> 비언어 소통의 힘을 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 비언어 소통, 팀 조율, 공감 능력 |

## 게임 구성

```
공동 목표 카드: 20장
행동 타일: 각 플레이어 10개
신호 카드: 몸짓·눈빛·위치 신호 가이드
결과 점수판
```

## 기본 규칙

```
규칙:
  게임 내내 말하기 금지
  글쓰기·손가락 글자 금지
  눈빛·몸짓·위치만 사용

목표:
  공동 목표 카드의 과제 완수
  예) "팀원 모두 같은 색 타일 선택"
      "숫자 합이 20이 되도록 배치"
      "특정 패턴 완성"

시간 제한:
  3분 내 완수

실패 시:
  다시 시도 (최대 3회)
```

## 목표 카드 예시

```
패턴 맞추기:
  "모두 같은 색 타일을 앞에 놓기"
  → 누가 먼저 색을 정할 것인가?

숫자 합산:
  "타일 숫자의 합이 정확히 15"
  → 각자 얼마를 낼 것인가?

순서 정하기:
  "키 순서대로 줄 서기"
  → 서로 비교하며 위치 조정

역할 분담:
  "A는 왼쪽, B는 오른쪽 구역 담당"
  → 누가 어느 역할인지 합의
```

## 허용되는 신호

```
눈빛: 시선 방향·눈썹 움직임
고개: 끄덕임·가로젓기·기울임
손: 가리키기·손바닥 방향
몸: 위치 이동·방향 전환
표정: 동의·반대·확인 표현
```

## 교육적 효과

```
비언어 소통:
  언어 외 소통 채널 인식
  → 전체 소통의 70% 비언어

공감:
  상대방 의도 파악
  → 감정 이입 능력

팀 조율:
  공통 이해 없이 협력
  → 암묵적 협력 능력
```

## 변형 모드

```
부분 침묵:
  한 명만 말할 수 있음
  → 리더십 역할 탐구

역할 침묵:
  특정 역할만 말 못함
  → 포용적 소통 훈련

시간 압박:
  1분 내 완수
  → 긴박한 팀 소통
```

## 게임 후 토론

```
"말 없이 소통하면서
 어떤 점이 가장 어려웠나요?"

"팀원 중 가장 잘 소통한 사람은?
 어떤 신호가 효과적이었나요?"

"실제 팀 작업에서도
 비언어 소통이 중요한 경우는?"
```
