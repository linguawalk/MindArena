# 집단 지성 (Collective Intelligence)

> 한 명의 전문가보다
> 다양한 비전문가 집단이
> 더 정확할 수 있습니다.
> 집단 지성의 조건과 한계를 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 30-45분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 집단 의사결정, 다양성 가치, 합의 형성 |

## 게임 구성

```
추정 문제 카드: 40장 (정답이 있는 추정 문제)
개인 답안지: 플레이어당 10장
집단 답안지: 라운드당 1장
전문가 카드: 특수 역할 10장
```

## 기본 규칙

```
개인 추정:
  문제 카드 공개
  각자 비밀리에 답 적기
  예) "서울 인구는 몇 명인가?"

평균 계산:
  개인 답 평균 = 집단 답

전문가 판단:
  전문가 카드 보유자 1명이
  최종 답 조정 가능

오차 비교:
  개인 최선 답 vs 집단 평균 vs 전문가 답
  어느 것이 정답에 가까운가?
```

## 추정 문제 예시

```
수치 추정:
  "에베레스트 산의 높이(m)?"
  "대한민국 국민 평균 나이?"
  "현재 지구상 나무의 수?"

확률 추정:
  "동전 10번 던져 7번 이상 앞면 확률?"
  "한국에서 100세 이상 인구 비율?"

가치 추정:
  "모나리자의 보험 가액은?"
  "2024년 한국 GDP는?"
```

## 집단 지성의 조건

```
작동하는 조건:
  다양성: 다른 배경·관점
  독립성: 서로 영향받지 않고 판단
  분산화: 다양한 정보 접근
  합산: 적절한 집계 방법

실패하는 조건:
  군중 심리: 남을 따라하기
  정보 폭포: 앞사람 답 보고 따르기
  집단 사고: 반대 의견 억압
```

## 실험 라운드

```
라운드 1: 완전 독립 추정
  서로 답 모르고 진행
  → 진정한 집단 지성

라운드 2: 순차 공개
  앞사람 답 보고 다음 사람 추정
  → 정보 폭포 실험

라운드 3: 토론 후 추정
  15분 토론 후 개인 답
  → 집단 사고 실험

세 라운드 결과 비교:
  어느 방법이 가장 정확했는가?
```

## 교육적 효과

```
집단 지성:
  다양성이 정확도를 높임
  → 팀 구성의 중요성

독립적 사고:
  남의 의견에 흔들리지 않기
  → 비판적 사고력

합의 vs 평균:
  집단 결정의 다양한 방법
  → 의사결정 메커니즘
```
