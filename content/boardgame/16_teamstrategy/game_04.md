# 리더 없는 팀 (Leaderless Team)

> 리더가 없다면 팀은 어떻게 움직이는가?
> 자연스럽게 리더가 생기는가?
> 모두가 평등하게 결정할 수 있는가?
> 리더십의 본질을 탐구합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-6명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 자기 조직화, 리더십 본질, 집단 역학 |

## 게임 구성

```
복잡한 임무 카드: 15장
역할 금지 카드: "리더 금지" 표시
관찰 기록지: 관찰자용
행동 분석표
```

## 기본 규칙

```
규칙:
  공식 리더 없음
  역할 배정 없음
  단, 임무는 완수해야 함

임무 수행:
  복잡한 임무 카드 1장 받기
  15분 내 팀 합의로 완수

관찰:
  1명이 관찰자 역할 (돌아가며)
  팀 역학 기록

성찰:
  임무 완수 후 역학 분석
```

## 임무 카드 예시

```
임무 1 - 자원 배분:
  "팀에 100점이 주어졌습니다.
   5가지 프로젝트에 배분하세요.
   단, 모든 팀원이 동의해야 합니다."

임무 2 - 갈등 해결:
  "두 가지 상충되는 방향 중
   팀의 방향을 정하세요."

임무 3 - 창의 과제:
  "팀 이름과 로고, 슬로건을 만드세요.
   모두가 만족해야 합니다."

임무 4 - 위기 대응:
  "갑작스러운 위기 상황에서
   즉각적인 대응 계획을 세우세요."
```

## 관찰 기록지

```
누가 먼저 말을 꺼냈나?
누구의 의견이 가장 많이 채택됐나?
반대 의견은 어떻게 처리됐나?
침묵한 사람은 누구인가?
갈등이 있었다면 어떻게 해결됐나?
비공식 리더가 나타났는가?
```

## 리더십 패턴 분석

```
자연 발생 리더십 유형:
  전문성 리더: 가장 잘 아는 사람
  관계 리더: 조율·중재하는 사람
  추진 리더: 결정하고 밀어붙이는 사람
  지원 리더: 뒤에서 돕는 사람

리더 없는 팀의 함정:
  아무도 결정 안 함 (마비)
  강한 한 명이 사실상 독점
  시간 낭비·비효율
```

## 교육적 효과

```
리더십 본질:
  리더는 만들어지는가, 생겨나는가?
  → 리더십 이론 체험

집단 역학:
  공식 구조 없이 작동하는 팀
  → 조직 행동학 이해

자기 인식:
  자신의 자연스러운 역할
  → 리더십 강점 발견
```

## 게임 후 토론

```
"리더 없이도 잘 작동했나요?
 어떤 문제가 생겼나요?"

"자연스럽게 리더 역할을 한 사람은?
 왜 그 사람이 리더가 됐을까요?"

"좋은 리더의 조건은 무엇인가요?"
```
