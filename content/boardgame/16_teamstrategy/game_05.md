# 최후의 합의 (Final Consensus)

> 모든 팀원이 동의해야 합니다.
> 단 한 명의 반대도 허용되지 않습니다.
> 완전한 합의는 가능한가?
> 협력과 설득의 극한 훈련.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐⭐⭐ |
| 목표 역량 | 완전 합의, 경청, 설득, 양보의 기술 |

## 게임 구성

```
합의 주제 카드: 25장 (난이도별)
입장 카드: 강한 동의·동의·중립·반대·강한 반대
양보 카드: 20장 (조건부 수용 가능 사항)
시간 트랙: 30분 제한
합의 보드: 전원 동의 시 완성
```

## 기본 규칙

```
주제 공개:
  합의 주제 카드 1장
  예) "우리 팀의 5가지 핵심 가치 선정"

초기 입장:
  각자 비밀 입장 카드 선택
  동시 공개

토론:
  반대자는 반대 이유 설명
  찬성자는 반대자 설득 시도
  양보 카드로 조건 제시 가능

재투표:
  토론 후 다시 입장 표명
  전원 동의 시 합의 성공

시간 제한:
  30분 내 합의 못 하면 실패
```

## 합의 주제 카드 예시

```
쉬운 주제:
  "팀 이름을 정하세요"
  "오늘 점심 메뉴를 정하세요"

중간 주제:
  "팀의 가장 중요한 규칙 1가지"
  "우선 해결할 문제 선택"

어려운 주제:
  "팀원 중 대표 1명 선출"
  "제한된 자원 배분 방식 결정"
  "반드시 하나를 포기해야 할 때"
```

## 양보 카드 예시

```
"이 조건이라면 동의합니다:
  ①다음 결정 때 내 의견 우선 반영
  ②특정 사항만 제외하고 동의
  ③일정 기간 후 재검토 조건
  ④내가 원하는 역할 배정"
```

## 완전 합의의 어려움

```
만장일치 함정:
  진짜 동의 vs 압박에 의한 동의
  → 진정성 있는 합의만 유효

소수 보호:
  1명의 반대를 무시하지 않기
  → 그 반대가 중요한 이유일 수 있음

시간 압박:
  합의 시간이 길수록 피로
  → 급한 양보 vs 진정한 합의
```

## 교육적 효과

```
경청:
  반대 의견을 끝까지 듣기
  → 진정한 경청 훈련

설득:
  논리·감정·이익으로 설득
  → 다양한 설득 방법

양보의 기술:
  내가 원하는 것 포기하기
  → 협상의 핵심 기술
```
