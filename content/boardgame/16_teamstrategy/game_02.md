# 역할 교환 (Role Swap)

> 리더가 실무자가 되고
> 실무자가 리더가 됩니다.
> 역할이 바뀌면 관점이 바뀝니다.
> 모든 역할을 이해하는 팀이 강합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 역할 이해, 공감, 다기능 팀워크 |

## 역할 구성

```
기본 역할 4개:
  리더: 전체 방향 결정·조율
  분석가: 정보 수집·분석·보고
  실행가: 결정 사항 실행
  소통가: 내외부 소통·문서화

추가 역할 (8인 시):
  비판가: 약점·리스크 지적
  창의가: 새로운 아이디어 제안
  조정자: 갈등 중재
  지원자: 다른 역할 보조
```

## 게임 구성

```
임무 카드: 30장 (팀 과제)
역할 카드: 각 역할당 1장
역할 가이드: 역할별 행동 지침
평가 시트: 역할 수행도 평가
```

## 기본 규칙

```
1라운드:
  역할 배정
  임무 카드 1장 수행 (10분)
  역할 수행도 평가

역할 교환:
  모든 플레이어 역할 이동
  (리더→분석가→실행가→소통가→리더)

2-4라운드:
  동일 임무 또는 새 임무 반복
  매 라운드 다른 역할 경험

성찰:
  4개 역할 모두 경험 후
  어떤 역할이 가장 어려웠는지 토론
```

## 임무 카드 예시

```
임무 1: 새로운 학교 급식 메뉴 제안
  분석가: 영양·비용·선호도 조사
  창의가: 새로운 메뉴 아이디어
  리더: 최종 메뉴 3개 선정
  소통가: 제안서 작성

임무 2: 학급 규칙 5가지 만들기
임무 3: 동네 문제 해결 방안 제시
임무 4: 팀 프로젝트 1주일 계획 수립
```

## 역할별 어려움

```
리더:
  "모든 것을 결정해야 한다는 부담"
  "팀원이 동의하지 않을 때"

분석가:
  "충분한 정보가 없을 때"
  "분석은 했는데 무시될 때"

실행가:
  "지시가 불명확할 때"
  "실행 중 문제가 생길 때"

소통가:
  "모두를 이해시키기 어려울 때"
  "기록이 많아질 때"
```

## 교육적 효과

```
역할 공감:
  다른 역할의 어려움 직접 체험
  → 팀 내 상호 이해

다기능성:
  여러 역할 수행 능력
  → 팀 유연성

자기 인식:
  어떤 역할이 자신에게 맞는가
  → 강점 발견
```

## 게임 후 토론

```
"어떤 역할이 가장 어려웠나요?
 예상과 달랐던 점은?"

"리더가 가장 중요한가요,
 아니면 모든 역할이 동등한가요?"

"팀에서 자신의 자연스러운 역할은
 무엇인 것 같나요?"
```
