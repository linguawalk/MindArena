# 기후 협상가 (Climate Negotiator)

> 기후 변화는 모두의 문제이지만
> 해결 비용은 불평등하게 분배됩니다.
> 선진국과 개발도상국, 현재와 미래.
> 기후 협상의 딜레마를 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 4-8명 |
| 시간 | 45-60분 |
| 난이도 | ⭐⭐⭐⭐⭐ |
| 목표 역량 | 기후 정책 이해, 국제 협상, 세대 간 형평성 |

## 역할 카드

```
선진국 그룹:
  미국: GDP 최대, 역사적 최대 배출국
  EU: 친환경 선도, 탄소세 도입
  일본: 기술력, 경제 의존도

개발도상국 그룹:
  중국: 현재 최대 배출국, 성장 필요
  인도: 빠른 성장, 석탄 의존
  아프리카 연합: 최소 배출, 최대 피해

특수 역할:
  군소도서국: 해수면 상승 피해
  석유 수출국: 화석연료 경제
  미래 세대 대표: 2100년 시점
```

## 게임 구성

```
국가 보드: 역할당 1개 (GDP·배출량·취약성)
협상 카드: 40장 (제안·요구·조건)
기후 트랙: 현재 온도 상승 추이
시간 트랙: 협상 기한 (10라운드)
합의 보드: 공동 결의문
```

## 협상 의제

```
의제 1: 감축 목표
  "2050년까지 탄소중립 달성"
  각국 감축량 배분 협상

의제 2: 재원 조달
  "선진국이 개도국에 1조 달러 지원"
  누가 얼마나 낼 것인가

의제 3: 기술 이전
  "청정 기술 특허 공유"
  지식재산권 vs 기후 위기

의제 4: 손실과 피해
  "기후 피해국에 보상"
  역사적 책임 vs 현재 역량

의제 5: 감축 검증
  "독립적 검증 기구 설치"
  주권 vs 투명성
```

## 기본 규칙

```
포지션 설정:
  역할에 맞는 초기 입장 정하기

협상 라운드:
  ①제안 카드로 협상안 제시
  ②다른 국가 수락·거부·수정
  ③연합 형성 가능

기후 변화:
  합의 실패 시 기후 트랙 악화
  특정 국가 피해 발생

최종 합의:
  10라운드 내 공동 결의문 채택
  미채택 시 기후 재앙
```

## 핵심 딜레마

```
무임 승차:
  합의 안 해도 혜택 받음
  → 공유지의 비극

역사적 책임:
  과거 배출로 오늘 혜택 본 국가
  → 누가 더 감축해야 하는가

개발권:
  가난한 나라도 성장할 권리
  → 기후 vs 경제 발전
```

## 교육적 효과

```
국제 정치:
  국가 이익의 충돌과 협력
  → 국제 관계 이해

기후 과학:
  온도 상승과 영향
  → 기후 변화 지식

협상 능력:
  다자 협상 전략
  → 국제 협력 사고
```
