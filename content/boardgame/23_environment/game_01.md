# 먹이 사슬 전략 (Food Chain Strategy)

> 작은 것이 큰 것에게 먹히고
> 큰 것이 사라지면 작은 것이 넘칩니다.
> 먹이 사슬의 균형이 무너지면
> 생태계 전체가 흔들립니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-5명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐ |
| 목표 역량 | 생태계 이해, 시스템 사고, 균형 관리 |

## 게임 구성

```
생물 타일: 생산자·1차소비자·2차소비자·최상위포식자
  각 20개
생태계 보드: 5×5 그리드
에너지 토큰: 50개
환경 카드: 20장 (기후·재해·인간 개입)
```

## 생태계 구성

```
에너지 흐름:
  태양 → 식물 (생산자)
         ↓
       초식동물 (1차 소비자)
         ↓
       육식동물 (2차 소비자)
         ↓
       최상위 포식자

에너지 효율:
  각 단계에서 10%만 전달
  → 상위로 갈수록 개체수 적음
```

## 기본 규칙

```
생태계 설정:
  보드에 생물 타일 배치
  에너지 토큰 분배

매 라운드:
  ①에너지 생산 (식물 광합성)
  ②먹이 관계 적용
    (포식자가 먹이 개체수 감소)
  ③번식 (에너지 충분 시)
  ④환경 카드 적용

균형 붕괴:
  특정 개체 멸종 시
  연쇄 반응 발생

승리:
  10라운드 후 가장 안정적 생태계
```

## 환경 카드 예시

```
자연 재해:
  "가뭄": 식물 에너지 -30%
  "산불": 특정 지역 생물 -50%

인간 개입:
  "벌목": 생산자 구역 제거
  "포획": 최상위 포식자 -3
  "외래종 유입": 새 포식자 등장

기후:
  "온난화": 일부 종 번성, 일부 종 쇠퇴
  "한파": 변온동물 활동 감소
```

## 핵심 실험

```
늑대 재도입 실험 (옐로스톤):
  늑대 제거 → 사슴 급증
  → 식물 파괴 → 강변 침식
  → 강 흐름 변화

늑대 재도입 → 사슴 감소
  → 식물 회복 → 강변 안정
  → 강 흐름 정상화

"늑대가 강을 바꿨다"
→ 영양 연쇄 효과 체험
```

## 교육적 효과

```
시스템 사고:
  연결된 요소들의 상호작용
  → 복잡계 이해

균형 감각:
  과다·과소 모두 문제
  → 지속 가능성 사고

생태 지식:
  먹이 사슬·에너지 흐름
  → 생물학 기초
```
