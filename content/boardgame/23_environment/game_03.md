# 생태계 설계자 (Ecosystem Designer)

> 황폐해진 땅에 생태계를 복원합니다.
> 어떤 종을 먼저 도입해야 하는가?
> 순서가 틀리면 실패합니다.
> 생태적 천이를 이해하는 게임.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 2-4명 |
| 시간 | 40-55분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 생태적 천이, 종간 관계, 복원 생태학 |

## 게임 구성

```
황폐지 보드: 5×5 그리드 (초기 황무지)
생물 카드: 선구종·중간종·극상종 각 15장
상호작용 카드: 공생·경쟁·포식 관계 20장
환경 조건 트랙: 토양·수분·일조량
```

## 생태적 천이 단계

```
1단계 - 선구 단계:
  이끼·지의류·잡초
  척박한 환경에서 생존
  토양 형성 시작

2단계 - 초원 단계:
  풀·작은 초본식물
  토양 안정화
  곤충·소형 동물 서식 시작

3단계 - 관목 단계:
  작은 나무·관목
  토양 풍부해짐
  다양한 동물 유입

4단계 - 극상림 단계:
  큰 나무·다양한 생물
  안정적 생태계
  자기 조절 능력
```

## 기본 규칙

```
시작:
  황무지에서 시작
  선구종부터 도입 가능

생물 배치:
  적합한 환경 조건 확인
  전제 조건 충족 시 배치

천이 진행:
  선구종 → 환경 개선 → 다음 단계 가능

상호작용:
  공생: 서로 도움
  경쟁: 자원 다툼 → 한 종 감소
  포식: 개체수 조절

목표:
  극상림 달성한 구역 수로 승리
```

## 생물 카드 예시

```
선구종:
  이끼류: 암석 분해, 토양 형성
  질소고정 식물: 토양 영양 증가

중간종:
  참나무 묘목: 선구종 필요
  다람쥐: 나무 씨앗 분산
  지렁이: 토양 개선

극상종:
  참나무 성목: 중간 단계 필요
  독수리: 최상위 포식자
  버섯류: 분해자 역할
```

## 실제 복원 사례

```
체르노빌 제외 구역:
  인간 없자 생태계 자연 회복
  늑대·곰·희귀조 귀환

한국 광릉 숲:
  500년 보전된 극상림
  희귀종의 마지막 피난처

새만금:
  간척으로 생태계 파괴
  복원 시도와 어려움
```

## 교육적 효과

```
생태 지식:
  천이 과정·종간 관계
  → 생물학 심화 개념

시스템 사고:
  순서와 조건의 중요성
  → 복잡계 이해

환경 윤리:
  한번 파괴된 생태계 복원의 어려움
  → 보전의 중요성
```
