# 멸종 위기 구출 (Endangered Rescue)

> 하루에 150종이 사라집니다.
> 제한된 자원으로
> 어떤 종을 구할 것인가?
> 생물 다양성 보전의 딜레마.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 35-50분 |
| 난이도 | ⭐⭐⭐⭐ |
| 목표 역량 | 생물 다양성, 우선순위 결정, 보전 윤리 |

## 게임 구성

```
멸종 위기종 카드: 40장
  (실제 멸종 위기종 기반)
보전 자원 카드: 예산·인력·시간·기술 각 20개
서식지 보드: 5개 지역
멸종 타이머: 10라운드
```

## 멸종 위기종 카드 정보

```
각 카드 포함 정보:
  종 이름·사진
  현재 개체수
  멸종 위협 요인
  보전 비용 (자원)
  생태계 역할 (키스톤종 여부)
  상징성 (플래그십종 여부)
  고유성 (진화적 독특함)
```

## 기본 규칙

```
위협 단계:
  매 라운드 위협 카드 공개
  특정 종 개체수 감소

보전 결정:
  제한된 자원으로 어떤 종 구할지 결정
  팀 합의 필요

보전 활동:
  서식지 보호·개체수 증식
  밀렵 방지·외래종 제거

멸종:
  개체수 0 = 멸종
  생태계 영향 즉시 발생

승리:
  10라운드 후 가장 많은 종 생존
```

## 멸종 위기종 예시

```
카리스마틱 메가파우나:
  아무르표범 (개체수 < 100)
  마운틴고릴라 (개체수 < 1,000)
  → 보전 비용 높음, 인식 제고 효과

키스톤종:
  꿀벌: 식물 수분
  상어: 해양 생태계 균형
  → 생태적 영향 막대

숨겨진 중요종:
  특정 박쥐: 해충 방제
  특정 균류: 나무 성장 지원
  → 보전 우선순위 낮지만 중요
```

## 보전 딜레마

```
딜레마 1 - 카리스마 vs 기능:
  귀여운 팬더 vs 꼭 필요한 꿀벌
  어느 쪽을 먼저 구해야 하는가?

딜레마 2 - 비용 효과:
  한 종 구하는 데 드는 비용으로
  열 종을 부분적으로 도울 수 있다면?

딜레마 3 - 고유성:
  다른 어디에도 없는 종 vs
  생태계에 중요한 종

딜레마 4 - 인간 vs 야생:
  보전 지역 설정 시
  지역 주민 생계와 충돌
```

## 실제 보전 사례

```
성공 사례:
  흰꼬리수리 (DDT 금지 후 회복)
  혹등고래 (포경 금지 후 회복)
  한국 황새 (복원 사업 성공)

실패 사례:
  도도새: 이미 멸종
  태즈메이니아늑대: 마지막 1마리 사망
  양쯔강 돌고래: 공식 멸종 선언
```

## 교육적 효과

```
생물 다양성:
  다양성 = 생태계 안정성
  → 보전의 과학적 근거

윤리적 판단:
  어떤 생명이 더 중요한가?
  → 환경 윤리학

우선순위:
  제한된 자원의 최적 배분
  → 실용적 보전 정책
```
