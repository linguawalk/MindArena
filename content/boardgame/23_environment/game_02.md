# 탄소 발자국 (Carbon Footprint)

> 하루 동안 당신은 얼마나 많은
> 탄소를 배출하는가?
> 일상의 선택이 지구에 미치는 영향을
> 숫자로 체험합니다.

## 기본 정보

| 항목 | 내용 |
|------|------|
| 인원 | 3-6명 |
| 시간 | 30-40분 |
| 난이도 | ⭐⭐ |
| 목표 역량 | 환경 의식, 소비 패턴 분석, 실천적 환경 행동 |

## 게임 구성

```
생활 카드: 60장 (일상 활동·소비)
탄소 토큰: 각 활동당 배출량 표시
목표 카드: 개인 탄소 한도
절감 카드: 20장 (친환경 대안)
지구 건강 트랙
```

## 기본 규칙

```
하루 시뮬레이션:
  아침·점심·저녁·이동·여가
  각 시간대 생활 카드 선택

탄소 계산:
  선택한 활동의 탄소 합산
  목표 한도와 비교

절감 도전:
  절감 카드로 탄소 줄이기
  대안 생활 방식 선택

지구 건강:
  전체 플레이어 탄소 합산
  지구 건강 트랙 변화

승리:
  개인 목표 달성 + 지구 건강 유지
```

## 생활 카드 탄소량

```
이동:
  승용차 출퇴근 (10km): 2.3kg CO₂
  대중교통: 0.4kg CO₂
  비행기 (서울-제주): 90kg CO₂
  자전거·도보: 0kg CO₂

식품:
  소고기 스테이크 (200g): 5.4kg CO₂
  닭고기 (200g): 0.7kg CO₂
  채소 식사: 0.2kg CO₂
  커피 1잔: 0.2kg CO₂

에너지:
  에어컨 2시간: 0.8kg CO₂
  스마트폰 충전: 0.005kg CO₂
  세탁기 1회: 0.6kg CO₂

소비:
  티셔츠 1장 구매: 7kg CO₂
  스마트폰 1대: 70kg CO₂
```

## 절감 카드 예시

```
"오늘 채식 선택": -3kg CO₂
"대중교통 이용": -1.9kg CO₂
"에어컨 1도 높이기": -0.3kg CO₂
"중고 물건 구매": -5kg CO₂
"로컬 푸드 선택": -1kg CO₂
```

## 1인당 탄소 예산

```
현재 한국 1인 평균: 약 13톤/년 (35kg/일)
파리협정 목표: 2톤/년 (5.5kg/일)

→ 현재의 약 85% 감축 필요
→ 게임에서 이 차이를 체험
```

## 교육적 효과

```
환경 의식:
  일상 선택의 환경 영향
  → 실천적 환경 행동

수치 감각:
  탄소량을 구체적 숫자로 인식
  → 추상적 문제의 구체화

집단 행동:
  개인 행동이 지구에 미치는 영향
  → 사회적 책임 의식
```
