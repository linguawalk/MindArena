# 아이디어 전쟁 (Idea War)

> 아이디어가 세상을 바꿉니다.
> 자신의 아이디어를 발전시키고
> 다른 아이디어와 경쟁하며
> 사회적 영향력을 확대하는
> 지식 기반 전략 게임입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐⭐ (상) |
| 교육 분야 | 혁신 이론, 사회 변화, 아이디어 전파, 전략적 사고 |

---

## 게임 목표

10라운드 후 **사회적 영향력 점수**가
가장 높은 플레이어가 승리합니다.

또는 자신의 아이디어가 **사회 전체에 전파
(영향력 100 달성)**되면 즉시 승리합니다.

---

## 준비물

- 사회 영향력 보드 (5개 영역)
- 아이디어 카드 덱 (60장)
- 발전 카드 덱 (40장)
- 저항 카드 덱 (30장)
- 동맹 카드 20장
- 영향력 토큰 (플레이어별 색상)
- 사회 변화 트랙
- 지식 자원 토큰 (40개)
- 행동 포인트 토큰 (플레이어당 4개)
- 점수 기록판

### 사회 영향력 보드

```
┌──────────┬──────────┬──────────┐
│  학문·교육 │ 정치·정책  │ 경제·산업 │
│  영향력   │  영향력   │  영향력  │
│  [0-20]  │  [0-20]  │  [0-20]  │
├──────────┴──────────┴──────────┤
│         문화·예술                │
│           영향력                 │
│           [0-20]                │
├─────────────────────────────────┤
│         과학·기술                │
│           영향력                 │
│           [0-20]                │
└─────────────────────────────────┘

총 영향력 = 5개 영역 합산 (최대 100)
```

## 아이디어 카드 종류

```
혁명적 아이디어 (Revolutionary):
  기본 영향력: 15
  전파 속도: 빠름
  저항: 강함 (기득권의 저항)
  특성: 전파 시 연쇄 효과 (+5 추가)

점진적 아이디어 (Incremental):
  기본 영향력: 8
  전파 속도: 느림
  저항: 약함
  특성: 안정적 성장, 저항 거의 없음

융합 아이디어 (Fusion):
  기본 영향력: 12
  전파 속도: 중간
  저항: 중간
  특성: 두 분야를 결합, 시너지 효과

기초 아이디어 (Foundational):
  기본 영향력: 10
  전파 속도: 매우 느림
  저항: 없음
  특성: 다른 아이디어의 기반이 됨
        (이 아이디어 위에 다른 아이디어 발전 시 +3)

파괴적 아이디어 (Disruptive):
  기본 영향력: 20
  전파 속도: 빠름
  저항: 매우 강함
  특성: 기존 아이디어 1개 무효화 가능
        단, 실패 시 자신도 큰 타격
```

## 게임 준비

### 1. 시작 아이디어 선택

각 플레이어가 아이디어 카드 3장을 받고
그 중 1장을 선택해서 자신의 초기 아이디어로 설정합니다.

### 2. 초기 영향력 배치

각 플레이어가 자신의 초기 아이디어를
가장 잘 맞는 영역에 배치합니다.

```
예시:
  과학적 발견 아이디어 → 과학·기술 영역
  교육 혁신 아이디어 → 학문·교육 영역
```

### 3. 지식 자원 배분

각 플레이어에게 지식 자원 토큰 5개를 줍니다.

### 4. 행동 포인트 배분

매 라운드 시작 시 행동 포인트 4개를 받습니다.

## 게임 규칙

### 라운드 구조

```
1단계: 사회 변화 카드 공개
2단계: 지식 자원 생산
3단계: 행동 단계 (플레이어 순서대로)
4단계: 영향력 전파 체크
5단계: 저항 처리
```

### 1단계: 사회 변화 카드

매 라운드 사회 변화 카드 1장을 공개합니다.

```
사회 변화 카드 예시:

디지털 혁명:
  효과: 과학·기술, 경제·산업 영역 영향력 +3
        이번 라운드 기술 관련 아이디어 전파 2배

교육 위기:
  효과: 학문·교육 영역 영향력 -2
        교육 아이디어 저항 감소 (전파 쉬워짐)

경제 침체:
  효과: 경제·산업 영역 영향력 -3
        혁명적 아이디어 저항 감소

문화 르네상스:
  효과: 문화·예술 영역 영향력 +5
        모든 영역 아이디어 전파 속도 +1

정치적 변화:
  효과: 정치·정책 영역 새로운 기회
        정치 관련 아이디어 영향력 2배

세계화:
  효과: 모든 영역 영향력 +1
        융합 아이디어 시너지 +3
```

### 2단계: 지식 자원 생산

현재 보유한 아이디어 수만큼
지식 자원 토큰을 획득합니다.

```
지식 자원 생산:
  기본 아이디어 1개: +1 자원/라운드
  발전된 아이디어: +2 자원/라운드
  전파된 아이디어: +3 자원/라운드
```

### 3단계: 행동 단계

각 플레이어가 행동 포인트를 사용합니다.

**①아이디어 연구 (포인트 1):**
아이디어 카드 덱에서 카드 2장을 뽑아
1장을 선택하고 1장을 버립니다.

**②아이디어 발전 (포인트 2):**
보유한 아이디어 카드에 발전 카드를 추가합니다.

```
발전 카드 예시:

이론적 증거 (Theoretical Evidence):
  효과: 아이디어 기본 영향력 +5
        학문·교육 영역 저항 -2
  비용: 지식 자원 3개

실용적 적용 (Practical Application):
  효과: 아이디어 전파 속도 +2
        경제·산업 영역 영향력 +3
  비용: 지식 자원 2개

대중화 (Popularization):
  효과: 문화·예술 영역 영향력 +4
        저항 -1 (전체)
  비용: 지식 자원 2개

정치적 지지 (Political Support):
  효과: 정치·정책 영역 영향력 +5
        정부 저항 완전 제거
  비용: 지식 자원 4개

과학적 증명 (Scientific Proof):
  효과: 아이디어 신뢰도 +10
        모든 영역 저항 -3
  비용: 지식 자원 5개

미디어 전파 (Media Spread):
  효과: 이번 라운드 영향력 전파 3배
  비용: 지식 자원 3개
```

**③아이디어 전파 (포인트 1):**
자신의 아이디어를 선택한 영역에 전파합니다.

```
전파 규칙:
  현재 아이디어 영향력 vs 해당 영역 저항값 비교

  아이디어 영향력 > 저항:
    전파 성공
    해당 영역 영향력 토큰 배치
    영향력 +아이디어 기본값

  아이디어 영향력 ≤ 저항:
    전파 실패
    아이디어 영향력 -1 (마모)
```

**④저항 공략 (포인트 1):**
특정 영역의 저항을 약화시킵니다.

```
공략 방법:
  지식 자원 2개 소비 = 저항 -1

저항 현황:
  학문·교육: 기본 저항 3
  정치·정책: 기본 저항 5
  경제·산업: 기본 저항 4
  문화·예술: 기본 저항 2
  과학·기술: 기본 저항 4
```

**⑤동맹 제안 (포인트 1):**
다른 플레이어와 임시 동맹을 맺습니다.

```
동맹 효과:
  동맹 기간: 2라운드
  공동 전파: 두 플레이어 아이디어 영향력 합산
  공동 저항: 저항 공략 시 자원 절반
  이익 공유: 전파 성공 시 영향력 75%씩

동맹 해제:
  언제든 선언 가능
  단, 해제 후 2라운드 동안 재동맹 불가
```

**⑥경쟁 아이디어 공격 (포인트 2):**
다른 플레이어의 아이디어를 약화시킵니다.

```
공격 방법:
  반박 카드 사용 + 지식 자원 3개

반박 카드 예시:
  논리적 오류 지적: 상대 아이디어 신뢰도 -3
  대안 제시: 상대 영역 영향력 -2, 자신 +1
  역사적 반증: 상대 아이디어 발전 카드 1개 제거
  여론 조작: 상대 다음 전파 저항 +2

방어:
  상대방 공격 시 방어 카드 사용 가능
  방어 비용: 지식 자원 2개
```

### 4단계: 영향력 전파 체크

각 영역에서 영향력 토큰이 가장 많은 아이디어가
해당 영역을 "지배"합니다.

```
지배 효과:
  지배 아이디어 보유자: +3 영향력/라운드
  지배 경쟁 (동점): 다음 라운드 결전

전체 영향력 계산:
  5개 영역의 영향력 토큰 합산
  100 달성 시 즉시 승리
```

### 5단계: 저항 처리

미전파 아이디어는 저항을 받아 약화됩니다.

```
저항 효과:
  전파 안 된 라운드마다: 아이디어 영향력 -1
  저항값 초과 마모: 아이디어 카드 버림

새 아이디어 획득:
  버린 아이디어 대신 새 카드 1장 획득
```

## 특수 아이디어 시스템

### 아이디어 진화

특정 조건을 충족하면 아이디어가 진화합니다.

```
진화 조건:
  같은 아이디어 3개 영역 전파 성공
  발전 카드 3개 이상 부착
  동맹 아이디어와 융합

진화 효과:
  아이디어 이름 업그레이드
  기본 영향력 +10
  모든 영역 저항 -2
  특수 능력 획득
```

### 아이디어 융합

두 플레이어가 동맹 중일 때
아이디어를 융합할 수 있습니다.

```
융합 조건:
  동맹 중인 두 플레이어
  두 아이디어 모두 발전 카드 2개 이상

융합 효과:
  새로운 융합 아이디어 카드 생성
  두 아이디어의 영향력 합산
  양측 공동 소유
  이익은 60:40 분배 (먼저 제안한 쪽이 60%)
```

## 역사적 아이디어 카드

게임에 실제 역사적 아이디어를 포함할 수 있습니다.

```
민주주의 (Athenian Democracy):
  시대: 고대 그리스
  영향력: 18
  전파 대상: 정치·정책, 학문·교육
  발전 경로: 직접민주주의 → 대의민주주의

과학 혁명 (Scientific Revolution):
  시대: 16-17세기
  영향력: 20
  전파 대상: 과학·기술, 학문·교육
  발전 경로: 코페르니쿠스 → 뉴턴 → 아인슈타인

산업혁명 아이디어 (Industrial Revolution):
  시대: 18-19세기
  영향력: 17
  전파 대상: 경제·산업, 과학·기술
  발전 경로: 증기기관 → 전기 → 컴퓨터

진화론 (Theory of Evolution):
  시대: 19세기
  영향력: 15
  전파 대상: 과학·기술, 문화·예술
  저항: 높음 (종교적 저항)

인터넷 혁명 (Internet Revolution):
  시대: 20세기 말
  영향력: 22
  전파 대상: 모든 영역
  특성: 모든 영역 동시 전파 가능
```

## 전략 가이드

### 초보자 전략

**1. 집중 전파**
한 영역에 집중해서 지배력을 확보하세요.
여러 영역에 분산하면 어느 것도 지배하지 못합니다.

**2. 저항 먼저 약화**
저항이 낮은 영역(문화·예술)에서 먼저 성공하고
점차 저항이 높은 영역으로 확장하세요.

**3. 발전 카드 우선**
아이디어를 먼저 충분히 발전시키고 전파하세요.
약한 아이디어로 전파하면 계속 실패합니다.

### 중급 전략

**1. 동맹 타이밍**
혼자 저항을 뚫기 어려운 정치·정책 영역에서
동맹이 매우 효과적입니다.

**2. 공격 vs 방어**
선두 플레이어의 아이디어를 공격해서
영향력 확장을 방해하세요.

**3. 사회 변화 예측**
사회 변화 카드 패턴을 분석해서
유리한 변화가 오기 전에 준비하세요.

### 고급 전략

**1. 아이디어 진화 목표**
3개 영역 전파 성공으로 아이디어를 진화시키면
게임 후반 압도적으로 유리해집니다.

**2. 동맹 역이용**
동맹으로 공동 전파한 후
적절한 시점에 동맹을 끊고 단독 지배를 노리세요.

**3. 파괴적 아이디어 타이밍**
파괴적 아이디어는 상대가 지배하는 영역을
무너뜨리는 데 사용하세요.

## 변형 규칙

### 역사 아이디어 전쟁
- 역사적 아이디어 카드만 사용
- 실제 역사 흐름 재현
- 역사·사회 교육 연계

### 협력 아이디어 전쟁
- 모든 플레이어 협력
- 외부 "무지" 카드가 아이디어를 방해
- 지식 확산의 어려움 체험

### 토론 아이디어 전쟁
- 아이디어 전파 시 실제 토론 필요
- 다른 플레이어들이 저항 역할
- 논증 능력 훈련

## 교육적 활용

### 사회·역사 교육 연계
- **아이디어 전파**: 사상과 혁신의 전파 과정
- **사회 변화**: 아이디어가 사회를 바꾸는 방식
- **저항과 극복**: 혁신에 대한 저항의 역사

### 창의성 교육 연계
- **아이디어 발전**: 초기 아이디어를 발전시키는 과정
- **융합 사고**: 다른 분야 아이디어의 결합
- **영향력**: 아이디어의 사회적 가치 측정

### 수업 활용 방법
