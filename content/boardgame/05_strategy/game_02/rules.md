# 네트워크 전쟁 (Network War)

> 노드와 연결선으로 이루어진 네트워크를
> 전략적으로 구축하고 상대방의 네트워크를
> 끊어내는 그래프 이론 기반 전략 게임입니다.
> 연결의 힘으로 승부합니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 25-35분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 그래프 이론, 네트워크 사고, 전략적 연결, 수학 |

---

## 게임 목표

게임 종료 시 **자신의 네트워크가 가장 강력한
(연결된 노드 수 × 연결선 수)** 플레이어가 승리합니다.

또는 상대방의 네트워크를 완전히 분리시키면
즉시 승리합니다.

## 게임 준비

### 1. 허브 노드 배치

허브 노드 마커 10개를 보드에 무작위로 배치합니다.

```
허브 노드 특성:
  점수 가치: 일반 노드의 3배
  연결 보너스: 인접 연결선 강도 +1
  획득 조건: 해당 위치 점령 + 연결선 2개 이상
```

### 2. 시작 노드 배치

각 플레이어가 시작 노드 3개를 보드 모서리 구역에 배치합니다.

```
2인 게임:
  플레이어 1: 좌상단 구역 (1-3행, 1-3열)
  플레이어 2: 우하단 구역 (5-7행, 5-7열)

3인 게임:
  플레이어 1: 좌상단
  플레이어 2: 우상단
  플레이어 3: 하단 중앙

4인 게임:
  각 플레이어 모서리 구역
```

### 3. 강화 카드 배분

각 플레이어에게 강화 카드 4장을 나눠줍니다.

## 게임 규칙

### 턴 진행

플레이어는 번갈아 가며 행동합니다.
매 턴 다음 중 **2가지**를 선택해서 실행합니다.

**①노드 배치:**
자신의 기존 노드에 인접한 빈 위치에
새 노드를 배치합니다.

**②연결선 배치:**
자신의 인접한 두 노드 사이에
연결선을 배치합니다.

**③노드 강화:**
자신의 노드에 방어 토큰을 추가합니다.

```
방어 토큰 효과:
  방어 토큰 1개: 공격 1회 무효화
  방어 토큰 2개: 공격 2회 무효화 + 반격 가능
  방어 토큰 3개: 난공불락 (직접 공격 불가)
```

**④연결선 공격:**
상대방의 연결선 1개에 방해 토큰을 올립니다.

```
공격 효과:
  방해 토큰 1개: 연결선 약화 (점수 -1)
  방해 토큰 2개: 연결선 절단 (제거)
  단, 상대방이 방어 토큰으로 방어 가능
```

**⑤노드 점령:**
상대방의 고립된 노드(연결선 없는 노드)를
자신의 노드로 전환합니다.
비용: 자신의 인접 노드 필요.

**⑥강화 카드 사용:**
손패의 강화 카드 1장을 사용합니다.

---

## 강화 카드 목록

```
슈퍼 허브 (Super Hub):
  효과: 선택한 자신의 노드를 허브 노드로 업그레이드
        점수 3배 + 인접 연결 강화
  희귀도: ⭐⭐⭐

네트워크 폭발 (Network Burst):
  효과: 자신의 노드에서 인접한 모든 빈 칸에
        즉시 노드 배치 (최대 4개)
  희귀도: ⭐⭐⭐

연결 폭풍 (Connection Storm):
  효과: 자신의 인접 노드들 사이에
        모든 가능한 연결선 즉시 배치
  희귀도: ⭐⭐⭐

방어막 (Shield):
  효과: 선택한 자신의 노드에 방어 토큰 3개 즉시 추가
  희귀도: ⭐⭐

네트워크 절단 (Network Cut):
  효과: 상대방의 연결선 2개에 방해 토큰 배치
  희귀도: ⭐⭐

허브 탈취 (Hub Capture):
  효과: 인접 허브 노드를 즉시 점령
        (방어 토큰 무시)
  희귀도: ⭐⭐⭐⭐

재연결 (Reconnect):
  효과: 절단된 자신의 연결선 1개 복구
        + 방해 토큰 모두 제거
  희귀도: ⭐⭐

전략적 우회 (Strategic Bypass):
  효과: 인접하지 않아도 2칸 거리 연결선 배치
  희귀도: ⭐⭐⭐

정보 차단 (Information Block):
  효과: 상대방이 다음 턴 강화 카드 사용 불가
  희귀도: ⭐⭐

네트워크 분석 (Network Analysis):
  효과: 상대방의 강화 카드 1장 확인
        원하면 그 카드 버리기 가능
  희귀도: ⭐⭐
```

## 네트워크 강도 계산

### 기본 강도 공식

```
네트워크 강도 = 연결된 노드 수 × 연결선 수
              + 허브 노드 수 × 3
              + 방어 토큰 수
```

### 연결 보너스

```
선형 연결 (─): 기본
방사형 연결 (★): 중앙 노드에서 4방향 연결 = +5 보너스
루프 연결 (○): 닫힌 경로 완성 = +3 보너스
삼각형 연결 (△): 3노드 삼각형 = +2 보너스
```

### 분리 페널티

```
고립된 노드 (연결선 없음): 해당 노드 점수 0
분리된 서브네트워크: 각 서브네트워크 독립 계산
```

## 승리 조건

### 즉시 승리

다음 조건 달성 시 즉시 승리합니다.

```
완전 분리 승리:
  상대방의 모든 노드가 고립 상태
  (연결선 없음 또는 서브네트워크로 분리)

허브 독점 승리:
  보드의 허브 노드 7개 이상 점령
  (총 10개 중 70% 이상)

네트워크 지배 승리:
  자신의 노드가 보드 전체 노드의 50% 이상
```

### 라운드 종료 승리 (10라운드 후)

```
최종 점수:
  네트워크 강도 + 허브 노드 보너스 + 연결 보너스

가장 높은 점수 플레이어 승리
```

## 전략 가이드

### 초보자 전략

**1. 허브 노드 우선**
허브 노드는 일반 노드의 3배 점수입니다.
초반에 허브 노드 방향으로 확장하세요.

**2. 연결선 밀도 높이기**
노드만 많아도 연결선이 없으면 점수가 낮습니다.
노드와 연결선을 균형 있게 배치하세요.

**3. 방어 우선**
핵심 노드(허브·연결 허브)에 방어 토큰을 먼저 배치하세요.

### 중급 전략

**1. 루프 완성**
닫힌 루프를 완성하면 +3 보너스를 받습니다.
루프 경로를 미리 계획하세요.

**2. 상대 분리 작전**
상대방의 네트워크에서 취약한 연결선을 찾아
집중 공격으로 분리하세요.

**3. 카드 타이밍**
강화 카드는 중요한 순간에 사용하세요.
특히 "네트워크 절단"은 상대의 허브 연결을 끊는 데 활용합니다.

### 고급 전략

**1. 다중 위협**
두 방향에서 동시에 확장해서
상대방이 하나만 막도록 강요하세요.

**2. 허브 탈취 타이밍**
상대방이 방어 토큰을 다 소모했을 때
허브 탈취 카드를 사용하면 최대 효과입니다.

**3. 네트워크 토폴로지 최적화**
방사형 구조가 선형 구조보다 더 강한 네트워크를 만듭니다.
중앙 허브에서 여러 방향으로 확장하세요.

## 수학적 분석

### 그래프 이론 연결

```
연결 그래프:
  모든 노드가 연결된 그래프 = 최대 점수
  V: 노드 수, E: 연결선 수
  완전 연결 그래프: E = V(V-1)/2

최소 신장 트리:
  모든 노드를 연결하는 최소 연결선 수
  = V - 1개

네트워크 취약점:
  단절점(cut vertex): 제거 시 그래프 분리되는 노드
  교량(bridge): 제거 시 그래프 분리되는 연결선
  → 상대방의 단절점과 교량을 공격하세요
```

### 클러스터링 계수

```
클러스터링 계수 =
  실제 연결된 이웃 쌍 / 가능한 이웃 쌍
  → 높을수록 조밀한 네트워크
  → 방사형 구조 = 낮은 클러스터링
  → 루프 구조 = 높은 클러스터링
```

## 이벤트 카드

매 3라운드마다 이벤트 카드 1장을 공개합니다.

```
네트워크 폭풍:
  모든 플레이어의 연결선 1개 무작위 제거

통신 혁명:
  이번 라운드 모든 연결선 배치 비용 없음

사이버 공격:
  모든 방어 토큰 1개 감소

네트워크 융합:
  인접한 두 플레이어의 노드 연결 가능 (협상)

정보 공개:
  모든 플레이어의 강화 카드 1장 공개

네트워크 재편:
  각 플레이어가 자신의 노드 1개를 다른 위치로 이동
```

## 변형 규칙

### 수학 네트워크 전쟁
- 연결선에 가중치(1-5) 부여
- 최소 비용 신장 트리 완성이 목표
- 그래프 이론 심화 학습

### 협력 네트워크
- 모든 플레이어가 하나의 거대 네트워크 건설
- 외부 "바이러스" 카드가 연결선을 끊음
- 협력으로 네트워크 유지

### 속도 네트워크
- 모래시계 30초 제한 턴
- 빠른 판단과 직관 훈련

## 교육적 활용

### 수학 연계
- **그래프 이론**: 노드·연결선·경로 개념
- **위상수학**: 연결성과 분리 개념
- **최적화**: 최소 연결·최대 네트워크

### 사고력 연계
- **공간적 사고**: 2D 네트워크 구조 파악
- **전략적 사고**: 공격과 방어의 균형
- **패턴 인식**: 취약점과 기회 발견

### 수업 활용 방법
1. 그래프 이론 기초 설명 (10분)
2. 2인 기본 게임 (25분)
3. "네트워크의 취약점은?" 분석 (10분)
4. 수학 변형 버전 도전 (20분)
