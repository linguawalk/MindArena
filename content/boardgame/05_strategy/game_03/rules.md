# 진화 경쟁 (Evolution Race)

> 자신의 생물 종을 환경에 맞게 진화시키고
> 다른 종보다 더 잘 적응해서
> 생태계를 지배하는 전략 게임입니다.
> 자연선택의 원리가 게임의 핵심입니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-5명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 진화론, 생태학, 자연선택, 적응 전략 |

---

## 게임 목표

8라운드 후 **생존 점수(개체 수 × 적응도 × 환경 점수)**가
가장 높은 플레이어가 승리합니다.

또는 모든 다른 종을 멸종시키면 즉시 승리합니다.

---

## 준비물

- 생태계 보드 (4개 환경 구역)
- 종 카드 (플레이어별 기본 종 1장)
- 특성 카드 덱 (80장)
- 환경 변화 카드 덱 (30장)
- 먹이 토큰 (40개)
- 개체 수 마커 (플레이어별 20개)
- 적응도 트랙 (0-10)
- 멸종 마커
- 진화 포인트 토큰

### 생태계 보드 구조

```
┌──────────────┬──────────────┐
│   열대우림    │    초원      │
│  먹이: 풍부  │  먹이: 보통  │
│  위험: 높음  │  위험: 낮음  │
│  특성 보너스: │  특성 보너스: │
│  위장·독성   │  속도·시력   │
├──────────────┼──────────────┤
│   해양       │   사막       │
│  먹이: 보통  │  먹이: 희귀  │
│  위험: 중간  │  위험: 높음  │
│  특성 보너스: │  특성 보너스: │
│  수영·발광   │  내열·절수   │
└──────────────┴──────────────┘
```

## 기본 종 카드

```
각 플레이어는 다음 기본 종 중 1개 선택:

초식 동물 (Herbivore):
  기본 개체 수: 5
  기본 적응도: 3
  특성: 없음
  먹이: 식물성 먹이 전용
  강점: 먹이 소비 적음
  약점: 포식자에 취약

잡식 동물 (Omnivore):
  기본 개체 수: 4
  기본 적응도: 4
  특성: 없음
  먹이: 모든 먹이 가능
  강점: 다양한 환경 적응
  약점: 특화 없음

육식 동물 (Carnivore):
  기본 개체 수: 3
  기본 적응도: 5
  특성: 없음
  먹이: 다른 종 사냥
  강점: 다른 종 개체 수 감소시킴
  약점: 먹이(다른 종) 없으면 굶음

곤충 (Insect):
  기본 개체 수: 8
  기본 적응도: 2
  특성: 군집 생활
  먹이: 식물성·부패물
  강점: 빠른 번식
  약점: 낮은 개별 적응도

식물 (Plant):
  기본 개체 수: 6
  기본 적응도: 3
  특성: 광합성
  먹이: 필요 없음 (광합성)
  강점: 먹이 불필요, 환경 개선
  약점: 이동 불가, 초식동물에 취약
```

## 특성 카드 시스템

### 특성 카드 종류

```
신체 특성:
  날개 (Wings):
    효과: 이동 시 환경 이동 비용 없음
          조류 포식자 회피 가능
    적응도 보너스: +2
    비용: 진화 포인트 3

  외골격 (Exoskeleton):
    효과: 포식 당할 때 개체 수 손실 -1
          건조 환경 저항
    적응도 보너스: +1
    비용: 진화 포인트 2

  독성 (Toxin):
    효과: 포식자가 사냥 후 적응도 -1
          경쟁종 인접 시 개체 수 -1 (상대방)
    적응도 보너스: +2
    비용: 진화 포인트 3

  위장 (Camouflage):
    효과: 포식 공격 1회 무효화/라운드
          환경 점수 +1 (서식 환경)
    적응도 보너스: +1
    비용: 진화 포인트 2

  발광 (Bioluminescence):
    효과: 야간 환경에서 추가 먹이 획득
          다른 종 유인 가능
    적응도 보너스: +1
    비용: 진화 포인트 2

행동 특성:
  군집 생활 (Social):
    효과: 개체 수 8 이상 시 번식 2배
          포식 피해 -2
    적응도 보너스: +1
    비용: 진화 포인트 2

  영역 표시 (Territorial):
    효과: 자신의 환경 구역에 타종 진입 방해
          먹이 독점 가능
    적응도 보너스: +2
    비용: 진화 포인트 3

  이동 생활 (Migratory):
    효과: 매 라운드 무료로 환경 이동 가능
          모든 환경 적응도 +1
    적응도 보너스: +1
    비용: 진화 포인트 2

생리 특성:
  내열성 (Heat Resistance):
    효과: 사막 환경 적응도 +3
          열대 위험 무시
    적응도 보너스: +1
    비용: 진화 포인트 1

  절수 능력 (Water Conservation):
    효과: 사막에서 먹이 소비 절반
          건조 환경 생존 가능
    적응도 보너스: +1
    비용: 진화 포인트 1

  야행성 (Nocturnal):
    효과: 주간 포식 피해 없음
          야간 먹이 2배
    적응도 보너스: +1
    비용: 진화 포인트 2

  빠른 번식 (Rapid Reproduction):
    효과: 번식 단계에서 개체 수 +3 추가
          최대 개체 수 +5
    적응도 보너스: +0
    비용: 진화 포인트 2
```

## 게임 규칙

### 라운드 구조

```
1단계: 환경 변화 카드 공개
2단계: 먹이 배분
3단계: 진화 단계 (특성 획득)
4단계: 이동 단계
5단계: 포식 단계
6단계: 번식 단계
7단계: 멸종 체크
```

### 1단계: 환경 변화

환경 변화 카드 1장을 공개합니다.

```
환경 변화 카드 예시:

빙하기 시작:
  효과: 열대우림 먹이 -3, 초원 먹이 -2
        내열성 특성 무효화
        이 라운드 내열성 보너스 없음

가뭄:
  효과: 모든 환경 먹이 -2
        절수 능력 특성 보너스 +2

화산 폭발:
  효과: 사막 구역 진입 불가 (1라운드)
        인접 구역 먹이 -1

유성 충돌:
  효과: 무작위 환경 1개 먹이 0
        해당 환경의 모든 종 이동 강제

온난화:
  효과: 사막 먹이 +3
        해양 먹이 -1
        내열성 종 적응도 +2

풍요의 계절:
  효과: 모든 환경 먹이 +2
        번식 단계 개체 수 +1 추가
```

### 2단계: 먹이 배분

각 환경 구역에 먹이 토큰을 배치합니다.

```
기본 먹이 배분:
  열대우림: 6개
  초원: 4개
  해양: 4개
  사막: 2개

환경 변화 카드 효과 적용 후 최종 배분
```

### 3단계: 진화 단계

각 플레이어가 진화 포인트를 사용해서
특성 카드를 획득하거나 기존 특성을 강화합니다.

```
진화 포인트 획득:
  매 라운드 기본 1포인트
  적응도 5 이상: +1포인트
  적응도 8 이상: +2포인트
  허브 환경 점령: +1포인트

특성 강화:
  특성 카드를 2번 보유하면 효과 2배
  (같은 특성 카드 2장 = 강화)
```

### 4단계: 이동 단계

각 플레이어가 자신의 종을 환경 구역으로 이동시킵니다.

```
이동 규칙:
  인접 환경으로만 이동 가능
  이동 비용: 개체 수의 20% 손실
  날개 특성: 이동 비용 없음
  이동 생활 특성: 무료 이동

환경 적응도:
  자신의 종이 해당 환경에 맞는 특성 보유 시
  적응도 보너스 적용
```

### 5단계: 포식 단계

육식 동물이 다른 종을 사냥합니다.

```
포식 규칙:
  육식 종 vs 다른 종 (같은 환경)
  포식자 적응도 vs 피식자 적응도 비교

포식 결과:
  포식자 적응도 > 피식자 적응도:
    피식자 개체 수 -(포식자 적응도 - 피식자 적응도)
    포식자 개체 수 +1 (먹이 충족)

  포식자 적응도 ≤ 피식자 적응도:
    포식 실패
    포식자 개체 수 -1 (에너지 소모)

특성 효과 적용:
  독성: 포식 성공 후 포식자 적응도 -1
  위장: 포식 1회 무효화
  외골격: 피해 -1
```

### 6단계: 번식 단계

각 종이 번식해서 개체 수를 늘립니다.

```
번식 규칙:
  먹이 토큰 소비 → 개체 수 증가

기본 번식:
  먹이 토큰 1개 = 개체 수 +1
  (최대 현재 개체 수의 50%까지)

먹이 부족:
  먹이 < 개체 수 × 0.5:
    부족분만큼 개체 수 감소

번식 보너스:
  군집 생활 특성: 개체 수 8+ → 번식 2배
  빠른 번식 특성: +3 추가
```

### 7단계: 멸종 체크

개체 수가 0이 된 종은 멸종합니다.

```
멸종 조건:
  개체 수 0
  적응도 0
  먹이 완전 부족 3라운드 연속

멸종 시:
  해당 플레이어 게임 탈락
  단, 관찰자로 남아 이벤트 발생 역할 가능
```

## 점수 계산

### 게임 종료 (8라운드 후)

```
생존 점수 = 개체 수 × 적응도 × 환경 점수

환경 점수:
  해당 환경에 가장 많은 개체 = 3점
  해당 환경에 두 번째 = 2점
  해당 환경에 있음 = 1점

보너스:
  특성 카드 3개 이상: +10점
  모든 환경 진출: +15점
  최다 개체 수: +10점
  최고 적응도: +10점
```

## 전략 가이드

### 초보자 전략

**1. 환경 특화**
자신의 종에 맞는 특성이 보너스를 주는
환경 1개에 집중하세요.

**2. 먹이 확보 우선**
번식보다 먹이 확보가 먼저입니다.
먹이가 없으면 개체 수가 줄어듭니다.

**3. 육식 동물 주의**
육식 동물이 있으면 방어 특성을 먼저 확보하세요.
독성이나 위장이 효과적입니다.

### 중급 전략

**1. 환경 변화 예측**
환경 변화 카드의 패턴을 분석하세요.
빙하기 카드가 많으면 내열성보다 냉한성 특성이 유리합니다.

**2. 적응도 최적화**
적응도 5 이상부터 진화 포인트 보너스를 받습니다.
적응도 강화가 장기적으로 유리합니다.

**3. 특성 시너지**
날개 + 이동 생활 = 이동 완전 자유.
군집 생활 + 빠른 번식 = 폭발적 개체 수 증가.

### 고급 전략

**1. 먹이사슬 조절**
육식 동물이라면 피식자 종을 너무 빨리 멸종시키지 마세요.
먹이가 없으면 자신도 굶습니다.

**2. 환경 독점**
한 환경을 완전히 독점하면
다른 종이 진입하기 어렵습니다.
영역 표시 특성과 결합하세요.

**3. 진화 타이밍**
환경 변화에 맞춰 특성을 미리 획득하면
변화 직후 큰 이점을 얻습니다.

## 변형 규칙

### 협력 진화 경쟁
- 모든 종이 협력해서 대멸종 이벤트 생존
- 먹이사슬 균형 유지가 목표
- 생태계 교육 최적화

### 급진 진화
- 매 라운드 진화 포인트 2배
- 더 빠른 특성 획득
- 소요 시간: 20분

### 역할극 진화
- 각 플레이어가 실제 동물 역할
- 해당 동물의 실제 특성 적용
- 생물학 교육 연계

## 교육적 활용

### 생물 교육 연계
- **자연선택**: 적응도와 생존의 관계
- **적응**: 환경에 맞는 특성 발달
- **먹이사슬**: 포식자·피식자 관계
- **생물 다양성**: 다양한 종의 공존

### 전략 교육 연계
- **적응 전략**: 환경 변화에 대한 대응
- **자원 관리**: 먹이와 개체 수 균형
- **위험 관리**: 포식과 멸종 회피

### 수업 활용 방법
1. 자연선택 개념 설명 (10분)
2. 3인 게임 1판 (35분)
3. "어떤 특성이 가장 유리했나?" 토론 (10분)
4. 실제 동물 적응 사례 연결 (10분)
