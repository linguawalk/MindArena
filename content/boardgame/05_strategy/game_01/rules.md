# 자원 제국 (Resource Empire)

> 제한된 자원을 효율적으로 관리하고
> 생산·교역·투자를 통해 제국을 건설하는
> 경제 전략 게임입니다.
> 자원의 흐름을 지배하는 자가 승리합니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 자원 관리, 경제 전략, 기회비용, 투자 |

---

## 게임 목표

8라운드 후 **총 자산(보유 자원 + 건설물 가치 + 교역 수익)**이
가장 높은 플레이어가 승리합니다.

---

## 준비물

- 게임 보드 (5×5 자원 격자)
- 자원 타일 (목재·철광·식량·금·에너지 각 15개)
- 건설 카드 40장
- 교역 카드 30장
- 투자 카드 20장
- 시장 가격 보드
- 플레이어 보드 (개인 자원 관리)
- 행동 토큰 (플레이어당 4개)
- 점수 기록판

### 자원 격자 보드

```
   1열   2열   3열   4열   5열
1행[목재][철광][식량][금  ][에너]
2행[식량][목재][금  ][에너][철광]
3행[철광][에너][목재][식량][금  ]
4행[금  ][식량][철광][목재][에너]
5행[에너][금  ][에너][철광][식량]

각 칸: 해당 자원 생산지
플레이어가 소유하면 매 라운드 자원 생산
```

## 자원 시스템

### 자원 종류와 기본 가치

```
목재 (Wood):
  기본 가치: 2 코인
  용도: 건설·제지·연료
  생산: 보통

철광 (Iron):
  기본 가치: 4 코인
  용도: 무기·기계·건설
  생산: 드묾

식량 (Food):
  기본 가치: 3 코인
  용도: 인구 유지·교역
  생산: 풍부

금 (Gold):
  기본 가치: 6 코인
  용도: 교역·투자·건설
  생산: 희귀

에너지 (Energy):
  기본 가치: 5 코인
  용도: 생산력 강화·첨단 건설
  생산: 보통
```

### 시장 가격 변동

```
시장 가격 보드:
  매 라운드 주사위로 1-2개 자원 가격 변동
  +1 코인 또는 -1 코인

가격 범위:
  최저: 기본 가치 -2
  최고: 기본 가치 +4

가격 변동 요인:
  해당 자원 과잉 공급 → 가격 하락
  해당 자원 부족 → 가격 상승
```

## 게임 준비

### 1. 자원 격자 소유권 설정

각 플레이어는 시작 자원 격자 2칸을 선택합니다.
선택 순서: 시계 방향으로 번갈아 선택.

### 2. 초기 자원 배분

```
각 플레이어 시작 자원:
  목재: 3개
  철광: 1개
  식량: 3개
  금: 2개
  에너지: 2개
  시작 자금: 20 코인
```

### 3. 시장 가격 초기화

모든 자원을 기본 가치로 설정합니다.

## 게임 규칙

### 라운드 구조

```
1단계: 자원 생산
2단계: 시장 가격 변동
3단계: 행동 단계 (플레이어 순서대로)
4단계: 교역 정산
```

### 1단계: 자원 생산

자신이 소유한 격자 칸에서 자원을 생산합니다.

```
기본 생산:
  소유 칸 1개 = 해당 자원 1개/라운드

건설 보너스:
  광산 건설: 철광·금 생산 +1
  농장 건설: 식량 생산 +2
  발전소 건설: 에너지 생산 +2
  제재소 건설: 목재 생산 +2
```

### 2단계: 시장 가격 변동

시장 주사위를 굴려서 자원 가격을 변동시킵니다.

```
주사위 1: 목재 +1
주사위 2: 철광 -1
주사위 3: 식량 +1
주사위 4: 금 +1
주사위 5: 에너지 -1
주사위 6: 모든 자원 +1 (호황)
주사위 7: 무작위 2개 -1 (불황)
```

### 3단계: 행동 단계

각 플레이어가 행동 토큰 4개를 사용합니다.

**①자원 격자 획득 (행동력 2):**
비어있는 자원 격자 1칸을 획득합니다.
획득 비용: 해당 자원 현재 가치 × 3 코인.

**②건설 (행동력 1-2):**
건설 카드를 사용해서 자신의 격자에 건물을 짓습니다.

```
건설 카드 예시:

광산 (Mine):
  비용: 철광 2 + 목재 1 + 금 10
  효과: 해당 격자 생산량 +1
  행동력: 2

농장 (Farm):
  비용: 목재 2 + 식량 1 + 금 8
  효과: 식량 생산 +2
  행동력: 1

발전소 (Power Plant):
  비용: 철광 3 + 에너지 1 + 금 15
  효과: 에너지 생산 +2
        다른 건물 생산 +1 (인접 칸)
  행동력: 2

무역항 (Trade Port):
  비용: 목재 3 + 금 12
  효과: 교역 수수료 면제
        교역 한도 +2
  행동력: 2

연구소 (Research Lab):
  비용: 에너지 2 + 금 15 + 철광 1
  효과: 매 라운드 무작위 투자 카드 1장
  행동력: 2

창고 (Warehouse):
  비용: 목재 3 + 금 8
  효과: 자원 보관 한도 +5
  행동력: 1
```

**③교역 (행동력 1):**
다른 플레이어 또는 시장과 자원을 거래합니다.

```
시장 교역:
  시장 가격으로 자원 구매·판매
  한 번에 최대 3개

플레이어 교역:
  합의된 가격으로 자유롭게 거래
  한 번에 최대 5개
  무역항 보유 시 한도 없음
```

**④투자 (행동력 1):**
투자 카드를 구매해서 미래 수익을 확보합니다.

```
투자 카드 예시:

자원 선물 거래:
  비용: 금 10
  효과: 3라운드 후 선택 자원 5개 획득
  위험: 해당 자원 가격 하락 시 손실

무역 독점:
  비용: 금 20
  효과: 다음 2라운드 특정 자원 교역 시
        5% 수수료 징수
  위험: 다른 플레이어의 견제

기술 혁신:
  비용: 에너지 3 + 금 15
  효과: 모든 건물 생산량 +1 (2라운드)
  위험: 없음 (안전 투자)

시장 예측:
  비용: 금 8
  효과: 다음 라운드 가격 변동 사전 확인
  위험: 예측 활용 못하면 손실
```

**⑤자원 격자 공격 (행동력 2):**
다른 플레이어의 격자를 빼앗으려 시도합니다.

```
공격 조건:
  철광 3개 + 금 15 소비

방어 조건:
  방어자가 철광 2개 + 금 10으로 방어 가능

결과:
  공격 성공: 해당 격자 소유권 이전
  방어 성공: 공격자 자원 반환 없음 (손실)
  군사 건물 보유 시: 공격·방어 강화
```

## 점수 계산

### 게임 종료 (8라운드 후)

```
자원 가치:
  보유 자원 × 현재 시장 가격

건설물 가치:
  광산: 20점
  농장: 15점
  발전소: 25점
  무역항: 18점
  연구소: 22점
  창고: 10점

교역 수익:
  게임 중 교역으로 얻은 누적 수익

투자 수익:
  투자 카드의 실현 수익

격자 소유:
  소유 격자 1칸당 5점

최종 점수 = 자원 가치 + 건설물 가치
           + 교역 수익 + 투자 수익
           + 격자 소유 점수
```

## 전략 가이드

### 초보자 전략

**1. 다양한 자원 확보**
초반에 다양한 자원 격자를 확보하세요.
단일 자원에 집중하면 가격 변동에 취약합니다.

**2. 건설 우선**
자원을 팔기보다 건설에 투자하면
장기적으로 더 많은 자원을 생산합니다.

**3. 시장 가격 관찰**
가격이 높을 때 팔고 낮을 때 사는
기본적인 시장 원리를 활용하세요.

### 중급 전략

**1. 무역항 우선 건설**
무역항은 교역 한도를 늘려서
시장 가격 차익을 극대화합니다.

**2. 투자 타이밍**
가격이 낮을 때 선물 거래를 구매하면
가격 상승 시 큰 수익을 얻습니다.

**3. 공격과 방어 균형**
철광을 충분히 확보해서
공격과 방어 모두에 대비하세요.

### 고급 전략

**1. 독점 전략**
특정 자원 격자를 독점하고
무역 독점 카드로 수수료를 징수하세요.

**2. 연구소 투자**
연구소는 무작위 투자 카드를 제공해서
예상치 못한 이익을 만듭니다.

**3. 후반 역전**
후반에 고가 자원을 대량 확보하면
점수 역전이 가능합니다.

## 변형 규칙

### 협력 자원 제국
- 팀이 공동 자원 풀 관리
- 외부 위협 카드가 자원을 감소시킴
- 협력으로 위협 방어

### 단기 자원 제국
- 5라운드로 단축
- 시작 자원 +2배
- 소요 시간: 20분

### 역사 자원 제국
- 특정 역사 시대 테마
- 해당 시대 자원·건설물 사용
- 역사 학습 연계

## 교육적 활용

### 경제 교육 연계
- **수요·공급**: 시장 가격 변동 원리
- **기회비용**: 자원 사용의 선택
- **투자**: 단기 vs 장기 수익
- **무역**: 비교우위와 교역 이익

### 수학 연계
- **계산**: 자원 가치·점수 계산
- **확률**: 주사위 가격 변동 예측
- **최적화**: 자원 배분 전략

### 수업 활용 방법
1. 경제 기본 개념 연결 (10분)
2. 3인 게임 1판 (35분)
3. "최적의 자원 전략은?" 토론 (10분)
4. 실제 경제 현상과 비교 (10분)
