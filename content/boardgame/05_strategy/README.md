# 전략 게임 (Strategy Games)

> 장기적 계획과 전술적 판단으로
> 상대방보다 유리한 위치를 점하는 게임입니다.
> 자원 관리, 위험 평가, 기회 포착이 핵심입니다.

---

## 카테고리 특징

- **장기 계획**: 여러 수 앞을 내다보는 전략
- **자원 관리**: 제한된 자원의 최적 활용
- **리스크 관리**: 위험과 기회의 균형
- **교육적 가치**: 전략적 사고, 의사결정, 기회비용 이해

## 교육 목표

- 장기적 전략 수립 능력
- 자원 배분과 기회비용 이해
- 리스크와 보상의 균형 판단
- 상대방 전략 분석 및 대응 능력
