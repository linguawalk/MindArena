# 시장 지배자 (Market Master)

> 다양한 산업 분야에 투자하고
> 시장 점유율을 높여서
> 경제적 지배력을 확보하는
> 경제 전략 게임입니다.
> 시장의 흐름을 읽고 선점하세요.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-5명 |
| 소요 시간 | 35-45분 |
| 권장 연령 | 13세 이상 |
| 난이도 | ⭐⭐⭐⭐ (상) |
| 교육 분야 | 경제학, 시장 원리, 투자 전략, 경쟁 분석 |

## 게임 목표

10라운드 후 **총 자산(현금 + 주식 가치 + 건물 가치)**이
가장 높은 플레이어가 승리합니다.

## 준비물

- 시장 보드 (6개 산업 구역)
- 주식 카드 (산업별 10장 = 60장)
- 시장 가격 트랙 (산업별)
- 투자 카드 40장
- 이벤트 카드 30장
- 시장 점유율 마커
- 화폐 (1·5·10·50·100 단위)
- 산업 건물 미니어처 (플레이어별)
- 주주 카드 (특수 역할)
- 경쟁 분석 보드

금융 산업:
  초기 주가: 60

  초기 주가: 30

  성장률: 낮음 (+4%/라운드 평균)
  변동성: 낮음 (±8%)

부동산 산업:
  초기 주가: 55

## 게임 준비

### 1. 초기 자금 배분

```
3인: 각 300 코인
4인: 각 250 코인
5인: 각 200 코인
```

### 2. 주주 카드 배분

각 플레이어에게 주주 카드 1장을 무작위로 줍니다.

```
주주 카드 종류:

기관 투자자 (Institutional Investor):
  특권: IT·금융 주식 매수 수수료 없음
        분기 배당금 20% 추가

개인 투자자 (Individual Investor):
  특권: 소액 투자 시 세금 면제
        시장 정보 1개 사전 획득/라운드

헤지 펀드 매니저 (Hedge Fund Manager):
  특권: 공매도 가능 (주가 하락에 베팅)
        레버리지 투자 가능 (2배 투자)

산업 전문가 (Industry Expert):
  특권: 선택한 산업의 이벤트 카드 미리 확인
        해당 산업 주식 내부 정보 활용

중앙은행 (Central Bank):
  특권: 이자율 조정 가능 (1회/게임)
        금융 위기 이벤트 한 번 무효화
```

### 3. 시장 가격 초기화

모든 산업을 초기 주가로 설정합니다.

## 게임 규칙

### 라운드 구조

```
1단계: 이벤트 카드 공개
2단계: 시장 가격 변동
3단계: 행동 단계 (플레이어 순서대로)
4단계: 배당금 지급
5단계: 시장 분석 단계
```

### 1단계: 이벤트 카드 공개

이벤트 카드 1장을 공개합니다.

```
이벤트 카드 예시:

기술 혁신:
  효과: IT 산업 주가 +30%
  지속: 1라운드
  연계: 에너지 산업 +5% (전력 수요)

금융 위기:
  효과: 금융 주가 -40%
        모든 산업 -10%
  지속: 2라운드
  연계: 부동산 -20% (대출 감소)

전염병 발생:
  효과: 의료 산업 +50%
        식품 산업 +10%
  지속: 3라운드
  연계: 항공·관광 산업 폭락 (해당 없으면 무시)

에너지 위기:
  효과: 에너지 산업 +40%
        IT·제조업 -15% (비용 증가)
  지속: 2라운드
  연계: 대체 에너지 기업 +20%

부동산 버블 붕괴:
  효과: 부동산 -50%
        금융 -20% (부실 대출)
  지속: 3라운드
  연계: 건설·자재 산업 -30%

규제 강화:
  효과: 선택된 산업 -20%
        규제 준수 비용 발생
  지속: 2라운드

경기 호황:
  효과: 모든 산업 +10%
        배당률 +2%
  지속: 2라운드

정치적 불안:
  효과: 변동성 높은 산업 -20%
        안전 자산 (식품·의료) +5%
  지속: 1라운드
```

### 2단계: 시장 가격 변동

이벤트 카드 + 수요·공급 원리로 가격이 변동됩니다.

```
가격 변동 공식:
  기본 변동: ±5% (무작위)
  이벤트 효과: 이벤트 카드에 명시
  수요 효과: 해당 산업 주식 다수 보유 시 +3%
  공급 효과: 해당 산업 주식 매도 多 시 -3%

가격 범위:
  최저: 초기 주가 × 0.2
  최고: 초기 주가 × 5
```

### 3단계: 행동 단계

각 플레이어가 행동을 선택합니다.

**①주식 매수 (행동 1회):**
원하는 산업의 주식을 구매합니다.

```
매수 규칙:
  현재 시장 가격으로 구매
  한 번에 1-5주 구매 가능
  수수료: 거래 금액의 2% (기관 투자자: 없음)

레버리지 투자 (헤지 펀드 전용):
  보유 현금의 2배까지 투자 가능
  이익 2배, 손실 2배
```

**②주식 매도 (행동 1회):**
보유 중인 주식을 판매합니다.

```
매도 규칙:
  현재 시장 가격으로 판매
  한 번에 1-5주 판매 가능
  수수료: 거래 금액의 2%

공매도 (헤지 펀드 전용):
  보유하지 않은 주식 판매 가능
  나중에 더 낮은 가격으로 구매해서 차익
  위험: 가격 상승 시 무제한 손실
```

**③산업 건물 건설 (행동 1회):**
특정 산업 구역에 자신의 건물을 건설합니다.

```
건설 비용: 해당 산업 주가 × 5
건설 효과:
  해당 산업 점유율 +2%
  매 라운드 추가 배당 +해당 산업 주가 × 1%
  다른 플레이어 해당 산업 투자 수익 -3%
```

**④시장 조사 (행동 1회):**
다음 이벤트 카드를 미리 확인합니다.

```
조사 비용: 20 코인
효과: 다음 라운드 이벤트 카드 미리 확인
      해당 정보로 투자 전략 수립 가능
      단, 다른 플레이어에게 공개 금지
```

**⑤투자 카드 사용 (행동 1회):**
보유한 투자 카드를 사용합니다.

```
투자 카드 예시:

내부 정보:
  효과: 다음 이벤트에서 선택 산업 +20% 추가
  비용: 없음 (단, 법적 위험: 다른 플레이어 신고 가능)

시장 조작:
  효과: 선택 산업 주가 ±15% 조정
  비용: 50 코인
  위험: 규제 강화 이벤트 시 역효과

적대적 M&A:
  효과: 다른 플레이어의 산업 건물 1개 흡수
  비용: 해당 건물 가치의 150%
  위험: 상대방이 방어 카드로 방어 가능

분산 투자 펀드:
  효과: 모든 산업에 균등 투자
        위험 분산 효과
  비용: 100 코인

공격적 인수:
  효과: 선택 산업 점유율 +5%
  비용: 80 코인
```

### 4단계: 배당금 지급

각 산업의 보유 주식에 따라 배당금을 지급합니다.

```
배당금 = 보유 주식 수 × 현재 주가 × 배당률

예시:
  IT 산업 주가 80, 배당률 5%, 보유 3주
  배당금 = 3 × 80 × 5% = 12 코인

산업 건물 보유 시:
  추가 배당 = 현재 주가 × 1%/건물
```

### 5단계: 시장 분석

플레이어들이 시장 점유율을 확인합니다.

```
시장 점유율 계산:
  각 산업별 보유 주식 수 / 총 유통 주식 수 × 100

시장 지배 보너스:
  한 산업 점유율 30%+: +5%p 배당 추가
  한 산업 점유율 50%+: 가격 결정권 획득
                       (이번 라운드 해당 산업 가격 ±10% 조정)
  한 산업 점유율 70%+: 독점 선언 가능
                       (다른 플레이어 해당 산업 거래 수수료 10%)
```

## 점수 계산

### 게임 종료 (10라운드 후)

```
최종 자산 = 보유 현금
           + (보유 주식 수 × 최종 주가)
           + 건물 가치
           + 누적 배당금

건물 가치 = 해당 산업 최종 주가 × 3

추가 보너스:
  최다 산업 점유율: +50 코인
  전체 자산 1위: +20 코인
  가장 많은 배당 수익: +30 코인
  가장 다양한 포트폴리오 (5개 이상 산업): +25 코인
```

## 전략 가이드

### 초보자 전략

**1. 분산 투자**
하나의 산업에 집중하면 이벤트 충격이 큽니다.
3개 이상 산업에 분산 투자하세요.

**2. 배당 수익 활용**
금융·에너지 산업은 배당률이 높습니다.
초반에 안정적인 배당 수익을 확보하세요.

**3. 이벤트 주시**
이벤트 카드가 산업 주가에 큰 영향을 줍니다.
이벤트 발생 전에 미리 투자하는 것이 중요합니다.

### 중급 전략

**1. 시장 조사 활용**
이벤트 카드를 미리 아는 것이 가장 강력한 정보입니다.
투자 타이밍을 최적화하세요.

**2. 산업 건물 전략**
산업 건물은 장기적으로 꾸준한 수익을 제공합니다.
점유율이 높은 산업에 건물을 지으세요.

**3. 공매도 활용 (헤지 펀드)**
이벤트로 하락이 예상되는 산업을 공매도하면
큰 수익을 얻을 수 있습니다.

### 고급 전략

**1. 독점 전략**
한 산업을 집중 매수해서 70% 점유율을 달성하면
다른 플레이어의 거래에 수수료를 부과할 수 있습니다.

**2. 정보 비대칭 활용**
주주 카드 능력을 최대한 활용해서
다른 플레이어보다 더 많은 정보를 확보하세요.

**3. 타이밍 매매**
버블이 형성되는 산업을 정점에 팔고
폭락한 산업을 저점에 사는 전략입니다.

## 경제 개념 연결

### 게임 내 경제 원리

```
수요와 공급:
  많은 플레이어가 매수 = 가격 상승
  많은 플레이어가 매도 = 가격 하락

기회비용:
  A 산업 투자 = B 산업 투자 포기
  최적 포트폴리오 선택의 중요성

위험과 수익:
  IT (고위험·고수익) vs 식품 (저위험·저수익)
  리스크 프리미엄 개념

독점과 경쟁:
  시장 점유율 70%+ = 독점 이익
  경쟁 감소 = 가격 결정력

거품과 붕괴:
  과도한 기대 = 버블
  현실 조정 = 폭락
  사이클 이해
```

## 교육적 활용

### 경제 교육 연계
- **주식 시장**: 주가 변동과 거래 원리
- **투자 전략**: 분산 투자와 포트폴리오
- **시장 구조**: 독점·경쟁·과점
- **경기 순환**: 호황·불황·버블

### 수학 연계
- **퍼센트 계산**: 주가 변동·배당률
- **기댓값**: 투자 수익 기대값
- **확률**: 이벤트 발생 확률

### 수업 활용 방법
1. 주식 시장 기본 개념 설명 (15분)
