# 지식 경매 (Knowledge Auction)

> 지식으로 돈을 벌고
> 번 돈으로 더 많은 지식을 사는
> 경제와 지식이 결합된 게임입니다.
> 무엇을 알고 무엇에 투자할지
> 전략적 판단이 승부를 결정합니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-6명 |
| 소요 시간 | 30-40분 |
| 권장 연령 | 12세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 지식 활용, 경제적 사고, 전략적 투자, 리스크 관리 |

---

## 게임 목표

게임 종료 시 **지식 자산(획득한 지식 카드)과
보유 자금의 합산 가치**가 가장 높은 플레이어가 승리합니다.

---

## 준비물

- 지식 카드 120장 (분야별 20장)
- 지식 화폐 (1·5·10·50 단위)
  - 각 플레이어 시작 자금: 100 코인
- 경매 보드
- 투자 수익 카드 30장
- 지식 채권 카드 20장
- 시장 변동 주사위
- 점수 계산표

### 지식 화폐 구성

```
1 코인: 40장
5 코인: 30장
10 코인: 20장
50 코인: 10장
```

### 지식 카드 가치

```
분야별 기본 가치:
  과학: 15-40 코인
  역사: 10-30 코인
  수학: 20-45 코인
  사회: 10-25 코인
  예술: 15-35 코인
  자연: 10-30 코인

희귀 카드 (각 분야 5장):
  기본 가치 × 2배
  게임 종료 시 추가 보너스
```

## 게임 준비

### 1. 초기 자금 배분

각 플레이어에게 시작 자금 100 코인을 줍니다.

```
배분:
  50 코인 × 1장
  10 코인 × 3장
  5 코인 × 2장
  1 코인 × 10장
```

### 2. 경매 시장 설정

지식 카드를 섞어서 경매 보드에 5장을 공개합니다.
나머지는 더미로 쌓습니다.

### 3. 시작 지식 배분

각 플레이어에게 지식 카드 2장을 무작위로 줍니다.
이 카드들은 경매 없이 무료로 획득합니다.

---

## 게임 규칙

### 라운드 구조

각 라운드는 4단계로 진행됩니다.

```
1단계: 지식 테스트 (수입 단계)
2단계: 시장 변동 (환경 변화)
3단계: 경매 (투자 단계)
4단계: 세트 보너스 확인
```

### 1단계: 지식 테스트

각 플레이어가 자신이 보유한 지식 카드 중
1장을 선택해서 문제를 풀어 수입을 얻습니다.

**진행:**
①카드 선택: 테스트할 카드 1장 선택.
②문제 공개: 다른 플레이어가 문제를 읽어줍니다.
③답변: 30초 이내 답변.
④수입:
```
정답: 카드 기본 가치만큼 코인 획득
오답: 카드 기본 가치의 25% 손실 (최소 0)
```

**추가 도전:**
다른 플레이어도 같은 문제에 도전할 수 있습니다.
```
먼저 도전하고 정답: 기본 가치의 50% 추가 획득
나중에 도전하고 정답: 기본 가치의 25% 추가 획득
틀리면: 5 코인 손실
```

### 2단계: 시장 변동

시장 변동 주사위를 굴려서 이번 라운드의
지식 시장 상황을 결정합니다.

```
주사위 결과:
  1: 과학 가치 +50% (과학 카드 거래 활발)
  2: 역사 가치 +50%
  3: 수학 가치 +50%
  4: 사회 가치 +50%
  5: 모든 희귀 카드 가치 +30%
  6: 시장 불안 (모든 가치 -20%, 이번 라운드만)
```

### 3단계: 경매

경매 보드의 카드 5장 중 1장을 경매합니다.
경매는 3단계로 진행됩니다.

**①카드 검토:**
경매 카드의 문제를 모두 읽습니다.
(정답은 아직 공개하지 않음)

**②입찰:**
모든 플레이어가 동시에 입찰 금액을 결정합니다.
입찰 금액을 손에 쥐고 동시에 공개합니다.

**③낙찰:**
가장 높은 금액을 제시한 플레이어가 낙찰받습니다.
```
낙찰: 입찰 금액 지불 + 카드 획득
동일 금액: 재입찰 (두 플레이어만 참여)
```

**④보충:**
경매된 카드 자리에 더미에서 새 카드 1장을 공개합니다.

**투자 전략:**
```
현재 가치 < 입찰 금액: 투기 (위험)
현재 가치 ≈ 입찰 금액: 적정 (안정)
현재 가치 > 입찰 금액: 저평가 매수 (유리)
```

### 4단계: 세트 보너스 확인

현재 보유한 카드 세트 조합을 확인합니다.
세트가 완성되면 즉시 보너스 코인을 받습니다.

```
세트 보너스:
  동일 분야 3장: +20 코인
  동일 분야 5장: +50 코인
  2개 분야 각 3장: +35 코인
  전체 6개 분야 각 1장: +80 코인
  희귀 카드 3장: +60 코인
```

## 지식 채권

### 채권 발행

자금이 부족할 때 지식 채권을 발행해서
다른 플레이어에게 빌릴 수 있습니다.

```
채권 조건:
  금액: 10-50 코인
  이자율: 20%
  상환 기간: 2라운드
```

**채권 절차:**
①채권 카드 작성 (금액·이자율·상환 기간).
②다른 플레이어에게 제안.
③합의하면 즉시 자금 수령.
④기간 내 원금 + 이자 상환.

**채무 불이행:**
상환 기간을 넘기면 이자율이 2배로 증가합니다.
게임 종료 시 미상환 채무는 점수에서 차감됩니다.

## 특수 카드

### 투자 수익 카드

```
지식 펀드:
  5 코인 투자 → 매 라운드 3 코인 수익
  총 10라운드 유효

지식 독점:
  특정 분야 카드를 3장 이상 보유 시
  다른 플레이어의 해당 분야 테스트 수익 10% 징수

지식 특허:
  특정 문제에 정답을 맞히면
  다음 라운드 같은 문제 출제 시 2 코인 수익
```

### 시장 조작 카드

```
지식 인플레이션:
  선택한 분야 카드 가치 이번 라운드 +30%

지식 거품:
  경매 카드 1장의 최저 입찰가를 50% 올림

정보 독점:
  경매 전 카드 1장의 정답을 먼저 확인
```

## 전략 가이드

### 초보자 전략

**1. 자신 있는 분야 집중**
정답률이 높은 분야의 카드를 집중 매수하세요.
지식 테스트에서 안정적인 수입이 생깁니다.

**2. 세트 보너스 목표**
동일 분야 카드 3장 세트를 먼저 만드세요.
+20 코인 보너스는 안정적인 수익원입니다.

**3. 과도한 투기 금지**
자금의 50% 이상을 한 카드에 투자하지 마세요.
자금이 바닥나면 경매에 참여할 수 없습니다.

### 중급 전략

**1. 시장 변동 예측**
시장 변동 주사위 결과를 관찰하세요.
특정 분야가 자주 올라가면 해당 분야 카드를 미리 매수하세요.

**2. 경쟁자 분석**
다른 플레이어의 보유 카드를 파악하고
세트 완성을 방해하는 경매 전략을 사용하세요.

**3. 채권 활용**
좋은 카드가 나왔을 때 자금이 부족하면
채권을 발행해서라도 매수하는 것이 유리할 수 있습니다.

### 고급 전략

**1. 포트폴리오 다변화**
단일 분야 집중보다 다양한 분야 카드를
골고루 보유하면 시장 변동 리스크가 줄어듭니다.

**2. 경매 시장 조작**
자신이 원하는 카드를 다른 플레이어가 눈치채지 못하게
다른 카드에 관심 있는 척 입찰하세요.

**3. 타이밍 경매**
게임 후반에 고가치 카드가 나올 때를 대비해
자금을 비축해두는 전략이 중요합니다.

## 게임 종료

5라운드 완료 후 게임이 끝납니다.

### 최종 자산 계산

```
최종 자산 = 보유 자금
          + 지식 카드 현재 가치 합계
          + 세트 보너스
          + 투자 수익 카드 누적 수익
          - 미상환 채무 원금 + 이자

지식 카드 현재 가치:
  기본 가치 × 시장 배율
  (마지막 시장 변동 적용)
```

### 점수 계산 예시

```
플레이어 A:
  보유 자금: 45 코인
  지식 카드 가치:
    양자역학 (35) + 산업혁명 (25) + 황금비율 (30) = 90 코인
  세트 보너스: 없음
  채무: 없음
  총 자산: 45 + 90 = 135 코인

플레이어 B:
  보유 자금: 20 코인
  지식 카드 가치:
    과학 카드 5장 = 150 코인
  세트 보너스: 동일 분야 5장 = +50 코인
  채무: -15 코인
  총 자산: 20 + 150 + 50 - 15 = 205 코인

→ 플레이어 B 승리!
```

## 변형 규칙

### 주식 시장 지식 경매
- 지식 카드가 주식처럼 거래됨
- 매 라운드 가격이 수요에 따라 변동
- 더 복잡한 경제 시뮬레이션

### 팀 지식 경매
- 2인 1팀으로 자금과 카드 공유
- 팀 합산 자산으로 승부
- 협력적 투자 전략 필요

### 단기 지식 경매
- 3라운드로 단축
- 빠른 의사결정 훈련
- 소요 시간: 20분

### 전문가 지식 경매
- 각 플레이어가 전문 분야 설정
- 전문 분야 카드 획득 시 가치 1.5배
- 전문성의 경제적 가치 학습

## 교육적 활용

### 경제 교육 연계
- **경매**: 시장 가격 결정 원리
- **투자**: 리스크와 수익의 균형
- **채권**: 부채와 이자의 개념
- **포트폴리오**: 자산 다변화 전략

### 지식 연계
- **다분야 지식**: 경제적 가치로 동기 부여
- **전략적 사고**: 지식 투자 계획
- **의사결정**: 불확실한 상황에서 선택

### 수업 활용 방법
1. 경제 개념 (경매·투자·채권) 설명 (10분)
2. 소규모 3인 게임 시범 (15분)
3. 본게임 진행 (35분)
4. "최적의 투자 전략은?" 토론 (10분)
