# 타임라인 마스터 (Timeline Master)

> 역사적 사건·과학적 발견·예술 작품·기술 혁신을
> 시간 순서대로 배열하는 게임입니다.
> 정확한 연대 지식과 시대적 맥락 이해가
> 승부를 결정합니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-6명 |
| 소요 시간 | 20-30분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐ (보통) |
| 교육 분야 | 역사적 사고, 시간 감각, 맥락적 이해, 인과 관계 |

---

## 게임 목표

자신의 카드를 타임라인에 정확하게 배치해서
**가장 먼저 손패를 모두 소진**한 플레이어가 승리합니다.

---

## 준비물

- 타임라인 카드 200장
  - 역사 카드 60장 (앞: 사건명 / 뒤: 연도)
  - 과학 카드 50장
  - 기술 카드 40장
  - 예술 카드 30장
  - 자연 카드 20장
- 타임라인 트랙 (펼치면 2m)
- 시대 구분 마커
  - 고대 (~ 500년)
  - 중세 (500~1500년)
  - 근세 (1500~1800년)
  - 근대 (1800~1900년)
  - 현대 (1900~2000년)
  - 최현대 (2000년~)
- 모래시계 (30초)
- 점수 토큰

## 게임 준비

### 1. 타임라인 트랙 설치

타임라인 트랙을 펼치고
시대 구분 마커를 적절한 위치에 놓습니다.

```
타임라인:
|──BC────|──고대──|──중세──|──근세──|──근대──|──현대──|──최현대──|
BC2333  AD500   1500   1800   1900   2000
```

### 2. 시작 카드 배치

카드를 섞어서 1장을 뽑아 연도를 공개하고
타임라인 트랙에 놓습니다.

```
예시: "한글 창제 (1443년)" 카드가 시작 카드
타임라인: ... [한글 창제] ...
```

### 3. 손패 배분

각 플레이어에게 카드 5장을 나눠줍니다.
카드의 뒷면(연도)을 보지 않고 앞면만 봅니다.

---

## 게임 규칙

### 턴 진행

플레이어는 번갈아 가며 진행합니다.

**①카드 선택**
손패에서 카드 1장을 선택합니다.

**②위치 선언**
타임라인에서 카드를 놓을 위치를 선언합니다.

```
예시:
  현재 타임라인: [고조선 건국] ... [한글 창제] ... [프랑스 혁명]
  
  손패 카드: "임진왜란"
  위치 선언: "한글 창제와 프랑스 혁명 사이에 놓겠습니다."
```

**③검증**
카드를 뒤집어 연도를 확인합니다.

```
임진왜란 = 1592년
한글 창제 = 1443년
프랑스 혁명 = 1789년

1443 < 1592 < 1789 → 정확! ✓
```

**④결과**

**정답:**
타임라인에 카드를 배치합니다.
손패가 줄어듭니다.

**오답:**
카드를 타임라인에 배치하지 못하고
더미에서 카드 1장을 추가로 받습니다.

### 특수 규칙

**정확한 연도 맞히기:**
위치뿐만 아니라 정확한 연도까지 말하면 보너스 점수.

```
"임진왜란은 1592년이고
 한글 창제(1443년)와 프랑스 혁명(1789년) 사이입니다."
→ 정확한 연도 보너스: +3점
```

**도전 규칙:**
다른 플레이어가 위치 선언에 이의를 제기할 수 있습니다.
도전 토큰 1개 사용.
카드를 뒤집기 전에 도전해야 합니다.

```
도전 성공: 도전자 +5점, 원래 플레이어 카드 1장 추가
도전 실패: 도전자 카드 1장 추가
```

## 변형 규칙

### 주제별 타임라인

특정 주제만 사용하는 버전:

```
한국사 타임라인:
  한국사 카드 30장만 사용
  학교 역사 수업 복습에 최적

과학사 타임라인:
  과학 카드만 사용
  과학 발전 흐름 학습

기술 혁신 타임라인:
  기술 카드만 사용
  기술 발전 역사 학습
```

### 팀 타임라인

2인 1팀으로 진행:
팀원끼리 상의해서 위치 결정.
단, 카드 앞면만 보고 상의 (뒷면은 공개 금지).

### 역방향 타임라인

최신에서 오래된 순으로 배열:
다른 관점으로 역사 흐름 이해.

### 연도 추정 타임라인

정확한 연도 대신 50년 이내 오차 허용:
정확한 암기보다 시대 감각 훈련.

```
오차 범위 점수:
  정확: 5점
  ±10년: 4점
  ±25년: 3점
  ±50년: 2점
  ±50년 초과: 0점
```

## 교육적 활용

### 역사 교육 연계
- **연대기**: 사건의 시간적 순서 파악
- **인과 관계**: 앞 사건이 뒤 사건에 미친 영향
- **시대적 맥락**: 같은 시대 다른 지역 비교

### 사고력 연계
- **시간 감각**: 역사적 시간 스케일 이해
- **비교 분석**: 동시대 사건 비교
- **추론**: 연도를 몰라도 전후 맥락으로 추론

### 수업 활용 방법
1. 특정 시대 카드만 선별해서 사용 (10분)
2. 수업 내용과 연결된 카드로 복습 (15분)
3. 전체 카드로 역사 흐름 파악 (25분)
4. 학생이 새 카드 제작 (20분)
