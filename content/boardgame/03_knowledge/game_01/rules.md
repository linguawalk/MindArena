# 지식 피라미드 (Knowledge Pyramid)

> 쉬운 문제부터 어려운 문제까지
> 피라미드 구조로 올라가며
> 다양한 분야의 지식을 겨루는 게임입니다.
> 높은 층으로 올라갈수록 더 큰 점수를 얻습니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-6명 |
| 소요 시간 | 20-30분 |
| 권장 연령 | 10세 이상 |
| 난이도 | ⭐⭐ (보통) |
| 교육 분야 | 다분야 지식, 논리적 사고, 지식 통합 |

---

## 게임 목표

피라미드의 각 층에서 문제를 풀어
**가장 높은 점수**를 얻은 플레이어가 승리합니다.
높은 층의 문제일수록 더 많은 점수를 얻습니다.

---

## 준비물

- 피라미드 게임 보드
- 문제 카드 덱 (층별)
  - 1층 (⭐): 60장
  - 2층 (⭐⭐): 50장
  - 3층 (⭐⭐⭐): 40장
  - 4층 (⭐⭐⭐⭐): 30장
  - 5층 (⭐⭐⭐⭐⭐): 20장
- 분야 주사위 (6면: 과학·역사·수학·언어·예술·자연)
- 점수 토큰
- 모래시계 (30초)
- 플레이어 마커

### 피라미드 보드 구조

```
         [5층]          ← 최고 난이도 (50점)
        [4층][4층]       ← 고난이도 (30점)
      [3층][3층][3층]    ← 중난이도 (20점)
    [2층][2층][2층][2층] ← 기초 (10점)
  [1층][1층][1층][1층][1층] ← 입문 (5점)

각 칸: 플레이어 마커 위치
```

## 문제 카드 구성

### 분야별 문제 예시

**과학 분야:**
```
1층 (⭐):
  Q: 물의 화학식은?
  A: H₂O

2층 (⭐⭐):
  Q: 빛의 3원색은?
  A: 빨강·초록·파랑

3층 (⭐⭐⭐):
  Q: DNA의 이중 나선 구조를 발견한 과학자 2명은?
  A: 왓슨·크릭 (1953년)

4층 (⭐⭐⭐⭐):
  Q: 광합성의 화학 반응식을 쓰세요.
  A: 6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂

5층 (⭐⭐⭐⭐⭐):
  Q: 상대성 이론에서 질량-에너지 등가 원리를
     설명하고 그 공식의 의미를 서술하세요.
  A: E=mc² (에너지=질량×빛의속도²)
     질량과 에너지는 서로 전환 가능
```

**역사 분야:**
```
1층 (⭐):
  Q: 한국 최초의 국가는?
  A: 고조선

2층 (⭐⭐):
  Q: 조선을 건국한 왕은?
  A: 이성계 (태조)

3층 (⭐⭐⭐):
  Q: 임진왜란이 발생한 연도와
     이순신 장군의 주요 해전 2개는?
  A: 1592년 / 한산도 대첩·명량 해전

4층 (⭐⭐⭐⭐):
  Q: 산업혁명이 영국에서 먼저 시작된
     3가지 이유를 설명하세요.
  A: 풍부한 석탄·철광석 / 식민지 시장 /
     의회 민주주의로 인한 자유로운 경제 활동

5층 (⭐⭐⭐⭐⭐):
  Q: 1·2차 세계대전의 발생 원인과 결과를
     비교 분석하고 공통점과 차이점을 서술하세요.
  A: (다각도 분석 필요 — 심사위원 판정)
```

**수학 분야:**
```
1층 (⭐):
  Q: 삼각형의 내각의 합은?
  A: 180°

2층 (⭐⭐):
  Q: 피타고라스 정리를 공식으로 쓰세요.
  A: a² + b² = c²

3층 (⭐⭐⭐):
  Q: 원의 넓이 공식과 원주 공식을 쓰세요.
  A: 넓이=πr² / 원주=2πr

4층 (⭐⭐⭐⭐):
  Q: 등차수열의 일반항과 합 공식을 쓰고
     3, 7, 11, 15...의 10번째 항을 구하세요.
  A: aₙ=a₁+(n-1)d / Sₙ=n(a₁+aₙ)/2
     a₁₀ = 3 + 9×4 = 39

5층 (⭐⭐⭐⭐⭐):
  Q: 미분의 개념을 직관적으로 설명하고
     f(x)=x³+2x²-5x+1의 도함수를 구하세요.
  A: 순간 변화율 / f'(x)=3x²+4x-5
```

**언어 분야:**
```
1층 (⭐):
  Q: "지구"를 영어로?
  A: Earth

2층 (⭐⭐):
  Q: 한국어 "은/는"과 "이/가"의 차이는?
  A: 은/는=주제 표시 / 이/가=주어 표시

3층 (⭐⭐⭐):
  Q: 셰익스피어의 4대 비극을 모두 말하세요.
  A: 햄릿·오셀로·리어왕·맥베스

4층 (⭐⭐⭐⭐):
  Q: 은유와 직유의 차이를 예시와 함께 설명하세요.
  A: 직유="~같다/처럼" 사용
     은유=직접 "A는 B다"
     예: "그는 사자 같다" vs "그는 사자다"

5층 (⭐⭐⭐⭐⭐):
  Q: 언어가 사고를 결정한다는
     사피어-워프 가설을 설명하고
     그 한계를 비판하세요.
  A: (논증 필요 — 심사위원 판정)
```

**예술 분야:**
```
1층 (⭐):
  Q: 색의 3원색은?
  A: 빨강·파랑·노랑

2층 (⭐⭐):
  Q: 모나리자를 그린 화가는?
  A: 레오나르도 다빈치

3층 (⭐⭐⭐):
  Q: 인상주의의 특징 3가지를 설명하세요.
  A: 빛의 변화 포착 / 야외 작업 / 짧은 붓터치

4층 (⭐⭐⭐⭐):
  Q: 바흐·모차르트·베토벤을 시대 순으로 나열하고
     각각의 음악적 특징을 설명하세요.
  A: 바흐(바로크)→모차르트(고전)→베토벤(고전→낭만)

5층 (⭐⭐⭐⭐⭐):
  Q: 현대 예술에서 "예술이란 무엇인가?"에 대한
     뒤샹의 도전을 설명하고 그 의미를 평가하세요.
  A: (논증 필요 — 심사위원 판정)
```

**자연 분야:**
```
1층 (⭐):
  Q: 지구에서 가장 큰 동물은?
  A: 흰긴수염고래

2층 (⭐⭐):
  Q: 광합성이 일어나는 세포 기관은?
  A: 엽록체

3층 (⭐⭐⭐):
  Q: 먹이 사슬에서 생산자·소비자·분해자의
     역할을 각각 설명하세요.
  A: 생산자=광합성 / 소비자=다른 생물 섭취 /
     분해자=유기물 분해

4층 (⭐⭐⭐⭐):
  Q: 판 구조론을 설명하고
     지진과 화산의 발생 원리를 연결하세요.
  A: 지각판 이동 → 경계에서 충돌·분리 →
     지진(판의 움직임) / 화산(마그마 분출)

5층 (⭐⭐⭐⭐⭐):
  Q: 기후 변화의 메커니즘을 설명하고
     생태계에 미치는 영향을 다각도로 분석하세요.
  A: (다각도 분석 필요 — 심사위원 판정)
```

## 게임 규칙

### 게임 준비

**1. 문제 카드 배치**
각 층별 문제 카드를 섞어서
피라미드 보드의 해당 층에 뒤집어 놓습니다.

**2. 시작 위치**
모든 플레이어는 피라미드 1층에서 시작합니다.

**3. 선공 결정**
나이가 가장 적은 플레이어가 선공입니다.

### 턴 진행

**①분야 선택**
분야 주사위를 굴립니다.
나온 분야의 문제를 풀어야 합니다.
(단, 1라운드는 원하는 분야 선택 가능)

**②층 선택**
현재 자신의 마커가 있는 층 또는
그 위 층의 문제를 선택할 수 있습니다.

```
현재 2층에 있으면: 2층 또는 3층 문제 선택 가능
단, 층을 건너뛸 수 없음 (2층 → 4층 불가)
```

**③문제 카드 공개**
선택한 층의 문제 카드를 공개합니다.
모래시계를 뒤집습니다. (30초)

**④답변**
30초 이내에 답변합니다.

**⑤채점**
다른 플레이어들이 정답 카드를 확인해서
정오를 판정합니다.

**정답:**
해당 층의 점수를 획득합니다.
마커를 해당 층으로 이동합니다.

**오답:**
점수 없음.
마커는 현재 위치 유지.

### 특수 규칙

**도전 규칙:**
다른 플레이어가 오답을 낸 문제에
즉시 도전할 수 있습니다.
도전 성공 시 해당 점수의 50% 추가 획득.
도전 실패 시 자신의 점수 10점 감점.

**힌트 규칙:**
한 게임에 최대 2번, 다른 플레이어에게 힌트를 요청할 수 있습니다.
힌트를 준 플레이어는 5점을 받습니다.
힌트를 받고 정답을 맞히면 점수의 50%만 획득합니다.

**연속 정답 보너스:**
3문제 연속 정답 시 +10점 보너스.
5문제 연속 정답 시 +25점 보너스.

## 점수 체계

```
층별 기본 점수:
  1층: 5점
  2층: 10점
  3층: 20점
  4층: 30점
  5층: 50점

보너스 점수:
  도전 성공: 해당 층 점수 × 50%
  힌트 제공: +5점
  연속 3문제: +10점
  연속 5문제: +25점
  피라미드 완등 (5층 도달): +30점

페널티:
  도전 실패: -10점
  힌트 받고 정답: 점수 × 50%
```

## 게임 종료

다음 중 하나가 발생하면 게임이 끝납니다.
- 한 플레이어가 피라미드 5층에 도달
- 문제 카드 덱이 소진
- 합의로 종료 (라운드 수 제한 설정)

## 전략 가이드

### 초보자 전략

**1. 자신 있는 분야 집중**
분야 주사위가 불리하게 나오면
힌트 요청을 전략적으로 사용하세요.

**2. 낮은 층 안정적 공략**
1-2층 문제를 연속으로 맞혀
연속 정답 보너스를 노리세요.

**3. 도전 신중히**
도전 실패 시 -10점이므로
확신이 있을 때만 도전하세요.

### 중급 전략

**1. 층 선택 타이밍**
다른 플레이어가 높은 층에 있을 때
자신도 빠르게 올라가야 합니다.

**2. 도전 기회 활용**
다른 플레이어가 높은 층에서 오답을 낼 때
도전으로 큰 점수를 획득할 수 있습니다.

**3. 힌트 전략**
힌트를 주면 5점을 받으면서
상대방의 점수를 절반으로 줄입니다.

### 고급 전략

**1. 점수 분석**
현재 점수 차이를 계산해서
역전에 필요한 고층 문제에 집중하세요.

**2. 분야 준비**
게임 전에 6가지 분야를 골고루 준비하세요.
특정 분야에 약하면 큰 불이익입니다.

**3. 상대 견제**
선두 플레이어가 쉬운 문제에 도전할 때
도전으로 점수 차이를 줄이세요.

## 변형 규칙

### 팀 지식 피라미드
- 2인 1팀으로 구성
- 팀원이 번갈아 문제 답변
- 팀 합산 점수로 승부

### 전문가 피라미드
- 각 플레이어가 전문 분야 1개 선택
- 전문 분야 문제는 2배 점수
- 다른 분야 문제는 정상 점수

### 빠른 피라미드
- 모래시계 15초로 단축
- 1-3층만 사용
- 소요 시간: 10-15분

### 교과서 피라미드
- 특정 교과 과목에 집중
- 학교 수업 내용 기반 문제
- 학습 복습에 최적화

---

## 교육적 활용

### 교과 연계
- **과학**: 기본 개념부터 심화 원리까지
- **역사**: 사실 암기부터 분석적 이해까지
- **수학**: 공식 암기부터 응용 문제까지
- **국어/영어**: 어휘부터 문학 분석까지

### 사고력 연계
- **기억력**: 다양한 분야 지식 정리
- **분석력**: 고층 문제의 복잡한 분석
- **표현력**: 구술 답변 능력

### 수업 활용 방법
1. 해당 단원 복습용으로 문제 카드 제작 (20분)
2. 수업 마무리 복습 게임 (15분)
3. 학기말 종합 복습 대회 (30분)
4. 학생이 직접 문제 출제 (20분)
