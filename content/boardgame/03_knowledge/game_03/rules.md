# 분야 대결 (Field Battle)

> 각 플레이어가 전문 분야를 선택하고
> 분야별 지식으로 영역을 확장하는
> 지식 기반 영역 전쟁 게임입니다.
> 자신의 분야를 지키면서
> 다른 분야를 침략하세요.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 2-4명 |
| 소요 시간 | 25-35분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 전문 지식, 융합 사고, 전략적 판단, 논증 |

---

## 게임 목표

게임 종료 시 가장 많은 **지식 영역**을
차지한 플레이어가 승리합니다.
지식 영역은 문제를 맞혀서 획득하고
방어전에서 지키거나 공격전으로 빼앗을 수 있습니다.

---

## 준비물

- 지식 지도 보드 (6×6 격자)
- 분야 카드 6장 (과학·역사·수학·사회·예술·자연)
- 문제 카드 덱 (분야별 30장씩 = 180장)
- 영역 마커 (플레이어별 색상)
- 전투 주사위 2개
- 방어 토큰 (플레이어당 5개)
- 동맹 토큰 (플레이어당 2개)
- 점수 기록판

### 지식 지도 보드

```
     과학    역사    수학    사회    예술    자연
   ┌──────┬──────┬──────┬──────┬──────┬──────┐

   ├──────┼──────┼──────┼──────┼──────┼──────┤

   ├──────┼──────┼──────┼──────┼──────┼──────┤

   ├──────┼──────┼──────┼──────┼──────┼──────┤

   ├──────┼──────┼──────┼──────┼──────┼──────┤

   ├──────┼──────┼──────┼──────┼──────┼──────┤

각 셀 = 특정 분야 교차점
예: (과학, 역사) = "과학사" 영역
    (수학, 예술) = "수학과 예술의 교차" 영역
```

## 게임 준비

### 1. 분야 선택

각 플레이어가 순서대로 전문 분야를 선택합니다.

```
2인 게임: 각 2개 분야
3인 게임: 각 2개 분야 (1개 분야는 중립)
4인 게임: 각 1-2개 분야
```

선택한 분야는 자신의 "홈 영역"이 됩니다.

### 2. 초기 영역 배치

각 플레이어는 자신의 홈 분야 행과 열에
영역 마커를 배치합니다.

```
예시 (2인 게임):
  플레이어 1 (과학, 수학 선택):
    과학 행 전체 + 수학 행 전체 = 12칸

  플레이어 2 (역사, 사회 선택):
    역사 행 전체 + 사회 행 전체 = 12칸

  중립: 예술·자연 행 = 12칸
```

### 3. 문제 카드 배치

각 분야의 문제 카드를 해당 위치에 뒤집어 놓습니다.

---

## 게임 규칙

### 턴 진행

자신의 턴에 다음 중 하나를 선택합니다.

**①영역 확장 (자신의 영역 인접 칸 공략)**
**②방어 강화 (자신의 영역에 방어 토큰 배치)**
**③동맹 체결 (다른 플레이어와 임시 동맹)**
**④중립 영역 공략**

### 영역 확장 (공격)

**대상 선택:**
자신의 영역에 인접한 빈 칸 또는 상대방 영역을 선택합니다.

**문제 출제:**
해당 칸의 행·열 분야 중 하나를 선택해서
문제 카드를 뽑습니다.

```
예시: (과학, 역사) 칸 공략
  → 과학 문제 또는 역사 문제 선택
```

**답변:**
45초 이내에 답변합니다.

**결과:**
```
정답: 해당 칸에 자신의 영역 마커 배치
오답: 해당 칸은 중립 상태 유지
```

**방어전:**
상대방 영역을 공격할 때
방어 토큰이 있으면 방어자도 같은 문제에 답변합니다.

```
공격자 정답 + 방어자 오답: 공격 성공 (영역 빼앗김)
공격자 정답 + 방어자 정답: 방어 성공 (영역 유지)
공격자 오답: 공격 실패 (방어 토큰 소모 없음)
```

### 방어 강화

자신의 영역 1칸에 방어 토큰을 배치합니다.
방어 토큰이 있는 칸은 공격 시 방어전 발생.
방어 토큰은 방어전 후 제거됩니다.

### 동맹 체결

다른 플레이어 1명과 임시 동맹을 맺습니다.
동맹 기간: 2라운드.

```
동맹 효과:
  동맹국의 영역을 공격하지 않는 약속
  동맹국의 문제 답변 시 힌트 1회 가능
  동맹 해제 시 다음 라운드부터 효력 없음
```

동맹 토큰을 사용하면 동맹을 맺습니다.
게임당 최대 2번 동맹 가능.

## 융합 영역 규칙

### 교차 영역

두 분야가 교차하는 칸은 **융합 영역**으로
두 분야의 지식이 모두 필요합니다.

```
융합 영역 예시:
  (과학, 역사) = "과학사"
    → 과학 문제 + 역사 문제 모두 정답 필요

  (수학, 예술) = "수리 예술"
    → 수학 문제 + 예술 문제 모두 정답 필요

  (사회, 자연) = "환경 사회학"
    → 사회 문제 + 자연 문제 모두 정답 필요
```

**융합 영역 공략:**
두 문제 모두 정답 시 해당 칸 획득.
두 문제 중 1개만 정답 시 "반점령" 상태.
(다음 턴에 나머지 1문제만 풀면 완전 점령)

**융합 영역 점수:**
```
단일 분야 영역: 1점
융합 영역 (반점령): 0.5점
융합 영역 (완전 점령): 3점
```

## 문제 카드 예시

### 과학 × 역사 융합 문제

```
카드 #01:
  과학 문제: 뉴턴이 발견한 운동 법칙 3가지는?
  역사 문제: 과학 혁명이 일어난 시대와 대표 과학자는?

카드 #02:
  과학 문제: 페니실린의 발견 원리는?
  역사 문제: 페니실린이 2차 세계대전에 미친 영향은?
```

### 수학 × 예술 융합 문제

```
카드 #01:
  수학 문제: 황금비(φ)의 값은 약 얼마인가?
  예술 문제: 황금비가 사용된 유명 예술 작품 2개는?

카드 #02:
  수학 문제: 프랙탈의 수학적 특성은?
  예술 문제: 프랙탈 패턴이 나타나는 자연 현상 2가지는?
```

### 사회 × 자연 융합 문제

```
카드 #01:
  사회 문제: 탄소세의 개념과 목적은?
  자연 문제: 탄소 순환의 과정을 설명하세요.

카드 #02:
  사회 문제: 환경 난민이 발생하는 사회적 원인은?
  자연 문제: 해수면 상승의 과학적 메커니즘은?
```

## 특수 이벤트 카드

매 3라운드마다 이벤트 카드를 1장 공개합니다.

```
이벤트 카드 목록:

지식 폭풍:
  다음 라운드 모든 문제 난이도 1단계 상승.

영역 재편:
  모든 플레이어가 영역 마커 2개를 이동 가능.

중립 해제:
  중립 영역 3칸이 무작위로 공개 입찰.
  가장 높은 점수를 제시한 플레이어가 획득.

동맹 파기:
  현재 모든 동맹 즉시 해제.

지식 교환:
  두 플레이어가 합의하면 영역 1칸씩 교환 가능.

전문가 등장:
  특정 분야 문제를 다음 턴에 2배 점수로 획득.
```

## 전략 가이드

### 초보자 전략

**1. 홈 영역 우선 방어**
자신의 전문 분야 영역부터 방어 토큰으로 강화하세요.
홈 영역을 빼앗기면 점수 회복이 어렵습니다.

**2. 인접 영역 먼저 공략**
자신의 영역과 인접한 칸부터 확장하세요.
멀리 있는 칸은 경로를 만들기 어렵습니다.

**3. 융합 영역 노리기**
융합 영역은 3점으로 일반 영역의 3배입니다.
두 분야를 모두 아는 영역을 집중 공략하세요.

### 중급 전략

**1. 상대방 약점 파악**
상대방이 어떤 분야를 어려워하는지 파악하고
그 분야 문제가 많은 칸을 공략하세요.

**2. 동맹 타이밍**
혼자 이기기 어려울 때 동맹을 제안하세요.
하지만 동맹이 너무 강해지면 나중에 부담이 됩니다.

**3. 반점령 활용**
융합 영역의 반점령 상태를 여러 칸 만들어두고
한꺼번에 완전 점령하는 전략도 유효합니다.

### 고급 전략

**1. 이벤트 예측**
3라운드마다 이벤트가 발생합니다.
이벤트 직전에 전략을 조정하세요.

**2. 영역 가치 계산**
단순 칸 수보다 융합 영역의 가치를 우선시하세요.
3점짜리 융합 영역 1칸 = 1점짜리 일반 영역 3칸.

**3. 동맹 배신 타이밍**
동맹이 자신에게 불리해지면 적절한 시점에 해제하세요.
단, 신뢰가 깨지면 다음 동맹 제안이 어려워집니다.

## 게임 종료

다음 중 하나가 발생하면 게임이 끝납니다.
- 전체 36칸이 모두 점령됨
- 5라운드 완료
- 한 플레이어가 18칸 이상 점령 (과반수)

### 최종 점수 계산

```
영역 점수:
  일반 영역 1칸: 1점
  융합 영역 반점령: 0.5점
  융합 영역 완전 점령: 3점

보너스:
  홈 분야 전체 점령: +10점
  연속 5칸 이상 영역: +5점
  상대 홈 분야 침략 성공: +5점

최종 점수가 높은 플레이어 승리
```

## 변형 규칙

### 개인 교사 분야 대결
- 각 플레이어가 실제 전공·관심 분야 선택
- 자신만의 문제 카드 직접 제작
- 더 깊은 전문 지식 필요

### 협력 분야 대결
- 팀이 함께 모든 36칸 점령 목표
- 외부 '난이도 상승' 카드가 방해
- 순수 협력 버전

### 토너먼트 분야 대결
- 1 vs 1 토너먼트 형식
- 각 경기 20분
- 누적 점수로 최종 승자 결정

## 교육적 활용

### 교과 연계
- **통합 교과**: 여러 분야 지식의 융합적 이해
- **창의 융합**: 분야 간 연결 사고
- **자기 주도 학습**: 전문 분야 심화 학습 동기

### 사고력 연계
- **전략적 사고**: 영역 확장 계획
- **메타 인지**: 자신의 강약점 파악
- **협상**: 동맹 체결과 전략적 소통

### 수업 활용 방법
1. 학생이 전공 분야 선택 + 문제 카드 제작 (30분)
2. 수업 시간에 게임 진행 (35분)
3. 융합 영역 토론 (15분)
4. 가장 창의적인 융합 사례 발표 (10분)
