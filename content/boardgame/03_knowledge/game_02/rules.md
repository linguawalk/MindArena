# 개념 연결 (Concept Chain)

> 하나의 개념에서 시작해서
> 논리적으로 연결되는 개념을 이어가는
> 지식 연결 게임입니다.
> 얼마나 멀리, 얼마나 창의적으로
> 개념을 연결할 수 있는지 겨룹니다.

---

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플레이어 | 3-6명 |
| 소요 시간 | 15-25분 |
| 권장 연령 | 11세 이상 |
| 난이도 | ⭐⭐⭐ (중상) |
| 교육 분야 | 개념 이해, 지식 연결, 창의적 사고, 논증 능력 |

---

## 게임 목표

시작 개념 카드에서 출발해서
가장 긴 개념 연결 사슬을 만든 플레이어가 승리합니다.
단, 모든 연결에는 **논리적 근거**가 있어야 합니다.

---

## 준비물

- 개념 카드 120장 (분야별 20장씩)
  - 과학 / 역사 / 수학 / 사회 / 예술 / 자연
- 연결 카드 60장 (연결 유형별)
- 도전 토큰 (플레이어당 3개)
- 점수 토큰
- 모래시계 (45초)
- 연결 기록판

### 개념 카드 예시

```
과학:
  중력 / 빛 / 원자 / 진화 / DNA / 에너지
  열역학 / 전자기 / 양자 / 화학 반응 / ...

역사:
  혁명 / 제국 / 무역 / 전쟁 / 민주주의
  식민지 / 문명 / 종교 / 발명 / 외교 / ...

수학:
  소수 / 무한 / 확률 / 함수 / 기하
  방정식 / 통계 / 집합 / 논리 / 위상 / ...

사회:
  경제 / 법률 / 문화 / 교육 / 환경
  인구 / 미디어 / 권력 / 불평등 / 복지 / ...

예술:
  음악 / 회화 / 건축 / 문학 / 영화
  조각 / 무용 / 사진 / 디자인 / 연극 / ...

자연:
  생태계 / 기후 / 진화 / 멸종 / 공생
  먹이사슬 / 적응 / 생물다양성 / 자원 / ...
```

### 연결 카드 유형

```
인과 관계 (→):
  "A가 B의 원인이다."
  예: 중력 → 조석 현상

포함 관계 (⊃):
  "A는 B를 포함한다."
  예: 생태계 ⊃ 먹이사슬

유사성 (~):
  "A와 B는 유사한 원리를 가진다."
  예: 진화 ~ 문화 변화

대립 관계 (↔):
  "A와 B는 대립한다."
  예: 민주주의 ↔ 독재

역사적 연결 (⇒):
  "A가 역사적으로 B로 이어졌다."
  예: 산업혁명 ⇒ 도시화

비유 관계 (≈):
  "A는 B에 비유될 수 있다."
  예: DNA ≈ 설계도

응용 관계 (→):
  "A의 원리가 B에 응용된다."
  예: 확률 → 보험
```

## 게임 준비

### 1. 시작 개념 선택

개념 카드를 섞어서 1장을 공개합니다.
이것이 이번 라운드의 시작 개념입니다.

```
예시: 시작 개념 = "중력"
```

### 2. 손패 배분

각 플레이어에게 개념 카드 5장을 나눠줍니다.
연결 카드 3장도 나눠줍니다.

### 3. 연결 기록판 설정

```
연결 기록판:
[중력] → ? → ? → ? → ? → ...
```

---

## 게임 규칙

### 라운드 진행

플레이어는 번갈아 가며 개념을 연결합니다.

### 턴 순서

**①개념 선택**
손에 있는 개념 카드 중 1장을 선택합니다.

**②연결 카드 선택**
연결 유형을 나타내는 연결 카드 1장을 선택합니다.

**③연결 선언**
이전 개념과 선택한 개념을 연결하는
**논리적 근거**를 30초 이내에 설명합니다.

```
예시:
  이전 개념: "중력"
  연결 카드: 인과 관계 (→)
  새 개념: "조석 현상"
  설명: "달의 중력이 바닷물을 당겨서
         조석 현상이 발생합니다."
```

**④도전 기회**
다른 플레이어들이 연결의 논리성에 이의를 제기할 수 있습니다.
과반수가 동의하면 연결 무효.
연결이 유효하면 도전자 -5점.

**⑤카드 보충**
개념 카드 1장과 연결 카드 1장을 보충합니다.

### 점수 계산

```
기본 연결 점수:
  인과 관계: 10점
  포함 관계: 8점
  유사성: 12점
  대립 관계: 10점
  역사적 연결: 10점
  비유 관계: 15점
  응용 관계: 12점

보너스 점수:
  분야 전환 연결 (다른 분야로): +5점
  창의적 연결 (심사위원 인정): +10점
  3개 이상 연속 사슬 완성: +8점
  사슬 길이 보너스:
    10개 이상: +20점
    15개 이상: +35점
    20개 이상: +50점
```

### 사슬 끊기

플레이어가 연결을 못하거나 도전에 의해 연결이 무효가 되면
사슬이 끊어집니다.
끊어진 사슬의 총 길이에 따라 점수가 계산됩니다.
새 시작 개념 카드로 다음 사슬을 시작합니다.

## 연결 예시

### 사슬 예시 1: 중력에서 민주주의까지

```
[중력]
  → (인과) [조석 현상]
    ⊃ (포함) [해양 생태계]
      ~ (유사) [사회 생태계]
        ⇒ (역사) [민주주의]

설명:
  중력 → 조석 현상: 달의 중력이 조석 유발
  조석 현상 ⊃ 해양 생태계: 조석이 해양 생태계의 핵심 요소
  해양 생태계 ~ 사회 생태계: 둘 다 균형과 순환 구조
  사회 생태계 ⇒ 민주주의: 균형 잡힌 사회 구조가
                            민주주의로 발전

총 4개 연결, 분야 전환 2회
기본 점수: 10+8+12+10 = 40점
분야 전환 보너스: +10점
합계: 50점
```

### 사슬 예시 2: DNA에서 음악까지

```
[DNA]
  → (인과) [유전]
    ⊃ (포함) [진화]
      → (응용) [인공지능]
        ≈ (비유) [창의성]
          ~ (유사) [음악]

설명:
  DNA → 유전: DNA가 유전 정보 담당
  유전 ⊃ 진화: 유전이 진화의 기반
  진화 → 인공지능: 진화 알고리즘이 AI에 응용
  인공지능 ≈ 창의성: AI가 창의적 작업 수행 (비유)
  창의성 ~ 음악: 음악은 창의성의 표현

총 5개 연결, 분야 전환 3회
기본 점수: 10+8+12+15+12 = 57점
분야 전환 보너스: +15점
합계: 72점
```

## 도전 규칙 상세

### 도전 가능 조건

다음 경우에 도전할 수 있습니다.
①연결의 논리적 근거가 부정확하거나 억지스러울 때.
②연결 카드 유형이 잘못 사용됐을 때.
③설명이 30초를 초과했을 때.

### 도전 절차

**①도전 선언:**
"도전!"을 외칩니다.
도전 토큰 1개를 사용합니다.

**②근거 제시:**
왜 연결이 잘못됐는지 설명합니다.

**③심판 투표:**
도전자와 원래 플레이어를 제외한 나머지 플레이어들이
연결 유효 여부를 투표합니다.

**④결과:**
```
도전 성공 (과반수 무효):
  연결 무효 처리
  도전자: +5점 (부분 보상)
  연결자: 0점

도전 실패 (과반수 유효):
  연결 유지
  도전자: -5점
  연결자: 정상 점수
```

## 특수 카드

### 와일드 개념 카드
```
사용법: 어떤 개념이든 원하는 개념으로 사용 가능
조건: 사용 시 연결 설명을 더 자세히 해야 함
보너스: 성공 시 +5점 추가
```

### 점프 연결 카드
```
사용법: 현재 사슬의 마지막 개념 대신
        2단계 전 개념에서 연결 가능
조건: 더 긴 연결 경로 제시 필요
보너스: 성공 시 +10점 추가
```

### 역방향 카드
```
사용법: 사슬을 역방향으로 연결
        (새 개념 → 기존 마지막 개념)
조건: 역방향 논리 설명 필요
보너스: 성공 시 +8점 추가
```

## 변형 규칙

### 협력 개념 연결
- 팀이 함께 하나의 긴 사슬 만들기
- 팀원이 번갈아 연결
- 목표: 20개 이상의 사슬

### 제한 분야 개념 연결
- 지정된 2개 분야의 카드만 사용
- 더 좁은 범위에서의 창의적 연결
- 특정 교과 복습에 활용

### 역방향 개념 연결
- 목표 개념 카드를 먼저 공개
- 시작 개념에서 목표 개념까지 연결
- 경로 계획 능력 필요

### 시간 압박 개념 연결
- 모래시계 15초로 단축
- 빠른 연상과 직관 필요
- 심화 버전

## 교육적 활용

### 교과 연계
- **과학**: 과학 개념 간 연결 관계 이해
- **사회**: 사회 현상의 인과 관계 분석
- **국어**: 개념 설명과 논증 능력 향상
- **통합**: 교과 간 융합 사고

### 사고력 연계
- **연결적 사고**: 다른 분야 개념 연결
- **논증**: 연결의 근거 설명
- **비판적 사고**: 타인의 연결 평가

### 수업 활용 방법
1. 단원 핵심 개념 카드로 시작 (5분)
2. 팀별 개념 사슬 만들기 (20분)
3. 가장 창의적인 사슬 발표 (10분)
4. 교과서에서 연결 찾기 (15분)
