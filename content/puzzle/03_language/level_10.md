## 문제 1

다음 최소 대립쌍(minimal pair)을 분석하고
한국어 음운 체계를 설명하세요.

불/뿔, 달/탈, 발/팔
살/쌀, 가/카, 바/파
물/뮬(?), 한/헌, 금/긁(?)

(1) 최소 대립쌍의 원리를 설명하세요.
(2) 한국어의 자음 체계(평음/격음/경음)를 분석하세요.
(3) 이 구분이 외국인 한국어 학습에서 어려운 이유는?

### 힌트 1
최소 대립쌍: 하나의 음소만 다른 두 단어.
불/뿔: ㅂ/ㅂ의 차이가 아닌 ㅜ/ㅜ+ㄹ?
아니면 불(火)/뿔(角): ㅂ/ㅃ의 차이.

### 힌트 2
한국어 삼중 대립:
평음: ㅂ, ㄷ, ㄱ, ㅅ, ㅈ
격음: ㅍ, ㅌ, ㅋ, ㅎ, ㅊ
경음: ㅃ, ㄸ, ㄲ, ㅆ, ㅉ

### 정답
최소대립쌍=한음소차이 / 삼중대립(평·격·경)=한국어고유특성 / 외국인학습어려움=삼중구분없는언어들

### 해설
한국어 음운론 분석:
최소 대립쌍(minimal pair):
한 음소(phoneme)의 차이로 의미가 구별되는 쌍.
불[bul]/뿔[pul]: ㅂ(유성)·ㅍ의 차이? → 실제로 불/풀.
살[sal]/쌀[ssal]: ㅅ/ㅆ (평음/경음).
가[ka]/카[kha]: ㄱ/ㅋ (평음/격음).
삼중 대립:
평음(tense): ㅂ, ㄷ, ㄱ - 보통 강도.
격음(aspirated): ㅍ, ㅌ, ㅋ - 기식음(숨이 나옴).
경음(fortis): ㅃ, ㄸ, ㄲ - 긴장음(성문 긴장).
세계 언어 중 드문 3중 대립.
외국인 학습 어려움:
영어: p/b (유성/무성) 2중 대립.
한국어: 평음/격음/경음 3중 대립.
영어 화자: ㅂ/ㅍ/ㅃ 구별 매우 어려움.
청각적·운동적 훈련이 필요합니다.

## 문제 2

다음 한국어 형태론을 분석하세요.

형태소 분석:
먹었습니다 = 먹 + 었 + 습니다
공부하지않겠습니다 = 공부 + 하 + 지 + 않 + 겠 + 습니다
아름다움 = 아름답 + ㅁ (불규칙)

(1) 각 형태소의 의미와 기능을 설명하세요.
(2) 한국어 어미 체계의 특성을 분석하세요.
(3) 불규칙 활용의 원리를 설명하세요.

### 힌트 1
형태소: 의미를 가진 최소 언어 단위.
먹(어근) + 었(과거시제) + 습니다(공손형).

### 힌트 2
아름답+ㅁ: "아름답" → "아름다움"
ㅂ불규칙: ㅂ + 모음 → 오/우 변환.

### 정답
형태소분석=의미최소단위분리 / 한국어=교착어(어미축적) / 불규칙=음운환경변화

### 해설
한국어 형태론 분석:
형태소 분석:
먹(어근)+었(과거)+습니다(청자높임·평서).
공부(명사어근)+하(동사화)+지(부정연결)+않(부정)+겠(의지)+습니다.
교착어 특성:
각 형태소가 독립적 의미를 가지며 순차 결합.
어미만 수십 가지: -아/어, -으면, -고, -지, -는데...
굴절어(영어)와 달리 형태 변형 없이 첨가.
불규칙 활용:
ㅂ불규칙: 아름답+아→아름다워(ㅂ→우/오).
ㄷ불규칙: 걷+어→걸어(ㄷ→ㄹ).
ㅅ불규칙: 짓+어→지어(ㅅ 탈락).
르불규칙: 모르+아→몰라(르→ㄹㄹ).
이런 불규칙은 역사적 음운 변화의 흔적입니다.

## 문제 3

다음 촘스키의 변형생성문법 개념을
한국어에 적용해 분석하세요.

심층 구조 vs 표층 구조:
"나는 밥을 먹었다."
"밥을 나는 먹었다."
→ 표층 구조는 다르지만 심층 구조는 같음.

재귀 구조:
"민준은 [서아가 [지호를 좋아한다고] 말했다고] 들었다."

변형 규칙:
수동화: "민준이 사과를 먹었다" → "사과가 민준에 의해 먹혔다."

### 힌트 1
심층 구조: 의미적 기저 구조.
표층 구조: 실제 발화 형태.
같은 심층 구조 → 다양한 표층 실현.

### 힌트 2
재귀성: 문장 안에 문장이 무한히 포함될 수 있음.
이것이 인간 언어의 창의성의 원천.

### 정답
심층=의미기저·표층=실현형태 / 재귀성=무한문장생성 / 변형규칙=구조이동

### 해설
변형생성문법 한국어 적용:
심층·표층 구조:
동일 심층: "나는 밥을 먹었다" = "밥을 나는 먹었다".
다른 표층: 어순 변화, 강조점 차이.
촘스키: 언어 능력(competence) vs 언어 수행(performance).
재귀성(recursion):
"민준은 [... [지호를 좋아한다고] 말했다고] 들었다."
이론적으로 무한 중첩 가능.
이것이 유한한 규칙으로 무한한 문장을 생성하는 원리.
촘스키: 재귀성이 인간 언어의 핵심 특성.
수동 변형:
능동: 민준이(주어) + 사과를(목적어) + 먹었다.
수동: 사과가(주어) + 민준에 의해(행위자) + 먹혔다.
한국어 수동은 영어보다 덜 발달(보조 용언 활용 많음).

## 문제 4

다음 사회언어학의 핵심 개념인
언어 변이(variation)를 분석하세요.

같은 화자가 다르게 말하는 경우:
공식 상황: "오늘 회의를 시작하겠습니다."
비공식: "야, 우리 시작하자."
SNS: "오늘 회의 ㄱㄱ"
문자: "오늘 회의 시작할게요~"

(1) 어떤 요소가 언어 선택에 영향을 미치나요?
(2) 코드 전환(code-switching)이란?
(3) 디지털 언어의 특성을 분석하세요.

### 힌트 1
언어 변이 요소: 청자, 상황, 매체, 목적.
같은 화자가 상황에 따라 다른 언어 코드 사용.

### 힌트 2
코드 전환: 두 가지 언어·방언·스타일을 전환.
"안녕하세요, How are you?" (한영 코드 전환).

### 정답
청자·상황·매체·목적에따른변이 / 코드전환=상황적응적언어선택 / 디지털언어=축약·이모티콘·신조어

### 해설
언어 변이와 코드 전환:
언어 변이 요소:
청자(audience design): 누구에게 말하는가.
상황(situation): 공식/비공식.
매체(channel): 구어/문어/디지털.
목적(purpose): 정보 전달/감정 표현/관계 유지.
코드 전환(code-switching):
언어·방언·스타일 간 전환.
다국어 화자: 한국어↔영어 전환.
단일어 화자: 공식체↔비공식체 전환.
기능: 내집단 표시, 거리 조절, 의미 강조.
디지털 언어 특성:
축약: "ㄱㄱ"(가자가자), "ㅋㅋ"(웃음).
이모티콘: 감정 보충.
신조어: "TMI", "레전드", "킹받는다".
비선형성: 링크, 이미지, 텍스트 혼합.
비격식성: 구어체의 문어화.
디지털 언어는 새로운 언어 변이형을 만들어가고 있습니다.

## 문제 5

다음 한국어 담화 표지(discourse marker)를
분석하세요.

"음...", "그러니까", "있잖아",
"뭐랄까", "어떻게 말하지",
"그게 아니라", "아니 그게 아니고"

(1) 각 표지의 기능을 설명하세요.
(2) 구어와 문어에서 담화 표지의 차이는?
(3) 이것이 언어 처리에서 하는 역할은?

### 힌트 1
담화 표지: 담화 구조를 조직하거나
화자의 태도를 나타내는 표현.
"음...": 발화 계획 중.
"그러니까": 설명·요약.

### 힌트 2
구어에서 자연스러운 담화 표지가
문어에서는 삭제됨.
실제 대화에서 없으면 어색함.

### 정답
발화계획·구조조직·태도표시기능 / 구어=풍부·문어=삭제 / 실시간처리지원

### 해설
담화 표지 분석:
기능별 분류:
발화 계획: "음...", "어...", "그게..."
→ 생각할 시간을 버는 채움말(filler).
구조 조직: "그러니까", "그래서", "그런데"
→ 담화 단위 연결·전환.
청자 관여: "있잖아", "알지?", "맞죠?"
→ 청자의 주의·동의 확인.
태도 표시: "솔직히", "사실은", "뭐랄까"
→ 화자의 태도·불확실성 표시.
수정: "아니 그게 아니고", "그게 아니라"
→ 이전 발화 수정.
구어 vs 문어:
구어: 담화 표지 풍부 → 실시간 처리 지원.
문어: 대부분 삭제 → 계획된 언어.
담화 표지를 모르면 자연스러운 대화 불가.
외국어 학습의 고급 단계에서 중요.

## 문제 6

다음 한국어 텍스트의 화제-평언 구조를
분석하세요.

"코끼리는 코가 길다."
"그 영화는 나는 재미없었어."
"서울은 물가가 비싸다."
"민준이는 성격이 좋다."

(1) 화제(topic)와 평언(comment)을 구분하세요.
(2) 한국어의 이중 주어 구문을 설명하세요.
(3) 영어로 번역 시 어떤 어려움이 있나요?

### 힌트 1
화제: "은/는"으로 표시되는 담화의 주제.
평언: 화제에 대한 서술.
"코끼리는"(화제) + "코가 길다"(평언).

### 힌트 2
이중 주어: "코끼리는 코가 길다"에서
"코끼리"와 "코" 둘 다 주어 역할.

### 정답
화제=은/는·평언=서술부 / 이중주어=한국어특수구문 / 영어번역어려움=소유구문변환필요

### 해설
화제·평언 구조 분석:
기본 구조:
"코끼리는(화제) 코가 길다(평언)."
"서울은(화제) 물가가 비싸다(평언)."
화제: 담화에서 이야기하는 대상(은/는).
평언: 화제에 대한 서술.
이중 주어 구문:
"코끼리는(대주어) 코가(소주어) 길다."
대주어: 담화 화제, 전체 관점.
소주어: 서술의 직접 주어.
의미: "코끼리의 특성 중 코가 길다."
영어 번역 어려움:
"코끼리는 코가 길다" → "Elephants have long trunks."
구조 자체가 다름: 한국어(주어 중심) vs 영어(소유 구문).
"그 영화는 나는 재미없었어" → "I didn't find that movie interesting."
한국어의 화제 구조가 영어 목적어로 전환.
한국어는 주제 현저(topic-prominent) 언어입니다.

## 문제 7

다음 한국어 경어법의 역사적 변화를
분석하세요.

조선시대: 복잡한 6단계 경어법.
일제강점기: 일본어 영향으로 변화.
현대: 4단계(해라/해요/합쇼/합셔) 정도로 단순화.
최근: "존댓말 피로감", "나이 대신 직책".

(1) 경어법이 단순화되는 이유는?
(2) 청년 세대의 경어법 변화를 분석하세요.
(3) 직장에서의 경어법 변화는 무엇을 의미하나요?

### 힌트 1
경어법 단순화: 사회 구조 변화(민주화, 개인주의).
청년 세대: 나이 서열보다 역할·관계 중심.

### 힌트 2
"~님" 호칭의 확산, 직책 호칭 선호.
"형/언니/오빠/누나" 외 새로운 호칭 등장.

### 정답
단순화=사회민주화·개인주의 / 청년=서열탈피·관계중심 / 직장=수평문화반영

### 해설
경어법 역사적 변화:
조선시대:
왕·귀족·중인·평민 등 엄격한 신분제 반영.
6단계 이상의 복잡한 경어 체계.
현대 변화 원인:
민주화: 신분제 붕괴, 법적 평등.
도시화: 익명 사회, 전통 관계 약화.
개인주의: 개인 정체성 강화.
미디어: 표준어 보급, 방언·경어 변화.
청년 세대 변화:
나이 서열 거부: "나이 대신 직책·역할".
새로운 호칭: "~님"의 보편화.
반말 확산: 수평적 관계 지향.
직장 경어법 변화:
직급 대신 "영어 이름" 사용(스타트업).
"~님" 호칭 도입(카카오, 네이버 등).
수직적 조직 → 수평적 조직 문화 반영.
경어법의 변화는 사회 변화의 언어적 반영입니다.

## 문제 8

다음 한국어의 의미론적 관계를
분석하세요.

동의 관계: 죽다 = 사망하다 = 돌아가다 = 숨지다
반의 관계: 삶↔죽음, 기쁨↔슬픔
상하 관계: 동물→포유류→개→진돗개
전체-부분: 몸→팔→손→손가락
다의 관계: 눈(eye/snow), 밤(night/chestnut)

(1) 유의어들의 미묘한 차이를 분석하세요.
(2) 반의어의 유형(모순·반대·방향)을 설명하세요.
(3) 의미 관계가 언어 이해에 어떻게 활용되나요?

### 힌트 1
죽다/사망하다/돌아가다/숨지다: 모두 죽음이지만
등록(register)·격식·문맥이 다릅니다.

### 힌트 2
모순 반의어: A이면 B가 아님(살다↔죽다).
반대 반의어: 연속선상(덥다↔춥다, 중간값 존재).
방향 반의어: 관계 방향(가르치다↔배우다).

### 정답
유의어=등록·격식·맥락차이 / 반의어3유형=모순·반대·방향 / 의미망=이해와생성에활용

### 해설
의미론적 관계 분석:
유의어(synonymy) 차이:
죽다: 일상적, 중립적.
사망하다: 공식적, 뉴스·서류.
돌아가다: 높임말, 완곡한 표현.
숨지다: 갑작스러운 죽음, 뉴스 보도.
반의어 유형:
①모순 반의어(complementary):
삶↔죽음 (중간값 없음, A 아니면 B).
②반대 반의어(gradable antonyms):
덥다↔춥다 (연속선상, 미지근함 존재).
크다↔작다, 빠르다↔느리다.
③방향 반의어(relational antonyms):
사다↔팔다, 가르치다↔배우다, 부모↔자녀.
관계의 방향이 반대.
의미 네트워크(semantic network):
어휘들이 의미적으로 연결된 망.
이 망을 통해 언어 이해·생성 가능.
"개"를 보면 "동물", "충성", "짖다" 등 활성화.

## 문제 9

다음 한국어 텍스트에서
전제(presupposition)를 찾고 분석하세요.

"민준이는 담배를 끊었다."
"서아가 또 지각했다."
"당신의 왕관을 잃지 마세요."
"이번에는 반드시 합격하겠다."

각 문장에 숨겨진 전제는?

### 힌트 1
전제: 발화에 암묵적으로 가정된 내용.
"민준이는 담배를 끊었다" → 민준이는 담배를 피웠다(전제).

### 힌트 2
"또 지각했다" → 이전에도 지각했다(전제).
"왕관을 잃지 마세요" → 당신에게 왕관이 있다(전제).

### 정답
담배끊음→피웠다전제 / 또지각→이전지각전제 / 왕관→왕관소유전제 / 이번에는→이전실패전제

### 해설
전제(presupposition) 분석:
"민준이는 담배를 끊었다":
전제: 민준이는 이전에 담배를 피웠다.
부정해도 전제는 유지: "민준이는 담배를 끊지 않았다"→여전히 이전에 피웠다.
"서아가 또 지각했다":
전제: 서아는 이전에도 지각한 적이 있다.
"또": 반복을 나타내는 전제 유발어(trigger).
"당신의 왕관을 잃지 마세요":
전제: 당신에게 왕관이 있다(소유·정체성의 비유).
실제: 광고·동기부여 문구, 비유적 전제.
"이번에는 반드시 합격하겠다":
전제: 이전에 합격하지 못한 적이 있다.
"이번에는": 이전 시도의 실패를 함의.
전제의 중요성:
발화의 의미를 이해하려면 전제 파악이 필수.
전제를 통한 암묵적 메시지 전달.
광고·정치 언어에서 전제 조작이 흔합니다.

## 문제 10

다음 언어 습득 이론을 한국어 학습에
적용하여 분석하세요.

행동주의(스키너):
자극→반응→강화로 언어 습득.
"엄마"라고 하면 칭찬받아 반복 학습.

생득주의(촘스키):
언어 습득 장치(LAD) 선천적.
모든 아이가 문법 오류를 비슷한 방식으로 함.

상호작용주의:
언어 입력 + 사회적 상호작용.
비고츠키의 근접 발달 영역.

(1) 각 이론의 강점과 한계를 분석하세요.
(2) 한국 아이의 경어법 습득은 어떤 이론으로 설명되나요?
(3) 최적의 한국어 교육 방법은?

### 힌트 1
경어법: 타고난 것? 가르쳐서 배우는 것?
아이들이 경어를 언제부터 어떻게 배우는가.

### 힌트 2
경어법 습득: 사회적 상호작용 필수.
어른들의 반응(강화)+ 언어 입력+사회 규칙.

### 정답
행동주의=습관형성강점·창의설명못함 / 생득주의=보편문법·경어법설명어려움 / 상호작용=경어법습득최적설명

### 해설
언어 습득 이론 비교:
행동주의(Skinner):
강점: 반복·연습의 효과, 외국어 교육 응용.
한계: 아이들의 창의적 오류("먹었다"→"먹었다") 설명 못함.
창의적 언어 사용(한 번도 들은 적 없는 문장 생성).
생득주의(Chomsky):
강점: 언어 보편성 설명, 빠른 습득 설명.
한계: 경어법 같은 사회·문화적 언어 습득 어려움.
입력이 없어도 언어를 안다는 주장의 검증 어려움.
상호작용주의(Vygotsky):
강점: 사회적 상호작용의 역할 강조.
경어법 습득: 어른과의 상호작용에서 배움.
근접 발달 영역: 혼자 못하지만 도움받아 할 수 있는 영역.
한국어 교육 최적 방법:
기초: 구조적 입력(행동주의적 반복).
중급: 의미 협상·상호작용(상호작용주의).
고급: 실제 사회적 맥락에서 사용(사회언어학).
경어법: 실제 상황 시뮬레이션 필수.
