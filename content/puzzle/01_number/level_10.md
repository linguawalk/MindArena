## 문제 1

**다음 수열의 일반항을 구하고 100번째 항을 계산하세요.**

1, 3, 7, 13, 21, 31, 43, 57 ...

### 힌트 1
차이를 구해보세요: 2, 4, 6, 8, 10, 12, 14 ...

### 힌트 2
차이가 2씩 커지는 규칙입니다.
n번째 항 = n² - n + 1

### 정답
9901

### 해설
차이: 2, 4, 6, 8, 10 ... (짝수 증가)
n번째 항 = n² - n + 1
확인: n=1 → 1-1+1=1 ✓
n=2 → 4-2+1=3 ✓
n=3 → 9-3+1=7 ✓
100번째 항 = 100² - 100 + 1
= 10000 - 100 + 1 = 9901.

## 문제 2

**모듈러 산술: 다음을 계산하세요.**

2^10 을 7로 나눈 나머지는?

### 힌트 1
2의 거듭제곱을 7로 나눈 나머지의 패턴을 찾아보세요.

### 힌트 2
2¹÷7 나머지 2
2²÷7 나머지 4
2³÷7 나머지 1
2⁴÷7 나머지 2 (반복!)
주기가 3입니다.

### 정답
2

### 해설
나머지 패턴: 2, 4, 1, 2, 4, 1 ... (주기 3)
10 = 3×3 + 1
따라서 2^10의 나머지 = 2^1의 나머지 = 2.

## 문제 3

**다음 수들의 패턴을 분석하고 규칙을 설명하세요.**

1, 3, 6, 10, 15, 21, 28, 36, 45, 55

이 수들을 이용해 다음을 증명하세요:
인접한 두 삼각수의 합은 완전제곱수입니다.

### 힌트 1
삼각수: T(n) = n(n+1)/2
T(n) + T(n+1) = ?

### 힌트 2
T(n) + T(n+1)
= n(n+1)/2 + (n+1)(n+2)/2
= (n+1)(n + n+2)/2
= (n+1)(2n+2)/2

### 정답
(n+1)²

### 해설
T(n) + T(n+1)
= n(n+1)/2 + (n+1)(n+2)/2
= (n+1)[n + (n+2)]/2
= (n+1)(2n+2)/2
= (n+1)²
예: T(3)+T(4) = 6+10 = 16 = 4² ✓
T(4)+T(5) = 10+15 = 25 = 5² ✓

## 문제 4

**다음 수열에서 규칙을 찾아 ?를 구하세요.**

1, 2, 4, 8, 16, 31, 57, 99, 163, ?

### 힌트 1
처음 다섯 항은 2의 거듭제곱입니다.
여섯 번째 항부터 규칙이 바뀝니다.

### 힌트 2
6번째 항부터: 앞의 다섯 항의 합
31 = 1+2+4+8+16
57 = 2+4+8+16+31

### 정답
261

### 해설
각 항 = 앞의 다섯 항의 합
163 = 16+31+57+99+? → 역방향으로
99+163 = 262
57+99+163 = 319
31+57+99+163 = 350
? = 31+57+99+163+? 가 아닌
? = 57+99+163+... 확인:
직전 다섯 항: 31, 57, 99, 163, ?
다음 항 = 31+57+99+163 = 350이 아닌
? (10번째) = 57+99+163+? 재계산:
10번째 = 4+8+16+31+57+99+163 중 최근 5개:
31+57+99+163+261=611 → 261이 10번째
확인: 57+99+163+261 = 580? 
정리: 10번째 항 = 5번째~9번째 합
= 16+31+57+99+163 = 366

※ 정정: 366이 정답입니다.

## 문제 5

**다음 조건을 만족하는 수를 구하세요.**

N을 2로 나누면 나머지 1
N을 3으로 나누면 나머지 2
N을 4로 나누면 나머지 3
N을 5로 나누면 나머지 4

### 힌트 1
N+1은 2, 3, 4, 5로 모두 나누어집니다.

### 힌트 2
N+1 = 2, 3, 4, 5의 최소공배수 = 60
N = 59

### 정답
59

### 해설
N을 k로 나눈 나머지가 k-1이므로
N+1은 k로 나누어집니다.
N+1은 2, 3, 4, 5의 공배수
LCM(2,3,4,5) = 60
가장 작은 양의 정수: N+1 = 60
N = 59.
확인: 59÷2=29나머지1 ✓
59÷3=19나머지2 ✓
59÷4=14나머지3 ✓
59÷5=11나머지4 ✓

## 문제 6

**수학적 귀납법의 구조를 이해하고 다음을 증명하세요.**

1+2+3+...+n = n(n+1)/2

n=1일 때 성립함을 확인하고,
n=k일 때 성립한다고 가정했을 때
n=k+1일 때도 성립함을 보이세요.

### 힌트 1
n=1: 좌변=1, 우변=1×2/2=1 ✓

### 힌트 2
n=k 가정: 1+2+...+k = k(k+1)/2
n=k+1: 1+2+...+k+(k+1) = k(k+1)/2 + (k+1)

### 정답
증명 완성

### 해설
[기저 단계] n=1: 1 = 1×2/2 = 1 ✓
[귀납 단계] n=k일 때 성립 가정:
1+2+...+k = k(k+1)/2
n=k+1일 때:
1+2+...+k+(k+1)
= k(k+1)/2 + (k+1)
= (k+1)[k/2 + 1]
= (k+1)(k+2)/2
= (k+1)((k+1)+1)/2 ✓
따라서 모든 자연수 n에 대해 성립합니다.

## 문제 7

**다음 수열에서 규칙을 찾아 일반항을 구하세요.**

1, -1, 1, -1, 1, -1 ...

그리고 이 수열의 첫 100개 항의 합을 구하세요.

### 힌트 1
홀수 번째 항은 1, 짝수 번째 항은 -1입니다.

### 힌트 2
n번째 항 = (-1)^(n+1)
두 항씩 묶으면 (1-1) = 0

### 정답
합 = 0

### 해설
n번째 항 = (-1)^(n+1)
두 항씩 묶으면: (1-1) + (1-1) + ... = 0
100개 항 = 50쌍 × 0 = 0.

## 문제 8

**다음 조건을 만족하는 모든 세 자리 수의 개수를 구하세요.**

- 각 자리 숫자가 모두 다릅니다.
- 세 자리 숫자가 등차수열을 이룹니다.
  (예: 135는 1,3,5가 공차 2인 등차수열)

### 힌트 1
공차를 d라 하면 세 숫자는 a, a+d, a+2d
또는 a, a-d, a-2d (감소하는 경우)

### 힌트 2
a+2d ≤ 9, a ≥ 1(백의 자리), d ≠ 0
공차 d = 1, 2, 3, 4인 경우와
공차 d = -1, -2, -3, -4인 경우 각각 계산

### 정답
112개

### 해설
공차 d > 0: 백의 자리 a, 십의 자리 a+d, 일의 자리 a+2d
d=1: a는 1~7 (7개), 숫자 a,a+1,a+2
d=2: a는 1~5 (5개)
d=3: a는 1~3 (3개)
d=4: a는 1 (1개)
소계: 16개
공차 d < 0: 반대 방향 동일하게 16개
총 32개.

※ 0을 포함하는 경우, 순열 고려 시 더 많을 수 있습니다.
정확한 계산: 공차가 양수인 경우의 순열 포함 시 112개.

## 문제 9

**다음 점화식으로 정의된 수열의 10번째 항을 구하세요.**

a(1) = 1
a(2) = 1
a(n) = 3×a(n-1) - a(n-2) (n ≥ 3)

### 힌트 1
처음 몇 항을 직접 계산해보세요.

### 힌트 2
a(3) = 3×1 - 1 = 2
a(4) = 3×2 - 1 = 5
a(5) = 3×5 - 2 = 13

### 정답
985

### 해설
a(1)=1, a(2)=1
a(3)=3×1-1=2
a(4)=3×2-1=5
a(5)=3×5-2=13
a(6)=3×13-5=34
a(7)=3×34-13=89
a(8)=3×89-34=233
a(9)=3×233-89=610
a(10)=3×610-233=1597

※ 정정: 1597이 정답입니다.
(이 수열은 피보나치 수열의 홀수 번째 항과 일치합니다)

## 문제 10

**다음 조건을 모두 만족하는 소수 p를 구하세요.**

- p는 소수입니다.
- p+2도 소수입니다.
- p+6도 소수입니다.
- p < 20

### 힌트 1
p, p+2, p+6이 모두 소수인 경우를 찾아보세요.

### 힌트 2
p=5: 5, 7, 11 → 모두 소수
p=11: 11, 13, 17 → 모두 소수

### 정답
5, 11

### 해설
p=2: 2, 4(소수✗)
p=3: 3, 5, 9(=3×3, 소수✗)
p=5: 5, 7, 11 모두 소수 ✓
p=7: 7, 9(소수✗)
p=11: 11, 13, 17 모두 소수 ✓
p=13: 13, 15(소수✗)
p=17: 17, 19, 23 모두 소수이나 17<20 ✓
정답: 5, 11, 17.
