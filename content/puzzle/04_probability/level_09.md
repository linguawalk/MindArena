## 문제 1

혼합 전략 내쉬 균형:
가위바위보에서 상대가 가위만 낸다면
최적 전략은 바위.
하지만 상대가 이를 알면 보를 냅니다.

순수 전략이 없을 때 혼합 전략 균형은?
각 선택을 동일한 확률(1/3)로 낼 때
상대방이 어떤 선택을 해도
기댓값이 같음을 보이세요.

### 힌트 1
상대가 가위만 낸다고 가정.
나의 가위(무승부), 바위(승), 보(패)의 기댓값.

### 힌트 2
나의 전략이 1/3씩일 때:
상대가 가위를 내든, 바위를 내든, 보를 내든
나의 기댓값이 모두 0임을 보이세요.

### 정답
1/3 혼합 전략이 내쉬 균형, 기댓값=0

### 해설
나의 전략: 가위(1/3), 바위(1/3), 보(1/3).
상대가 가위를 낼 때:
E = (1/3)×0 + (1/3)×1 + (1/3)×(-1) = 0.
상대가 바위를 낼 때:
E = (1/3)×(-1) + (1/3)×0 + (1/3)×1 = 0.
상대가 보를 낼 때:
E = (1/3)×1 + (1/3)×(-1) + (1/3)×0 = 0.
어떤 선택이든 기댓값 = 0.
1/3 혼합 전략이 유일한 내쉬 균형.
상대가 내 전략을 알아도 대응할 방법이 없습니다.

## 문제 2

베이지안 업데이트:
처음에 동전이 공정할 확률 0.5,
편향(앞면 0.7)일 확률 0.5로 추정합니다.

동전을 3번 던져 앞면 3번이 나왔을 때
편향된 동전일 사후 확률은?

### 힌트 1
P(편향|앞앞앞) = P(앞앞앞|편향)×P(편향) / P(앞앞앞).

### 힌트 2
P(앞앞앞|공정) = (0.5)³ = 0.125.
P(앞앞앞|편향) = (0.7)³ = 0.343.

### 정답
약 73.3%

### 해설
P(앞앞앞) = P(앞앞앞|공정)×P(공정) + P(앞앞앞|편향)×P(편향)
= 0.125×0.5 + 0.343×0.5
= 0.0625 + 0.1715 = 0.234.
P(편향|앞앞앞) = (0.343×0.5) / 0.234
= 0.1715 / 0.234 ≈ 0.733.
처음 50%이던 편향 확률이 증거(앞면 3번)로 73.3%로 업데이트.
베이지안 추론: 새 증거로 믿음을 지속 업데이트.
과학적 방법론의 수학적 기초.

## 문제 3

확률 그래프 이론:
에르되시-레니 랜덤 그래프 G(n,p):
n개의 노드, 각 간선이 독립적으로 확률 p로 존재.

n=5, p=0.5일 때
(1) 간선의 기댓값은?
(2) 연결 그래프일 확률은 대략 얼마인가요?
(3) 거대 연결 요소가 나타나는 임계 확률은?

### 힌트 1
가능한 간선 수: C(5,2) = 10.
각 간선 존재 확률: 0.5.

### 힌트 2
임계 확률: p = 1/n 근처에서 거대 연결 요소 등장.

### 정답
(1) 5개 (2) 약 34% (3) p = 1/n = 0.2

### 해설
(1) E(간선 수) = C(5,2)×0.5 = 10×0.5 = 5개.
(2) 연결 그래프 확률:
정확한 계산은 복잡하지만 p=0.5에서 약 34%.
p가 클수록 연결 확률 증가.
(3) 임계 현상:
p < 1/n: 작은 요소들만 존재.
p = 1/n: 거대 연결 요소 탄생.
p > log(n)/n: 거의 확실히 연결.
상전이(phase transition): 물리의 상변화와 유사.
소셜 네트워크, 인터넷 구조 분석에 적용.

## 문제 4

확률 부등식의 응용:
마르코프 부등식:
P(X ≥ a) ≤ E(X)/a (X ≥ 0)

기댓값이 100인 시험에서
150점 이상 받을 확률의 상한은?

체비쇼프 부등식으로
평균 100, 표준편차 15인 시험에서
70점 이하 또는 130점 이상일 확률의 상한은?

### 힌트 1
마르코프: P(X ≥ 150) ≤ 100/150 = 2/3.

### 힌트 2
체비쇼프: |X-100| ≥ 30 = 2σ.
P(|X-100| ≥ 30) ≤ 1/4.

### 정답
마르코프: 2/3 / 체비쇼프: 1/4

### 해설
마르코프 부등식:
P(X ≥ 150) ≤ E(X)/150 = 100/150 = 2/3.
매우 느슨한 상한이지만 분포를 몰라도 성립.
체비쇼프 부등식:
|X-100| ≥ 30, k=30/15=2.
P(|X-100| ≥ 30) ≤ 1/2² = 1/4.
실제 정규 분포에서 2σ 밖은 약 5%이므로
체비쇼프의 25%는 훨씬 느슨합니다.
확률 부등식: 분포를 몰라도 성립하는 보편적 한계.

## 문제 5

확률 암호학 입문:
일회용 암호 패드(One-Time Pad):
메시지 M, 키 K (무작위), 암호문 C = M XOR K.

M = 10110 (이진수), K = 01101 (무작위)일 때
(1) C는?
(2) K를 모르면 C에서 M을 추론할 수 있나요?
(3) 완전 비밀성이란?

### 힌트 1
XOR: 같으면 0, 다르면 1.
10110 XOR 01101 = ?

### 힌트 2
완전 비밀성: P(M|C) = P(M).
암호문이 메시지에 대한 정보를 전혀 주지 않음.

### 정답
(1) 11011 (2) 불가능 (3) 암호문이 메시지 정보 없음

### 해설
(1) C = 10110 XOR 01101:
1XOR0=1, 0XOR1=1, 1XOR1=0, 1XOR0=1, 0XOR1=1.
C = 11011.
(2) K가 완전 무작위이면:
어떤 C도 모든 M에 동일한 확률로 대응.
정보 이론적으로 해독 불가.
(3) 완전 비밀성(Shannon, 1949):
P(M|C) = P(M).
암호문을 봐도 메시지 확률이 변하지 않음.
일회용 패드만이 완전 비밀성 달성.
단점: 키 길이 = 메시지 길이, 재사용 불가.

## 문제 6

통계적 가설 검정:
동전이 공정한지 검정합니다.
귀무 가설 H₀: p = 0.5 (공정)
대립 가설 H₁: p ≠ 0.5 (편향)

동전을 100번 던져 60번 앞면이 나왔습니다.
유의 수준 5%에서 귀무 가설을 기각할 수 있나요?

(H₀ 하에서: 평균=50, 표준편차=5)

### 힌트 1
검정 통계량 Z = (X - μ) / σ = (60-50)/5.

### 힌트 2
Z = 2.0.
유의 수준 5%: |Z| > 1.96이면 기각.

### 정답
|Z|=2.0 > 1.96이므로 귀무 가설 기각 (p < 0.05)

### 해설
Z = (60-50)/5 = 2.0.
유의 수준 5%의 임계값: ±1.96.
|2.0| > 1.96 → 귀무 가설 기각.
결론: 통계적으로 유의미하게 편향된 동전.
p-value ≈ 0.046 < 0.05.
단, 이것이 "확실히 편향"을 의미하지는 않습니다.
5%의 오류 가능성(1종 오류)이 있습니다.

## 문제 7

랜덤 워크:
수직선 위에서 출발점 0에서 시작.
매 스텝: +1 (확률 0.5) 또는 -1 (확률 0.5).

(1) n스텝 후 위치의 기댓값은?
(2) n스텝 후 위치의 분산은?
(3) 원점 복귀 확률은?

### 힌트 1
각 스텝: E(스텝) = 0.5×1 + 0.5×(-1) = 0.
V(스텝) = E(X²) - [E(X)]² = 1.

### 힌트 2
n스텝: E(위치) = 0, V(위치) = n.
표준편차 = √n.

### 정답
(1) 0 (2) n (3) 1 (반드시 돌아옴, 무한 시간 후)

### 해설
(1) E(위치) = n×E(스텝) = n×0 = 0.
(2) V(위치) = n×V(스텝) = n×1 = n.
표준편차 = √n → 퍼짐이 √n에 비례.
(3) 원점 복귀 확률:
1차원 랜덤 워크는 확률 1로 원점 귀환.
"취한 사람은 반드시 집에 돌아간다."
2차원: 확률 1로 귀환.
3차원: 귀환 확률 < 1 (약 34%).
"취한 새는 집에 못 돌아갈 수 있다."

## 문제 8

확률 알고리즘:
퀵소트의 평균 시간 복잡도를 확률로 분석합니다.

n개 원소 중 피벗을 무작위로 선택할 때
피벗이 중간 50% (n/4 ~ 3n/4) 안에 있을 확률은?

이 경우 양쪽이 각각 최대 3n/4이 됩니다.

(1) 피벗이 "좋은" 위치일 확률은?
(2) 연속으로 k번 좋은 피벗을 선택할 확률은?
(3) 평균적으로 몇 번 시도해야 좋은 피벗을 얻나요?

### 힌트 1
중간 50%: n/4 ~ 3n/4 → n/2개 위치.
좋은 피벗 확률 = 1/2.

### 힌트 2
기하 분포: 좋은 피벗이 나올 때까지 평균 시도 수 = 1/p.

### 정답
(1) 1/2 (2) (1/2)^k (3) 평균 2번

### 해설
(1) 좋은 피벗: n/4~3n/4 범위 = n/2개.
확률 = (n/2)/n = 1/2.
(2) k번 연속 좋은 피벗: (1/2)^k.
(3) 기하 분포 기댓값: 1/p = 1/(1/2) = 2번.
퀵소트 평균 O(n log n):
절반씩 분할이 평균적으로 O(log n) 단계.
각 단계 O(n) 작업 → 평균 O(n log n).
랜덤화 알고리즘의 위력:
최악 O(n²)이지만 평균 O(n log n).
랜덤화로 최악 케이스를 회피.

## 문제 9

확률과 정보:
압축 이론의 기초.

공정한 동전(엔트로피 1비트) vs
편향 동전(앞면 3/4, 뒷면 1/4).

편향 동전의 엔트로피는?
100번 던진 결과를 저장할 때
공정한 동전은 100비트가 필요합니다.
편향 동전은 몇 비트가 필요한가요?

### 힌트 1
H = -[0.75×log₂(0.75) + 0.25×log₂(0.25)].
log₂(0.75) ≈ -0.415, log₂(0.25) = -2.

### 힌트 2
편향 동전의 엔트로피 ≈ 0.811비트.
100번 → 약 81.1비트.

### 정답
엔트로피 ≈ 0.811비트 / 100번 ≈ 81비트

### 해설
H = -[0.75×(-0.415) + 0.25×(-2)]
= -(-0.311 - 0.5)
= 0.811비트.
100번 시퀀스 최소 저장: 81.1비트.
허프만 코딩: 빈번한 결과(앞면)에 짧은 코드.
앞면: "0" (1비트), 뒷면: "10" (2비트).
평균: 0.75×1 + 0.25×2 = 1.25비트 (최적에 가까움).
엔트로피 = 이론적 최소 비트 수.
데이터 압축의 수학적 한계.

## 문제 10

종합 확률 퍼즐:
3개의 상자에 공이 다음과 같이 들어있습니다.

상자 1: 흰 공 2개, 검은 공 1개.
상자 2: 흰 공 1개, 검은 공 2개.
상자 3: 흰 공 1개, 검은 공 1개.

다음 실험을 합니다:
①주사위를 던져 1~2면이면 상자1, 3~4면이면 상자2, 5~6면이면 상자3을 선택.
②선택된 상자에서 공 1개를 꺼냄.
③꺼낸 공이 흰 공이었을 때 상자3에서 꺼냈을 확률은?

### 힌트 1
P(상자1)=1/3, P(상자2)=1/3, P(상자3)=1/3.
P(흰|상자1)=2/3, P(흰|상자2)=1/3, P(흰|상자3)=1/2.

### 힌트 2
P(흰) = (1/3)×(2/3) + (1/3)×(1/3) + (1/3)×(1/2).
P(상자3|흰) = P(흰|상자3)×P(상자3) / P(흰).

### 정답
약 31.6%

### 해설
P(흰) = (1/3)×(2/3) + (1/3)×(1/3) + (1/3)×(1/2)
= 2/9 + 1/9 + 1/6
= 4/18 + 2/18 + 3/18
= 9/18 = 1/2.
P(상자3|흰) = [(1/2)×(1/3)] / (1/2)
= (1/6) / (1/2)
= 1/3 ≈ 33.3%.

재계산:
P(흰|상자3)×P(상자3) = (1/2)×(1/3) = 1/6.
P(흰) = 2/9+1/9+1/6 = 4/18+2/18+3/18 = 9/18 = 1/2.
P(상자3|흰) = (1/6)/(1/2) = 2/6 = 1/3 ≈ 33.3%.
