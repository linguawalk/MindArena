## 문제 1

이항 분포 B(10, 0.3)에서
X의 기댓값과 분산을 구하세요.

### 힌트 1
이항 분포 B(n,p)의 기댓값: E(X) = np.
분산: V(X) = np(1-p).

### 힌트 2
n=10, p=0.3.
E(X) = 10×0.3.
V(X) = 10×0.3×0.7.

### 정답
기댓값=3, 분산=2.1

### 해설
E(X) = np = 10×0.3 = 3.
V(X) = np(1-p) = 10×0.3×0.7 = 2.1.
표준편차 σ = √2.1 ≈ 1.449.
해석: 10번 시행에서 평균 3번 성공,
실제 값은 3 ± 1.449 범위에 주로 분포.

## 문제 2

어느 공장의 불량률이 20%입니다.
제품 5개를 검사할 때
불량품이 2개 이하일 확률은?

### 힌트 1
X ~ B(5, 0.2).
P(X≤2) = P(X=0) + P(X=1) + P(X=2).

### 힌트 2
P(X=k) = C(5,k) × 0.2^k × 0.8^(5-k).

### 정답
약 94.2%

### 해설
P(X=0) = C(5,0)×0.2⁰×0.8⁵ = 1×1×0.32768 = 0.32768.
P(X=1) = C(5,1)×0.2¹×0.8⁴ = 5×0.2×0.4096 = 0.4096.
P(X=2) = C(5,2)×0.2²×0.8³ = 10×0.04×0.512 = 0.2048.
P(X≤2) = 0.32768+0.4096+0.2048 = 0.94208 ≈ 94.2%.

## 문제 3

정규 분포 N(μ, σ²)의 특성을 분석하세요.

어느 학교 시험 점수가
평균 70점, 표준편차 10점인
정규 분포를 따릅니다.

(1) 60~80점 사이에 있을 확률은?
(2) 90점 이상일 확률은?
(3) 상위 16%에 들려면 몇 점 이상?

(정규 분포: μ±σ 구간에 약 68%, μ±2σ에 약 95%)

### 힌트 1
60~80점 = μ±σ 구간.
90점 = μ+2σ.

### 힌트 2
P(μ-σ ≤ X ≤ μ+σ) ≈ 0.68.
P(X > μ+σ) ≈ (1-0.68)/2 = 0.16.

### 정답
(1) 약 68% (2) 약 2.5% (3) 80점 이상

### 해설
μ=70, σ=10.
(1) 60~80 = 70±10 = μ±σ → 약 68%.
(2) 90점 = μ+2σ.
P(X>90) = (1-0.95)/2 = 0.025 = 2.5%.
(3) 상위 16% = P(X>x) = 0.16.
P(X>μ+σ) ≈ 0.16이므로 x = μ+σ = 80점.

## 문제 4

포아송 분포:
어느 콜센터에 1시간에 평균 4건의 전화가 옵니다.
1시간에 정확히 3건 올 확률은?

포아송 분포: P(X=k) = (e^(-λ) × λ^k) / k!
(λ=평균, e≈2.718)

### 힌트 1
λ=4, k=3.
P(X=3) = e^(-4) × 4³ / 3!

### 힌트 2
e^(-4) ≈ 0.0183.
4³ = 64.
3! = 6.

### 정답
약 19.5%

### 해설
P(X=3) = e^(-4) × 4³ / 3!
= 0.0183 × 64 / 6
= 0.0183 × 10.667
≈ 0.195.
포아송 분포: 단위 시간·공간에서
드물게 발생하는 사건의 횟수 모델.
λ = 평균 발생 횟수.
콜센터, 방사선 붕괴, 교통사고 등에 적용.

## 문제 5

두 확률 변수 X, Y가 독립이고
E(X)=3, E(Y)=5,
V(X)=4, V(Y)=9일 때

(1) E(X+Y) = ?
(2) E(2X-3Y) = ?
(3) V(X+Y) = ?
(4) V(2X-3Y) = ?

### 힌트 1
E(aX+bY) = aE(X) + bE(Y).
독립이면 V(X+Y) = V(X) + V(Y).

### 힌트 2
V(aX) = a²V(X).
독립이면 V(aX+bY) = a²V(X) + b²V(Y).

### 정답
(1)8 (2)-9 (3)13 (4)97

### 해설
(1) E(X+Y) = E(X)+E(Y) = 3+5 = 8.
(2) E(2X-3Y) = 2E(X)-3E(Y) = 6-15 = -9.
(3) V(X+Y) = V(X)+V(Y) = 4+9 = 13 (독립).
(4) V(2X-3Y) = 4V(X)+9V(Y) = 4×4+9×9 = 16+81 = 97.
독립인 경우 분산은 항상 더해집니다 (빼기 아님).

## 문제 6

큰 수의 법칙과 중심극한정리를
설명하는 실험입니다.

동전을 n번 던질 때 앞면 비율이
0.5에 가까워지는 과정을 분석하세요.

n=10: 앞면 7번 → 비율 0.7.
n=100: 앞면 52번 → 비율 0.52.
n=1000: 앞면 503번 → 비율 0.503.

(1) 이 현상의 수학적 원리는?
(2) 도박사의 오류란?
(3) 중심극한정리란?

### 힌트 1
큰 수의 법칙: n이 커질수록 표본 평균이 모평균에 수렴.

### 힌트 2
도박사의 오류: "이번에 앞면이 자주 나왔으니 다음에는 뒷면이 나올 것"이라는 잘못된 생각.

### 정답
큰수의법칙=n증가시수렴 / 도박사오류=독립사건혼동 / 중심극한=표본평균정규분포수렴

### 해설
큰 수의 법칙:
표본 크기 n이 커질수록
표본 평균 → 모집단 평균.
비율 → 실제 확률.
도박사의 오류:
"앞면이 많이 나왔으니 다음은 뒷면"이라는 오류.
각 동전 던지기는 독립적, 이전 결과 영향 없음.
중심극한정리:
표본 크기가 충분히 크면
표본 평균의 분포가 정규 분포에 근사.
σ가 아무리 이상해도 n≥30이면 근사 가능.
통계학의 핵심 정리입니다.

## 문제 7

다음 확률 게임의 공정성을 분석하세요.

게임: 주사위 2개를 던져
합이 7이면 +5점,
합이 2 또는 12이면 +10점,
그 외에는 -2점.

이 게임은 공정한가요?
(기댓값으로 판단)

### 힌트 1
전체: 36가지.
합=7: 6가지, 합=2: 1가지, 합=12: 1가지.

### 힌트 2
E = 5×(6/36) + 10×(2/36) + (-2)×(28/36).

### 정답
기댓값 ≈ -0.389 → 불공정 (참가자에게 불리)

### 해설
P(합=7) = 6/36 = 1/6.
P(합=2 또는 12) = 2/36 = 1/18.
P(그 외) = 28/36 = 7/9.
E = 5×(6/36) + 10×(2/36) + (-2)×(28/36)
= 30/36 + 20/36 - 56/36
= -6/36 = -1/6 ≈ -0.167점.
기댓값이 음수이므로 참가자에게 불리한 게임.
공정한 게임: 기댓값 = 0.

## 문제 8

표본 추출과 통계 추정:
어느 도시의 유권자 100만 명 중
1,000명을 무작위로 조사했더니
60%가 A 후보를 지지했습니다.

(1) 표본 비율은?
(2) 모집단 비율의 추정값은?
(3) 왜 전수 조사를 하지 않나요?

### 힌트 1
표본 비율 = 조사 결과 / 표본 수.
표본 비율로 모집단 비율을 추정합니다.

### 힌트 2
전수 조사: 100만 명 모두 조사.
표본 조사: 1,000명으로 추정.

### 정답
(1) 0.6(60%) (2) 약 60% (3) 비용·시간·현실성

### 해설
표본 비율 p̂ = 600/1000 = 0.6.
모집단 비율 추정값 ≈ 60%.
전수 조사를 안 하는 이유:
비용: 100만 명 조사는 천문학적 비용.
시간: 선거 전에 완료 불가.
정확성: 오히려 표본이 더 정확할 수 있음.
중심극한정리: 충분한 표본(n≥30)이면 정규 분포 근사.
오차 범위: ±1/√n = ±1/√1000 ≈ ±3.2%.

## 문제 9

조건부 확률의 역설:
심프슨의 역설을 분석하세요.

치료법 A와 B를 비교한 데이터:

전체 결과:
A: 400명 중 300명 회복 (75%)
B: 600명 중 500명 회복 (83%)

경증 환자:
A: 100명 중 90명 회복 (90%)
B: 500명 중 450명 회복 (90%)

중증 환자:
A: 300명 중 210명 회복 (70%)
B: 100명 중 50명 회복 (50%)

어떤 치료법이 더 좋은가요?

### 힌트 1
전체: B가 좋아 보임.
경증·중증 각각: A가 같거나 좋음.

### 힌트 2
환자 구성 비율이 다르기 때문에 역설이 발생합니다.
A는 중증 환자 비율이 높고, B는 경증 환자 비율이 높습니다.

### 정답
실제로는 A가 더 좋음 (심프슨의 역설)

### 해설
심프슨의 역설: 전체를 합칠 때 방향이 역전되는 현상.
A는 중증(300명) 위주, B는 경증(500명) 위주.
중증은 회복률이 원래 낮음.
A: 중증 환자도 70% 회복 → 매우 효과적.
B: 중증은 50%만 회복.
경증·중증 각각 비교하면 A가 우수하거나 동일.
전체 통계만 보면 오해할 수 있습니다.
데이터 분석 시 그룹 구성을 반드시 확인해야 합니다.

## 문제 10

도박 게임의 파산 확률:
A가 1만원, B가 2만원을 가지고 동전 게임을 합니다.
매번 1,000원씩 걸어 앞면이면 A가 받고
뒷면이면 B가 받습니다.
A가 파산(0원)할 확률은?

(공정한 게임: p = 1/2)

### 힌트 1
도박꾼의 파산 이론(Gambler's Ruin).
공정한 게임에서 파산 확률 = 상대방 돈 / 전체 돈.

### 힌트 2
A의 파산 확률 = B의 초기 금액 / (A+B의 초기 금액)
= 2만 / (1만+2만) = 2/3.

### 정답
A의 파산 확률 = 2/3

### 해설
도박꾼의 파산 이론(공정한 게임):
P(A 파산) = B의 초기 금액 / 전체 금액
= 20,000 / 30,000 = 2/3.
P(B 파산) = A의 초기 금액 / 전체 금액
= 10,000 / 30,000 = 1/3.
같은 확률의 게임이지만
초기 자본이 적으면 파산 확률이 높습니다.
카지노가 항상 이기는 이유:
카지노의 초기 자본 >> 개인의 자본.
파산 확률이 거의 1에 수렴.
