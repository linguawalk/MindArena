## 문제 1

콜모고로프 공리 체계:
확률의 수학적 정의를 분석하세요.

공리 1: P(A) ≥ 0 (비음성)
공리 2: P(Ω) = 1 (전체 확률)
공리 3: 서로소인 사건들의 합집합:
P(A₁∪A₂∪...) = P(A₁)+P(A₂)+...

(1) 이 공리에서 P(∅)=0을 증명하세요.
(2) P(Aᶜ) = 1-P(A)를 증명하세요.
(3) P(A∪B) = P(A)+P(B)-P(A∩B)를 증명하세요.

### 힌트 1
P(∅)=0: Ω와 ∅이 서로소이고 Ω∪∅=Ω임을 활용.

### 힌트 2
P(Aᶜ): A와 Aᶜ은 서로소, A∪Aᶜ=Ω.
공리 2,3 활용.

### 정답
공리에서 유도 가능 (아래 해설 참조)

### 해설
(1) P(∅)=0:
Ω = Ω∪∅ (서로소).
공리3: P(Ω) = P(Ω)+P(∅).
1 = 1+P(∅).
∴ P(∅) = 0.
(2) P(Aᶜ)=1-P(A):
Ω = A∪Aᶜ (서로소).
P(Ω) = P(A)+P(Aᶜ).
1 = P(A)+P(Aᶜ).
∴ P(Aᶜ) = 1-P(A).
(3) P(A∪B) = P(A)+P(B)-P(A∩B):
A∪B = A∪(B\A) (서로소).
P(A∪B) = P(A)+P(B\A).
B = (A∩B)∪(B\A) (서로소).
P(B) = P(A∩B)+P(B\A).
P(B\A) = P(B)-P(A∩B).
∴ P(A∪B) = P(A)+P(B)-P(A∩B).

## 문제 2

시그마 대수(σ-algebra)와 측도론:
왜 모든 부분집합에 확률을 부여할 수 없나요?

비가측 집합의 존재(비탈리 집합):
[0,1]에서 측도 이론과 일치하는
확률을 모든 부분집합에 부여하면 모순이 발생합니다.

(1) 이 문제의 핵심을 설명하세요.
(2) 시그마 대수가 해결책인 이유는?
(3) 실용적 확률에서 이것이 문제가 되는가?

### 힌트 1
선택 공리(Axiom of Choice)를 사용하면
측도가 정의되지 않는 집합을 만들 수 있습니다.

### 힌트 2
σ-대수: 사건들의 모임으로, 모든 부분집합보다 제한적.
우리가 "관심 있는" 사건들만 포함.

### 정답
비가측집합존재→전체부분집합불가 / σ대수=가측집합한정 / 실용적으로는문제없음

### 해설
문제의 핵심:
[0,1]의 모든 부분집합에 평행이동 불변 측도 부여 불가.
비탈리(1905): 선택 공리로 비가측 집합 구성.
이 집합에는 "확률"을 의미 있게 정의할 수 없음.
σ-대수 해결:
모든 부분집합 대신 "잘 정의된" 사건의 모임.
닫힘 성질: 합집합, 여집합에 닫혀있음.
보렐 σ-대수: 열린 집합으로 생성된 σ-대수.
실용적 의미:
일상적 확률 문제에서는 전혀 문제없음.
이산 확률: 모든 부분집합이 가측.
연속 확률: 보렐 집합이면 충분.
수학적 엄밀성을 위한 이론적 기초.

## 문제 3

특성 함수(Characteristic Function):
확률 변수 X의 특성 함수:
φ(t) = E[e^(itX)] = E[cos(tX)] + i·E[sin(tX)]

(1) 공정한 동전(±1)의 특성 함수는?
(2) 특성 함수의 주요 성질은?
(3) 중심극한정리와의 관계는?

### 힌트 1
P(X=1)=P(X=-1)=1/2.
φ(t) = (1/2)e^(it) + (1/2)e^(-it) = cos(t).

### 힌트 2
합의 특성 함수 = 각 특성 함수의 곱.
n개 합의 정규화 → 정규 분포의 특성 함수 수렴.

### 정답
φ(t)=cos(t) / 합의특성함수=곱 / CLT=특성함수수렴으로증명

### 해설
(1) φ(t) = (1/2)e^(it) + (1/2)e^(-it)
= (e^(it)+e^(-it))/2 = cos(t).
(2) 특성 함수의 성질:
φ(0) = 1 (항상).
|φ(t)| ≤ 1.
독립 합: φ_{X+Y}(t) = φ_X(t)·φ_Y(t).
분포를 유일하게 결정.
(3) 중심극한정리 증명:
n개 독립 동일 분포의 합을 정규화.
φ_n(t) = [φ(t/√n)]^n → e^(-t²/2).
e^(-t²/2)는 표준 정규 분포의 특성 함수.
특성 함수는 확률론의 강력한 도구.

## 문제 4

마르팅게일 이론:
확률 과정 {Xₙ}이 마르팅게일이면:
E[Xₙ₊₁ | X₀,...,Xₙ] = Xₙ.

(1) 공정한 게임에서의 자산이 마르팅게일임을 보이세요.
(2) 마르팅게일 수렴 정리를 설명하세요.
(3) 옵션 가격 결정에서의 응용은?

### 힌트 1
공정한 게임: 기댓값=0인 베팅.
현재 자산 + 베팅 기댓값 = 현재 자산.

### 힌트 2
마르팅게일 수렴: 유계 마르팅게일은 거의 확실히 수렴.

### 정답
공정게임자산=마르팅게일 / 유계마르팅게일수렴 / 블랙숄즈=위험중립측도

### 해설
(1) 공정한 게임의 자산:
E[X_{n+1}|X₀,...,Xₙ] = Xₙ + E[베팅 결과] = Xₙ + 0 = Xₙ.
∴ 마르팅게일.
(2) 마르팅게일 수렴 정리:
E[|Xₙ|] ≤ M (유계)이면
Xₙ → X∞ (거의 확실하게 수렴).
랜덤 워크는 유계가 아니므로 수렴 안 함.
(3) 블랙-숄즈 모델:
위험 중립 측도(Q) 하에서 주가는 마르팅게일.
옵션 가격 = E_Q[할인된 미래 페이오프].
금융 파생상품 가격 결정의 수학적 기초.
노벨 경제학상(1997): 블랙-숄즈-머튼.

## 문제 5

확률 불등식의 심화:
호에프딩 부등식:
n개 독립 유계 확률 변수의 합 Sₙ에 대해:
P(Sₙ - E[Sₙ] ≥ t) ≤ exp(-2t²/Σ(bᵢ-aᵢ)²)
(aᵢ ≤ Xᵢ ≤ bᵢ)

동전 100번 던져 앞면 횟수가 65 이상일 확률의 상한은?

### 힌트 1
X = 앞면 횟수, E[X] = 50.
각 Xᵢ ∈ {0,1}, aᵢ=0, bᵢ=1.
t = 65-50 = 15.

### 힌트 2
P(X ≥ 65) ≤ exp(-2×15² / 100×1²).

### 정답
약 0.011 = 1.1%

### 해설
P(X-50 ≥ 15) ≤ exp(-2×225/100)
= exp(-4.5)
≈ 0.0111 = 1.11%.
실제 이항 분포에서: P(X≥65) ≈ 0.0018.
호에프딩은 상한이므로 실제보다 느슨합니다.
호에프딩 부등식의 활용:
기계 학습: 일반화 오차 경계.
알고리즘 분석: 확률적 알고리즘의 오차 보장.
표본 복잡도: 필요한 데이터 수 결정.

## 문제 6

확률 과정의 분류:
다음 확률 과정들을 분류하고 특성을 설명하세요.

(1) 브라운 운동(위너 과정)
(2) 포아송 과정
(3) 가우시안 과정
(4) 레비 과정

### 힌트 1
브라운 운동: 연속 경로, 정규 증분, 독립 증분.
포아송 과정: 이산 점프, 지수 대기 시간.

### 힌트 2
가우시안 과정: 임의 유한 집합이 다변량 정규.
레비 과정: 독립 정류 증분의 일반화.

### 정답
연속=브라운·가우시안 / 이산점프=포아송·레비 / 일반화=레비가포아송+브라운포함

### 해설
브라운 운동 B(t):
B(0)=0, 연속 경로.
B(t)-B(s) ~ N(0, t-s) (독립 정규 증분).
물리: 분자 운동, 금융: 주가 모델.
포아송 과정 N(t):
N(t)-N(s) ~ Poisson(λ(t-s)).
이산 점프, 지수 대기.
콜센터, 방사선 붕괴.
가우시안 과정:
임의 유한 부분집합 → 다변량 정규 분포.
가우시안 프로세스 회귀(머신러닝).
레비 과정:
독립 정류 증분의 가장 일반적 형태.
브라운 운동 + 포아송 과정의 합을 포함.
금융의 점프-확산 모델.

## 문제 7

엔트로피와 정보의 심화:
KL 발산(쿨백-라이블러 발산):
D_KL(P||Q) = Σ P(x) × log(P(x)/Q(x))

P = (1/2, 1/2), Q = (3/4, 1/4)일 때
D_KL(P||Q)와 D_KL(Q||P)를 구하고
비대칭성을 설명하세요.

### 힌트 1
D_KL(P||Q) = (1/2)×log(1/2 / 3/4) + (1/2)×log(1/2 / 1/4).
log₂ 사용.

### 힌트 2
log₂(2/3) ≈ -0.585, log₂(2) = 1.

### 정답
D_KL(P||Q) ≈ 0.208비트 / D_KL(Q||P) ≈ 0.189비트

### 해설
D_KL(P||Q):
= (1/2)×log₂(2/3) + (1/2)×log₂(2)
= (1/2)×(-0.585) + (1/2)×1
= -0.292 + 0.5 = 0.208비트.
D_KL(Q||P):
= (3/4)×log₂(3/2) + (1/4)×log₂(1/2)
= (3/4)×0.585 + (1/4)×(-1)
= 0.439 - 0.25 = 0.189비트.
비대칭성: D_KL(P||Q) ≠ D_KL(Q||P).
KL 발산은 거리가 아님 (삼각 부등식 불성립).
해석: P를 기준으로 Q를 근사할 때의 "정보 손실".
응용: 머신러닝 손실 함수, 정보 이론, 통계 추론.

## 문제 8

랜덤 행렬 이론 입문:
n×n 랜덤 행렬의 고유값 분포.

가우시안 직교 앙상블(GOE):
원소가 독립 정규 분포인 대칭 행렬.
위그너 반원 법칙: n→∞일 때
정규화된 고유값 분포 → 반원 분포.

(1) 위그너 반원 법칙의 의미는?
(2) 실생활 응용은?
(3) 왜 "반원"인가?

### 힌트 1
고유값을 √n으로 나누면 [-2,2] 위의 반원에 수렴.
밀도: ρ(x) = (1/2π)√(4-x²).

### 힌트 2
응용: 핵물리학(에너지 준위), 통신 이론, 머신러닝.

### 정답
랜덤행렬고유값=반원분포 / 핵물리·통신·ML응용 / 수학적보편성

### 해설
위그너 반원 법칙:
랜덤 대칭 행렬의 고유값을 정규화하면
[-2,2] 구간의 반원 분포로 수렴.
ρ(x) = (1/2π)√(4-x²).
왜 반원?
중심극한정리의 행렬 버전.
독립성과 대칭성에서 수학적으로 도출.
실생활 응용:
핵물리: 원자핵의 에너지 준위 통계.
무선 통신: MIMO 안테나 시스템.
머신러닝: 신경망 가중치 초기화.
금융: 상관 행렬 분석(소음 vs 신호).
랜덤 행렬 이론은 현대 수학·물리·공학의 교차점.

## 문제 9

확률과 계산 복잡도:
BPP(Bounded-error Probabilistic Polynomial time):
확률 알고리즘으로 다항 시간에 풀 수 있는 문제.

소수 판별(밀러-라빈 알고리즘):
n이 소수일 확률적 판별.
k번 반복 시 오류 확률 ≤ (1/4)^k.

(1) k=10이면 오류 확률은?
(2) BPP와 P의 관계는?
(3) 무작위성이 계산에 주는 이점은?

### 힌트 1
오류 확률 ≤ (1/4)^10 = 1/4^10.

### 힌트 2
BPP ⊆ P²(PSPACE). 대부분 학자는 BPP = P로 추측.

### 정답
(1) 약 10^(-6) (2) P⊆BPP, BPP=P추측 (3) 속도·간결성

### 해설
(1) (1/4)^10 = 1/1,048,576 ≈ 10^(-6).
10번 반복으로 오류 확률이 100만분의 1.
(2) BPP와 P:
P ⊆ BPP (결정론적 알고리즘은 확률적 알고리즘).
대부분 학자: BPP = P (무작위성이 진정한 도움 안 됨).
AKS 알고리즘(2002): 소수 판별이 실제로 P임을 증명.
(3) 무작위성의 이점:
알고리즘 단순화.
평균적 성능 개선.
최악 케이스 회피.
암호학: 진정한 무작위성이 보안의 핵심.

## 문제 10

확률론의 미해결 문제들:
다음 미해결/최근 해결된 문제들을 탐구하세요.

(1) 콜모고로프 복잡도와 랜덤성:
가장 무작위한 수열이란?

(2) 브라운 운동의 프랙탈 차원:
2차원에서 브라운 운동의 경로 차원 = 2.
3차원에서 경로 차원 = ?

(3) 랜덤 그래프의 임계 현상:
G(n,p)에서 해밀턴 순환이 나타나는 임계 p는?

### 힌트 1
콜모고로프 복잡도: 수열을 생성하는 가장 짧은 프로그램.
무작위 = 압축 불가능.

### 힌트 2
브라운 운동 3차원 경로: 자기 교차 없음, 차원 < 2.

### 정답
(1) 압축불가능한수열 (2) 3차원에서도4/3 (3) p=log(n)/n

### 해설
(1) 콜모고로프 랜덤성:
C(x) = x를 생성하는 최단 프로그램 길이.
무작위 수열: C(x) ≈ |x| (압축 불가).
역설: 가장 무작위한 수열은 정의 불가.
(최단 프로그램의 존재는 결정 불가능)
(2) 브라운 운동 경로 차원:
2차원: 경로가 평면을 채움 → 하우스도르프 차원 = 2.
3차원: 자기 교차 없음 → 하우스도르프 차원 = 2.
(3, 4차원도 동일)
로른(1982): 3차원 이상에서도 브라운 경로 차원 = 2.
(3) 랜덤 그래프 해밀턴 순환:
p = log(n)/n 근처에서 해밀턴 순환 임계.
포즈나(1983): 정확한 임계 결정.
