# 메커니즘 퍼즐 2단계 — 시스템과 순환

> 개별 원인-결과를 넘어
> 시스템 전체의 작동 방식을 탐구합니다.
> 순환과 균형이 어떻게 세상을 유지하는지
> 이해합니다.

## 문제 1

다음 물 순환의 메커니즘을 완성하세요.

```
바다 → ___ → 구름 → ___ → 강 → 바다
```

각 단계의 메커니즘을 설명하세요.

### 힌트 1
바다에서 물이 어떻게 공기 중으로 올라가나요?

### 힌트 2
구름에서 물이 어떻게 다시 내려오나요?

### 정답
바다→증발→구름→강수(비/눈)→강→바다

### 해설
물 순환 메커니즘:
①증발: 태양 에너지 → 바다 물 → 수증기.
②상승: 수증기 → 상공으로 이동.
③응결: 차가운 상공에서 수증기 → 물방울(구름).
④강수: 구름 물방울 모임 → 비/눈.
⑤유출: 강 → 바다로 귀환.
에너지원: 태양 (증발), 중력 (강수, 유출).
순환의 중요성:
담수 공급, 기온 조절, 영양분 이동.
지구 물 총량: 변하지 않음 (보존).

## 문제 2

다음 먹이 사슬의 메커니즘을 분석하세요.

```
풀 → 토끼 → 여우 → 독수리
```

여우 개체수가 갑자기 급증하면
각 생물의 개체수는 어떻게 변할까요?

### 힌트 1
여우가 많아지면 토끼는?

### 힌트 2
토끼가 줄면 풀은? 여우는 결국?

### 정답
토끼감소→풀증가·여우도결국감소(먹이부족)→균형회복

### 해설
먹이 사슬 피드백:
여우 증가 → 토끼 포식 증가 → 토끼 감소.
토끼 감소 → 풀 감소 완화 → 풀 회복.
토끼 감소 → 여우 먹이 부족 → 여우 감소.
여우 감소 → 토끼 회복.
로트카-볼테라 동학:
포식자-피식자 진동.
인구 수가 주기적으로 오르내림.
음성 피드백 = 안정적 순환.
생태계 균형: 단일 종 급증은 자기 교정됨.

## 문제 3

다음 경제 순환의 메커니즘을 설명하세요.

가계 → 기업 → 가계의 순환에서:
가계: 노동 공급, 소득 수령.
기업: 노동 구매, 상품 판매.

이 순환이 멈추면 어떻게 될까요?

### 힌트 1
가계가 소비를 줄이면 기업 수입은?

### 힌트 2
기업 수입 감소 → 고용은? → 가계 소득은?

### 정답
소비감소→기업수입감소→해고→소득감소→소비추가감소 (불황의 악순환)

### 해설
경제 순환 메커니즘:
정상 순환:
가계 소득 → 소비 → 기업 수입 → 임금 → 가계 소득.
불황 악순환 (양성 피드백):
소비 감소 → 기업 매출 감소 → 해고.
해고 → 소득 감소 → 추가 소비 감소.
→ 불황 심화 (절약의 역설).
정부 개입:
재정 정책: 정부 지출로 순환 자극.
통화 정책: 금리 인하로 소비·투자 촉진.
케인즈: "불황 시 정부가 지출해야 한다."

## 문제 4

다음 도시 열섬 효과의 메커니즘을 분석하세요.

도시가 주변 농촌보다 2~5°C 더 따뜻합니다.
왜 이런 현상이 생기나요?

### 힌트 1
도시와 농촌의 표면 차이를 생각해보세요.
(콘크리트, 아스팔트 vs 흙, 나무)

### 힌트 2
인간 활동에서 발생하는 열은?

### 정답
콘크리트·아스팔트=높은열흡수+인공열+증발냉각없음 → 도시 온도 상승

### 해설
도시 열섬 메커니즘:
①표면 재질:
콘크리트·아스팔트: 비열 낮음, 열 흡수 많음.
나무·풀: 증산으로 냉각 효과.
②인공열:
자동차, 에어컨, 산업 활동.
③증발 냉각 부재:
도시: 포장 → 빗물 흡수 없음.
농촌: 토양·식물이 물 흡수 → 증발 냉각.
④건물 협곡 효과:
건물 사이에 열 갇힘.
해결책:
녹지 확대, 옥상 정원, 밝은 색 포장재.
응용: 도시 계획에서 기후 고려 필수.

## 문제 5

다음 항생제 내성의 메커니즘을 설명하세요.

항생제를 많이 쓸수록
약이 듣지 않는 세균이 증가합니다.
왜 이런 일이 일어나나요?

### 힌트 1
세균 집단에는 다양한 개체가 있습니다.
일부는 우연히 항생제에 내성을 가집니다.

### 힌트 2
항생제 사용 시 내성 없는 세균은 죽고
내성 있는 세균만 살아남으면?

### 정답
항생제→내성없는세균사멸→내성세균만생존·번식→내성세균집단화

### 해설
자연 선택 메커니즘:
①세균 집단: 유전적 다양성 존재.
②항생제 사용: 선택압 발생.
③내성 없는 세균: 사멸.
④내성 있는 세균: 생존 + 번식.
⑤다음 세대: 내성 세균이 대부분.
⑥항생제 효과 감소.
다윈의 자연 선택:
변이 + 선택 + 유전 = 진화.
항생제 내성은 진화의 실시간 관찰.
대응 전략:
항생제 사용 최소화 (선택압 감소).
새 항생제 개발 (다른 선택압).
병용 요법 (여러 내성 동시 획득 어려움).

## 문제 6

다음 복잡한 피드백 시스템을 분석하세요.

북극 빙하 감소의 피드백:

빙하 감소 → ___ → ___ → 빙하 추가 감소.

이 메커니즘의 이름과 위험성은?

### 힌트 1
흰 빙하와 어두운 바다의 햇빛 반사율(알베도)을 비교하세요.

### 힌트 2
빙하가 줄면 → 더 많은 어두운 바다 → 햇빛 흡수 증가 → 온도 상승.

### 정답
빙하감소→어두운바다노출→햇빛흡수증가→추가온난화→추가빙하감소 (양성피드백=알베도피드백)

### 해설
알베도 피드백 메커니즘:
알베도: 빛 반사율.
흰 빙하: 알베도 80-90% (햇빛 대부분 반사).
어두운 바다: 알베도 6% (햇빛 대부분 흡수).
양성 피드백 (악순환):
빙하 감소 → 어두운 바다 증가.
햇빛 흡수 증가 → 온도 상승.
온도 상승 → 빙하 추가 감소.
→ 자기 강화 사이클.
위험성:
임계점(tipping point): 되돌릴 수 없는 지점.
기후 붕괴 가속화.
다른 기후 피드백:
영구동토층 해빙 → 메탄 방출 → 온난화.
식물 증가 → CO₂ 흡수 (음성 피드백).

## 문제 7

다음 인터넷 바이럴의 메커니즘을 수학적으로 분석하세요.

콘텐츠가 100명에게 공유됩니다.
각 사람이 10% 확률로 다시 공유.
1세대 공유자 = 10명.

이것이 계속되면 어떻게 될까요?

### 힌트 1
매 세대 공유자 수: 100 → 10 → 1 → 0.1...
이것은 감소하는 수열입니다.

### 힌트 2
반대로 각 사람이 10명에게 전파, 전파율 100%이면?

### 정답
전파율<1이면소멸 / 전파율>1이면지수적증가(바이럴) / 임계점=전파율1

### 해설
바이럴 메커니즘의 수학:
기본 재생산 수 R₀:
R₀ < 1: 소멸 (각 공유가 1명 미만 도달).
R₀ = 1: 유지 (안정).
R₀ > 1: 지수적 성장 (바이럴).
예시:
전파율 10%: R₀=0.1 → 소멸.
전파율 200%: R₀=2 → 지수 성장.
바이럴 조건: R₀ > 1.
감염병 R₀:
코로나 초기: R₀≈2.5-3.
독감: R₀≈1.2-1.4.
홍역: R₀≈12-18.
R₀를 1 이하로 만들기 = 방역의 목표.

## 문제 8

다음 토양 형성 메커니즘을 분석하세요.

비옥한 토양이 만들어지는 데 수백~수천 년이 걸립니다.
어떤 과정으로 토양이 형성되나요?

### 힌트 1
암석이 작은 조각으로 부서지는 과정은?

### 힌트 2
생물체가 토양 형성에 어떻게 기여하나요?

### 정답
물리적풍화+화학적분해+생물활동+유기물축적=토양형성

### 해설
토양 형성 메커니즘 (토양 생성작용):
①물리적 풍화:
온도 변화, 물 동결-해빙, 바람.
암석 → 작은 입자.
②화학적 분해:
빗물 + CO₂ = 약산 (탄산).
광물 용해, 산화.
③생물 활동:
이끼·지의류: 바위에 달라붙어 분해.
지렁이: 토양 혼합, 영양분 순환.
균류·세균: 유기물 분해.
④유기물 축적:
죽은 생물 분해 → 부식질 형성.
시간:
1cm 토양: 100-1000년.
파괴: 집중 강우 → 1년에 수cm 유실.

## 문제 9

다음 주식 시장 거품의 메커니즘을 분석하세요.

주가 상승 → 투자자 유입 → 주가 추가 상승 → 더 많은 투자자 유입.

이 메커니즘의 구조와 결과는?

### 힌트 1
이것은 양성 피드백 메커니즘입니다.
하지만 주가는 무한히 오를 수 없습니다.

### 힌트 2
자산 가격이 실제 가치를 크게 초과하면 어떻게 될까요?

### 정답
양성피드백=거품형성 → 임계점에서 신뢰상실 → 급격한하락(거품붕괴)

### 해설
자산 거품 메커니즘:
형성 단계:
초기 상승 → 수익 기대 → 투자 유입.
투자 유입 → 가격 상승 → 추가 유입.
모멘텀 투자: 오르기 때문에 삼.
거품 단계:
가격 > 내재 가치 (펀더멘털 괴리).
"이번엔 다르다"는 믿음.
붕괴 단계:
신뢰 상실 (어떤 사건 계기).
매도 → 가격 하락 → 패닉 매도.
역방향 양성 피드백.
역사적 사례:
1637년 튤립 버블, 1929년 대공황.
2000년 닷컴 버블, 2008년 주택 거품.
교훈: 양성 피드백은 결국 반전됨.

## 문제 10

다음 언어 보존/소멸 메커니즘을 분석하세요.

세계에서 매 2주마다 한 언어가 사라집니다.
현재 7000개 → 100년 후 절반 이상 소멸 예측.

언어 소멸의 메커니즘은?

### 힌트 1
사람들이 언어를 선택할 때 무엇을 고려하나요?
(취업, 교육, 의사소통)

### 힌트 2
소수 언어 사용자가 줄어들면 언어의 가치는?

### 정답
경제적이점→지배언어선택→소수언어전승단절→사용자감소→언어소멸 (양성피드백)

### 해설
언어 소멸 메커니즘:
네트워크 효과:
더 많은 사람이 쓸수록 언어의 가치 증가.
소수 언어: 네트워크 효과 약함.
경제적 선택:
지배 언어 = 취업, 교육, 사회 진출.
부모: 자녀에게 지배 언어 교육.
세대 전승 단절:
아이들이 소수 언어 안 배움.
마지막 사용자 사망 → 언어 소멸.
양성 피드백:
사용자 감소 → 언어 가치 감소.
→ 추가 이탈 → 추가 감소.
보존 메커니즘:
공식 지위 부여 (웨일스어, 히브리어 부활).
디지털 기록, 교육 정책.
공동체 자부심과 정체성.
