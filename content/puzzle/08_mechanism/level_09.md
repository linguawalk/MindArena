## 문제 1

다음 비만의 다층 메커니즘을 분석하세요.

비만은 단순히 "너무 많이 먹고
너무 적게 운동하는" 문제가 아닙니다.
생물학·심리학·사회·경제 메커니즘을
통합 분석하세요.

### 힌트 1
생물: 호르몬, 장 미생물, 유전자.
심리: 스트레스 식이, 감정 식이.

### 힌트 2
사회: 식품 환경, 식품 사막.
경제: 건강식 비용, 가공식품 마케팅.

### 정답
생물(호르몬·유전)+심리(스트레스식이)+사회(식품환경)+경제(가격·마케팅)=복합원인

### 해설
비만의 다층 메커니즘:
생물학적:
렙틴 저항성: 포만감 신호 감소.
인슐린 저항성: 지방 저장 증가.
장 미생물: 비만형 미생물 군집.
유전자: 30-70% 유전성.
심리적:
스트레스 → 코르티솔 → 복부 지방.
감정 식이: 부정 감정 → 고칼로리 음식.
수면 부족 → 그렐린↑ → 식욕 증가.
사회적:
식품 사막: 건강식 접근 어려운 지역.
사회적 규범: 음식 문화.
경제적:
초가공식품: 싸고, 맛있고, 접근 쉬움.
마케팅: 과식 유도.
건강식: 비쌈.
통합 해결:
개인 의지력만으로 부족.
환경 설계 변화 + 경제 인센티브.
의료 지원 + 사회 규범 변화.

## 문제 2

다음 전쟁 발발의 메커니즘을 다층 분석하세요.

1차 세계대전 발발 (1914):
사라예보 암살 → 세계 대전.
이 작은 사건이 대전쟁으로 이어진
메커니즘의 다층 분석은?

### 힌트 1
구조적 원인 (화약통): 동맹 체계, 군비 경쟁.
촉발 원인 (불꽃): 암살.

### 힌트 2
연쇄 반응: 동맹 의무 → 차례로 참전.
각국의 인식과 오판도 메커니즘.

### 정답
구조=동맹망+군비경쟁+민족주의+제국주의 / 촉발=암살 / 연쇄=의무이행→확전

### 해설
1차 세계대전 다층 메커니즘:
구조적 조건 (화약통):
①동맹 체계: 삼국 협상 vs 삼국 동맹.
②군비 경쟁: 특히 독일-영국 해군.
③민족주의: 범슬라브주의 vs 범게르만주의.
④제국주의: 식민지 경쟁.
⑤경직된 군사 계획: 슐리펜 계획.
촉발 사건:
프란츠 페르디난트 암살.
오스트리아-헝가리의 세르비아 최후통첩.
연쇄 반응:
오스트리아 → 세르비아 선전포고.
러시아 동원 (범슬라브 의무).
독일 선전포고 (오스트리아 동맹).
프랑스 개입 (러시아 동맹).
독일의 벨기에 침공 → 영국 개입.
인지 실패:
각국이 단기전 예상.
상대방 의지 오판.
교훈: 구조적 취약성이 있으면 작은 사건도 대재앙.

## 문제 3

다음 인터넷 경제의 네트워크 효과 메커니즘을 분석하세요.

구글, 페이스북, 아마존이 독점적 지위를
유지하는 메커니즘은?

### 힌트 1
직접 네트워크 효과: 사용자↑ → 가치↑.
간접 네트워크 효과: 한쪽 증가 → 다른 쪽 가치↑.

### 힌트 2
전환 비용: 다른 플랫폼으로 이동하기 어려운 이유는?

### 정답
네트워크효과(직접+간접)+데이터축적+전환비용+보완재생태계=독점자기강화

### 해설
플랫폼 독점 메커니즘:
직접 네트워크 효과:
페이스북: 사용자 多 → 더 많은 친구 연결 가능.
더 가치 있음 → 더 많은 가입자.
간접 네트워크 효과:
구글: 사용자 多 → 광고주 More.
광고주 많음 → 더 좋은 서비스 → 사용자 More.
데이터 축적:
사용자 데이터 → 더 좋은 알고리즘.
더 좋은 서비스 → 더 많은 사용자.
→ 더 많은 데이터. (데이터 해자)
전환 비용:
페이스북: 친구 네트워크 이전 어려움.
아마존: 구매 기록, 리뷰, 프라임.
구글: 습관, 이메일, 문서.
보완재 생태계:
앱 스토어, 개발자 생태계.
탈출 어렵게 만드는 구조.
규제 딜레마:
독점 규제 vs 서비스 품질.
데이터 이동성, 상호운용성 요구.

## 문제 4

다음 도시 젠트리피케이션의 메커니즘을 다층 분석하세요.

낙후 지역이 개발되면서 원주민이 쫓겨나는
메커니즘과 대안을 분석하세요.

### 힌트 1
예술가 → 카페 → 부동산 → 부유층 입주.
각 단계의 메커니즘은?

### 힌트 2
원주민 퇴출 메커니즘: 임대료 상승, 재개발.
저항 메커니즘: 협동조합, 임대료 규제.

### 정답
저렴지역→예술가유입→매력도↑→자본유입→임대료↑→원주민퇴출=젠트리피케이션

### 해설
젠트리피케이션 메커니즘:
1단계: 선구자 (개척자):
예술가, 창의 노동자 → 저렴한 공간 선호.
독특한 문화·분위기 형성.
2단계: 초기 젠트리피케이션:
카페, 소규모 상점.
외부인 관심.
임대료 소폭 상승.
3단계: 자본 유입:
부동산 투자자 유입.
대형 체인 진입.
임대료 급등.
4단계: 원주민 퇴출:
원주민: 임대료 감당 불가.
예술가도 퇴출 (자신이 만든 분위기에서).
5단계: 동질화:
독특성 상실.
고급화된 획일적 상권.
저항 메커니즘:
커뮤니티 토지 신탁.
임대료 안정화 정책.
지역 자본 우선 정책.
젠트리피케이션의 역설: 개선이 원주민을 쫓아냄.

## 문제 5

다음 기술 혁신의 파괴적 메커니즘을 분석하세요.

스트리밍이 음반 산업을,
우버가 택시 산업을 어떻게 파괴했나요?
공통 메커니즘은?

### 힌트 1
하위 시장 진입 → 기존 업체 무시.
기술 개선 → 주류 시장 장악.

### 힌트 2
기존 업체의 딜레마: 새 기술 채택 시
기존 사업 잠식.

### 정답
하위시장진입+기존업체의혁신자딜레마+기술개선→주류장악=파괴적혁신공통패턴

### 해설
파괴적 혁신 공통 메커니즘:
크리스텐슨 모델:
1단계: 하위 시장 진입.
스트리밍: MP3 (낮은 음질, 편리함).
우버: 비공식 카풀 (저렴, 편리함).
2단계: 기존 업체 무시.
음반사: MP3 = "진짜 음악 아님."
택시: 우버 = "불법."
3단계: 지속적 개선.
스트리밍: 음질 개선, 콘텐츠 확대.
우버: 신뢰성, 안전, 다양한 서비스.
4단계: 주류 시장 장악.
혁신자 딜레마:
기존 업체: 새 기술 채택 = 기존 수익 잠식.
합리적으로 무시 → 결국 파괴됨.
공통 패턴:
기술 + 비즈니스 모델 혁신.
규제 환경 변화.
소비자 행동 변화.

## 문제 6

다음 언어의 의미 변화 메커니즘을 분석하세요.

단어의 의미가 시간이 지나면서
어떻게 변하나요?

예시:
"나쁘다" (bad) → 슬랭에서 "좋다".
"인터넷" → "접속"에서 "생활 전체"로.

### 힌트 1
의미 변화 유형: 확장, 축소, 향상, 저하, 이동.

### 힌트 2
사용자 집단 → 사회 전체로 의미 확산.
은유적 사용 → 새 의미 고착.

### 정답
사용자집단변형+은유확산+사회승인=의미변화 / 권력집단언어가규범화

### 해설
의미 변화 메커니즘:
유형:
①확장: "마우스" → 컴퓨터 장치.
②축소: "meat" (모든 음식) → 고기만.
③향상: "knight" (하인) → 기사.
④저하: "villain" (농부) → 악당.
⑤이동: "awful" (경외스러운) → 끔찍한.
메커니즘:
①혁신 용어: 새 사물/개념 → 기존 단어 재사용.
②은유: 추상 개념에 구체 단어 적용.
③사용자 집단: 특정 집단의 언어 → 사회 확산.
④권력 관계: 지배 집단 언어가 규범.
디지털 시대 가속:
밈: 단어·표현의 빠른 의미 변화.
인터넷 신조어: 빠른 창조와 소멸.
글로벌화: 의미 변화의 국제적 동조.
언어 = 살아있는 시스템.
의미 변화 = 사회 변화의 반영.

## 문제 7

다음 팬데믹 후 원격 근무의
다층 메커니즘을 분석하세요.

원격 근무가 도시, 주거, 직장 문화에
미치는 연쇄 효과를 분석하세요.

### 힌트 1
도시: 도심 공동화, 지방 이주.
주거: 더 넓은 공간 수요.

### 힌트 2
직장 문화: 성과 중심 vs 시간 중심.
젠더: 돌봄 노동의 재분배.

### 정답
원격근무→도시공동화+지방활성화+공간수요변화+성과문화전환+돌봄재분배=다층파급

### 해설
원격 근무 연쇄 효과:
도시 공간:
도심 사무실 수요 감소.
상업 부동산 위기.
도심 소비 감소.
인구 이동:
비싼 도시 → 저렴한 지방.
"디지털 노마드" 증가.
지방 도시 활성화.
주거 수요:
좁은 도심 → 넓은 교외.
작업실 공간 수요.
스마트 홈 기술 수요.
직장 문화:
시간 기반 → 성과 기반.
비동기 소통 증가.
자율성 vs 고립.
젠더 효과:
돌봄 노동 유연성.
하지만: 여성에게 돌봄 부담 증가 가능.
경계 모호화 (일-생활).
새 균형:
하이브리드 근무: 유연성 + 연결.
도심 재활용: 주거, 복합 용도.

## 문제 8

다음 심리적 안전감의 조직 메커니즘을 분석하세요.

구글 아리스토텔레스 프로젝트:
최고 팀의 1번 특성 = 심리적 안전감.

심리적 안전감이 팀 성과에 미치는
메커니즘은?

### 힌트 1
심리적 안전감: 위험 감수, 실수 인정, 아이디어 제시가
처벌받지 않을 것이라는 믿음.

### 힌트 2
심리적 안전감 높음 → 학습, 혁신, 협력.
낮음 → 침묵, 감추기, 방어.

### 정답
심리안전→실수공유→학습↑→위험감수→혁신↑→협력↑→팀성과↑=선순환

### 해설
심리적 안전감 메커니즘:
안전감 → 행동 변화:
①실수 공개: 처벌 없음 → 실수 공유.
②학습: 공유된 실수 = 집단 학습.
③위험 감수: 안전 → 새 아이디어 제시.
④혁신: 아이디어 → 실험 → 혁신.
⑤협력: 취약성 공유 → 신뢰 → 협력.
낮은 안전감 → 성과 저하:
①침묵: 중요 정보 숨김.
②감추기: 실수 은폐 → 개선 없음.
③방어: 자기 보호 → 협력 저하.
④집단 사고: 이의 제기 없음.
에드몬슨 연구:
안전감 높은 팀 = 더 많은 실수 보고.
역설: 보고가 많다 = 더 좋은 팀 (숨기지 않음).
구축 메커니즘:
리더의 취약성 공개.
실수 = 학습 기회 프레이밍.
이의 제기 = 의무 규범.
경청과 감사 표현.

## 문제 9

다음 기후 변화 해결의 다층 메커니즘을 분석하세요.

기후 변화 해결을 위한
기술·경제·사회·정치 메커니즘의
통합 분석은?

### 힌트 1
기술: 재생에너지, 탄소 포집.
경제: 탄소세, 배출권 거래.

### 힌트 2
사회: 소비 행동 변화.
정치: 국제 협약, 국내 규제.

### 정답
기술개발+경제인센티브+사회규범변화+정치제도=다층해결필요 / 단층해결불충분

### 해설
기후 변화 해결의 다층 메커니즘:
기술 층위:
재생 에너지: 비용 지속 하락.
탄소 포집: 대기 중 CO₂ 제거.
에너지 저장: 배터리, 수소.
스마트 그리드: 효율적 분배.
경제 층위:
탄소세: 탄소 배출 비용화.
배출권 거래: 감축 인센티브.
녹색 금융: 저탄소 투자 촉진.
보조금: 재생에너지 경쟁력.
사회 층위:
소비 행동: 식단, 이동, 소비 변화.
사회 규범: "기후 행동 = 의무."
교육: 기후 리터러시.
정치 층위:
파리 협약: 국제 목표 설정.
탄소 국경세: 무역 연계.
규제: 배출 기준.
상호 강화:
기술 개선 → 비용 하락 → 경제 인센티브 변화.
사회 규범 → 정치 압력 → 정책 변화.
정책 → 기술 투자 → 혁신.
단일 층위 해결 불충분.

## 문제 10

발전 구간 종합:
다음 현상의 메커니즘을 가장 깊이
분석해보세요.

"왜 좋은 의도가 나쁜 결과를 낳는가?"

역사적 사례와 메커니즘을 분석하고
일반 원리를 도출하세요.

### 힌트 1
코브라 효과, 의도하지 않은 결과.
인센티브의 왜곡 효과.

### 힌트 2
2차 효과, 3차 효과.
시스템 사고의 부재.

### 정답
단선적사고+시스템무시+인센티브왜곡+2차효과무시=좋은의도나쁜결과

### 해설
좋은 의도의 나쁜 결과 메커니즘:
코브라 효과 (인도, 영국 식민지):
문제: 코브라 많음.
해결책: 코브라 잡으면 포상.
결과: 사람들이 코브라 사육 → 개체수 증가.
좌초: 포상 폐지 → 코브라 방류.
메커니즘:
인센티브 왜곡: 보상이 다른 행동 유발.
시스템 반응 무시.
2차 효과 무시.
다른 사례:
댐 건설: 홍수 방지 → 생태 파괴, 이주민.
도로 확장: 교통 완화 → 더 많은 차 → 혼잡.
원조: 지역 경제 지원 → 현지 산업 파괴.
항생제: 감염 치료 → 내성 증가.
일반 원리:
①시스템은 개입에 반응한다 (적응).
②인센티브는 예상치 못한 행동 유발.
③2차·3차 효과를 고려해야.
④단순 해결책은 복잡한 문제에 부적합.
해결:
시스템 사고: 전체 연결 파악.
파일럿 테스트: 소규모 실험.
피드백 루프: 지속적 모니터링.
