# 메커니즘 퍼즐 3단계 — 복합 메커니즘

> 입문 구간의 마지막 단계입니다.
> 단순한 원인-결과를 넘어
> 여러 메커니즘이 복합적으로 작용하는
> 현상을 탐구합니다.

## 문제 1

다음 현상의 복합 메커니즘을 분석하세요.

비가 많이 온 뒤 산사태가 발생합니다.
단순히 비가 원인일까요?

### 힌트 1
비 외에 어떤 조건이 함께 있어야 산사태가 발생하나요?
(경사, 토양, 식생, 인간 활동)

### 힌트 2
각 요소가 어떻게 상호작용하나요?

### 정답
강우+급경사+불안정토양+식생훼손+지진등 복합 조건의 임계 초과

### 해설
산사태 복합 메커니즘:
①강우: 토양 포화 → 무게 증가 + 점착력 감소.
②경사: 중력 성분 증가.
③토양 특성: 점토 → 물 먹으면 미끄러움.
④식생 훼손:
뿌리 = 토양 결합력. 벌목 → 결합력 감소.
⑤인간 활동: 사면 절개, 지하수 변화.
임계점 개념:
각 요소 혼자로는 산사태 안 일어남.
여러 요소가 임계값을 넘으면 발생.
복합 위험 = 단순 합 이상.
예방:
경사면 식생 복원, 배수로 설치, 사면 보강.

## 문제 2

다음 수면 부족의 연쇄 효과를 분석하세요.

수면 부족 → 여러 신체·정신 증상 → 더 심한 수면 부족.

이 악순환의 전체 메커니즘은?

### 힌트 1
수면 부족이 스트레스 호르몬에 미치는 영향은?

### 힌트 2
스트레스 호르몬 증가 → 수면 질은?

### 정답
수면부족→코르티솔↑→불안·스트레스→수면어려움→수면부족악순환

### 해설
수면 부족 악순환:
①수면 부족 → 코르티솔(스트레스 호르몬) 증가.
②코르티솔 → 각성 증가, 불안.
③인지 기능 저하 → 업무 실수 → 스트레스.
④스트레스 → 수면 어려움.
⑤악순환 지속.
추가 메커니즘:
카페인 의존: 수면 부족 → 카페인 섭취 → 수면 방해.
면역 저하: 수면 부족 → 면역 세포 감소.
식욕 조절 장애: 그렐린↑ (배고픔 호르몬) → 과식.
탈출 메커니즘:
수면 위생 개선, 스트레스 관리, 점진적 회복.

## 문제 3

다음 교통 혼잡의 메커니즘을 분석하세요.

고속도로에서 사고 없이도 교통 정체가 발생합니다.
"유령 정체(phantom traffic jam)"의 메커니즘은?

### 힌트 1
한 차가 갑자기 브레이크를 밟으면
뒤차들은 어떻게 반응하나요?

### 힌트 2
이 반응이 파동처럼 뒤로 전파되면?

### 정답
한차 급제동→뒤차 과도제동→감속파 역방향전파→정체→사고없는교통체증

### 해설
유령 정체 메커니즘:
①한 차가 약간 제동.
②뒤차: 반응 지연 + 과도 제동 (안전 마진).
③과도 제동이 더 강한 제동 유발.
④감속 파동이 후방으로 전파.
⑤파동이 충분히 강해지면 완전 정지.
수학: 비선형 파동 방정식.
관찰: 원인 지점은 이미 해소됐는데 정체 지속.
해결:
차간 거리 유지 (파동 흡수).
일정 속도 유지.
자율주행차: 정밀 제어로 파동 억제.
도로 속도 가변 제한: 교통 흐름 최적화.

## 문제 4

다음 기술 확산의 S커브 메커니즘을 분석하세요.

새 기술 채택 곡선:
처음에는 느리게 → 갑자기 폭발적 성장 → 다시 느려짐.

왜 S자 모양이 되나요?

### 힌트 1
초기: 혁신자와 얼리어답터만 채택.
중기: 다수가 채택 (네트워크 효과).

### 힌트 2
후기: 잠재 채택자가 줄어들면?

### 정답
초기=소수혁신자(느림)→임계질량=네트워크효과폭발→포화(느림)=S커브

### 해설
S커브 메커니즘:
1단계 (초기, 느림):
혁신자·얼리어답터만.
위험 감수, 높은 비용.
소수 → 네트워크 효과 작음.
2단계 (성장, 폭발):
임계 질량 도달.
네트워크 효과 활성화.
입소문, 사회 규범 변화.
양성 피드백: 사용자↑ → 가치↑ → 사용자 추가.
3단계 (성숙, 느림):
포화 (대부분 이미 채택).
새 채택자 감소.
음성 피드백 (시장 한계).
예시: 스마트폰, 인터넷, 전기차.
로저스 혁신 확산 이론.

## 문제 5

다음 금융 위기의 연쇄 메커니즘을 분석하세요.

2008년 금융 위기의 핵심 메커니즘:

서브프라임 모기지 → ___ → ___ → 글로벌 금융 위기.

빈칸을 채우고 각 단계를 설명하세요.

### 힌트 1
서브프라임 모기지가 복잡한 금융 상품으로 묶여
전 세계에 팔렸습니다.

### 힌트 2
주택 가격이 하락하면 이 금융 상품들의 가치는?

### 정답
서브프라임→MBS·CDO묶어판매→주택가격하락→금융상품가치폭락→은행파산위기→신용동결→실물경제침체

### 해설
2008 금융 위기 메커니즘:
①서브프라임 대출: 신용 낮은 사람에게 주택 대출.
②증권화: 대출을 MBS, CDO로 묶어 전 세계 판매.
③위험 분산 착각: 복잡성 → 위험 평가 어려움.
④주택 거품 붕괴: 주택 가격 하락.
⑤MBS·CDO 가치 폭락.
⑥은행 자본 손실 → 파산 위기 (리먼 브라더스).
⑦신용 시장 동결: 은행들 서로 대출 거부.
⑧실물 경제 타격: 기업 자금난 → 해고.
교훈:
금융 혁신 = 위험 제거 아님 (이전일 뿐).
복잡성 = 위험 불투명성.
시스템적 위험: 연결이 위기 전파.

## 문제 6

다음 언어 습득의 메커니즘을 분석하세요.

아이들이 문법 규칙을 배우지 않아도
자연스럽게 언어를 습득합니다.
어떤 메커니즘이 작동하나요?

### 힌트 1
아이들이 언어를 배우는 환경을 생각해보세요.
(반복, 피드백, 사회적 상호작용)

### 힌트 2
촘스키의 언어 습득 장치(LAD) 이론을 생각해보세요.

### 정답
생득적언어능력+풍부한입력+사회적피드백+패턴인식=자연언어습득

### 해설
언어 습득 복합 메커니즘:
①생득적 언어 능력 (촘스키):
인간 뇌: 언어 구조 인식 능력 내장.
보편 문법: 모든 언어의 공통 구조.
②입력 처리:
대량 언어 입력 → 패턴 인식.
통계적 학습: 자주 나오는 조합 습득.
③사회적 피드백:
이해 성공/실패로 언어 조정.
부모의 확장 반응 (child-directed speech).
④임계기 (결정적 시기):
생후 ~7세: 뇌의 언어 회로 최적 형성.
이후: 습득 어려워짐.
응용:
외국어 교육: 몰입식 환경.
이중 언어 교육: 어릴수록 효과적.

## 문제 7

다음 창의성의 메커니즘을 분석하세요.

창의적 아이디어는 어떻게 생성되나요?
"유레카!" 순간의 메커니즘은?

### 힌트 1
의식적 노력 → 막힘 → 무의식 처리 → 통찰.
이 4단계를 생각해보세요.

### 힌트 2
서로 다른 분야의 지식이 연결될 때
창의적 아이디어가 나옵니다.

### 정답
준비→부화(무의식처리)→통찰(갑작스런연결)→검증 / 다양한경험이원료

### 해설
창의성 메커니즘 (왈라스 모델):
①준비 (preparation):
문제 집중적 탐구.
의식적 노력, 막힘 경험.
②부화 (incubation):
의식적 탐구 중단.
무의식 처리 계속.
다른 경험이 새 연결고리 제공.
③통찰 (illumination):
갑작스런 아이디어 출현.
서로 다른 개념의 새로운 연결.
④검증 (verification):
아이디어 논리적 검토, 실행.
뇌 메커니즘:
기본 상태 네트워크 (DMN): 휴식 시 활성 → 창의성.
서로 다른 뇌 영역 연결.
원격 연합: 멀리 떨어진 개념 연결.
다양한 경험 = 창의성의 원료.

## 문제 8

다음 사회 규범의 형성 메커니즘을 분석하세요.

"악수"라는 인사 방식은 어떻게 보편적 규범이 됐을까요?

### 힌트 1
초기에 왜 악수가 시작됐나요?
(무기 없음 증명, 신뢰 신호)

### 힌트 2
일부 사람이 악수를 하면 다른 사람들은
왜 따라하게 될까요?

### 정답
초기신뢰신호→사회적조정게임→임계질량→규범화→이탈비용증가→고착

### 해설
사회 규범 형성 메커니즘:
①기원: 무기를 안 가진 증거 (빈 손).
또는 계약 확인 (악수 = 봉인).
②조정 게임:
상대가 악수하면 나도 악수 = 최선.
상대가 절하면 나도 절 = 최선.
어떤 규범이든 공유되면 안정.
③임계 질량:
충분한 사람이 채택 → 따르는 것이 이익.
④고착 (lock-in):
이탈 비용: 악수 안 하면 무례.
사회적 제재.
⑤세계화:
강한 문화(서양 비즈니스)의 확산.
비즈니스 규범 → 글로벌 표준.
교훈: 규범은 역사적 우연 + 조정 게임 + 네트워크 효과.

## 문제 9

다음 과학 혁명의 메커니즘을 분석하세요.

코페르니쿠스, 다윈, 아인슈타인의 혁명.
새 이론이 기존 이론을 대체하는 메커니즘은?

### 힌트 1
쿤(Kuhn)의 패러다임 전환 이론.
정상 과학 → 위기 → 혁명 → 새 정상 과학.

### 힌트 2
기존 패러다임이 설명 못하는 "변칙"이 쌓일 때.

### 정답
정상과학→변칙누적→위기→패러다임전환→새정상과학 (쿤의과학혁명구조)

### 해설
과학 혁명 메커니즘 (쿤):
①정상 과학:
지배적 패러다임 (세계관 + 방법론).
퍼즐 풀기: 패러다임 내에서 문제 해결.
②변칙 (anomaly):
패러다임으로 설명 안 되는 현상.
처음엔 무시 또는 임시방편.
③위기:
변칙 누적 → 패러다임 신뢰 감소.
다양한 대안 제시.
④혁명:
새 패러다임 등장.
이전 패러다임과 통약 불가 (incommensurable).
⑤새 정상 과학:
새 패러다임으로 정착.
예시:
천동설 → 지동설 (코페르니쿠스).
특수 창조 → 진화론 (다윈).
뉴턴 역학 → 상대성/양자 (아인슈타인/플랑크).

## 문제 10

입문 구간 종합: 다음 복잡한 메커니즘을 분석하세요.

인터넷이 민주주의에 미치는 영향.
긍정적 메커니즘과 부정적 메커니즘을 모두 분석하세요.

### 힌트 1
긍정: 정보 접근성, 시민 참여, 권력 감시.
부정: 필터 버블, 가짜 뉴스, 극단화.

### 힌트 2
각 메커니즘이 어떻게 작동하는지,
그리고 서로 어떻게 상호작용하는지 분석하세요.

### 정답
긍정=정보민주화·참여증가 / 부정=필터버블·극단화 / 알고리즘이핵심메커니즘

### 해설
인터넷-민주주의 복합 메커니즘:
긍정 메커니즘:
①정보 접근: 권력 감시, 부패 폭로.
②참여 증가: 청원, 시민 저널리즘.
③조직화: 소수 목소리 결집.
④투명성: 정부 정보 공개.
부정 메커니즘:
①필터 버블:
알고리즘이 선호 정보만 제공.
다른 관점 차단 → 확증 편향.
②가짜 뉴스:
제작·전파 비용 낮음.
감정 유발 → 빠른 확산.
③극단화:
같은 생각끼리만 소통 → 극단화.
④해외 개입: 가짜 뉴스로 민주주의 교란.
핵심: 알고리즘이 정보 환경 결정.
해결: 알고리즘 투명성, 미디어 리터러시.
