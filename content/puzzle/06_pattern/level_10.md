## 문제 1

다음 거듭제곱 잉여(power residue) 패턴을 분석하세요.

```
1² mod 13 = 1
2² mod 13 = 4
3² mod 13 = 9
4² mod 13 = 3
5² mod 13 = 12
6² mod 13 = 10
7² mod 13 = 10
8² mod 13 = 12
9² mod 13 = 3
10² mod 13 = 9
11² mod 13 = 4
12² mod 13 = 1
```

(1) 대칭 패턴을 설명하세요.
(2) 이차잉여(quadratic residue)란?
(3) 13 mod 4와의 관계는?

### 힌트 1
n² ≡ (13-n)² (mod 13).
(13-n)² = 169-26n+n² ≡ n² (mod 13).

### 힌트 2
이차잉여: 제곱수가 될 수 있는 mod p 값.
p=13(소수): 이차잉여는 6개, 비이차잉여 6개.

### 정답
n과(p-n)이같은잉여 / 이차잉여=제곱가능값 / 13≡1(mod4)이면-1이이차잉여

### 해설
대칭 패턴: n²≡(p-n)² (mod p).
6²=10, 7²=10 (6+7=13).
이차잉여 mod 13: {1,3,4,9,10,12}.
비이차잉여: {2,5,6,7,8,11}.
오일러 기준:
a는 p의 이차잉여 ↔ a^((p-1)/2)≡1 (mod p).
르장드르 기호 (a/p):
+1: 이차잉여, -1: 비이차잉여, 0: p|a.
이차 상호 법칙(가우스): 수론의 보석.

## 문제 2

다음 분할 함수와 모듈러 패턴을 분석하세요.

라마누잔의 합동 관계:
```
p(5n+4) ≡ 0 (mod 5)
p(7n+5) ≡ 0 (mod 7)
p(11n+6) ≡ 0 (mod 11)
```

검증:
p(4)=5 ≡ 0 (mod 5) ✓
p(9)=30 ≡ 0 (mod 5) ✓
p(14)=135 ≡ 0 (mod 5) ✓

(1) 이 패턴의 놀라운 점은?
(2) 라마누잔이 어떻게 발견했는가?
(3) 현대 수학에서의 증명 방법은?

### 힌트 1
5, 7, 11이 모두 소수이고
지수가 각각 4, 5, 6 = (p-1) 형태.

### 힌트 2
라마누잔: 직관과 계산으로 발견.
증명: 모듈러 형식(modular forms) 이론.

### 정답
소수주기와분할수의비자명관계 / 직관적발견 / 모듈러형식이론

### 해설
라마누잔의 합동:
p(5n+4) ≡ 0 (mod 5): 모든 n에 대해.
이 패턴은 전혀 명백하지 않음.
라마누잔(1919): 직관적으로 발견, 부분 증명.
왓슨(1938): 5와 7에 대해 증명.
아틀린(1967): 11에 대해 증명.
현대 증명: 모듈러 형식.
에타 함수: η(τ) = q^(1/24)Π(1-qⁿ).
분할 함수 = 에타 함수의 역수.
모듈러 대칭이 합동 패턴 설명.
수학적 경이: 분할 수와 소수 사이의 비자명 연결.

## 문제 3

다음 지수 생성 함수(EGF) 패턴을 분석하세요.

```
EGF: G(x) = Σ aₙ xⁿ/n!
```

벨 수(Bell numbers):
B₀=1, B₁=1, B₂=2, B₃=5, B₄=15, B₅=52.

EGF: B(x) = e^(eˣ-1).

(1) 벨 수가 세는 것은?
(2) 벨 삼각형을 그리세요.
(3) 점화식은?

### 힌트 1
벨 수: n개 원소 집합의 분할 수.
B₃=5: {1}{2}{3}, {12}{3}, {13}{2}, {1}{23}, {123}.

### 힌트 2
벨 삼각형:
1
1 2
2 3 5
5 7 10 15

### 정답
집합분할수 / 벨삼각형점화식 / Bₙ=Σᵢ₌₀ⁿ C(n,i)Bᵢ

### 해설
벨 수 B(n): n원소 집합의 분할 방법 수.
B₀=1, B₁=1, B₂=2, B₃=5, B₄=15, B₅=52.
벨 삼각형:
각 행의 첫 수 = 이전 행 마지막 수.
각 수 = 왼쪽 + 위의 수.
1
1  2
2  3  5
5  7  10 15
15 20 27 37 52
점화식: B(n+1) = Σᵢ₌₀ⁿ C(n,i)B(i).
EGF: e^(eˣ-1)의 계수로 복구 가능.

## 문제 4

다음 수론적 함수들의 곱셈성을 분석하세요.

완전수(perfect number):
σ(n) = n의 모든 약수의 합.
완전수: σ(n) = 2n.

```
σ(6) = 1+2+3+6 = 12 = 2×6 ✓
σ(28) = 1+2+4+7+14+28 = 56 = 2×28 ✓
σ(496) = ? (검증)
```

(1) σ는 곱셈적 함수인가?
(2) 짝수 완전수의 패턴은?
(3) 홀수 완전수는 존재하는가?

### 힌트 1
σ(mn)=σ(m)σ(n) (gcd(m,n)=1): 곱셈적.
짝수 완전수: 2^(p-1)(2^p-1), 2^p-1이 소수.

### 힌트 2
메르센 소수: 2^p-1 형태의 소수.
p=2: 3(소수) → 6(완전).
p=3: 7(소수) → 28(완전).
p=5: 31(소수) → 496(완전).

### 정답
곱셈적함수 / 2^(p-1)(2^p-1) / 홀수완전수=미발견·존재여부미결

### 해설
σ(6)=12, σ(28)=56, σ(496)=992=2×496 ✓.
짝수 완전수: 유클리드-오일러 정리.
2^(p-1)(2^p-1)이 완전수 ↔ 2^p-1이 메르센 소수.
알려진 메르센 소수: p=2,3,5,7,13,17,19...
무한히 많은지 미해결.
홀수 완전수:
알려진 것 없음.
만약 존재하면: >10^1500.
2000년 이상 미해결 문제.

## 문제 5

다음 유클리드 알고리즘의 패턴을 분석하세요.

gcd(a,b) = gcd(b, a mod b).

```
gcd(1071, 462):
1071 = 2×462 + 147
462 = 3×147 + 21
147 = 7×21 + 0
gcd = 21
```

(1) 알고리즘의 단계 수는 최대 몇 번인가?
(2) 최악의 경우는 어떤 입력인가?
(3) 확장 유클리드 알고리즘이란?

### 힌트 1
최악: 피보나치 수 쌍 (F(n), F(n-1)).
단계 수 ≤ log_φ(max(a,b)) × 2.

### 힌트 2
확장 유클리드: gcd(a,b) = ax+by 표현.
디오판토스 방정식의 해.

### 정답
O(logn)단계 / 피보나치쌍이최악 / 확장=베주항등식

### 해설
유클리드 알고리즘:
단계 수: O(log(min(a,b))).
피보나치가 최악:
gcd(F(n), F(n-1))은 n-2단계 필요.
1071=F(16)?... 정확히는 피보나치에 가까운 수.
확장 유클리드 알고리즘:
gcd(a,b)=d이면 ax+by=d인 정수 x,y 존재.
베주 항등식(Bézout's identity).
응용: 모듈러 역원 계산.
RSA에서 d=e⁻¹ (mod φ(n)) 계산.

## 문제 6

다음 수 체계의 패턴을 분석하세요.

p진수(p-adic numbers):
일반 수의 절댓값: |n|∞ = 일반적인 크기.
p진 절댓값: |n|p = p⁻ᵛₚ(n).
vp(n) = n에서 p가 인수로 등장하는 횟수.

```
|12|₃ = 3⁻ᵛ₃(12) = 3⁻¹ (12=4×3¹)
|18|₃ = 3⁻ᵛ₃(18) = 3⁻² (18=2×3²)
|27|₃ = 3⁻ᵛ₃(27) = 3⁻³ (27=3³)
```

(1) |6|₂, |6|₃을 구하세요.
(2) p진 절댓값의 역설적 성질은?
(3) p진수가 수학에서 중요한 이유는?

### 힌트 1
6=2×3: |6|₂=2⁻¹=1/2, |6|₃=3⁻¹=1/3.

### 힌트 2
p진 절댓값: p의 고차 거듭제곱 = "작은" 수.
27은 3진법에서 "0에 가까움".

### 정답
|6|₂=1/2·|6|₃=1/3 / 고차거듭제곱=작은수 / 페르마마지막정리증명도구

### 해설
p진 절댓값:
|n|p = p⁻ᵛₚ(n).
6=2¹×3¹: |6|₂=2⁻¹=0.5, |6|₃=3⁻¹≈0.333.
역설:
2진법에서: 2,4,8,16,...이 0에 수렴!
|2ⁿ|₂=2⁻ⁿ→0.
완전히 다른 거리 개념.
중요성:
오스트로프스키 정리: ℚ의 절댓값 = 일반 또는 p진.
와일스의 페르마 마지막 정리 증명: p진수 핵심.
수론의 현대 도구.

## 문제 7

다음 격자 이론의 패턴을 분석하세요.

정수 격자 ℤⁿ에서의 패턴:
2D 격자 ℤ²: 각 점 (m,n).

가우스 정수: a+bi (a,b∈ℤ).
노름: |a+bi|² = a²+b².

페르마의 두 제곱수 정리:
소수 p = a²+b²의 합 ↔ p=2 또는 p≡1 (mod 4).

(1) 어떤 소수가 두 제곱수의 합인가요?
(2) 가우스 정수에서 소수의 패턴은?
(3) 라이프니츠 공식과의 연결은?

### 힌트 1
p=2: 1²+1².
p≡1(mod4): 5=1²+2², 13=2²+3², 17=1²+4².
p≡3(mod4): 3,7,11,19... 두 제곱수의 합 불가.

### 힌트 2
π/4 = 1-1/3+1/5-1/7+... (라이프니츠).
이것이 두 제곱수 합과 관련!

### 정답
p≡1mod4또는p=2 / 가우스소수=p≡3mod4 / 라이프니츠π=4Σ(-1)^n/(2n+1)

### 해설
두 제곱수의 합:
p=2=1²+1².
p≡1(mod4): 5,13,17,29,37,41,...
p≡3(mod4): 절대 불가 (3,7,11,19,23,...).
가우스 정수에서:
p≡1(mod4): (a+bi)(a-bi)로 인수분해.
p≡3(mod4): 가우스 정수에서도 소수.
라이프니츠 공식 연결:
π = 4(1-1/3+1/5-1/7+...).
= 4Σ_{n=0}^∞ (-1)ⁿ/(2n+1).
이것이 격자 점 세기와 관련됩니다.

## 문제 8

다음 멱급수 패턴을 분석하세요.

베르누이 수(Bernoulli numbers):
B₀=1, B₁=-1/2, B₂=1/6, B₃=0, B₄=-1/30...

생성 함수: t/(eᵗ-1) = Σ Bₙtⁿ/n!

(1) Bₙ=0 (n홀수, n>1)인 이유는?
(2) 합의 공식과의 관계:
    1ᵏ+2ᵏ+...+nᵏ = ?
(3) 리만 제타 함수와의 관계는?

### 힌트 1
t/(eᵗ-1) + t/2 = (t/2)coth(t/2): 짝함수.
홀수 차항 = 0 (B₁ 제외).

### 힌트 2
Σᵢ₌₁ⁿ iᵏ = (베르누이 다항식)/(k+1).

### 정답
생성함수짝함수성질 / faulhaber공식 / ζ(2k)=베르누이수포함

### 해설
베르누이 수:
B₀=1, B₁=-1/2, B₂=1/6, B₄=-1/30, B₆=1/42...
홀수(n>1): Bₙ=0 (생성 함수의 짝함수성).
파울하버 공식:
1ᵏ+2ᵏ+...+nᵏ = (1/(k+1))Σⱼ₌₀ᵏ C(k+1,j)Bⱼnᵏ⁺¹⁻ʲ.
가우스: Σᵢ₌₁ⁿ i = n(n+1)/2 = (1/2)B₀n²+(1/2)B₁n...
리만 제타 함수:
ζ(2k) = (-1)ᵏ⁺¹ (2π)²ᵏ B₂ₖ / (2×(2k)!).
ζ(2)=π²/6, ζ(4)=π⁴/90.
오일러가 발견한 아름다운 공식.

## 문제 9

다음 타일링 수의 패턴을 분석하세요.

1×n 보드를 1×1과 1×2 타일로 채우는 방법 수:
```
n=1: 1
n=2: 2 (1+1, 2)
n=3: 3 (1+1+1, 1+2, 2+1)
n=4: 5
n=5: 8
```

(1) 이 수열은?
(2) 다른 타일 조합으로 어떻게 변하나?
(3) 행렬로 표현하면?

### 힌트 1
피보나치 수열!
T(n) = T(n-1) + T(n-2).

### 힌트 2
1×1과 1×3 타일: T(n)=T(n-1)+T(n-3).
행렬: [T(n+1);T(n)] = [1 1;1 0]ⁿ [1;1].

### 정답
피보나치수열 / 점화식변화 / 행렬거듭제곱

### 해설
T(n) = F(n+1) (피보나치).
마지막 타일이 1×1: T(n-1)가지.
마지막 타일이 1×2: T(n-2)가지.
T(n) = T(n-1)+T(n-2).
1×3 타일 추가:
T(n) = T(n-1)+T(n-2)+T(n-3) (트리보나치).
1,1,2,4,7,13,24...
행렬 표현:
[T(n+1)]   [1 1 0]ⁿ  [T(1)]
[T(n)  ] = [1 0 0]  ×[T(0)]
[T(n-1)]   [0 1 0]   [T(-1)]

## 문제 10

심화 구간 입문: 다음 패턴들을 통합하는
메타 패턴을 찾으세요.

①피보나치의 황금비 수렴.
②파스칼 삼각형의 피보나치 대각선.
③타일링의 피보나치.
④식물의 피보나치 나선.
⑤황금 직사각형의 자기 유사성.

(1) 이 모든 것을 연결하는 핵심 원리는?
(2) φ = 1 + 1/φ의 의미는?
(3) 자기 유사성이 패턴에서 하는 역할은?

### 힌트 1
자기 유사성: 전체 = 부분의 반복.
피보나치: f(n) = f(n-1)+f(n-2) → 자기 참조.

### 힌트 2
φ = 1+1/φ: 황금비는 자기 자신으로 정의.
이것이 모든 피보나치 현상의 핵심.

### 정답
자기유사성·자기참조 / φ=극한비율=자연의최적분할 / 패턴=자기유사구조

### 해설
통합 원리: 자기 유사성(self-similarity).
피보나치: 앞 두 항의 합 = 자기 참조 성장.
황금비: φ=1+1/φ = 무한 연분수 [1;1,1,1,...].
자기 유사 직사각형: φ×1 → 1×(φ-1)=1×1/φ.
자연: 식물의 성장이 자기 유사적.
파이토탁시스: 잎/씨앗의 나선 = 피보나치 수.
진화적 이유: 새 요소를 최대한 공간 효율적으로.
황금각 = 360°/φ² ≈ 137.5°.
패턴과 자기 유사성:
가장 깊은 패턴 = 자기 자신을 닮은 구조.
프랙탈, 재귀, 피보나치가 모두 이 원리.
