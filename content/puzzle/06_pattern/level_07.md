## 문제 1

다음 패턴을 수학적으로 일반화하세요.

```
1 = 1
1+3 = 4
1+3+5 = 9
1+3+5+7 = 16
1+3+5+7+9 = 25
```

(1) 규칙을 공식으로 표현하세요.
(2) 처음 n개 홀수의 합은?
(3) 이것을 기하학적으로 설명하세요.

### 힌트 1
1, 4, 9, 16, 25 = 1², 2², 3², 4², 5².

### 힌트 2
처음 n개 홀수의 합 = n².
기하: n×n 정사각형.

### 정답
처음n개홀수의합=n² / 기하=n×n정사각형

### 해설
1+3+5+...+(2n-1) = n².
기하학적 증명:
n=1: * (1칸).
n=2: 1칸 + L자 3칸 = 4칸.
n=3: 4칸 + L자 5칸 = 9칸.
각 단계에서 L자 모양(홀수 개)을 추가하면
정사각형이 됩니다.
이것이 "홀수의 합 = 제곱수"의 시각적 증명.

## 문제 2

다음 수열의 일반항을 구하고
100번째 항을 계산하세요.

```
수열: 3, 7, 11, 15, 19, 23...
```

그리고 이 수열에서 100보다 작은
항의 개수는?

### 힌트 1
등차수열: 초항=3, 공차=4.
aₙ = 3 + (n-1)×4 = 4n-1.

### 힌트 2
100번째: 4×100-1 = 399.
100보다 작으려면: 4n-1 < 100 → n < 25.25.

### 정답
a₁₀₀=399 / 100미만=25개

### 해설
aₙ = 3+(n-1)×4 = 4n-1.
a₁₀₀ = 4×100-1 = 399.
4n-1 < 100:
4n < 101.
n < 25.25.
n = 1, 2, ..., 25 → 25개.
a₂₅ = 4×25-1 = 99 < 100 ✓.
a₂₆ = 4×26-1 = 103 > 100 ✓.

## 문제 3

다음 모듈러 패턴을 분석하세요.

```
1² mod 4 = 1
2² mod 4 = 0
3² mod 4 = 1
4² mod 4 = 0
5² mod 4 = 1
```

(1) 패턴을 설명하세요.
(2) 홀수²과 짝수²의 mod 4 값은?
(3) 이것이 의미하는 수학적 사실은?

### 힌트 1
홀수²: 항상 mod 4 = 1.
짝수²: 항상 mod 4 = 0.

### 힌트 2
홀수 = 2k+1이면 (2k+1)² = 4k²+4k+1 = 4(k²+k)+1.
mod 4 = 1.

### 정답
홀수²mod4=1·짝수²mod4=0 / 정수는4로나눠1또는0만

### 해설
홀수 n=2k+1:
n² = (2k+1)² = 4k²+4k+1 = 4(k(k+1))+1.
n² mod 4 = 1.
짝수 n=2k:
n² = 4k².
n² mod 4 = 0.
수학적 함의:
모든 완전제곱수는 4로 나누면 나머지가 0 또는 1.
나머지가 2 또는 3이면 완전제곱수가 아님.
빠른 완전제곱수 판별법.

## 문제 4

다음 패턴에서 n번째 행의 합을 구하고
일반 공식을 유도하세요.

```
행 1: 1              합=1
행 2: 3  5           합=8
행 3: 7  9  11       합=27
행 4: 13 15 17 19    합=64
```

### 힌트 1
합: 1, 8, 27, 64 = 1³, 2³, 3³, 4³.

### 힌트 2
n행 합 = n³.
n행의 첫 항 = n²-n+1.

### 정답
n행합=n³ / 세제곱수

### 해설
n행 첫 항: n²-n+1.
n행 마지막 항: n²+n-1.
n행 합 = n/2 × (첫 항+마지막 항)
= n/2 × (n²-n+1+n²+n-1)
= n/2 × 2n²
= n³.
아름다운 결과: 연속 홀수를 특정 방식으로 배열하면
각 행의 합 = 세제곱수!
1³+2³+3³+...+n³ = [n(n+1)/2]² = (삼각수)².

## 문제 5

다음 이진 패턴을 분석하세요.

```
n=1: 1
n=2: 10
n=3: 11
n=4: 100
n=5: 101
n=6: 110
n=7: 111
n=8: 1000
```

(1) 2ⁿ이 2진법에서 어떻게 표현되는가?
(2) 2ⁿ-1이 2진법에서 어떻게 표현되는가?
(3) n번째 2진수의 자릿수는?

### 힌트 1
2ⁿ = 1 뒤에 0이 n개.
2ⁿ-1 = 1이 n개.

### 힌트 2
n번째 수의 자릿수 = ⌊log₂(n)⌋+1.

### 정답
2ⁿ=1+0×n / 2ⁿ-1=1×n / 자릿수=⌊log₂n⌋+1

### 해설
2ⁿ의 2진법: 1 다음 0이 n개.
2¹=10, 2²=100, 2³=1000.
2ⁿ-1의 2진법: 1이 n개.
2¹-1=1, 2²-1=11, 2³-1=111.
자릿수:
1~1: 1자리. 2~3: 2자리. 4~7: 3자리. 8~15: 4자리.
n의 자릿수 = ⌊log₂(n)⌋+1.
컴퓨터에서 n비트로 표현 가능한 수: 0~2ⁿ-1.

## 문제 6

다음 수열의 패턴을 찾아
닫힌 형식(closed form)을 구하세요.

```
S(n) = 1 + 2 + 3 + ... + n = ?
S(n) = 1² + 2² + 3² + ... + n² = ?
S(n) = 1³ + 2³ + 3³ + ... + n³ = ?
```

### 힌트 1
첫 번째: 가우스 공식 n(n+1)/2.
두 번째: n(n+1)(2n+1)/6.

### 힌트 2
세 번째: [n(n+1)/2]².
= (삼각수)².

### 정답
n(n+1)/2 / n(n+1)(2n+1)/6 / [n(n+1)/2]²

### 해설
Σk = n(n+1)/2 (삼각수, 가우스).
Σk² = n(n+1)(2n+1)/6 (피라미드수).
Σk³ = [n(n+1)/2]² (삼각수의 제곱).
아름다운 관계: (Σk)² = Σk³.
n=3: (1+2+3)² = 36 = 1³+2³+3³ = 1+8+27 = 36 ✓.

## 문제 7

다음 순열 패턴을 분석하세요.

n개 원소의 순열 수:
n=1: 1
n=2: 2
n=3: 6
n=4: 24
n=5: 120

(1) 패턴을 공식으로 표현하세요.
(2) n!의 마지막 자리 숫자는 n≥5이면?
(3) n!에 포함된 0의 개수 (마지막) 공식은?

### 힌트 1
n! = n×(n-1)×...×2×1.
n≥5이면 5×2=10이 인수로 포함 → 마지막 자리=0.

### 힌트 2
마지막 0의 개수 = 5의 인수 개수.
⌊n/5⌋+⌊n/25⌋+⌊n/125⌋+...

### 정답
n! / n≥5이면마지막자리=0 / ⌊n/5⌋+⌊n/25⌋+...

### 해설
n! = 1×2×3×...×n.
n≥5: 2와 5가 인수 → 10이 인수 → 마지막 자리 0.
5! = 120, 10! = 3628800.
마지막 0의 개수:
10!: ⌊10/5⌋ = 2개의 0.
25!: ⌊25/5⌋+⌊25/25⌋ = 5+1 = 6개.
100!: ⌊100/5⌋+⌊100/25⌋+⌊100/125⌋ = 20+4+0 = 24개.

## 문제 8

다음 재귀 패턴을 분석하고
하노이 탑과의 관계를 설명하세요.

T(1) = 1.
T(n) = 2T(n-1) + 1.

(1) T(2), T(3), T(4), T(5)를 구하세요.
(2) 일반항 T(n)은?
(3) 64개 원판 하노이 탑은 몇 번 이동?

### 힌트 1
T(1)=1, T(2)=3, T(3)=7, T(4)=15, T(5)=31.
T(n) = 2ⁿ-1.

### 힌트 2
64개: T(64) = 2⁶⁴-1 ≈ 1.8×10¹⁹.

### 정답
1,3,7,15,31 / T(n)=2ⁿ-1 / 2⁶⁴-1≈1.8×10¹⁹

### 해설
T(n) = 2ⁿ-1 (앞서 4단계에서 배운 패턴).
하노이 탑: n개 원판 최소 이동 횟수 = 2ⁿ-1.
T(1)=1, T(2)=3, T(3)=7, T(4)=15, T(5)=31.
64개: 2⁶⁴-1 = 18,446,744,073,709,551,615.
초당 1번 이동 시 약 5,850억 년.
이것이 "하노이 탑의 저주" 전설의 기원.

## 문제 9

다음 패턴에서 규칙을 일반화하고
증명해보세요.

```
2² - 1² = 3 = 2+1
3² - 2² = 5 = 3+2
4² - 3² = 7 = 4+3
5² - 4² = 9 = 5+4
n² - (n-1)² = ?
```

### 힌트 1
n² - (n-1)² = n² - (n²-2n+1) = 2n-1.

### 힌트 2
2n-1 = n + (n-1).

### 정답
2n-1 = n+(n-1)

### 해설
n² - (n-1)² = n² - n² + 2n - 1 = 2n-1.
= n + (n-1).
연속 제곱수의 차 = 홀수.
이것이 홀수의 합이 제곱수가 되는 이유.
1 = 1²-0².
3 = 2²-1².
5 = 3²-2².
7 = 4²-3².
1+3+5+7 = 4² = 16 (인접 제곱수 차의 합 = 제곱수).

## 문제 10

다음 수열을 분석하고
이것이 패턴인지 우연인지 판단하세요.

```
1/1 = 1.000000...
1/7 = 0.142857142857...
2/7 = 0.285714285714...
3/7 = 0.428571428571...
```

(1) 1/7의 순환마디는?
(2) 2/7, 3/7의 순환마디와 1/7의 관계는?
(3) 이 패턴의 수학적 원인은?

### 힌트 1
1/7 = 0.(142857) 순환마디=142857.
2/7 = 0.(285714): 142857을 다르게 시작.

### 힌트 2
142857의 순환 배열:
142857→285714→428571→571428→714285→857142.

### 정답
순환마디=142857 / 2/7=142857의순환 / 7은소수이므로완전순환

### 해설
1/7 = 0.142857142857...
순환마디: 142857 (6자리).
2/7: 285714 (142857을 285714로 회전).
3/7: 428571.
4/7: 571428.
5/7: 714285.
6/7: 857142.
모두 142857의 순환 배열!
수학적 이유:
7은 소수, 10은 7의 원시근.
10의 거듭제곱이 mod 7에서 6주기.
142857 × 7 = 999999.
142857 × 1,2,...,6 = 142857의 순환.
