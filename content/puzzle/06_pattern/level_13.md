## 문제 1

다음 음악과 패턴의 관계를 탐구하세요.

바흐의 푸가:
주제 → 대위적 변형 → 다시 주제.
수학적 대칭: 역행(retrograde), 역전(inversion), 확대(augmentation).

리만의 음악 이론:
조성 관계를 토넷츠(Tonnetz) 격자로.
```
... F - C - G - D - A ...
   / \ / \ / \ / \ /
  Ab - Eb - Bb - F ...
```

(1) 푸가의 변형이 군론적 연산과 대응되는 방식은?
(2) 토넷츠 격자의 수학적 구조는?
(3) 음악 패턴의 자기 유사성은?

### 힌트 1
역행: 시간 역전 T.
역전: 음정 반전 I.
역행역전: TI.
이 네 가지 {원형,역행,역전,역행역전}이 군 Z₂×Z₂.

### 힌트 2
토넷츠: 5도권과 단3도·장3도의 격자.
위상: 도넛(토러스) 구조.

### 정답
변형=Z₂×Z₂군 / 토넷츠=격자·토러스위상 / 음악=자기유사구조

### 해설
푸가의 군론:
주제 P: 원형.
역행 R: 음표 순서 역전.
역전 I: 음정 방향 반전.
역행역전 RI: R후 I.
{P,R,I,RI}: Z₂×Z₂ (클라인 4원군).
쇤베르크 12음 기법: 이 군을 12개 음 조합에 적용.
토넷츠:
수평: 5도 관계 (7반음).
대각선: 장3도(4반음), 단3도(3반음).
위상: 12음의 등가로 토러스.
무한 격자 → 유한 토러스.
음악의 자기 유사성:
바흐: 같은 주제가 다른 성부에서 반복.
프랙탈 음악: 자기 유사적 멜로디 패턴.

## 문제 2

다음 유전자 코드의 패턴을 탐구하세요.

DNA 코드:
4개 염기: A, T, G, C (또는 RNA: A, U, G, C).
코돈(codon): 3개 염기 = 아미노산 1개.
4³ = 64가지 코돈 → 20가지 아미노산.

대칭 패턴:
3번째 염기가 종종 무관 (흔들림 가설).
A↔G (퓨린), C↔T (피리미딘).

(1) 코돈 표의 수학적 구조는?
(2) 유전 코드의 오류 수정 패턴은?
(3) 생명 코드와 수학적 코드의 공통점은?

### 힌트 1
64코돈 → 20아미노산: 평균 3.2코돈/아미노산.
64를 20으로 나누면 여분 코드 → 오류 내성.

### 힌트 2
흔들림: 3번째 염기 변이가 아미노산 변화 안 함.
오류 수정: 해밍 코드와 유사.

### 정답
64→20=여분코드 / 흔들림=3번째염기무관·오류내성 / 생명=오류수정코드

### 해설
유전 코드 수학:
4⁴=64코돈, 20아미노산 + 정지코돈 3개.
여분(redundancy): 오류 내성.
메티오닌만 코돈 1개, 세린은 6개.
흔들림 가설 (크릭, 1966):
3번째 염기의 일부 변이는 같은 아미노산 코딩.
A↔G, U↔C 교환 가능.
생물학적 오류 수정.
해밍 코드와의 유사성:
해밍: 여분 비트로 오류 감지·수정.
유전 코드: 여분 코돈으로 돌연변이 내성.
진화적 최적화:
유해 돌연변이를 최소화하는 코드 배열.
계산 실험: 실제 코드가 거의 최적.

## 문제 3

다음 사회 네트워크의 패턴을 탐구하세요.

파워 법칙(power law) 네트워크:
P(k) ∝ k^(-α): 연결도 k를 가진 노드 비율.
대부분 노드 = 낮은 연결, 소수 = 허브.

바라바시-알베르트 모델:
선호 연결(preferential attachment):
새 노드가 기존 연결 많은 노드에 연결 선호.
→ 부익부 패턴 → 멱함수 분포.

(1) 선호 연결이 멱함수를 만드는 수학은?
(2) 소세계 현상과 파워 법칙의 관계는?
(3) 이 패턴이 사회에 주는 함의는?

### 힌트 1
연결도 k가 되는 확률: 계속 성장하는 네트워크.
P(k) ∝ k^(-3) (BA 모델의 지수=3).

### 힌트 2
소세계: 높은 클러스터 + 짧은 평균 거리.
파워 법칙: 허브 존재 → 짧은 경로.

### 정답
선호연결→부익부→멱함수k^-3 / 허브=짧은경로 / 불평등·취약성함의

### 해설
바라바시-알베르트 모델:
n번째 노드가 노드 i에 연결될 확률 ∝ kᵢ.
마스터 방정식: dP(k)/dt = ... → P(k) ∝ k⁻³.
지수 α=3: BA 모델의 보편적 지수.
소세계와 멱함수:
허브 존재 → 어떤 노드도 허브 근처에 있음.
평균 거리 O(log log n) (ultra-small world).
반면 무작위 제거에 강함, 표적 공격에 약함.
사회적 함의:
인터넷: 허브 사이트 공격 취약.
부의 분배: 파레토 법칙 (80:20).
전염병: 허브 노드가 핵심 전파자.
정책: 허브 중심 전략의 효율성과 위험성.

## 문제 4

다음 물리학의 대칭 패턴을 탐구하세요.

뇌터 정리:
연속 대칭 → 보존 법칙.

```
시간 이동 대칭 → 에너지 보존
공간 이동 대칭 → 운동량 보존
회전 대칭 → 각운동량 보존
게이지 대칭 → 전하 보존
```

(1) 대칭과 보존의 수학적 연결은?
(2) 대칭이 깨질 때 어떤 패턴이 나타나나요?
(3) 표준 모형의 대칭 패턴 U(1)×SU(2)×SU(3)은?

### 힌트 1
해밀턴 역학: H가 어떤 변환에 불변이면 대응 양 보존.
[H,Q]=0 → dQ/dt=0.

### 힌트 2
대칭 깨짐: 힉스 메커니즘.
U(1)×SU(2) → U(1): 전자기만 남음.

### 정답
해밀턴교환자=0→보존 / 자발적대칭깨짐→질량생성 / 표준모형=게이지군

### 해설
뇌터 정리:
라그랑지안이 연속 변환에 불변 → 보존류 존재.
에너지: L이 시간 이동에 불변.
운동량: L이 공간 이동에 불변.
양자역학: [H,Q]=0 → Q 보존.
대칭 깨짐:
자발적 대칭 깨짐: 바닥 상태가 대칭 없음.
멕시코 모자 포텐셜: 대칭적 위치 → 불안정.
힉스 메커니즘: W,Z 보손이 질량 획득.
표준 모형:
U(1): 전자기 (광자).
SU(2): 약력 (W±, Z).
SU(3): 강력 (글루온 8개).
패턴: 대칭 → 힘의 전달 입자.

## 문제 5

다음 패턴과 언어의 관계를 탐구하세요.

촘스키 계층(Chomsky hierarchy):
정규 언어 ⊂ 문맥 자유 ⊂ 문맥 의존 ⊂ 귀납적.

```
정규: (ab)* - 유한 오토마타
문맥자유: aⁿbⁿ - 푸시다운 오토마타
문맥의존: aⁿbⁿcⁿ - 선형 유계 오토마타
귀납적: 모든 계산 가능 언어 - 튜링 기계
```

(1) 각 언어 클래스의 패턴 복잡도는?
(2) 자연 언어는 어느 클래스인가요?
(3) 언어 패턴과 수열 패턴의 공통 구조는?

### 힌트 1
정규: 상태만으로 기억.
문맥자유: 스택으로 중첩 구조.
자연 언어: 문맥 의존 특성 있음.

### 힌트 2
aⁿbⁿ: n을 기억 필요 → 정규 아님.
aⁿbⁿcⁿ: 두 카운터 필요 → 문맥자유 아님.

### 정답
정규<문맥자유<문맥의존<귀납 / 자연언어=문맥의존근처 / 언어=패턴의생성규칙

### 해설
촘스키 계층:
정규 언어: aaabbb 패턴 (고정 반복).
문맥자유: aⁿbⁿ (중첩 가능).
문맥의존: aⁿbⁿcⁿ (세 카운터).
귀납적: 모든 계산 가능.
자연 언어:
"더 많은 쥐를 더 많은 고양이가 더 많이 쫓았다."
교차 의존: 문맥자유보다 강함.
스위스-독일어: aⁿbⁿcⁿ 구조.
일반 합의: 문맥 의존에 가까움.
하지만 완전한 분류: 미해결.
언어와 수열:
둘 다 생성 규칙 (문법, 점화식).
둘 다 계층적 구조.
촘스키 = 수열 복잡도의 언어 버전.

## 문제 6

다음 경제학의 패턴을 탐구하세요.

게임 이론의 반복 패턴:
죄수의 딜레마 반복 게임.

```
협력(C) vs 배신(D):
둘 다 C: 각 3점.
한쪽 D, 한쪽 C: D=5, C=0.
둘 다 D: 각 1점.
```

팃포탯(Tit-for-Tat) 전략:
1라운드: C.
이후: 상대방 이전 선택 복사.

(1) 왜 팃포탯이 강력한가요?
(2) 팃포탯의 수학적 조건은?
(3) 협력의 창발 패턴은?

### 힌트 1
팃포탯: 협력 유도 + 배신 즉시 응징.
미래 이익 현재가치: δ × 이익.

### 힌트 2
협력 균형 조건: δ ≥ (5-3)/(5-1) = 1/2.
할인율 δ > 0.5이면 협력이 균형.

### 정답
협력유도+즉각응징 / δ≥1/2이면협력균형 / 창발=반복+기억+응징

### 해설
팃포탯의 강점:
단순: 기억 1라운드만.
협력적: 먼저 협력.
응징적: 배신에 즉각 응징.
용서: 상대가 돌아오면 협력 재개.
수학적 분석:
무한 반복 게임, 할인율 δ.
팃포탯 vs 항상 C 가치: 3/(1-δ).
배신 후 팃포탯 대응: 5+δ/(1-δ).
협력이 최적 조건: 3/(1-δ) ≥ 5+δ/(1-δ).
→ δ ≥ 1/2.
협력의 창발:
반복 + 기억 + 응징 능력 = 협력 가능.
진화: 이타적 행동의 수학적 기원.
인간 사회의 협력 패턴을 설명.

## 문제 7

다음 정보 이론의 패턴을 탐구하세요.

데이터 압축의 한계:
섀넌의 소스 코딩 정리:
최적 압축 = 엔트로피 H(X).
H(X) = -Σ P(x)log₂P(x).

허프만 코딩의 패턴:
빈도 높은 심볼 → 짧은 코드.
빈도 낮은 심볼 → 긴 코드.

```
A: 50% → 0
B: 25% → 10
C: 12.5% → 110
D: 12.5% → 111
```

(1) 이 코드의 평균 길이는?
(2) 엔트로피와 비교하면?
(3) 무손실 압축의 한계는?

### 힌트 1
평균 길이: 0.5×1+0.25×2+0.125×3+0.125×3 = 1.75.
엔트로피: -0.5log2-0.25log4-0.25log8 = 1.75.

### 힌트 2
허프만 = 최적 정수 코드.
산술 코딩: 비정수 → 엔트로피에 더 가까움.

### 정답
평균1.75비트 / 엔트로피=1.75(최적달성) / 한계=엔트로피(더이상압축불가)

### 해설
허프만 코딩:
A(0.5): 1비트, B(0.25): 2비트, C,D(0.125각): 3비트.
평균 길이 = 0.5×1+0.25×2+0.125×3+0.125×3 = 1.75비트.
엔트로피 = -0.5log₂0.5-0.25log₂0.25-0.25log₂0.125
= 0.5+0.5+0.375+0.375 = 1.75비트.
최적 달성!
압축 한계:
섀넌: H(X) 비트 이하 압축 불가.
H(X)=0: 완전 예측 가능 (압축 극대).
H(X)=log₂n: 완전 균등 (압축 불가).
ZIP, MP3, JPEG: 섀넌 한계에 근접하는 실용 알고리즘.

## 문제 8

다음 뇌의 패턴 인식을 탐구하세요.

게슈탈트 원리:
근접성(proximity), 유사성(similarity),
연속성(continuity), 폐합성(closure),
대칭성(symmetry).

```
* * *    * * *
* * * vs * . *
* * *    * * *
```
두 패턴의 지각적 차이.

(1) 게슈탈트 원리가 수학적으로 표현되면?
(2) 패턴 완성(completion)의 수학은?
(3) 딥러닝의 패턴 인식과 인간의 차이는?

### 힌트 1
근접성: 거리 함수.
유사성: 특성 공간의 군집.
연속성: 측지선 (짧은 경로).

### 힌트 2
폐합성: 불완전한 패턴 → 최솟값 에너지 완성.
호프필드 네트워크: 에너지 최소화로 패턴 완성.

### 정답
게슈탈트=거리·군집·측지선 / 완성=에너지최소화 / 딥러닝=피처통계·인간=개념이해

### 해설
게슈탈트의 수학:
근접성: 거리 d(x,y) < ε이면 같은 그룹.
유사성: 특성 공간의 클러스터링 (k-means).
연속성: 곡선의 최소 곡률 (스플라인).
폐합성: 불완전 → 에너지 최소 완성.
호프필드 네트워크:
E = -½ΣᵢⱼWᵢⱼsᵢsⱼ.
에너지 최소화 = 저장된 패턴 복원.
연상 기억: 부분 패턴 → 전체 패턴.
딥러닝 vs 인간:
딥러닝: 통계적 피처, 큰 데이터 필요.
인간: 적은 예시로 개념 형성 (few-shot).
인간: 인과 이해, AI: 상관 관계.
적대적 예시: AI는 쉽게 속음, 인간은 아님.

## 문제 9

다음 패턴과 진화의 관계를 탐구하세요.

진화 알고리즘:
자연 선택을 모방한 최적화.

```
1. 초기 집단 생성 (무작위)
2. 적합도 평가
3. 선택 (좋은 것 남김)
4. 교배·변이
5. 2로 돌아감
```

피셔의 기본 정리:
"집단의 평균 적합도 증가율 = 적합도의 분산."

(1) 진화 알고리즘이 찾는 패턴은?
(2) 탐색-활용 딜레마와 패턴은?
(3) 자연 선택이 발견하는 패턴의 한계는?

### 힌트 1
진화: 적합도 함수의 최대값 탐색.
유전 알고리즘: 해 공간의 패턴 탐색.

### 힌트 2
탐색(exploration): 새 영역 탐색.
활용(exploitation): 알려진 좋은 영역 집중.
ε-탐욕 전략.

### 정답
적합도함수최적해 / ε-탐욕딜레마 / 국소최대에갇힐수있음

### 해설
유전 알고리즘:
해 = 염색체 (이진 문자열 등).
교배: 두 해의 부분 조합.
변이: 무작위 변형.
선택: 적합도 높은 해 선호.
탐색-활용 딜레마:
탐색만: 최적 해 놓침.
활용만: 국소 최대에 갇힘.
균형: ε-탐욕, 볼츠만 온도, UCB.
자연 선택의 한계:
국소 최대 함정.
"완벽한 설계" 불가능.
진화: 충분히 좋은 것 (satisficing).
기러기 V자 편대: 최적 패턴 발견.
하지만 인간 눈 = 뒤집힌 설계 (진화 제약).

## 문제 10

마스터 구간의 시작으로
자신만의 패턴 퍼즐을 설계하세요.

조건:
①실생활과 깊이 연결됨.
②직관과 다른 패턴적 결과.
③풀이 과정에서 핵심 패턴 개념 학습.
④10~12단계 수준의 난이도.
⑤명확한 수학적 정답이 있을 것.

### 힌트 1
반직관적 패턴: 지수 성장, 멱함수, 복리, 카오스.

### 힌트 2
실생활: 금융, 생물, 소셜 미디어, 물리.

### 정답
개인 창작 (다양한 답 가능)

### 해설
예시 퍼즐: "소셜 미디어 바이럴 패턴"

문제:
게시물 A가 처음 10명에게 공유됩니다.
각 사람은 24시간 내에 10명에게 더 공유합니다.
단, 이미 본 사람은 다시 보지 않습니다.

소셜 네트워크 크기가 100만 명일 때:
(1) 며칠 후 모든 사람이 보게 되나요?
(2) 바이럴의 지수 성장이 멈추는 이유는?
(3) SIR 모델로 분석하면?

수학:
n일 후 시청자: 10 × 10ⁿ = 10ⁿ⁺¹.
10ⁿ⁺¹ ≥ 10⁶ → n ≥ 5.
5일 후 모든 사람 도달.
하지만 실제: SIR 모델의 S(취약) 감소로 성장 둔화.
포함 개념: 지수 성장, 포화, SIR 모델, 네트워크.
