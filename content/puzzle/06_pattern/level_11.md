## 문제 1

다음 모듈러 형식(modular form)의 패턴을 분석하세요.

람다 함수:
λ(τ) = (θ₂(τ)/θ₃(τ))⁴

세타 함수 계수 패턴:
θ₃(q)² = Σ r₂(n)qⁿ
r₂(n): n을 두 제곱수의 합으로 나타내는 방법 수.

```
r₂(1)=4: (±1,0),(0,±1)
r₂(2)=4: (±1,±1)
r₂(5)=8: (±1,±2),(±2,±1)
r₂(3)=0: 불가능
```

(1) r₂(n)의 패턴을 설명하세요.
(2) n이 4k+3 형태의 소인수를 홀수 번 가지면?
(3) 야코비의 두 제곱수 정리란?

### 힌트 1
r₂(n)=0 ↔ n이 4k+3 형태 소인수를 홀수 번 포함.
r₂(3)=0: 3≡3(mod4), 홀수 번.

### 힌트 2
야코비: r₂(n) = 4(d₁(n)-d₃(n)).
d₁(n): 4k+1 형태 약수 수.
d₃(n): 4k+3 형태 약수 수.

### 정답
r₂(n)=4(d₁-d₃) / 4k+3소인수홀수=0 / 야코비정리

### 해설
야코비의 두 제곱수 정리:
r₂(n) = 4(d₁(n)-d₃(n)).
d₁(n): n의 약수 중 4k+1 형태.
d₃(n): n의 약수 중 4k+3 형태.
예시:
n=5: 약수 1(≡1),5(≡1) → d₁=2,d₃=0 → r₂=8 ✓.
n=3: 약수 1(≡1),3(≡3) → d₁=1,d₃=1 → r₂=0 ✓.
n=2: 약수 1(≡1),2(≡2) → 2≡2는 어느 쪽? → r₂=4.
모듈러 형식과 수론의 깊은 연결.

## 문제 2

다음 다항식 패턴을 분석하세요.

체비쇼프 다항식:
T₀(x)=1, T₁(x)=x, T₂(x)=2x²-1.
점화식: Tₙ(x) = 2xTₙ₋₁(x) - Tₙ₋₂(x).

```
T₃(x) = 4x³-3x
T₄(x) = 8x⁴-8x²+1
```

(1) T₃, T₄를 검증하세요.
(2) Tₙ(cos θ) = cos(nθ)임을 확인하세요.
(3) 체비쇼프 다항식의 영점은?

### 힌트 1
T₃: 2x(2x²-1)-x = 4x³-2x-x = 4x³-3x ✓.

### 힌트 2
Tₙ(cos θ)=cos(nθ)이면 영점:
cos(nθ)=0 → nθ=(2k-1)π/2 → θ=(2k-1)π/(2n).
xₖ = cos((2k-1)π/(2n)).

### 정답
T₃=4x³-3x·T₄=8x⁴-8x²+1 / Tn(cosθ)=cos(nθ) / xk=cos((2k-1)π/2n)

### 해설
체비쇼프 다항식:
점화식: Tₙ = 2xTₙ₋₁-Tₙ₋₂.
T₄: 2x(4x³-3x)-(2x²-1) = 8x⁴-6x²-2x²+1 = 8x⁴-8x²+1 ✓.
삼각 항등식 연결:
cos(nθ)의 다중각 공식이 Tₙ(cosθ).
cos(2θ)=2cos²θ-1 → T₂(x)=2x²-1 ✓.
영점: x=cos((2k-1)π/(2n)), k=1,...,n.
n=2: x=cos(π/4)=√2/2, x=cos(3π/4)=-√2/2.
→ T₂(x)=2x²-1의 영점 ✓.
응용: 수치 근사에서 최적 표본점.

## 문제 3

다음 그래프 이론의 패턴을 분석하세요.

크로마틱 다항식(chromatic polynomial):
그래프 G의 k-채색 수를 k의 다항식으로.

경로 그래프 Pₙ (n꼭짓점):
P(Pₙ, k) = k(k-1)ⁿ⁻¹.

완전 그래프 Kₙ:
P(Kₙ, k) = k(k-1)(k-2)...(k-n+1).

(1) P₃(삼각형)의 크로마틱 다항식은?
(2) k=3이면 삼각형의 채색 수는?
(3) 크로마틱 다항식과 4색 정리의 관계는?

### 힌트 1
삼각형=K₃: P(K₃,k)=k(k-1)(k-2).
k=3: 3×2×1=6.

### 힌트 2
4색 정리: 평면 그래프 P(G,4)>0.
채색 다항식이 양수 ↔ k-채색 가능.

### 정답
k(k-1)(k-2) / 6가지 / 4색정리=P(G,4)>0보장

### 해설
크로마틱 다항식:
삭제-수축(deletion-contraction):
P(G,k) = P(G-e,k) - P(G/e,k).
삼각형=K₃: P=k(k-1)(k-2).
k=3: 6 (3!=6, 완전 그래프).
4색 정리(채색 다항식 관점):
모든 평면 그래프에 대해 P(G,4)>0.
→ 4색으로 인접 채색 가능.
증명(1976): 컴퓨터 보조.
크로마틱 근: P(G,k)=0의 근들.
버크호프(1912): P(G,k)≥0 (k=자연수)이길 기대.
하지만 복소수 근 존재: 버크호프-루이스 추측.

## 문제 4

다음 정수론의 아름다운 패턴을 분석하세요.

가우스 합(Gauss sum):
G(χ,ψ) = Σ_{a mod p} χ(a)ψ(a).

특수 경우:
G = Σ_{a=0}^{p-1} e^{2πia²/p}.

p=5의 경우:
G = e^0 + e^{2πi/5} + e^{8πi/5} + e^{18πi/5} + e^{32πi/5}.

(1) |G|² = p임을 설명하세요.
(2) 가우스가 이 합으로 무엇을 증명했나요?
(3) 현대 수학에서의 응용은?

### 힌트 1
|G|² = G×G* = p (p 소수).
이는 이차 상호 법칙 증명에 핵심.

### 힌트 2
G² = (-1)^{(p-1)/2} × p.
p≡1(mod4): G²=p, G=√p.
p≡3(mod4): G²=-p, G=i√p.

### 정답
|G|²=p(절댓값제곱=p) / 이차상호법칙증명 / 암호·수론핵심도구

### 해설
가우스 합의 성질:
G² = (-1)^{(p-1)/2} × p.
p=5(≡1 mod4): G²=5, G=√5.
p=3(≡3 mod4): G²=-3, G=i√3.
가우스의 업적:
이차 상호 법칙의 6번째 증명에 활용.
(가우스는 일생 8개 증명).
이차 상호 법칙:
(p/q)(q/p) = (-1)^{(p-1)(q-1)/4}.
현대 응용:
지수 합: 암호학, 수론.
랑랜즈 프로그램의 기초.
가우스 합 → 모듈러 형식 → 페르마 마지막 정리.

## 문제 5

다음 조합론적 수 패턴을 분석하세요.

스털링 수(Stirling numbers):
제1종 스털링 수 s(n,k): n! 순열에서 k개 사이클.
제2종 스털링 수 S(n,k): n원소를 k개 비공 집합으로 분할.

```
S(4,2)=7:
{1}{234},{2}{134},{3}{124},{4}{123}
{12}{34},{13}{24},{14}{23}
```

(1) S(n,1)과 S(n,n)은?
(2) 벨 수와의 관계는?
(3) 점화식은?

### 힌트 1
S(n,1)=1: 모두 한 집합.
S(n,n)=1: 각각 하나씩.

### 힌트 2
B(n) = Σᵢ₌₁ⁿ S(n,k).
S(n,k) = k×S(n-1,k) + S(n-1,k-1).

### 정답
S(n,1)=1·S(n,n)=1 / B(n)=ΣS(n,k) / kS(n-1,k)+S(n-1,k-1)

### 해설
S(n,k) (제2종 스털링):
S(n,1)=1, S(n,n)=1.
S(4,2)=7 (검증 ✓).
점화식: S(n,k)=kS(n-1,k)+S(n-1,k-1).
해석: n번째 원소를 기존 k집합에 추가(k가지)
또는 새 집합 생성(이전 S(n-1,k-1)).
벨 수 연결: B(n)=Σᵢ₌₀ⁿ S(n,k).
B(4)=S(4,1)+S(4,2)+S(4,3)+S(4,4)=1+7+6+1=15 ✓.
폴링 다항식: xⁿ = Σᵢ₌₀ⁿ S(n,k)x(x-1)...(x-k+1).

1D 셀룰러 오토마타 규칙 30:
초기: 가운데 셀만 1.
규칙: 현재셀+왼쪽+오른쪽 → 다음 상태.

```
규칙 30 (이진수 00011110):
111→0, 110→0, 101→0, 100→1
```

단순 규칙도 튜링 완전이 될 수 있음.
"모든 계산적으로 동등한 시스템은 같은 능력."
새로운 과학(A New Kind of Science, 2002).

## 문제 7

다음 다변수 패턴을 분석하세요.

슈르 다항식(Schur polynomial):
s_λ(x₁,...,xₙ): 영 다이어그램 λ와 연관.

단순 사례:
λ=(2,1): 두 행, 첫 행 2칸, 둘째 행 1칸.

s_(2,1)(x,y) = x²y + xy²

(1) 대칭성을 확인하세요.
(2) s_(n)(x,y) = Σ xⁱyⁿ⁻ⁱ는?
(3) 슈르 다항식의 조합론적 의미는?

### 힌트 1
s_(2,1)(x,y): x와 y를 교환해도 같음 (대칭).
s_(n)(x,y) = xⁿ+xⁿ⁻¹y+...+yⁿ (완전 대칭 다항식).

### 힌트 2
반표준 영 타블로: λ형태에 숫자 채우기.
s_(2,1): 11|2, 12|1 등.

### 정답
x,y교환불변 / xⁿ+xⁿ⁻¹y+...+yⁿ / 반표준영타블로의수

### 해설
슈르 다항식:
대칭군의 표현론에서 핵심.
반표준 영 타블로로 정의:
λ=(2,1)의 타블로: 행은 증가, 열은 강증가.
s_(2,1)(x,y) = x²y+xy² (두 타블로에서).
완전 동차 대칭 다항식:
s_(n) = h_n = Σ xⁱ₁xⁱ₂...xᵢₖ (i₁≤i₂≤...≤iₖ).
응용:
조합론: RSK 대응.
표현론: GL(n)의 표현 분류.
기하학: 그라스만 다양체의 코호몰로지.

## 문제 8

다음 암호학적 패턴을 분석하세요.

해시 함수의 눈사태 효과(avalanche effect):
입력 1비트 변화 → 출력 ~50% 비트 변화.

SHA-256 패턴:
"abc" → 특정 256비트 해시.
"abd" → 완전히 다른 해시.

(1) 이것이 왜 필요한가요?
(2) 선형 피드백 시프트 레지스터(LFSR) 패턴은?
(3) 해시 충돌의 확률론적 분석은?

### 힌트 1
눈사태: 작은 변화 → 큰 변화.
예측 불가능성 = 보안의 핵심.

### 힌트 2
생일 공격: √(2ⁿ) = 2^(n/2) 시도 후 충돌.
SHA-256: 충돌 찾기 ≈ 2^128 시도.

### 정답
예측불가성=보안 / LFSR=선형패턴(취약) / 생일공격=2^(n/2)

### 해설
눈사태 효과:
좋은 해시 함수의 필수 조건.
SAC(Strict Avalanche Criterion): 1비트 변화 → 정확히 50% 비트 변화.
LFSR:
선형 점화식: xₙ=xₙ₋₁ XOR xₙ₋₃.
주기가 길지만 선형 → 취약 (베르마세그 알고리즘).
비선형 조합이 필요.
생일 공격:
n비트 해시: ~2^(n/2)번 시도로 충돌.
n=128: 2^64 ≈ 10^19 시도.
SHA-256: n=256, 충돌 ≈ 2^128.
생일 역설: 23명에서 50% 확률로 같은 생일.
= √(365) ≈ 19명 (수학적 이유).

## 문제 9

다음 대수적 수의 패턴을 분석하세요.

대수적 수: 정수 계수 다항식의 근.
초월수: 대수적이 아닌 수.

```
√2: x²-2=0의 근 → 대수적
∛3: x³-3=0의 근 → 대수적
φ=(1+√5)/2: x²-x-1=0의 근 → 대수적
π: 초월수 (린데만, 1882)
e: 초월수 (에르미트, 1873)
```

(1) 대수적 수의 개수는?
(2) 초월수의 비율은?
(3) 리우빌 수란?

### 힌트 1
대수적 수: 가산 무한개.
초월수: 비가산 무한개.
"거의 모든" 실수가 초월수.

### 힌트 2
리우빌 수: Σ10^(-n!) = 0.1100010000...
이 수가 초월수임을 증명한 최초 사례.

### 정답
대수적=가산무한 / 초월수=비가산(거의모두) / 리우빌=최초초월수

### 해설
대수적 수의 개수:
각 다항식의 근의 수: 유한.
다항식 수: 가산 무한.
→ 대수적 수 = 가산 무한.
초월수:
실수 = 비가산 무한.
대수적 = 가산 무한.
→ 초월수 = 비가산 무한.
"거의 모든" 실수가 초월수!
리우빌 수 (1851):
L = 0.1100010000000000000000010000...
n번째 자릿수: n!번째 위치에만 1.
대수적 수 근사 한계로 초월성 증명.
최초로 초월수의 존재 증명.

## 문제 10

다음 패턴의 미래를 탐구하세요.

AI와 패턴 발견:
딥마인드의 AlphaFold: 단백질 구조 패턴.
알파지오메트리(AlphaGeometry): 기하 증명 패턴.
알파코드(AlphaCode): 코딩 패턴.

수학의 새로운 패턴 발견:
그린-타오 정리(2004):
소수 중에 임의로 긴 등차수열이 있다.
5,11,17,23,29 (공차=6).

(1) 그린-타오 정리가 놀라운 이유는?
(2) AI가 새로운 수학 패턴을 발견할 수 있나요?
(3) 패턴 발견의 미래는?

### 힌트 1
소수는 "무작위"처럼 보이지만
임의로 긴 등차수열 존재.
정규성과 랜덤성의 공존.

### 힌트 2
AI: 알파텐서(AlphaTensor)가 행렬 곱 알고리즘 발견.
