## 문제 1

다음 세 가지 기하학을 비교하세요.

유클리드 기하 (평면):
- 삼각형 내각 합 = 180도.
- 평행선은 만나지 않음.
- 피타고라스 정리 성립.

구면 기하 (지구 표면):
- 삼각형 내각 합 > 180도.
- 모든 대원은 만남 (평행선 없음).

쌍곡 기하 (안장면):
- 삼각형 내각 합 < 180도.
- 한 직선 밖 점에서 평행선 무한히 많음.

(1) 지구에서 북극→적도→90도 동진→북극의 삼각형 내각 합은?
(2) 각 기하에서 원의 둘레와 반지름 관계는?

### 힌트 1
북극→적도: 경도 0.
적도→90도 동진: 위도 0.
90도→북극: 경도 90.
각 꼭짓점의 각도를 구하세요.

### 힌트 2
북극 각도: 90도 (경도 0과 90의 차이).
적도의 두 꼭짓점: 각 90도.

### 정답
270도 / 유클리드=2πr, 구면<2πr, 쌍곡>2πr

### 해설
구면 삼각형:
북극 꼭짓점: 90도 (경선 0°와 90° 사이).
적도 왼쪽 꼭짓점: 90도 (경선과 적도 직각).
적도 오른쪽 꼭짓점: 90도.
내각 합 = 90+90+90 = 270도.
(180도 초과 = 양의 곡률)
원의 둘레:
평면: C = 2πr.
구면: C < 2πr (곡률이 양수 → 작아짐).
쌍곡: C > 2πr (곡률이 음수 → 커짐).

## 문제 2

다음 대칭 변환군을 분석하세요.

정사각형의 대칭군 D4:
- 회전: 0°, 90°, 180°, 270° (4가지)
- 반사: 가로축, 세로축, 대각선 2개 (4가지)
- 총 8가지 대칭 변환

(1) D4의 원소 8개를 나열하세요.
(2) 90도 회전 후 가로축 반사와
    가로축 반사 후 90도 회전은 같은가요?
(3) 군의 닫힘 성질을 설명하세요.

### 힌트 1
회전: r0(항등), r1(90°), r2(180°), r3(270°).
반사: s1(가로), s2(세로), s3(대각선/), s4(대각선\).

### 힌트 2
r1 후 s1 ≠ s1 후 r1 (비가환).

### 정답
8원소 / 순서에따라다름(비가환군) / 두변환의합성=항상군내원소

### 해설
D4 원소:
r0: 항등 (0도 회전).
r1: 90도 회전.
r2: 180도 회전.
r3: 270도 회전.
s1: 가로축 반사.
s2: 세로축 반사.
s3: 주대각선 반사.
s4: 부대각선 반사.
비가환성:
r1∘s1 ≠ s1∘r1.
(90도 후 가로반사 ≠ 가로반사 후 90도)
닫힘 성질:
임의의 두 대칭 합성 → 항상 D4의 원소.
군의 4조건: 닫힘, 결합법칙, 항등원, 역원.

## 문제 3

다음 타일링(테셀레이션) 문제를 분석하세요.

정다각형 중 평면을 완전히 덮을 수 있는 것은?

정삼각형 (내각 60°): 60×? = 360.
정사각형 (내각 90°): 90×? = 360.
정육각형 (내각 120°): 120×? = 360.
정오각형 (내각 108°): 108×? = 360.
정칠각형 (내각 ≈128.6°): 가능?

### 힌트 1
꼭짓점 주위 각도 합 = 360도이어야 합니다.
n각형의 내각 = (n-2)×180/n.

### 힌트 2
정오각형: 360/108 ≈ 3.33 (정수 아님 → 불가).
정삼각형: 360/60 = 6 → 가능.

### 정답
정삼각형(6개), 정사각형(4개), 정육각형(3개)만 가능

### 해설
평면 타일링 조건:
꼭짓점에서 만나는 각도 합 = 360도.
정삼각형(60°): 6개 = 360 ✓.
정사각형(90°): 4개 = 360 ✓.
정오각형(108°): 360/108 ≈ 3.33 ✗.
정육각형(120°): 3개 = 360 ✓.
정칠각형(≈128.6°): 360/128.6 ≈ 2.8 ✗.
오직 3가지 정다각형만 평면 타일링 가능.
벌집 구조: 정육각형 타일링 (효율 최대).
반정다각형 타일링: 두 종류 이상 조합으로 더 가능.

## 문제 4

다음 꼭짓점에서 만나는 정다면체들의
슐레플리 기호를 분석하세요.

슐레플리 기호 {p,q}: p각형 면, 각 꼭짓점에 q개 면.
정사면체: {3,3}
정육면체: {4,3}
정팔면체: {3,4}
정십이면체: {5,3}
정이십면체: {3,5}

조건: 꼭짓점의 각도 합 < 360도.

(1) {3,3}의 꼭짓점 각도 합은?
(2) 왜 {3,6}은 정다면체가 아닌가요?
(3) 5가지만 존재하는 이유를 설명하세요.

### 힌트 1
{3,3}: 정삼각형(60°) 3개 → 180도.
{3,6}: 정삼각형(60°) 6개 → 360도.

### 힌트 2
꼭짓점 각도 합 < 360도 → 볼록 다면체.
= 360도 → 평면 타일링.
> 360도 → 불가.

### 정답
{3,3}=180° / {3,6}=360°(타일링만가능) / 5개=각도조건만족경우

### 해설
슐레플리 기호 {p,q} 조건:
꼭짓점 각도 합 = q × (p-2)×180/p < 360.
{3,3}: 3×60 = 180 < 360 ✓ (정사면체).
{4,3}: 3×90 = 270 < 360 ✓ (정육면체).
{3,4}: 4×60 = 240 < 360 ✓ (정팔면체).
{5,3}: 3×108 = 324 < 360 ✓ (정십이면체).
{3,5}: 5×60 = 300 < 360 ✓ (정이십면체).
{3,6}: 6×60 = 360 = 360 → 타일링 (평면).
{4,4}: 4×90 = 360 → 타일링.
조건을 만족하는 {p,q}는 딱 5가지뿐.
따라서 플라톤 입체는 5가지만 존재.

## 문제 5

다음 아핀 변환(affine transformation)을 분석하세요.

아핀 변환: y = Ax + b
A는 행렬, b는 평행이동 벡터.

변환 T1: A=[[2,0],[0,2]], b=[1,1].
변환 T2: A=[[0,-1],[1,0]], b=[0,0].
(T2는 90도 회전)

(1) T1을 점 (1,1)에 적용하면?
(2) T2를 점 (3,0)에 적용하면?
(3) T1∘T2(T2 먼저 후 T1)를 점 (1,0)에 적용하면?

### 힌트 1
T1: 2배 확대 후 (1,1) 이동.
(1,1) → (2,2) → (3,3).

### 힌트 2
T2: 90도 회전.
(3,0) → (0,3).

### 정답
(1)(3,3) (2)(0,3) (3)T2먼저(0,1)후T1적용=(1,3)

### 해설
(1) T1(1,1):
2×(1,1) + (1,1) = (2,2)+(1,1) = (3,3).
(2) T2(3,0):
[[0,-1],[1,0]]×(3,0) = (0×3+(-1)×0, 1×3+0×0) = (0,3).
(3) T1∘T2(1,0):
먼저 T2: (0,3)×(1,0)...
T2(1,0) = (0×1+(-1)×0, 1×1+0×0) = (0,1).
T1(0,1) = 2×(0,1)+(1,1) = (0,2)+(1,1) = (1,3).
결과: (1,3).

## 문제 6

다음 입체의 위상적 성질을 분석하세요.

클라인 병(Klein bottle):
뫼비우스 띠 두 개를 가장자리끼리 붙인 것.

```
클라인 병의 특성:
- 3D 공간에서 자기 교차 없이 표현 불가
- 4D 공간에서 완전한 표현 가능
```

(1) 클라인 병의 면의 수는?
(2) 안과 밖이 구분되나요?
(3) 구, 토러스, 클라인 병의 오일러 수를 비교하세요.

### 힌트 1
뫼비우스 띠: 1면, 1가장자리.
두 뫼비우스 띠를 가장자리로 붙이면?

### 힌트 2
클라인 병: 1면 (단면 곡면).
오일러 수: χ = V - E + F.

### 정답
면=1개 / 안밖구분없음 / 구χ=2·토러스χ=0·클라인χ=0

### 해설
클라인 병:
면: 1개 (비방향성 단면 곡면).
안과 밖: 구분 없음 (뫼비우스 띠처럼).
3D에서 자기 교차 없이 표현 불가.
4D에서만 완전 임베딩 가능.
오일러 수 비교:
구(genus=0): χ = 2.
토러스(genus=1): χ = 0.
클라인 병: χ = 0 (토러스와 동일한 오일러 수).
하지만 클라인 병 ≠ 토러스 (방향성 다름).
비방향성 곡면: 뫼비우스 띠, 클라인 병, 사영 평면.

## 문제 7

다음 결정 구조(crystal structure)를 분석하세요.

입방체 격자계:
단순 입방: 꼭짓점에만 원자.
체심 입방(BCC): 꼭짓점 + 중심에 원자.
면심 입방(FCC): 꼭짓점 + 각 면 중심에 원자.

(1) 각 격자의 단위 셀당 원자 수는?
(2) 충전율(packing fraction)이 가장 높은 것은?
(3) 금속 결정에서 FCC가 많은 이유는?

### 힌트 1
꼭짓점 원자: 8개 꼭짓점 각 1/8 = 1개.
면 원자: 6개 면 각 1/2 = 3개.
체심 원자: 중심 = 1개.

### 힌트 2
단순 입방: 1개.
BCC: 1+1 = 2개.
FCC: 1+3 = 4개.

### 정답
SC=1 / BCC=2 / FCC=4 / FCC충전율=74%최고

### 해설
단위 셀당 원자 수:
SC: 8×(1/8) = 1.
BCC: 8×(1/8) + 1 = 2.
FCC: 8×(1/8) + 6×(1/2) = 4.
충전율:
SC: π/6 ≈ 52%.
BCC: π√3/8 ≈ 68%.
FCC: π/(3√2) ≈ 74%.
FCC = 최밀 충전 구조.
구를 가장 효율적으로 쌓는 방법 = FCC or HCP (둘 다 74%).
금, 은, 알루미늄, 구리: FCC 구조.
FCC의 이점: 충전율 최고 + 미끄럼면 많음 = 연성.

## 문제 8

다음 투영(projection)의 종류와 특성을 분석하세요.

지도 투영법:
메르카토르 도법: 각도 보존 (정각).
람베르트 도법: 넓이 보존 (정적).
스테레오그래픽: 원 보존.

(1) 어떤 도법도 거리·각도·넓이를 동시에 보존할 수 없는 이유는?
(2) 메르카토르 도법에서 그린란드와 아프리카 크기 왜곡은?
(3) 구에서 평면으로 정보 손실이 불가피한 이유는?

### 힌트 1
구는 가우스 곡률이 양수.
평면은 곡률 0.
가우스의 놀라운 정리: 곡률은 국소 등거리 변환에서 보존.

### 힌트 2
메르카토르: 고위도로 갈수록 넓이 과장.
그린란드(실제): 아프리카의 약 1/14 크기.

### 정답
가우스정리=곡률불보존 / 그린란드=아프리카1/14지만지도에서비슷 / 구↔평면=위상동치지만등거리불가

### 해설
보존 불가능성:
가우스 놀라운 정리: 국소 등거리 변환에서 가우스 곡률 보존.
구(곡률>0) ≠ 평면(곡률=0).
→ 거리 보존 투영 불가.
각도·넓이·거리 중 하나만 선택 가능.
메르카토르 왜곡:
그린란드 실제 면적: 약 216만 km².
아프리카 실제 면적: 약 3,037만 km².
비율: 약 1:14.
메르카토르에서는 비슷해 보임 (위도 오차).
위상적 관점:
구와 평면: 위상적 비동치 (구는 컴팩트, 평면은 비컴팩트).
구를 투영하려면 적어도 한 점 제거 필요 (무한점).

## 문제 10

다음 공간 분류 문제를 종합적으로 분석하세요.

2D 컴팩트 곡면의 분류 정리:
모든 콤팩트 방향성 곡면은 genus g의 토러스.
모든 콤팩트 비방향성 곡면은 cross-cap k개.

g=0: 구.
g=1: 토러스.
g=2: 이중 토러스.
k=1: 사영 평면.
k=2: 클라인 병.

(1) 각 곡면의 오일러 수는?
(2) 방향성과 비방향성의 차이는?
(3) 3D 닫힌 다양체의 분류는?

### 힌트 1
방향성 곡면 오일러 수: χ = 2-2g.
비방향성: χ = 2-k.

### 힌트 2
방향성: 곡면 위에 일관된 방향 정의 가능.
비방향성: 뫼비우스 띠처럼 방향이 뒤집힘.

### 정답
χ=2-2g(방향성)·2-k(비방향성) / 방향=안팎구분 / 3D=기하화추측(페렐만)

### 해설
오일러 수:
구(g=0): χ=2.
토러스(g=1): χ=0.
이중토러스(g=2): χ=-2.
사영평면(k=1): χ=1.
클라인병(k=2): χ=0.
방향성 vs 비방향성:
방향성: 법벡터가 연속적으로 정의.
비방향성: 법벡터가 반전되는 경로 존재.
