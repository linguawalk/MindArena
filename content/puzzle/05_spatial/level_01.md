# 공간 퍼즐 1단계 — 도형과 방향 입문

> 공간 퍼즐의 시작입니다.
> 도형을 관찰하고 방향을 파악하는 것이 핵심입니다.
> 머릿속에서 도형을 회전하고 변환해보세요.

---

## 문제 1

다음 도형에서 꼭짓점의 개수를 구하세요.

```
    *
   / \
  /   \
 *-----*
```

### 힌트 1
꼭짓점은 선이 만나는 점입니다.

### 힌트 2
삼각형의 모서리를 세어보세요.

### 정답
3개

### 해설
삼각형의 꼭짓점: 위쪽 1개 + 왼쪽 아래 1개 + 오른쪽 아래 1개 = 3개.
삼각형은 꼭짓점 3개, 변 3개, 내각 3개를 가집니다.

## 문제 2

다음 도형의 이름은 무엇인가요?

```
*-------*
|       |
|       |
*-------*
```

### 힌트 1
네 개의 꼭짓점을 가지고 있습니다.

### 힌트 2
네 변의 길이가 모두 같아 보입니다.

### 정답
정사각형 (또는 직사각형)

### 해설
네 꼭짓점, 네 변을 가진 사각형입니다.
가로와 세로의 길이가 같으면 정사각형,
다르면 직사각형입니다.
아스키아트에서는 정확한 비율 표현이 어렵습니다.

## 문제 3

다음 도형을 시계 방향으로 90도 회전하면
어떤 모양이 되나요?

```
*
|
|
*--*
```

### 힌트 1
왼쪽 아래가 직각인 L자 모양입니다.

### 힌트 2
시계 방향 90도: 위→오른쪽, 오른쪽→아래.

### 정답
```
*--*
   |
   *
```
(위로 뻗은 L이 오른쪽으로 뻗는 모양)

### 해설
시계 방향 90도 회전:
원래: 세로로 긴 변이 위아래, 짧은 변이 오른쪽 아래.
회전 후: 세로 변이 가로로, 짧은 변이 아래쪽.
```
*--*
   |
   *
```

## 문제 4

다음 그림에서 삼각형은 몇 개인가요?

```
    *
   / \
  *---*
 / \ / \
*---*---*
```

### 힌트 1
작은 삼각형부터 세어보세요.

### 힌트 2
작은 삼각형 4개 + 큰 삼각형을 찾아보세요.

### 정답
5개

### 해설
작은 삼각형: 4개 (아래 줄 3개 + 가운데 역삼각형 1개 포함하면...)
실제로:
아래 왼쪽 작은 삼각형: 1개.
아래 가운데 작은 삼각형: 1개.
아래 오른쪽 작은 삼각형: 1개.
위쪽 작은 삼각형: 1개.
전체를 아우르는 큰 삼각형: 1개.
총 5개.

## 문제 5

다음 주사위의 반대쪽 면을 구하세요.

일반 주사위의 규칙:
- 1의 반대: 6
- 2의 반대: 5
- 3의 반대: 4

주사위의 윗면이 3일 때 아랫면은?

### 힌트 1
마주보는 면의 합은 항상 7입니다.

### 힌트 2
3 + ? = 7.

### 정답
4

### 해설
주사위에서 마주보는 면의 합 = 7.
3의 반대 = 7 - 3 = 4.
앞면이 2이면 뒷면은 5.
앞면이 1이면 뒷면은 6.
이것은 표준 주사위의 규칙입니다.

## 문제 6

다음 도형에서 선의 개수를 구하세요.

```
*---*---*
|   |   |
*---*---*
|   |   |
*---*---*
```

### 힌트 1
가로선과 세로선을 따로 세어보세요.

### 힌트 2
가로선: 3줄 × 2개 = 6개.
세로선: 3줄 × 2개 = 6개.

### 정답
12개

### 해설
가로선: 각 행에 2개씩, 3행 = 6개.
세로선: 각 열에 2개씩, 3열 = 6개.
총 12개.
(3×3 격자에는 꼭짓점 9개, 선분 12개가 있습니다)

## 문제 7

다음 도형의 넓이는?
(각 칸의 넓이 = 1)

```
*-*-*
|X|X|
*-*-*
|X| |
*-*-*
```
(X 표시된 칸이 색칠된 부분)

### 힌트 1
X 표시가 된 칸의 수를 세어보세요.

### 힌트 2
위 행: 2개, 아래 행: 1개.

### 정답
3

### 해설
X로 표시된 칸:
위 행: 2칸.
아래 행: 1칸.
총 3칸 = 넓이 3.
도형의 넓이 = 단위 정사각형의 개수.

## 문제 8

다음 중 나머지와 다른 하나는?

도형 A: 삼각형
도형 B: 원
도형 C: 사각형
도형 D: 오각형

### 힌트 1
각 도형의 특성을 비교해보세요.

### 힌트 2
꼭짓점의 유무를 기준으로 분류하면?

### 정답
도형 B (원)

### 해설
A(삼각형), C(사각형), D(오각형): 모두 다각형 (직선으로 이루어진 도형, 꼭짓점 있음).
B(원): 곡선으로 이루어진 도형, 꼭짓점 없음.
원은 꼭짓점과 변이 없는 유일한 도형입니다.

## 문제 9

다음 방향 안내를 따라 최종 위치를 구하세요.

출발: (0, 0)
① 오른쪽으로 3칸
② 위로 2칸
③ 왼쪽으로 1칸
④ 아래로 1칸

### 힌트 1
좌표를 단계별로 추적하세요.

### 힌트 2
오른쪽=+x, 위=+y, 왼쪽=-x, 아래=-y.

### 정답
(2, 1)

### 해설
출발: (0, 0).
① 오른쪽 3: (3, 0).
② 위 2: (3, 2).
③ 왼쪽 1: (2, 2).
④ 아래 1: (2, 1).
최종 위치: (2, 1).

## 문제 10

다음 도형을 보고 대칭축의 개수를 구하세요.

정삼각형의 대칭축은 몇 개인가요?
정사각형의 대칭축은 몇 개인가요?
원의 대칭축은 몇 개인가요?

### 힌트 1
대칭축: 도형을 접었을 때 완전히 겹치게 하는 선.

### 힌트 2
정삼각형: 꼭짓점에서 반대쪽 변의 중점으로.
정사각형: 가로·세로·대각선 방향.

### 정답
정삼각형: 3개 / 정사각형: 4개 / 원: 무한개

### 해설
정삼각형: 각 꼭짓점에서 반대쪽 변의 중점으로 3개.
정사각형: 가로 1개 + 세로 1개 + 대각선 2개 = 4개.
원: 중심을 지나는 모든 직선이 대칭축 → 무한개.
정n각형의 대칭축 수 = n개.
