# 공간 퍼즐 3단계 — 입체 도형 입문

> 2D에서 3D로 넘어갑니다.
> 입체 도형을 머릿속에서 펼치고 접어보세요.
> 전개도와 입체의 관계가 핵심입니다.

---

## 문제 1

정육면체(주사위 모양)의
면, 꼭짓점, 모서리의 개수를 구하세요.

```
    *---*
   /|  /|
  *---* |
  | *-|-*
  |/  |/
  *---*
```

### 힌트 1
면: 앞·뒤·위·아래·왼·오른쪽.

### 힌트 2
꼭짓점: 모서리가 만나는 점.
모서리: 면과 면이 만나는 선.

### 정답
면: 6개 / 꼭짓점: 8개 / 모서리: 12개

### 해설
정육면체:
면(face): 앞·뒤·위·아래·좌·우 = 6개.
꼭짓점(vertex): 8개 (각 모서리의 끝점).
모서리(edge): 12개 (각 면의 변).
오일러 공식: V - E + F = 2.
8 - 12 + 6 = 2 ✓.

## 문제 2

다음 전개도를 접으면 어떤 입체가 되나요?

```
  [위]
[좌][앞][우][뒤]
  [아래]
```
(십자 모양의 전개도)

### 힌트 1
십자 모양으로 펼쳐진 6개의 정사각형.

### 힌트 2
6개의 면이 모두 정사각형입니다.

### 정답
정육면체 (큐브)

### 해설
십자형 전개도:
중앙 가로 4개: 앞·오른쪽·뒤·왼쪽 면.
위아래 1개씩: 위·아래 면.
접으면 6면체 = 정육면체.
정육면체 전개도: 11가지 서로 다른 형태가 있습니다.

## 문제 3

다음 정육면체를 위에서 본 모양(평면도)은?

```
    *---*
   /|  /|
  *---* |
  | *-|-*
  |/  |/
  *---*
```

### 힌트 1
위에서 내려다보면 윗면만 보입니다.

### 힌트 2
정육면체의 윗면은 정사각형입니다.

### 정답
```
*---*
|   |
*---*
```
(정사각형)

### 해설
정육면체를 위에서 보면:
윗면(정사각형)만 보입니다.
앞면·옆면은 아래로 숨어서 안 보입니다.
평면도(위): 정사각형.
정면도(앞): 정사각형.
측면도(옆): 정사각형.

## 문제 4

다음 쌓기 나무의 개수를 구하세요.

```
앞에서 본 모양:
*-*
|2|
*-*
(숫자는 그 위치에 쌓인 개수)

  앞
*-*-*
|2|1|
*-*-*
```

### 힌트 1
각 칸의 숫자가 그 위치에 쌓인 큐브 수입니다.

### 힌트 2
왼쪽 칸: 2개, 오른쪽 칸: 1개.

### 정답
3개

### 해설
위에서 본 배치도에서:
왼쪽 앞: 2개 쌓임.
오른쪽 앞: 1개 쌓임.
총 2+1 = 3개.
쌓기 나무 문제는 위에서 본 그림의 숫자를 더합니다.

## 문제 5

정삼각형 4개로 만들 수 있는 입체 도형은?

```
정삼각형:
   *
  / \
 /   \
*-----*
```

### 힌트 1
4개의 정삼각형 면으로 이루어진 입체.

### 힌트 2
피라미드와 비슷하지만 밑면도 삼각형입니다.

### 정답
정사면체 (tetrahedron)

### 해설
정사면체:
면: 4개 (정삼각형).
꼭짓점: 4개.
모서리: 6개.
오일러: 4 - 6 + 4 = 2 ✓.
플라톤 입체 중 가장 단순한 것.
5가지 플라톤 입체: 정사면체·정육면체·정팔면체·정십이면체·정이십면체.

## 문제 6

다음 전개도 중 정육면체가 되는 것은?

전개도 A:
```
[1][2][3][4]
   [5]
   [6]
```

전개도 B:
```
[1][2][3]
[4][5]
[6]
```

전개도 C:
```
[1][2][3][4][5]
         [6]
```

### 힌트 1
정육면체 전개도: 6개의 정사각형이
연결된 특정 패턴이어야 합니다.

### 힌트 2
전개도 C: 5개가 일렬이면 마지막 면이
접었을 때 이미 있는 면과 겹칩니다.

### 정답
A와 B (C는 불가)

### 해설
전개도 A: 가로 4개 + 위아래 1개씩 = 유효한 십자형 변형 ✓.
전개도 B: L자 배치 = 유효한 전개도 ✓.
전개도 C: 5개 일렬 + 1개 = 접으면 면이 겹침 ✗.
정육면체 전개도 11가지 모두 유효하며
나머지 배치들은 접으면 면이 겹칩니다.

## 문제 7

원기둥의 전개도를 설명하세요.
어떤 도형들로 이루어져 있나요?

```
원기둥:
   ___
  /   \
 |     |
 |     |
  \___/
```

### 힌트 1
원기둥을 펼치면 옆면은 어떤 모양?
위아래 면은 어떤 모양?

### 힌트 2
옆면을 펼치면 직사각형이 됩니다.
위아래는 원입니다.

### 정답
원 2개 + 직사각형 1개

### 해설
원기둥 전개도:
위 뚜껑: 원 (반지름 r).
아래 바닥: 원 (반지름 r).
옆면: 직사각형 (가로 = 2πr, 세로 = 높이 h).
겉넓이 = 2×πr² + 2πr×h.
직사각형의 가로가 원의 둘레와 같음.

## 문제 8

다음 입체를 정면·측면·위에서 본
세 가지 그림(삼각법)을 그리세요.

입체: 밑면이 정사각형인 사각뿔.

```
      *        ← 꼭대기
     /|\
    / | \
   /  |  \
  *---+---*   ← 밑면
  |   |   |
  *---+---*
```

### 힌트 1
정면에서 보면: 삼각형.
위에서 보면: 정사각형 + 대각선.

### 힌트 2
옆에서 보면: 삼각형 (정면과 같은 모양).

### 정답
정면: 삼각형 / 측면: 삼각형 / 위: 정사각형+대각선

### 해설
사각뿔의 삼각법:
정면도:
```
  *
 / \
*---*
```
측면도 (정면과 동일):
```
  *
 / \
*---*
```
평면도(위):
```
*---*
|\  /|
| \/ |
| /\ |
|/  \|
*---*
```
(정사각형 + 꼭짓점에서 교차하는 대각선)

## 문제 9

정육면체를 대각선으로 잘랐을 때
단면의 모양은?

```
    *---*
   /|  /|
  *---* |
  | *-|-*
  |/  |/
  *---*

한 꼭짓점과 대각선으로 반대쪽 꼭짓점을
포함하는 평면으로 자르기.
```

### 힌트 1
정육면체의 세 꼭짓점을 포함하는 단면.
어떤 다각형이 될까요?

### 힌트 2
3개의 꼭짓점을 지나는 평면 → 삼각형?
6개의 꼭짓점을 지나면?

### 정답
직사각형 (또는 정사각형)

### 해설
정육면체 대각 단면:
4개의 꼭짓점을 지나는 평면으로 자르면 직사각형.
특히 정육면체의 공간 대각선을 포함하는 평면:
단면 = 직사각형 (가로 = 모서리 길이, 세로 = 면 대각선).
정육면체를 잘랐을 때 나오는 단면:
삼각형, 사각형, 오각형, 육각형 모두 가능.

## 문제 10

다음 쌓기 나무를 앞에서 봤을 때
모양을 그리세요.

```
위에서 본 그림 (숫자=높이):
*-*-*
|3|2|
*-*-*
|1|2|
*-*-*
```
(앞쪽이 아래 행)

### 힌트 1
앞에서 볼 때: 각 열에서 가장 높은 것이 보입니다.
왼쪽 열: max(3,1)=3.
오른쪽 열: max(2,2)=2.

### 힌트 2
왼쪽에 3층, 오른쪽에 2층이 보입니다.

### 정답
```
*
* *
* *
```
(왼쪽 3칸, 오른쪽 2칸)

### 해설
앞에서 보기:
왼쪽 열: 앞(1층)+뒤(3층) → 가장 높은 3층이 보임.
오른쪽 열: 앞(2층)+뒤(2층) → 2층이 보임.
앞면도:
```
*
*-*
*-*
```
(왼쪽 3단, 오른쪽 2단)
