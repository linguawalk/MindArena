# 공간 퍼즐 4단계 — 공간 추론 기초

## 문제 1

다음 쌓기 나무 구조에서
보이지 않는 큐브는 몇 개인가요?

```
앞에서 본 모양:    옆에서 본 모양:
*-*               *-*
|2|               |2|
*-*               *-*
```

위에서 본 모양:
```
*-*
|?|
*-*
```

### 힌트 1
앞과 옆에서 모두 높이 2로 보입니다.
위에서 보면 1×1 칸입니다.

### 힌트 2
1×1×2 = 2개 큐브인데 앞에서 1개만 보입니다.

### 정답
1개 (뒤쪽 큐브가 보이지 않음)

### 해설
1×1 바닥에 2층으로 쌓인 구조:
앞에서: 2층 높이로 보임.
옆에서: 2층 높이로 보임.
총 큐브 2개 중 앞에서 볼 때 뒤쪽 1개는
앞쪽 큐브에 가려져 보이지 않습니다.
가려진 큐브 = 1개.

## 문제 2

다음 전개도를 접었을 때
마주보는 면끼리 올바르게 연결하세요.

```
      [A]
   [B][C][D]
      [E]
      [F]
```

### 힌트 1
C가 앞면이라고 가정하면:
B=왼쪽, D=오른쪽, A=위, E=아래.

### 힌트 2
F는 E 아래에 있으므로 접으면 뒷면.

### 정답
A↔F (위↔뒤), B↔D (왼↔오른), C↔E (앞↔아래)

### 해설
C를 앞면으로 설정:
C(앞) ↔ ?
B(왼) ↔ D(오른).
A(위) ↔ ?.
E는 C 아래 → C(앞)의 아래면.
F는 E 아래 → 접으면 C(앞)의 반대인 뒷면.
A(위) ↔ F(뒤)의 반대편.
정확한 마주보기:
A ↔ F (위 ↔ 뒤로 이어진 면).
B ↔ D (왼 ↔ 오른).
C ↔ E (앞 ↔ 아래로 이어진 면의 반대).

## 문제 3

다음 도형을 잘랐을 때 단면의 모양을 구하세요.

구를 임의의 평면으로 자르면?
원기둥을 축에 수직으로 자르면?
원기둥을 축에 평행하게 자르면?

### 힌트 1
구: 어떻게 잘라도 단면은?
원기둥 수직 절단: 윗면과 같은 모양?

### 힌트 2
원기둥 평행 절단: 긴 방향으로 자르면?

### 정답
구=원 / 원기둥수직=원 / 원기둥평행=직사각형

### 해설
구의 단면:
어떤 평면으로 잘라도 항상 원.
중심을 지나면 최대원(대원).
원기둥 수직 절단:
축에 수직 = 밑면과 평행.
단면 = 원.
원기둥 평행 절단:
축에 평행 = 옆면을 가름.
단면 = 직사각형 (가로=지름, 세로=높이의 일부).

## 문제 4

다음 그림에서 정육면체를 이루려면
몇 개의 큐브가 더 필요한가요?

```
현재 상태 (위에서 본 그림, 숫자=높이):
*-*-*
|2|1|
*-*-*
|1|0|
*-*-*
목표: 3×3×3 정육면체의 왼쪽 앞 2×2 부분
```

목표 높이: 모두 3.

### 힌트 1
각 칸의 현재 높이와 목표 높이(3)의 차이를 구하세요.

### 힌트 2
(3-2)+(3-1)+(3-1)+(3-0) = ?

### 정답
8개

### 해설
각 칸에 필요한 추가 큐브:
(행1,열1): 3-2 = 1개.
(행1,열2): 3-1 = 2개.
(행2,열1): 3-1 = 2개.
(행2,열2): 3-0 = 3개.
합계: 1+2+2+3 = 8개.

## 문제 5

다음 두 입체를 합쳤을 때 만들어지는
새로운 입체의 겉넓이는?
(각 면의 넓이 = 1)

입체 A: 1×1×1 정육면체.
입체 B: 1×1×1 정육면체.
합치는 방법: 한 면을 완전히 맞붙임.

### 힌트 1
각 정육면체의 겉넓이: 6×1 = 6.
두 개의 총 겉넓이: 12.

### 힌트 2
맞붙은 면 2개는 안쪽으로 숨어서 보이지 않음.
12 - 2 = 10.

### 정답
10

### 해설
각 정육면체 겉넓이: 6.
두 개 합산: 12.
맞붙은 면: 각 1개씩, 총 2개가 내부로 숨음.
겉넓이 감소: 2.
합친 입체 겉넓이: 12 - 2 = 10.
이 원리로 큐브를 여러 개 붙일 때
겉넓이를 계산할 수 있습니다.

## 문제 6

다음 좌표에서 점 A를 y축에 대해
반사한 점 A'를 구하세요.

A = (3, 2, 1) (3D 좌표)

### 힌트 1
y축 반사: x 좌표의 부호가 바뀝니다.
y, z 좌표는 변하지 않습니다.

### 힌트 2
(x, y, z) → (-x, y, z).

### 정답
A' = (-3, 2, 1)

### 해설
y축에 대한 반사 (3D):
y축은 x=0인 평면(y-z 평면)에 포함.
y-z 평면을 거울로 반사:
x → -x, y → y, z → z.
A(3,2,1) → A'(-3,2,1).
3D에서의 반사 변환.

## 문제 7

다음 도형을 x축 방향으로 2배,
y축 방향으로 3배 확대하면
꼭짓점들의 새 위치는?

원래 꼭짓점: (0,0), (2,0), (2,1), (0,1).

### 힌트 1
x 좌표 × 2, y 좌표 × 3.

### 힌트 2
(x, y) → (2x, 3y).

### 정답
(0,0), (4,0), (4,3), (0,3)

### 해설
확대 변환:
(0,0) → (0×2, 0×3) = (0,0).
(2,0) → (2×2, 0×3) = (4,0).
(2,1) → (2×2, 1×3) = (4,3).
(0,1) → (0×2, 1×3) = (0,3).
원래: 가로 2, 세로 1 직사각형.
확대 후: 가로 4, 세로 3 직사각형.
넓이: 2 → 12 (6배 확대).

## 문제 8

다음 패턴에서 빈칸을 채우세요.

```
정면:     옆면:     평면:
*-*       *-*       *-*-*
|1|       |1|       |1|1|
*-*       *-*       *-*-*
```

위 삼각법에서 입체를 복원하면?

### 힌트 1
정면: 1×1 정사각형 → 높이 1, 폭 1.
옆면: 1×1 정사각형 → 높이 1, 깊이 1.
평면: 1×2 직사각형 → 폭 1, 깊이 2.

### 힌트 2
모든 조건을 만족하는 입체는?

### 정답
1×2×1 직육면체 (가로1, 세로2, 높이1)

### 해설
삼각법 복원:
정면(앞): 폭=1, 높이=1.
측면(옆): 깊이=1, 높이=1.
평면(위): 폭=1, 깊이=2.
→ 가로(폭)=1, 세로(깊이)=2, 높이=1.
1×2×1 직육면체.
삼각법: 세 방향의 투영으로 입체를 특정.

## 문제 9

정육면체를 꼭짓점에서 꼭짓점으로
대각선을 그었을 때 대각선의 길이는?
(모서리 길이 = 1)

```
    *---*
   /|  /|
  *---* |
  | *-|-*
  |/  |/
  *---*
```

### 힌트 1
3D 피타고라스 정리:
d² = a² + b² + c².

### 힌트 2
a=1, b=1, c=1.
d² = 1+1+1 = 3.

### 정답
√3 ≈ 1.732

### 해설
정육면체 공간 대각선:
모서리 a=b=c=1.
3D 피타고라스: d = √(a²+b²+c²).
d = √(1+1+1) = √3 ≈ 1.732.
면 대각선: √(1²+1²) = √2 ≈ 1.414.
공간 대각선 > 면 대각선 > 모서리 길이.
√3 > √2 > 1.

## 문제 10

다음 도형 조각들을 조합하여
4×4×4 정육면체를 만들려고 합니다.

사용 가능한 조각:
조각 A: 2×2×2 정육면체 (8개 단위).
조각 B: 1×1×4 막대 (4개 단위).
조각 C: 1×1×1 정육면체 (1개 단위).

4×4×4 = 64개 단위.
조각 A 4개를 사용했을 때
남은 공간을 채우려면 B와 C를 어떻게 써야 하나요?

### 힌트 1
A 4개: 4×8 = 32개 단위.
남은 공간: 64-32 = 32개 단위.

### 힌트 2
B 7개: 7×4 = 28개 단위.
남은 4개는 C 4개로.

### 정답
B 8개(32개) 또는 B 7개+C 4개 등 다양한 조합 가능

### 해설
남은 공간: 32개 단위.
가능한 조합:
B만 사용: 32/4 = 8개.
B 7개(28) + C 4개(4) = 32.
B 6개(24) + C 8개(8) = 32.
등 여러 조합.
핵심: 남은 공간 = 32개 단위를 채울 수 있으면 됨.
공간 분할의 다양한 해법이 가능합니다.
