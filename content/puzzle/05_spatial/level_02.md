# 공간 퍼즐 2단계 — 도형의 변환

> 도형이 어떻게 변하는지 탐구합니다.
> 회전, 반사, 이동의 세 가지 변환을 익힙니다.
> 머릿속에서 도형을 움직여보세요.

---

## 문제 1

다음 도형을 좌우 반사하면 어떤 모양이 되나요?

```
*
|\
| \
*--*
```

### 힌트 1
좌우 반사: 왼쪽과 오른쪽이 뒤바뀝니다.
세로축을 기준으로 뒤집습니다.

### 힌트 2
빗변이 왼쪽 위에서 오른쪽 아래로 향했다면
반사 후에는 오른쪽 위에서 왼쪽 아래로.

### 정답
```
  *
 /|
/  |
*--*
```

### 해설
좌우 반사(세로축 대칭):
원래: 빗변이 오른쪽 아래 방향.
반사: 빗변이 왼쪽 아래 방향.
직각은 왼쪽 아래에서 오른쪽 아래로 이동.
```
  *
 /|
/  |
*--*
```

## 문제 2

다음 도형을 180도 회전하면 어떤 모양이 되나요?

```
*--*
|
*
```

### 힌트 1
180도 회전: 위아래, 좌우가 모두 뒤바뀝니다.

### 힌트 2
원래: 위쪽에 가로선, 왼쪽 아래에 세로선.
180도 후: 아래쪽에 가로선, 오른쪽 위에 세로선.

### 정답
```
   *
   |
*--*
```

### 해설
180도 회전 = 점대칭.
원래 도형의 각 점이 중심을 기준으로 반대 위치로 이동.
위의 가로선 → 아래로.
왼쪽 아래 세로선 → 오른쪽 위로.
결과:
```
   *
   |
*--*
```

## 문제 3

다음 패턴에서 규칙을 찾아
빈칸에 들어갈 도형을 선택하세요.

```
▲  ▶  ▼  ?
```

선택지:
A: ▲  B: ◀  C: ▶  D: ▼

### 힌트 1
각 도형이 어떻게 변하고 있나요?
방향에 주목하세요.

### 힌트 2
▲(위) → ▶(오른쪽) → ▼(아래) → ?(왼쪽)?
시계 방향으로 90도씩 회전합니다.

### 정답
B: ◀

### 해설
시계 방향 90도 회전 패턴:
▲(위) → ▶(오른쪽) → ▼(아래) → ◀(왼쪽).
다음은 다시 ▲(위)로 돌아오는 4단계 순환.
공간 패턴 인식: 방향의 순환적 변화.

## 문제 4

다음 도형에서 몇 번 접으면
완전히 겹치는지 구하세요.

```
*-------*
|       |
|       |
|       |
*-------*
```
(정사각형)

가로로 한 번 접기, 세로로 한 번 접기,
대각선으로 한 번 접기 — 각각 겹치나요?

### 힌트 1
접었을 때 완전히 겹치려면 대칭이어야 합니다.

### 힌트 2
정사각형의 대칭축: 가로, 세로, 대각선 2개.

### 정답
모두 겹침 (정사각형은 4개의 대칭축)

### 해설
정사각형의 대칭축:
가로 중심선: 위아래 절반이 겹침 ✓.
세로 중심선: 좌우 절반이 겹침 ✓.
대각선 1 (왼위→오른아래): 겹침 ✓.
대각선 2 (오른위→왼아래): 겹침 ✓.
정사각형은 4방향 모두 대칭이므로 모두 완전히 겹칩니다.

## 문제 5

다음 두 도형이 합동인지 판단하세요.

도형 A:
```
*-*
|X|
*-*
```

도형 B:
```
*---*
|   |
*---*
```

### 힌트 1
합동: 모양과 크기가 완전히 같은 도형.

### 힌트 2
두 도형의 가로와 세로 비율을 비교하세요.

### 정답
합동이 아님

### 해설
합동: 크기와 모양이 모두 같아야 함.
도형 A: 정사각형 (가로=세로=1).
도형 B: 직사각형 (가로=3, 세로=1).
크기가 다르므로 합동이 아닙니다.
단, 닮음 관계일 수도 있습니다 (모양은 같지만 크기 다름).
하지만 A는 정사각형, B는 직사각형이므로 닮음도 아닙니다.

## 문제 6

다음 도형을 오른쪽으로 2칸, 위로 1칸 이동하면
꼭짓점 A의 새 위치는?

```
좌표:
   3|
   2|  *A
   1| /
   0|*
    +--------
    0 1 2 3
```

A의 현재 위치: (2, 2).

### 힌트 1
이동 변환: 모든 점이 같은 방향으로 같은 거리만큼 이동.

### 힌트 2
오른쪽 2칸: x+2.
위 1칸: y+1.

### 정답
(4, 3)

### 해설
이동 변환 (평행이동):
원래 A: (2, 2).
오른쪽 2칸: x = 2+2 = 4.
위 1칸: y = 2+1 = 3.
새 위치: (4, 3).
평행이동은 도형의 모양과 크기를 변환하지 않습니다.

## 문제 7

다음 도형의 둘레를 구하세요.
(각 칸의 변 길이 = 1)

```
*-*-*
|   |
*   *
|   |
*-*-*
```

### 힌트 1
바깥쪽 선분의 수를 세어보세요.

### 힌트 2
위: 2, 아래: 2, 왼쪽: 2, 오른쪽: 2.

### 정답
8

### 해설
직사각형(가로 2, 세로 2)의 둘레:
위 변: 2.
아래 변: 2.
왼쪽 변: 2.
오른쪽 변: 2.
둘레 = 2+2+2+2 = 8.
또는: 둘레 = 2×(가로+세로) = 2×(2+2) = 8.

## 문제 8

다음 도형에서 점 P를 중심으로
90도 시계 방향 회전한 점 A'의 위치는?

```
   3| A(1,3)
   2|
   1|   P(2,1)
   0|
    +--------
    0 1 2 3
```

### 힌트 1
P를 원점으로 이동 후 회전, 다시 P만큼 이동.
A-P = (1-2, 3-1) = (-1, 2).

### 힌트 2
시계 방향 90도 회전: (x,y) → (y,-x).
(-1, 2) → (2, 1).
다시 P 더하기: (2+2, 1+1) = (4, 2).

### 정답
(4, 2)

### 해설
P(2,1)을 중심으로 회전:
A-P = (1-2, 3-1) = (-1, 2).
시계 방향 90도: (x,y) → (y,-x).
(-1, 2) → (2, 1).
P 더하기: (2+2, 1+1) = (4, 2).
A' = (4, 2).

## 문제 9

다음 도형들 중 점대칭인 것을 모두 고르세요.

A: 정삼각형
B: 정사각형
C: 정육각형
D: 원
E: 직각삼각형

### 힌트 1
점대칭: 중심점을 기준으로 180도 회전했을 때
원래 도형과 완전히 겹치는 도형.

### 힌트 2
홀수 꼭짓점 다각형(정삼각형, 정오각형)은
점대칭이 아닙니다.

### 정답
B(정사각형), C(정육각형), D(원)

### 해설
점대칭 조건: 180도 회전 후 자기 자신과 겹침.
A(정삼각형): 꼭짓점 3개 → 홀수 → 점대칭 ✗.
B(정사각형): 꼭짓점 4개 → 짝수 → 점대칭 ✓.
C(정육각형): 꼭짓점 6개 → 짝수 → 점대칭 ✓.
D(원): 모든 방향으로 대칭 → 점대칭 ✓.
E(직각삼각형): 일반적으로 점대칭 ✗.
짝수 꼭짓점의 정다각형과 원은 점대칭입니다.

## 문제 10

다음 도형을 3가지 변환으로 분류하세요.

변환 A: 도형을 위로 5칸 올렸다.
변환 B: 도형을 세로축을 기준으로 뒤집었다.
변환 C: 도형을 중심점을 기준으로 45도 돌렸다.
변환 D: 도형을 2배로 확대했다.

평행이동·반사·회전·닮음변환 중 각각 해당하는 것은?

### 힌트 1
평행이동: 방향과 거리만 변화.
반사: 거울처럼 뒤집기.

### 힌트 2
회전: 각도만큼 돌리기.
닮음변환(확대·축소): 크기 변화.

### 정답
A=평행이동 / B=반사 / C=회전 / D=닮음변환

### 해설
평행이동: 방향·거리 이동, 모양·크기 불변 → A.
반사(대칭이동): 축을 기준으로 뒤집기 → B.
회전: 중심점 기준으로 각도만큼 회전 → C.
닮음변환: 크기가 변하는 변환 → D.
A,B,C는 합동 변환(크기 불변).
D는 닮음 변환(크기 변함).
