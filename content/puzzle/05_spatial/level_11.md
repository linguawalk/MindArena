## 문제 1

다음 컴퓨터 그래픽의 수학적 기초를 분석하세요.

동차 좌표(homogeneous coordinates):
2D점 (x,y) → 3D점 (x,y,1).
평행이동을 행렬 곱으로 표현:

```
[1 0 tx] [x]   [x+tx]
[0 1 ty] [y] = [y+ty]
[0 0  1] [1]   [  1 ]
```

(1) 왜 동차 좌표를 사용하나요?
(2) 3D 투영 변환을 동차 좌표로 표현하면?
(3) GPU가 4×4 행렬을 사용하는 이유는?

### 힌트 1
동차 좌표 없이는 평행이동이 선형 변환이 아님.
동차 좌표로 모든 아핀 변환을 행렬 곱으로.

### 힌트 2
3D→2D 투영:
[f 0 0 0] [x]   [fx]
[0 f 0 0] [y] = [fy]
[0 0 1 0] [z]   [z ]
          [1]   [0 ]
결과: (fx/z, fy/z).

### 정답
통일된행렬곱표현 / 투영=4×4행렬 / GPU=회전·이동·스케일·투영통합

### 해설
동차 좌표의 필요성:
선형 대수: 행렬 곱 = 선형 변환 (원점 고정).
평행이동: 원점이 이동 → 선형 변환 아님.
동차 좌표: 1차원 추가 → 평행이동도 행렬 곱.
모든 기하 변환을 통일된 행렬 곱으로.
3D 컴퓨터 그래픽:
모델 변환 (M): 객체 배치.
뷰 변환 (V): 카메라 위치.
투영 변환 (P): 3D→2D.
MVP = P×V×M: 단일 행렬 곱.
GPU 최적화: 4×4 행렬 곱 하드웨어 가속.
하나의 파이프라인으로 모든 변환 처리.

## 문제 2

다음 로봇공학의 공간 수학을 분석하세요.

순기구학(Forward Kinematics):
로봇 팔의 각 관절 각도 → 끝점 위치.

DH(Denavit-Hartenberg) 매개변수:
각 관절을 4×4 변환 행렬로 표현.
T_total = T₁ × T₂ × T₃ × ...

역기구학(Inverse Kinematics):
끝점 목표 위치 → 관절 각도 계산.

(1) 2관절 로봇 팔의 끝점 위치 계산은?
(2) 역기구학이 순기구학보다 어려운 이유는?
(3) 자유도(DOF)와 가동 공간의 관계는?

### 힌트 1
2관절: 길이 L1, L2, 각도 θ1, θ2.
끝점 x = L1cosθ1 + L2cos(θ1+θ2).
끝점 y = L1sinθ1 + L2sin(θ1+θ2).

### 힌트 2
역기구학: 여러 해가 존재하거나 없을 수 있음.
비선형 방정식 → 수치 해법 필요.

### 정답
순기구학=행렬곱 / 역기구학=비선형방정식·다중해 / DOF=관절수=가동공간차원

### 해설
2관절 순기구학:
T1: 관절1 회전 (θ1).
T2: 링크1 이동 + 관절2 회전 (L1, θ2).
T3: 링크2 이동 (L2).
T_total = T1 × T2 × T3.
끝점 = T_total × [0,0,0,1]ᵀ.
역기구학 어려움:
여러 해: 팔꿈치 위/아래 두 가지.
해 없음: 목표가 가동 범위 밖.
비선형: 삼각함수 포함.
수치 해법: 야코비안 역행렬, 경사 하강법.
자유도:
3D 공간에서 완전 제어: 6 DOF (위치 3+방향 3).
7 DOF: 여분 자유도 = 장애물 회피.
인간 팔: 7 DOF.

## 문제 3

다음 컴퓨터 비전의 공간 수학을 분석하세요.

에피폴라 기하(Epipolar Geometry):
두 카메라로 3D 공간 재구성.

본질 행렬(Essential Matrix) E:
두 카메라의 상대 위치·방향 인코딩.
x'ᵀEx = 0 (두 카메라의 대응점).

(1) 에피폴라 선이란?
(2) 삼각측량으로 깊이 추정하는 방법은?
(3) SLAM(동시 위치 추정 및 지도 작성)이란?

### 힌트 1
에피폴라 선: 한 카메라의 점 → 다른 카메라에서 직선.
대응점 탐색을 선으로 제한.

### 힌트 2
삼각측량: 두 광선의 교점 = 3D 점.
기저선 길이와 시차(disparity)로 깊이 계산.

### 정답
에피폴라선=대응점탐색제약 / 삼각측량=두광선교점 / SLAM=이동하며지도생성

### 해설
에피폴라 기하:
두 카메라 C₁, C₂와 3D점 P.
C₁에서 본 P의 상: p₁.
에피폴라 선: C₁p₁P를 C₂에 투영한 선.
p₂는 반드시 이 선 위에 있음.
대응점 탐색: 2D 탐색 → 1D 탐색으로 감소.
삼각측량:
C₁에서 p₁ 방향 광선.
C₂에서 p₂ 방향 광선.
교점 = 3D점 P.
시차 d = p₁.x - p₂.x → 깊이 Z = fB/d.
SLAM:
카메라/라이다로 이동하면서 동시에:
현재 위치 추정(Localization).
환경 지도 생성(Mapping).
자율주행·AR·드론에 핵심.

## 문제 4

다음 메시 처리(mesh processing)를 분석하세요.

3D 메시: 삼각형들의 집합으로 3D 표면 표현.
```
정점(V): 3D 좌표.
면(F): 세 정점의 인덱스.
반-모서리(half-edge): 방향 있는 모서리.
```

(1) 메시 평활화(smoothing)의 수학적 방법은?
(2) 라플라시안 평활화란?
(3) 파라메트리제이션(parametrization)이란?

### 힌트 1
라플라시안: 각 정점을 이웃 평균으로 이동.
v_i ← (1/N) Σ v_j (이웃 j).

### 힌트 2
파라메트리제이션: 3D 메시 → 2D 평면 펼치기.
UV 맵: 텍스처 적용을 위한 2D 좌표.

### 정답
라플라시안=이웃평균이동 / 파라메트리제이션=3D→2D전개 / UV맵=텍스처좌표

### 해설
라플라시안 평활화:
각 정점의 새 위치 = 이웃 정점들의 평균.
v_i^(t+1) = v_i^t + λ L v_i^t.
L = 라플라시안 행렬.
효과: 노이즈 제거, 표면 부드럽게.
단점: 수축(shrinkage) 발생.
개선: 이중 라플라시안, 법벡터 보존.
파라메트리제이션:
3D 표면 → 2D 평면 펼치기.
왜곡 최소화: 각도 보존(공형)·넓이 보존(등적).
UV 맵: 텍스처 좌표 (0~1 범위).
응용: 게임 텍스처, 의류 패턴, 지도.
전개 불가능: 구면 → 왜곡 불가피.

## 문제 5

다음 위상 데이터 분석(TDA)을 분석하세요.

점구름(point cloud) 데이터:
```
*   *
  *
*     *
  *
```

영구 호몰로지(persistent homology):
다양한 스케일에서 위상적 특성(구멍) 추적.

(1) 체크 복합체(Čech complex)란?
(2) 바코드(barcode) 다이어그램은?
(3) 데이터 분석에서의 응용은?

### 힌트 1
체크 복합체: 각 점 주위에 반지름 r인 구.
구들이 겹치는 관계로 단순 복합체 구성.

### 힌트 2
바코드: 각 위상적 특성이 생성-소멸되는 스케일.
긴 바코드 = 실제 구조, 짧은 바코드 = 노이즈.

### 정답
체크=반지름r구의교집합복합체 / 바코드=특성의생명주기 / 응용=형상분석·의학·재료

### 해설
체크 복합체:
반지름 r 증가에 따라 복합체 변화.
r 작음: 점들 분리 (H₀ = 점 수).
r 중간: 루프 형성 (H₁ > 0).
r 큼: 모두 연결 (H₀=1, H₁=0).
영구 호몰로지:
각 위상 특성의 "생성-소멸" 기록.
(생성 r, 소멸 r) = 바코드 막대.
긴 막대 = 실제 구조적 특성.
짧은 막대 = 노이즈.
응용:
의학: 종양 형태 분석, 뇌 신경망.
재료: 다공성 구조 분석.
자연어: 문서 의미 공간 구조.
센서 네트워크: 커버리지 구멍 탐지.

## 문제 6

다음 최적 수송(optimal transport)을 분석하세요.

몬주-칸토로비치 문제:
한 분포 μ를 다른 분포 ν로 "운반"하는
최소 비용 방법.

와서스테인 거리(Wasserstein distance):
두 확률 분포 사이의 기하학적 거리.

(1) 모래 더미 문제(earth mover's distance)란?
(2) 와서스테인 GAN이란?
(3) 최적 수송의 수학적 구조는?

### 힌트 1
모래 더미: 한 모양의 더미를 다른 모양으로
최소 일(이동거리×양)로 이동.

### 힌트 2
GAN: 생성자가 만든 분포 ↔ 실제 분포.
와서스테인 거리로 두 분포 차이 측정.

### 정답
모래더미=최소이동비용=W거리 / WGAN=안정적학습 / 구조=볼록최적화

### 해설
최적 수송 직관:
μ(모래 더미) → ν(목표 모양).
T: 수송 계획 (어디서 어디로).
비용 = ∫|x-T(x)|² dμ(x).
최소화 = 최적 수송 계획.
와서스테인 거리:
W₂(μ,ν) = min_T √(∫|x-T(x)|²dμ).
두 분포 사이의 "기하학적" 거리.
KL 발산보다 자연스러운 거리.
와서스테인 GAN:
기존 GAN: 수렴 불안정, 모드 붕괴.
WGAN: 와서스테인 거리로 학습 → 안정.
립시츠 조건: 판별자의 기울기 제한.
수학적 구조:
볼록 최적화 문제.
쌍대 문제: 칸토로비치 쌍대성.
연속체 역학: 수송 방정식.

## 문제 7

다음 계산 기하학(computational geometry)의
핵심 알고리즘을 분석하세요.

볼록 껍질(Convex Hull):
점들의 집합을 감싸는 가장 작은 볼록 다각형.

```
   *
*     *
  *  *
*     *
   *
→
  /*\
*/   \*
  \ /
*/   \*
  \*/
```

(1) 그레이엄 스캔 알고리즘이란?
(2) 시간 복잡도는?
(3) 응용 분야는?

### 힌트 1
그레이엄 스캔:
가장 아래 점 p₀ 선택.
나머지 점을 p₀ 기준 각도로 정렬.
스택으로 볼록 껍질 구성.

### 힌트 2
정렬: O(n log n).
스캔: O(n).
전체: O(n log n).

### 정답
각도정렬후스택스캔 / O(nlogn) / GIS·충돌감지·패턴인식

### 해설
그레이엄 스캔:
1. 가장 아래(왼쪽) 점 p₀ 선택.
2. 나머지 점을 p₀ 기준 극각으로 정렬.
3. 스택 순회: 시계 방향 회전이면 팝.
4. 모든 점 처리 후 스택 = 볼록 껍질.
핵심: 세 점의 방향(외적 부호)으로 볼록성 판단.
O(n log n): 정렬이 병목.
응용:
GIS: 지역 경계, 영역 계산.
충돌 감지: 게임, 로봇공학.
패턴 인식: 형태 기술자.
컴퓨터 비전: 객체 외곽선.
최적화: 선형 프로그래밍 실행 가능 영역.

## 문제 8

다음 공간 데이터 구조를 분석하세요.

KD-트리(k-dimensional tree):
k차원 공간의 점들을 이진 트리로 구조화.

```
2D 예시:
   7|*
   6|      *
   5| *
   4|         *
   3|   *
    +----------
    1 2 3 4 5 6

분할:
x=3으로 좌우 분할 후
각각 y값으로 분할.
```

(1) KD-트리 구성 알고리즘은?
(2) 최근접 이웃 탐색 방법은?
(3) 공간 인덱싱의 다른 방법은?

### 힌트 1
KD-트리 구성:
중앙값으로 공간 분할.
재귀적으로 좌우 서브트리 구성.

### 힌트 2
최근접 탐색:
리프 노드까지 내려가며 후보 찾기.
역추적하며 더 가까운 후보 갱신.

### 정답
중앙값으로재귀분할 / 역추적탐색=O(logn)평균 / R트리·옥트리·BSP트리

### 해설
KD-트리 구성:
1. 현재 깊이에서 축 선택 (0→x, 1→y, 2→z...).
2. 해당 축에서 중앙값으로 분할.
3. 좌우 재귀 구성.
시간: O(n log n) 구성, O(log n) 평균 탐색.
최근접 이웃:
1. 쿼리점으로 리프 내려감.
2. 현재 최근접 후보 갱신.
3. 역추적: 분할 초구가 후보보다 가까우면 다른 쪽 탐색.
다른 공간 인덱싱:
R-트리: 직사각형 범위 기반, DB용.
옥트리: 3D 균등 분할, 게임.
BSP 트리: 평면으로 분할, 렌더링.
쿼드트리: 2D 균등 분할.

## 문제 9

다음 공간과 AI의 연결을 분석하세요.

신경망의 기하학:
입력 공간 ℝⁿ → 은닉층 → 출력 ℝᵐ.

유니버설 근사 정리:
충분히 큰 신경망은 임의의 연속 함수를 근사.

(1) 신경망이 공간을 어떻게 변환하나요?
(2) 과적합의 기하학적 의미는?
(3) 기하학적 딥러닝(Geometric DL)이란?

### 힌트 1
신경망: 선형 변환 + 비선형 활성화.
층마다 공간을 "구부리고 늘림".

### 힌트 2
기하학적 딥러닝: 대칭성을 활용하는 신경망.
CNN = 평행이동 대칭, GNN = 그래프 대칭.

### 정답
신경망=공간변형·클래스분리 / 과적합=결정경계복잡 / 기하DL=대칭성이용

### 해설
신경망의 기하학적 해석:
각 층: 공간 변환 (선형+비선형).
분류: 클래스를 선형 분리 가능하게 공간 변형.
2D 예: XOR 문제 → 공간 접어서 선형 분리.
결정 경계(decision boundary):
선형 층: 초평면 경계.
비선형: 복잡한 곡면 경계.
과적합: 너무 복잡한 경계 → 새 데이터 일반화 실패.
기하학적 딥러닝 (브론스테인):
대칭성을 구조에 내재화.
CNN: 평행이동 동변(equivariant).
GNN: 그래프 순열 동변.
SE(3)-동변 네트워크: 3D 회전·이동 불변.
단백질 구조 예측(AlphaFold)에 응용.

## 문제 10

다음 미래 공간 이론을 탐구하세요.

끈 이론의 공간:
10차원 또는 11차원 시공간.
여분 6차원: 칼라비-야우 다양체로 컴팩트화.

양자 중력의 공간:
루프 양자 중력: 공간이 이산적 (스핀 네트워크).
홀로그래피 원리: n+1차원 중력 = n차원 양자장론.

(1) 칼라비-야우 다양체란?
(2) 홀로그래피 원리의 의미는?
(3) 공간의 본질은 연속인가 이산인가?

### 힌트 1
칼라비-야우: 복소수 3차원 (실수 6차원).
리치 곡률=0인 콤팩트 케흘러 다양체.

### 힌트 2
홀로그래피: 정보는 경계(2D)에 저장.
3D 세계가 2D 경계의 투영?

### 정답
칼라비야우=복소3차원Ricci=0 / 홀로그래피=n차원=n-1차원경계 / 공간본질=미해결

### 해설
칼라비-야우 다양체:
끈 이론: 10D 시공간 = 4D(우리세계) + 6D(작은).
6D: 칼라비-야우 다양체로 말려 있음.
수학적으로 아름다운 복소 다양체.
슈퍼대칭 보존, 리치 곡률=0.
홀로그래피 원리 (말다세나):
AdS/CFT 대응:
n+1차원 중력 = n차원 경계 양자장론.
"우주의 정보는 경계에 저장됨."
블랙홀 정보 역설 해결 단초.
공간의 연속 vs 이산:
고전 물리: 연속.
루프 양자 중력: 이산 (플랑크 스케일 픽셀).
끈 이론: 최소 길이 있음.
실험적 검증: 현재 불가능.
미해결: 가장 깊은 물리학 질문.
