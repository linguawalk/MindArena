## 문제 1

다음 세 방향의 투영도로
원래 입체를 복원하세요.

```
정면:     측면:     평면:
*-*-*     *-*       *-*-*
|   |     | |       |   |
*-*-*     | |       *-*-*
|   |     *-*       
*-*-*     
```

### 힌트 1
정면: 2×2 격자 → 폭=2, 높이=2.
측면: 1×2 → 깊이=1, 높이=2.
평면: 2×1 → 폭=2, 깊이=1.

### 힌트 2
모든 조건: 폭=2, 깊이=1, 높이=2.

### 정답
2×1×2 직육면체 (가로2, 깊이1, 높이2)

### 해설
삼각법 복원:
정면(폭×높이): 2×2.
측면(깊이×높이): 1×2.
평면(폭×깊이): 2×1.
→ 가로=2, 깊이=1, 높이=2.
2×1×2 직육면체.
부피=4, 겉넓이=2×(2×1+1×2+2×2)=2×8=16.

## 문제 2

다음 전개도를 접어서 만든 정육면체에서
면 A의 위쪽에 오는 면과
면 A의 오른쪽에 오는 면을 구하세요.

```
   [위]
[왼][A][오른]
   [아래]
   [뒤]
```

### 힌트 1
A를 앞면으로 놓겠습니다.
왼=왼쪽, 오른=오른쪽, 위=위, 아래=아래.

### 힌트 2
뒤는 아래의 아래에 있으므로
아래를 접으면 뒤가 됩니다.
뒤 = 앞(A)의 반대면.

### 정답
A의 위쪽 = 위 / A의 오른쪽 = 오른

### 해설
A를 앞면 기준:
위(A 위) → 접으면 위면 ✓.
왼(A 왼) → 접으면 왼면 ✓.
오른(A 오른) → 접으면 오른면 ✓.
아래(A 아래) → 접으면 아랫면.
뒤(아래 아래) → 접으면 A의 반대인 뒷면.
A의 위쪽 = 위면.
A의 오른쪽 = 오른면.

## 문제 3

다음 입체를 물에 넣었다가 꺼낸 뒤
물이 묻은 면의 넓이(겉넓이)를 구하세요.

입체: 3×2×1 직육면체.
조건: 3×2 면이 바닥에 닿아 있어
바닥면에는 물이 묻지 않습니다.

### 힌트 1
전체 겉넓이에서 바닥면(3×2=6)을 빼면 됩니다.

### 힌트 2
전체 겉넓이: 2×(3×2+2×1+3×1) = 2×11 = 22.
물 묻은 면: 22-6 = 16.

### 정답
16

### 해설
전체 겉넓이:
앞뒤: 2×(3×1) = 6.
좌우: 2×(2×1) = 4.
위아래: 2×(3×2) = 12.
합계: 22.
바닥면(3×2=6)은 물 안 묻음.
물 묻은 면: 22-6 = 16.

## 문제 4

다음 도형들을 조합하여
주어진 모양을 완성하세요.

목표 모양 (위에서 본 그림):
```
*-*-*-*
|X|X|X|
*-*-*-*
|X| |X|
*-*-*-*
|X|X|X|
*-*-*-*
```

사용 가능한 조각:
조각 A: L자 (3칸)
조각 B: L자 (3칸)
조각 C: 직선 (2칸)

X칸=8개를 A×2+C×1로 채울 수 있나요?

### 힌트 1
A+A+C = 3+3+2 = 8칸.
목표 X칸도 8개.

### 힌트 2
빈칸(가운데)을 피해 배치하세요.
L자 두 개를 대칭으로 배치.

### 정답
가능 (L자 2개로 모서리, 직선 1개로 가운데 행 배치)

### 해설
배치 방법 예시:
A1 (L자): 왼위모서리 (위행 왼2개 + 왼열 중간1개).
A2 (L자): 오른위모서리 (대칭).
C (직선): 아래행 가운데 2개.
남은 칸: 왼아래, 오른아래 모서리.
다양한 배치 방법이 가능합니다.
공간 조각 맞추기 = 테트로미노/폴리오미노 퍼즐.

## 문제 5

정육면체를 다음과 같이 잘랐을 때
만들어지는 조각의 수와 각 조각의 모양을 구하세요.

정육면체를 세 방향 (가로·세로·높이) 각각
정중앙으로 한 번씩 자릅니다.

### 힌트 1
가로로 한 번: 2조각.
세로로 한 번: 4조각.
높이로 한 번: 8조각.

### 힌트 2
각 조각의 크기는 원래의 1/8.

### 정답
8개의 동일한 작은 정육면체

### 해설
3방향 각 1회 절단:
가로 절단: 위아래 2조각.
세로 절단: 앞뒤로 각각 2 → 4조각.
높이 절단: 좌우로 각각 2 → 8조각.
총 8개의 작은 정육면체.
원래 부피의 1/8 크기.
원래 모서리 a → 각 조각 모서리 a/2.

## 문제 6

다음 입체의 겉넓이와 부피를 구하세요.

계단 모양 입체:
```
위에서 본 그림 (숫자=높이):
*-*-*
|2|1|
*-*-*
|1|1|
*-*-*
각 칸의 밑면 넓이=1
```

### 힌트 1
부피: 각 칸의 (밑넓이×높이) 합산.
2×1 + 1×1 + 1×1 + 1×1 = 5.

### 힌트 2
겉넓이: 윗면 + 옆면 + 바닥.
계단 효과로 옆면이 추가됩니다.

### 정답
부피=5 / 겉넓이=18

### 해설
부피: 2+1+1+1 = 5.
겉넓이 계산:
바닥(전체 4칸): 4.
윗면(각 높이 위): 4칸 = 4.
앞면: 높이2+높이1=3 (2칸 폭에서 각 높이).
뒷면: 1+1=2.
왼쪽: 2+1=3.
오른쪽: 1+1=2.
내부 계단 옆면 추가: 높이 차이 1짜리 2면.
합계: 4+4+3+2+3+2 = 18.

## 문제 7

다음 도형을 90도씩 4번 회전했을 때
원래 모양으로 돌아오지 않는 도형은?

A: 정사각형
B: 직사각형 (가로≠세로)
C: 정삼각형
D: 원

### 힌트 1
90도 4번 = 360도 = 원래 모양으로.
하지만 회전 중간에 다른 모양이 보이는 것은?

### 힌트 2
정삼각형을 90도 회전하면 원래와 다릅니다.
120도 회전해야 같아집니다.

### 정답
B (직사각형), C (정삼각형)

### 해설
A(정사각형): 90도 회전마다 동일 모양.
B(직사각형 가로≠세로): 90도 회전하면 다른 모양.
180도 회전해야 원래 모양 복원.
C(정삼각형): 120도 회전해야 동일.
90도 회전 시 다른 모양.
D(원): 어떤 각도로 회전해도 동일.
회전 대칭: n각형은 360/n도 회전마다 동일.
정삼각형: 120도, 정사각형: 90도, 정육각형: 60도.

## 문제 8

다음 조건으로 최대 부피의 직육면체를 설계하세요.

조건: 모든 모서리 길이의 합 = 24.
(직육면체는 4개의 동일한 세트 모서리가 있음)

### 힌트 1
모서리 합: 4(a+b+c) = 24.
→ a+b+c = 6.

### 힌트 2
산술-기하 평균 부등식:
a+b+c ≥ 3×(abc)^(1/3).
등호: a=b=c일 때.

### 정답
a=b=c=2인 정육면체, 부피=8

### 해설
a+b+c = 6 조건에서 부피 abc 최대화.
AM-GM 부등식: (a+b+c)/3 ≥ (abc)^(1/3).
6/3 ≥ (abc)^(1/3).
2 ≥ (abc)^(1/3).
abc ≤ 8.
등호 조건: a=b=c=2.
최대 부피 = 2×2×2 = 8.
→ 같은 모서리 합에서 정육면체가 부피 최대.

## 문제 9

다음 3D 미로를 통과하는 경로를 찾으세요.

```
층1 (아래):        층2 (위):
*-*-*-*           *-*-*-*
|S| |E|           | |E| |
*-*-*-*           *-*-*-*
| | | |           | | | |
*-*-*-*           *-*-*-*
(S=시작, E=탈출구)
(층1의 E 위치에 층2로 올라가는 계단)
(층2의 E가 출구)
```

이동 가능: 상하좌우 + 계단 오르기.
벽: 각 칸 사이의 일부 선.

### 힌트 1
S(층1 왼위) → 계단 위치(층1 오른위) → 층2 → 출구.

### 힌트 2
층1에서 오른쪽으로 이동 후 계단 올라가기.

### 정답
S → 오른쪽2칸(층1 E) → 층2로 올라가기 → 왼쪽1칸(층2 E=출구)

### 해설
3D 미로 경로:
1. 층1 시작점(S).
2. 오른쪽으로 2칸 이동.
3. 층1의 E(계단) 위치 도달.
4. 계단으로 층2 올라가기.
5. 층2에서 왼쪽 1칸.
6. 층2의 E(출구) 도달.
3D 미로: 층 간 이동이 추가된 공간 탐색.

## 문제 10

다음 종합 문제를 풀어보세요.

정육면체의 겉면을 빨강·파랑·노랑·초록·보라·주황
6가지 색으로 칠합니다.
마주보는 면은 같은 색 없이, 모든 면이 다른 색.

(1) 가능한 색칠 방법은 몇 가지인가요?
(2) 위면=빨강으로 고정했을 때 나머지 배치는?
(3) 회전으로 같은 모양이 되는 것은 같은 것으로 볼 때?

### 힌트 1
정육면체는 24가지 회전 대칭이 있습니다.

### 힌트 2
총 배치: 6! = 720.
회전 동일: 24개씩 묶임.
720/24 = 30.

### 정답
(1) 720가지 (2) 5!=120가지 (3) 30가지

### 해설
(1) 6개 면에 6가지 색을 배치: 6! = 720.
(2) 위면 고정 후 나머지 5면: 5! = 120.
(3) 회전 대칭 고려:
정육면체 회전 대칭 수 = 24.
구별되는 색칠 = 6!/24 = 30가지.
[계산: 한 면을 고정(6가지 선택), 그 면을 위로.
남은 4가지 측면 배치: 4!/4 = 6 (원형 배열).
아래 면: 1가지.
합계: 6×1×6×5/(중복없이) = 30.]
