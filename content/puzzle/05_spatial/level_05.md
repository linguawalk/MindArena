# 공간 퍼즐 5단계 — 공간 변환의 심화

> 기초 구간 중반입니다.
> 복합 변환과 3D 공간 분석을 탐구합니다.
> 두 가지 이상의 변환이 연속으로 일어날 때를 다룹니다.

## 문제 1

다음 도형에 두 가지 변환을 순서대로 적용하세요.

원래 점: A(1, 2).
변환 1: x축에 대해 반사.
변환 2: 오른쪽으로 3 이동.

최종 위치는?

### 힌트 1
x축 반사: (x, y) → (x, -y).
A(1,2) → (1,-2).

### 힌트 2
오른쪽 3 이동: x+3.
(1,-2) → (4,-2).

### 정답
(4, -2)

### 해설
변환 1 (x축 반사):
A(1,2) → A'(1,-2).
y 좌표의 부호 반전.
변환 2 (오른쪽 3 이동):
A'(1,-2) → A''(4,-2).
x 좌표에 3 더하기.
최종: (4,-2).
변환의 순서가 바뀌면 결과가 달라질 수 있습니다.

## 문제 2

다음 변환을 역순으로 적용하면 어떻게 될까요?

원래: 오른쪽 3 이동 후 x축 반사.
역순: x축 반사 후 오른쪽 3 이동.

A(1, 2)에 각각 적용했을 때 결과를 비교하세요.

### 힌트 1
원래 순서:
(1,2) → (4,2) → (4,-2).

### 힌트 2
역순:
(1,2) → (1,-2) → (4,-2).

### 정답
원래 순서: (4,-2) / 역순: (4,-2) — 이 경우 같음

### 해설
원래 순서:
(1,2) →[오른쪽3]→ (4,2) →[x축반사]→ (4,-2).
역순:
(1,2) →[x축반사]→ (1,-2) →[오른쪽3]→ (4,-2).
이 특수한 경우에는 결과가 같습니다.
일반적으로 변환의 순서가 중요합니다.
예: 회전 후 이동 ≠ 이동 후 회전.

## 문제 4

다음 정육면체 전개도에서
접었을 때 면 1과 마주보는 면은?

```
         [2]
   [3][1][4][6]
         [5]
```

### 힌트 1
1을 앞면으로 놓을 때:
3=왼쪽, 4=오른쪽, 2=위, 5=아래.

### 힌트 2
6은 4(오른쪽)의 오른쪽에 있습니다.
4를 오른쪽으로 접으면 6은 뒷면이 됩니다.

### 정답
면 1의 반대 = 면 6

### 해설
1을 기준으로:
2(위) ↔ 5(아래) 마주봄.
3(왼) ↔ 4를 넘어선 6 → 1(앞) ↔ 6(뒤) 마주봄.
4(오른) ↔ 3(왼) 마주봄.
따라서:
1 ↔ 6 (앞↔뒤).
2 ↔ 5 (위↔아래).
3 ↔ 4 (왼↔오른).

## 문제 5

다음 도형의 부피와 겉넓이를 구하세요.
(단위: cm)

```
    *-------*
   /|      /|  높이=2
  *-------* |
  |       | |
  | *-----|-*  깊이=3
  |/      |/
  *-------*
     가로=4
```

### 힌트 1
직육면체 부피 = 가로 × 세로 × 높이.

### 힌트 2
겉넓이 = 2×(가로×세로 + 세로×높이 + 가로×높이).

### 정답
부피=24cm³ / 겉넓이=52cm²

### 해설
부피 = 4×3×2 = 24cm³.
겉넓이:
앞뒤: 2×(4×2) = 16.
좌우: 2×(3×2) = 12.
위아래: 2×(4×3) = 24.
합계: 16+12+24 = 52cm².
또는: 2×(4×3 + 3×2 + 4×2) = 2×26 = 52cm².

## 문제 6

다음 L자 모양 도형을 3D로 확장하면
부피는 얼마인가요?
(두께=2, 각 칸=1×1)

```
위에서 본 모양:
*-*
|X|
*-*-*
|X|X|
*-*-*
```

### 힌트 1
위에서 본 면적 × 두께 = 부피.
X 칸의 수: 3개.

### 힌트 2
면적=3, 두께=2.

### 정답
6

### 해설
위에서 본 면적: X 칸 3개 = 3cm².
두께(높이): 2cm.
부피 = 면적 × 두께 = 3×2 = 6cm³.
이 방법은 균일한 높이의 입체에 적용합니다.
(프리즘의 부피 = 밑넓이 × 높이)

## 문제 7

다음 두 직육면체를 붙여서 만든
새 입체의 겉넓이를 구하세요.

직육면체 A: 2×2×2 (겉넓이=24).
직육면체 B: 2×2×2 (겉넓이=24).
붙이는 방법: 2×2 면을 완전히 맞붙임.

### 힌트 1
두 겉넓이 합: 24+24=48.
맞붙은 면 2개(각 4cm²)가 내부로.

### 힌트 2
48 - 2×4 = 48-8 = 40.

### 정답
40cm²

### 해설
A 겉넓이: 6×(2×2) = 24.
B 겉넓이: 6×(2×2) = 24.
합산: 48.
맞붙은 면: A의 2×2면 + B의 2×2면 = 각 4cm².
총 8cm²가 내부로 숨음.
새 겉넓이: 48-8 = 40cm².
결과: 4×2×2 직육면체 겉넓이 = 2×(8+8+16) = 40 ✓.

## 문제 8

다음 도형을 x=2 직선을 축으로
360도 회전시키면 어떤 입체가 될까요?

```
y
4|
3|  *---*
2|  |   |
1|  *---*
0+----------x
   0 1 2 3
```
(x=2를 축으로 회전, 사각형은 x=2~3, y=1~3)

### 힌트 1
x=2 축으로 회전하면 원통형 입체.
사각형의 각 점이 원을 그립니다.

### 힌트 2
x=2에서 x=3까지: 반지름 1인 원.
y=1에서 y=3까지: 높이 2.

### 정답
속이 찬 원기둥 (반지름=1, 높이=2)

### 해설
x=2를 축으로 회전체:
사각형(x=2~3, y=1~3):
x=2~3 부분이 회전 → 반지름 최대 1.
x=2가 축 → 반지름 = 3-2 = 1.
y=1~3 → 높이 = 2.
결과: 반지름 1, 높이 2인 원기둥.
부피 = π×1²×2 = 2π ≈ 6.28.

## 문제 9

다음 정육면체 8개로 이루어진 2×2×2 구조에서
각 작은 정육면체에 번호를 매기고
대각선 방향으로 반대편에 있는 쌍을 찾으세요.

```
위층:          아래층:
*-*-*          *-*-*
|1|2|          |5|6|
*-*-*          *-*-*
|3|4|          |7|8|
*-*-*          *-*-*
```
(앞왼=1, 앞오=2, 뒤왼=3, 뒤오=4,
아래앞왼=5, 아래앞오=6, 아래뒤왼=7, 아래뒤오=8)

### 힌트 1
공간 대각선: 가장 멀리 떨어진 꼭짓점끼리.
1은 위-앞-왼쪽 → 반대는 아래-뒤-오른쪽.

### 힌트 2
1↔8, 2↔7, 3↔6, 4↔5.

### 정답
1↔8 / 2↔7 / 3↔6 / 4↔5

### 해설
2×2×2 구조에서 공간 대각선 쌍:
1(위앞왼) ↔ 8(아래뒤오른): 모든 좌표 반대.
2(위앞오른) ↔ 7(아래뒤왼): 모든 좌표 반대.
3(위뒤왼) ↔ 6(아래앞오른): 모든 좌표 반대.
4(위뒤오른) ↔ 5(아래앞왼): 모든 좌표 반대.
3D 공간에서 점(x,y,z)의 반대는 (-x,-y,-z).

## 문제 10

다음 조건을 모두 만족하는 입체를 설계하세요.

조건:
①위에서 보면 정사각형.
②앞에서 보면 삼각형.
③옆에서 보면 삼각형.
④부피가 최대가 되도록.

### 힌트 1
위=정사각형, 앞=삼각형, 옆=삼각형.
어떤 입체가 이 조건을 만족하나요?

### 힌트 2
밑면이 정사각형이고 꼭대기가 점인 입체.

### 정답
사각뿔 (정사각형 밑면 피라미드)

### 해설
조건 분석:
위에서=정사각형: 밑면이 정사각형.
앞에서=삼각형: 옆에서 보면 삼각형.
옆에서=삼각형: 앞에서 보면 삼각형.
→ 사각뿔(밑면 정사각형, 꼭짓점 1개).
부피 최대화:
사각뿔 부피 = (1/3)×밑넓이×높이.
같은 밑면에서 높이가 높을수록 부피 증가.
밑면 a×a, 높이 h: V = a²h/3.
제약 조건 내에서 h를 최대로.
