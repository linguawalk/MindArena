# 공간 퍼즐 7단계 — 공간의 깊은 층위

> 발전 구간에 오신 것을 환영합니다.

## 문제 1

다음 도형들을 위상적으로 분류하세요.
(위상적 동치: 연속 변형으로 서로 변환 가능)

A: 원 (O)
B: 정사각형 (□)
C: 삼각형 (△)
D: 8자 모양 (∞)
E: 도넛 모양 (토러스)
F: 구

### 힌트 1
위상적 동치: 찢거나 붙이지 않고 늘리거나 구부려서 변환.
원, 정사각형, 삼각형은 모두 하나의 폐곡선.

### 힌트 2
8자: 자기 교차 있음 → 원과 다름.
도넛: 구멍 1개 → 구와 다름.

### 정답
{A,B,C}동치 / {D}별도 / {F}별도 / {E}별도

### 해설
위상적 동치 (2D):
A(원)=B(정사각형)=C(삼각형): 모두 단순 폐곡선.
연속 변형으로 서로 변환 가능.
D(8자): 자기 교차점 1개 → 다른 위상.
3D:
F(구)≠E(도넛): 구멍의 수가 다름.
위상 불변량: 오일러 특성수, 구멍의 수(genus).
구: genus=0, 도넛: genus=1.
위상수학: 연속 변형에서 변하지 않는 성질 연구.

## 문제 2

다음 뫼비우스 띠의 성질을 분석하세요.

```
뫼비우스 띠: 직사각형 종이의 한 끝을
180도 비틀어 붙인 것.

    ___________
   /           \
  /    (반틀림)  \
 /_______________\
```

(1) 면의 수는?
(2) 가장자리(경계)의 수는?
(3) 중간선을 따라 잘랐을 때 어떻게 되나요?

### 힌트 1
뫼비우스 띠: 안과 밖이 없는 단면 곡면.
개미가 표면을 따라 걸으면 양면을 모두 지남.

### 힌트 2
중간선 자르기: 분리되지 않고 2배 긴 띠가 됩니다.

### 정답
면=1개 / 가장자리=1개 / 자르면=2배긴띠(한번꼬임)

### 해설
뫼비우스 띠의 특성:
면: 1개 (안팎 구분 없는 단면 곡면).
일반 종이: 2면, 2가장자리.
뫼비우스: 1면, 1가장자리.
중간선 자르기:
분리되지 않고 한 개의 긴 띠.
길이 2배, 한 번 꼬인 띠.
1/3 지점 자르기: 두 개의 연결된 띠.
위상수학의 핵심 예시.
비방향성(non-orientable) 곡면의 대표.

## 문제 3

4차원 초입방체(하이퍼큐브, tesseract)의
꼭짓점, 모서리, 면, 셀의 수를 구하세요.

패턴 분석:
- 0차원 점: 꼭짓점 1
- 1차원 선분: 꼭짓점 2, 모서리 1
- 2차원 정사각형: 꼭짓점 4, 모서리 4, 면 1
- 3차원 정육면체: 꼭짓점 8, 모서리 12, 면 6, 셀 1

### 힌트 1
n차원에서 (n+1)차원으로:
꼭짓점 ×2, 모서리 = 이전 모서리×2 + 이전 꼭짓점.
면 = 이전 면×2 + 이전 모서리.

### 힌트 2
꼭짓점: 8×2=16.
모서리: 12×2+8=32.
면: 6×2+12=24.
셀: 1×2+6=8.

### 정답
꼭짓점=16 / 모서리=32 / 면=24 / 셀=8

### 해설
n차원 초입방체 패턴:
차원 | 꼭짓점 | 모서리 | 면  | 셀
0    |  1    |  0    |  0  | 0
1    |  2    |  1    |  0  | 0
2    |  4    |  4    |  1  | 0
3    |  8    |  12   |  6  | 1
4    |  16   |  32   |  24 | 8
오일러 특성수 검증:
3D: 8-12+6 = 2 ✓.
4D: 16-32+24-8 = 0.
4D 오일러 수 = 0 (짝수 차원).

## 문제 4

다음 그래프의 오일러 경로가
존재하는지 판단하세요.

```
그래프 A:       그래프 B:
*---*           *---*
|   |           |   |
*---*           *---*---*
                    |
                    *
```

오일러 경로: 모든 간선을 정확히 한 번씩 지나는 경로.

### 힌트 1
오일러 경로 조건:
홀수 차수 꼭짓점이 0개 (오일러 회로) 또는
정확히 2개 (오일러 경로).

### 힌트 2
그래프 A: 각 꼭짓점의 차수를 세어보세요.
그래프 B: 차수 3인 꼭짓점이 있나요?

### 정답
A=존재(오일러회로) / B=존재(홀수차수꼭짓점2개)

### 해설
그래프 A (정사각형):
각 꼭짓점 차수: 2, 2, 2, 2 (모두 짝수).
홀수 차수 = 0개 → 오일러 회로 존재.
그래프 B (T자형):
꼭짓점 차수: 2, 2, 3, 3, 1 (홀수 꼭짓점 2개).
→ 오일러 경로 존재 (시작=홀수, 끝=홀수).
홀수 차수가 3개 이상이면 오일러 경로 없음.
쾨니히스베르크 다리 문제: 오일러가 해결한 최초의 그래프 이론.

## 문제 5

다음 공간 퍼즐을 풀어보세요.

3×3×3 정육면체를 만들기 위해
1×1×3 막대 조각을 사용합니다.
정확히 9개의 막대로 3×3×3을 채울 수 있나요?

### 힌트 1
3×3×3 = 27개 단위.
막대 1개 = 3개 단위.
27/3 = 9개. 개수는 맞습니다.

### 힌트 2
색칠 논리: 3×3×3을 3색으로 칠해보세요.
각 막대는 반드시 3가지 색을 모두 포함합니다.

### 정답
가능

### 해설
9개의 1×1×3 막대로 3×3×3 채우기:
수학적 가능성:
27단위 / 3단위 = 9개 막대.
실제 배치:
층별로 3×3을 가로 3막대 × 3층 = 9막대.
또는 다양한 방향 혼합.
3색 체크:
x,y,z 각 방향으로 0,1,2 라벨링.
각 막대: (x+y+z) mod 3 값이 0,1,2를 하나씩 포함.
→ 9개 막대로 완전히 채울 수 있음.

## 문제 6

다음 정다면체들의 이중성을 분석하세요.

플라톤 입체:
- 정사면체 (4면): 꼭짓점 4, 모서리 6, 면 4
- 정육면체 (6면): 꼭짓점 8, 모서리 12, 면 6
- 정팔면체 (8면): 꼭짓점 6, 모서리 12, 면 8

정육면체와 정팔면체는 서로 "쌍대"입니다.
쌍대: 면의 중심을 연결하면 반대 입체가 됩니다.

(1) 정사면체의 쌍대는?
(2) 정십이면체(12면)의 쌍대는?
(3) 오일러 공식 V-E+F=2를 확인하세요.

### 힌트 1
쌍대 관계: 면↔꼭짓점이 교환됩니다.
정육면체(V=8,F=6) ↔ 정팔면체(V=6,F=8).

### 힌트 2
정사면체: V=F=4 → 자기 자신이 쌍대.

### 정답
정사면체=자기쌍대 / 정십이면체↔정이십면체 / V-E+F=2확인

### 해설
플라톤 입체 쌍대:
정사면체(4,6,4) ↔ 정사면체(자기 쌍대).
정육면체(8,12,6) ↔ 정팔면체(6,12,8).
정십이면체(20,30,12) ↔ 정이십면체(12,30,20).
오일러 공식 확인:
정사면체: 4-6+4 = 2 ✓.
정육면체: 8-12+6 = 2 ✓.
정팔면체: 6-12+8 = 2 ✓.
쌍대: V와 F가 교환, E는 동일.
5가지 플라톤 입체 = 3개의 쌍대 쌍.

## 문제 7

다음 프랙탈의 4단계 반복 후
둘레와 넓이를 구하세요.

코흐 눈송이:
1단계: 정삼각형 (변 길이=1).
2단계: 각 변을 3등분하고 가운데에 정삼각형 추가.

단계별 패턴:
- 변의 수: 3, 12, 48, ...
- 각 변의 길이: 1, 1/3, 1/9, ...
- 둘레: 3, 4, 16/3, ...

### 힌트 1
각 단계에서 변의 수 ×4, 각 변 길이 ×1/3.
둘레 = 변수 × 변길이.

### 힌트 2
n단계 둘레 = 3 × (4/3)^n.
n→∞: 둘레 → ∞.

### 정답
4단계 둘레=3×(4/3)⁴=768/81≈9.48 / 넓이는 수렴(유한)

### 해설
코흐 눈송이:
n단계 변수: 3×4^n.
n단계 변길이: (1/3)^n.
n단계 둘레: 3×(4/3)^n.
4단계: 3×(4/3)⁴ = 3×256/81 = 768/81 ≈ 9.48.
넓이 수렴:
추가되는 삼각형 넓이의 합 = 유한값.
초기 넓이(√3/4)에 수렴하는 급수를 더함.
역설: 둘레=무한, 넓이=유한.
프랙탈 차원: log4/log3 ≈ 1.26 (1과 2 사이).

## 문제 8

다음 좌표 변환 행렬을 분석하세요.

2D 회전 행렬 (θ도 회전):
```
[cos θ  -sin θ] [x]   [x']
[sin θ   cos θ] [y] = [y']
```

(1) θ=90도일 때 행렬은?
(2) 점 (3,1)을 90도 회전한 결과는?
(3) 회전 행렬의 행렬식(determinant)은?

### 힌트 1
cos90°=0, sin90°=1.
행렬: [[0,-1],[1,0]].

### 힌트 2
[0,-1][3]   [0×3+(-1)×1]   [-1]
[1, 0][1] = [1×3+0×1   ] = [ 3]

### 정답
(1) [[0,-1],[1,0]] (2) (-1,3) (3) 1

### 해설
90도 회전 행렬:
[[cos90, -sin90], [sin90, cos90]]
= [[0,-1],[1,0]].
점 (3,1) 적용:
x' = 0×3 + (-1)×1 = -1.
y' = 1×3 + 0×1 = 3.
결과: (-1, 3).
행렬식: cos²θ + sin²θ = 1.
회전 행렬의 행렬식 = 항상 1.
→ 크기(넓이) 변화 없음.
회전은 등거리 변환: 거리, 각도, 넓이 보존.

## 문제 9

다음 위상 문제를 분석하세요.

컵과 도넛은 위상적으로 동치입니까?

```
컵:          도넛:
 ___          ___
|   |        /   \
|   |       |  O  |
|   |        \___/
 ---
 |_|  ← 손잡이
```

### 힌트 1
위상적 동치: 찢거나 붙이지 않고 연속 변형.
구멍의 개수(genus)가 위상 불변량.

### 힌트 2
컵: 손잡이에 구멍 1개.
도넛: 구멍 1개.

### 정답
동치 (genus=1로 같음)

### 해설
위상적 동치 분석:
컵(손잡이 있는): 구멍 1개 (손잡이 구멍).
도넛(토러스): 구멍 1개.
둘 다 genus=1 → 위상적으로 동치.
컵을 늘리고 변형하면 도넛이 됩니다.
반면:
구(genus=0) ≠ 도넛(genus=1).
"위상학자는 커피잔과 도넛을 구별 못한다"는 농담.
위상 불변량: genus, 오일러 수, 기본 군.

## 문제 10

다음 공간 퍼즐을 설계하고 풀어보세요.

4×4 격자에서 시작점 S(1,1)에서 출발하여
목표점 E(4,4)까지 이동합니다.

조건:
①오른쪽 또는 위로만 이동.
②같은 칸을 두 번 방문 불가.
③(2,2), (3,3) 칸은 통과 불가.

(1) 가능한 경로의 수를 구하세요.
(2) 막힌 칸 없이 이동 가능한 경우의 수와 비교하세요.

### 힌트 1
막힌 칸 없는 경우: C(6,3) = 20가지.
(4×4 격자에서 오른쪽 3번, 위쪽 3번 = C(6,3))

### 힌트 2
(2,2)와 (3,3) 모두 통과하는 경로를 빼면 됩니다.
포함-배제 원리 적용.

### 정답
전체20 - 두칸모두통과하는경우 = 20-4=16가지

### 해설
전체 경로 (제약 없음): C(6,3) = 20.
(2,2)만 통과: S→(2,2)→E.
S→(2,2): C(2,1)=2.
(2,2)→E: C(4,2)=6.
합: 2×6=12.
(3,3)만 통과: S→(3,3)→E.
S→(3,3): C(4,2)=6.
(3,3)→E: C(2,1)=2.
합: 6×2=12.
(2,2)와 (3,3) 모두 통과:
S→(2,2)→(3,3)→E: 2×2×2=... 재계산 필요.
S→(2,2): 2, (2,2)→(3,3): C(2,1)=2, (3,3)→E: 2.
합: 2×2×2=8.
포함-배제: 12+12-8=16.
제약 있는 경로: 20-16=4가지.
