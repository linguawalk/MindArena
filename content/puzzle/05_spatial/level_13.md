## 문제 1

다음 공간과 음악의 관계를 탐구하세요.

음고 공간(pitch space):
음계를 원형으로 배열 (5도권, circle of fifths).
전조(modulation)는 이 공간에서의 이동.

리듬 공간:
박자를 n각형의 꼭짓점으로 표현.
리듬 패턴 = 꼭짓점 부분집합.

(1) 5도권의 기하학적 구조는?
(2) 유클리드 리듬(Euclidean rhythm)이란?
(3) 음악 이론의 공간적 표현이 주는 통찰은?

### 힌트 1
5도권: 12음을 원으로 배치, 완전5도 간격.
C-G-D-A-E-B-F#-Db-Ab-Eb-Bb-F-C.

### 힌트 2
유클리드 리듬: n개 위치에 k개 온음을 
최대한 고르게 배치.
브레센햄 알고리즘과 동일.

### 정답
5도권=Z12순환군 / 유클리드리듬=고른박자배치=브레센햄 / 공간화=음악구조명확화

### 해설
5도권의 수학:
Z₁₂: 12원소 순환군.
완전5도 = 7반음 이동.
12와 7은 서로소 → 모든 음 순환.
조성 조바꿈 = Z₁₂에서의 이동.
유클리드 리듬:
E(k,n): n스텝에 k온음을 최대 균등 배치.
E(3,8): [1,0,0,1,0,0,1,0] = 쿠바 트레스 리듬.
E(5,8): = 차차차.
E(2,3): = 탱고.
브레센햄 직선 알고리즘과 수학적 동치.
음악 공간화의 통찰:
조성 관계 = 기하 거리.
화음 진행 = 공간 경로.
리듬 복잡성 = 기하 불균일성.

## 문제 2

다음 공간과 생물학의 관계를 탐구하세요.

형태 발생(morphogenesis):
튜링 패턴 - 반응-확산 시스템으로
생물학적 패턴 생성.

```
activator u: ∂u/∂t = Du∇²u + f(u,v)
inhibitor v: ∂v/∂t = Dv∇²v + g(u,v)
```

(1) 얼룩말 줄무늬, 표범 점박이의 수학적 원리는?
(2) 뼈대 형성에서 공간 정보는 어떻게 전달되나요?
(3) 위상과 형태 발생의 관계는?

### 힌트 1
활성인자-억제인자: 짧은 범위 활성 + 긴 범위 억제.
초기 조건 + 경계 조건 → 패턴 결정.

### 힌트 2
위상 전사 인자(HOX 유전자): 위치 정보를 유전자 발현으로.
단순 자기 조직화 + 발달 제약.

### 정답
튜링패턴=반응확산방정식해 / HOX=위치정보유전자 / 위상=불변특성이발달안정성

### 해설
튜링 패턴(1952):
반응-확산 시스템의 공간 불안정성.
균일 상태 → 패턴 (자발적 대칭 깨짐).
조건: Dv/Du >> 1 (억제인자가 빠르게 확산).
패턴: 줄무늬, 점, 미로, 반점.
동물 패턴:
얼룩말: 1D 줄무늬 (긴 파장).
표범: 2D 점 (짧은 파장).
도마뱀 비늘: 6각형 타일링.
발달과 공간:
HOX 유전자: 몸의 앞뒤 축을 따라 발현.
위치 정보 → 기관 발달 지시.
척추동물 팔다리: 위치 의존적 발달.
위상의 역할:
발달 회로의 위상적 불변성.
변형에도 안정적인 발달 과정.

## 문제 3

다음 공간과 건축의 관계를 탐구하세요.

프렉탈 건축:
자연의 프랙탈 구조를 건축에 응용.

위상 최적화(topology optimization):
주어진 하중에서 최적 구조 자동 설계.

```
최적화 전:    최적화 후:
*-------*     *--*--*
|       |  →  |\ | /|
|       |     | \|/ |
*-------*     *--*--*
```

(1) 위상 최적화의 수학적 원리는?
(2) 프랙탈 구조가 건축에 강한 이유는?
(3) 자연 구조와 공학 구조의 공간적 공통점은?

### 힌트 1
위상 최적화: 밀도 분포 ρ(x) 최적화.
목적: 하중 하 변위 최소화.
제약: 재료 총량.

### 힌트 2
프랙탈: 같은 구조의 반복 → 스케일 독립적 강도.
나무, 뼈, 해변: 자연의 프랙탈 구조.

### 정답
위상최적화=밀도분포최적화 / 프랙탈=스케일독립강도 / 자연공학=최적공간분포공통

### 해설
위상 최적화:
SIMP 방법: 각 요소의 밀도 0~1.
목적 함수: 컴플라이언스 최소화.
제약: Σρᵢ ≤ V (재료 총량).
결과: 재료가 효율적으로 분포.
에어버스 항공기 부품: 40% 경량화.
프랙탈 강도:
하중이 분산되는 경로가 많음.
스케일 변화에도 동일한 강도 유지.
뼈 구조: 해면골(trabecular bone) = 최적화된 트러스.
에펠탑: 구스타브 에펠의 뼈 내부 구조 영감.
자연 vs 공학:
둘 다 최소 재료 + 최대 강도 추구.
자연: 진화로 최적화.
공학: 알고리즘으로 최적화.
공간 분포: 응력을 따르는 재료 배치.

## 문제 4

다음 공간과 도시 계획의 관계를 탐구하세요.

공간 통사론(Space Syntax):
도시 공간의 연결성과 통행 예측.

네트워크 분석:
도심 = 연결 중심성 높음.
골목 = 낮음.

(1) 도시 격자와 유기적 도시의 공간적 차이는?
(2) 보행자 흐름을 예측하는 수학적 방법은?
(3) 15분 도시 개념의 공간적 분석은?

### 힌트 1
연결 중심성: 얼마나 많은 경로가 이 노드를 통과하는가.
통행 예측: 공간 통사론 모델.

### 힌트 2
15분 도시: 일상 필요시설이 도보 15분 이내.
등시선(isochrone): 같은 시간 내 도달 가능 영역.

### 정답
격자=균등접근·유기=자연성장 / 공간통사론으로보행예측 / 15분도시=등시선분석

### 해설
도시 공간 유형:
격자형 (뉴욕, 서울 강남):
균등한 접근성.
방향성 명확.
단조로울 수 있음.
유기적 (파리 구시가, 서울 종로):
역사적 성장.
공간적 다양성.
길 찾기 복잡.
공간 통사론 (힐리어):
각 거리를 노드로 표현.
연결성(connectivity) 계산.
통합도(integration) = 도시 중심성 예측.
보행자 흐름과 높은 상관관계.
15분 도시:
모든 주민이 15분 내 모든 서비스 접근.
등시선 분석: 도보/자전거 거리 원.
다핵 도시 구조 필요.
공간 정의(spatial justice)와 연결.

## 문제 5

다음 공간과 예술의 관계를 탐구하세요.

원근법(perspective)의 수학:
르네상스의 투영 기하학 발견.

비유클리드 예술:
에셔(M.C. Escher)의 쌍곡 공간 타일링.
포앙카레 원판 모델.

```
에셔의 Circle Limit 시리즈:
원 중심에서 경계로 갈수록 작아지지만
쌍곡 기하에서는 모두 같은 크기.
```

(1) 원근법이 투영 기하학과 연결되는 방식은?
(2) 에셔의 무한 반복 패턴의 수학적 기반은?
(3) 예술이 수학적 공간 이론을 어떻게 시각화하는가?

### 힌트 1
원근법: 소실점 = 평행선의 만남.
투영 기하: 무한원점이 포함된 공간.

### 힌트 2
에셔 Circle Limit: 쌍곡 기하의 포앙카레 원판 모델.
프리즈 군, 벽지 군: 평면 대칭 분류.

### 정답
원근법=투영기하·소실점=무한원점 / 에셔=쌍곡타일링·포앙카레원판 / 예술=수학시각화

### 해설
원근법의 수학:
소실점: 평행선들이 모이는 점.
투영 기하: 무한원점을 공식으로 포함.
교차비(cross-ratio): 투영 불변량.
르네상스 화가들이 무의식적으로 투영 기하 발견.
에셔와 수학:
프리즈 군: 1D 반복 패턴 7가지.
벽지 군: 2D 평면 대칭 17가지.
에셔의 패턴: 벽지 군의 시각적 표현.
Circle Limit: 쌍곡 기하의 포앙카레 원판.
경계에 가까울수록 쌍곡 거리 증가.
예술과 수학:
예술: 직관으로 수학적 구조 발견.
수학: 예술에서 엄밀한 이론 추출.
상호 영감: 복잡한 추상 개념의 시각화.

## 문제 6

다음 공간과 게임의 관계를 탐구하세요.

게임 공간 설계:
레벨 디자인의 공간 이론.
길목(chokepoint), 개방 구역, 은폐 지점.

위상학적 게임 공간:
이동 그래프: 노드=위치, 간선=이동 가능.
게임 진행 = 그래프 탐색.

(1) 미로 게임의 최적 설계 원리는?
(2) 오픈 월드 게임의 공간 구조는?
(3) 프로시저럴 생성(procedural generation)의 수학은?

### 힌트 1
완벽한 미로: 스패닝 트리 (모든 노드 연결, 사이클 없음).
불완전한 미로: 사이클 추가 → 다중 경로.

### 힌트 2
프로시저럴 생성: 알고리즘으로 콘텐츠 자동 생성.
노이즈 함수 (펄린 노이즈): 자연스러운 지형.

### 정답
미로=스패닝트리기반 / 오픈월드=계층적볼로노이 / 프로시저럴=노이즈+L시스템

### 해설
미로 설계:
완벽한 미로: 스패닝 트리.
단 하나의 경로 → 논리 퍼즐.
재귀적 분할, 프림 알고리즘, 크루스칼.
루프 추가: 여러 경로 → 탐색적 게임.
오픈 월드 공간:
계층적 구조: 지역 → 도시 → 건물.
볼로노이: 각 지역의 영향권.
지형 생성: 프랙탈 노이즈.
흥미 곡선: 긴장/이완의 공간적 배치.
프로시저럴 생성:
펄린 노이즈: 지형, 구름, 텍스처.
L-시스템: 식물, 자연 구조.
셀룰러 오토마타: 동굴, 도시.
보로노이: 마을 위치, 영역 분할.
수학: 결정론적 알고리즘 + 시드 값.

## 문제 7

다음 공간과 언어의 관계를 탐구하세요.

의미 공간(semantic space):
단어를 고차원 벡터로 표현.
word2vec: 단어 = 300차원 벡터.

공간적 관계:
"왕" - "남자" + "여자" ≈ "여왕"
(벡터 덧셈으로 유추)

(1) 의미 공간의 기하학적 특성은?
(2) 문장 임베딩과 문단의 공간은?
(3) 언어 모델의 내부 표현은 어떤 공간인가요?

### 힌트 1
word2vec: 동시 출현 행렬의 저차원 투영.
주성분 방향: 의미적으로 의미 있는 축.

### 힌트 2
언어 모델: 트랜스포머의 어텐션 = 고차원 공간에서의 벡터 조작.

### 정답
의미공간=고차원유클리드 / 문장임베딩=문장의벡터 / LLM내부=복잡한고차원다양체

### 해설
word2vec 기하:
각 단어 = 300차원 벡터.
유사한 단어 = 가까운 벡터.
관계 = 벡터 차이 방향.
"왕-남자+여자=여왕": 성별 벡터 덧셈.
의미 공간 특성:
국소 구조: 동의어·유의어 클러스터.
전역 방향: 의미론적 축 (성별, 시제, 크기).
다양체 가설: 언어는 고차원 공간의 저차원 다양체.
언어 모델 내부:
트랜스포머: 각 토큰 = 고차원 벡터.
어텐션: 벡터 간 관계 계산.
층마다 다른 추상화 수준.
해석 가능성: 내부 공간의 구조 이해 연구 중.
다양체 구조: 문법, 의미, 맥락의 층위.

## 문제 8

다음 공간과 의학의 관계를 탐구하세요.

의료 영상의 공간:
CT: 3D 밀도 함수.
MRI: 조직 특성 3D 맵.
fMRI: 4D (공간+시간) 활성도.

형태 측정학(morphometrics):
뇌 구조의 모양 변화를 측정.

(1) 의료 영상의 등록(registration)이란?
(2) 통계적 형태 모델(SSM)이란?
(3) 위상 데이터 분석이 의학에서 하는 역할은?

### 힌트 1
등록: 두 영상을 같은 좌표계로 정렬.
강체 변환 + 비강체 변형.

### 힌트 2
SSM: 훈련 데이터에서 형태 변화의 주성분 추출.
평균 형태 + 주성분으로 표현.

### 정답
등록=좌표정렬·강체+비강체 / SSM=형태PCA / TDA=암조직위상분석

### 해설
의료 영상 등록:
강체(rigid): 회전+이동 (뼈).
어파인(affine): 선형 변환.
탄성(elastic): 비선형 변형 (뇌).
목적: 시간별 변화 추적, 아틀라스 구성.
통계적 형태 모델:
n명의 뼈/기관 메시 → 대응점 찾기.
PCA: 평균 + 주성분 변형.
새로운 형태 = 평균 + Σαᵢ×PCᵢ.
임상 응용: 수술 계획, 진단.
위상 데이터 분석 응용:
암 조직: 세포 배열의 위상적 특성.
영구 호몰로지로 종양 특성 분류.
뇌 네트워크: 기능적 연결의 위상.
단백질 구조: 구멍과 포켓 탐지 (약물 표적).

## 문제 9

다음 공간과 사회의 관계를 탐구하세요.

공간적 불평등:
접근성 지도: 의료·교육·교통의 공간 분포.
식품 사막(food desert): 신선 식품 접근 불가 지역.

지리 가중 회귀(GWR):
공간적으로 변하는 회귀 계수.
"같은 변수가 지역마다 다른 효과."

(1) 공간적 자기상관(spatial autocorrelation)이란?
(2) 모란 지수(Moran's I)는?
(3) 공간 정의(spatial justice)의 수학적 측도는?

### 힌트 1
공간적 자기상관: 가까운 지역이 비슷한 값.
"모든 것은 다른 모든 것과 관련되지만, 가까운 것이 더 관련됨." (토블러 제1법칙)

### 힌트 2
모란 I: -1(완전 음의 상관) ~ +1(완전 양의 상관).
0: 공간 패턴 없음.

### 정답
공간자기상관=인접유사성 / 모란I=공간상관계수 / 공간정의=접근성균등화측도

### 해설
공간적 자기상관:
토블러 제1법칙: 인접 지역의 유사성.
긍정적 자기상관: 부촌 옆에 부촌.
부정적: 체커보드 패턴 (드문 경우).
모란 지수 I:
I = n/(W) × Σᵢ Σⱼ wᵢⱼ(xᵢ-x̄)(xⱼ-x̄) / Σᵢ(xᵢ-x̄)².
wᵢⱼ: 공간 가중치 행렬.
범위: -1 ~ +1.
공간 정의 측도:
접근성 지표: 평균 이동 거리.
지니 계수: 공간 분포의 불평등.
등시선 커버리지: 서비스 도달 인구 비율.
GWR: 지역별 다른 정책 효과 파악.

## 문제 10

마스터 구간의 시작으로
자신만의 공간 퍼즐을 설계하세요.

다음 조건을 만족하는 공간 퍼즐:
①실생활과 깊이 연결됨.
②직관과 다른 공간적 결과.
③풀이 과정에서 핵심 공간 개념 학습.
④10~12단계 수준의 난이도.
⑤명확한 정답이 있을 것.

### 힌트 1
반직관적 공간 결과:
고차원, 위상, 곡률, 프랙탈 관련.

### 힌트 2
실생활: 건축, 의학, AI, 게임, 도시.

### 정답
개인 창작 (다양한 답 가능)

### 해설
예시 퍼즐 설계: "엘리베이터 최적 배치"

문제:
100층 건물에 엘리베이터 3대를 배치합니다.
각 층에서 엘리베이터까지의 평균 이동 층수를
최소화하려면 어떻게 배치해야 하나요?

(모든 층이 동일한 이용 빈도라고 가정)

수학:
3대를 1,2,3 위치에 배치 시
각 층은 가장 가까운 엘리베이터 이용.
보로노이 분할 (1D).
목적 함수: Σ min(|층-e₁|, |층-e₂|, |층-e₃|).

풀이:

평균 이동: 약 8.3층.
이것이 1D 등주 문제의 이산 버전.

포함 개념:
보로노이 다이어그램, 등주 문제, 최적화, 1D 공간 분할.
