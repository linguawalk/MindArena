# 전략 퍼즐 2단계 — 게임 트리와 역방향 분석

> 게임의 끝에서부터 거꾸로 생각합니다.
> 상대방의 최선 수를 예측하고
> 내 최선 수를 결정합니다.

## 문제 1

돌 나누기 게임:
12개의 돌을 두 더미로 나눕니다.
두 사람이 번갈아 한 더미에서만 돌을 가져갑니다.
(한 번에 1개 이상, 더미 전체 가능)
마지막 돌을 가져가는 사람이 집니다.

현재 상태: (5, 7) 두 더미.
선공의 최선 수는?

### 힌트 1
님(Nim) 게임입니다.
두 더미의 XOR 값을 계산하세요.

### 힌트 2
5 XOR 7 = 101 XOR 111 = 010 = 2.
XOR ≠ 0이면 선공이 유리합니다.

### 정답
5개 더미에서 3개 제거 → (2, 7) 또는 7개 더미에서 5개 제거 → (5, 2)

### 해설
님 게임 전략:
XOR(5,7) = 2 ≠ 0 → 현재 플레이어 유리.
목표: XOR = 0이 되는 상태로 만들기.
5 XOR 7 = 2.
5를 5 XOR 2 = 7로? 아니면...
7을 7 XOR 2 = 5로 만들기:
7개 더미에서 2개 제거 → (5, 5).
5 XOR 5 = 0 → 상대방이 불리한 상태.
검증: (5,5)에서 상대가 5개 더미에서 k개 제거.
→ (5, 5-k), XOR = k ≠ 0.
→ 내가 대칭으로 대응: (5-k, 5-k), XOR = 0.
→ 항상 이길 수 있음.

## 문제 2

스위치 게임:
1, 0, 1, 0, 1 다섯 개의 스위치가 있습니다.
두 사람이 번갈아 연속된 두 스위치를 동시에 뒤집습니다.
(01↔10, 11→00, 00→11)
모든 스위치가 0이 되면 그 상태를 만든 사람이 집니다.

현재 상태: 1, 0, 1, 0, 1.
이 상태에서 선공이 이길 수 있나요?

### 힌트 1
가능한 이동: (1,2), (2,3), (3,4), (4,5) 위치 뒤집기.

### 힌트 2
상태를 이진수로 보고 가중치를 계산해보세요.
님 값(Grundy value) 개념이 필요합니다.

### 정답
가능한 이동을 따라 분석 필요 (복잡한 위치)

### 해설
스위치 게임 분석:
현재: 1,0,1,0,1.
가능한 이동:
(1,2): 1,0→0,1 → 0,1,1,0,1.
(2,3): 0,1→1,0 → 1,1,0,0,1.
(3,4): 1,0→0,1 → 1,0,0,1,1.
(4,5): 0,1→1,0 → 1,0,1,1,0.
각 상태에서 추가 분석 필요.
목표 상태 0,0,0,0,0에 도달하는 최단 경로 탐색.
스프라그-그런디 이론으로 님 값 계산.
게임 트리의 복잡성: 완전한 분석은 깊은 탐색 필요.

## 문제 3

3단계 결정 게임:
A가 먼저 선택: 왼쪽(L) 또는 오른쪽(R).
L 선택 시 B가 선택: 위(U) 또는 아래(D).
R 선택 시 B가 선택: 위(U) 또는 아래(D).

결과 (A의 이익, B의 이익):
L-U: (3, 2)
L-D: (1, 4)
R-U: (2, 3)
R-D: (4, 1)

역방향 귀납으로 균형을 구하세요.

### 힌트 1
B의 최적 선택 먼저:
L일 때: B는 U(2) vs D(4) → D 선택.
R일 때: B는 U(3) vs D(1) → U 선택.

### 힌트 2
B의 반응을 알면 A의 선택:
L → (1, 4) (B가 D 선택).
R → (2, 3) (B가 U 선택).
A는 2 > 1이므로 R 선택.

### 정답
A: R, B: U → 결과 (2, 3)

### 해설
역방향 귀납(Backward Induction):
1단계(끝): B의 최적 반응.
L일 때: D(4>2) 선택.
R일 때: U(3>1) 선택.
2단계(처음): A의 예측.
L 선택 시 결과: (1, 4) (B가 D 선택).
R 선택 시 결과: (2, 3) (B가 U 선택).
A는 2 > 1이므로 R 선택.
균형: (R, U) → 보수 (2, 3).
부분게임 완전 균형(Subgame Perfect Equilibrium).

## 문제 4

카드 게임 전략:
1~9까지의 카드가 있습니다.
두 사람이 번갈아 카드를 하나씩 가져갑니다.
세 장의 합이 15가 되면 승리.
(가져간 카드 모두 사용 가능)

선공의 최선 전략은?

### 힌트 1
이것은 틱택토와 동형입니다.
3×3 마방진을 생각해보세요.

### 힌트 2
마방진:
2 7 6
9 5 1
4 3 8
어떤 행·열·대각선도 합=15.

### 정답
마방진의 틱택토와 같이 중앙(5) 선택

### 해설
마방진과 틱택토의 동형:
```
2|7|6
9|5|1
4|3|8
```
세 수의 합=15 ↔ 한 줄을 모두 차지.
세 장의 합이 15가 되는 조합은 정확히 8가지:
(마방진의 8개 라인).
따라서 이 게임 = 틱택토.
최선 전략: 틱택토와 동일 (5 먼저, 공격과 방어 병행).
완벽한 플레이: 무승부.
이것이 게임 동형(isomorphism)의 예시.

## 문제 5

폭탄 제거 게임:
```
[B][_][_]
[_][B][_]
[_][_][B]
```
B는 폭탄, _는 빈칸.
두 사람이 번갈아 빈칸을 선택합니다.
선택한 칸과 인접한 폭탄이 있으면 그 폭탄이 제거됩니다.
마지막 폭탄을 제거한 사람이 집니다.

선공의 최선 수는?

### 힌트 1
각 빈칸을 선택했을 때 제거되는 폭탄 수를 계산하세요.

### 힌트 2
중앙(2,2)칸: 대각선 폭탄 B(1,1), B(3,3)에 인접? 아니면...
격자 인접: 상하좌우.

### 정답
폭탄 2개와 인접한 칸 선택

### 해설
격자 위치 (행, 열):
B들: (1,1), (2,2), (3,3).
빈칸들: (1,2),(1,3),(2,1),(2,3),(3,1),(3,2).
인접(상하좌우)으로 제거되는 폭탄:
(1,2): B(1,1)에 인접 → 1개 제거.
(2,1): B(1,1), B(2,2)에 인접 → 2개 제거.
(2,3): B(2,2), B(3,3)에 인접 → 2개 제거.
(3,2): B(3,3)에 인접 → 1개 제거.
최선: (2,1) 또는 (2,3) 선택으로 2개 제거.
폭탄이 1개 남으면 상대가 마지막 폭탄 제거 → 상대 승.
전략: 짝수개씩 제거 유도.

## 문제 6

점 연결 게임:
3×3 격자의 꼭짓점들을 인접 꼭짓점과 선으로 연결.
선을 그어 정사각형 한 변을 완성하는 사람이
그 정사각형 점수를 얻음.
모든 선이 그어지면 총점 비교.

선공의 최선 전략은 무엇인가요?

### 힌트 1
상자가 완성되기 직전(3변)인 상자를 만들지 않는 것이 중요합니다.

### 힌트 2
불가피하게 3변 상자가 생기면
연속 완성(chain)으로 손해를 최소화합니다.

### 정답
3변 상자 만들지 않기 + 연속 연결(chain) 전략

### 해설
점 연결 게임(Dots and Boxes) 전략:
초반: 3변 상자를 만들지 않기.
한 변을 추가하면 상대가 완성 → 상대 점수.
중반: 불가피하게 상자를 열어야 할 때
가장 작은 연속 체인을 선택.
연쇄 전략:
긴 체인을 상대에게 줄 때
마지막 2개는 남기고 넘김 (sacrifice).
상대가 2개 가져가면 다음 큰 체인을 내가 선점.
짝수/홀수 체인의 전략적 차이가 핵심.
완전한 분석: 조합 게임 이론.

## 문제 7

신호 전략:
두 팀이 비밀 전달 게임을 합니다.
A팀(송신)이 숫자 1~8 중 하나를 B팀(수신)에 전달.
1번의 질문(예/아니오)만 허용.
B팀이 맞히면 A팀 승, 틀리면 B팀 승.

A팀의 최선 전략은?

### 힌트 1
1번의 질문으로 8개 중 하나를 특정할 수 없습니다.

### 힌트 2
질문의 형식을 창의적으로 사용할 수 있습니다.
예: "숫자가 1~4 사이인가요?" (2진 탐색)

### 정답
한 번의 예/아니오로 8개를 구분 불가. 최선은 50% 확률.

### 해설
정보 이론적 분석:
1번의 예/아니오 질문: 1비트 정보.
8개 구분: log₂(8) = 3비트 필요.
불가능: 1번 질문으로 8개 구분.
최선 전략:
"숫자가 1~4인가?" → 4개로 압축.
하지만 여전히 4개 중 하나 = 알 수 없음.
현실 해결:
3번 질문으로 완전 구분 가능 (이진 탐색).
1번만으로는 최대 2개 구분.
게임 설계의 정보 이론적 한계.

## 문제 8

최소최대 전략:
게임 트리:
```
        A
       / \
      B   C
     /\ /\
    1 2 3 4
```
A는 최대화 플레이어, B와 C는 최소화 플레이어.
리프 값: B의 자식=1,2 / C의 자식=3,4.

A의 최적 선택은?

### 힌트 1
최소화 플레이어 B: min(1,2) = 1.
최소화 플레이어 C: min(3,4) = 3.

### 힌트 2
A는 max(1,3) = 3 선택.
A는 C 방향으로.

### 정답
A: C 방향 → 결과 3

### 해설
미니맥스(Minimax) 알고리즘:
리프 단계: 값 확정 (1,2,3,4).
B 단계 (최소화): min(1,2) = 1.
C 단계 (최소화): min(3,4) = 3.
A 단계 (최대화): max(1,3) = 3.
A는 C를 선택하고 값 3을 보장.
C는 3을 선택 (min(3,4)=3).
체스·바둑 AI의 기본 알고리즘.
알파-베타 가지치기: 불필요한 탐색 제거.

## 문제 9

협력 vs 경쟁:
죄수의 딜레마:
```
           B: 협력    B: 배신
A: 협력    (3, 3)    (0, 5)
A: 배신    (5, 0)    (1, 1)
```
단 1회 게임에서 A의 최선 전략은?

### 힌트 1
B가 협력하든 배신하든 A의 배신이 더 나은 결과를 줍니다.

### 힌트 2
B 협력 시: A 협력(3) < A 배신(5).
B 배신 시: A 협력(0) < A 배신(1).

### 정답
배신 (지배 전략)

### 해설
지배 전략(Dominant Strategy):
어떤 상대 전략에도 최선인 전략.
B 협력: A 배신(5) > A 협력(3).
B 배신: A 배신(1) > A 협력(0).
→ 배신이 지배 전략.
내쉬 균형: (배신, 배신) → (1, 1).
역설:
둘 다 협력했으면 (3,3)으로 더 좋은 결과.
하지만 개인 합리성 → 사회 비효율.
현실 응용:
군비 경쟁, 환경 문제, 공공재 딜레마.
반복 게임에서 협력 가능 (팃포탯).

## 문제 10

마지막 수 전략:
바둑판 위의 간단한 게임:
```
A . .
. . .
. . B
```
A는 왼쪽 위, B는 오른쪽 아래.
번갈아 상하좌우로 1칸 이동.
같은 칸에 만나면 그 칸으로 이동한 사람이 집니다.
(A가 먼저)

A의 최선 전략은?

### 힌트 1
A와 B의 맨해튼 거리를 계산하세요.
거리 = |행 차이| + |열 차이|.

### 힌트 2
거리가 홀수: 먼저 이동한 사람이 유리.
거리가 짝수: 나중에 이동한 사람이 유리.

### 정답
거리=4(짝수)이므로 B가 이론적으로 유리. A는 거리를 홀수로 만들기 불가.

### 해설
맨해튼 거리:
A(0,0), B(2,2): |2-0|+|2-0| = 4 (짝수).
추격 게임 분석:
거리가 홀수: 추격자(A)가 한 수만에 도달 가능.
거리가 짝수: 도망자(B)가 항상 같은 거리 유지 가능.
B는 A가 가까워질 때마다 반대 방향으로.
A는 거리=4 (짝수)이므로 불리.
무한 격자라면 B는 영원히 도망 가능.
유한 격자(3×3): 모서리·가장자리에서 선택지 제한.
경계 조건이 전략을 복잡하게 만듦.
