## 문제 1

포커 블러핑:
A가 강한 패(H)를 가질 확률 1/3,
약한 패(L)를 가질 확률 2/3.
강한 패: 항상 베팅.
약한 패: 확률 p로 블러핑(베팅).

B는 A의 베팅을 보고 콜(call) 또는 폴드(fold).
콜 시: H이면 B 손실 2, L이면 B 이득 1.

B의 최적 전략은?

### 힌트 1
B의 기댓값 (콜 시):
P(H|베팅)×(-2) + P(L|베팅)×(1).

### 힌트 2
P(H|베팅) = P(베팅|H)×P(H) / P(베팅).
P(베팅) = 1×(1/3) + p×(2/3).

### 정답
p=1/2이면 B 무차별, p<1/2이면 폴드, p>1/2이면 콜

### 해설
베이즈 업데이트:
P(H|베팅) = (1×1/3) / (1/3+2p/3) = 1/(1+2p).
P(L|베팅) = (p×2/3) / (1/3+2p/3) = 2p/(1+2p).
B의 기댓값:
E(콜) = -2/(1+2p) + 2p/(1+2p) = (2p-2)/(1+2p).
E(콜) = 0: 2p-2=0 → p=1.
재계산: E = -2×P(H|베팅) + 1×P(L|베팅).
= -2/(1+2p) + 2p/(1+2p) = (2p-2)/(1+2p).
p=1이면 E=0 (무차별).
p<1이면 E<0 (폴드 유리).
A의 최적 블러핑: p=1/2로 균형.

## 문제 2

보험 시장 역선택:
고위험자(H): 사고 확률 0.5, 보험 수요 높음.
저위험자(L): 사고 확률 0.1, 보험 수요 낮음.
보험사: 타입 구분 불가.

균등 요율 보험(all-or-nothing):
기대 손실 기반 가격 설정.

고위험자 50%, 저위험자 50%이면
공정 요율은? 그리고 무슨 일이 일어나나요?

### 힌트 1
평균 사고 확률 = 0.5×0.5 + 0.5×0.1 = 0.3.
공정 요율 = 손실×0.3.

### 힌트 2
저위험자: 0.3 요율 > 0.1 실제 위험 → 탈퇴.
고위험자만 남음 → 요율 0.5로 상승.

### 정답
공정요율=0.3×손실 → 저위험자탈퇴 → 시장붕괴(레몬시장)

### 해설
역선택 스파이럴:
1. 요율 0.3 설정.
2. 저위험자(0.1): 과다 지불 → 탈퇴.
3. 고위험자만 남음 → 요율 0.5로 상승.
4. 비용 증가 → 저위험자 추가 탈퇴.
5. 극단적 경우: 고위험자만 남음.
분리 균형 해결책:
고위험자: 높은 프리미엄 + 완전 보장.
저위험자: 낮은 프리미엄 + 부분 보장 (공제).
고위험자가 저위험자 플랜을 선택 불리하게 설계.
실제: 자기선택(self-selection) 메뉴.

## 문제 3

취업 협상 게임:
지원자 타입: 고능력(H, 확률 0.6) 또는 저능력(L, 확률 0.4).
고능력자 기여 가치: 100.
저능력자 기여 가치: 40.
기업은 타입 모름.

기업이 모든 지원자에게 같은 임금 제안.
기업의 최적 임금 제안은?

### 힌트 1
임금 W를 제안 시 모두 수락한다면:
기업 이익 = 0.6×(100-W) + 0.4×(40-W).

### 힌트 2
기업 이익 = 76 - W.
W > 76이면 음수.
최대 W = 76 (기대 가치).

### 정답
W = 76 (기대 가치 = 0.6×100+0.4×40)

### 해설
기대 가치 계산:
E[가치] = 0.6×100 + 0.4×40 = 60+16 = 76.
기업 최적: W = 76.
이익 = 0 (경쟁 시장 가정).
정보 비대칭의 결과:
고능력자: W=76 < 100 → 임금 손해.
저능력자: W=76 > 40 → 임금 이익.
교차 보조: 저능력자가 고능력자 희생으로 이익.
해결:
분리 계약 설계.
교육 신호 (스펜스).
성과 기반 보수.
확률 갱신:
시험·면접으로 타입 추정 → 베이즈 업데이트.

## 문제 4

다단계 베이지안 게임:
A: 강한 타입(S, 확률 2/3), 약한 타입(W, 1/3).
B는 A의 타입을 모름.
A의 전략: 강하게 행동(F) 또는 약하게 행동(Q).

보수:
S-F vs B: (2, -1), S-Q vs B: (1, 0).
W-F vs B: (-1, 1), W-Q vs B: (0, 0).
B의 대응: 도전(D) 또는 수용(A).

반분리 균형이 존재하나요?

### 힌트 1
S 타입: F 선택 (강하게 행동이 우월).
W 타입: 섞기 (F와 Q를 확률적으로).

### 힌트 2
B의 믿음 갱신:
F를 보면 P(S|F) 업데이트.
W 타입이 F를 q 확률로 선택.

### 정답
반분리 균형: S=F 확실, W=q 확률로 F

### 해설
반분리(semi-separating) 균형:
S 타입: 항상 F (강하게 행동).
W 타입: q로 F, (1-q)로 Q.
B의 믿음 갱신 (F 관찰 시):
P(S|F) = (2/3×1) / (2/3×1 + 1/3×q) = 2/(2+q).
B의 최적 대응:
D vs A 결정: P(S|F)×(-1) + P(W|F)×(1) ≷ 0.
균형 조건으로 q 결정.
완전 분리 vs 반분리:
완전 분리: 각 타입이 구별되는 신호.
반분리: 일부만 구별.
혼합: 구별 안 됨.

## 문제 5

스크리닝 설계:
독점 기업이 두 타입의 소비자에게 판매.
고가치 소비자(H, 50%): 가치=100.
저가치 소비자(L, 50%): 가치=50.
생산 비용=0.

기업이 가격을 두 개 설정: PH, PL.
(수량은 각 1단위)
IC 제약: H가 저가 플랜 선택 안 하도록.
IR 제약: L이 구매하도록.

최적 가격은?

### 힌트 1
정보 비대칭 없으면: PH=100, PL=50 → 이익=75.
정보 비대칭: H가 L 플랜 선택 가능.

### 힌트 2
IC: H는 PH 플랜에서 이익 ≥ PL 플랜에서 이익.
100-PH ≥ 100-PL → PH ≤ PL. 이는 항상 만족.
실제 IC: 100-PH ≥ 50-PL (H가 L 플랜 선택 유인).

### 정답
PL=50, PH=100-정보지대=100-(100-50)=50? 재계산 필요

### 해설
최적 스크리닝:
IR(L): 50-PL ≥ 0 → PL ≤ 50.
최적: PL = 50.
IC(H): 100-PH ≥ 100-PL = 50 → PH ≤ 50.
정보 지대: H는 PH=50에서 50 얻음.
기업 이익: 50+50=100 (각 1명에게 50).
완전 정보: PH=100, PL=50 → 이익=150.
정보 비대칭 비용: 150-100=50.
최적 메뉴:
PL을 낮춰 L 참여 유도.
H에게 정보 지대 허용.
효율성-지대 추출 트레이드오프.

## 문제 6

체리 피킹 전략:
보험사가 고객 풀에서 저위험 고객을 선별 유치.
전략: 특정 직업군(교사, 공무원)에 할인.

이것이 전략적으로 왜 유리한가요?
사회적으로 어떤 문제가 있나요?

### 힌트 1
특정 직업 = 저위험 신호.
할인 → 저위험 고객 유치 → 이익 증가.

### 힌트 2
고위험 고객은 다른 보험사로.
업계 전체로 고위험 고객 집중 문제.

### 정답
전략=저위험선별로이익↑ / 사회문제=고위험자보험료급등·시장실패

### 해설
체리 피킹(cream-skimming):
전략적 신호 활용: 특정 그룹 = 저위험.
가격 차별로 저위험 유치.
경쟁사: 고위험 고객 집중 → 손실.
균형: 모든 보험사가 체리 피킹 시도.
사회적 문제:
고위험자 보험 접근성 저하.
보험 시장 분열.
사회적 연대 원리 훼손.
규제 해결:
공동 요율: 모든 고객 같은 가격.
의무 가입: 역선택 방지 (건강보험).
위험 조정(risk adjustment): 고위험 가입자 보상.
정보 비대칭의 사회적 효과.

## 문제 7

최후통첩 게임 변형:
100달러를 A가 제안, B가 수락/거절.
하지만 B의 최소 수락액은 비공개.
B 타입:
공정 타입(F, 50%): 30달러 미만 거절.
이기적 타입(S, 50%): 1달러도 수락.

A의 최적 제안은?

### 힌트 1
제안 X (A가 100-X 제안 시 A 몫):
X ≥ 30: 모든 타입 수락 → A = 100-X.
X < 30: S 타입만 수락 → A = 0.5×(100-X).

### 힌트 2
X=30: A = 70.
X=1: A = 0.5×99 = 49.5.
A는 X=30 선택.

### 정답
X=30 제안 → A=70

### 해설
불완전 정보 최후통첩:
X ≥ 30 제안 시 (확실한 수락):
A의 이익 = 100-X (X=30이면 70).
X < 30 제안 시 (F 타입 거절):
A의 기대 이익 = 0.5×(100-X).
X=30에서 0.5×(100-30)=35 < 70.
최적: X=30 제안, A=70.
정보의 가치:
F 타입 비율이 높으면 더 공정한 제안 필요.
F 비율 100%: X=50 제안 (50:50).
F 비율 0%: X=1 제안.
사전 정보가 협상 전략을 결정.

## 문제 8

전략적 무지(strategic ignorance):
협상 전 정보를 일부러 수집하지 않는 전략.

상황:
A와 B가 협상.
A가 B의 최저 수락가를 알면:
그 가격만큼 제안 가능 (A에게 유리).
하지만 B는 A가 정보를 수집했다면 더 높게 요구.

A가 정보를 수집해야 하나요?

### 힌트 1
정보 수집 = 신호: "나는 당신의 약점을 안다."
B의 반응: 방어적으로 더 높게 요구.

### 힌트 2
전략적 무지: 정보를 모르는 척 → B의 경계 낮춤.
때로는 무지가 유리.

### 정답
상황 의존적: B가 강경파이면 무지가 유리, 약한 협상가면 정보 수집 유리

### 해설
전략적 무지 게임:
정보 수집의 역설:
정보 수집 = 상대방에게 신호.
"나는 최저 수락가를 알고 있다" → 상대 방어.
무지의 이점:
"나는 얼마가 적정한지 모른다" → 상대 안심.
상대가 실제 최저 수락가에 가깝게 제안.
베이지안 협상:
상대의 타입(강경/유화)에 따라 전략 조정.
정보 수집 여부 자체가 전략.
실제 협상:
협상 전문가: 많은 정보 수집이 기본.
하지만 정보 사용 방식도 전략적.
"당신이 알고 있다는 것을 드러내지 않기."

## 문제 9

경매 담합:
n개 기업이 경매에 참여.
담합: 미리 낙찰자 결정 → 낮은 가격으로 낙찰.
낙찰 이익을 담합 기업들이 나눔.

(1) 담합의 내부 안정성 문제는?
(2) 경매 설계로 담합을 방지하는 방법은?
(3) 담합 탐지 패턴은?

### 힌트 1
담합 = 카르텔 = 죄수의 딜레마.
배신 유인: 낙찰 기업이 이익 독점 가능.

### 힌트 2
2위 가격 경매: 진실 표현이 지배 전략 → 담합 어려움.
무작위 배정: 예측 불가 → 담합 이익 불확실.

### 정답
담합=카르텔딜레마 / 2위경매=담합방지 / 탐지=가격비교·로테이션패턴

### 해설
담합의 딜레마:
담합 결정: 낙찰 기업 선정.
배신 유인: 다른 기업이 조금 낮게 입찰 → 낙찰.
이익 분배 문제: 누가 얼마 받는가.
반복 게임: 미래 협력 위해 현재 협력.
경매 설계 대응:
2위 가격 경매: 진실 입찰 지배 전략.
→ 담합 기업이 배신 유인 없음... 하지만 낮게 짜는 담합 가능.
밀봉 1위 경매 + 높은 예비 가격.
여러 라운드 경매.
담합 탐지:
가격 패턴: 특정 기업이 번갈아 낙찰.
낙찰가 클러스터: 이상하게 유사한 입찰.
시장 점유율 고정: 특정 기업이 특정 지역.
불법: 대부분 국가에서 담합은 경쟁법 위반.

## 문제 10

위기 협상:
납치 상황에서 협상가의 전략.
범인: 인질 해방 vs 탈출 추구.
협상가: 시간 벌기 vs 즉각 요구 수용.

게임 트리:
```
협상가: 수용 또는 지연
범인: 해방 또는 유지
```

하지만 이것은 반복 게임.
신뢰할 수 있는 약속이 핵심.

(1) 왜 요구를 즉시 수용하면 안 되는가?
(2) 신뢰 구축의 전략은?
(3) 약속의 신뢰성을 높이는 방법은?

### 힌트 1
즉시 수용 = 납치 가치 높임 → 더 많은 납치 유발.
시간 = 범인의 비용 증가 + 협상력.

### 힌트 2
신뢰 구축: 작은 요구 수용 → 상호 신뢰.
약속 신뢰성: 확인 가능한 약속만.

### 정답
즉시수용=나쁜선례 / 시간지연+부분수용=전략 / 확인가능약속=신뢰성

### 해설
위기 협상의 게임 이론:
즉시 수용의 위험:
납치 성공 → 미래 납치 증가 (반복 게임 효과).
전략: "우리는 테러리스트와 협상 안 한다."
시간 지연의 이점:
범인의 피로 증가.
탈출 기회 포착.
요구 사항 완화.
신뢰 구축:
작은 증거 제공 (인질 안전 확인).
단계별 상호 수용.
제3자 개입 (중재자).
약속 신뢰성:
확인 가능한 약속: "X를 하면 Y를 제공."
