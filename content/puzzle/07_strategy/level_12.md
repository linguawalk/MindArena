## 문제 1

균형 개념의 계층:
게임 이론의 균형 정제 계층.

```
내쉬 균형 (NE)
  ↓ 더 강한 요구
부분게임 완전 균형 (SPE)
  ↓
완전 베이지안 균형 (PBE)
  ↓
순차 균형 (SE)
  ↓
전향 귀납 균형
```

(1) 각 균형 개념이 추가로 요구하는 것은?
(2) 가장 정제된 균형이 항상 최선인가요?
(3) 현실에서 어느 균형 개념이 적절한가요?

### 힌트 1
NE → SPE: 균형 외 경로에서도 최적.
SPE → PBE: 믿음의 일관성 추가.

### 힌트 2
더 강한 정제 = 더 적은 균형.
하지만 균형이 없어질 수도 있음.

### 정답
각단계=추가제약 / 정제≠최선(존재안할수도) / 현실=상황에따라다름

### 해설
균형 정제의 의미:
NE: 단순 상호 최적 반응.
SPE: 모든 부분게임에서 NE (비신뢰성 위협 제거).
PBE: 믿음 + 순차 합리성.
SE: PBE + 일관성 강화 (크렙스-윌슨).
전향 귀납: 균형 외 경로 행동도 합리적.
균형 정제의 문제:
너무 강한 정제 → 균형 없음.
현실과 괴리 (인간은 L2 수준).
상황별 적절한 개념:
단기 게임: NE.
반복 관계: SPE + 포크 정리.
신호 게임: PBE.
헌신 문제: SE.
실험 경제학: 행동 게임 이론.

## 문제 2

전략적 사고의 인지적 한계:
인간 두뇌의 전략 처리 용량.

실험 결과:
체스: 전문가는 L4~5 수준 계산.
포커: 전문가는 L3~4 수준.
경제 게임: 대부분 L1~2 수준.

(1) 왜 전략적 사고에 한계가 있나요?
(2) 이것이 게임 이론에 주는 함의는?
(3) AI가 인간의 한계를 어떻게 극복하나요?

### 힌트 1
인지 부하: 작업 기억 = 7±2 청크.
전략 탐색: 지수적 증가 vs 선형 처리.

### 힌트 2
게임 이론: 무한 귀납 가정.
현실: 유한 수준의 귀납.
→ 완전 합리성 가정 비현실적.

### 정답
인지한계=작업기억·처리속도 / 함의=유한수준균형 / AI=탐색+평가병렬화

### 해설
인지적 한계의 게임 이론:
작업 기억: 7±2 항목 동시 처리.
전략 트리: 깊이 d, 분기 b → b^d 경우.
인간: d=3~5, 선택적 탐색.
AI: 병렬 처리 + 신경망 평가.
행동 게임 이론의 교훈:
완전 합리성 → 유한 합리성.
수준-k 모형: 현실적 묘사.
인지 계층: 인구의 k 분포.
AI의 극복:
몬테카를로 트리 탐색: 확률적 탐색.
신경망 평가: 패턴 기반 빠른 평가.
자기 대결: 경험 없이 학습.
미래: AI + 인간 = 인지 증강.

## 문제 3

전략의 역설들:
게임 이론의 유명한 역설들.

①죄수의 딜레마: 개인 합리 → 집단 비합리.
②투표 역설: 개인 합리 선호 → 집단 순환.
③자기 실현 예언: 믿음이 현실 창조.
④역선택 스파이럴: 정보 비대칭 → 시장 붕괴.

(1) 이 역설들의 공통 구조는?
(2) 제도가 역설을 어떻게 해결하나요?
(3) 해결 불가능한 역설이 있나요?

### 힌트 1
공통 구조: 개인 최적 ≠ 집단 최적.
= 외부성 (내가 남에게 영향, 비용 안 냄).

### 힌트 2
제도: 외부성을 내부화.
세금, 보조금, 규제, 규범.

### 정답
공통=외부성·구성의오류 / 제도=내부화 / 해결불가=투표역설(애로정리)

### 해설
전략 역설의 공통 구조:
외부성: 내 행동이 남에게 영향.
구성의 오류: 개별 합리 → 집단 비합리.
자기 참조: 예측이 현실에 영향.
제도적 해결:
죄수 딜레마: 법, 규범, 반복 게임.
자기 실현: 조정 기관, 명확한 기대 설정.
역선택: 의무 가입, 공시, 인증.
해결 불가능한 역설:
애로 불가능성: 어떤 투표도 모든 조건 동시 충족 불가.
집단 선호의 근본적 한계.
전략의 철학적 함의:
집단이 개인보다 "덜 합리적"?
제도 설계 = 개인 인센티브 재조정.
게임 이론 = 사회 공학의 도구.

## 문제 4

전략과 윤리:
게임 이론적 최적이 윤리적으로 옳은가?

사례들:
①최후통첩 게임: 99:1 제안은 합리적이지만 불공정.
②담합: 기업에 최적, 소비자에 해로움.
③군비 경쟁: 상호 최적, 상호 해로움.
④플랫폼 독점: 플랫폼 최적, 사회 후생 감소.

(1) 게임 이론이 윤리를 포함할 수 있나요?
(2) 공정성의 수학적 표현은?
(3) 윤리적 게임 이론이란?

### 힌트 1
공정성 선호: 불공정에 응징 (최후통첩 거절).
이타주의: 타인 이익도 효용에 포함.

### 힌트 2
페어 게임 이론: 불공정도 효용 함수에 포함.
파인버그: 상호성 모형.

### 정답
공정성=효용함수확장 / 페어게임이론=윤리포함 / 메커니즘=윤리적인센티브설계

### 해설
윤리적 게임 이론:
표준 게임 이론: 물질적 이기심.
확장:
불평등 혐오 (페어·슈미트):
U = u - α×max(u-uⱼ, 0) - β×max(uⱼ-u, 0).
상호성 (라빈): 좋은 의도 → 좋게 반응.
이타주의: 타인 효용을 내 효용에 포함.
윤리 메커니즘 설계:
규칙 = 윤리적 행동이 개인 이익과 일치하도록.
탄소세: 환경 파괴 = 비용.
반독점법: 담합 = 처벌.
사회적 규범: 공정성 = 사회적 승인.
한계:
모든 윤리를 수학화 불가.
문화적 차이: 공정성 기준 다양.
게임 이론 = 도구, 가치 판단 = 외부.

## 문제 5

복잡계와 전략:
주식 시장 = 전략적 복잡계.

참여자들이 서로의 행동을 예측하며 거래.
예측이 현실에 영향 → 예측 자체가 게임.

효율적 시장 가설 vs 행동 경제학:
(1) EMH의 게임 이론적 기반은?
(2) 왜 예측이 어려운가요?
(3) 알파(초과 수익)는 가능한가요?

### 힌트 1
EMH: 모든 공개 정보 = 이미 가격에 반영.
새로운 정보만 가격 변화.
→ 예측 불가 = 랜덤워크.

### 힌트 2
행동 경제학: 인지 편향 → 체계적 오류 → 예측 가능.
하지만 예측 → 거래 → 가격 조정 → 예측 무효.

### 정답
EMH=공통지식균형 / 예측어려움=자기소멸예측 / 알파=일시적(정보우위)

### 해설
효율적 시장의 게임 이론:
EMH = 공통 지식의 내쉬 균형.
모두 알면 가격에 반영 → 초과 수익 없음.
그로스만-스티글리츠 역설:
정보 수집 비용 > 알파 → 아무도 수집 안 함.
→ 정보 미반영 → 알파 가능.
→ 정보 수집 → 반영 → 알파 소멸.
균형: 일부만 정보 수집 (혼합 전략).
자기 소멸 예측:
예측 발표 → 모두 행동 → 예측 실현 or 소멸.
"이 주식 오를 것" → 모두 삼 → 오름 (자기 실현).
또는: 모두 삼 → 고평가 → 하락 (자기 소멸).
알파:
정보 우위: 빠른 처리, 내부 정보.
인지 우위: 다른 사람의 오류 활용.
지속 불가능: 경쟁으로 소멸.

## 문제 6

전략과 진화:
인간 협력의 진화적 기원.

혈연 선택: 유전자 공유 → 이타적 행동.
해밀턴 규칙: rb > c (r=친족계수, b=이익, c=비용).

호혜적 이타주의: 반복 게임에서 협력 진화.
트리버스(1971): 팃포탯 = 진화적 안정.

(1) 해밀턴 규칙과 게임 이론의 연결은?
(2) 비친족 협력의 진화 메커니즘은?
(3) 인간 대규모 협력의 특수성은?

### 힌트 1
혈연 선택: 유전자 = "자신의 복사본" 도움.
rb > c: 도움의 비용 < 유전자 전파 이익.

### 힌트 2
호혜적 이타주의: 반복 게임 + 기억.
강한 호혜성: 처벌 비용을 감수하고 불공정자 처벌.

### 정답
해밀턴=유전자수준게임 / 호혜=반복게임 / 인간=제도·규범으로대규모협력

### 해설
협력의 진화 게임 이론:
혈연 선택:
r = 친족 계수 (형제=0.5, 사촌=0.125).
rb > c: 이타 행동 유전자 전파.
유전자 = "플레이어", 생물 = 도구.
호혜적 이타주의:
비친족 협력: 반복 게임 + 기억.
팃포탯 = 진화적 안정 전략.
조건: 상호 작용 충분히 반복.
강한 호혜성:
익명 집단에서도 처벌.
처벌 비용 감수 = 초합리적.
인간 규범: "내가 손해봐도 불공정자 처벌."
인간 대규모 협력:
가족·부족 넘는 협력 = 인간 특수성.
제도, 법, 종교, 규범.
가상의 믿음: 국가, 화폐, 기업 = 협력 기반.
하라리: 인간은 허구를 믿을 수 있어 대규모 협력.

## 문제 7

전략적 복잡도의 한계:
계산 복잡도와 게임 이론의 교차점.

PPAD 완전성:
내쉬 균형 찾기 = PPAD-완전.
다항 시간 알고리즘 존재 미지.

(1) PPAD란?
(2) 내쉬 균형이 PPAD 완전이라는 의미는?
(3) 근사 균형은 어떻게 구하나요?

### 힌트 1
PPAD: 방향 있는 그래프에서 경로 끝 찾기 문제류.
P에 속하지 않을 가능성 높음.

### 힌트 2
ε-근사 균형: 모든 플레이어가 ε 이상 이익 없음.
ε > 0이면 더 쉽게 찾을 수 있음.

### 정답
PPAD=고정점탐색류 / 내쉬=PPAD완전=다항시간미지 / 근사=ε균형사용

### 해설
PPAD 완전성 (다상그프타-고드빌론, 2006):
PPAD: 방향 그래프에서 소스에서 싱크 찾기.
브라우어 고정점 탐색 = PPAD.
내쉬 균형 찾기 = PPAD.
의미:
P ≠ NP처럼: PPAD ≠ P일 가능성.
= 내쉬 균형 찾기 = 다항 시간 알고리즘 없을 수 있음.
하지만 P에 속하지 않음을 증명도 안 됨.
알고리즘적 게임 이론:
실용적: ε-근사 균형 사용.
ε 작을수록 어려움.
특수 게임: 2인 제로합 = LP 풀이 가능.
응용:
온라인 광고 경매: 근사 VCG.
라우팅: 근사 내쉬 찾기.
포커 AI: ε-Nash CFR 알고리즘.

## 문제 8

전략과 사회 제도:
제도 = 게임의 규칙.

노스(1990): 제도 = "사회의 게임 규칙."
공식 규칙 + 비공식 규범.

제도 변화의 게임 이론:
기존 제도 = 규칙.
변화 시도 = 새 게임.
새 균형이 더 좋아도 전환 비용 존재.

(1) 제도가 왜 경로 의존적인가요?
(2) 제도 변화의 조건은?
(3) 좋은 제도 설계의 원칙은?

### 힌트 1
QWERTY: 더 좋은 드보락 키보드가 있어도 전환 안 함.
전환 비용 > 장기 이익.

### 힌트 2
제도 변화 조건:
충격 + 이해관계 변화 + 새 연합 형성.

### 정답
경로의존=전환비용+조정실패 / 변화=충격+연합 / 좋은제도=자기강화+적응성

### 해설
제도의 경로 의존성:
QWERTY 키보드: 초기 타자기 속도 문제.
네트워크 효과: 모두 QWERTY → 전환 불이익.
더 좋은 드보락 키보드도 도입 실패.
제도 = 자기 강화(self-reinforcing) 균형.
제도 변화 조건:
외부 충격: 전쟁, 경제 위기, 기술 혁명.
이해관계 변화: 새 플레이어 등장.
선도자: 전환 비용 감수하는 기업/국가.
임계 질량: 충분한 전환자 → 눈사태 효과.
좋은 제도 설계:
자기 강화: 지키는 것이 이익.
적응성: 환경 변화에 유연.
포용성: 다수의 이익.
집행 가능: 위반 비용 확실.
노스의 교훈: 부국 = 좋은 제도.

## 문제 9

전략의 철학적 기초:
합리성 가정의 철학적 문제.

완전 합리성:
①완전한 선호.
②이행성.
③베이즈 업데이트.
④기대 효용 극대화.

이 가정들의 철학적 문제는?

### 힌트 1
완전한 선호: 모든 결과 비교 가능?
이행성: A>B, B>C, C>A인 경우?

### 힌트 2
기대 효용: 알레 역설 (확실한 것 선호).
베이즈 업데이트: 어디서 시작 믿음?

### 정답
선호완전성의문·이행성위반·알레역설·비합리베이즈

### 해설
합리성 가정의 문제:
완전한 선호:
비교 불가 결과 (삶 vs 자유).
문화적 선호 다양성.
이행성:
순환 선호 실험에서 관찰.
소리, 색, 다차원 대상.
기대 효용 (알레 역설):
확실한 100만 달러 > 불확실한 기댓값이 같은 복권.
기대 효용 이론 위반.
베이즈 업데이트:
사전 믿음 어디서? (기저율 무시)
확인 편향: 믿음에 맞는 정보만 업데이트.
대안 이론:
전망 이론 (카너만-트버스키): 손실 회피, 준거점.
유한 합리성 (사이먼): 만족화 (satisficing).
이중 처리 이론 (시스템 1/2).
철학적 함의:
"합리성"의 정의가 문화적·상황적.
게임 이론 = 합리성의 수학, 인간 행동의 근사.

## 문제 10

심화 구간 종합: 전략이란 무엇인가?

다음 층위에서 통합 답변을 구성하세요.

①수학적: 게임 이론, 균형, 최적화.
②심리적: 인지 한계, 편향, 감정.
③사회적: 제도, 규범, 문화.
④진화적: 자연 선택, 적합도.
⑤철학적: 합리성, 자유의지, 윤리.

각 층위의 핵심과 한계, 미해결 문제는?

### 힌트 1
각 층위가 전략에 대해 다른 측면을 드러냅니다.
통합적 이해가 핵심입니다.

### 힌트 2
전략의 한계: 계산 복잡도, 정보 부족, 인지 한계.

### 정답
다층적전략이해 / 각층위의공헌과한계 / 통합이론미완성

### 해설
전략의 다층적 이해:
수학적:
최적화, 균형, 메커니즘 설계.
한계: 계산 복잡도, 균형 다수성.
미해결: PPAD vs P, 균형 선택.
심리적:
인지 한계, 사회적 선호, 감정.
한계: 개인 차이, 측정 어려움.
미해결: 진정한 합리성 기준.
사회적:
제도, 규범, 문화 = 게임 규칙.
한계: 제도 변화 예측 어려움.
미해결: 좋은 제도의 보편 원칙.
진화적:
자연 선택 = 전략 필터.
한계: 문화 진화 속도.
미해결: 이타주의의 완전한 설명.
철학적:
자유의지: 전략 선택이 진정 자유로운가?
윤리: 최적이 올바른가?
미해결: 규범적 게임 이론.
통합 전략 이론:
수학 × 심리 × 사회 × 진화 × 철학.
현재: 분리된 접근.
미래: 통합적 전략 과학.
