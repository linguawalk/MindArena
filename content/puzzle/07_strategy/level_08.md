## 문제 1

최적 메커니즘 설계 — 마이어슨 정리:
단일 물건 경매, 단일 구매자.
구매자 가치 v: 균등분포 [0, 1].
판매자 최적 예비 가격은?

### 힌트 1
최적 예비 가격: 가상 가치(virtual value) = v - (1-F(v))/f(v).
균등분포: F(v)=v, f(v)=1.
가상 가치 = v-(1-v) = 2v-1.

### 힌트 2
최적: 가상 가치 ≥ 0일 때 판매.
2v-1 ≥ 0 → v ≥ 1/2.
예비 가격 = 1/2.

### 정답
예비가격=1/2, 판매확률=1/2, 기대수입=1/4

### 해설
마이어슨 최적 경매(1981):
균등분포 [0,1]:
최적 예비 가격 r = 1/2.
v ≥ 1/2이면 판매: 확률 1/2.
기대 가격 = E[v|v≥1/2] = 3/4.
기대 수입 = (1/2)×(3/4) = 3/8.
아니면: 기대 수입 = 1/4?
정확: E[수입] = r×P(v<r) + E[v|v≥r]×P(v≥r).
= 1/2×0 + (3/4)×(1/2) = 3/8.
또는 r=1/2 직접: E = ∫₁/₂¹ v dv = [v²/2]₁/₂¹ = 1/2-1/8 = 3/8.
일반 결과:
더 많은 구매자 → 예비 가격 낮춤 (경쟁이 수입 보장).
독점 판매자 → 예비 가격으로 정보 지대 추출.

## 문제 2

약속과 헌신 장치:
오디세우스 문제:
사이렌의 노래를 들으면 배를 몰아 암초로.
오디세우스의 해결책: 돛대에 묶이기.

전략적 함의:
(1) 왜 헌신 장치가 전략적으로 유리한가요?
(2) 현실의 헌신 장치 예시는?
(3) 헌신의 신뢰성 조건은?

### 힌트 1
미래의 자신을 구속 → 상대방의 예측 변화.
"나는 절대 X를 하지 않겠다" → 상대 전략 변화.

### 힌트 2
신뢰할 수 있는 헌신: 번복 비용이 매우 높음.
태울 배: 퇴로 차단 → 전투 의지 신뢰.

### 정답
헌신=상대전략변화→자신유리 / 탈출불가=태운배·계약·평판 / 번복불가=신뢰성

### 해설
헌신 장치 (Commitment Device):
역설: 선택지 제거가 유리.
"나는 협상 안 한다" 약속:
→ 상대가 더 좋은 조건 제시.
→ 나는 더 유리한 위치.
현실 예시:
탈출로 차단: 코르테스의 배 태우기.
계약: 위약금으로 번복 비용 높임.
평판: 과거 행동이 미래 신뢰성.
예비 비용: 협상 전 투자.
중앙은행 독립성: 인플레이션 약속 신뢰성.
신뢰성 조건:
번복 비용 > 번복 이익.
제3자가 관찰 가능.
즉각적이고 자동적 처벌.

## 문제 3

정보 설계 (Bayesian Persuasion):
발신자(S)가 수신자(R)에게 신호 보냄.
수신자가 특정 행동을 취하길 발신자가 원함.

상황:
제품 품질: 좋음(G, 확률 0.4) 또는 나쁨(B, 0.6).
구매자: G이면 구매 (이익 1), B이면 불구매.
판매자: 구매 선호.
판매자가 검사 결과를 조작(공개 방식 선택) 가능.

최적 정보 공개 전략은?

### 힌트 1
완전 공개: P(구매)=0.4 (G만 구매).
완전 비공개: P(G)=0.4 < 0.5 → 구매 안 함.

### 힌트 2
최적 신호: G이면 항상 "G" 보고.
B이면 확률 2/3로 "G", 1/3로 "B" 보고.

### 정답
최적=B를2/3확률로G로표시 → P(구매)=0.4+0.6×2/3=0.8

### 해설
베이지안 설득 (캐머러-겐트조우, 2011):
목표: 구매자가 구매 결정하도록 신호 설계.
구매 조건: P(G|신호="G") ≥ 0.5.
최적 신호 구조:
G → "G" (확률 1).
B → "G" (확률 p), "B" (확률 1-p).
신호 "G" 수신 시:
P(G|"G") = 0.4/(0.4+0.6p) ≥ 0.5.
0.4 ≥ 0.5×(0.4+0.6p) = 0.2+0.3p.
0.2 ≥ 0.3p → p ≤ 2/3.
최대 p=2/3: P(구매) = 0.4+0.6×(2/3) = 0.4+0.4 = 0.8.
정보 설계의 힘:
사실에 기반하면서도 정보 공개 방식으로 행동 유도.
현실: 광고, 신용 평가, 감사 보고서.

## 문제 4

동적 가격 설정 — 라쳇 효과:
규제 기관이 기업의 비용을 모름.
기업이 효율성 개선 → 규제 기관이 목표 상향.
기업의 최적 반응은?

### 힌트 1
라쳇 효과(Ratchet Effect):
올해 잘하면 → 내년 목표 높아짐.
최적: 올해 일부러 낮은 성과.

### 힌트 2
동적 인센티브 문제:
단기 성과와 장기 인센티브의 충돌.
해결: 장기 계약, 성과 공유.

### 정답
기업=성과숨김전략 / 사회손실=비효율 / 해결=장기계약·성과공유

### 해설
라쳇 효과 (Weitzman, 1980):
소련 계획 경제에서 처음 관찰.
메커니즘:
올해 성과 좋음 → 내년 목표 상향.
기업: 올해 성과 의도적 제한.
→ 효율성 개선 유인 사라짐.
동적 인센티브 문제:
오늘의 행동 → 내일의 제약.
시간 불일치(time inconsistency).
해결책:
장기 계약: 미리 목표 고정.
성과 공유: 개선의 일부를 기업에 귀속.
롤링 기준: 자신의 과거보다는 산업 평균 기준.
현실: 기업-본사 관계, 정부 보조금, 의료 수가.

## 문제 5

전략적 보완과 대체:
전략적 보완(strategic complements):
상대가 더 많이 → 나도 더 많이.
전략적 대체(strategic substitutes):
상대가 더 많이 → 나는 더 적게.

쿠르노(수량) vs 버트랑(가격) 경쟁:
어느 것이 보완이고 어느 것이 대체인가요?

### 힌트 1
쿠르노: 상대 생산량↑ → 내 최적 생산량↓.
반응함수 기울기 음수.

### 힌트 2
버트랑: 상대 가격↑ → 내 최적 가격↑.
반응함수 기울기 양수.

### 정답
쿠르노=전략적대체 / 버트랑=전략적보완

### 해설
전략적 보완 vs 대체:
쿠르노 (수량 경쟁):
∂Q₁*/∂Q₂ < 0 (전략적 대체).
상대 증산 → 내 감산 최적.
안정적: 균형으로 수렴.
버트랑 (가격 경쟁):
∂P₁*/∂P₂ > 0 (전략적 보완).
상대 가격↑ → 내 가격↑ 최적.
불안정 가능: 가격 전쟁 또는 담합.
외부 충격의 전파:
보완: 충격이 증폭 (금융 위기 전파).
대체: 충격이 흡수.
정책 함의:
보완 → 정책 효과 증폭 (양날의 검).
대체 → 정책 효과 부분 상쇄.

## 문제 6

역선택과 도덕적 해이의 복합:
건강 보험에서 두 문제 동시 발생.
역선택: 아픈 사람만 가입.
도덕적 해이: 가입 후 건강 관리 소홀.

(1) 두 문제의 수학적 차이는?
(2) 공동 해결책은?
(3) 의무 가입의 효과는?

### 힌트 1
역선택: 사전 정보 비대칭.
도덕적 해이: 사후 행동 비관찰.

### 힌트 2
의무 가입: 역선택 해결 (고위험만 가입 방지).
하지만 도덕적 해이: 해결 안 됨.

### 정답
역선택=사전정보·도덕해이=사후행동 / 공제제도=양쪽완화 / 의무가입=역선택만해결

### 해설
역선택 vs 도덕적 해이:
역선택:
발생 시점: 계약 전.
원인: 숨겨진 타입.
해결: 스크리닝, 신호, 의무 가입.
도덕적 해이:
발생 시점: 계약 후.
원인: 숨겨진 행동.
해결: 모니터링, 인센티브 정렬, 공제.
공동 해결책:
공제(deductible): 손실 일부 부담.
역선택 완화: 고위험자 비용 체감.
도덕적 해이 완화: 주의 유인.
트레이드오프: 공제 높을수록 위험 공유 감소.
의무 가입:
역선택 해결: 저위험자도 가입 → 풀 확장.
도덕적 해이 악화 가능: 선택 없음 → 더 소홀.
보험 설계의 핵심 딜레마.

## 문제 7

알고리즘 게임 이론:
n명의 에이전트가 공유 자원 배분.
각 에이전트: 처리 시간 tᵢ (비공개).
사회 최적: 총 완료 시간 최소화.
각자의 인센티브: 자신의 완료 시간 최소화.

진실 유인 메커니즘이 가능한가요?

### 힌트 1
VCG 적용 가능.
하지만 계산 복잡도 문제.

### 힌트 2
근사 알고리즘 + 진실 유인의 트레이드오프.
완전 최적화는 NP-hard일 수 있음.

### 정답
VCG=진실유인가능 / 하지만계산복잡도=실용성문제 / 근사VCG존재

### 해설
알고리즘 메커니즘 설계:
일반 스케줄링:
VCG 적용: 진실 유인 + 효율성.
하지만: 최적 스케줄링 = NP-hard.
VCG 계산 자체가 어려움.
근사 VCG (아퀴닐라-마이어슨):
근사 알고리즘 + VCG 변형.
진실 유인 유지 + 다항 시간.
근사율: 최적의 (1+ε) 배 이내.
인터넷 응용:
광고 경매: 구글, 메타의 실시간 입찰.
클라우드 컴퓨팅: 자원 배분.
라우팅: 네트워크 경로 선택.
계산 복잡도와 인센티브의 긴장:
효율적 알고리즘 = 전략적 조작 가능.
진실 유인 알고리즘 = 때로 비효율.

## 문제 8

공공 정보와 사적 정보:
두 투자자가 자산 가치 추정.
공공 신호: σ_p (모두 관찰).
사적 신호: σᵢ (개인만 관찰).

공공 정보가 늘면 협력이 증가하나요?

### 힌트 1
공공 신호 강화: 모든 사람의 믿음이 비슷해짐.
→ 조정 게임에서 협력 증가.

### 힌트 2
하지만 글로벌 게임: 공공 신호 강화 → 다중 균형 복귀.
→ 불안정성 증가 가능.

### 정답
공공정보=조정촉진 / 하지만과도하면다중균형복귀=불안정

### 해설
모리스-신 정보 효과 (2002):
글로벌 게임에서:
공공 신호 약함 → 유일 균형 (안정).
공공 신호 강함 → 다중 균형 복귀 (불안정).
역설: 중앙은행의 투명성이 오히려 불안정 유발.
메커니즘:
공공 신호: 모든 사람이 같은 정보.
모두가 "모두 투자할 것"이라고 예상 → 실제 투자.
자기 실현 예언 강화.
정책 함의:
중앙은행 투명성의 두 얼굴.
너무 명확한 신호 → 금융 군집(herding).
모호한 신호 → 다양한 예상 → 안정.
앨런-고르통 이견: 불투명성의 전략적 가치.

## 문제 9

플랫폼 전략:
양면 시장(two-sided market):
플랫폼이 두 그룹(A, B)을 연결.
A 그룹 수: nA, B 그룹 수: nB.
A의 가입 가치: αnB (B가 많을수록 가치).
B의 가입 가치: βnA (A가 많을수록 가치).

최적 가격 책정은?

### 힌트 1
네트워크 외부성: 한쪽 가입 → 다른 쪽 가치 증가.
닭-달걀 문제: 어느 쪽을 먼저 끌어올까?

### 힌트 2
최적: 보조금 측과 수익 측 구분.
가격 탄력성과 네트워크 효과 고려.

### 정답
낮은탄력성측=높은가격 / 네트워크효과큰측=보조금 / 닭달걀=보조금으로시작

### 해설
양면 시장 가격 이론 (로쉐-티롤, 2003):
최적 가격 조건:
P_A - MC_A = (1/ε_A - σ_A/P_B) × 가격 마크업.
단순화:
ε 낮은 측: 가격 높게.
네트워크 효과 큰 측: 보조금 또는 낮은 가격.
실제 사례:
신용카드: 소비자 무료/보상, 가맹점 수수료.
플랫폼: 사용자 무료, 광고주 유료.
게임 콘솔: 하드웨어 손실, 소프트웨어 이익.
닭-달걀 해결:
한쪽을 무료/보조금으로 먼저 유치.
임계 질량 달성 후 수익화.
아마존: 구매자 유치 → 판매자 유치.

## 문제 10

복잡계 전략:
창발 전략(emergent strategy):
의도하지 않았지만 행동 패턴에서 전략이 나타남.

민츠버그의 전략 유형:
①계획 전략 (의도적).
②창발 전략 (우발적).
③실현 전략 (실제).

(1) 게임 이론과 창발 전략의 차이는?
(2) 적응 전략의 수학적 모델은?
(3) 불확실성 환경에서 최적 전략 접근은?

### 힌트 1
게임 이론: 완전한 계획 + 예측.
창발: 실시간 적응 + 학습.
두 가지의 결합이 현실적.

### 힌트 2
강화 학습: 환경에서 보상 신호로 전략 학습.
Q-러닝: 상태-행동 가치 함수 추정.

### 정답
게임이론=예측기반·창발=적응기반 / 강화학습=수학적모델 / 불확실=포트폴리오+실험

### 해설
창발 전략의 수학:
강화 학습:
Q(s,a) = r + γ×max Q(s',a').
상태 s에서 행동 a의 장기 가치.
환경과 상호작용으로 Q 학습.
적응적 게임 이론:
진화 알고리즘: 전략 집단의 자연 선택.
허구 게임 플레이(Fictitious Play): 과거 통계로 전략 업데이트.
후회 최소화(Regret Minimization): 과거 후회 줄이는 방향.
불확실성 전략:
포트폴리오: 여러 전략 병행.
실험: 소규모 시험 후 확장.
옵션: 미래 선택 유지.
계획+적응: 사전 계획 + 실시간 조정.
현실: 전략 컨설팅의 핵심 질문.
