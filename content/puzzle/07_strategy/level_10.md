## 문제 1

내쉬 균형의 존재 증명:
브라우어 고정점 정리를 이용한 증명.

내쉬 정리(1950):
유한 플레이어, 유한 전략의 게임에서
혼합 전략 내쉬 균형은 항상 존재한다.

(1) 브라우어 고정점 정리란?
(2) 내쉬 균형 = 고정점인 이유는?
(3) 이 정리의 한계는?

### 힌트 1
브라우어: 연속 함수 f: D→D (D: 볼록 컴팩트)는
반드시 고정점 x* = f(x*)를 가진다.

### 힌트 2
최적 반응 함수 BR: Σ→Σ (혼합 전략 공간).
균형 = 모든 플레이어의 BR의 고정점.

### 정답
브라우어=고정점정리 / 균형=BR의고정점 / 한계=존재만증명(유일성·선택불보장)

### 해설
내쉬 균형 존재 증명:
혼합 전략 공간 Σ: 볼록 컴팩트 집합.
최적 반응 BR(σ): 상반응 대응(correspondence).
카쿠타니 고정점 정리 (BR이 집합 값):
BR: Σ→2^Σ, 볼록 값, 폐그래프.
→ 고정점 σ* ∈ BR(σ*) 존재.
이것이 내쉬 균형.
수학적 아름다움:
위상수학(고정점) + 게임이론(균형)의 연결.
내쉬의 박사 논문: 27페이지, 2개 정리.
한계:
존재 증명이지 유일성 아님.
다중 균형: 어느 것이 실현될지 불명.
계산 복잡도: 균형 찾기 PPAD-완전.

## 문제 2

완전 베이지안 균형(PBE):
불완전 정보 게임의 엄밀한 균형 개념.

조건:
①전략: 각 플레이어의 최적 반응.
②믿음: 베이즈 법칙으로 업데이트.
③일관성: 균형 경로 상의 믿음은 베이즈 법칙.

다음 신호 게임에서 PBE를 구하세요:
강한 타입(S, 2/3) → 신호 보냄(C) 또는 안 보냄(NC).
약한 타입(W, 1/3) → C 또는 NC.

수신자: 신호 보고 투자(I) 또는 비투자(NI).

보수 (타입, 신호, 수신자):
S-C-I: (2,1), S-C-NI: (0,0), S-NC-I: (3,1), S-NC-NI: (1,0).
W-C-I: (1,1), W-C-NI: (-1,0), W-NC-I: (2,1), W-NC-NI: (0,0).

### 힌트 1
분리 균형 확인: S→NC, W→C.
수신자: C 보면 W → NI. NC 보면 S → I.

### 힌트 2
S: NC 선택 시 I → 보수 3. C 선택 시 NI → 보수 0.
S: NC 선호 ✓.
W: C 선택 시 NI → 보수 -1. NC 선택 시 I → 보수 2.
W: NC 선호. 분리 균형 깨짐.

### 정답
혼합균형탐색필요 / 풀링균형=(둘다NC)가능성

### 해설
PBE 분석:
풀링 균형 (둘 다 NC):
수신자 믿음 (NC 관찰): P(S|NC) = 2/3.
수신자 기댓값 (I): 2/3×1+1/3×1 = 1 > 0.
수신자: I 선택.
S: NC→I=3, C→? (균형 외 경로 믿음 자유).
W: NC→I=2, C→? .
둘 다 NC: 풀링 균형 가능.
균형 외 경로: C 관찰 시 믿음 자유.
P(W|C)=1 설정 → NI → 이탈 유인 없음.
풀링 균형 (NC, NC) with P(W|C)=1 존재.

## 문제 3

반복 게임의 포크 정리:
할인율 δ가 충분히 높으면
어떤 결과도 (파레토 개선 균형으로) 달성 가능.

정확한 진술:
게임 G의 어떤 결과 v가
개인 합리성 조건 v ≥ min-max 값을 만족하면
충분히 높은 δ에서 그 결과가 균형 결과.

(1) min-max 값이란?
(2) 포크 정리의 한계는?
(3) 정확한 균형 전략 구성은?

### 힌트 1
min-max: 상대가 나를 최소화할 때의 나의 최대값.
= 상대가 최악으로 굴어도 내가 보장받는 값.

### 힌트 2
포크 정리 한계:
균형이 너무 많음 → 어느 것이 실현될지 모름.
공개 개입 (public monitoring) 필요.

### 정답
min-max=상대최악시보장값 / 포크=달성가능결과무한 / 균형선택문제심화

### 해설
포크 정리:
min-max 값 m_i:
m_i = max_{σᵢ} min_{σ₋ᵢ} uᵢ(σᵢ, σ₋ᵢ).
= 상대가 나를 처벌해도 내가 보장받는 최소.
포크 정리:
v ≥ m (모든 플레이어) → δ 충분히 크면 달성 가능.
죄수의 딜레마: m = 1 (DD 보수).
(3,3) > (1,1) → 달성 가능.
무한 반복 + 공개 관찰 + 충분한 인내 → 협력.
한계:
균형이 (연속체만큼) 많음.
균형 선택 문제 해결 안 됨.
유한 반복: 포크 정리 성립 안 함.
불완전 모니터링: 약한 포크 정리만.

## 문제 4

역경매와 조달:
정부가 계약 업체 선정 (역경매).
업체들의 비용 ci ~ [0,1] (균등분포, 독립).
정부 목표: 최저 비용 선택 + 진실 유인.

비크레이 역경매:
낙찰: 최저 입찰 업체.
지불: 두 번째 낮은 입찰가.

(1) 진실 입찰이 지배 전략인 이유는?
(2) 기대 조달 비용은?
(3) 두 번째 낮은 비용의 기대값은?

### 힌트 1
2위 가격 역경매 (최저가 낙찰, 2위 최저가 지불):
입찰 b > c: 손해 없이 낙찰 확률 감소.
입찰 b < c: 낙찰 시 손해.
→ b = c 지배 전략.

### 힌트 2
n명 입찰, 균등분포 [0,1]:
2번째 최솟값 기대 = (n-1)/(n+1).
n=2: 기대 비용 = 1/3.

### 정답
진실입찰=지배전략 / n=2기대비용=1/3 / 일반n=(n-1)/(n+1)

### 해설
비크레이 역경매:
b = c: 낙찰 시 이익 = 지불-비용 = c₍₂₎-c ≥ 0 (c가 최저이면).
b > c: 낙찰 확률 감소 (손해 없지만 낙찰 기회 손실).
b < c: 낙찰 시 이익 = c₍₂₎-b? 아니면 손해?
정확히: 낙찰 시 지불 = 2위 최저가.
b < c이면 낙찰되어도 손해: 지불-비용 < 0.
→ b = c 최적.
n명 균등분포 최솟값의 순서 통계:
E[X₍ₖ₎] = k/(n+1).
2번째 최솟값 = 2번째 작은 값:
n=2: E[min₂] = E[X₍₂₎] = 2/3? 아니면 1/3?
실제: 최솟값 E[X₍₁₎] = 1/(n+1).
2번째 최솟값 E[X₍₂₎] = 2/(n+1).
n=2: E[X₍₂₎] = 2/3 (최댓값의 기대).
조달: 최솟값을 원함 → E[X₍₁₎] = 1/3.
2위 지불 = 두 번째 낮은 입찰 = n=2에서 E = 1/3+?
수입 동등성: 1위와 2위 가격 경매 기대 수입 동일.

## 문제 5

전략형과 전개형 게임의 동등성:
모든 전개형 게임은 전략형으로 변환 가능.
하지만 반드시 같은 균형을 가지지는 않음.

예시:
```
A: L 또는 R.
R이면 B: U 또는 D.
보수: L→(1,1), R-U→(2,2), R-D→(0,3).
```

전략형의 내쉬 균형은?
전개형의 역방향 귀납 균형은?

### 힌트 1
전략형:
A: {L, R}, B: {U, D} (항상).

```
        B:U      B:D
A:L    (1,1)    (1,1)
A:R    (2,2)    (0,3)
```

### 힌트 2
전략형 균형: (L,D) = (1,1) 확인.
B: D 선택 시 A가 L → B 이탈? U→(2,2)>D→(1,1). 이탈.

### 정답
전개형균형=(R,U) / 전략형에(L,D)도균형 / (L,D)=비신뢰성위협

### 해설
균형 개념의 세밀화:
전략형 내쉬 균형:
(L, D): A=1, B=1.
A 이탈: R → B가 D이면 0 < 1. 안 함.
B 이탈: U → 하지만 A가 L이면 B=(1,1)→(1,1) 무차별.
→ (L,D)는 내쉬 균형.
하지만 비신뢰성 위협:
B의 D 위협: "A가 R 선택시 D로 처벌."
실제 R이면: B는 U(2) > D(3)? 아니 D(3) > U(2).
B는 실제로 D 선택 (3>2).
→ A는 R → B는 D → A는 L.
역방향 귀납: R이면 B는 D(3>2). A: R→0, L→1.
A는 L 선택. 균형 = (L, D).
수정: B의 보수를 재확인.
R-U: (2,2), R-D: (0,3) → B는 D(3) > U(2). D 선택.
A는 R→0, L→1. A: L 선택.
역방향 귀납 균형 = (L, D)!

## 문제 7

협력 게임의 핵(Core)과 안정성:
매칭 이론 — 안정적 결혼 문제.

4명(A,B) (남성), 4명(a,b) (여성).
선호:
A: a > b, B: b > a.
a: A > B, b: A > B.

안정 매칭이란?
게일-샤플리 알고리즘으로 구하세요.

### 힌트 1
안정 매칭: 블로킹 쌍이 없는 매칭.
블로킹 쌍: 서로 현재 파트너보다 상대를 선호.

### 힌트 2
게일-샤플리 (남성 제안):
1라운드: A→a, B→b.
a: A 수락. b: A vs B → A 선호 → B 거절.
2라운드: B→a.
a: A vs B → A 선호 → B 거절.
B 제안 대상 없음.

### 정답
안정매칭=(A-a, B-b) / b가A선호해도A는a와매칭

### 해설
게일-샤플리 알고리즘:
남성 제안 버전:
A→a (1지망), B→b (1지망).
a: A 유지. b: A 없으므로 B 유지.
결과: (A-a, B-b).
블로킹 쌍 확인:
(A-b): A는 a>b이므로 b 선호 안 함. 블로킹 아님.
(B-a): a는 A>B이므로 A 선호. B는 a>b이므로 a 선호.
→ (B-a)는 블로킹 쌍!
재검토: a가 B를 거절했으므로 (A-a)는 안정.
(B-a): B가 a 원하고 a가 B 원함 → 블로킹 쌍.
하지만 a는 현재 A와 더 행복 → a는 B로 이동 안 함.
안정 매칭 = (A-a, B-b) 불안정? 수정 필요.
정확: (B-a)에서 a는 A>B이므로 B 선호 안 함.
블로킹 쌍 아님. (A-a, B-b) 안정.

## 문제 8

순차 합리성과 전향 귀납:
역방향 귀납과 반대 방향.

전향 귀납(Forward Induction):
"상대방이 왜 이 신호를 보냈는가?"

```
게임:
A: Out (보수 2,0) 또는 In.
In이면: 두 플레이어 동시 선택.
        B:L    B:R
A:L    (3,1)  (0,0)
A:R    (0,0)  (1,3)
```

전향 귀납 예측은?

### 힌트 1
A가 In을 선택했다면: Out(2)보다 나은 결과 기대.
→ A는 최소 2 이상 기대.
A:L가 3 > 2이므로 A는 L을 의도.

### 힌트 2
B: A가 In을 선택했다면 L을 의도 (전향 귀납).
→ B는 L 선택 (1>0).
균형: In, (L,L) → (3,1).

### 정답
전향귀납=(In,L,L)→(3,1) / A의In신호=L의도

### 해설
전향 귀납:
A가 Out을 포기했다면 이유가 있음.
A의 최선: In-L = 3 > Out = 2.
In-R = 1 < Out = 2 → A는 R 선택 안 함.
→ A가 In을 선택했다면 L 의도.
B는 이 추론을 하고 L 선택 (1>0).
역방향 귀납과의 차이:
역방향: 끝에서 거꾸로 (미래 → 현재).
전향 귀납: 관찰된 행동에서 의도 추론 (과거 → 현재).
균형 정제:
전향 귀납 = 강한 균형 정제.
역방향 귀납 = 부분게임 완전 균형.
전향 귀납 > 역방향 귀납 (더 강한 요구).

## 문제 9

경매 이론의 최전선:
복잡한 경매 환경:
① 상호 보완적 가치: A와 B 함께 가치 더 높음.
② 상호 대체적 가치: A 또는 B 중 하나면 충분.
③ 예산 제약: 총 지출 한도.

이 환경에서 VCG의 한계는?

### 힌트 1
상호 보완: 조합 입찰 필요.
예산 제약: VCG 지불이 예산 초과 가능.

### 힌트 2
VCG + 예산 제약: 진실 유인 깨질 수 있음.
대안: CA-VCG (조합 VCG), 코어 가격 설정.

### 정답
VCG한계=예산제약·계산복잡도 / 조합VCG·코어가격이대안

### 해설
VCG의 한계:
예산 균형:
VCG 세금 합 < 비용 → 보조 필요.
예산 제약 + VCG: 지불 불가 시 전략 왜곡.
계산 복잡도:
최적 배분 = NP-hard (조합 경매).
VCG 계산 자체가 어려움.
담합 취약성:
특정 조건에서 입찰자들이 담합 가능.
최적 경매 (마이어슨) 적용 어려움.
현실 대안:
코어 가격 설정: 코어 안의 가격 선택.
FA 경매(Ascending, FAA): 단계적 가격 인상.
PO-VCG: 파레토 최적 + VCG.
프랙티컬 메커니즘:
완벽한 이론 > 실용적 타협.
간단성, 투명성, 계산 가능성이 중요.

## 문제 10

심화 구간 입문 종합:
다음 상황의 전략 구조를 수학적으로 분석하세요.

기후 협약 게임:
n개 국가가 온실가스 감축 여부 결정.
감축 비용: c (각국 동일).
감축 국가 수 k이면 각국 이익: b(k) (k에 증가).
b(n) > c > b(1): 전체 감축이 이익, 혼자 감축은 손해.

(1) 이 게임의 구조는?
(2) 내쉬 균형은?
(3) 국제 협약 메커니즘은?

### 힌트 1
각국 감축 여부: 다자 죄수의 딜레마.
비감축 = 지배 전략 (b(k) 증가해도 자신 비용 c).

### 힌트 2
내쉬 균형: 모두 비감축 (공공재 문제).
사회 최적: 모두 감축 (n×b(n) - n×c > 0).

### 정답
다자죄수딜레마 / 균형=모두비감축 / 협약=처벌조건부·무역연계

### 해설
기후 게임의 수학:
국가 i의 이익:
감축 시: b(k) - c.
비감축 시: b(k).
→ 비감축이 항상 c만큼 유리.
지배 전략: 비감축.
내쉬 균형: 모두 비감축 → b(1)×n 이익.
사회 최적: 모두 감축 → b(n)×n - c×n.
b(n)×n - c×n > b(1)×n (b(n)>c+b(1)/n 가정).
메커니즘 해결:
조건부 협약: k개국 이상 서명 시 발효.
최저 규모: 협약 이익이 탈퇴보다 높아야.
무역 연계: 비감축 국가에 무역 제재.
기술 이전: 감축 비용 c 줄이기.
반복 게임: 미래 협력 위협.
파리 협약의 구조:
자발적 서약 (NDC) + 투명성 메커니즘.
강제력 부족 → 약한 균형.
