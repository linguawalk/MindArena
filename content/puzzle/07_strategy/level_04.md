> 내쉬 균형의 개념을 깊이 탐구하고
> 다양한 전략적 상황에 적용합니다.

## 문제 1

다음 보수 행렬에서 내쉬 균형을 모두 찾으세요.

```
           B: 왼쪽    B: 오른쪽
A: 위      (2, 1)    (0, 0)
A: 아래    (0, 0)    (1, 2)
```

### 힌트 1
내쉬 균형: 어느 플레이어도 단독으로 이탈 유인 없는 상태.

### 힌트 2
(위, 왼쪽): A=2, B=1.
A가 아래로? → (0,0), A 손해. B가 오른쪽으로? → (0,0), B 손해.
→ 내쉬 균형.

### 정답
(위, 왼쪽)과 (아래, 오른쪽) 두 개의 내쉬 균형

### 해설
내쉬 균형 확인:
(위, 왼쪽): A=2, B=1.
A 이탈: 아래→0 < 2. 이탈 안 함.
B 이탈: 오른쪽→0 < 1. 이탈 안 함.
→ 내쉬 균형 ✓.
(아래, 오른쪽): A=1, B=2.
A 이탈: 위→0 < 1. 이탈 안 함.
B 이탈: 왼쪽→0 < 2. 이탈 안 함.
→ 내쉬 균형 ✓.
(위, 오른쪽): A=0.
A 이탈: 아래→1 > 0. 이탈 함 → 균형 아님.
(아래, 왼쪽): B=0.
B 이탈: 오른쪽→2 > 0. 이탈 함 → 균형 아님.
두 순수 전략 균형 = 조정 게임(coordination game).

## 문제 2

성 대결 게임(Battle of the Sexes):
커플이 각각 원하는 이벤트:
A: 축구 선호. B: 오페라 선호.
하지만 함께 하는 것이 혼자보다 중요.

```
           B: 축구    B: 오페라
A: 축구    (3, 1)    (0, 0)
A: 오페라  (0, 0)    (1, 3)
```

(1) 순수 전략 내쉬 균형은?
(2) 혼합 전략 균형은?
(3) 어떻게 조정할 수 있나요?

### 힌트 1
(축구, 축구)와 (오페라, 오페라) 두 균형.
혼합 전략: A의 축구 확률 p, B의 축구 확률 q.

### 힌트 2
A 무차별 조건: 3q+0(1-q) = 0×q+1×(1-q).
3q = 1-q → 4q = 1 → q = 1/4.

### 정답
순수: (축구,축구),(오페라,오페라) / 혼합: A=3/4축구, B=1/4축구

### 해설
혼합 전략 균형:
B가 q 확률로 축구를 낼 때 A의 기댓값:
축구: 3q, 오페라: 1-q.
A 무차별: 3q = 1-q → q = 1/4.
A가 p 확률로 축구를 낼 때 B의 기댓값:
축구: p, 오페라: 3(1-p).
B 무차별: p = 3-3p → p = 3/4.
혼합 균형: (3/4, 1/4).
기댓값: 3×(3/4)×(1/4) = 9/16 ≈ 0.56 (두 순수 균형보다 낮음).
조정 방법:
사전 의사소통.
번갈아 양보 (반복 게임).
협조 기관 (중재자).

## 문제 3

공공재 게임:
4명이 각자 10만원을 가집니다.
원하는 만큼 공공 자금에 기부.
모인 금액을 2배로 늘려 4명에게 균등 분배.

각자 얼마를 기부해야 하나요?

### 힌트 1
내가 1원 기부 → 공공 자금 2원 증가 → 내 몫 0.5원.
순이익: -0.5원 (기부하면 손해).

### 힌트 2
모두 기부 안 함 = 내쉬 균형 (사회 최적은 아님).
모두 기부 시 각 20만원 (10×2×4÷4).

### 정답
내쉬 균형: 기부 0 / 사회 최적: 모두 기부

### 해설
공공재 게임 분석:
1원 기부의 순이익: 2×(1/4) - 1 = -0.5.
기부는 항상 손해 → 지배 전략: 기부 안 함.
내쉬 균형: 모두 0 기부 → 각 10만원.
사회 최적: 모두 10만원 기부 → 총 80만원 → 각 20만원.
사회 딜레마:
개인 합리성 → 집단 손실.
공공재 문제의 수학적 표현.
해결책:
처벌 메커니즘: 기부 안 한 사람 제재.
조건부 협력: 다른 사람 기부 확인 후 기부.
작은 집단: 평판 효과.
반복 게임: 협력 유지 가능.

## 문제 4

스태그 헌트(Stag Hunt):
두 사냥꾼이 각각 선택:
함께 사슴 사냥 또는 혼자 토끼 사냥.

```
           B: 사슴    B: 토끼
A: 사슴    (4, 4)    (0, 2)
A: 토끼    (2, 0)    (2, 2)
```

(1) 내쉬 균형은?
(2) 어느 균형이 더 좋은가요?
(3) 사회적 트랩이란?

### 힌트 1
(사슴, 사슴): A=4, B=4.
이탈? A가 토끼로: 2 < 4. 안 함. B도 마찬가지.
→ 균형.

### 힌트 2
(토끼, 토끼): A=2, B=2.
이탈? A가 사슴으로: 0 < 2. 안 함 (B가 토끼이면 사슴 사냥 혼자 못함).
→ 균형.

### 정답
두 균형: (사슴,사슴)=(4,4), (토끼,토끼)=(2,2)

### 해설
스태그 헌트 분석:
(사슴, 사슴): 파레토 우월 균형 (둘 다 더 좋음).
(토끼, 토끼): 위험 지배 균형 (더 안전함).
사슴 사냥의 위험:
상대가 토끼를 선택하면 0.
토끼는 확실히 2.
위험 회피 → 토끼 선택 → 나쁜 균형.
사회적 트랩:
더 좋은 균형이 있어도 위험 회피로 나쁜 균형에.
군비 경쟁, 환경 보호, 기술 표준화.
해결: 신뢰 구축, 의사소통, 조정 기관.

## 문제 5

쿠르노 경쟁:
두 기업이 동시에 생산량 결정.
시장 가격: P = 100 - Q₁ - Q₂.
한계비용: c = 10.
각 기업의 이윤 최대화 생산량은?

### 힌트 1
기업 1의 이윤: π₁ = (P-c)Q₁ = (90-Q₂-Q₁)Q₁.
∂π₁/∂Q₁ = 90-Q₂-2Q₁ = 0.

### 힌트 2
반응함수: Q₁ = (90-Q₂)/2.
대칭: Q₂ = (90-Q₁)/2.

### 정답
Q₁=Q₂=30, P=40, 각 이윤=900

### 해설
쿠르노 균형:
Q₁ = (90-Q₂)/2, Q₂ = (90-Q₁)/2.
연립방정식:
Q₁ = (90-Q₂)/2 → 2Q₁ = 90-Q₂.
Q₂ = (90-Q₁)/2 → 2Q₂ = 90-Q₁.
2Q₁+Q₂=90, Q₁+2Q₂=90.
Q₁=Q₂=30.
P = 100-30-30 = 40.
π₁ = (40-10)×30 = 900.
비교:
독점: Q=45, P=55, π=2025.
쿠르노: 각 Q=30, 총Q=60, P=40.
완전 경쟁: Q=90, P=10.
쿠르노는 독점과 경쟁 사이.

## 문제 6

버트랑 경쟁:
두 기업이 가격을 동시에 설정.
(동일 제품, 낮은 가격 기업이 모든 고객 획득)
한계비용 c = 10.

균형 가격은?

### 힌트 1
한 기업이 P > c로 가격 설정 시
다른 기업은 P-ε으로 이익 가능.

### 힌트 2
이 경쟁이 계속되면 결국 P = c.

### 정답
P₁ = P₂ = c = 10 (한계비용 = 경쟁 가격)

### 해설
버트랑 역설:
단 2개 기업이어도 완전 경쟁 결과.
이유: 가격 경쟁의 치킨 게임.
어느 기업도 P > c이면 다른 기업이 P-ε으로 시장 독점.
→ P = c까지 낮아짐.
쿠르노 vs 버트랑:
쿠르노 (수량 경쟁): P > c, 이윤 > 0.
버트랑 (가격 경쟁): P = c, 이윤 = 0.
실제 시장:
제품 차별화: 완전한 버트랑 아님.
용량 제약: 쿠르노에 가까움.
반복 게임: 암묵적 담합 가능.
버트랑 역설 해결: 에지워스 사이클, 혼합 전략.

## 문제 7

레오(Leao) 시장 게임:
두 기업이 시장 진입 여부 결정.
진입 비용: 각 F = 50.
두 기업 모두 진입 시 각 이윤 = 30 (비용 후).
한 기업만 진입 시 이윤 = 100.

```
           B: 진입    B: 미진입
A: 진입    (-20,-20)  (50, 0)
A: 미진입  (0, 50)    (0, 0)
```

내쉬 균형과 효율적 결과는?

### 힌트 1
(진입, 미진입)과 (미진입, 진입): 두 균형.
(진입, 진입): (-20,-20) - 균형 아님.

### 힌트 2
사회 효율: 한 기업만 진입 = 총 50.
두 기업 진입 = 총 -40 (비효율).

### 정답
두 순수 균형: (진입,미진입), (미진입,진입) / 효율적=한 기업만 진입

### 해설
진입 게임:
(진입, 미진입): A=50, B=0.
A 이탈: 미진입→0 < 50. 안 함.
B 이탈: 진입→-20 < 0. 안 함 ✓.
(미진입, 진입): 대칭으로 균형 ✓.
(진입, 진입): A=-20 → 미진입으로 이탈.
균형 아님.
(미진입, 미진입): B가 진입→50 > 0. 이탈 ✓.
균형 아님.
조정 문제:
어느 균형에 도달하는가?
사전 의사소통, 진입 순서, 역사.
실제: 선점 효과 → 먼저 진입한 기업이 시장 차지.

## 문제 8

다단계 게임:
연쇄점 게임(Chain Store Paradox):
본사(독점)가 각 지역에서 잠재적 진입자와 대결.
20개 지역, 순서대로.

전략:
진입자 진입 시: 본사가 가격 전쟁(비용) 또는 수용.
단순 역방향 귀납: 마지막 지역 → 수용.
→ 앞 지역도 모두 수용.

하지만 현실에서는?

### 힌트 1
20개 지역 모두 순수 역방향 귀납 → 모두 진입.
하지만 현실에서 본사는 초기에 강하게 대응.

### 힌트 2
평판 효과: 초기 가격 전쟁 → 후기 진입 억제.
역방향 귀납의 한계.

### 정답
이론=모두진입 / 현실=평판으로진입억제 / 유한반복게임의역설

### 해설
연쇄점 역설(Selten, 1978):
역방향 귀납: 마지막 지역부터 모두 수용.
모든 지역에서 진입 허용.
현실과 괴리:
실제 기업은 초기에 강하게 싸움.
이유: 평판(reputation) 효과.
초기 강경 대응 → 후기 진입 억제 효과.
인식론적 게임 이론:
역방향 귀납이 성립하려면 공통 지식 필요.
현실: 완전 합리성 가정 의심.
불완전 정보: 본사 타입 불확실 → 강경 타입 가장.
유한 반복 vs 무한 반복의 차이.

## 문제 9

투표 역설:
3명(A,B,C)이 3가지 정책(X,Y,Z) 중 선택.
선호:
A: X > Y > Z.
B: Y > Z > X.
C: Z > X > Y.

다수결로 결정하면?

### 힌트 1
X vs Y: A(X)+C(X) vs B(Y) → X 승.
Y vs Z: A(Y)+B(Y) vs C(Z) → Y 승.
X vs Z: B(Z)+C(Z) vs A(X) → Z 승.

### 힌트 2
X>Y>Z>X: 순환!
사회적 선호의 비추이성.

### 정답
투표 역설: X>Y, Y>Z, Z>X (순환)

### 해설
콩도르세 역설(1785):
개인 선호는 합리적이지만 집계 선호는 비합리적.
X>Y (A+C 2:1), Y>Z (A+B 2:1), Z>X (B+C 2:1).
X>Y>Z>X: 순환 → 사회적 선호 불존재.
애로의 불가능성 정리(1951):
합리성 + 파레토 + 무관 독립성 + 비독재 → 불가능.
어떤 투표 규칙도 이 조건들을 동시 충족 불가.
실제 함의:
의제 설정 권한 = 권력.
어떤 안건부터 투표하느냐가 결과 결정.
전략적 투표(strategic voting): 진짜 선호 숨기기.

## 문제 10

코사 노스트라 문제:
마피아 조직의 신뢰 딜레마:
조직원들이 법정에서 묵비권(C) vs 자백(D).

```
           B: 묵비권   B: 자백
A: 묵비권  (-1, -1)   (-10, 0)
A: 자백    (0, -10)   (-5, -5)
```

(1) 내쉬 균형은?
(2) 마피아가 침묵을 강제하는 방법은?
(3) 반복 게임에서의 변화는?

### 힌트 1
고전적 죄수의 딜레마.
지배 전략: 자백.

### 힌트 2
마피아: 자백자 처벌 → 보수 변화.
(-10, 0) → (-10-처벌, 0) → (-∞, 0).
자백의 유인 제거.

### 정답
균형=자백 / 처벌위협=침묵강제 / 반복게임=협력가능

### 해설
마피아의 게임 이론:
단기: 자백이 지배 전략.
마피아 해결:
자백자 처벌(-K) → 자백의 보수 감소.
K > 5이면: 자백(-5-K) < 묵비(-1).
묵비가 우월 전략으로 전환.
오메르타(Omerta): 침묵의 코드 = 처벌 약속.
신뢰할 수 있는 약속(credible commitment).
반복 게임:
무한 반복: 팃포탯으로 협력 가능.
미래 손실 > 현재 이익 → 침묵 유지.
현실 조직:
명예 코드, 사회적 규범, 처벌 위협.
제도가 개인 인센티브를 바꿈.
