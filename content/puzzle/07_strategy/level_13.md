## 문제 1

전략과 생태학:
포식자-피식자 전략 게임.
로트카-볼테라 방정식:
dx/dt = αx - βxy (피식자)
dy/dt = δxy - γy (포식자)

(1) 게임 이론으로 이 동학을 설명하세요.
(2) 진화적 안정 전략과 생태 균형의 연결은?
(3) 인간 경쟁에 대한 함의는?

### 힌트 1
피식자: 회피 vs 방어 전략.
포식자: 공격 vs 에너지 보존 전략.
균형: 공진화.

### 힌트 2
생태 균형 = 게임 이론 균형.
포식자 과잉 → 피식자 감소 → 포식자 감소.
= 부의 피드백 = 안정 균형.

### 정답
생태균형=ESS / 공진화=전략적상호적응 / 인간=시장경쟁유사

### 해설
생태-전략 연결:
로트카-볼테라 = 전략적 상호 의존.
피식자 전략: 빠른 번식(r 전략) vs 적은 번식+강한 방어(K 전략).
포식자 전략: 추격 vs 매복 vs 협동 사냥.
공진화 (coevolution):
포식자와 피식자가 서로를 선택압으로.
군비 경쟁: 치타 ↔ 가젤 속도 경쟁.
ESS와 생태 균형:
생태 균형 = 어떤 돌연변이 전략도 침입 불가.
= 진화적 안정 전략의 생태적 표현.
인간 경쟁:
기업 생태계: 포식자(대기업)-피식자(스타트업).
시장 균형 = 전략적 공진화.
군비 경쟁: 실리콘밸리 인재 경쟁.

## 문제 2

전략과 언어:
협상에서 언어의 전략적 역할.

언어 행위 이론 (오스틴, 설):
①발화 행위: 말하기.
②발화 수반 행위: 약속, 위협, 요청.
③발화 효과 행위: 실제 효과.

게임 이론에서:
위협: 믿을 수 있는가?
약속: 자기 구속적인가?
싸구려 토크: 정보를 전달하는가?

(1) 위협의 신뢰성 조건은?
(2) 언어가 전략을 어떻게 변환하는가?
(3) 협상에서 프레이밍의 효과는?

### 힌트 1
믿을 수 있는 위협: 실행이 자신에게도 이익.
믿을 수 없는 위협: 실행 시 자신도 손해.

### 힌트 2
프레이밍: "90% 생존" vs "10% 사망" = 같은 정보, 다른 반응.
= 동일한 게임, 다른 선택.

### 정답
위협신뢰=실행이익 / 언어=게임틀재구성 / 프레이밍=인지편향활용

### 해설
언어와 전략:
믿을 수 있는 위협:
"협상 결렬 시 소송." → 소송 비용 감수 의지?
헌신 장치: "이미 변호사 고용" → 위협 신뢰.
언어의 전략적 기능:
약속: 자신을 구속 (헌신 장치).
위협: 상대 행동 변화.
신호: 정보 전달.
프레이밍: 게임 틀 재구성.
프레이밍 효과 (트버스키-카너만):
손실 프레임: "200명 죽는다" → 위험 회피.
이익 프레임: "400명 살린다" → 위험 추구.
같은 결과, 다른 선택.
전략적 응용:
협상: "이 조건이 당신에게 최선" vs "저 조건이 우리에게 최소."
광고: 손실 강조 vs 이득 강조.
정치: 프레임 전쟁.

## 문제 3

전략과 건축:
도시 설계의 전략 게임.

게임 이론적 도시 문제:
①주차 외부성: 모두 차를 가져오면 주차 불가.
②젠트리피케이션: 개발 → 원주민 퇴출.
③공유 공간: 공원 = 공공재.
④NIMBY: 모두 필요하지만 내 뒷마당에는 반대.

각 문제의 전략 구조와 해결책은?

### 힌트 1
주차: 죄수의 딜레마 (모두 가져오면 모두 손해).
젠트리피케이션: 연쇄 반응 게임.

### 힌트 2
NIMBY: 공공재 게임 + 지역 이기주의.
해결: 보상 + 분산 배치.

### 정답
각문제=다른게임구조 / 공통=외부성 / 해결=제도·보상·설계

### 해설
도시 전략 게임:
주차 외부성:
모두 차 → 주차 불가 → 모두 손해.
해결: 주차 요금 (피구세), 대중교통.
젠트리피케이션:
초기 예술가 → 카페 → 부동산 상승 → 원주민 퇴출.
선점 효과 + 연쇄 반응.
해결: 임대료 규제, 지역 개발 이익 공유.
공원 (공공재):
무임승차: 관리 안 하고 이용.
해결: 지역세, 공동 관리.
NIMBY:
쓰레기 처리장, 정신병원 등 = 사회 필수.
해결: 보상 + 투명한 입지 선정.
보상: "당신 지역에 오는 대신 세금 할인."
도시 설계의 교훈:
공간 배치 = 인센티브 구조.
좋은 도시 설계 = 좋은 메커니즘 설계.

## 문제 4

전략과 의학:
의료 의사결정의 게임 이론.

시나리오들:
①항생제 내성: 각자 최대 사용 → 내성 증가.
②임상 시험: 참여 vs 무임승차.
③장기 기증: 공공재 딜레마.
④의료 보험: 역선택 + 도덕적 해이.

가장 복잡한 문제를 선택해 분석하세요.

### 힌트 1
항생제 내성: 공유지의 비극.
각자 사용 최적화 → 사회 전체 내성 증가.

### 힌트 2
해결: 사용 규제, 새 항생제 개발 인센티브, 진단 개선.

### 정답
항생제=공유지비극 / 규제+인센티브해결 / 글로벌조율필요

### 해설
항생제 내성 게임:
개인 최적: 효과 있는 항생제 사용.
집단 결과: 내성 증가 → 효과 감소.
죄수의 딜레마 / 공유지의 비극.
글로벌 공공재 문제:
한 국가의 규제 = 다른 국가 이익.
무임승차 유인.
해결 메커니즘:
사용 규제: 처방 기준 강화.
새 항생제 인센티브: 수익 보장 계약.
신속 진단: 불필요한 처방 감소.
교육: 환자 인식 개선.
국제 조율: WHO 프레임워크.
게임 이론 교훈:
의료 = 전략적 상호 의존.
메커니즘 설계 = 의료 정책.
글로벌 조율 없이는 해결 불가.

## 문제 5

전략과 스포츠:
게임 이론으로 스포츠 분석.

페널티킥:
키커: 왼쪽(L) 또는 오른쪽(R).
골키퍼: L 또는 R.
성공 확률:

```
           GK:L    GK:R
Kicker:L   0.6     0.9
Kicker:R   0.9     0.6
```

(1) 혼합 전략 균형은?
(2) 실제 데이터와 비교하면?
(3) 다른 스포츠에서의 응용은?

### 힌트 1
키커 무차별 조건 (q: GK가 L 확률):
0.6q+0.9(1-q) = 0.9q+0.6(1-q).
0.9-0.3q = 0.6+0.3q.

### 힌트 2
0.3 = 0.6q → q = 0.5.
대칭이므로 키커도 p=0.5.

### 정답
균형=각0.5 / 실데이터≈균형지지 / 테니스서브·농구등응용

### 해설
페널티킥 혼합 균형:
q: GK가 L 선택 확률.
키커 무차별: 0.6q+0.9(1-q) = 0.9q+0.6(1-q).
0.9-0.3q = 0.6+0.3q → q=0.5.
키커도 p=0.5 (대칭 보수 행렬).
실제 데이터 (팔라시오-우에르타, 2003):
프로 페널티킥: 혼합 전략 균형 근처.
실증 지지: 전문가는 균형에 가까움.
다른 스포츠:
테니스 서브: 넓은 서브 vs 몸 서브.
농구 수비: 맨투맨 vs 지역.
야구 투구: 패스트볼 vs 변화구.
모두 혼합 전략 균형.
패턴 예측: 운동 선수들이 무의식적으로 균형 학습.

## 문제 6

전략과 외교:
국제 관계의 게임 이론.

쿠바 미사일 위기 (1962):
미국 vs 소련.
미국: 봉쇄(B) 또는 공습(A).
소련: 철수(W) 또는 유지(M).

```
           소련:W     소련:M
미국:B    (3, 3)    (2, 1)
미국:A    (4, 1)    (1, 4)
```

(1) 내쉬 균형은?
(2) 실제 결과와의 연결은?
(3) 핵 억지의 게임 이론은?

### 힌트 1
(B,W): 미=3,소=3.
미 이탈: A→소련 W면 4 > 3. 이탈 ✓.
(B,W) 균형 아님.

### 힌트 2
(A,M): 미=1,소=4.
미 이탈: B→소련 M면 2 > 1. 이탈 ✓.
균형 아님.
혼합 전략 균형 탐색.

### 정답
순수균형없음 / 혼합균형존재 / 실제=협상으로(B,W)달성

### 해설
쿠바 미사일 위기 분석:
순수 전략 균형 없음 (치킨 게임 구조).
혼합 균형:
p: 미국이 B 선택 확률.
소련 무차별: 3p+1(1-p) = p+4(1-p).
2p+1 = 4-3p → 5p=3 → p=3/5.
q: 소련이 W 선택 확률.
미국 무차별: 3q+2(1-q) = 4q+1(1-q).
q+2 = 3q+1 → q=0.5.
실제 결과:
소련이 철수 (미국의 봉쇄에 응).
케네디-흐루쇼프 협상.
미국: 쿠바 불침공 약속 (비밀 터키 미사일 철수).
= 게임을 협조적으로 재구성.
핵 억지 (MAD):
상호 확증 파괴: 공격 → 보복 → 양측 전멸.
공격 보수 = 매우 낮음 → 균형이 비공격.

## 문제 7

전략과 법률:
법경제학의 전략 게임.

소송 게임:
원고: 소송 또는 합의.
피고: 합의 또는 항전.
각자의 소송 비용, 이길 확률.

(1) 소송이 발생하는 조건은?
(2) 합의 범위는?
(3) 배상 규칙이 전략에 미치는 영향은?

### 힌트 1
합의 조건: 원고 최저 수락 < 피고 최대 제시.
원고 최저: P×J - C_P (P=승소확률, J=판결액, C=비용).

### 힌트 2
피고 최대: P×J + C_D (소송 비용 절감).
합의 범위: P×J-C_P < 합의액 < P×J+C_D.
범위 = C_P+C_D > 0.

### 정답
합의범위=소송비용크기 / 소송=낙관편향시발생 / 영국식=소송억제

### 해설
소송 게임 분석:
합의 범위: [P×J-C_P, P×J+C_D].
범위 크기 = C_P+C_D (총 소송 비용).
소송 발생 조건:
합의 범위 없음 = 소송.
원고 낙관편향: P_원고 > P_피고.
→ 원고 최저 > 피고 최대 → 합의 불가.
소송 발생.
배상 규칙:
미국식: 각자 비용 부담.
→ 합의 범위 = C_P+C_D (크다).
→ 소송 적음.
영국식: 패소자 비용 부담.
→ 소송 억제 (비용 위험 큼).
→ 대신 소액 소송 억제.
응용:
지식재산 소송 전략.
소송 자금 조달 (litigation finance).
집단 소송의 메커니즘.

## 문제 8

전략과 교육:
교육 제도의 게임 이론.

입시 경쟁 게임:
n명 학생이 제한된 자리를 두고 경쟁.
각자 공부 시간 tᵢ 선택.
합격 확률: tᵢ/Σtⱼ.
공부 비용: tᵢ.

(1) 내쉬 균형은?
(2) 사회 손실은?
(3) 더 나은 제도는?

### 힌트 1
n=2: π₁ = t₁/(t₁+t₂) × V - t₁.
FOC: t₂V/(t₁+t₂)² = 1.
대칭: t* = V/4 (n=2).

### 힌트 2
총 공부 시간: 2×V/4 = V/2.
가치: V (한 명 합격).
사회 손실: V/2 (공부 비용).

### 정답
균형=t*=V/4·사회손실=V/2 / n증가=과잉공부 / 해결=추첨·다양한경로

### 해설
입시 경쟁의 게임 이론:
n명 대칭 균형: t* = V(n-1)/n².
n→∞: t* → 0, 하지만 총 공부 = n×t* = V(n-1)/n → V.
총 비용 V, 가치 V → 순이익 0.
렌트 소멸(rent dissipation): 경쟁이 가치를 소멸.
현실:
한국 수능: 극심한 경쟁 → 과잉 교육.
사교육비: 사회 손실.
해결책:
추첨: 공부 인센티브 감소.
다양한 입학 경로: 경쟁 분산.
졸업 기준 강화: 입학보다 졸업 어렵게.
교육 가치 자체 증진: 경쟁이 아닌 학습.

## 문제 9

전략과 환경:
탄소 시장의 게임 이론.

배출권 거래제(ETS):
각 기업 감축 비용 다름.
배출권 거래 → 효율적 감축.

게임 이론 문제:
①기업의 로비 전략.
②할당량 협상.
③국제 탄소 시장.

(1) ETS의 효율성 원리는?
(2) 기업의 전략적 행동은?
(3) 국제 탄소 시장의 딜레마는?

### 힌트 1
ETS 효율성: 한계 감축 비용이 낮은 기업이 더 감축.
배출권 가격 = 모든 기업의 한계 비용 균등화.

### 힌트 2
기업 전략: 초기 할당량 최대화 로비.
국제: 탄소 누출 (규제 없는 국가로 이전).

### 정답
ETS=한계비용균등=효율 / 기업=로비·전략적보고 / 국제=탄소누출·무임승차

### 해설
ETS 게임 이론:
효율성:
기업 A: 감축 비용 10원/톤.
기업 B: 감축 비용 30원/톤.
배출권 가격 20원: A가 팔고 B가 삼.
→ 총 비용 최소화.
기업 전략:
할당량 로비: 초기 할당 많을수록 이익.
전략적 보고: 감축 비용 과대 신고.
미래 규제 예상: 현재 과잉 배출.
국제 탄소 누출:
강한 규제 국가 기업 → 약한 규제 국가 이전.
탄소 국경 조정 메커니즘 (CBAM): EU의 해결책.
무임승차: 일부 국가 불참.
파리 협약 한계: 강제력 없음.

## 문제 10

마스터 구간 시작:
자신만의 전략 퍼즐을 설계하세요.

조건:
①실생활과 깊이 연결됨.
②게임 이론 개념이 핵심.
③반직관적이지만 명확한 답.
④여러 전략 개념이 통합됨.
⑤사회적 함의가 있음.

### 힌트 1
좋은 전략 퍼즐: 단순한 상황에서 복잡한 전략 구조.
반직관: 개인 최적 ≠ 집단 최적.

### 힌트 2
실생활: 교통, 의료, 교육, 환경, 경제.

### 정답
개인 창작 (다양한 답 가능)

### 해설
예시 퍼즐: "음식 배달 플랫폼 게임"

상황:
배달 플랫폼이 음식점, 배달원, 소비자를 연결.
세 주체 모두 플랫폼에 의존.

게임 구조:
음식점: 가입 vs 미가입.
배달원: 가입 vs 다른 직업.
소비자: 사용 vs 직접 주문.

외부성:
음식점 많음 → 소비자 가치↑ → 배달원 수입↑.
배달원 많음 → 배달 빠름 → 소비자 사용↑.
네트워크 효과: 모두 연결.

전략 분석:
초기: 닭-달걀 딜레마 (아무도 시작 안 함).
해결: 플랫폼 보조금으로 한쪽 먼저 유치.
성숙기: 독점화 → 수수료 인상 → 모두 피해.
= 공유지의 비극 (플랫폼이 공유지).

사회적 함의:
플랫폼 규제의 필요성.
수수료 상한, 데이터 공유, 공개 표준.
협동조합 모델: 참여자가 플랫폼 소유.
