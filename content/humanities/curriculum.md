# MindArena 인문사회 트랙 커리큘럼

**목표:** 역사·철학·경제·심리·사회를 비판적으로 이해하고 자신의 관점을 구성하는 능력
**방식:** 퍼즐(30%) + AI 보드게임(70%)
**특징:** 정답이 없는 질문 중심 — AI가 소크라테스·역사 인물·이해관계자 역할
**범위:** 역사 → 철학·윤리 → 경제 → 심리 → 사회·정치 → 문화

---

## 교육 방식 상세

```
퍼즐 (30%)
  개념·용어·사실 확인 (역사 연대·철학자·경제 지표)
  논리 구조 분석 (논증 오류 찾기)
  자료 해석 (통계·지도·사료 읽기)

AI 보드게임 (70%)
  소크라테스형:   AI가 끝없이 "왜?"를 질문
  역할극형:       AI가 역사 인물·이해관계자 역할
  토론형:         AI가 반대 입장 논거 제시
  시뮬레이션형:   AI가 여러 역할을 번갈아 맡음
  딜레마형:       AI가 압박 상황 제시, 결정 요구

AI 역할 유형
  소크라테스:     사용자 답변에 계속 "왜?" 질문
  역사 인물:      당대 맥락 속에서 직접 대화
  반론자:         사용자 주장에 가장 강한 반론
  다중 이해관계자: 한 상황에서 여러 입장 순차 제시
  심판:           논거의 강도를 평가하고 피드백
```

---

## 커리큘럼 구조

```
26개 단원 × 평균 6개 역량 = 약 156개 핵심 역량

특징: 모든 단원이 현재와 연결됨
```
