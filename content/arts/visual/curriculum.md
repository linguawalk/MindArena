# MindArena 미술 트랙 커리큘럼 v2

**목표:** 색채·조형·미술사·디자인을 눈으로 직접 체험하며 감각적으로 이해
**방식:** 캐주얼 게임(60%) + 퍼즐(20%) + AI 보드게임(20%)
**핵심 변화:** Canvas API·SVG 기반 시각 인터랙션 게임 전면 도입
**범위:** 색채 → 조형 원리 → 미술사 → 디자인 → 재료 → 창작·비평

---

## 시각 인터랙티브 게임 기술 기반

```
Canvas API (브라우저 내장)
  직접 그리기:  마우스·터치로 선·도형 그리기
  색상 조작:    실시간 색채 혼합·변환
  이미지 처리:  필터·효과 적용
  애니메이션:   움직이는 시각 효과

CSS + JavaScript
  색상 슬라이더:  HSL·RGB 실시간 조작
  드래그앤드롭:   요소 배치·구성
  비교 뷰어:     원본 vs 변형 나란히 보기
  타이머:        틀린 그림 찾기 속도 측정
```
