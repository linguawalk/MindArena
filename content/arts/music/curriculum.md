# MindArena 음악 트랙 커리큘럼 v2

**목표:** 기초 악전부터 실용음악 이론까지 — 눈+귀+손으로 동시에 학습
**방식:** 퍼즐(30%) + AI 보드게임(30%) + 청각·인터랙티브 게임(40%)
**핵심 변화:** 모든 단원에 Web Audio API 기반 청음·연주 게임 추가
**범위:** 기초 악전 → 화성학 → 스케일·모드 → 코드·텐션 → 편곡 → 창작

---

## 청각·인터랙티브 게임 기술 기반

```
Web Audio API (브라우저 내장)
  음정 재생:    특정 Hz로 음 생성·재생
  화음 재생:    여러 음을 동시에 재생
  리듬 재생:    타이밍 기반 시퀀서
  음원 분석:    주파수 분석 (FFT)

인터랙티브 UI
  피아노 건반:  클릭·터치로 음 재생
  음정 슬라이더: 주파수 연속 조절
  드럼 패드:    리듬 입력
  악보 편집기:  음표 배치·재생
```

---

## 커리큘럼 구조

```
28개 단원 × 평균 7개 역량 = 약 196개 핵심 역량
청각·인터랙티브 게임: 28개 (단원당 1개)
AI 보드게임: 13개
퍼즐: 단원당 2-3문제
```
