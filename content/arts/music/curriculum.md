# MindArena 음악 트랙 커리큘럼

**목표:** 기초 악전부터 실용음악 이론(화성학·편곡·재즈이론)까지
**방식:** 퍼즐(이론 확인) + AI 보드게임(청음·작곡·분석 적용)
**비율:** 퍼즐 60% : AI 보드게임 40%
**범위:** 기초 악전 → 화성학 → 스케일·모드 → 코드·텐션 → 편곡 → 장르 이론

---

## 커리큘럼 구조

```
28개 단원 × 평균 7개 역량 = 약 196개 핵심 역량

레벨 구분
  L1 Seed   (단원 1-6):   기초 악전
  L2 Sprout (단원 7-12):  기초 화성학
  L3 Branch (단원 13-18): 스케일·모드·텐션
  L4 Tree   (단원 19-24): 재즈 이론·편곡
  L5 Forest (단원 25-28): 고급 화성·창작
```
