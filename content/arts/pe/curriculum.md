# MindArena 체육 트랙 커리큘럼 v2

**목표:** 스포츠 과학·전술·윤리 이해 + 모바일 신체활동 실제 수행
**방식:** 퍼즐(20%) + AI 보드게임(40%) + 모바일 신체활동 게임(40%)
**특징:** 스마트폰 센서·카메라·GPS를 활용한 실제 움직임 측정
**범위:** 운동과학 → 전술 → 심리 → 윤리 → 경영 → 역사철학

---

## 모바일 신체활동 게임 기술 기반

```
스마트폰 활용 기술
  가속도계    → 걸음수·점프높이·달리기 속도·운동강도 측정
  자이로스코프 → 자세·균형·회전 측정
  GPS         → 이동거리·경로·속도·실외 미션
  카메라      → 동작 자세 분석 (MediaPipe AI)
  마이크      → 박자·리듬 감지·구령 인식
  타이머      → 반응속도·기록 측정

게임 유형
  📱 런너형    실제 달리기 + 즉석 퀴즈 장애물
  📱 동작형    카메라로 자세 인식·교정
  📱 측정형    센서로 운동 데이터 수집·분석
  📱 리듬형    박자에 맞춰 신체 동작 수행
  📱 탐험형    GPS로 실외 미션·오리엔티어링
  📱 배틀형    실시간 운동 기록 경쟁
  📱 챌린지형  일정 기간 목표 달성 미션
```
