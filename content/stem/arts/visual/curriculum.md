# MindArena 미술 트랙 커리큘럼

**목표:** 시각적 인터랙션으로 미술 감각과 이론을 자연스럽게 체득
**방식:** 캐주얼 게임(80%) + 미니 퍼즐(20%)
**특징:** 텍스트 학습 최소화 — 보고·선택·만들며 배우는 방식
**범위:** 색채 이론 → 조형 원리 → 미술사 → 디자인 → 창작

---

## 교육 방식 상세

```
캐주얼 게임 유형 (80%)
  🎨 색상 게임:   슬라이더·팔레트·매칭으로 색 감각 훈련
  🔍 관찰 게임:   틀린 그림 찾기·세부 묘사 맞히기
  🧩 구성 게임:   조각 맞추기·배치하기·균형 잡기
  🖼️ 감상 게임:   작품 분류·시대 맞히기·작가 추론
  ✏️ 창작 게임:   간단한 드로잉·색칠·패턴 완성

미니 퍼즐 (20%)
  개념 확인:      미술 용어·원리·역사 퀴즈
  AI 해설:        게임 완료 후 AI가 미술 원리 설명

AI 역할
  게임 진행자:    힌트 제공·난이도 조절
  해설자:         게임 후 미술 원리 자연스럽게 설명
  비평가:         창작 게임에서 피드백 제공
```

---

## 커리큘럼 구조

```
20개 단원 × 평균 6개 역량 = 약 120개 핵심 역량
각 역량마다 캐주얼 게임 1-2개 + 미니 퍼즐 연결
```
