# MindArena 체육 트랙 커리큘럼

**목표:** 스포츠 과학·전술·윤리를 통해 신체와 경쟁을 깊이 이해
**방식:** 퍼즐(30%) + AI 보드게임(70%)
**특징:** 직접 운동보다 스포츠를 분석·설계·토론하는 지적 접근
**범위:** 운동 과학 → 스포츠 전술 → 스포츠 심리 → 윤리·사회 → 스포츠 경영

---

## 교육 방식 상세

```
퍼즐 (30%)
  스포츠 규칙·역사·기록 퀴즈
  신체 과학 개념 확인
  통계·데이터 해석 문제

AI 보드게임 (70%)
  전술 대결형:   AI가 상대 감독, 전술 의사결정 대결
  시뮬레이션형:  AI가 선수·코치·구단주 역할
  토론형:        AI가 반론자로 스포츠 윤리 토론
  분석형:        AI가 데이터 분석 파트너

AI 역할 유형
  상대 감독:     나의 전술에 맞대응
  데이터 분석가: 통계로 결정 근거 제시
  선수:          훈련·계약·부상 시나리오 협의
  심판·위원회:   규정 위반·윤리 문제 판정
  역사 속 인물:  과거 스포츠 명장면 재현
```

---

## 커리큘럼 구조

```
22개 단원 × 평균 6개 역량 = 약 132개 핵심 역량
퍼즐: 단원당 평균 2개 역량 확인
AI 보드게임: 단원당 1-2개 시나리오
```
