# MindArena 예체능 트랙 커리큘럼

**목표:** 예술과 신체의 언어를 이해하고 창의적 표현 능력 기르기
**방식:** AI가 각 역량에 맞는 문제를 실시간 생성
**범위:** 음악 이론 → 미술·디자인 → 체육·스포츠 → 융합 창작

---

## 커리큘럼 구조

```
18개 단원 × 평균 6개 역량 = 약 108개 핵심 역량
특징: STEM과 예술의 연결 강조
      수학·과학 트랙과 교차 연결
```
