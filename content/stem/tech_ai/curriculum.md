# MindArena 기술·AI 트랙 커리큘럼

**목표:** 기술과 AI의 원리를 이해하고 비판적으로 사용하는 능력
**방식:** AI가 각 역량에 맞는 문제를 실시간 생성
**범위:** 인터넷·데이터 → AI 원리 → 사회적 영향 → 미래 기술

---

## 커리큘럼 구조

```
20개 단원 × 평균 6개 역량 = 약 120개 핵심 역량
특징: 기술 비판적 사고 + 실용적 디지털 리터러시
      인문사회 트랙과 긴밀히 연결
```
