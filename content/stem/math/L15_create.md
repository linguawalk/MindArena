# 수학 L15: 나만의 수학 문제 만들기

**핵심 질문:** 좋은 수학 문제란 무엇인가?
**레벨:** L5 Forest 3단계
**소요 시간:** 40-50분
**연결 개념:** 메타인지, 역설계, 창의적 수학 사고

---

## 개념 소개

수학을 가장 깊이 이해한 사람은
문제를 **푸는** 사람이 아니라
문제를 **만드는** 사람입니다.

좋은 수학 문제의 조건:
1. **명확한 조건** — 무엇을 구하는지 분명해야 합니다
2. **유일한 정답** — 풀이 방법은 여러 가지여도 답은 하나
3. **적절한 난이도** — 너무 쉽지도, 너무 어렵지도 않게
4. **의미 있는 개념** — 중요한 수학 원리를 담아야 합니다
5. **흥미로운 맥락** — 실생활 연결이 있으면 더 좋습니다

---

## 기본 문제 1: 문제 분석

다음 두 문제를 비교하세요.

**문제 A:**
"30의 약수를 모두 구하시오."

**문제 B:**
"두 자리 수 N이 있습니다. N의 약수의 합이
N 자신과 같을 때, N을 구하시오.
(단, 1과 N 자신은 약수에서 제외합니다.)"

어느 문제가 더 좋은 문제인가요? 왜 그런가요?

**분석:**
- 문제 A: 단순 계산, 기억력 테스트
- 문제 B: 개념 이해 필요, 탐구 과정이 중요,
          28이라는 아름다운 답(완전수) 발견

**정답: 28** (1+2+4+7+14 = 28)
이런 수를 **완전수(Perfect Number)**라고 합니다.
다음 완전수는 496, 그 다음은 8128입니다.

## 기본 문제 2: 문제 변형

다음 기본 문제를 더 흥미로운 문제로 바꿔보세요.

**원래 문제:** "2x + 3 = 11을 풀어라."

**변형 예시:**
"편의점에서 같은 종류의 음료 2병과 3,000원짜리 과자를
샀더니 11,000원이었습니다. 음료 1병의 가격은?"

**나만의 변형:**
같은 수학적 내용(일차방정식)을 담지만
더 흥미로운 맥락을 만들어 보세요.

*이 문제에는 정해진 답이 없습니다.
창의적인 변형이 목표입니다.*

## 심화 문제: 나만의 문제 설계

다음 조건을 모두 만족하는 수학 문제를 만드세요.

**조건:**
- L1-L14에서 배운 개념 중 하나를 사용
- 실생활 맥락이 있을 것
- 풀이 과정이 2-3단계 이상일 것
- 정답이 유일할 것

**설계 순서:**
1. 사용할 개념 선택 (예: 확률, 방정식, 수열...)
2. 실생활 맥락 정하기 (예: 스포츠, 음식, SNS...)
3. 숫자와 조건 설정
4. 정답 확인
5. 풀이 검증

**예시 문제:**
"인스타그램에서 첫 날 팔로워가 10명이었습니다.
매일 전날의 1.5배씩 팔로워가 늘어난다면
1주일 후 팔로워는 몇 명인가요?
단, 소수점 이하는 버립니다."

**풀이:**
- 1일차: 10명
- 2일차: 15명
- 3일차: 22명
- 4일차: 33명
- 5일차: 49명
- 6일차: 73명
- 7일차: 109명

등비수열: 10 × 1.5⁶ = 10 × 11.39 ≈ **113명**
(정확한 정수 계산: 113명)

## 실생활 연결

**수학 올림피아드 문제 출제의 비밀**

IMO(국제수학올림피아드) 문제는
전 세계 수학자들이 수년에 걸쳐 만듭니다.

좋은 올림피아드 문제의 특징:
- **우아함:** 복잡한 계산 없이 핵심 아이디어로 풀림
- **놀라움:** 처음엔 어려워 보이지만 핵심을 찾으면 단순
- **보편성:** 어떤 배경 지식 없이도 이해 가능한 문제

수학에서 문제를 만드는 능력은
새로운 수학을 발견하는 능력과 같습니다.

**당신이 만든 문제가 누군가에게**
**영감을 줄 수 있습니다.**

## 🏆 수학 트랙 완주!

**L1부터 L15까지 수학 트랙을 완주했습니다.**

여정을 돌아보면:
```
L1-3   Seed:   수의 기본 구조와 비율
L4-6   Sprout: 방정식·함수·그래프
L7-9   Branch: 확률·통계·수열
L10-12 Tree:   기하·논리·경우의 수
L13-15 Forest: 귀납법·게임이론·창작
```

수학은 계산이 아닙니다.
패턴을 발견하고, 논리로 증명하고,
세상을 수학의 언어로 표현하는 것입니다.
