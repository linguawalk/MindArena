# MindArena 수학 트랙 커리큘럼

**목표:** Brilliant.org 수준의 수학 학습 경험 제공
**방식:** AI가 각 역량에 맞는 문제를 실시간 생성
**범위:** 중학 기초 ~ 고교 전과정 + 대학 입문

---

## 커리큘럼 구조

```
25개 단원 × 평균 7개 역량 = 약 175개 핵심 역량
AI 문제 생성: 역량당 무제한
난이도 조절: 쉬움 / 보통 / 어려움 / 심화
```
