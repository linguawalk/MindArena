# 수학 L5: 함수란 무엇인가

**핵심 질문:** 입력과 출력의 관계를 어떻게 표현하는가?
**레벨:** L2 Sprout 2단계
**소요 시간:** 20-25분
**연결 개념:** 정의역, 치역, 합성함수, 알고리즘

## 개념 소개

자판기에 500원을 넣으면 콜라가 나옵니다.

500원 → [자판기] → 콜라

이것이 함수의 본질입니다.

**입력(Input) → [규칙] → 출력(Output)**

f(x) = 2x + 1

- x=3 → f(3) = 7
- x=5 → f(5) = 11

**핵심 성질:** 같은 입력은 항상 같은 출력을 냅니다.

## 기본 문제 2

다음 중 함수인 것을 고르세요.

- A) 학번 → 학생 이름
- B) 이름 → 학번
- C) 날짜 → 그날의 최고기온
- D) 키 → 몸무게

**정답:** A, C

**해설:**
- B: 동명이인이 있을 수 있음 (하나의 입력에 여러 출력)
- D: 같은 키여도 몸무게가 다를 수 있음

하나의 입력에 반드시 하나의 출력만 대응해야 합니다.

## 심화 문제

**합성함수**

- f(x) = 2x + 1
- g(x) = x²

g(f(x))를 구하세요. (f의 출력 → g의 입력)

**풀이:** g(f(x)) = g(2x+1) = (2x+1)²

x=2일 때: f(2)=5, g(5)=25
확인: (2×2+1)² = 25 ✓

함수를 함수에 넣는 것 — 이것이 프로그래밍의 함수 호출과 정확히 같은 원리입니다.

## 실생활 연결

**넷플릭스 추천 알고리즘**

실제로 여러 함수가 합성되어 작동합니다:

- g(나이·성별·지역) = 인구통계 점수
- h(시청시간·완주율) = 선호도 점수
- k(g, h) = 최종 추천 목록

오늘 당신의 추천 목록은 수십 개의 함수가 합성된 결과입니다. 함수를 이해하면 알고리즘이 보입니다.
