# 수학 L4: 방정식의 탄생

**핵심 질문:** 미지수를 x로 쓰기 시작한 이유는?
**레벨:** L2 Sprout 1단계
**소요 시간:** 20-25분
**연결 개념:** 등식의 성질, 일차방정식, 연립방정식

---

## 개념 소개

"어떤 수에 3을 더하면 7이 됩니다."

고대 이집트인은 이것을 그림으로 표현했습니다.
9세기 아랍 수학자 알콰리즈미가 **al-jabr(알자브르)**라는 체계를 만들었습니다. → **Algebra(대수학)**의 어원입니다.

17세기 데카르트가 미지수를 x, y, z로, 알려진 수를 a, b, c로 표기하는 관습을 확립했습니다.

x + 3 = 7 → x = 4

이 단순한 표기법이 수학 전체를 바꾸었습니다.

---

## 기본 문제 1

다음 방정식을 푸세요.

- x + 5 = 12
- 3x = 21
- x - 4 = 9
- x ÷ 2 = 6

**정답:** x=7 / x=7 / x=13 / x=12
**핵심 원리:** 등호 양쪽에 같은 연산을 하면 등식이 유지됩니다.

---

## 기본 문제 2

문장을 방정식으로 표현하고 푸세요.

"어떤 수의 2배에서 3을 빼면 11입니다."

**방정식:** 2x - 3 = 11
**풀이:** 2x = 14 → x = 7
**확인:** 2×7 - 3 = 11 ✓

---

## 심화 문제

**연립방정식**

- 사과 3개 + 바나나 2개 = 1,700원
- 사과 1개 + 바나나 3개 = 1,100원

사과와 바나나 각각의 가격은?

**풀이:**
3x + 2y = 1700 ... ①
x + 3y = 1100 ... ②

②×3 - ①: 7y = 1600 → y = 200원 (바나나)
x = 1100 - 3×200 = 500원 (사과)

**정답:** 사과 500원, 바나나 200원

---

## 실생활 연결

**배달 앱 요금 비교**

- A 서비스: 기본료 2,000원 + km당 500원
- B 서비스: 기본료 없음 + km당 800원

몇 km 이상이면 A가 유리한가?

2000 + 500x = 800x
2000 = 300x
**x ≈ 6.7km**

6.7km 이상 배달이면 A 서비스가 더 저렴합니다. 방정식 하나로 최적의 선택이 가능합니다.
