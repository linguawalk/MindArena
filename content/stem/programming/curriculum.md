# MindArena 프로그래밍 트랙 커리큘럼

**목표:** Brilliant.org 컴퓨터과학 과정 동등 수준
**방식:** AI가 각 역량에 맞는 문제를 실시간 생성
**범위:** 알고리즘 사고 → 자료구조 → 언어 → 시스템 → 응용

---

## 커리큘럼 구조

```
22개 단원 × 평균 7개 역량 = 약 154개 핵심 역량
특징: 코드 없이도 개념 이해 가능하게 설계
      실제 코딩은 별도 실습 환경에서 진행
```
