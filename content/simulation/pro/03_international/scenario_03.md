## 시나리오 설정

### 배경: 가상의 2030년 기후 위기

```
기후 현황 (가상 2030년):
  지구 평균 온도 상승:
    산업화 이전 대비 +1.8°C (달성)
    파리협정 목표 1.5°C 이미 초과
    2°C 이내 유지: 사실상 불가능

  극한 기상:
    50년 만의 홍수: 연 3-4회
    전례 없는 폭염: 연 6-8회
    해수면 상승: 30cm (도서 국가 위기)
    산호초 90% 소멸

  배출 현황:
    글로벌 CO₂ 배출: 정점 후 감소 중
    하지만 목표 대비 여전히 20% 초과
    주요 배출국 약속 이행률: 58%
```

```
  2030년 피지·투발루:
    해수면 상승으로 일부 섬 주민 이주

    일부 선진국 새 정부:
    "기후 비용이 과도하다" 협정 재검토

→ 유엔 기후특별정상회의 (COP40) 소집
   파리협정 전면 재협상 또는 강화
```

## 핵심 협상 의제

### 의제 1: 온도 목표

```
현행: 1.5°C 억제 (사실상 달성 불가)
       2°C 이내 유지

재협상 쟁점:
  옵션 A: 1.5°C 목표 유지 (상징적)
  옵션 B: 2°C로 공식 하향 조정
  옵션 C: 국가별 차등 목표
  옵션 D: 온도 목표 폐기 + 배출량 목표

찬반:
  피해국: "1.5°C 포기는 우리 사형 선고"
  선진국: "현실적 목표로 실천 강화"
  과학계: "1.5°C 이미 넘었지만 목표 상징 중요"
```

### 의제 2: 감축 의무

```
현행: 국가결정기여 (NDC) 자발적 제출
       법적 구속력 없음

재협상 쟁점:
  법적 구속력 부여 여부:
    위반 시 제재 메커니즘
    탄소세 국제화
    무역 연계 조건

  차별화 원칙:
    선진국 더 많이 감축
    개발도상국 유예
    신흥국 중간 수준

  검증 체계:
    독립적 배출량 검증
    위성 모니터링 의무화
    투명성 강화
```

### 의제 3: 기후 재원

```
현행: 선진국 연 1,000억 달러 약속 (미달성)

재협상 쟁점:
  규모:
    피해국 요구: 연 1조 달러
    선진국 제안: 연 3,000억 달러
    현실적 합의점: ?

  구성:
    보조금 vs 대출 비율
    민간 자본 포함 여부
    기후 손실·피해 기금 (별도)

  손실과 피해 (Loss & Damage):
    기후 변화로 이미 발생한 피해 보상
    선진국 역사적 책임 인정 여부
    법적 배상 vs 자발적 기여
```

### 의제 4: 탄소 시장

```
현행: 파리협정 6조 탄소 시장 규정

재협상 쟁점:
  국제 탄소 크레딧:
    품질 기준 강화
    이중 계산 방지
    자연 기반 해결책 (산림) 포함 여부

  탄소 국경 조정:
    EU 탄소국경세 국제화
    개도국 반발 (무역 장벽 논란)

  넷제로 달성 방법:
    탄소 포집·저장 기술 역할
    원자력 포함 여부
    자연 흡수원 (숲·바다) 인정 범위
```

### 의제 5: 기술 이전

```
현황:
  개발도상국: 청정 기술 필요하지만 비용 과다
  선진국: 기술 보호 vs 기후 협력

쟁점:
  지식재산권 완화 (기후 기술)
  기술 이전 의무화 여부
  능력 배양 프로그램
  태양광·풍력·ESS 기술 공유
```

### 오전 세션 (3시간)

```

  "태평양 도서국 투발루,
```

## 실제 사례 참고

### 사례 1: 교토의정서 실패 (1997-2012)

```
내용: 선진국 의무 감축 + 개도국 면제
실패 원인:
  미국 비준 거부
  중국·인도 면제로 실효성 낮음
  목표 달성률 저조

교훈:
  법적 구속력 + 미국 불참 = 실패
  개도국 포함 없이는 불완전
```

### 사례 2: 파리협정 (2015)

```
혁신: 자발적 NDC + 글로벌 목표
성과: 195개국 참여 (보편성)
한계:
  구속력 없음
  목표 달성률 58%
  1.5°C 이미 초과 위기

교훈:
  포용성 vs 강제성 트레이드오프
  자발성이 참여율은 높이지만 실천 낮춤
```

### 사례 3: COP28 두바이 (2023)

```
성과:
  "화석연료로부터의 전환" 첫 명시
  손실·피해 기금 출범 (7억 달러)
한계:
  "단계적 폐지" 아닌 "전환"
  재원 규모 턱없이 부족

교훈:
  합의의 언어가 얼마나 중요한가
  산유국 의장국의 협상 왜곡
```

### 사례 4: 한국의 기후 외교

```
NDC 상향: 2018년 대비 40% 감축 (2030년)
탄소중립: 2050년 목표
현실:
  석탄 발전 여전히 높음
  산업 구조 전환 속도 느림
  기후 재원 기여 낮음

교훈:
  선언과 이행의 간극
  중견국 기후 외교의 역할
```
