## 핵심 협상 의제

### 의제 1: 비핵화

```
현황:
  북한 입장: "핵은 체제 생존의 보루"
             "완전한 비핵화는 협상 불가"
             "단계적·동시적 조치만 가능"
  미국 입장: "CVID (완전하고 검증가능하며
             불가역적인 비핵화) 선행"
  한국 입장: "단계적 비핵화 + 상응 조치"
  중국 입장: "쌍중단 (핵+군사훈련 동시 중단)
              후 협상"

협상 공간:
  단계적 비핵화 시퀀스:
    1단계: 핵 동결 (신규 생산 중단)
    2단계: 핵 능력 신고·검증
    3단계: 핵 폐기

  검증 방식:
    IAEA 사찰 범위
    현장 접근 수준
    위성 모니터링

  상응 조치:
    제재 완화 (어느 단계에서?)
    안전 보장 (어떤 형태로?)
    경제 지원 (규모와 조건)
```

### 의제 2: 평화 체제

```
현황:
  정전협정 (1953년) → 평화협정으로 전환
  주한미군 문제 연계
  북한: "미군 철수 전제"
  미국: "미군 주둔 협상 불가"
  한국: "한미동맹 유지 전제"

협상 공간:
  평화협정 당사자:
    남북미 3자? 남북미중 4자?
  주한미군:
    규모 조정? 역할 재정의?
    완전 철수는 협상 불가
  불가침 조약:
    남북 간 불가침 재확인
    미북 간 적대 관계 종식 선언

  종전 선언 vs 평화협정:
    종전 선언: 정치적 선언 (구속력 낮음)
    평화협정: 법적 구속력 (검증 필요)
```

### 의제 3: 경제 협력

```
대북 제재 현황:
  UN 안보리 결의 다수
  미국 독자 제재
  한국: 5·24 조치 (2010년~)

제재 완화 로드맵:
  조건부 완화: 비핵화 단계에 연동
  인도주의 예외: 식량·의약품
  개성공단 재개?
  금강산 관광 재개?
  남북 철도·도로 연결?

경제 지원 패키지:
  한국: 에너지 지원
  미국: 금융 제재 해제
  국제사회: 개발 지원
```

### 의제 4: 인권

```
현황:
  UN 북한인권 결의 매년 통과
  탈북민 증언 + 위성 사진
  북한: "내정 간섩 거부"

협상에서 인권 위치:
  미국·한국 일부: "인권 포함 필수"
  실용주의파: "비핵화 우선, 인권은 나중"
  NGO: "인권 없는 합의는 반쪽"

쟁점:
  인산가족 상봉
  납치자·억류자 문제 (한국·일본)
  정치범 수용소
```

### 의제 5: 군사 긴장 완화

```
즉각 조치 가능:
  군사분계선 우발적 충돌 방지
  핫라인 복구
  군사 훈련 규모 축소

중기 조치:
  비무장지대 평화 지대화
  해상 경계선 (NLL) 재합의

장기 과제:
  재래식 군비 통제
  군사력 균형
```

## 진행자 가이드

### 핵심 긴장 구조

```
비핵화 vs 체제 보장:
  "북한이 핵을 포기하면
   리비아 카다피처럼 될 수 있습니다.
   그 두려움을 어떻게 해소할 것인가?"

미국 vs 중국:
  "미국은 북한 비핵화를 원하지만
   중국은 현상 유지를 원합니다.
   두 강대국의 이익이 충돌할 때
   한국은 어떻게 해야 하는가?"

단기 vs 장기:
  "지금 불완전한 합의와
   완전한 합의를 기다리는 것 중
   어느 것이 더 위험한가?"
```

### 북한팀 연기 지원

```
북한팀은 가장 어려운 역할입니다.
다음 관점을 유지하도록 지원:

  "우리는 미국에 당한 역사가 있다"
  "핵이 없으면 우리는 없다"
  "경제 어렵지만 주권이 먼저"
  "중국도 우리를 통제하려 한다"

북한팀이 너무 유연하면:
  "최고지도자 지시를 잊으셨나요?"
  "군부가 이 합의를 수용할까요?"
```
