# Pro 시뮬레이션 - 국제 관계 (International Relations)

> 국내 정치보다 더 복잡한 것이 국제 정치입니다.
> 강제할 수 있는 법도 없고
> 심판도 없으며
> 약속을 지키지 않아도 처벌받지 않습니다.
> 오직 힘과 이익과 신뢰만이 작동합니다.
> 국제 관계 시뮬레이션은
> 무정부적 세계 질서에서
> 국가가 어떻게 생존하고 협력하는지를
> 직접 체험하는 과정입니다.

---

## 카테고리 특징

```
국내 거버넌스와의 핵심 차이:

국내:  법과 제도가 행위자를 구속
국제:  강제 가능한 상위 권위체 없음
       (무정부 상태, Anarchy)

핵심 개념:
  국가 이익 (National Interest):
    모든 외교의 출발점
    이익이 없으면 협력도 없다

  힘의 균형 (Balance of Power):
    패권국 견제 + 동맹 형성
    약소국의 생존 전략

  신뢰와 평판 (Credibility):
    국제 관계는 반복 게임
    오늘의 배신이 내일의 고립

  상호의존 (Interdependence):
    경제 연계가 갈등 비용 높임
    하지만 의존도가 취약점도 됨

  다자주의 (Multilateralism):
    국제 기구·조약·규범
    약소국의 목소리 확대 수단
```

## 03_international 고유의 역할 구조

```
일반적 역할 구성:
  국가 대표단 (복수):
    각국 외교부 + 협상팀
    국가별 비공개 이익 보유
  국제 기구:
    UN·WTO·IMF 등
    중립적 조율자 역할
  비국가 행위자:
    다국적 기업·NGO·언론
    현대 국제 관계의 핵심 요소
  국내 정치 압박:
    외교관도 국내 정치 압박 받음
    합의가 국내 비준 필요

특징:
  동시 다자 협상 (Multilateral Negotiation)
  비공개 양자 협상 허용
  정보 비대칭 + 기만 전략 허용
  비준 절차 시뮬레이션 포함
```

## 진행 구조

```
표준 3일 구조:
  Day 1: 각국 포지션 수립 + 양자 접촉
  Day 2: 다자 협상 + 위기 + 연합 형성
  Day 3: 최종 합의 시도 + 비준 + 성찰

추가 특징:
  각국 대표단 작전실 별도 운영
  양자·다자 협상 동시 진행
  국내 비준 절차 시뮬레이션
  국제 여론·언론 영향 반영
```
