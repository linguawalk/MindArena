## 시나리오 설정

### 핵심 사건

```
가상 기업 '아시아반도체 (AsiaSemi)'는
세계 3위 반도체 파운드리 기업입니다.

갑자기 두 개의 인수 제안이 동시에 들어왔습니다.

제안 1: 미국 '테크자이언트 (TechGiant)'
  → 적대적 인수 시도 (주주에게 직접 제안)
  → 제안가: 주당 85달러 (시가 대비 +38%)
  → 목적: 반도체 공급망 미국 내 통합

제안 2: 중국 '드래곤칩 (DragonChip)'
  → 우호적 인수 제안 (경영진에게 먼저 접촉)
  → 제안가: 주당 92달러 (시가 대비 +50%)
  → 목적: 중국 반도체 자립 + 기술 확보

아시아반도체는 선택의 기로에 섰습니다.
그리고 이것은 단순한 기업 거래가 아닙니다.
반도체 패권 경쟁의 핵심 전장이 됩니다.
```

## 아시아반도체 (AsiaSemi) 프로필

```
설립: 28년 전
본사: 가상 국가 코리아나 (Koreana)
시가총액: 약 680억 달러
직원: 42,000명 (코리아나 28,000 + 해외 14,000)
연매출: 약 180억 달러
영업이익률: 18.3%

기술력:
  3nm 공정 기술 보유 (세계 3위)
  독자 설계 자동화 툴 (EDA) 보유
  특허: 12,400건

고객사:
  애플: 매출의 22%
  엔비디아: 매출의 18%
  AMD: 매출의 12%
  기타 글로벌 팹리스: 48%

약점:
  장비 80%를 ASML (네덜란드) 의존
  미국 기술 라이선스 다수 포함
  최근 2분기 연속 실적 미달

주주 구성:
  코리아나 국민연금: 8.2%
  창업주 가문: 12.4%
  외국인 기관투자자: 54.3%
  국내 기관: 18.1%
  소액주주: 7.0%
```

## 실제 사례 참고

### 사례 1: ARM 인수 시도 (엔비디아, 2020-2022)

```
상황: 엔비디아가 ARM을 400억 달러에 인수 시도
결과: 영국·EU·미국·중국 모두 반대로 무산
교훈: 반도체 기업 M&A의 지정학적 복잡성
      국가 안보 = 최강의 거부권
```

### 사례 2: TSMC와 미·중 사이

```
상황: 대만 TSMC에 대한 미·중 양측 압박
미국: 애리조나 공장 건설 압박 + 보조금
중국: 대만 통일 시 TSMC 접수 위협
TSMC: 양쪽과 관계 유지하며 독자 생존

교훈: 핵심 기술 기업의 전략적 중립 가능성과 한계
```

### 사례 3: 삼성·SK하이닉스의 지정학

```
상황: 미국 반도체법 (CHIPS Act) 수혜 조건
      중국 공장 확장 제한 조건
결과: 한국 기업들의 미묘한 줄타기
교훈: 동맹국이어도 이해관계는 충돌한다
```

### 사례 4: 퀄컴의 NXP 인수 실패 (2018)

```
상황: 퀄컴이 NXP를 440억 달러에 인수 시도
결과: 중국 규제 당국 승인 거부로 무산
      미·중 무역전쟁의 일환
교훈: M&A는 양국 관계의 인질이 될 수 있다
```
