# Pro 시나리오 01: 국가 경제 위기 대응

> 외환보유액이 급감하고 있습니다.
> 국제 신용등급이 강등 직전입니다.
> IMF는 구제금융 조건을 제시했습니다.
> 대기업 3곳이 동시에 유동성 위기에 빠졌습니다.
> 당신의 팀은 72시간 안에 결정을 내려야 합니다.

## 시나리오 설정

### 배경: 가상 국가 코리아나 (Koreana)

```
국가 기본 정보:
  인구: 5,200만 명
  GDP: 약 1조 8,000억 달러
  주요 산업: 반도체·자동차·조선·석유화학
  수출 의존도: GDP의 42%
  외환보유액: 현재 2,100억 달러
              (6개월 전 3,400억 달러)
  국가 신용등급: AA- (강등 경고 발령 중)
  정부 부채: GDP의 52%
```

### 위기의 전개

```
6개월 전:
  글로벌 반도체 수요 급감
  주력 수출 기업 실적 악화 시작

3개월 전:
  미국 금리 급등으로 달러 강세
  외국인 투자자 자금 이탈 시작
  원화 가치 15% 절하

1개월 전:
  3대 그룹 중 하나인 '한성그룹'
  유동성 위기 공개
  주식시장 패닉 (-18%)

현재 (Day 0):
  외환보유액 2,100억 달러
  (적정 수준 3,200억 달러의 66%)
  IMF 긴급 협의 요청
  무디스 신용등급 강등 검토 공시
  한성그룹 외 2개 대기업 추가 위기
  시중은행 뱅크런 조짐
```

## 참여 팀 구성

### 팀 A: 정부 경제 위기 대응팀 (4-5명)

```
구성:
  기획재정부 장관 (팀장)
  한국은행 총재
  금융위원회 위원장
  청와대 경제수석
  외교부 차관 (국제 협력 담당)

비공개 정보:
  실제 외환보유액: 1,850억 달러
  (공식 발표 2,100억보다 낮음)
  IMF와 사전 비밀 협의 진행 중
  대통령 재가 가능 최대 재정 투입: 50조 원

공개 목표:
  외환 시장 안정
  금융 시스템 붕괴 방지
  IMF 구제금융 없이 해결

비밀 목표:
  IMF 조건 최소화 (구제금융 불가피 시)
  총선 전 위기 종식 (6개월 내)
```

### 팀 B: 대기업 그룹 경영진 (3-4명)

```
구성:
  한성그룹 회장 (팀장)
  그룹 CFO
  구조조정 전문 컨설턴트
  법률 자문

비공개 정보:
  실제 부채 규모: 공시의 1.4배
  해외 비밀 계좌 자산: 2조 원
  채권단 비밀 협의 진행 중
  해외 매각 가능 자산 목록

공개 목표:
  그룹 생존
  핵심 사업 유지
  임직원 고용 보호

비밀 목표:
  오너 일가 지분 최대 보호
  핵심 계열사 분리 보호
  정부 지원금 최대화
```

### 팀 C: IMF·국제 금융 기관 (2-3명)

```
구성:
  IMF 아시아태평양 국장 (팀장)

  세계은행 대표

비공개 정보:
  내부 분석: 코리아나 실제 위기 수준 심각
  구제금융 조건 초안 (공개 조건보다 엄격)
  미국 재무부 비공개 지시사항

공개 목표:
  코리아나 경제 안정화
  국제 금융 시스템 안정
  대출 조건 이행 확보

비밀 목표:
  미국 금융 기관 이익 보호
  구조조정을 통한 시장 개방 확대
  채무 상환 최우선 확보
```

### 팀 D: 시민사회·노동·언론 (2-3명)

```
구성:
  민주노총 위원장 (팀장)
  시민단체 연합 대표
  주요 언론사 경제부장

비공개 정보:

  해외 언론 인터뷰 예약 완료

공개 목표:
  서민·노동자 피해 최소화
  구조조정 과정 투명성 확보
  재벌 책임 추궁

비밀 목표:
  IMF 구제금융 조건 세부 공개
  특정 정치 세력 지원
  언론 주도권 확보
```

### 팀 E: 외국인 투자자·채권단 (2-3명)

```
구성:
  글로벌 헤지펀드 대표 (팀장)

  국제 신용평가사 대표

  투자금 회수 + 이익 극대화
  코리아나 시장 안정 (투자 환경 유지)

비밀 목표:
  위기를 활용한 자산 저가 매입

## 실제 사례 연결

### 1997년 한국 IMF 외환위기

```
타임라인:
  1997년 1월: 한보철강 부도
  1997년 7월: 기아자동차 부도
  1997년 11월: 외환보유액 고갈
  1997년 12월: IMF 구제금융 신청

IMF 조건:
  고금리 정책 (금리 30%까지 인상)
  기업·금융 구조조정
