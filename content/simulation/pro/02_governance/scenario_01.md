# Pro 시나리오 04: 국가 재정 위기 대응

> 국가 부채가 GDP의 100%를 넘었습니다.
> 국민연금은 15년 후 고갈됩니다.
> 복지는 줄일 수 없고 세금은 올리기 어렵습니다.
> 지금 고통을 선택할 것인가,
> 미래 세대에게 부담을 넘길 것인가?
> 민주주의는 이 질문에 어떻게 답하는가?

## 시나리오 설정

### 배경: 코리아나의 재정 위기

```
현재 상황 (가상 2032년):
  GDP: 약 2조 달러
  국가 부채: GDP의 108% (약 216조 원)
  재정 적자: GDP의 5.8% (연간 약 116조 원)
  국채 금리: 5.2% (3년 전 2.1%에서 급등)
  신용등급: A (AA에서 강등, 추가 강등 경고)

구조적 원인:
  고령화 가속:
    65세 이상 인구: 전체 38%
    노인 복지 지출: 10년간 3배 증가
  저성장 고착:
    잠재성장률: 1.2% (2000년대 5%에서 하락)
    세수 증가율: 성장률 하회
  복지 확대:
    20년간 복지 지출 GDP 대비 10% → 22%
    건강보험 적자 누적: 45조 원
    국민연금 고갈 예상: 2047년

단기 위기:
  국제 신용평가사 AA → A 강등 (3개월 전)
  외국인 국채 보유 비율 급감 (-18%)
  국채 금리 급등으로 이자 비용 급증
  IMF 권고: "즉각적 재정 건전화 필요"
```

## 재정 위기의 구조

### 수입 vs 지출 현황

```
연간 세입 (약 600조 원):
  소득세:      22% (132조)
  법인세:      18% (108조)
  부가가치세:  17% (102조)
  사회보험료:  25% (150조)
  기타:        18% (108조)

연간 세출 (약 716조 원):
  복지·사회보장: 38% (272조) ← 최대
  교육:          13% (93조)
  국방:          10% (72조)
  인프라:         8% (57조)
  이자 비용:     10% (72조) ← 급증
  기타:          21% (150조)

연간 적자: 116조 원
→ 매년 국채 발행으로 충당
→ 이자 비용이 복지 지출 증가율 추월
```

### 미래 전망 (현 추세 유지 시)

```
2035년: 국가 부채 GDP 130%
         국채 금리 7% 도달 예상
         이자 비용이 복지 예산 초과

2040년: 국가 부채 GDP 165%
         신용등급 BBB 강등 우려
         국민연금 수령 개시 세대 급증

2047년: 국민연금 고갈 예상
         건강보험 준고갈 상태
         사회 안전망 붕괴 위험
```

## 정책 옵션

### 분야 1: 세입 확대

```
옵션 A-1: 법인세 인상
  현행 25% → 28%
  추가 세수: 연 약 18조 원
  찬성: "기업이 더 부담해야"
  반대: "투자·고용 위축"

옵션 A-2: 소득세 최고세율 인상
  현행 45% → 50% (1억 이상)
  추가 세수: 연 약 12조 원
  찬성: "고소득자 더 부담"
  반대: "세수 효과 제한적, 조세 회피"

옵션 A-3: 부가가치세 인상
  현행 10% → 13%
  추가 세수: 연 약 35조 원
  찬성: "가장 효과적 세수 확보"
  반대: "저소득층 역진성"

옵션 A-4: 금융투자소득세 강화
  주식·부동산 양도차익 과세 강화
  추가 세수: 연 약 8조 원
  찬성: "불로소득 과세"
  반대: "자본 이탈·시장 위축"

옵션 A-5: 탄소세 도입
  탄소 배출량 기준 과세
  추가 세수: 연 약 15조 원
  찬성: "환경+세수 동시 효과"
  반대: "산업 경쟁력 약화"
```

### 분야 2: 세출 감축

```
옵션 B-1: 연금 개혁
  수급 개시 연령: 65세 → 68세
  소득대체율: 40% → 35%
  절감 효과: 연 약 20조 원 (장기)
  찬성: "지속 가능성 확보"
  반대: "노인 빈곤 심화"

옵션 B-2: 건강보험 본인 부담 확대
  현행 20% → 30% (외래 진료)
  절감 효과: 연 약 12조 원
  찬성: "도덕적 해이 방지"
  반대: "취약계층 의료 접근성 저하"

옵션 B-3: 복지 지출 구조 조정
  중복 복지 통합 + 비효율 제거
  절감 효과: 연 약 15조 원
  찬성: "효율성 향상"
  반대: "수혜자 반발"

옵션 B-4: 공무원 인건비 동결
  향후 5년간 임금 인상 동결
  절감 효과: 연 약 8조 원
  찬성: "공공 부문 허리 졸라야"
  반대: "우수 인재 이탈"

옵션 B-5: 국방비 삭감
  현행 GDP 2.8% → 2.3%
  절감 효과: 연 약 10조 원
  찬성: "과도한 국방비"
  반대: "안보 위협"
```

### 분야 3: 구조 개혁

```
옵션 C-1: 공기업 민영화
  대상: 주요 공기업 10개사
  예상 수입: 일시 약 80조 원
  찬성: "비효율 해소 + 재정 확보"

옵션 C-3: 세대 간 재정 위원회 설치
  미래 세대 이익 대변 독립 기관
```

## 진행자 가이드

### 민주주의적 긴장 부각

```
다수결의 한계:
  "68%가 복지 삭감에 반대합니다.
   하지만 71%가 재정 건전화가 필요하다고 합니다.
   민주주의는 이 모순을 어떻게 해결하나요?"

선거 주기의 문제:
  "이 개혁의 효과는 20년 후에 나타납니다.
   하지만 2년 후에 선거가 있습니다.
   어떻게 하시겠습니까?"

이익단체 압박:
  "노인연합 10만 명이 국회 앞에 있습니다.
   경제인연합은 투자 철회를 경고합니다.
   노동조합은 파업을 선언했습니다.
   어느 목소리를 더 들어야 하나요?"
```

### 교착 상태 해소 방법

```
방법 1: 여론조사 추가 발표
  특정 정책 찬반 여론 공개
  → 정치적 계산 변화 유도

방법 2: 전문가 증언 추가
  IMF 대표 등장: 긴박성 증가
  → 결정 압박

방법 3: 시간 제한 선언
  "48시간 내 합의 없으면
   신용등급 즉각 강등"
  → 위기감으로 교착 해소
```

## 실제 사례 참고

### 사례 1: 스웨덴의 연금 개혁 (1990년대)

```
상황: 재정 위기 + 연금 고갈 위기
방법: 여야 초당적 합의
      자동 안정화 장치 도입
      (경제 상황에 따라 급여 자동 조정)
결과: 지속 가능한 연금 시스템 구축
교훈: 정치적 합의가 가능하다
      하지만 스웨덴 특유의 협의 문화 필요
```

### 사례 2: 그리스 재정 위기 (2010-2015)

```
상황: GDP 대비 부채 180%
      EU·IMF 구제금융 신청
조건: 강력한 긴축 (복지 삭감·세금 인상)
결과: 경제 25% 위축
      실업률 27% 도달
      사회 불안 심화
교훈: 재정 위기 방치 시 더 큰 고통
      외부 강제 긴축의 사회적 비용
```

### 사례 3: 일본의 잃어버린 30년

```
상황: 1990년대 부동산 버블 붕괴 후
      재정 확대로 부채 급증
현재: GDP 대비 부채 260% (세계 최고)
결과: 저금리로 위기 봉합 중
      하지만 구조 개혁 지연
교훈: "나중에 하자"는 더 큰 위기를 만든다
```

### 사례 4: 한국의 실제 재정 상황

```
현실 (2024년):
  국가 부채: GDP의 51% (관리 가능 수준)
  하지만 고령화 가속 + 저성장으로
  2050년 GDP 100% 돌파 예상

쟁점:
  연금 개혁: 수차례 시도, 번번이 실패
  세금 인상: 정치적 저항으로 어려움
  복지 확대 요구: 계속 증가

교훈: 지금 논의하지 않으면 코리아나 시뮬레이션이
      현실이 될 수 있다
```
