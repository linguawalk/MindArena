# Pro 시뮬레이션 - 국가 거버넌스 (National Governance)

> 기업 경영보다 더 복잡한 것이 국가 경영입니다.
> 이익이 충돌하는 수천만 명의 국민,
> 3-5년마다 바뀌는 정권,
> 수십 년이 걸리는 구조적 변화.
> 국가 거버넌스 시뮬레이션은
> 민주주의의 작동 원리와 한계를
> 직접 체험하는 과정입니다.

---

## 카테고리 특징

```
01_corporate와의 차이:
  기업:  이익 극대화 → 비교적 명확한 목표
  국가:  공익 추구 → 목표 자체가 논쟁의 대상

핵심 특징:
  합법성 (Legitimacy):
    결정이 옳은가보다 누가 어떻게 결정했는가
  대표성 (Representation):
    누구의 목소리가 정책에 반영되는가
  지속성 (Continuity):
    단기 정권과 장기 국가 이익의 충돌
  책임성 (Accountability):
    결정의 결과에 누가 책임지는가
```

## 02_governance 고유의 역할 구조

```
일반적 역할 구성:
  행정부:   정책 집행 + 예산 편성
  입법부:   법안 심의 + 예산 의결
  사법부:   위헌 심사 (옵저버)
  시민사회: 여론 + 압력 단체
  언론:     의제 설정 + 정보 공개
  전문가:   객관적 분석 + 정책 대안

특징:
  민주주의적 절차가 시뮬레이션의 핵심
  다수결만이 답이 아님
  소수 의견 보호 메커니즘 포함
```

---

## 진행 구조

```
표준 3일 구조:
  Day 1: 위기 인식 + 각 기관 포지션 수립
  Day 2: 입법·행정 협상 + 여론 전쟁
  Day 3: 최종 의결 + 헌법·법률적 검토 + 성찰

추가 특징:
  의회 표결 시뮬레이션 포함
  헌법재판소 위헌 심사 가능
  여론조사 결과 실시간 반영
  언론 보도가 협상에 영향
```
