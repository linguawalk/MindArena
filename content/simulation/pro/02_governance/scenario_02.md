# Pro 시나리오 05: 헌법 개정 협상

> 헌법은 국가의 근본 법입니다.
> 모든 법률과 정책의 기준이 되고
> 한 세대가 다음 세대에게 물려주는
> 가장 중요한 사회적 계약입니다.
> 지금 우리는 어떤 나라를 만들고 싶은가?
> 누구의 목소리가 헌법에 담겨야 하는가?
> 합의는 가능한가?

## 시나리오 설정

### 배경: 코리아나 헌법 개정 논의

```
현행 헌법 상황:
  제정: 가상 1988년 (민주화 직후)
  마지막 개정: 가상 1997년
  경과: 35년간 큰 개정 없음

개정 논의 배경:
  정치 구조:
    대통령제 + 단임 5년 (제왕적 대통령 비판)
    여소야대 반복으로 입법 교착 빈번
    지방분권 요구 증가

  사회 변화:
    디지털 기본권 (인터넷·데이터·AI)
    기후·환경권 헌법 명시 요구
    양성평등·다양성 조항 강화 요구
    동물권 논의 시작

  경제 구조:
    경제민주화 조항 강화 vs 완화 논쟁
    토지 공개념 강화 요구
    기업 지배구조 헌법 반영

  안보 환경:
    한반도 평화 프로세스 반영
    핵 무장 논의 (헌법적 허용 여부)
    사이버 안보 조항 신설

헌법 개정 요건:
  국회 재적의원 2/3 이상 찬성
  국민투표 과반 찬성
  → 매우 높은 문턱 → 광범위한 합의 필요
```

## 핵심 개정 쟁점

### 쟁점 1: 권력 구조 개편

```
현행: 대통령제 (5년 단임)

옵션 A: 4년 중임제 대통령제
  대통령 임기 4년, 1회 중임 허용
  장점: 책임 정치 강화
  단점: 장기 집권 우려

옵션 B: 이원집정부제
  대통령 (외교·안보) + 총리 (내치)
  장점: 권력 분산
  단점: 책임 소재 불명확

옵션 C: 의원내각제
  총리 중심 국정 운영
  대통령은 상징적 국가원수
  장점: 민의 직접 반영
  단점: 한국 정치 문화와 부합 여부

옵션 D: 현행 유지 + 권한 축소
  대통령 권한 일부 국회·총리 이전
  장점: 급격한 변화 없이 개혁
  단점: 근본 문제 해결 불완전
```

### 쟁점 2: 기본권 확대

```
신설 요구 기본권:

디지털 기본권:
  인터넷 접근권
  데이터 자기결정권
  AI 피해 구제권
  알고리즘 투명성 요구권

환경·기후권:
  "건강한 환경에서 살 권리"
  기후 위기 대응 국가 의무 명시
  미래 세대의 환경권

사회경제적 기본권:
  주거권 강화 (집 없이는 삶이 없다)
  돌봄권 (아이·노인 돌봄 국가 책임)
  문화 향유권

평등권 확대:
  성별 정체성 차별 금지 명시
  장애인 권리 강화
  이주민·난민 기본권
```

### 쟁점 3: 지방분권

```
현행: 중앙집권적 구조

개정 방향:
  지방 자치 실질화
  자치 입법권·과세권 강화
  수도권 집중 완화 헌법 명시
  지역 균형 발전 의무 조항

쟁점:
  서울 중심 vs 지방 균형
  효율성 vs 형평성
  국가 통합 vs 지역 자율
```

### 쟁점 4: 경제 조항

```
현행 경제 조항 논쟁:

재벌 개혁파:
  경제민주화 강화
  기업 지배구조 헌법 명시
  토지 공개념 강화

시장 자유파:
  과도한 규제 조항 삭제
  경제 활동 자유 확대
  민간 영역 국가 개입 제한

쟁점:
  헌법으로 경제 구조 규율할 수 있는가?
  재벌 해체 vs 경쟁력 유지
```

### 쟁점 5: 안보·평화 조항

```
현행: 자유민주적 기본질서 + 평화 통일 추구

논쟁 1: 평화 조항 강화
  "침략 전쟁 영원히 포기"
  "핵 무기 보유·사용 금지"
  vs "핵 억제력 헌법 허용"

논쟁 2: 자위대·군사력
  현행 국군 조항 강화
  사이버 안보 조항 신설
  동맹 관계 헌법 반영 여부

논쟁 3: 통일 조항
  현행 "자유민주적 기본질서에 입각한 통일"
  평화 공존 강조로 수정 요구
  통일 조항 자체 삭제 주장 (현실론)
```

```

  법률 자문

비공개 정보:
  당 지도부 내부 방침:
    권력 구조: 4년 중임제 선호
    기본권 확대: 일부 수용 가능

  안정적 개정 + 집권당 유리 구조

비밀 목표:
  다음 선거 전 개정 완료 (집권 기간 활용)

## Day 2: 조문 협상 + 갈등 고조

### 오전 세션 (3시간)

```
09:00 밤사이 이벤트 발표 (30분):

  필수 이벤트 1:
  "여론조사 발표:
   국민 74%가 헌법 개정 필요하다고 응답.
   하지만 권력 구조 선호도:
   4년 중임제 38% / 이원집정부제 32%
   / 의원내각제 18% / 현행 유지 12%"
  → 국민 의견 분열 확인

  필수 이벤트 2:
  "헌법재판소, 현행 선거구 획정
   헌법 불합치 결정.
   2년 내 개선 입법 의무화."
  → 선거 제도 개혁 연계 압박

09:30 전체 협상 테이블 (90분):
  5대 쟁점 패키지 협상 시도
  교환 거래 (trade-off) 탐색:
    "권력 구조 양보 → 기본권 확대"
    "경제 조항 완화 → 환경권 신설"
    "안보 조항 강화 → 지방분권 확대"

11:00 조문 작성 워크숍 (60분):
  합의된 조항 실제 문구 작성
  법률 자문: 헌법 조문 형식 검토
  쟁점 조항 대안 문구 제시
```

### 오후 세션 (3시간)

```
14:00 선택적 이벤트 투입 (2-3개):

  이벤트 A:
  "보수 시민단체 대규모 집회:
   '성소수자 조항 삽입 시 개헌 반대'
   10만 명 국회 앞 집결."
  → 성평등 조항 협상 경색

  이벤트 B:
  "청년 단체, 기후권 헌법 명시 촉구
   단식 농성 돌입.
   '미래 세대의 생존권' 호소."
  → 환경권 압박 고조

```

   높은 합의 기준을 요구할까요?
   그것이 민주주의에 좋은 것인가요,
   나쁜 것인가요?"

가치 충돌 심화:
  "성소수자 권리와 종교 자유가
   충돌할 때 헌법은 어느 편을 들어야 하나요?
   헌법은 중립적일 수 있나요?"

미래 세대 문제:
  "오늘 만드는 헌법이
   50년 후 사람들을 구속합니다.
   그들의 동의를 어떻게 받을 수 있나요?"
```

### 조문 작성 지원

```
좋은 헌법 조문의 특징:
  ①명확성: 해석 논란 최소화
  ②포괄성: 다양한 상황 적용 가능
  ③실행 가능성: 구체적 의무 부여
  ④균형성: 권리와 의무 균형

나쁜 헌법 조문:
  너무 구체적 (법률 수준)
  너무 모호 (선언에 불과)
  정치적 타협의 흔적만 남은 문구
```
