# Pro 시나리오 06: 선거 제도 개혁

> 민주주의의 핵심은 선거입니다.
> 하지만 어떤 선거 제도를 선택하느냐에 따라
> 누가 권력을 갖는지가 완전히 달라집니다.
> 모든 정당은 자신에게 유리한 제도를 원합니다.
> 공정한 선거 제도란 과연 존재하는가?
> 그리고 누가 그것을 결정할 자격이 있는가?

## 시나리오 설정

### 배경: 코리아나 선거 제도 개혁 논의

```
현행 선거 제도:
  국회: 300석
    지역구: 253석 (소선거구 단순다수제)
    비례: 47석 (준연동형 비례대표)
  대통령: 단순다수제 직선
  지방: 광역·기초 의회 별도

현행 제도의 문제:
  거대 양당 유리 구조
  소수 정당 원내 진입 어려움
  지역 구도 고착화
  사표(死票) 비율: 40% 이상
  여성·청년·소수자 대표성 낮음
  위성 정당 창당 등 편법 만연

개혁 논의 배경:
  헌법재판소: 현행 선거구 획정 위헌
  시민사회: 선거 제도 개혁 요구
  소수 정당: 비례 확대 요구
  거대 양당: 기득권 유지 시도
  국제 기준: OECD 중 가장 낮은 비례성
```

## 선거 제도 기초 개념

```
주요 선거 제도 유형:

①소선거구 단순다수제 (FPTP):
  한 선거구에서 1명만 당선
  1등이 모든 것을 가져감
  예: 영국, 미국, 현재 코리아나 지역구

  특징:
    강한 정부 (양당제 형성)
    사표 많음
    지역 대표성 강함

②비례대표제:
  득표율에 비례해 의석 배분
  예: 네덜란드, 이스라엘

  특징:
    다당제 형성
    소수 의견 반영
    연립 정부 불가피
    정당 내 공천 권력 집중

③혼합형 제도:
  지역구 + 비례 병행
  연동형: 전체 의석을 비례로 계산
  병립형: 지역구·비례 별개 계산

④결선투표제:
  1차 투표 후 상위 2명 결선
  예: 프랑스 대통령선거

  특징:
    과반 지지 보장
    2회 투표 비용
    전략적 투표 패턴
```

## 핵심 개혁 쟁점

### 쟁점 1: 국회의원 선거 제도

```
옵션 A: 완전 비례대표제
  300석 전체를 득표율로 배분
  지역구 폐지
  정당 투표만

  찬성:
    "가장 공정한 민의 반영"
    사표 최소화
    소수 정당 진입 용이
  반대:
    "지역 대표 사라진다"
    정당 공천의 전횡 우려
    비례 명부 결정이 당권 집중

옵션 B: 권역별 비례대표제
  전국을 6-8개 권역으로 분할
  각 권역 내 비례 배분
  지역구 폐지 또는 대폭 축소

  찬성:
    완전 비례보다 지역성 반영
    소수 정당 진입 가능
  반대:
    여전히 지역구 의원 없음
    권역 설정이 정치적 논란

옵션 C: 연동형 비례대표제 강화
  현행 준연동형 → 완전 연동형
  비례 의석 100석 이상으로 확대
  위성 정당 방지 조항 강화

  찬성:
    지역구 유지 + 비례성 강화
    현실적 개혁
  반대:
    여전히 거대 양당 유리
    위성 정당 방지 실효성

옵션 D: 중대선거구제
  한 선거구에서 2-5명 당선
  기존 지역구 통합·확대

  찬성:
    소수 정당 진입 용이
    지역 대표성 유지
  반대:
    선거구 획정 어려움
    여전히 비례성 낮음
```

### 쟁점 2: 대통령 선거 제도

```
현행: 단순다수제 (1등 당선)

옵션 A: 결선투표제
  1차: 모든 후보 투표
  2차: 상위 2명 결선
  찬성: "과반 지지로 정당성 강화"
  반대: "2회 투표 비용·혼란"

옵션 B: 선호투표제 (Ranked Choice)
  1순위·2순위 선호 표시
  순위에 따라 당선자 결정
  찬성: "1회 투표로 결선 효과"
  반대: "유권자 혼란"

옵션 C: 현행 유지
  단순다수제 그대로
  찬성: "익숙한 제도"
  반대: "30% 지지로도 당선 가능"
```

### 쟁점 3: 의원 정수

```
현행: 300석

논쟁:
  확대론 (350-400석):
    "비례 확대를 위한 의석 추가"
    "인구 대비 의원 수 OECD 최저"
    반대: "의원 특권 확대" 여론 악화

  축소론 (250석):
    "세금 낭비 줄여야"
    반대: "대표성 더 낮아진다"

  현행 유지 (300석):
    "지역구 줄이고 비례 늘리기"
```

### 쟁점 4: 대표성 강화

```
여성 대표성:
  현행 여성 의원 비율: 19% (OECD 최하위권)
  방안:
    ①여성 할당제 (비례 50% 의무)
    ②교호 순번제 (남녀 교대 배치)
    ③자발적 권고 (현행)
  쟁점: 강제 할당 vs 자율

청년 대표성:
  현행 30대 이하 의원: 5% 미만
  방안:
    ①청년 할당 의석
    ②피선거권 연령 하향 (25세 → 18세)
  쟁점: 역차별 논란

비례 명부 공천:
  현행: 당 지도부 결정
  개혁안:
    ①당원 투표로 결정
    ②시민 배심원단 참여
  쟁점: 내부 민주주의 vs 당 자율
```

### 쟁점 5: 선거 관리

```
선거 연령:
  현행: 18세
  논의: 16세로 하향?
  쟁점: 정치 성숙도 vs 참정권 확대

선거 운동:
  현행: 선거 기간·방식 엄격 규제
  논의: 온라인 선거 운동 자유화
  쟁점: 공정성 vs 표현의 자유

투표 방식:
  현행: 사전투표 + 당일 투표
  논의: 우편투표, 온라인 투표 도입
  쟁점: 편의성 vs 보안·무결성
```

```
구성:
  원내대표 (팀장)
  선거제도 개혁 특위 간사
  당 선거 전략 담당

비공개 정보:
  내부 시뮬레이션 결과:
    완전 비례제 도입 시: 의석 -40석 예상
    연동형 강화 시: 의석 -20석 예상
    중대선거구 도입 시: 의석 -15석 예상
    현행 유지 시: 현상 유지
  당 지도부 방침:
    "의석 감소 최소화"
    "비례 확대는 30석 이내"

공개 목표:
  안정적 국정 운영 가능한 선거 제도

비밀 목표:
  기득권 의석 최대 방어
  소수 정당 협상력 제한
```

### 팀 B: 제1야당 (대형 진보 정당) (2-3명)

```
구성:
  원내대표 (팀장)

비공개 정보:
  내부 시뮬레이션 결과:

    연동형 강화: 의석 +8석 예상
```

```
    여야 양당 모두에게 캐스팅 보트
    비례 의석 100석 이상이 최소 조건

공개 목표:
  비례대표 대폭 확대
  소수 정당 원내 진입 보장

비밀 목표:

  캐스팅 보트로 조건 극대화
```

## Day 1: 제도 이해 + 포지션 수립

### 오전 세션 (3시간)

```
09:00 전체 브리핑 (45분):
  진행자: 선거 제도 기초 교육
  각국 제도 비교 자료 배포
  코리아나 현행 제도 문제점 정리
  비공개 패키지 배포

09:45 팀별 내부 전략 회의 (75분):
  각 제도 시뮬레이션 결과 검토
  협상 우선순위 설정
  레드라인 확정
  협상 카드 목록 작성

11:00 공개 입장 발표 (60분):
  각 팀 선거 제도 개혁 원칙 발표
  (팀당 8분)
  질의응답 없음 (1차 포지션 확인)
```

### 오후 세션 (3시간)

```
14:00 제도별 소위원회 (90분):

  소위 1: 국회의원 선거 제도
    (여당·야당·소수정당·선관위)
    4개 옵션 중 2개로 압축

  소위 2: 대통령 선거 제도
    (여당·야당·시민사회)
    결선투표제 도입 여부

  소위 3: 대표성 강화
    (소수정당·시민사회·여성·청년)
    할당제·연령 하향 논의

15:30 시뮬레이션 게임 (60분):
  각 선거 제도를 실제 적용해보기
  지난 선거 결과를 각 제도로 재계산
  → "만약 이 제도였다면 몇 석?"
  → 정당별 이해관계 수치로 확인
```

## Day 2: 협상 + 계산의 정치

### 오전 세션 (3시간)

```
09:00 밤사이 이벤트 발표 (30분):

  필수 이벤트 1:
  "여론조사 발표:
   '현행 선거 제도 바꿔야 한다' 74%
   하지만 '의원 정수 늘려야 한다' 찬성 18%
   → 비례 확대를 위해 의원 수 늘리면 여론 역풍"
  → 딜레마: 비례 확대 vs 의원 정수 동결
```

## 실제 사례 참고

### 사례 1: 한국 선거 제도 변천

```
1988년: 소선거구제 도입
2004년: 1인 2표 병립형 비례 도입
2020년: 준연동형 비례대표제 도입
         → 위성 정당 창당으로 무력화
2024년: 준연동형 유지 (위성 정당 재등장)

교훈:
  제도 설계보다 정치 문화가 중요
  법의 허점은 반드시 악용된다
  선거 제도 개혁은 끝이 없다
```

### 사례 2: 뉴질랜드의 선거 제도 개혁

```
과정:
  1993년: 국민투표로 소선거구제 → 혼합형 비례 전환
  이유: "승자독식 소선거구제 불공정"

결과:
  소수 정당 의회 진입
  연립 정부 일상화
  정책 결정 느려짐

교훈:
  국민투표로 제도 결정 → 정당성 강함
  제도 변화가 정치 문화를 바꾼다
  소선거구제의 단순성 vs 비례제의 공정성
```

### 사례 3: 독일 연동형 비례대표제

```
제도:
  지역구 50% + 비례 50%
  전체 의석을 비례 득표율로 결정
  초과 의석 허용 (의석 수 변동)

결과:
  2021년 선거: 736석 (정원 598석 대비 +138석)
  소수 정당 안정적 원내 진입
  연립 정부 필수

교훈:
  연동형은 의석 수 불안정 문제
  코리아나 적용 시 의원 정수 논란 불가피
```

### 사례 4: 이스라엘 완전 비례대표제

```
제도:
  전국 단일 선거구 완전 비례
  의석 진입 기준: 3.25%

결과:
  12개 이상 정당 의회 진입
  극소수 정당이 연립 조건 결정
  정치 불안정 + 잦은 조기 선거

교훈:
  완전 비례제의 대표성 vs 통치 가능성
  소수 정당 과다 발언권의 문제
```
