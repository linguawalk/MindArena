> 새로운 바이러스가 빠르게 확산되고 있습니다.

  각 팀 역할 확인

  총리팀 주재
  각 부처 상황 보고
  핵심 결정:

  결정 1: 국경 통제
    □ 발원국 입국 즉시 금지
    □ 발원국 입국자 격리 의무

    □ 즉시 전면 공개
    □ 최소 정보만 공개
    □ 24시간 후 공개
    → 시간 제한: 15분 내 결정

  결정 3: 병상 확보
    □ 즉시 비상 병상 동원
    □ 상황 지켜보며 준비
    → 시간 제한: 10분 내 결정

## 실제 사례 참고

### 사례 1: 한국 메르스 (2015)

```
초기 대응 실패:
  정보 비공개 → 국민 불신
  병원 목록 공개 안 함 → 자체 조사
  초기 과소 대응 → 빠른 확산

교훈:
  투명한 정보 공개가 신뢰 유지
  초기 과잉 대응이 과소 대응보다 낫다
  병원 간 정보 공유 필수
```

### 사례 2: 한국 코로나19 (2020)

```
성공 요인:
  3T (Test·Trace·Treat) 전략
  드라이브스루 검사소
  신속한 정보 공개
  시민 자발적 협조

한계:
  장기화에 따른 방역 피로
  경제적 약자 지원 부족
  변이 바이러스 대응 지연

교훈:
  초기 신속 대응이 핵심
  시민 신뢰가 방역의 기반
  경제·방역 동시 고려 필요
```

### 사례 3: 스웨덴의 집단면역 전략

```
전략: 봉쇄 없이 집단면역 추구
결과: 초기 사망률 높음
      장기적으로 경제 충격 적음
      하지만 노인 사망 비판

교훈:
  다른 가치관이 다른 전략을 낳는다
  정답은 없다
  선택에는 대가가 따른다
```

### 사례 4: WHO의 역할과 한계

```
비판:
  중국 눈치 보기
  팬데믹 선언 지연
  초기 정보 공유 부족

방어:
  회원국 정보 의존 구조
  강제 권한 없음
  정치적 압력 속 독립성 한계

교훈:
  국제 보건 거버넌스의 구조적 한계
  국가 간 협력 없이는 팬데믹 통제 불가
```
