# Pro 시뮬레이션 - 위기 관리 (Crisis Management)

> 위기는 예고 없이 찾아옵니다.
> 정보는 불완전하고
> 시간은 없으며
> 실수의 대가는 치명적입니다.
> 위기 관리 시뮬레이션은
> 극도의 불확실성 속에서
> 어떻게 결정하고 소통하며
> 조직을 이끄는지를 체험합니다.
> 평시에 훈련하지 않은 조직은
> 위기 때 무너집니다.

---

## 카테고리 특징

```
다른 카테고리와의 핵심 차이:

기업·거버넌스·국제 관계:
  시간 여유 있음
  정보 비교적 충분
  협상 중심

위기 관리:
  시간 극도로 부족 (시간 압박)
  정보 불완전·상충 (불확실성)
  결정 중심 (협상보다 행동)
  실시간 상황 변화 (동적 환경)

핵심 역량:
  신속 결정:
    완벽한 정보 없이 결정
    결정 지연의 비용 인식

  위기 소통:
    공중·언론·내부 소통
    잘못된 정보 관리
    신뢰 유지

  지휘 통제:
    조직 간 조율
    책임과 권한 명확화
    피로·스트레스 관리

  적응:
    초기 계획의 수정
    예상치 못한 변수 대응
```

## 04_crisis 고유의 진행 구조

```
위기 관리 시뮬레이션의 특징:

실시간 상황 변화:
  진행자가 시간 경과에 따라
  새로운 정보·이벤트 지속 투입
  팀은 계속 변하는 상황에 적응

타임 프레셔:
  각 결정에 시간 제한
  "10분 내에 결정하라"
  지연 자체가 결과에 영향

불완전 정보:
  초기 정보는 항상 부정확
  상충되는 정보 동시 제공
  "무엇이 사실인가" 판단이 핵심

다부처·다기관 조율:
  서로 다른 목표·우선순위
  관할권 충돌
  소통 실패의 현실적 묘사

표준 일일 구조:
  오전: 전날 상황 브리핑 + 새 이벤트
  오후: 대응 결정 + 실행 + 소통
  저녁: 결과 평가 + 다음날 준비
```

## B2B 활용

```
정부·공공:
  재난 관리 훈련 (행안부·소방청)
  보건 위기 대응 (질병청·복지부)
  금융 위기 대응 (금융위·한국은행)
  사이버 안보 훈련 (국정원·과기부)

기업:
  기업 위기 관리팀 훈련
  BCP (업무 연속성 계획) 훈련
  커뮤니케이션팀 위기 대응

군·안보:
  합동 작전 상황 훈련
  민군 협력 위기 대응
```
