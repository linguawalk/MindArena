# Pro 시뮬레이션 - 미래 시나리오 (Future Scenarios)

> 미래는 예측하는 것이 아니라
> 준비하는 것입니다.
> 미래 시나리오 시뮬레이션은
> 아직 오지 않은 세계를 상상하고
> 그 세계에서 어떤 결정을 내릴지
> 미리 체험하는 과정입니다.
> 틀린 예측보다 더 위험한 것은
> 예측 자체를 하지 않는 것입니다.

---

## 카테고리 특징

```
다른 카테고리와의 핵심 차이:

과거·현재 기반 시뮬레이션:
  알려진 사실 + 현실적 제약
  정답의 범위가 비교적 명확

미래 시나리오:
  불확실성이 핵심 조건
  복수의 미래가 동시에 가능
  "옳은 결정"이 나중에야 판명
  현재의 결정이 미래를 만든다

핵심 역량:
  시나리오 사고 (Scenario Thinking):
    단일 예측 대신 복수 시나리오
    각 시나리오 대비 계획

  장기적 관점:
    10-30년 후를 현재에서 결정
    복리의 힘 (초기 결정의 중요성)

  가치 기반 결정:
    미래가 불확실할 때
    가치가 나침반이 된다

  적응력:
    시나리오가 바뀌면 전략도 바뀐다
    유연성 vs 일관성
```

## 05_future 고유의 진행 구조

```
미래 시나리오 시뮬레이션의 특징:

시나리오 설정:
  복수의 미래 시나리오 제시
  (낙관·기준·비관 시나리오)
  각 팀이 다른 시나리오 담당 가능

현재 → 미래 연결:
  지금 결정이 20년 후 어떻게?
  복리 효과 + 경로 의존성

미래에서 현재로:
  "백캐스팅" 방법:
  원하는 미래를 먼저 설정
  거꾸로 현재 행동 도출

가치 충돌:
  미래 세대 vs 현재 세대
  확실성 vs 불확실성
  글로벌 vs 로컬

표준 3일 구조:
  Day 1: 미래 시나리오 분석 + 현재 현황
  Day 2: 정책·전략 협상 + 위기 대응
  Day 3: 장기 로드맵 합의 + 성찰
```
