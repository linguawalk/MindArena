## 시나리오 설정

### 배경: 가상의 2045년 코리아나

```
인구 현황:
  2045년 총인구: 4,200만 명
    (2024년 5,200만 명 대비 -19%)
  출산율: 0.65 (세계 최저)
  출생아: 연 17만 명
    (2024년 23만 명 대비 -26%)
  사망자: 연 45만 명
  자연 감소: 연 28만 명

연령 구조 (2045년):
  65세 이상: 46% (1,932만 명)
  15-64세: 47% (1,974만 명)
  0-14세: 7% (294만 명)
  노년부양비: 노인 1명당 생산인구 1.02명
    (2024년: 0.27명 → 완전 역전)

지역별 현황:
  수도권 인구 집중: 55% (2,310만 명)
  지방 소멸 위험 지역: 전체의 65%
  이미 소멸한 군: 38개 (2024년 대비 +28)
  무인 마을: 전국 1,200개
```

### 인구 절벽의 파급 효과

```
경제:
  GDP 성장률: 0.3% (잠재성장률 0.1%)
  1인당 GDP: 상승 (생산성 향상으로)
  하지만 총 GDP: 감소 중
  소비 시장: 축소 (-15% vs 2024년)
  부동산: 지방 전면 하락, 수도권 혼조

재정:
  국민연금: 2043년 고갈 (현실화)
  건강보험: 만성 적자
  노인 요양: GDP의 6.8% (2024년 1.8%)
  세수: 감소 (생산인구 감소)
  재정 적자: GDP의 7.2%

사회:
  빈집: 전국 290만 채 (2024년 140만)
  폐교: 연간 300개
  폐업 병원: 지방 종합병원 40% 소멸
  군 병력: 목표 대비 35% 부족
  소방·경찰: 지방 광역화 진행

이민:
  외국인 인구: 210만 명 (5%)
  이민 확대 논의 진행 중
  사회 통합: 과제
```

## 핵심 정책 쟁점

### 쟁점 1: 출산 장려 vs 현실 수용

```
20년간 출산 장려 결과:
  투자: 350조 원 (2006-2045년)
  결과: 출산율 1.08 → 0.65 (오히려 하락)
  교훈: "돈으로는 안 된다"

현실:
  출산율 회복 가능성 논쟁:
    낙관론: "프랑스는 회복했다 (1.8)"
    비관론: "한국은 구조가 다르다"
    현실론: "0.65→1.0도 30년"

쟁점:
  계속 출산 장려에 투자?
  아니면 인구 감소를 받아들이고 적응?
  두 전략의 배분은?

적응 전략:
  소규모 고효율 사회 설계
  AI·자동화로 인력 부족 보완
  이민 확대
  1인당 생산성 극대화
```

### 쟁점 2: 이민 정책

```
현황 (2045년):
  외국인 인구: 210만 명 (5%)
  이민 정책: 여전히 소극적
  사회 갈등: 일부 지역 이민자 반발

주요국 비교:
  캐나다: 인구의 23% 이민자 (성공적 통합)
  독일: 14% (갈등 + 기여 혼재)
  일본: 2.3% (여전히 폐쇄적)
  싱가포르: 30%+ (도시국가 특수)

코리아나 선택지:
  옵션 A: 적극적 이민 확대
    연간 30만 명 수용
    숙련 이민자 우대
    귀화 요건 완화

  옵션 B: 제한적 이민
    필요 분야 한정 (의료·돌봄·농업)
    임시 체류 중심
    문화 동질성 유지

  옵션 C: 현상 유지
    연간 5-10만 명 현 수준
    시장 자율에 맡김

쟁점:
  경제적 필요 vs 문화적 정체성
  사회 통합 비용
  이민자 권리
```

### 쟁점 3: 국토 재편

```
현황:
  지방 소멸 가속화
  수도권 집중 심화
  공공 서비스 유지 불가 지역 증가

선택지:
  집중화 (거점 도시 전략):
    인구를 거점 도시로 유도
    농어촌 공공 서비스 단계적 축소
    효율적이지만 "지방 포기" 비판

  분산화 (균형 발전):
    지방 투자 지속
    원격 근무 인프라 확충
    비용 높고 효과 불확실

  전략적 수축:
    소멸 지역은 자연으로 돌려보내기
    "축소의 미학" (Shrinking Cities)
    역발상이지만 저항 심함

쟁점:
  지방 주민의 이주 권리 vs 거주 권리
  재정 효율 vs 균형 발전
  소멸 지역 처리 방식
```

### 쟁점 4: 복지 재설계

```
현황 위기:
  국민연금 고갈 (2043년 현실화)
  현재 65세 이상 연금 수령자: 1,932만 명
  생산가능인구: 1,974만 명
  → 생산인구 1명이 노인 거의 1명 부양

재설계 방향:
  연금:
    수급 연령: 65 → 70세
    소득대체율: 40% → 30%
    보험료: 현재 18% → 25%+
    → 노인 빈곤 vs 재정 지속

  돌봄:
    노인 돌봄 인력 부족
    로봇·AI 돌봄 기술 도입
    가족 돌봄 vs 공공 돌봄

  의료:
    지방 병원 통폐합
    원격 의료 확대
    의대 정원 확대

쟁점:
  현 노인 세대 vs 미래 세대
  복지 축소 vs 재정 지속
  기술로 돌봄 대체 가능한가?
```

### 쟁점 5: 새로운 사회 모델

```
인구 감소 사회의 긍정 측면:
  환경 부하 감소
  1인당 자원 증가
  경쟁 완화
  삶의 질 향상 가능

새로운 사회 모델 탐색:
  소규모 고품질 사회:
    인구 감소를 문제가 아닌
    다른 사회로의 전환으로 인식
    "더 적지만 더 좋게"

  AI·기술 기반 사회:
    인구 감소를 기술로 보완
    생산성 극대화

  다문화 사회:
    이민으로 인구·문화 다양화
    새로운 한국 정체성

쟁점:
  인구 감소가 필연적이라면
  어떤 사회를 선택할 것인가?
  "좋은 축소"는 가능한가?
```

## 실제 사례 참고

### 사례 1: 일본의 축소 경험

```
현황:
  2010년 이후 인구 감소 시작
  빈집 900만 채 (2023년)
  지방 소멸 가속화
  이민에 소극적 → 기술 대체 선택

교훈:
  소극적 대응의 한계
  기술만으로는 부족
  지방 소멸 막기 어려움
  그러나 1인당 GDP는 유지
```

### 사례 2: 독일의 이민 전략

```
현황:
  2015년 100만 명 난민 수용
  현재 외국인 비율 15%
  사회 갈등 + 경제 기여 혼재

교훈:
  이민은 단기 인구 문제 해결
  하지만 통합 정책 필수
  극우 반발 관리 어려움
  경제 기여는 분명
```

### 사례 3: 스웨덴의 복지 모델

```
전략:
  높은 복지 + 출산 친화 정책
  육아휴직 부모 공동 사용
  공공 보육 강화

결과:
  출산율 1.7 유지 (한국 0.65 대비)
  이민 30% 포함
  높은 세금으로 가능

교훈:
  돈만이 아닌 구조 변화 필요
  양성 평등이 출산율의 핵심
  한국에 적용하려면 구조 전환 필요
```

### 사례 4: 한국 저출산 정책 실패

```
투자: 2006-2024년 280조 원
결과: 출산율 1.08 → 0.72 (계속 하락)

실패 원인:
  현금 지원 중심 (구조 변화 없음)
  주거·교육비 해결 안 됨
  성별 불평등 미해결
  경쟁 사회 구조 그대로

교훈:
  현금으로는 출산율 못 올린다
  구조적 변화 없이는 한계
  20년 대책이 왜 실패했는지 직시
```
