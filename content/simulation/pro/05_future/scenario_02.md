## 시나리오 설정

### 배경: 가상의 2035년 코리아나

```
기후 현황:
  지구 온도: +1.9°C (산업화 이전 대비)
  파리협정 1.5°C 목표: 이미 초과
  2°C 마지노선: 위협 중
  극한 기상: 이미 일상화

코리아나 탄소중립 현황:
  2030년 목표: 2018년 대비 40% 감축
  실제 달성: 28% 감축 (목표 미달)
  2050년 목표: 탄소중립
  현재 경로: 2050년까지 60% 감축 예상
  → 목표 달성 위해 추가 40% 감축 필요

주요 배출원 현황:
  에너지: 37% (발전·열 생산)
  산업: 28% (철강·시멘트·석유화학)
  수송: 18% (자동차·항공·해운)
  건물: 11% (냉난방·조명)
  농축산·폐기물: 6%
```

### 탄소중립의 현실적 과제

```
에너지 전환:
  재생에너지 비중: 현재 25% → 2050년 80% 목표
  원전: 현재 27% → 유지 or 확대 논쟁
  석탄: 현재 25% → 2035년 완전 폐기 예정
  하지만 재생에너지 간헐성 문제 미해결

산업 전환:
  철강: 수소환원제철로 전환 필요
        비용: 현재의 2.3배
        기간: 15-20년
  시멘트: 탄소 포집 기술 필요
  석유화학: 친환경 소재로 전환
  → 전환 비용 합계: 약 400조 원 (15년간)

고용 충격:
  탄소 집약 산업 일자리: 85만 명
  전환 과정 감소 예상: 45만 명
  녹색 일자리 창출 예상: 30만 명
  순 감소: 15만 명
  → 충청·경남 지역 집중 피해

기술 과제:
  수소 경제: 아직 상용화 초기
  탄소 포집·저장: 비용 문제
  전기차 배터리: 해결 중
  항공·해운: 대안 기술 미성숙
```

## 핵심 협상 의제

### 의제 1: 산업 전환 속도

```
쟁점:
  석탄 발전:
    2030년 완전 폐기 vs 2038년 단계적 폐기
  철강 수소전환:
    2040년 완료 vs 2050년 완료
  내연기관차:
    2030년 판매 금지 vs 2035년 판매 금지

빠른 전환:
  장점: 기후 목표 달성
  단점: 산업 경쟁력 타격
        실업 급증
        전기 요금 급등

느린 전환:
  장점: 경제 충격 완화
  단점: 기후 목표 미달
        미래 세대 부담
        탄소 국경세 피격
```

### 의제 2: 공정 전환 (Just Transition)

```
개념:
  탄소 집약 산업 노동자·지역사회
  보호하며 전환하는 것

현황:
  충남 (석탄 발전): 발전소 폐쇄 예정
  경남 (조선·철강): 산업 전환 필요
  → 지역 경제 붕괴 위기

쟁점:
  재원: 전환 지원 기금 규모
  대상: 노동자 개인 vs 지역 사회
  방식: 재교육 vs 소득 보전 vs 이주 지원

공정 전환 기금:
  정부 제안: 10년간 20조 원
  노동계 요구: 10년간 50조 원
  기업 부담: 어느 수준까지?
```

### 의제 3: 탄소 가격제

```
현행: 배출권 거래제 (ETS)
  탄소 가격: 톤당 3만 원
  적용 범위: 온실가스 72% 커버

강화 방안:
  탄소세 병행 도입:
    에너지·연료에 직접 세금
    소비자 행동 변화 유도
  탄소 가격 단계적 인상:
    2035년: 톤당 5만 원
    2040년: 톤당 10만 원
    2050년: 톤당 20만 원

쟁점:
  탄소 가격 상승 → 전기·난방 요금 상승
  저소득층 역진성 문제
  산업 경쟁력 문제 (탄소국경세)
  수입 활용: 기후 기금 vs 소득세 감면
```

### 의제 4: 녹색 투자·금융

```
필요 투자:
  2035-2050년 탄소중립 투자:
    에너지: 200조 원
    산업: 150조 원
    수송: 80조 원
    건물: 70조 원
    합계: 500조 원

재원 구성:
  정부: 150조 원
  민간: 350조 원 필요
  → 민간 투자 어떻게 유도?

녹색 금융:
  녹색 채권 발행
  기후 관련 공시 의무화
  ESG 투자 제도화
  기후 리스크 금융 감독

쟁점:
  그린워싱 방지
  중소기업 녹색 전환 자금
  해외 투자 유치
```

### 의제 5: 기술 혁신

```
핵심 기술 현황:
  수소: 생산 비용 아직 높음
  CCUS: 상용화 초기 (비용 높음)
  소형 원자로: 2030년대 상용화 예상
  전고체 배터리: 2027년 양산 예상

기술 투자 우선순위:
  단기 (2035-2040):
    태양광·풍력 + ESS
    전기차·수소차
    건물 에너지 효율

  중기 (2040-2045):
    수소 경제 본격화
    산업 탈탄소 기술

  장기 (2045-2050):
    항공·해운 대체 연료
    농축산 탄소 감축

쟁점:
  정부 R&D 집중 분야
  민간 기술 개발 인센티브
  국제 기술 협력
```

  기술 현실:
    수소 경제 상용화 5년 지연 가능성
    CCUS 비용 예상보다 2배 높음
  정치적 제약:
    탄소세 도입 시 지지율 -10%p
    내년 총선 → 고통스러운 결정 회피 압박

공개 목표:
  2050 탄소중립 달성

    탄소 가격: 톤당 5만 원까지

공개 목표:
  기술적으로 가능한 속도의 전환

비밀 목표:

    산업 원가: 약 1.5% 인상

## 실제 사례 참고

### 사례 1: 독일 에너지 전환

```
목표: 2045년 탄소중립
진행:
  탈원전 완료 (2023)
  재생에너지 50%+ 달성
  하지만 러시아 가스 의존 문제
교훈:
  전환 과정의 에너지 안보 위험
  정치적 일관성의 중요성
```

### 사례 2: 덴마크 성공 모델

```
성과: GDP 성장 + 탄소 감축 동시
비결:
  일관된 30년 에너지 정책
  풍력 기술 세계 1위
  시민 수용성 높음
교훈:
  장기 일관성 + 기술 투자
  에너지 전환이 새 산업 기회
```

### 사례 3: 한국 탄소중립 현실

```
2030년 NDC: 2018년 대비 40% 감축
실적 (2023): 약 25% 감축
격차: 15%p

주요 과제:
  석탄 발전 여전히 높음
  산업 전환 속도 느림
  탄소 가격 낮음 (톤당 3만 원)

교훈:
  선언 쉽고 이행 어렵다
  정권 교체마다 정책 변동
```

### 사례 4: 독일 석탄 탈출 (Kohleausstieg)

```
계획: 2038년 석탄 완전 폐기
공정 전환:
  석탄 지역에 400억 유로 지원
  노동자 조기 퇴직 패키지
  지역 재개발 투자
교훈:
  공정 전환 없이는 사회적 저항
  지역 공동체 단위 지원 필요
```
