# Pro 시뮬레이션 (Professional Simulation)

> Open 시뮬레이션이 입문이라면
> Pro 시뮬레이션은 몰입입니다.
> 최소 3일, 최대 수주에 걸쳐 진행되는
> 전문가 수준의 장기 시뮬레이션입니다.
> 역할은 더 깊고, 상황은 더 복잡하며,
> 결과는 더 현실에 가깝습니다.

---

## Open vs Pro 비교

| 항목 | Open 시뮬레이션 | Pro 시뮬레이션 |
|------|--------------|--------------|
| 소요 시간 | 1-2시간 | 최소 3일 이상 |
| 참여자 | 4-8명 | 8-30명 |
| 역할 깊이 | 단순 역할 | 조직·팀 단위 역할 |
| 정보 구조 | 모두 공개 | 역할별 비공개 정보 |
| 진행 방식 | 테이블 토론 | 실제 업무 방식 모사 |
| 결과물 | 구두 합의 | 문서·보고서·협약서 |
| 대상 | 일반인·학생 | 전문가·조직·교육기관 |
| 활용 | 교육·체험 | B2B·기업교육·학술 |

## Pro 시뮬레이션의 특징

### 1. 장기 몰입

```
Open: 1-2시간 내 결론
Pro:  Day 1 - 상황 파악 + 역할 적응
      Day 2 - 협상·갈등·위기 대응
      Day 3 - 최종 결정 + 성찰
      (시나리오에 따라 1-4주 연장 가능)
```

### 2. 조직 단위 역할

```
Open: 개인이 역할 수행
Pro:  팀이 역할 수행
      예: A팀 = 정부 부처
          B팀 = 대기업 경영진
          C팀 = 시민단체
          D팀 = 국제 기구
```

### 3. 비공개 정보 구조

```
Open: 모든 정보 공개 공유
Pro:  역할별 비밀 정보 보유
      → 정보 탐색·협상·기밀 유지가 핵심
      예: 정부팀만 아는 예산 한도
          기업팀만 아는 내부 전략
```

### 4. 실제 업무 방식 모사

```
Open: 테이블 토론
Pro:  공식 회의·브리핑·협상
      문서 작성·제출·검토
      보도자료·성명서·협약서 작성
      (실제 직무 역량 훈련 효과)
```

### 5. 전문 지식 요구

```
Open: 배경 지식 없어도 참여 가능
Pro:  해당 분야 기본 지식 필요
      또는 사전 학습 자료 제공
```

## Pro 시뮬레이션 진행 구조

### 표준 3일 구조

```
Day 0 (사전):
  역할 패키지 배포 (1주일 전)
  사전 학습 자료 제공
  역할별 비공개 미션 확인

Day 1: 상황 파악 + 포지션 수립
  오전: 전체 브리핑 (상황 공유)
  오후: 팀별 내부 회의 + 전략 수립
  저녁: 비공식 사전 접촉 허용

Day 2: 협상 + 위기 + 갈등
  오전: 공식 협상 라운드 1
  오후: 위기 이벤트 투입 + 긴급 대응
  저녁: 비공식 협상 + 연합 형성

Day 3: 결정 + 성찰
  오전: 최종 협상 + 결정 선언
  오후: 결과 발표 + 성찰 토론
  저녁: 전체 복기 + 피드백

Day 4+ (선택):
  후속 시나리오 진행
  결정 이후의 상황 전개
```

## B2B 활용 방안

### 기업 교육

```
임원 리더십 과정:
  경영 위기 대응 시뮬레이션
  글로벌 M&A 협상 훈련

신입·중간 관리자:
  팀 의사결정 + 갈등 관리
  다부서 협력 시뮬레이션
```

### 공공·정부 교육

```
공무원 정책 역량 교육:
  국가 위기 대응 훈련
  다부처 협력 정책 시뮬레이션

의회·지자체:
  정책 협상 + 이해관계 조율
```

### 대학·대학원

```
경영대학원 (MBA):
  케이스 스터디 심화 버전
  실전 협상 훈련

행정·정책 대학원:
  정책 결정 + 이해관계자 관리
  국제 협상 훈련
```
