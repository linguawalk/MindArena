# 시나리오 05: 인수합병 협상

> 인수하는 자와 인수되는 자,
> 그리고 그 사이에 있는 모든 이해관계자.
> 숫자로 표현되는 기업 가치 뒤에
> 수백 명의 삶이 달려 있습니다.
> 협상 테이블에서 무엇을 지키고 무엇을 내줄 것인가?

## 협상의 시작

메가푸드 M&A 팀이 프레시박스에
비공식 인수 의향을 타진했습니다.

**메가푸드의 초기 제안:**
```
인수 가격: 850억 원 (매출의 2.6배)
인수 방식: 주식 100% 인수
고용 보장: 핵심 인력 1년 보장
창업자: 2년 계약직 전환 (연봉 3억)
브랜드: 메가푸드 산하 독립 운영
```

**프레시박스의 초기 반응:**
```
대표이사 윤지수:
  "850억은 너무 낮다. 우리 성장성을
   반영하면 최소 1,500억은 받아야 한다."

투자자 대표:
  "런웨이가 8개월밖에 없다.
   현실적으로 협상해야 한다."

CTO 신준서:
  "인수 후 기술팀 독립성이 보장되지 않으면
   나는 남지 않겠다."
```

## 핵심 쟁점

### 쟁점 1: 기업 가치 (인수 가격)

```
메가푸드 입장 (850억):
  근거:
    현재 매출 기준 2.6배 (적자 기업)
    유사 기업 평균 멀티플 적용
    물류 인프라 구축 비용 별도 소요
    인수 후 통합 비용 고려

프레시박스 입장 (1,500억):
  근거:
    성장률 68% 반영 시 미래 가치
    구독 고객 42만 명의 LTV (생애가치)
    기술 자산 (알고리즘 + 특허)
    밀키트 시장 성장 프리미엄

객관적 참고:
  국내 유사 M&A 평균: 매출의 3-4배
  적자 스타트업 할인율: 30-40%
  합리적 범위: 950-1,300억 원 추정
```

### 쟁점 2: 창업자 처우

```
메가푸드 제안:
  2년 계약직 전환
  연봉 3억 원
  스톡옵션 없음

윤지수 요구:
  5년 고용 보장
  연봉 5억 원
  매출 연동 성과급
  경영 독립성 보장

핵심 갈등:
  메가푸드: "창업자는 직원이 되어야 한다"
  윤지수: "내 회사가 없어지는 것이 아니다"
```

### 쟁점 3: 고용 보호

```
현황: 프레시박스 직원 180명

메가푸드 제안:
  핵심 인력 (50명) 1년 고용 보장
  나머지 (130명) 통합 후 재배치 또는 정리

프레시박스 요구:
  전 직원 2년 고용 보장
  임금 동결 (삭감 없음)
  강제 부서 이동 금지

현실:
  메가푸드 내부에 유사 직무 중복 많음
  물류팀 40명은 메가푸드 기존 팀과 완전 중복
```

### 쟁점 4: 브랜드 및 운영 독립성

```
메가푸드 입장:
  "프레시박스 브랜드는 유지하되
   메가푸드 시스템으로 통합"
  → 구매·물류·마케팅 통합

프레시박스 입장:
  "독립 운영이 보장되지 않으면
   브랜드 가치가 훼손된다"
  → 별도 법인 유지 요구

핵심 갈등:
  통합의 시너지 vs 독립의 브랜드 가치
```

### 쟁점 5: 기술 자산 처리

```
CTO 신준서의 우려:
  "추천 알고리즘과 물류 최적화 시스템은
   우리 팀이 만든 것이다.
   메가푸드에 통합되면 우리 팀이
   의미 없어진다."

메가푸드의 의도:
  기술 시스템 전체 흡수
  개발팀 메가푸드 기술조직에 통합

프레시박스 요구:
  기술팀 독립 운영
  신준서 CTO 직위 유지
  기술 로드맵 독자 결정권
```

```
  카드: 인수 포기 선언

역할 4: 프레시박스 CTO 신준서
```

## 진행 방법

### 1단계: 배경 파악 (20분)

```
개인 준비:
  자신의 역할 BATNA 설정
  (Best Alternative To Negotiated Agreement)
  → "협상 결렬 시 나의 최선의 대안은?"

메가푸드 M&A 팀장의 BATNA:
  → 다른 밀키트 기업 인수
  → 자체 밀키트 사업 론칭

프레시박스 대표의 BATNA:
  → Series C 재도전
  → 다른 인수자 탐색
  → IPO 추진

투자자의 BATNA:
  → 강제 매각 (창업자 동의 없이 50% 지분 행사)
  → 파산 정리

CTO의 BATNA:
  → 경쟁사 이직
  → 독립 창업
```

### 2단계: 포지션 발표 (15분)

```
각 역할 2-3분 내 발표:
  현재 상황 인식
  최우선 목표
  절대 양보 불가 조건
  협상 가능한 범위
```

### 3단계: 협상 (60-70분)

```
Phase 1 (20분): 양측 포지션 확인
  메가푸드 팀 vs 프레시박스 팀
  각자 초기 조건 제시
  핵심 쟁점 5가지 확인

Phase 2 (30분): 조건 교환
  쟁점별 협상 진행
  패키지 딜 구성 가능
    예: "가격을 낮추는 대신 고용 보장 강화"
  역할 간 사적 협의 가능 (타임아웃 제도)
    각 팀 1회, 5분

Phase 3 (20분): 최종 합의 시도
  조건 패키지 최종 조율
  합의 또는 결렬 확인
```

**진행자용 이벤트 카드:**

```
이벤트 1 (Phase 1 종료 시):
  "밀키트 시장 1위 기업 '마이셰프'가
   프레시박스 인수에 관심 있다는
   소식이 흘러나왔습니다."
  → 경쟁 구도 형성 (프레시박스 협상력 강화)

이벤트 2 (Phase 2 중반):
  "프레시박스 최대 고객사가
   경쟁사로 이탈 조짐을 보이고 있습니다.
   구독 해지율이 이번 달 급등했습니다."
  → 프레시박스 가치 하락 압박

이벤트 3 (교착 상태 시):
  "CTO 신준서가 실리콘밸리 스타트업으로부터
   스톡옵션 포함 제안을 받았다고
   팀 내에서 알려졌습니다."
  → 기술 인력 이탈 위기 고조

이벤트 4 (선택적):
  "정부가 식품 스타트업 M&A 촉진을 위해
   인수 기업에 세제 혜택을 부여하는
   방안을 검토 중이라고 발표했습니다."
  → 메가푸드 인수 유인 증가

이벤트 5 (선택적):
  "프레시박스 직원들이 SNS에
   '우리를 팔지 마세요' 캠페인을
   시작했습니다. 언론이 주목하고 있습니다."
  → 여론 압박 발생
```

### 4단계: 결정 선언 (10분)


## 진행자 가이드

### 협상 역학 이해

```
협상의 핵심 원칙:
  ZOPA (Zone of Possible Agreement):
    합의 가능 범위 = 양측의 BATNA 사이
    메가푸드 상한: 1,200억 (내부 기준)
    프레시박스 하한: 950억 (투자자 수용 한계)
    ZOPA: 950-1,200억 원

  앵커링 효과:
    처음 제시한 숫자가 협상을 지배
    → 첫 제안의 중요성 강조
```

### 교착 상태 해소

```
"지금 가장 중요한 한 가지만 포기한다면
 무엇을 포기하고 무엇을 얻고 싶으신가요?"

"상대방이 왜 그 조건을 고수하는지
 이해하신다면, 어떻게 생각하시나요?"

"만약 3개월 후 프레시박스가 자금 부족으로
 문을 닫는다면 각자 어떤 결과가 되나요?"
```

### 감정적 순간 활용

```
창업자 역할이 감정적이 될 때:
  "지금 느끼는 감정을 협상 테이블에서
   어떻게 활용하실 건가요?"

투자자가 냉혹한 입장일 때:
  "투자자로서 직원들의 고용에 대해
   어떤 책임감을 느끼시나요?"
```
