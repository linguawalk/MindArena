# 시나리오 04: ESG 경영의 선택

> 성장과 지속가능성 사이에서
> 기업은 어떤 선택을 해야 하는가?
> 주주, 직원, 환경, 지역사회의 이해관계가
> 충돌하는 현실 속에서 판단을 내려야 합니다.

## 핵심 딜레마

이사회에서 네 가지 방향이 논의되고 있습니다.

### 옵션 A: 전면 ESG 전환 (3년 계획)

```
내용:
  2025년부터 친환경 소재(생분해·종이) 전환 시작
  2027년까지 플라스틱 라인 100% 전환
  R&D 투자: 3년간 150억 원
  생산 라인 교체: 3년간 200억 원

재무 영향:
  전환 기간 영업이익률: 8.2% → 3-4%
  전환 완료 후 예상 이익률: 6-7%
  총 투자 비용: 약 350억 원
  (현금 보유 120억 + 차입 230억 필요)

기대 효과:
  ESG 등급 B 이상 달성
  친환경 프리미엄 가격 적용 가능
  고객사 장기 계약 유지
  정부 지원금 수혜 가능

리스크:
  전환 기간 수익성 악화
  기술 전환 실패 가능성
  차입 부담 증가
  주가 단기 하락 예상
```

### 옵션 B: 단계적 전환 (5년 계획)

```
내용:
  신규 수주만 친환경 소재로 전환
  기존 계약은 만기까지 플라스틱 유지
  R&D 투자: 5년간 80억 원
  생산 라인 점진적 교체

재무 영향:
  영업이익률 유지: 7-8%
  총 투자 비용: 약 150억 원 (5년간)
  재무 안정성 유지

기대 효과:
  수익성 유지하며 전환
  리스크 분산
  기존 고객사 관계 유지

리스크:
  규제 대응 속도 부족
  고객사가 2026년까지 전환 요구 시 불이행
  ESG 등급 개선 속도 느림
  경쟁사 선제 전환 시 불리
```

### 옵션 C: 사업 다각화 (병행 전략)

```
내용:
  플라스틱 사업 유지하며 친환경 신사업 병행
  친환경 포장재 자회사 설립
  기존 사업 수익으로 신사업 투자

재무 영향:
  기존 사업 이익률 유지
  신사업 초기 손실: 연 20-30억 원 예상
  5년 후 자회사 분사 및 상장 목표

기대 효과:
  리스크 분산
  신사업 성공 시 기업가치 급등
  기존 직원 고용 유지

리스크:
  두 사업 동시 운영 역량 부담
  자원 분산
  신사업 실패 시 추가 손실
```

### 옵션 D: 현상 유지 + 규제 대응 최소화

```
내용:
  법적 최소 요건만 충족
  규제 시행 전까지 현행 유지
  단기 수익 극대화 후 매각 또는 사업 전환 검토

재무 영향:
  단기 이익률 극대화
  2027년까지 누적 이익 약 250억 원 추가

기대 효과:
  단기 주주 가치 극대화
  현금 축적

리스크:
  2027년 이후 사업 존속 불투명
  고객사 이탈 가속화
  브랜드 이미지 훼손
  규제 위반 시 과징금
  인재 이탈
```

## 이해관계자 입장

### 내부 이해관계자

```
대표이사 (창업주 아들, 42세):
  "우리 회사를 물려받아 더 오래 유지하고 싶다.
   ESG는 선택이 아니라 생존 문제다.
   하지만 직원들의 일자리도 지켜야 한다."

CFO (50세, 외부 영입):
  "숫자가 모든 것을 말한다.
   350억 투자에 대한 ROI가 불확실하다.
   주주에게 어떻게 설명할 것인가?"

생산이사 (48세, 내부 승진):
  "기술 전환은 가능하다.
   하지만 현장 직원들의 재교육과 사기가 걱정된다.
   충분한 시간과 지원이 필요하다."

영업이사 (45세):
  "고객사가 요구하면 해야 한다.
   하지만 가격을 올리면 경쟁사로 넘어간다.
   친환경이 프리미엄을 정당화할 수 있나?"

노조 위원장:
  "전환은 찬성한다.
   하지만 한 명도 해고 없이, 임금 삭감 없이
   전환해야 한다. 그것이 조건이다."
```

### 외부 이해관계자

```
주요 고객사 구매담당:
  "2026년까지 친환경 전환이 안 되면
   공급사를 교체할 수밖에 없다.
   하지만 단가는 기존과 동일하게 유지해 달라."

기관투자자 대표:
  "ESG 등급 B 이상 달성 전까지
   추가 투자는 없다.
   하지만 수익성이 너무 떨어지면 매각도 고려한다."

환경단체 대표:
  "그린워싱은 안 된다.
   실질적인 플라스틱 감축 수치를
   공개적으로 약속해야 한다."

지역 주민 대표:
  "공장 인근 환경 오염 문제를 해결해 달라.
   하지만 공장이 없어지면 지역 경제가 무너진다."
```

## 역할 배정

### 4인 진행 시

```
역할 1: 대표이사
  목표: 장기 생존 + 직원 보호 + 주주가치
  딜레마: 세 가지를 동시에 달성할 수 있는가?
  권한: 최종 결정권

역할 2: CFO
  목표: 재무 안정성 + 투자 수익률
  딜레마: 장기 투자 vs 단기 수익
  권한: 재무 거부권

역할 3: 기관투자자 대표
  목표: ESG 등급 개선 + 수익률 유지
  딜레마: ESG 요구 vs 수익성 요구
  권한: 투자 철회 카드

역할 4: 노조 위원장
  목표: 고용 보호 + 임금 유지
  딜레마: 전환 찬성 vs 고용 보장 요구
  권한: 파업 카드
```

### 6인 진행 시

```
위 4개 역할 + 다음 추가:

역할 5: 주요 고객사 구매담당
  목표: 친환경 포장재 확보 + 단가 유지
  딜레마: 공급사 교체 vs 관계 유지
  권한: 계약 해지 카드

역할 6: 환경단체 대표
  목표: 실질적 환경 개선
  딜레마: 기업과 협력 vs 대립
  권한: 여론 동원 + 언론 공개
```

### 8인 진행 시

```
위 6개 역할 + 다음 추가:

역할 7: 생산직 대표 직원
  목표: 고용 안정 + 새로운 기술 습득 기회
  딜레마: 변화 수용 vs 불안감
  권한: 현장 분위기 대변

역할 8: 정부 담당자 (환경부)
  목표: 규제 이행 + 산업 경쟁력 유지
  딜레마: 강력 규제 vs 산업 보호
  권한: 지원금 + 과징금 카드
```

## 진행 방법

### 1단계: 배경 파악 (20분)

```
개인 체크리스트:
  □ 그린팩의 재무 상황 파악
  □ 각 옵션의 비용과 효과 이해
  □ 자신의 역할과 이해관계 파악
  □ ESG란 무엇인지 개념 정리

ESG 개념 정리 (참고):
  E (환경): 탄소 배출, 폐기물, 자원 사용
  S (사회): 고용, 인권, 지역사회
  G (지배구조): 투명성, 이사회 구성, 부패 방지
```

### 2단계: 입장 발표 (15분)

```
각 역할이 2-3분 내 발표:
  현재 상황 인식
  선호 옵션과 이유
  절대 수용 불가 조건
  협상 가능한 부분
```

### 3단계: 이해관계자 협상 (50-60분)

```
라운드 1 (20분): 핵심 쟁점 도출
  "ESG 전환의 속도와 비용을 누가 부담하는가"
  "고용 보호와 수익성을 동시에 달성할 수 있는가"
  "단기 주주가치와 장기 지속가능성 중 무엇이 우선인가"

라운드 2 (25분): 조건 협상
  각 역할이 조건 카드 제시
  연합 형성 및 패키지 딜 논의

라운드 3 (15분): 최종 합의
  채택 옵션 + 수정 조건 확정
  각 역할의 수용 여부
```

**진행자용 이벤트 카드:**

```
이벤트 1 (라운드 1 종료 시):
  "환경단체가 그린팩의 플라스틱 폐기물
   불법 매립 의혹을 제기하는 기사를
   내일 오전 게재할 예정입니다."
  → 여론 위기 발생

이벤트 2 (교착 상태 시):
  "경쟁사 패키지원이 전면 ESG 전환을
   공식 선언하고 주가가 15% 상승했습니다."
  → 경쟁 압박 증가

이벤트 3 (합의 임박 시):
  "정부에서 친환경 포장재 전환 기업에
   설비 투자비의 30% 보조금 지급
   계획을 발표했습니다."
  → 재무 부담 완화

이벤트 4 (선택적):
  "주요 고객사가 친환경 포장재에
   단가 15% 프리미엄을 인정하겠다고
   비공개로 타진해왔습니다."
  → 수익성 전망 개선
```

### 4단계: 결정 선언 (5분)

```
발표 항목:
  ①채택 옵션 (A/B/C/D 또는 혼합)
  ②ESG 전환 일정표
  ③고용 보호 약속 내용
  ④주주 수익 보장 방안
  ⑤환경 목표 수치 (플라스틱 감축 %)
```

### 5단계: 성찰 토론 (20-30분)

```
토론 질문:

①"ESG는 기업의 비용인가, 투자인가?"
  각자의 관점에서 논하세요.

②이 시뮬레이션에서 가장 힘든 역할은
  무엇이었고 왜 그랬나요?

③단기 수익과 장기 지속가능성이 충돌할 때

④"기업의 목적은 이윤 극대화다"라는
  밀턴 프리드먼의 주장과
  "이해관계자 자본주의"를 비교해보세요.

⑤한국 기업들이 ESG를 실천하기 어려운
  현실적 이유는 무엇인가요?

## 진행자 가이드

### ESG 개념 배경 지식

```
ESG 등급이 중요한 이유:
  기관투자자 투자 기준 (국민연금 등 포함)
  금융기관 대출 조건
  해외 수출 시 바이어 요구사항
  우수 인재 채용 경쟁력

그린워싱이란:
  실질적 변화 없이 친환경 이미지만 홍보
  최근 규제 강화 + 소비자 반발 증가
  → "진짜 ESG"와 "보여주기 ESG" 구분 중요
```

### 갈등 심화 유도 질문

```
CFO에게:
  "10년 후 이 회사가 존재하지 않는다면
   지금의 수익성이 무슨 의미가 있나요?"

노조에게:
  "전환을 거부하면 2027년 이후
   공장 자체가 문을 닫을 수도 있습니다.
   그래도 지금 조건을 고수하시겠습니까?"

환경단체에게:
  "기업이 완전히 망해서 문을 닫으면
   환경에 더 좋을까요?"

투자자에게:
  "ESG를 요구하면서 단가는 올리지 말라고
   하는 것은 모순 아닌가요?"

## 실제 사례 참고

### 사례 1: SK이노베이션의 ESG 전환

```
상황: 정유·석유화학 기업의 ESG 전환 압박
전략: 배터리·그린 사업으로 포트폴리오 전환
결과: 단기 수익성 악화 → 장기 기업가치 상승
교훈: 전환 비용을 감수한 선제 투자의 중요성
```

### 사례 2: 유한킴벌리의 지속가능경영

```
전략: 우리강산 푸르게 푸르게 캠페인 (30년+)
결과: ESG 선도 기업 이미지 + 브랜드 가치 상승
교훈: ESG는 단기 비용이 아닌 장기 브랜드 자산
```

### 사례 3: H&M의 그린워싱 논란

```
상황: 친환경 컬렉션 출시 + 실상 폐기물 증가
결과: 소비자 반발 + 규제 당국 조사
교훈: 실질적 변화 없는 ESG 마케팅의 역효과
```

### 사례 4: 파타고니아의 급진적 ESG

```
전략: 환경 보호를 기업 목적으로 명문화
      "지구가 우리의 유일한 주주"
결과: 브랜드 충성도 극대화 + 프리미엄 유지
교훈: ESG를 기업 정체성으로 삼을 때의 가능성
```
