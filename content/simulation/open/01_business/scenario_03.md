## 진출 방식 옵션

### 방식 1: 직접 진출 (자사 설립)

```
내용: 현지 법인 설립 + 직접 운영
투자 규모: 높음 (현지 인력·사무소 필요)
통제력: 높음
속도: 느림 (6-12개월 준비)
리스크: 높음 (모든 비용 자체 부담)
적합 시장: 미국, 유럽 (대규모 시장)
```

### 방식 2: 현지 파트너십 (유통 계약)

```
내용: 현지 유통사와 독점/비독점 계약
투자 규모: 중간
통제력: 낮음 (파트너 의존)
속도: 빠름 (3-6개월)
리스크: 중간 (파트너 리스크)
적합 시장: 동남아, 중동
```

### 방식 3: 온라인 우선 (플랫폼 입점)

```
내용: 아마존·쇼피 등 글로벌 플랫폼 입점
투자 규모: 낮음
통제력: 중간
속도: 매우 빠름 (1-3개월)
리스크: 낮음 (플랫폼 의존성)
적합 시장: 미국, 동남아 동시 테스트
```

### 방식 4: M&A (현지 기업 인수)

```
내용: 현지 소규모 뷰티 브랜드 인수
투자 규모: 매우 높음 (50-100억 원+)
통제력: 매우 높음
속도: 중간 (인수 절차 6-12개월)
리스크: 매우 높음 (인수 실패 시 대규모 손실)
적합 시장: 미국, 유럽 (기존 유통망 활용)
```

## 핵심 딜레마

이사회에서 세 가지 전략 방향이 충돌하고 있습니다.

### 전략 A: 집중 돌파 (미국 단독 집중)

```
내용:
  미국 시장 단독 진출
  3년간 50억 원 집중 투자
  아마존 + 인플루언서 마케팅 중심

근거:
  "미국에서 성공하면 글로벌 브랜드 완성"
  "분산 투자는 모두 실패의 지름길"

리스크:
  실패 시 회사 재정에 치명적
  국내 사업 투자 여력 감소
  성공까지 3-5년 소요
```

### 전략 B: 단계적 확장 (동남아 → 미국)

```
내용:
  1단계: 동남아 3개국 동시 진출 (15억 원)
  2단계: 성과 확인 후 미국 진출 결정
  방식: 현지 파트너십 + 온라인 병행

근거:
  "문화적으로 가까운 동남아에서 경험 축적"
  "작은 성공을 발판으로 큰 시장 도전"

리스크:
  동남아 성공이 미국 성공을 보장하지 않음
  시간 소요로 경쟁사에 선점 당할 수 있음
```

### 전략 C: 멀티 마켓 (동시 다발 진출)

```
내용:
  미국·동남아·중동 동시 진출
  온라인 플랫폼 우선 테스트
  시장별 소규모 예산 배분

근거:
  "어느 시장이 터질지 모른다"
  "리스크 분산"

리스크:
  자원 분산으로 어느 시장도 제대로 못할 수 있음
  실행 역량 한계
  집중력 부족으로 모두 실패 가능
```

## 역할 배정

### 4인 진행 시

```
역할 1: 대표이사
  목표: 장기 성장 + 리스크 관리 균형
  고민: 이사회 압박 vs 현실적 역량
  성향: 신중한 도전

역할 2: 마케팅 이사
  목표: 브랜드 글로벌화 + SNS 파워 활용
  고민: 어느 시장에서 브랜드 스토리가 먹히나
  성향: 트렌드 민감, 공격적 확장

역할 3: 재무 이사
  목표: 재무 안정성 + 투자 효율
  고민: 얼마까지 베팅할 수 있나
  성향: 숫자 기반, 보수적 판단

역할 4: 외부 투자자 (VC)
  목표: 3년 내 기업가치 2배
  고민: 빠른 성장 vs 안정적 성장
  성향: 공격적 성장 요구
```

### 6인 진행 시

```
위 4개 역할 + 다음 추가:

역할 5: 동남아 현지 파트너 대표
  배경: 베트남 최대 뷰티 유통사 대표
  목표: 독점 계약 + 유리한 조건
  고민: 케어랩의 장기 파트너가 될 수 있나
  성향: 현지 시장 경험 강조

역할 6: 글로벌 뷰티 컨설턴트
  배경: 10년 이상 K-뷰티 해외 진출 지원 경험
  목표: 현실적 조언 제공
  고민: 클라이언트가 원하는 것 vs 해야 할 것
  성향: 중립적, 경험 기반 판단
```

## 진행자 가이드

### 핵심 긴장 포인트

```
①속도 vs 안정성:
  빠른 진출 vs 충분한 준비
  → "경쟁사가 먼저 선점하면 어떻게 되나요?"

②집중 vs 분산:
  한 시장 집중 vs 여러 시장 동시
  → "역량이 분산되면 어느 것도 못 한다는
     주장에 어떻게 대응하시겠나요?"

③단기 vs 장기:
  빠른 성과 vs 브랜드 구축
  → "3년 내 결과가 없으면 투자자가
     떠날 텐데 어떻게 하실 건가요?"
```

```
전략: 현지화 전략 + 프리미엄 포지셔닝
결과: 초기 성공 → 사드 사태로 급락
교훈: 정치적 리스크를 항상 고려해야 한다
```

```
전략: 현지 파트너십 + 온라인 우선
결과: 베트남·태국에서 안정적 성장
교훈: 단계적 접근이 리스크를 줄인다
```

### 사례 3: 닥터자르트의 미국 진출

```
전략: 세포라 입점 + 프리미엄 브랜딩

교훈: 유통 채널 선택이 브랜드 이미지를 결정
```
