# 시나리오 04: 부동산 정책의 선택

> 집은 삶의 터전인가, 투자 자산인가?
> 집값을 잡으면 집주인이 반발하고
> 집값을 올리면 무주택자가 절망합니다.
> 세금을 올리면 공급이 줄고
> 공급을 늘리면 환경이 파괴됩니다.
> 모두를 만족시킬 수 없는
> 부동산 정책의 딜레마가 시작됩니다.

---

## 배경: 한국 부동산의 현실

### 현황 데이터

```
주택 가격:
  서울 아파트 평균 매매가: 약 11억 원 (2024)
  서울 아파트 평균 전세가: 약 5억 원
  PIR (소득 대비 주택 가격 비율):
    서울: 약 25배 (연봉 25년치)
    OECD 평균: 8-10배

주택 소유:
  전국 자가 보유율: 57%
  서울 자가 보유율: 42% (절반 이상이 세입자)
  다주택자 (2채+): 전체 가구의 15%
  다주택자 보유 주택 비중: 전체의 28%

청년 주거:
  30대 자가 보유율: 29% (10년 전 41%)
  청년 월세 비중: 60%+ (급증)
  평균 내 집 마련 기간: 서울 기준 약 28년

공공임대:
  공공임대 비중: 전체 주택의 8%
  (OECD 평균 15%, 네덜란드 35%)
```

### 부동산 문제의 구조

```
수요 측:
  서울·수도권 집중 (인구·일자리)
  저금리 시대 투자 수요 급증
  갭투자·레버리지 활용 확산

공급 측:
  서울 재건축·재개발 규제
  신규 택지 부족
  건설 비용 상승

세금 측:
  취득세·보유세·양도세 복잡
  정권마다 세율 변동
  다주택자 과세 강화·완화 반복

금융 측:
  대출 규제 (LTV·DSR)
  금리 변동에 따른 시장 급변동
```

## 핵심 사건: 서울 집값 재급등

### 상황 설정

```
현재 상황:
  2년간 조정 후 서울 아파트 가격
  다시 급등세 (6개월간 +15%)
  전세 가격도 동반 상승
  역전세 위기 해소되며 전세 수요 급증

원인:
  금리 인하 기대감
  공급 부족 지속
  재건축 규제 완화 기대
  수도권 일자리 집중 심화

피해:
  무주택 서민 내 집 마련 더 어려워짐
  청년 월세 부담 급증
  전세 사기 피해자 여전히 많음

수혜:
  유주택자 자산 증가
  다주택 임대인 임대 수익 증가
  건설사 분양 수익 회복
```

## 정책 옵션

### 옵션 A: 공급 확대 (그린벨트 해제 + 재건축 촉진)

```
핵심 내용:
  서울 외곽 그린벨트 일부 해제
  재건축·재개발 규제 대폭 완화
  용적률 상향 (고층화 허용)
  3기 신도시 속도 조절 추진

예상 공급량:
  10년간 서울·수도권 50만 호 추가

찬성 논거:
  "집값은 결국 수요-공급의 문제"
  "규제 완화로 민간 공급 촉진"
  "시장 원리에 맡겨야"

반대 논거:
  "그린벨트 해제는 환경 파괴"
  "공급 효과는 10년 후 (단기 효과 없음)"
  "재건축 이익은 기존 집주인에게 집중"
  "투기 수요를 먼저 잡아야"
```

### 옵션 B: 수요 억제 (세금 강화 + 대출 규제)

```
핵심 내용:
  다주택자 보유세 강화
  종합부동산세 부활 및 강화
  양도소득세 중과세 유지
  LTV·DSR 대출 규제 강화
  갭투자 금지

예상 효과:
  투기 수요 감소
  다주택자 매물 출회

찬성 논거:
  "투기 수요를 잡아야 근본 해결"
  "다주택자 세금이 너무 낮다"
  "집은 거주 목적이어야"

반대 논거:
  "세금 강화는 임대 공급 감소"
  "결국 전세·월세 가격 상승"
  "재산권 침해"
  "다주택자도 임대 공급자"
```

### 옵션 C: 공공 주도 공급 확대

```
핵심 내용:
  공공임대 주택 대폭 확충
  (현 8% → 20% 목표)
  토지임대부 주택 확대
  공공 재건축 활성화
  사회주택 지원

예산: 10년간 약 200조 원

찬성 논거:
  "주거는 복지, 시장에 맡기면 안 된다"
  "저소득층·청년 주거 안정"
  "유럽 복지국가 모델"

반대 논거:
  "막대한 재정 투입"
  "공공임대 품질 낮음"
  "민간 시장 위축"
  "입지 선정 갈등"
```

### 옵션 D: 지역 분산 정책

```
핵심 내용:
  공공기관 추가 지방 이전
  지방 거점 도시 집중 육성
  수도권 기업 이전 인센티브
  지방 대학 강화

예상 효과:
  수도권 집중 완화
  지방 주택 수요 증가

찬성 논거:
  "서울 집값 문제는 서울 집중의 문제"
  "근본 원인 해결"
  "지방 균형 발전"

반대 논거:
  "기업·인재의 자발적 이동 어려움"
  "효과까지 20-30년 소요"
  "단기 집값 안정 효과 없음"
```

## 핵심 쟁점

```
쟁점 1: 집의 본질
  "집은 거주 공간인가, 투자 자산인가?
   정책의 목표를 어디에 둬야 하는가?"

쟁점 2: 공급 vs 수요 억제
  "집값을 잡으려면 공급을 늘려야 하는가,
   수요를 줄여야 하는가?"

쟁점 3: 세대 간 공정성
  "기성 세대는 집값 상승으로 자산을 키웠다.
   청년 세대는 그 기회를 빼앗겼다.
   이것은 공정한가?"

쟁점 4: 환경 vs 공급
  "주택 공급을 위해 그린벨트를 해제해도 되는가?"

쟁점 5: 재산권 vs 주거권
  "다주택자의 재산권과
   무주택자의 주거권이 충돌할 때
   어느 것이 우선인가?"
```

## 역할 배정

### 4인 진행 시

```
역할 1: 국토교통부 장관
  목표: 집값 안정 + 공급 확대
  딜레마: 어떤 공급이 실질적 효과를 내는가
  권한: 정책 입안권

역할 2: 다주택 임대인 대표 (3채 보유)
  배경: 60대, 노후를 임대 수익에 의존
  목표: 재산권 보호 + 세금 부담 완화
  딜레마: 사회적 비난 vs 정당한 투자
  권한: 임대 공급 축소 카드

역할 3: 무주택 청년 대표 (30대 직장인)
  배경: 서울 거주, 결혼 계획, 월세 75만 원
  목표: 내 집 마련 가능성 + 주거비 부담 완화
  딜레마: 지금 사야 하나 vs 더 기다려야 하나
  권한: 세대 여론 대변

역할 4: 건설업계 대표
  목표: 재건축 규제 완화 + 사업성 확보
  딜레마: 공급 확대 주장 vs 수익성 추구
  권한: 투자·고용 카드
```

### 6인 진행 시

```
위 4개 역할 + 다음 추가:

역할 5: 환경단체 대표
  목표: 그린벨트 보호 + 난개발 방지
  딜레마: 환경 보호 vs 주거 공급
  권한: 환경 영향 평가·소송 카드

역할 6: 전세 사기 피해자 대표
  목표: 피해 구제 + 전세 제도 개혁
  딜레마: 임시 구제 vs 제도 근본 개혁
  권한: 여론 동원 + 입법 청원
```

### 8인 진행 시

```
위 6개 역할 + 다음 추가:

역할 7: 지방 소도시 시장
  목표: 지역 인구 유지 + 빈집 문제 해결
  딜레마: 수도권 집중 완화 요구
  권한: 지방 정부 협의 카드

역할 8: 부동산 전문 경제학자
  목표: 객관적 데이터 기반 정책 제언
  딜레마: 학문적 진실 vs 정치적 현실
  권한: 전문가 권위
```

### 2단계: 정책 토론 (70-80분)

```
라운드 1 (20분): 현실 인식 공유
  각자 가장 시급한 문제 발표
  "지금 가장 고통받는 사람은 누구인가?"

라운드 2 (35분): 정책 협상
  단기 대책 vs 장기 구조 개편
  세금·공급·금융 정책 조합
  수혜자와 피해자 협상

라운드 3 (20분): 최종 정책 합의
  단기 (1-2년) 대책
  장기 (5-10년) 구조 개편
  재원 마련 방안
```

**진행자용 이벤트 카드:**

```
이벤트 1 (라운드 1 후):
  "서울 강남 한 아파트가 이번 달
   신고가를 경신하며 35억 원에 거래됐습니다.
   같은 날 청년 무주택자 커뮤니티에
   '집 포기 선언' 글이 쏟아졌습니다."
  → 양극화 현실 부각

이벤트 2 (공급 논의 시):
  "서울 그린벨트 해제 예정 지역 주민들이
   '환경 파괴 반대' 시위를 시작했습니다.
   해당 지역 시의원도 반대 성명을 냈습니다."
  → 공급 확대의 현실적 장벽

이벤트 3 (세금 논의 시):
  "다주택자 과세 강화 발표 후
   임대인들이 전월세 계약을 갱신하지 않겠다고
   선언하는 사례가 급증했습니다."
  → 세금 강화의 부작용

이벤트 4 (교착 상태 시):
  "전세 사기 피해자 한 명이
   보증금을 돌려받지 못해
   스스로 목숨을 끊었다는 뉴스가 보도됐습니다."
  → 감정적 긴장 고조

이벤트 5 (선택적):
  "싱가포르식 공공주택 모델 도입 시
   10년간 약 150조 원 예산이 필요하다는
   연구 결과가 발표됐습니다."
  → 공공 모델의 현실적 비용
```

### 3단계: 정책 선언 (10분)

```
발표 항목:
  ①핵심 정책 방향 (공급/수요/공공/분산)
  ②단기 대책 (1-2년)
  ③장기 구조 개편 (5-10년)
  ④재원 마련 방안
  ⑤가장 우선 보호할 계층
  ⑥포기하거나 양보할 부분
```

### 4단계: 성찰 토론 (25-30분)

```
토론 질문:

①"집값이 오르면 좋은 사람이 있고
  집값이 내리면 좋은 사람이 있습니다.
  정부는 어느 편을 들어야 하는가?"

②한국에서 부동산 정책이 번번이
  실패하는 이유는 무엇인가요?

③"내 집 마련은 개인의 노력으로
  달성해야 하는가, 국가가 보장해야 하는가?"

④싱가포르(공공주택 80%)와 한국을 비교하면
  무엇이 다른가요?

⑤만약 당신이 무주택 청년이라면,
  지금 집을 사야 할까요?
  이 질문에 정책이 어떻게 답해야 하나요?

⑥부동산 정책에서
  가장 과소 대표되는 목소리는 누구인가요?
```

## 진행자 가이드

### 감정적 쟁점 관리

```
다주택자 vs 무주택자 갈등:
  "다주택자도 불법을 한 것은 아닙니다.
   하지만 그 결과가 다른 사람의 기회를
   줄였다면 사회적 책임이 있을까요?"

환경 vs 공급:
  "그린벨트는 한번 해제하면
   되돌리기 어렵습니다.
   50만 호 공급이 그 가치와 바꿀 만한가요?"

세대 갈등:
  "기성 세대는 집값이 오르던 시대를 살았습니다.
   그것은 행운인가, 노력의 결과인가?"
```

### 현실 연결 질문

```
"지금 월세로 사는 분이 있다면
 그 경험이 이 정책 토론에서
 어떻게 느껴지시나요?"

"부모님이 집을 여러 채 갖고 있다면
 이 토론이 다르게 느껴질까요?"
```

## 실제 사례 참고

### 사례 1: 싱가포르 공공주택 (HDB)

```
정책: 전 국민 80%가 공공주택 거주
      정부가 토지 소유, 분양권 제한
결과: 안정적 주거 + 주거 불평등 최소화
교훈: 초기 강력한 국가 개입이 장기 안정 가능
      하지만 토지 국유화 등 근본 조건 필요
```

### 사례 2: 문재인 정부 부동산 (2017-2022)
