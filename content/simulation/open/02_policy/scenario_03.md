# 시나리오 03: 플랫폼 규제 딜레마

> 배달앱, 검색엔진, 소셜미디어, 쇼핑플랫폼.
> 디지털 플랫폼은 우리 삶을 편리하게 했지만
> 동시에 독점, 불공정, 감시, 중독의 문제를 낳았습니다.
> 혁신을 죽이지 않으면서 시장을 공정하게 만들 수 있는가?
> 규제의 칼날은 어디를 향해야 하는가?

---

| 항목 | 내용 |

## 배경: 플랫폼 경제의 현실

### 한국 플랫폼 시장 현황

```
배달 플랫폼:
  배달의민족 + 쿠팡이츠 시장 점유율: 95%+
  음식점 수수료: 매출의 15-27%
  배달 라이더: 약 30만 명 (특수고용직)
  소비자 배달비: 평균 3,000-5,000원

검색·광고 플랫폼:
  네이버 검색 점유율: 약 60%
  구글 점유율: 약 35%
  네이버 쇼핑 광고 논란 (검색 결과 조작 의혹)

쇼핑 플랫폼:
  쿠팡 시장 점유율: 약 25% (급성장)
  입점 판매자 수수료: 10-15%
  가격 통제·갑질 논란 지속

숙박·여행 플랫폼:
  야놀자·여기어때 복점 구조
  숙박업소 수수료: 10-20%

공통 문제:
  플랫폼 수수료 지속 인상
  입점 업체 종속성 심화
  데이터 독점 및 활용 논란
  플랫폼 노동자 권리 사각지대
```

### 글로벌 플랫폼 규제 동향

```
EU:
  디지털시장법 (DMA) 시행 (2023)
  게이트키퍼 (독점 플랫폼) 규제 강화
  구글·애플·메타 등 대상

미국:
  빅테크 반독점 소송 증가
  구글 검색 독점 판결 (2024)
  메타 인스타그램 분리 논의

한국:
  온라인 플랫폼 공정화법 추진 중
  공정거래위원회 플랫폼 조사 강화
  배달앱 수수료 규제 논의
```

## 핵심 사건: 배달플랫폼 수수료 갈등

### 사건 개요

```
발단:
  국내 1위 배달 플랫폼 '배달링'이
  음식점 수수료를 기존 15%에서 27%로
  대폭 인상한다고 공지했습니다.

배달링 현황:
  시장 점유율: 62%
  월간 주문 건수: 1억 5,000만 건
  가입 음식점: 약 45만 개
  배달 라이더: 18만 명
  연간 거래액: 약 35조 원
  모회사: 글로벌 배달 그룹 (독일계)

수수료 인상 배경:
  3년 연속 영업 손실
  글로벌 본사의 흑자 전환 압박
  라이더 비용 상승
  마케팅 비용 증가
```

### 이해관계자 반응

```
음식점 업주:
  "수수료 27%면 남는 게 없다"
  "메뉴 가격 올릴 수밖에 없다"
  "플랫폼 없이는 장사 안 된다"
  전국 자영업자 협의회 집단 반발

소비자:
  "음식 가격이 또 오르나"
  "배달비 이미 너무 비싸다"
  "그래도 편리함은 포기 못 해"

배달 라이더:
  "수수료 오르면 우리 수입도 오를 줄 알았는데"
  "실제로는 건당 단가 동결"
  "플랫폼이 수익만 챙긴다"

경쟁 플랫폼:
  "시장 독점이 이런 결과를 낳았다"
  "공정한 경쟁이 필요하다"

배달링:
  "지속 가능한 사업을 위해 불가피"
  "서비스 품질 향상에 재투자"
  "가격은 시장이 결정하는 것"
```

## 정책 옵션

### 옵션 A: 수수료 상한제 도입

```
핵심 내용:
  배달 플랫폼 수수료 법정 상한선 설정
  현행 유지 또는 15% 상한 제안
  위반 시 매출의 3% 과징금

찬성 논거:
  "독점 플랫폼의 수수료 착취 방지"
  "자영업자 보호"
  "소비자 가격 안정"

반대 논거:
  "가격 규제는 시장 왜곡"
  "플랫폼 투자 유인 감소"
  "혁신 저해"
  "결국 서비스 질 하락"

해외 사례:
  미국 뉴욕: 배달 수수료 23% 상한제 시행
  결과: 음식 가격 소폭 하락
         플랫폼 서비스 축소 우려
```

### 옵션 B: 플랫폼 분리 명령

```
핵심 내용:
  독점적 지위 남용 플랫폼에
  사업 부문 분리 명령
  배달링 → 배달 중개 + 직접 배달 분리
  데이터 공유 의무화

찬성 논거:
  "독점 구조 자체를 해소"
  "경쟁 촉진으로 자율 가격 조정"

반대 논거:
  "기업 분리는 극단적 조치"
  "글로벌 기업에 적용 어려움"
  "분리 후 서비스 품질 저하 우려"

해외 사례:
  미국 AT&T 분리 (1984)
  EU 구글 검색 분리 논의 중
```

### 옵션 C: 공공 플랫폼 설립

```
핵심 내용:
  정부 또는 지자체가 공공 배달 플랫폼 운영
  수수료: 2-5% (원가 수준)
  민간 플랫폼과 경쟁 구조 형성

찬성 논거:
  "수수료 경쟁으로 시장 가격 하락"
  "자영업자·소비자 선택권 확대"
  "데이터 공공 활용 가능"

반대 논거:
  "정부의 민간 시장 개입"
  "공공 플랫폼 운영 효율성 의문"
  "세금으로 민간 기업과 경쟁"
  "공주문 앱 실패 사례 (서울시)"

실제 사례:
  서울시 '제로배달 유니온': 수수료 2%
  결과: 점유율 1% 미만 (편의성 부족)
```

### 옵션 D: 자율 규제 + 투명성 강화

```
핵심 내용:
  수수료 산정 기준 공개 의무화
  알고리즘 투명성 요구
  플랫폼 노동자 보호 기준 마련
  플랫폼 간 데이터 이동권 보장

찬성 논거:
  "시장 자율에 맡기되 투명성 확보"
  "혁신 저해 최소화"
  "소비자·업주가 정보 기반 선택"

반대 논거:
  "자율 규제는 결국 자기 규제 실패"
  "독점 구조에서 투명성만으로 부족"
  "근본 문제 해결 못함"
```

## 추가 쟁점: 플랫폼 노동자 문제

```
배달 라이더 현황:
  법적 지위: 특수고용직 (사실상 자영업자)
  고용보험·산재보험: 일부만 적용
  수입: 월 200-350만 원 (불안정)
  사고 시 책임: 본인 부담

쟁점:
  플랫폼 노동자를 근로자로 인정할 것인가?

근로자 인정 시:
  4대 보험 의무 가입
  최저임금 적용
  → 플랫폼 비용 대폭 증가
  → 소비자 가격 인상 또는 서비스 축소

현행 유지 시:
  노동자 보호 사각지대 지속
  사고·질병 시 개인 책임
```

## 역할 배정

### 4인 진행 시

```
역할 1: 공정거래위원회 위원장
  목표: 공정한 시장 질서 확립
  딜레마: 규제 강도 vs 혁신 보호
  권한: 과징금·분리 명령 카드

역할 2: 배달링 한국 대표
  목표: 수익성 확보 + 사업 지속
  딜레마: 수수료 인상 불가피 vs 여론 악화
  권한: 서비스 축소·철수 카드

역할 3: 전국 자영업자 협의회 대표
  목표: 수수료 인하 + 공정한 계약
  딜레마: 플랫폼 없이는 생존 불가
  권한: 집단 이탈·파업 카드

역할 4: 소비자 단체 대표
  목표: 소비자 가격 보호 + 편의 유지
  딜레마: 규제가 오히려 가격 올릴 수도
  권한: 여론 동원 카드
```

### 6인 진행 시

```
위 4개 역할 + 다음 추가:

역할 5: 배달 라이더 노조 대표
  목표: 근로자 지위 인정 + 소득 보장
  딜레마: 지위 인정 시 일감 감소 가능
  권한: 파업·운행 거부 카드

역할 6: 스타트업 창업자 (신규 배달 플랫폼)
  목표: 공정한 경쟁 환경 조성
  딜레마: 규제가 기존 독점 보호할 수도
  권한: 혁신 대안 제시 카드
```

### 1단계: 배경 파악 (20분)

```
개인 확인:
  □ 플랫폼 수수료 구조 이해
  □ 각 이해관계자 입장 파악
  □ 자신의 역할 목표와 카드 확인

입장 정리:
  수수료 상한 설정 찬성/반대: ___
  공공 플랫폼 설립 찬성/반대: ___
  라이더 근로자 인정 찬성/반대: ___
  핵심 우선순위:
    □ 자영업자 보호
    □ 소비자 가격
    □ 플랫폼 혁신
    □ 노동자 권리
    □ 시장 공정성
```

### 2단계: 정책 토론 (60-70분)

```
라운드 1 (15분): 현황 인식 공유
  각자 가장 심각한 문제 발표
  우선 해결 과제 제시

라운드 2 (30분): 정책 협상
  수수료 규제 방향 협상
  플랫폼 노동자 지위 논의
  공공 플랫폼 찬반 토론

라운드 3 (20분): 최종 정책 패키지
  단기 대응책 + 장기 구조 개편
  각 이해관계자 수용 여부
```

**진행자용 이벤트 카드:**

```
이벤트 1 (라운드 1 후):
  "배달링이 수수료 인상에 반대하는

이벤트 2 (교착 상태 시):
  "배달 라이더 3명이 이번 달

  → 노동자 보호 이슈 부각

이벤트 3 (규제 논의 시):
  "배달링 본사가 규제 강화 시
   한국 시장 철수를 검토하겠다고
   비공식 발표했습니다."
  → 규제의 역효과 경고

이벤트 4 (선택적):
  "EU가 배달 플랫폼 노동자를

   최종 확정했습니다."
  → 글로벌 기준 변화
```

### 3단계: 정책 선언 (10분)

```
발표 항목:
  ①수수료 규제 방식과 수준
  ②플랫폼 노동자 지위
  ③공공 플랫폼 설립 여부
  ④데이터 투명성 요구 사항
  ⑤단계별 시행 일정
```

### 4단계: 성찰 토론 (20-30분)

```
토론 질문:

①"플랫폼 독점은 나쁜가?"
  독점의 장점(효율성, 규모의 경제)과
  단점(착취, 혁신 저해)을 비교하세요.

②"정부가 가격을 규제하면
  항상 부작용이 생긴다."
  이 주장에 동의하나요?

③배달앱 없이 살 수 있나요?
  우리는 얼마나 플랫폼에 종속되어 있나요?

④플랫폼 노동자는 노동자인가,
  자영업자인가?
  그 구분이 왜 중요한가요?

⑤"한국이 글로벌 플랫폼을
  규제하는 것이 가능한가?"
  현실적 한계는 무엇인가요?
```

## 진행자 가이드

### 핵심 긴장 유지

```
혁신 vs 규제:
  "규제를 강화하면 배달링이 철수할 수 있습니다.
   그러면 소비자와 자영업자는 어떻게 될까요?"

자영업자 딜레마:
  "플랫폼이 없으면 장사가 안 됩니다.
   하지만 수수료가 너무 높으면 남는 것도 없습니다.
   이 딜레마를 어떻게 해결할 수 있을까요?"

소비자 편의 vs 가격:
  "규제로 수수료가 낮아져도
   플랫폼이 배달비를 올리면
   소비자 부담은 같습니다."
```

## 실제 사례 참고

### 사례 1: 미국 뉴욕 배달 수수료 상한제

```
규제: 2023년 15% + 기타 수수료 총 23% 상한
결과: 음식점 수익 일부 개선
      배달링·도어대시 서비스 일부 축소
      배달비 소폭 인상 (소비자 전가)
교훈: 규제의 효과는 제한적, 부작용 존재
```

### 사례 2: 한국 공주문 앱 (서울시)

```
정책: 서울시 공공 배달앱 운영 (수수료 2%)
결과: 가입 음식점 적음, 이용자 극소수
      2023년 사실상 실패 인정
교훈: 편의성 없는 공공 플랫폼은 외면
```

### 사례 3: EU 디지털시장법 (DMA)

```
규제: 게이트키퍼 (빅 플랫폼) 행위 규제
      자사 서비스 우대 금지
      데이터 이동권 보장
결과: 구글·애플·메타 등 대응 중
      실효성은 아직 미지수
```
