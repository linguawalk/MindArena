# 시나리오 02: 에너지 전환 정책

> 탄소중립 2050을 선언한 대한민국.
> 원전, 재생에너지, 화석연료 사이에서
> 어떤 속도로, 어떤 방식으로 전환할 것인가?
> 경제, 환경, 안보, 지역사회가 충돌하는
> 에너지 정책의 딜레마가 시작됩니다.

## 배경: 한국의 에너지 현실

### 현황 데이터

```
에너지 현황 (2024년):
  전력 생산 비중:
    석탄:      30.2%
    원자력:    27.4%
    LNG:       28.3%
    재생에너지: 10.1%
    기타:       4.0%

  에너지 수입 의존도: 93% (세계 최고 수준)
  연간 에너지 수입액: 약 200조 원
  전력 요금: OECD 최하위 수준

탄소 배출:
  연간 CO₂ 배출: 약 6억 톤 (세계 11위)
  2030년 감축 목표: 2018년 대비 40% 감축
  2050년 목표: 탄소중립

원전 현황:
  가동 원전: 26기
  설계 수명 만료 예정: 10기 (2030년까지)
  신규 건설 중: 3기 (신한울 3·4호기 등)
```

### 에너지 전환의 핵심 과제

```
기술적 과제:
  재생에너지 간헐성 (해가 없으면 태양광 0)
  에너지 저장 시스템 (ESS) 부족
  송전망 인프라 노후화
  재생에너지 적합 부지 부족 (산지·해안 갈등)

경제적 과제:
  전력 요금 인상 불가피 (현재 원가 이하 판매)
  한국전력 누적 적자: 30조 원 이상
  에너지 집약 산업 (철강·석유화학) 경쟁력 약화

안보적 과제:
  에너지 수입 의존도 93% → 공급 불안
  러시아-우크라이나 전쟁 후 에너지 안보 중요성 부각
  원전 폐기 시 전력 공급 부족 우려

사회적 과제:
  원전 지역 주민 수용성
  태양광·풍력 발전소 지역 갈등
  석탄 발전소 폐쇄 지역 고용 문제
```

## 정책 옵션

### 옵션 A: 원전 중심 저탄소 전환

```
핵심 내용:
  기존 원전 수명 연장 (최대 20년)
  신규 원전 추가 건설 (2기 추가)
  소형모듈원자로 (SMR) 개발 투자
  재생에너지 보완적 확대 (30% 목표)
  석탄 발전 2035년까지 단계적 폐지

전력 비중 목표 (2040년):
  원자력: 40% / 재생에너지: 35%
  LNG: 20% / 기타: 5%

비용: 10년간 약 120조 원
전기요금 인상: 약 15-20%

찬성 논거:
  "탄소 없는 안정적 전력 공급"
  "재생에너지보다 낮은 발전 단가"
  "에너지 안보 강화"
  "한국 원전 기술의 수출 경쟁력"

반대 논거:
  "후쿠시마·체르노빌 같은 사고 위험"
  "핵폐기물 처리 문제 미해결"
  "원전은 건설에 10-15년 소요"
  "글로벌 탈원전 추세에 역행"
```

### 옵션 B: 재생에너지 급속 전환

```
핵심 내용:
  태양광·풍력 대규모 투자
  재생에너지 비중 2040년 60% 목표
  원전 설계 수명 종료 후 순차 폐기
  ESS (에너지 저장 장치) 대규모 구축
  스마트 그리드 전국 확대

전력 비중 목표 (2040년):
  재생에너지: 60% / LNG: 20%
  원자력: 15% / 기타: 5%

비용: 10년간 약 300조 원
전기요금 인상: 약 40-50%

찬성 논거:
  "기후 위기 대응 가장 빠른 방법"
  "에너지 자립도 향상 가능"
  "신산업·일자리 창출"
  "원전 사고 위험 제거"

반대 논거:
  "재생에너지 간헐성으로 전력 불안"
  "전기요금 급등으로 산업 경쟁력 약화"
  "설치 부지 부족 (국토 면적 한계)"
  "ESS 기술 아직 불완전"
```

### 옵션 C: 균형 전환 (점진적 믹스)

```
핵심 내용:
  원전 현수준 유지 (수명 연장, 신규 최소화)
  재생에너지 40% 목표 (현실적 속도)
  LNG 브릿지 연료로 활용
  수소 경제 병행 투자
  석탄 2030년까지 완전 폐지

전력 비중 목표 (2040년):
  재생에너지: 40% / 원자력: 25%
  LNG: 25% / 수소·기타: 10%

비용: 10년간 약 180조 원
전기요금 인상: 약 25-30%

찬성 논거:
  "현실적 전환 속도"
  "전력 안정성 유지"
  "경제적 충격 최소화"
  "기술 발전에 유연하게 대응"

반대 논거:
  "2050 탄소중립 달성 어려움"
  "LNG도 탄소 배출원"
  "명확한 방향 없이 정치적 타협만"
```

### 옵션 D: 에너지 절약 + 효율화 우선

```
핵심 내용:
  에너지 소비 자체를 줄이는 데 집중
  건물 에너지 효율 기준 강화
  전기요금 현실화 (원가 반영)
  에너지 집약 산업 구조 전환 지원
  시민 에너지 절약 문화 정착

목표:
  2030년까지 에너지 소비 20% 감소
  수요 감소로 공급 전환 부담 완화

비용: 10년간 약 50조 원
전기요금 인상: 즉각 30% (현실화)

찬성 논거:
  "가장 친환경적 에너지는 안 쓰는 에너지"
  "공급 전환보다 수요 감소가 빠름"
  "장기적으로 에너지 비용 절감"

반대 논거:
  "산업 경쟁력 약화"
  "국민 생활 수준 저하"
  "요금 인상에 대한 정치적 저항"
  "근본 전환 없이는 한계"
```

## 핵심 쟁점

```
쟁점 1: 원전의 역할
  "원전은 기후 위기 해결사인가,
   새로운 위험인가?"

쟁점 2: 전환 속도
  "빠른 전환(환경 우선)과
   느린 전환(경제 안정) 중 무엇이 맞는가?"

쟁점 3: 비용 부담
  "에너지 전환 비용을 누가 부담해야 하는가?
   기업? 소비자? 정부?"

쟁점 4: 에너지 안보
  "93% 수입 의존에서 자립도를 높이려면
   원전과 재생에너지 중 무엇이 더 효과적인가?"

쟁점 5: 지역 갈등
  "원전·태양광·풍력 발전소는 모두
   '우리 동네는 안 된다'는 님비 현상에 직면합니다.
   어떻게 해결할 것인가?"
```

## 역할 배정

### 4인 진행 시

```
역할 1: 산업통상자원부 장관
  목표: 안정적 전력 공급 + 탄소중립 달성
  딜레마: 원전 유지 vs 재생에너지 전환
  입장: 균형 전환 선호
  권한: 에너지 정책 입안권

역할 2: 환경부 장관
  목표: 탄소 감축 + 생태계 보호
  딜레마: 재생에너지도 환경 파괴 가능
  입장: 재생에너지 확대 + 수요 감축
  권한: 환경 영향 평가권

역할 3: 대형 제조업체 대표 (철강)
  목표: 전기요금 안정 + 탄소 규제 대응
  딜레마: 탄소 감축 의무 vs 원가 상승
  입장: 원전 유지로 저렴한 전기 공급
  권한: 산업계 여론 + 고용 카드

역할 4: 환경단체 대표
  목표: 원전 폐기 + 재생에너지 전환
  딜레마: 재생에너지 속도 vs 환경 파괴
  입장: 빠른 탈원전 + 재생에너지
  권한: 여론 동원 + 소송 카드
```

### 6인 진행 시

```
위 4개 역할 + 다음 추가:

역할 5: 원전 지역 주민 대표
  배경: 경북 경주 원전 인근 주민
  목표: 안전 보장 + 지역 경제 유지
  딜레마: 원전 폐기 시 지역 경제 붕괴
  입장: "폐기하려면 우리 지역 대책 먼저"
  권한: 지역 여론 + 사회적 갈등 카드

역할 6: 재생에너지 기업 대표
  배경: 태양광·풍력 사업자
  목표: 재생에너지 시장 확대
  딜레마: 부지 확보·주민 동의 어려움
  입장: "정부가 규제 완화와 지원을 해야"
  권한: 투자 및 고용 창출 카드
```

### 8인 진행 시

```
위 6개 역할 + 다음 추가:

역할 7: 기후 과학자
  목표: 기후 위기 데이터 기반 정책 촉구
  딜레마: 과학적 사실 vs 정치적 현실
  입장: "2030년까지 석탄 완전 폐지 필수"
  권한: 과학적 권위

역할 8: 일반 시민 대표 (중산층 가정)
  목표: 전기요금 부담 최소화
  딜레마: 환경 걱정 vs 생활비 부담
  입장: "좋은 정책이지만 내 요금은 오르면 안 돼"
  권한: 소비자 여론
```

## 실제 사례 참고

### 사례 1: 독일의 에너지 전환 (Energiewende)

```
전략: 2011년 탈원전 선언, 재생에너지 집중
현실: 전기요금 유럽 최고 수준
      러시아 가스 의존도 증가
      2022년 전쟁 후 에너지 위기
교훈: 에너지 안보 없는 전환의 위험
```

### 사례 2: 프랑스의 원전 유지

```
전략: 전력의 70% 이상 원전 (세계 최고)
현실: 낮은 탄소 배출 + 저렴한 전기요금
      노후 원전 관리 비용 급증
교훈: 원전의 장기 비용 과소평가 주의
```

### 사례 3: 덴마크의 재생에너지 성공

```
전략: 풍력 중심 재생에너지 전환
현실: 전력의 50%+ 풍력 (세계 선도)
      해상풍력 기술 수출 산업화
교훈: 지리적 조건 활용 + 장기 일관성
      (한국은 풍황 조건 덴마크보다 불리)
```

### 사례 4: 한국 에너지 정책 변동

```
문재인 정부: 탈원전 + 재생에너지
윤석열 정부: 원전 확대 + 재생에너지 보완
문제: 정권 교체마다 정책 방향 바뀜
교훈: 에너지 정책은 10-30년 장기 계획
      정치적 안정성이 핵심
```
