## 카테고리별 특징

### 퍼즐 (Puzzle)

```
대상: 개인
형식: 단계별 문제 풀기
목적: 논리·수학·언어·공간 사고력 훈련
특징: 1~15단계 난이도 체계
      입문 → 기초 → 발전 → 심화 → 마스터
```

### 보드게임 (Board Game)

```
대상: 2-8명
형식: 규칙 기반 게임 (승패 있음)
목적: 전략적 사고·협상·추리 훈련
특징: 30-120분 소요
      실제 플레이 가능한 완성 게임
```

### 시뮬레이션 Open

```
대상: 4-8명 (일반인·학생)
형식: 역할극 + 토론 (승패 없음)
목적: 의사결정·협상·현실 이해
특징: 1-2시간 소요
      무료 배포
```

### 시뮬레이션 Pro

```
대상: 8-30명 팀 단위 (전문가·조직)
형식: 장기 몰입 시뮬레이션
목적: 전문 역량 훈련·조직 학습
특징: 최소 3일 이상 소요
      B2B 교육 활용
```

| 비용 | 무료 | B2B 유료 |

## 활용 가이드

### 교육 기관

```
초·중·고:
  퍼즐 (01_number ~ 05_spatial)
  보드게임 (01_abstract ~ 03_knowledge)

대학·대학원:
  보드게임 전체
  시뮬레이션 Open 전체
  시뮬레이션 Pro (전공 연계)
```

### 기업·조직

```
팀 빌딩:
  보드게임 (04_cooperative ~ 06_negotiation)
  시뮬레이션 Open (1-2시간)

전문 교육:
  시뮬레이션 Pro (분야별 선택)
  맞춤형 시나리오 개발 가능
```

### 시민 교육

```
일반 시민:
  시뮬레이션 Open/국가정책 전체
  사회 현안 이해 + 토론

청년:
  시뮬레이션 Open 전체
  미래 시나리오 (Pro/05_future)
```
