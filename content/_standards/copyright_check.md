# MindArena 저작권 체크리스트

> 모든 콘텐츠 제작 시 반드시 확인해야 합니다.
> 저작권 문제는 플랫폼 운영의 핵심 리스크입니다.

---

## 기본 원칙

### 안전하게 사용할 수 있는 것
- ✅ 직접 창작한 문제·해설·예시
- ✅ 수학적 사실·공식·정리 (저작권 없음)
- ✅ 게임 규칙·알고리즘 (저작권 없음)
- ✅ 역사적 사실·과학적 개념 (저작권 없음)
- ✅ 공공 도메인 자료 (저작권 만료)
- ✅ 오픈소스 라이선스 자료 (라이선스 확인 후)

### 사용 금지 또는 주의가 필요한 것
- ❌ 타 출판사 문제집의 문제 직접 복사
- ❌ 수능·모의고사 문제 직접 사용
- ❌ 저작권 있는 도서의 지문 직접 인용
- ❌ 특정 시험 기관의 문제 무단 사용
- ⚠️ 유명 퍼즐 (스도쿠·루미큐브 등): 규칙은 가능, 특정 문제집 내용은 불가
- ⚠️ 실존 인물 관련 내용: 사실에 근거하지 않은 내용 금지

## 문제 창작 안전 가이드

### 기존 문제를 참고할 때
1. **아이디어만 참고**: 문제 구조·주제만 가져옴.
2. **완전히 새로 작성**: 문장·숫자·조건을 모두 바꿈.
3. **출처 없이 제출**: 완전히 새 창작물로 만듦.

### 피해야 할 패턴
```
❌ 나쁜 예:
"다음은 2024년 수능 수학 문제입니다. [원문 그대로]"

✅ 좋은 예:
비슷한 유형의 문제를 처음부터 직접 창작.
숫자·조건·서술을 모두 새로 만듦.
```

### 영감 vs 표절의 경계
- **영감** (허용): 같은 유형의 문제, 유사한 수학 개념.
- **표절** (금지): 문장·숫자·조건이 동일하거나 거의 같음.

## 특수 콘텐츠 주의사항

### 시뮬레이션 시나리오
- ✅ 실제 사건을 기반으로 하되 **픽션임을 명시**
- ✅ 역사적 사실은 자유롭게 활용
- ❌ 실존 인물의 특정 발언을 가공하여 사용 금지
- ❌ 특정 정당·기업·단체를 직접 지목하여 비판 금지

### 보드게임 소개
- ✅ 게임 규칙 설명 가능
- ✅ 전략·전술 분석 가능
- ❌ 게임 로고·캐릭터·아트워크 무단 사용 금지
- ❌ 게임 이름을 상업적으로 혼동 유발하는 방식 사용 금지

### AI 생성 콘텐츠
- ✅ AI로 생성한 문제·해설 사용 가능
- ✅ Claude와 협업하여 만든 콘텐츠는 우리 저작물
- ⚠️ AI가 기존 저작물을 그대로 재현할 수 있으므로 검토 필요
- ⚠️ 유명 문제집과 유사한 내용은 수정 후 사용

## 저작권 분쟁 예방 체크리스트

콘텐츠 업로드 전 다음을 확인하세요.

### 텍스트 콘텐츠
- [ ] 직접 창작한 내용인가?
- [ ] 타 출처에서 문장을 그대로 가져오지 않았는가?
- [ ] 현대 문학 작품의 지문을 무단 사용하지 않았는가?
- [ ] 수능·모의고사 기출 문제를 그대로 쓰지 않았는가?

### 수치·데이터
- [ ] 통계·데이터의 출처가 공공 기관인가?
- [ ] 출처가 필요한 경우 표기했는가?

### 문화·역사 콘텐츠
- [ ] 역사적 사실에 근거하는가?
- [ ] 특정 집단을 불필요하게 비하하지 않는가?
- [ ] 정치적 편향이 없는가?
