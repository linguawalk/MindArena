# MindArena 언어 트랙 커리큘럼

**목표:** 모국어 깊이 + 외국어 실용 + 언어로 다른 분야 게임 플레이 (CLIL)
**방식:** 퍼즐(40%) + AI 대화 게임(60%)
**특징:** 언어 자체를 배우는 것과 언어로 배우는 것을 동시에 제공
**범위:** 한국어 → 언어학 → 영어 → 중국어·일본어 → CLIL → 언어와 사회

---

## 교육 방식 상세

```
퍼즐 (40%)
  어휘·문법·독해·청해 문제
  맞춤법·표현 오류 찾기
  텍스트 분석·요약·추론

AI 대화 게임 (60%)
  대화 파트너형:  AI가 원어민으로 상황별 대화
  교정형:         AI가 사용자 표현을 교정·개선
  토론형:         외국어로 인문사회 주제 토론
  역할극형:       공항·회의·협상 등 실전 상황
  CLIL형:         다른 분야를 외국어로 학습

AI 역할 유형
  원어민 대화 상대:  자연스러운 일상 대화
  언어 교사:         오류 분석·교정·설명
  원어민 면접관:     취업·입학 면접 시뮬레이션
  비즈니스 파트너:   협상·프레젠테이션 상황
  문화 가이드:       언어와 문화를 함께 설명

언어 레벨 체계
  L1 Seed:    입문 (기초 표현)
  L2 Sprout:  초급 (일상 대화)
  L3 Branch:  중급 (주제별 토론)
  L4 Tree:    고급 (전문 분야)
  L5 Forest:  최고급 (언어로 창작·분석)
```

---

## 커리큘럼 구조

```
총 30개 단원 × 평균 6개 역량 = 약 180개 핵심 역량

파트 1: 한국어 (모국어) — 단원 1-8
파트 2: 언어학 기초 — 단원 9-11
파트 3: 영어 — 단원 12-20
파트 4: 중국어 — 단원 21-23
파트 5: 일본어 — 단원 24-26
파트 6: CLIL + 언어와 사회 — 단원 27-30
```
