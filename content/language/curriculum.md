# MindArena 언어 트랙 커리큘럼 (한국어)

**목표:** 한국어를 깊이 이해하고 정확하게 사용하며 창의적으로 표현하는 능력
**방식:** 퍼즐(40%) + AI 대화 게임(60%)
**범위:** 문법 기초 → 어휘·표현 → 읽기·쓰기 → 말하기·듣기 → 문학 → 언어학 → 창작
**수준:** 수능 국어 → 대학 글쓰기 → 전문 언어 능력

---

## 교육 방식 상세

```
퍼즐 (40%)
  문법·맞춤법·어휘 정확성 확인
  독해·추론·비판적 읽기
  언어 구조 분석

AI 대화 게임 (60%)
  교정형:     AI가 사용자 표현을 교정·개선
  토론형:     주제에 대해 한국어로 논증 대결
  창작형:     AI가 창작 파트너·편집자 역할
  분석형:     AI와 함께 텍스트·문학 작품 분석
  역할극형:   면접·발표·협상 등 실전 상황

AI 역할 유형
  언어 교사:     오류 분석·교정·설명
  편집자:        글쓰기 피드백·개선 제안
  토론 상대:     반론·심화 질문 제시
  독자:          사용자 글에 반응하는 일반 독자
  문학 평론가:   작품 해석·감상 토론
```

---

## 커리큘럼 구조

```
22개 단원 × 평균 7개 역량 = 약 154개 핵심 역량

영역 1: 문법과 어법         단원 1-5   (L1 Seed ~ L2 Sprout)

영역 5: 말하기와 듣기       단원 18-19 (L3 Branch)
영역 6: 한국 문학           단원 20-21 (L4 Tree)
영역 7: 언어학과 언어 문화  단원 22    (L5 Forest)
```
