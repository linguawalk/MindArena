# MindArena 융합 트랙 커리큘럼

**목표:** 분야 경계를 넘어 복합적 문제를 해결하는 최고 수준의 사고 훈련
**방식:** AI가 각 역량에 맞는 문제를 실시간 생성
**특징:** 다른 5개 트랙을 모두 이수하지 않아도 접근 가능하되
         깊이 있는 융합은 각 트랙 완료 후 해금

---

## 커리큘럼 구조

```
15개 단원 × 평균 6개 역량 = 약 90개 핵심 역량
특징: MindArena만의 독자적 콘텐츠
      모든 트랙의 개념을 연결하는 허브
```

## 전체 트랙 간 연결 지도

```
수학 ←→ 과학    (물리·화학의 수식)
수학 ←→ 프로그래밍  (알고리즘 수학)
수학 ←→ 예체능  (음악 수학·스포츠 역학)
수학 ←→ 기술AI  (머신러닝 수학)
과학 ←→ 기술AI  (양자컴퓨팅·바이오테크)
과학 ←→ 예체능  (스포츠 역학·음악 물리)
모두 ←→ 융합    (복잡계·정보이론·미래설계)
```
